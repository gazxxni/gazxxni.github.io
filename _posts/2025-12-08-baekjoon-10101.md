## 문제
창영이는 삼각형의 종류를 잘 구분하지 못한다. 따라서 프로그램을 이용해 이를 외우려고 한다.

삼각형의 세 각을 입력받은 다음, 


	세 각의 크기가 모두 60이면, Equilateral
	세 각의 합이 180이고, 두 각이 같은 경우에는 Isosceles
	세 각의 합이 180이고, 같은 각이 없는 경우에는 Scalene
	세 각의 합이 180이 아닌 경우에는 Error


를 출력하는 프로그램을 작성하시오.

---

## 입력
총 3개의 줄에 걸쳐 삼각형의 각의 크기가 주어진다. 모든 정수는 0보다 크고, 180보다 작다.

---

## 출력
문제의 설명에 따라 Equilateral, Isosceles, Scalene, Error 중 하나를 출력한다.

## 풀이

### 풀이 핵심 로직
삼각형의 세 각을 입력받아 각의 값에 따라 삼각형의 종류를 판별하는 문제입니다. 각이 모두 60도이면 'Equilateral' 삼각형이고, 각의 합이 180도이며 두 각이 같은 경우에는 'Isosceles', 모든 각이 서로 다르면 'Scalene'으로 판별합니다. 만약 각의 합이 180도가 아닐 경우 'Error'를 출력합니다.

### 동작 과정
1. **입력**: 세 각을 입력받는다. 예를 들어, `60`, `60`, `60`을 입력받으면 각각 `a`, `b`, `c`에 저장된다.
2. **Equilateral 판별**: 모든 각이 60도인지 확인한다. (`if a == b == c == 60:`) 이 경우 'Equilateral'을 출력.
3. **각의 합 검사**: 세 각의 합이 180인지 확인한다. (`elif a + b + c == 180:`)
   - 두 각이 같은 경우는 다음과 같이 판별한다:
     - `a`와 `b`가 같고, `c`는 다를 때 'Isosceles'.
     - `a`와 `c`가 같고, `b`는 다를 때 'Isosceles'.
     - `b`와 `c`가 같고, `a`는 다를 때 'Isosceles'.
   - 모든 각이 다를 경우 'Scalene'을 출력.
4. **Error 판별**: 위의 모든 조건을 만족하지 않으면 'Error'를 출력.

예시 입력으로 `60`, `60`, `60`을 주면 첫 번째 조건에 해당하므로 'Equilateral'이 출력된다. `60`, `60`, `60`을 주면 두 번째 조건에 해당하므로 'Isosceles'가 출력된다. `60`, `70`, `50`을 주면 세 번째 조건에 해당하므로 'Scalene'이 출력된다. 마지막으로 `60`, `60`, `80`을 주면 모든 조건을 만족하지 않으므로 'Error'가 출력된다.

### 시간 복잡도
이 알고리즘은 입력된 각을 상수 시간 내에 검사하므로 시간 복잡도는 O(1)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
c = int(input())


if a == b == c == 60:
    print('Equilateral')

elif a + b + c == 180:
    if a == b and b != c:
        print('Isosceles')
    elif a == c and b != c:
        print('Isosceles')
    elif b == c and a != c:
        print('Isosceles')
    else:
        print('Scalene')

else:
    print('Error')
```
