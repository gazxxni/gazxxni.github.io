## 풀이

### 풀이 핵심 로직
이 문제는 주어진 수 `s`보다 작거나 같은 최대 이른바 "삼각수"를 구하는 문제입니다. 삼각수는 1부터 n까지의 합으로, 해당 수는 `T_n = n * (n + 1) / 2`로 정의됩니다. 주어진 `s`에 대해 `n`을 찾기 위해, 이 공식을 변형하여 이차 방정식의 형태로 만들어 `n`의 값을 직접 계산합니다.

### 동작 과정
1. **입력**: 사용자로부터 정수 `s`를 입력받습니다. 예를 들어, `s = 10`이라고 가정합시다.
2. **이차 방정식 변형**: 삼각수의 공식을 `T_n = n * (n + 1) / 2`에서 `T_n <= s`를 만족하는 가장 큰 `n`을 찾기 위해 변형합니다. 이를 통해 `n`에 대한 이차 방정식을 얻습니다:
   \[ n^2 + n - 2s \leq 0 \]
3. **해 찾기**: 이차 방정식의 근의 공식을 사용하여 `n`을 구합니다. 이를 통해 `n`은 약식으로 다음과 같이 계산됩니다:
   \[ n = \frac{-1 + \sqrt{1 + 8s}}{2} \]
4. **소수점 이하 버리기**: `n`은 정수여야 하므로, 최종적으로 계산된 `n`을 정수로 변환하여 출력합니다.

예시 입력 `s = 10`인 경우:
- \( 1 + 8 * 10 = 81 \)
- \( \sqrt{81} = 9 \)
- \( n = \frac{-1 + 9}{2} = 4 \) (정수 변환 후 `4`가 출력됨)

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(1) \)입니다. 입력을 받고 계산하는 과정에서 반복문이나 재귀 호출이 없기 때문에 상수 시간에 실행됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

s = int(input())

a = ((1 + 8 * s) ** 0.5 - 1) / 2

print(int(a))
```
