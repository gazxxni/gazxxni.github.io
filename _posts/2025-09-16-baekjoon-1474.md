## 문제
세준이는 N개의 영어 단어를 이용해 길이가 M인 새로운 단어를 만들려고 한다. 새로운 단어는 N개의 단어를 순서대로 이어 붙이고, 각 단어의 사이에 _을 넣어서 만든다. 이렇게 만든 새로운 단어의 길이가 M이 아닌 경우 _를 추가해서 길이가 M이 되게 만들어야 한다.


	_는 단어와 단어 사이에만 추가할 수 있다. 따라서, 새로운 단어는 _으로 시작하거나, _로 끝날 수 없다.
	단어와 단어 사이에 있는 _의 개수는 모두 같아야 한다.
	
		모두 같게 만드는 것이 불가능한 경우 단어와 단어 사이에 있는 _의 개수의 최댓값과 최솟값의 차이는 1이 되어야 한다.
	
	


새로운 단어 중 사전 순으로 가장 앞서는 단어를 구해보자.

---

## 입력
첫째 줄에 N과 M이 주어진다. 둘째 줄부터 N개의 줄에 단어가 한 줄에 하나씩 주어진다.

---

## 출력
첫째 줄에 사전 순으로 가장 앞서는 단어를 출력한다.

알파벳 대문자, 소문자, 밑 줄의 순서는 'A'

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 N개의 단어를 이어 붙여 M 길이의 문자열을 만드는 과정에서, 각 단어 사이에 들어갈 밑줄(_)의 개수를 균등하게 나누는 것이 핵심입니다. 우리는 총 필요한 밑줄의 개수를 계산하고, 이를 N-1개의 구간에 고르게 나누되, 나눌 수 없는 경우에는 최대 1개의 차이가 나는 형태로 조정합니다. 이를 통해 사전순으로 가장 앞서는 문자열을 생성합니다.

### 동작 과정
1. **입력 처리**: N과 M을 입력받고, N개의 단어를 리스트로 저장합니다.
2. **총 길이 계산**: 각 단어의 길이를 모두 더하여 `total`을 구합니다.
3. **필요한 밑줄 계산**: `needed_underscores = M - total`을 통해 필요한 총 밑줄의 개수를 계산합니다.
4. **균등 분배**:
   - `aa = needed_underscores // (N - 1)`로 각 구간에 기본적으로 넣어야 할 밑줄 수를 계산합니다.
   - `bb = needed_underscores % (N - 1)`로 남은 밑줄 수를 계산합니다.
   - 각 구간에 `aa` 개의 밑줄을 넣고, 남은 `bb` 개의 밑줄을 첫 `bb` 개 구간에 하나씩 추가합니다.
5. **문자열 생성**: 첫 번째 단어를 결과 리스트에 추가한 후, 각 구간의 밑줄과 단어를 순서대로 추가하여 최종 문자열을 만듭니다.
6. **결과 출력**: 최종 문자열을 출력합니다.

예시 입력:
```
3 17
apple
banana
grape
```
- `N = 3`, `M = 17`
- 단어들의 총 길이는 15.
- 필요한 밑줄 수는 `17 - 15 = 2`.
- 각 구간에 넣을 밑줄 수는 `2 // 2 = 1`, 남은 밑줄 `2 % 2 = 0`.
- 따라서 각 구간에 1개의 밑줄을 삽입하면 `1, 1`로 고르게 나누어 1개의 밑줄이 추가됩니다.
- 최종 문자열은 `apple_banana_grape`가 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 단어를 입력받고 길이를 계산하는 과정에서 O(N) 시간이 필요하며, 문자열을 출력하는 과정도 O(N) 시간이 소요됩니다. 따라서 전체적인 시간 복잡도는 O(N)입니다.

## 코드
```python
n, m = map(int, input().split())
arr = [input().rstrip() for _ in range(n)]

def greedy(a, b):
    aa = a // b
    bb = a % b

    result = [aa] * b
    for i in range(b):
        if ord(arr[i + 1][0]) >= 97 and bb > 0:
            result[i] += 1
            bb -= 1
    
    for i in range(b - 1, 0, -1):
        if result[i] == aa and bb > 0:
            result[i] += 1
            bb -= 1

    result2 = ["_" * x for x in result]
    return result2

total = 0
for i in range(n):
    total += len(arr[i])

result = greedy(m - total, n- 1)

ans = [arr.pop(0)]
for i in range(n - 1):
    ans.append(result[i])
    ans.append(arr[i])

print(*ans, sep = '')
```
