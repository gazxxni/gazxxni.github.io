## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 1부터 N까지의 수를 이어서 썼을 때의 전체 자릿수를 계산하는 것입니다. 각 자리수의 범위에 따라 몇 개의 수가 있는지 계산하고, 각 자리수의 개수에 해당하는 곱을 더하여 최종 자릿수를 구합니다.

### 동작 과정
1. 입력값 `N`이 주어집니다.
2. `N`의 자리수 `l`을 구합니다. 예를 들어, `N = 23`일 경우 `l = 2`입니다.
3. 1자리 수 (1~9)부터 `l-1`자리 수 (10~99)까지의 자릿수를 계산합니다.
   - 1자리 수: 9개 (1~9), 각 수는 1자리 → 총 자릿수 = 9 * 1 = 9
   - 2자리 수: 90개 (10~99), 각 수는 2자리 → 총 자릿수 = 90 * 2 = 180
4. 마지막으로 `N`의 자리수인 `l`자리 수 (100~N)에서 남은 수의 자릿수를 계산합니다.
   - 예를 들어, `N = 23`인 경우, 10~23까지의 수가 14개이고, 각 수는 2자리 → 총 자릿수 = (23 - 10 + 1) * 2 = 28
5. 모든 자릿수를 합산하여 출력합니다.

예시 입력으로 `N = 23`일 경우:
- 1자리 수 자릿수: 9
- 2자리 수 자릿수: 28
- 총 자릿수: 9 + 28 = 37

### 시간 복잡도
O(log N) - 입력값 `N`의 자릿수를 구하는 것은 로그 시간 복잡도를 가집니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
