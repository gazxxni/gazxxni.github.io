## 문제
무인도에 갇힌 사람들을 구명보트를 이용하여 구출하려고 합니다. 구명보트는 작아서 한 번에 최대 2명씩 밖에 탈 수 없고, 무게 제한도 있습니다.

예를 들어, 사람들의 몸무게가 [70kg, 50kg, 80kg, 50kg]이고 구명보트의 무게 제한이 100kg이라면 2번째 사람과 4번째 사람은 같이 탈 수 있지만 1번째 사람과 3번째 사람의 무게의 합은 150kg이므로 구명보트의 무게 제한을 초과하여 같이 탈 수 없습니다.

구명보트를 최대한 적게 사용하여 모든 사람을 구출하려고 합니다.

사람들의 몸무게를 담은 배열 people과 구명보트의 무게 제한 limit가 매개변수로 주어질 때, 모든 사람을 구출하기 위해 필요한 구명보트 개수의 최솟값을 return 하도록 solution 함수를 작성해주세요.

제한사항


무인도에 갇힌 사람은 1명 이상 50,000명 이하입니다.
각 사람의 몸무게는 40kg 이상 240kg 이하입니다.
구명보트의 무게 제한은 40kg 이상 240kg 이하입니다.
구명보트의 무게 제한은 항상 사람들의 몸무게 중 최댓값보다 크게 주어지므로 사람들을 구출할 수 없는 경우는 없습니다.

limit
return

## 풀이
### 풀이 핵심 로직
구명보트를 최대한 적게 사용하기 위해, 사람들의 몸무게를 정렬한 뒤, 가장 가벼운 사람과 가장 무거운 사람을 짝지어 보트를 타게 합니다. 만약 두 사람의 몸무게 합이 구명보트의 무게 제한을 초과하지 않으면 둘이 함께 보트를 타고, 그렇지 않으면 무거운 사람만 보트를 탑니다. 이 과정을 반복하여 모든 사람을 구출하는 데 필요한 최소 보트의 개수를 구합니다.

### 동작 과정
1. **입력 데이터**: `people = [70, 50, 80, 50]`, `limit = 100`
2. **정렬**: `people`을 오름차순으로 정렬하면 `[50, 50, 70, 80]`이 됩니다.
3. **초기 포인터 설정**: `left` 포인터는 0 (가장 가벼운 사람), `right` 포인터는 3 (가장 무거운 사람)입니다.
4. **첫 번째 반복**:
   - `people[right] + people[left]` = `80 + 50` = `130`, 이 값은 `limit`인 `100`을 초과하므로, 무거운 사람(80kg)만 보트를 탑니다.
   - `right`를 2로 이동합니다 (이제 70kg).
5. **두 번째 반복**:
   - `people[right] + people[left]` = `70 + 50` = `120`, 이 값도 `100`을 초과하므로, 무거운 사람(70kg)만 보트를 탑니다.
   - `right`를 1로 이동합니다 (이제 50kg).
6. **세 번째 반복**:
   - `people[right] + people[left]` = `50 + 50` = `100`, 이 값은 `limit`을 초과하지 않으므로, 두 사람(50kg, 50kg)이 함께 보트를 탑니다.
   - `left`를 1로 이동하고, `right`를 0으로 이동합니다.
7. **종료 조건**: `left`가 `right`보다 커지면 반복을 종료합니다. 
8. **결과**: 총 보트의 개수는 3개입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)입니다. 이는 사람들을 정렬하는 데 걸리는 시간입니다. 이후의 반복문은 O(N)으로 처리되므로, 전체 시간 복잡도는 O(N log N)입니다.

## 코드
```python
def solution(people, limit):
    answer = 0
    people.sort()
    left = 0
    right = len(people) - 1
    
    while left <= right:
        answer += 1
        
        if people[right] + people[left] <= limit:
            left += 1
            
        right -= 1
        
    return answer
```
