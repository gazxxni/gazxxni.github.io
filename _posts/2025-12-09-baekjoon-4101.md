## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 두 양의 정수 a와 b를 비교하여, a가 b보다 큰 경우 "Yes"를 출력하고, 그렇지 않은 경우 "No"를 출력하는 문제입니다. 입력의 마지막은 두 개의 0으로 표시되므로 이를 체크하여 반복문을 종료합니다.

### 동작 과정
1. **입력**: 여러 개의 테스트 케이스가 주어지며, 각 줄마다 두 개의 정수가 a와 b가 입력됩니다.
2. **반복 처리**: 입력된 a와 b를 읽고, 두 수가 모두 0인 경우 반복을 종료합니다.
3. **비교**: a가 b보다 큰지 비교합니다.
   - a > b이면 "Yes"를 출력합니다.
   - 그렇지 않으면 "No"를 출력합니다.
4. **종료 조건**: 입력이 "0 0"일 경우 반복문을 종료합니다.

#### 예시 입력 및 단계별 설명
- 입력: 
```
1 2
2 1
3 3
0 0
```
- 단계별 출력:
  - 첫 번째 줄에서 a=1, b=2: a가 b보다 작으므로 "No" 출력.
  - 두 번째 줄에서 a=2, b=1: a가 b보다 크므로 "Yes" 출력.
  - 세 번째 줄에서 a=3, b=3: a가 b와 같으므로 "No" 출력.
  - 마지막 줄 "0 0"이 입력되면 반복 종료.

### 시간 복잡도
이 알고리즘은 각 테스트 케이스에 대해 상수 시간 O(1)만큼의 작업을 수행하므로 전체 입력에 대해 O(n) (n은 테스트 케이스의 수)입니다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
