## 문제
계속되는 폭우로 일부 지역이 물에 잠겼습니다. 물에 잠기지 않은 지역을 통해 학교를 가려고 합니다. 집에서 학교까지 가는 길은 m x n 크기의 격자모양으로 나타낼 수 있습니다. 

아래 그림은 m = 4, n = 3 인 경우입니다.



가장 왼쪽 위, 즉 집이 있는 곳의 좌표는 (1, 1)로 나타내고 가장 오른쪽 아래, 즉 학교가 있는 곳의 좌표는 (m, n)으로 나타냅니다. 

격자의 크기 m, n과 물이 잠긴 지역의 좌표를 담은 2차원 배열 puddles이 매개변수로 주어집니다. 오른쪽과 아래쪽으로만 움직여 집에서 학교까지 갈 수 있는 최단경로의 개수를 1,000,000,007로 나눈 나머지를 return 하도록 solution 함수를 작성해주세요.

제한사항


격자의 크기 m, n은 1 이상 100 이하인 자연수입니다.


m과 n이 모두 1인 경우는 입력으로 주어지지 않습니다.

물에 잠긴 지역은 0개 이상 10개 이하입니다.
집과 학교가 물에 잠긴 경우는 입력으로 주어지지 않습니다.


입출력 예

        
m
n
puddles
return

## 풀이
### 풀이 핵심 로직
이 문제는 동적 계획법을 사용하여 격자의 각 위치까지 도달하는 경로의 수를 계산하는 문제입니다. 각 셀 `(r, c)`에 도달할 수 있는 경로의 수는 위쪽 셀 `(r-1, c)`에서 오는 경로의 수와 왼쪽 셀 `(r, c-1)`에서 오는 경로의 수의 합으로 정의됩니다. 단, 물에 잠긴 지역에 도달할 수 없으므로 해당 셀에 대해서는 경로 수를 0으로 설정합니다.

### 동작 과정
1. 주어진 `m`과 `n`에 따라 `dp` 배열을 초기화합니다. `dp[r][c]`는 `(r, c)`에 도달할 수 있는 경로의 수를 저장합니다.
2. `dp[1][1]`을 1로 설정하여 시작점에서 해당 위치에 도달하는 경로가 하나임을 나타냅니다.
3. 이중 반복문을 통해 각 셀 `(r, c)`을 순회합니다:
   - 만약 셀 `(r, c)`가 물에 잠기지 않았다면, 위쪽 셀과 왼쪽 셀에서 오는 경로의 수를 더하여 `dp[r][c]`를 업데이트합니다.
   - 물에 잠긴 셀은 `dp[r][c]`를 0으로 유지합니다.
4. 마지막으로, 학교의 위치인 `dp[n][m]` 값을 1,000,000,007로 나눈 나머지를 반환합니다.

예시 입력 `m = 4`, `n = 3`, `puddles = [[2, 2]]`의 경우:
- 격자는 다음과 같이 초기화되고 업데이트됩니다:
  - 시작점 `dp[1][1] = 1`
  - `dp[1][2] = 1`, `dp[1][3] = 1` (1행은 모두 1)
  - `dp[2][1] = 1` (2열은 모두 1)
  - `dp[2][2]`는 물에 잠겨 있으므로 0
  - `dp[2][3] = 1` (2열에서 오는 경로)
  - `dp[3][1] = 1`, `dp[3][2] = 0`, `dp[3][3] = 1`
  - `dp[4][1] = 1`, `dp[4][2] = 0`, `dp[4][3] = 4` (위쪽과 왼쪽에서 모두 오는 경로의 수)
- 최종적으로 `dp[3][4]`의 값은 4가 되어 반환됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(m * n)입니다. 격자의 모든 셀을 한 번씩 방문하며 각 셀에 대해 상수 시간의 연산을 수행하기 때문입니다.

## 코드
```python
def solution(m, n, puddles):
    dp = [[0] * (m+1) for _ in range(n+1)]
    dp[1][1] = 1
    
    for r in range(1,n+1):
        for c in range(1,m+1):
            if [c, r] not in puddles:
                dp[r][c] += dp[r-1][c] + dp[r][c-1]
            
    
    
    return dp[n][m] % 1000000007
```
