## 풀이
### 풀이 핵심 로직
주어진 n x n 배열에 1부터 n^2까지의 정수를 시계방향으로 나선형 배치하기 위해, 방향을 정의하고 현재 위치에서 다음 위치로 이동하면서 숫자를 채워 나간다. 만약 다음 위치가 배열의 경계를 벗어나거나 이미 숫자가 채워져 있다면 방향을 전환하여 계속 진행한다.

### 동작 과정
1. n = 4인 경우를 예로 들면, 4x4 배열을 초기화하고 방향을 오른쪽(0, 1)으로 설정한다.
2. 1부터 16까지의 숫자를 차례로 배열에 채워 나간다.
   - (0, 0) 위치에 1을 채우고, 다음 위치(0, 1)로 이동.
   - (0, 1) 위치에 2를 채우고, 다음 위치(0, 2)로 이동.
   - (0, 2) 위치에 3을 채우고, 다음 위치(0, 3)로 이동.
   - (0, 3) 위치에 4를 채우고, 다음 위치(0, 4)로 이동하려고 시도하지만 경계를 벗어나므로 방향을 아래(1, 0)로 전환.
   - (1, 3) 위치에 5를 채우고, 다음 위치(2, 3)로 이동.
   - 이 과정을 반복하여 배열이 완성된다.
3. 최종적으로 배열은 다음과 같이 된다:
   ```
   [[1, 2, 3, 4],
    [12, 13, 14, 5],
    [11, 16, 15, 6],
    [10, 9, 8, 7]]
   ```

### 시간 복잡도
O(n^2)  
(n x n 배열을 채우기 위해 n^2번의 작업이 필요함)

## 코드
```python
def solution(n):
    answer = [[0] * n for _ in range(n)]

    directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]
    d = 0
    r, c = 0, 0
    for i in range(1, n * n + 1):
        answer[r][c] = i
        
        dr, dc = directions[d]
        nr = r + dr
        nc = c + dc
        
        if nc < 0 or nr < 0 or nc > n - 1 or nr > n - 1 or answer[nr][nc] != 0:
            d = (d + 1) % 4
            dr, dc = directions[d]
            nr = r + dr
            nc = c + dc
        
        r, c = nr, nc
        
    return answer
```
