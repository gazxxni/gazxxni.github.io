## 문제
서강대학교 컴퓨터공학과 실습실 R912호에는 현재 N개의 창문이 있고 또 N명의 사람이 있다. 1번째 사람은 1의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다.  2번째 사람은 2의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다. 이러한 행동을 N번째 사람까지 진행한 후 열려 있는 창문의 개수를 구하라. 단, 처음에 모든 창문은 닫혀 있다.

예를 들어 현재 3개의 창문이 있고 3명의 사람이 있을 때,


	1번째 사람은 1의 배수인 1,2,3번 창문을 연다. (1, 1, 1)
	2번째 사람은 2의 배수인 2번 창문을 닫는다. (1, 0, 1)
	3번째 사람은 3의 배수인 3번 창문을 닫는다. (1, 0, 0)


결과적으로 마지막에 열려 있는 창문의 개수는 1개 이다.

---

## 입력
첫 번째 줄에는 창문의 개수와 사람의 수 N(1 ≤ N ≤ 2,100,000,000)이 주어진다.

---

## 출력
마지막에 열려 있는 창문의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 각 창문이 열리거나 닫히는 조건을 이해하는 것입니다. 각 창문은 자신의 번호에 해당하는 약수의 개수에 따라 열리거나 닫힙니다. 일반적으로 약수는 쌍으로 존재하지만, 완전 제곱수는 홀수 개의 약수를 가집니다. 따라서, 열려 있는 창문은 완전 제곱수에 해당하는 창문들로, N까지의 완전 제곱수의 개수는 `⌊√N⌋`로 계산할 수 있습니다.

### 동작 과정
1. 첫 번째 사람(1번)은 모든 창문(1, 2, 3, ..., N)을 엽니다. (상태: 1, 1, 1, ..., 1)
2. 두 번째 사람(2번)은 2의 배수인 2번, 4번, 6번, ..., 창문을 닫습니다. (상태: 1, 0, 1, 0, 1, ..., 0)
3. 세 번째 사람(3번)은 3의 배수인 3번, 6번, ..., 창문을 닫습니다. (상태: 1, 0, 0, 0, 1, ..., 0)
4. 이러한 과정을 N번째 사람까지 반복합니다.

마지막 창문 상태에서 열려 있는 창문은 완전 제곱수에 해당하며, 이 개수는 `⌊√N⌋`로 알 수 있습니다.

예를 들어, N=3인 경우:
- 1번 사람: (1, 1, 1)
- 2번 사람: (1, 0, 1)
- 3번 사람: (1, 0, 0)
최종적으로 열려 있는 창문은 1개입니다.

### 시간 복잡도
O(1) - 입력 N에 대해 상수 시간 안에 결과를 계산할 수 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
print(int(n ** 0.5))
```
