## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
팩토리얼은 주어진 양의 정수 N에 대해 N! = N × (N-1) × (N-2) × ... × 1로 정의됩니다. 또한, 0!은 1로 정의됩니다. 이 문제에서는 재귀 함수를 사용하여 N의 팩토리얼을 계산하는 방식으로 문제를 해결합니다.

### 동작 과정
1. 사용자가 정수 N을 입력합니다. 예를 들어, N = 5라고 가정합니다.
2. `fac`라는 재귀 함수를 호출하여 5!을 계산합니다.
   - fac(5) → 5 × fac(4)
   - fac(4) → 4 × fac(3)
   - fac(3) → 3 × fac(2)
   - fac(2) → 2 × fac(1)
   - fac(1) → 1 (기저 사례)
3. 이제 각 단계에서 결과를 역으로 계산합니다:
   - fac(2) = 2 × 1 = 2
   - fac(3) = 3 × 2 = 6
   - fac(4) = 4 × 6 = 24
   - fac(5) = 5 × 24 = 120
4. 최종적으로 5! = 120이 출력됩니다.

### 시간 복잡도
시간 복잡도는 O(N)입니다. 이는 N에 대해 재귀 호출이 N번 이루어지기 때문입니다.

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
