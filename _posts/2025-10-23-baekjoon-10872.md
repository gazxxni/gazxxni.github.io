## 풀이

### 풀이 핵심 로직
이 코드는 재귀 함수를 사용하여 주어진 정수 n의 팩토리얼을 계산합니다. 팩토리얼은 n! = n × (n-1)!의 형태로 정의되며, 이 문제에서 n이 0일 경우 특별히 1로 정의됩니다.

### 동작 과정
1. 사용자로부터 정수 n을 입력받습니다.
2. `fac`라는 재귀 함수를 정의합니다. 이 함수는 n이 1일 경우 1을 반환하고, 그렇지 않으면 n과 `fac(n-1)`의 곱을 반환합니다.
3. n이 0일 경우, 팩토리얼의 정의에 따라 1을 출력합니다.
4. n이 0이 아닐 경우, `fac(n)`을 호출하여 n의 팩토리얼을 계산하고 그 결과를 출력합니다.

예시 입력:
- 입력: `5`
- 과정:
  - `fac(5)` 호출 → 5 × `fac(4)`
  - `fac(4)` 호출 → 4 × `fac(3)`
  - `fac(3)` 호출 → 3 × `fac(2)`
  - `fac(2)` 호출 → 2 × `fac(1)`
  - `fac(1)` 호출 → 1 반환
  - `fac(2)` → 2 × 1 = 2 반환
  - `fac(3)` → 3 × 2 = 6 반환
  - `fac(4)` → 4 × 6 = 24 반환
  - `fac(5)` → 5 × 24 = 120 반환
- 출력: `120`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 이는 재귀 호출이 n번 이루어지기 때문입니다. 각 호출에서 상수 시간 작업(곱셈과 반환)을 수행하므로 전체 시간은 n에 비례합니다.

---

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
