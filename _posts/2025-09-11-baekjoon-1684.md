## 문제
정수 N을 정수 D로 나눴을 때의 몫을 Q, 나머지를 R이라고 하면 항등식 R = N - Q×D가 성립한다.

n개의 정수로 된 수열이 있을 때, 모든 정수를 한 정수 D로 나눴을 때 나머지가 같아지는 경우가 있다. 그리고 수열에 따라서는 이러한 정수 D가 여러 개 존재할 수 있다.

n개의 정수로 된 수열이 주어졌을 때, 가장 큰 D를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 n(1 ≤ n ≤ 1,000)이 주어진다. 다음 줄에는 절댓값이 1,000,000을 넘지 않는 n개의 정수들이 주어진다.

---

## 출력
첫째 줄에 가장 큰 D를 출력한다. 항상 가장 큰 D가 존재하는 경우만 입력으로 주어진다.

## 풀이

### 풀이 핵심 로직
주어진 정수 배열의 모든 쌍의 차이의 최대공약수(GCD)를 구하면, 그 GCD가 바로 우리가 찾고자 하는 가장 큰 D가 됩니다. 이는 두 수의 차가 D로 나누어 떨어져야 한다는 성질에서 유도됩니다. 즉, D는 모든 두 수의 차이의 공약수여야 하므로, 모든 차이의 GCD를 계산하는 것이 문제를 해결하는 방법입니다.

### 동작 과정
1. 입력으로 주어진 정수 N의 개수 n을 읽고, n개의 정수를 배열에 저장합니다.
2. 배열의 중복된 값을 제거하여 새로운 배열을 생성합니다. (중복된 값은 GCD 계산에 영향을 주지 않기 때문)
3. 두 수의 차이의 절댓값을 계산하여 리스트에 저장합니다.
   - 예를 들어 배열이 [4, 8, 10]일 때, |8-4|, |10-4|, |10-8|를 계산하여 [4, 6, 2]를 얻습니다.
4. 이 차이들로부터 최대공약수를 계산하여 가장 큰 D를 찾습니다.
5. 결과를 출력합니다.

예시 입력:
```
3
4 8 10
```
단계별 설명:
- n = 3, arr = [4, 8, 10]
- 중복 제거 후 brr = [4, 8, 10]
- 차이 계산: |8-4|=4, |10-4|=6, |10-8|=2 → crr = [4, 6, 2]
- GCD 계산: GCD(4, 6) = 2, GCD(2, 2) = 2 → 최대 D = 2
- 출력: 2

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 이는 모든 쌍의 차이를 계산하기 위해 O(n^2) 연산이 필요하고, GCD를 계산하는 과정도 O(log M) (여기서 M은 두 수 중 큰 수) 만큼 시간이 소요됩니다. 하지만, n이 최대 1000이므로 이 복잡도는 허용 범위 내에 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline

'''
두 수의 차가 D로 나누어 떨어진다
(aᵢ - aⱼ) % D = 0 -> 성립해야함

따라서 |a₂ - a₁|, |a₃ - a₁|, ..., |aₙ - a₁|
이 숫자들의 최대공약수를 구해야함
'''

n = int(input())
arr = list(map(int, input().split()))

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
        
    return a

brr = list(set(arr))
crr = []

for i in range(len(brr) - 1):
    for j in range(i + 1, len(brr)):
        crr.append(abs(brr[j] - brr[i])) 
        
ans = crr[0]
for i in range(1, len(crr)): 
    ans = gcd(ans, crr[i])
    
print(ans)
```
