## 풀이

### 풀이 핵심 로직
주어진 문제는 두 수의 차가 어떤 정수 D로 나누어 떨어지는 조건을 만족하는 수를 찾는 것이다. 이 문제의 핵심 아이디어는 주어진 수들 간의 차이의 최대공약수(GCD)를 구하는 것으로, 이를 통해 모든 수의 차이가 D로 나누어지는지 판별할 수 있다.

### 동작 과정
1. **입력 처리**: 첫 번째 줄에서 수의 개수 `n`을 입력받고, 두 번째 줄에서 `n`개의 정수를 리스트 `arr`로 입력받는다.
2. **중복 제거**: `set()`을 사용하여 `arr`에서 중복된 값을 제거하고 이를 `brr` 리스트로 저장한다.
3. **차이 계산**: 중복이 제거된 리스트 `brr`의 모든 쌍 `(brr[i], brr[j])`에 대해 그 차이의 절대값을 계산하여 `crr` 리스트에 저장한다.
4. **최대공약수 계산**: `crr`의 첫 번째 원소를 초기값으로 설정하고, 나머지 원소들과 차례로 GCD를 계산하여 최종 GCD 값을 구한다.
5. **결과 출력**: 최종적으로 구한 GCD 값을 출력한다.

예시 입력:
```
5
1 3 6 10 15
```
단계별 설명:
- `n`은 5이고, `arr`는 `[1, 3, 6, 10, 15]`이다.
- 중복 제거 후 `brr`는 `[1, 3, 6, 10, 15]`로 동일하다.
- 차이 계산:
  - `(3-1)=2`, `(6-1)=5`, `(10-1)=9`, `(15-1)=14`
  - `(6-3)=3`, `(10-3)=7`, `(15-3)=12`
  - `(10-6)=4`, `(15-6)=9`
  - `(15-10)=5`
  - 결과적으로 `crr`는 `[2, 5, 9, 14, 3, 7, 12, 4, 9, 5]`가 된다.
- GCD 계산:
  - 초기값 `ans`에 `2`를 설정.
  - 나머지 원소들과 GCD를 차례로 계산하여 최종적으로 GCD 값을 구한다.
- 결과는 GCD 값이 출력된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)이다. 여기서 n은 입력된 수의 개수이다. 중복을 제거한 후 두 수의 차이를 계산하는 부분에서 이중 반복문을 사용하기 때문에 O(n^2) 시간이 소요된다. GCD 계산은 O(log(max(a, b)))의 시간 복잡도를 가지지만, 이중 반복문에 비해 상대적으로 적은 영향을 주므로 전체 시간 복잡도는 O(n^2)로 유지된다.

## 코드
```python
import sys
input = sys.stdin.readline

'''
두 수의 차가 D로 나누어 떨어진다
(aᵢ - aⱼ) % D = 0 -> 성립해야함

따라서 |a₂ - a₁|, |a₃ - a₁|, ..., |aₙ - a₁|
이 숫자들의 최대공약수를 구해야함
'''

n = int(input())
arr = list(map(int, input().split()))

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
        
    return a

brr = list(set(arr))
crr = []

for i in range(len(brr) - 1):
    for j in range(i + 1, len(brr)):
        crr.append(abs(brr[j] - brr[i])) 
        
ans = crr[0]
for i in range(1, len(crr)): 
    ans = gcd(ans, crr[i])
    
print(ans)
```
