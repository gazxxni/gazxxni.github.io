## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
주어진 문제는 N개의 테스트 케이스에 대해 특정 포맷의 문자열을 출력하는 것입니다. 각 줄에는 "Hello World, Judge i!"라는 형식으로 인사를 해야 하며, 여기서 i는 1부터 N까지의 숫자입니다. 반복문을 사용하여 1부터 N까지의 숫자를 출력 문자열에 포함시켜 해결할 수 있습니다.

### 동작 과정
1. 사용자로부터 N을 입력받습니다. 예를 들어, N이 3이라면 입력은 `3`입니다.
2. 반복문을 사용하여 1부터 N까지의 숫자를 순회합니다. 
   - 첫 번째 반복(iteration)에서는 i=1이므로 "Hello World, Judge 1!"이 출력됩니다.
   - 두 번째 반복에서는 i=2이므로 "Hello World, Judge 2!"가 출력됩니다.
   - 세 번째 반복에서는 i=3이므로 "Hello World, Judge 3!"가 출력됩니다.
3. 이 과정은 N까지 계속되며, 각 줄마다 포맷에 맞춘 문자열이 출력됩니다.

입력 예시:
```
3
```
출력:
```
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

### 시간 복잡도
이 프로그램의 시간 복잡도는 O(N)입니다. N은 입력받은 테스트 케이스의 수이며, N번의 반복을 수행하기 때문입니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
