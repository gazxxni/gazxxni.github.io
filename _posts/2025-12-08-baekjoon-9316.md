## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 N개의 테스트 케이스에 대해 각 줄에 "Hello World, Judge i!"라는 형식으로 출력하는 간단한 구현 문제입니다. 반복문을 사용하여 1부터 N까지의 숫자를 순회하면서 각 숫자를 포함한 문자열을 출력하면 됩니다.

### 동작 과정
1. 입력으로 N이 주어집니다. 예를 들어, N이 3이라면, 
2. 반복문을 사용하여 1부터 N까지의 숫자 i에 대해 다음과 같은 문자열을 출력합니다.
   - i = 1일 때: "Hello World, Judge 1!"
   - i = 2일 때: "Hello World, Judge 2!"
   - i = 3일 때: "Hello World, Judge 3!"
3. 최종적으로 출력 결과는 다음과 같습니다:
   ```
   Hello World, Judge 1!
   Hello World, Judge 2!
   Hello World, Judge 3!
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. N은 입력으로 주어진 테스트 케이스의 수로, 각 테스트 케이스에 대해 한 번씩 출력하기 때문입니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
