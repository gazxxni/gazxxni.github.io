## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

---

## 풀이

### 풀이 핵심 로직
주어진 숫자 N에 따라 1부터 N까지의 숫자를 순차적으로 출력하면서 각 숫자에 맞는 인사말을 만들어 출력하는 것이 핵심 아이디어입니다. 포매팅을 통해 문자열을 쉽게 구성할 수 있습니다.

### 동작 과정
1. 입력으로 N이 주어진다고 가정합니다. 예를 들어, N=3일 경우.
2. 반복문을 사용하여 1부터 N까지의 숫자를 순회합니다.
3. 각 반복에서 "Hello World, Judge i!" 형식의 문자열을 출력합니다.
4. 이 경우, 출력 결과는 다음과 같습니다:
   ```
   Hello World, Judge 1!
   Hello World, Judge 2!
   Hello World, Judge 3!
   ```

### 시간 복잡도
이 알고리즘은 N번의 반복을 수행하므로 시간 복잡도는 O(N)입니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
