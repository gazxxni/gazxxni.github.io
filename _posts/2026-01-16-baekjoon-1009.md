## 문제
재용이는 최신 컴퓨터 10대를 가지고 있다. 어느 날 재용이는 많은 데이터를 처리해야 될 일이 생겨서 각 컴퓨터에 1번부터 10번까지의 번호를 부여하고, 10대의 컴퓨터가 다음과 같은 방법으로 데이터들을 처리하기로 하였다.

1번 데이터는 1번 컴퓨터, 2번 데이터는 2번 컴퓨터, 3번 데이터는 3번 컴퓨터, ... ,

10번 데이터는 10번 컴퓨터, 11번 데이터는 1번 컴퓨터, 12번 데이터는 2번 컴퓨터, ...

총 데이터의 개수는 항상 ab개의 형태로 주어진다. 재용이는 문득 마지막 데이터가 처리될 컴퓨터의 번호가 궁금해졌다. 이를 수행해주는 프로그램을 작성하라.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 수 \(a\)의 \(b\)제곱을 구한 후, 결과를 10으로 나눈 나머지를 구하여 마지막 데이터가 처리될 컴퓨터의 번호를 결정하는 문제이다. \(a^b\)의 마지막 자리는 \(a\)의 마지막 자리수에 따라 주기적으로 반복되므로, 이 주기를 이용하여 계산할 수 있다.

### 동작 과정
1. 입력으로 테스트 케이스의 개수 \(T\)와 각 테스트 케이스에 대해 정수 \(a\)와 \(b\)를 받는다.
2. \(a\)의 마지막 자리수를 구하기 위해 \(a \% 10\)을 계산한다.
3. 각 마지막 자리수별로 \(b\)의 값에 따라 결과를 결정한다:
   - 마지막 자리수가 **0**이면, 결과는 항상 **10** (10번 컴퓨터).
   - 마지막 자리수가 **1**, **5**, **6**이면, 결과는 항상 해당 숫자 (각각 **1**, **5**, **6**).
   - 마지막 자리수가 **4** 또는 **9**이면, \(b\)의 짝수/홀수 여부에 따라 결정된다:
     - 짝수면 \(a^2 \% 10\), 홀수면 \(a \% 10\).
   - 나머지 숫자 (**2, 3, 7, 8**)의 경우, \(b\)를 4로 나눈 나머지에 따라 주기를 따른다:
     - \(b \% 4 == 0\)일 때는 \(a^4 \% 10\), \(b \% 4 == 1\)일 때는 \(a^1 \% 10\), \(b \% 4 == 2\)일 때는 \(a^2 \% 10\), \(b \% 4 == 3\)일 때는 \(a^3 \% 10\).

예시 입력:
```
3
1 2
3 3
4 4
```
- 첫 번째 테스트 케이스: \(1^2 \% 10 = 1\) (컴퓨터 1)
- 두 번째 테스트 케이스: \(3^3 = 27 \% 10 = 7\) (컴퓨터 7)
- 세 번째 테스트 케이스: \(4^4\)는 16의 마지막 자리 6 (짝수이므로 \(4^2\)의 마지막 자리) (컴퓨터 6)

### 시간 복잡도
각 테스트 케이스에 대해 상수 시간 \(O(1)\)이 소요되므로, 전체 시간 복잡도는 \(O(T)\)입니다. 여기서 \(T\)는 테스트 케이스의 수입니다.

## 코드
```python
import sys

t = int(sys.stdin.readline())

for _ in range(t):
    a, b = map(int, sys.stdin.readline().split())
    
    base = a % 10
    
    if base == 0:
        print(10)
    elif base in [1, 5, 6]:
        print(base)
    elif base in [4, 9]:
        remain = b % 2
        if remain == 0:
            print((base ** 2) % 10)
        else:
            print(base)
    else:
        remain = b % 4
        if remain == 0:
            print((base ** 4) % 10)
        else:
            print((base ** remain) % 10)
```
