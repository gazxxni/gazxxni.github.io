## 문제
네 점이 주어졌을 때, 네 점을 이용해 정사각형을 만들 수 있는지 없는지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 네 줄로 이루어져 있으며, 점의 좌표가 한 줄에 하나씩 주어진다. 점의 좌표는 -100,000보다 크거나 같고, 100,000보다 작거나 같은 정수이다. 같은 점이 두 번 이상 주어지지 않는다.

---

## 출력
각 테스트 케이스마다 입력으로 주어진 네 점을 이용해서 정사각형을 만들 수 있으면 1을, 없으면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 네 점을 이용해 정사각형을 만들 수 있는지를 판단하기 위해 각 점들 간의 거리의 제곱을 계산하여 이를 카운트합니다. 정사각형의 경우, 네 개의 변의 길이는 모두 같고, 두 대각선의 길이도 같으므로, 거리의 제곱을 카운트하여 8개의 변과 2개의 대각선이 나오는지를 체크합니다.

### 동작 과정
1. 입력으로 주어진 점들을 리스트에 저장합니다.
2. 두 점 사이의 거리의 제곱을 계산하여, 이를 딕셔너리에서 카운트합니다.
   - 예를 들어, 점 (0, 0), (1, 0), (1, 1), (0, 1) 가 주어진 경우, 네 점 사이의 거리 제곱은 (0,0)과 (1,0) 간의 거리 제곱은 1, (0,0)과 (1,1) 간의 거리 제곱은 2, (0,0)과 (0,1) 간의 거리 제곱은 1, ... 등으로 계산됩니다.
3. 거리 제곱 카운트에서 변의 개수(길이가 같은 쌍)와 대각선의 개수를 확인합니다.
   - 변의 개수는 8, 대각선의 개수는 2여야 합니다.
4. 조건을 만족하면 1을 출력하고, 그렇지 않으면 0을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력으로 주어지는 점의 개수는 항상 4개이므로, 상수 시간 안에 모든 계산이 이루어집니다.

## 코드
```python
from collections import defaultdict
t = int(input())

for _ in range(t):
    arr = []
    
    for i in range(4):
        x, y = map(int, input().split())
        arr.append((x, y))
    
    dic = defaultdict(int)
    for i in range(4):
        for j in range(4):
            if i != j:
                a = (arr[i][0] - arr[j][0]) ** 2 + (arr[i][1] - arr[j][1]) ** 2
                dic[a] += 1
    
    aa = False
    bb = False  
    for k, v in dic.items():
        if v == 8:
            aa = True
        
        elif v == 4:
            bb = True
            
    if aa and bb:
        print(1)
        
    else:
        print(0)
```
