## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수 N을 소수의 곱으로 표현하는 과정입니다. 이 문제에서는 N의 소인수들을 오름차순으로 출력해야 하므로, 먼저 2로 나누어 떨어지는 경우를 처리한 후, 홀수 소수를 이용해 나머지 소인수를 찾습니다.

### 동작 과정
1. **입력**: 정수 N을 입력받습니다.
   - 예시: N = 28
2. **2로 나누기**: N이 2로 나누어 떨어지는 동안 2를 출력하고 N을 2로 나눈 값으로 업데이트합니다.
   - 28 → 2 출력, N = 14
   - 14 → 2 출력, N = 7
3. **홀수 소수로 나누기**: 3부터 N의 제곱근까지 홀수 소수를 이용해 N을 나누어 떨어지는 소수들을 출력합니다.
   - i = 3, N = 7 → 나누어 떨어지지 않음
   - i = 5, N = 7 → 나누어 떨어지지 않음
   - i = 7, N = 7 → 7 출력 후 N = 1
4. **남은 N 확인**: 만약 N이 1보다 크면 그 자체가 소수이므로 출력합니다.
   - 이 경우 N = 1이므로 출력하지 않음.
5. **최종 출력**: 소인수들이 한 줄에 하나씩 오름차순으로 출력됩니다.
   - 출력: 2, 2, 7

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(√N)입니다. 이는 N을 소인수로 나누는 과정에서 최대 √N까지의 숫자를 확인하기 때문입니다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
