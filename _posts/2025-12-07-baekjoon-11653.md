## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수를 소수의 곱으로 표현하는 과정입니다. 주어진 정수 \( N \)에 대해, 먼저 2로 나누어 떨어지는 만큼 2를 출력하고, 이후 홀수 소수들로 나누어 나가면서 소인수를 출력합니다. 마지막으로 남은 수가 1보다 크면 그것도 소인수이므로 출력합니다.

### 동작 과정
1. **입력**: 정수 \( N \)을 입력받습니다. 예를 들어, \( N = 60 \)이라고 가정합니다.
2. **2로 나누기**: \( N \)이 2로 나누어 떨어지는 동안 2를 출력하고 \( N \)을 2로 나눈 몫으로 업데이트합니다. (60 → 30 → 15)
   - 출력: 2
   - 출력: 2
3. **홀수 소수로 나누기**: 이제 3부터 시작하여 홀수 소수로 \( N \)을 나누어 나갑니다. 
   - \( N = 15 \)는 3으로 나누어 떨어지므로 3을 출력하고 나누기.
     - 출력: 3 (15 → 5)
4. **남은 수 확인**: 5가 1보다 크므로 출력합니다.
   - 출력: 5
5. 최종 결과: 2, 2, 3, 5가 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(\sqrt{N}) \)입니다. 이는 2로 나누는 과정과 홀수 소수로 나누는 과정에서 각각 최대 \( \sqrt{N} \)까지 반복하기 때문입니다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
