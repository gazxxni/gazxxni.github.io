## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 알고리즘의 수행 횟수를 계산하고, 이를 다항식으로 나타낼 때 최고차항의 차수를 구하는 것입니다. 주어진 `n`에 대해 알고리즘의 수행 횟수를 수식으로 표현하면 `n * (n - 1) - (n - 1) * (n / 2)`로 나타낼 수 있으며, 이 수식을 통해 수행 횟수와 최고차항의 차수를 구할 수 있습니다.

### 동작 과정
1. 입력으로 정수 `n`을 받습니다. (예: n = 5)
2. 수행 횟수를 계산합니다:
   - `n * (n - 1)`는 모든 쌍의 비교를 의미합니다.
   - `(n - 1) * (n / 2)`는 중복된 쌍의 수를 의미합니다.
   - 따라서 수행 횟수는 `n * (n - 1) - (n - 1) * (n / 2)`로 계산됩니다.
3. 최고차항의 차수를 판단합니다:
   - 이 식의 최고차항은 `n^2`이므로 차수는 2입니다.
4. 결과를 출력합니다:
   - 첫째 줄에 수행 횟수, 둘째 줄에 차수를 출력합니다.

예시 입력: 
```
5
```
출력:
```
10
2
```
- 수행 횟수는 `5 * (5 - 1) - (5 - 1) * (5 / 2) = 20 - 10 = 10`입니다.
- 최고차항의 차수는 2입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 크기 `n`에 관계없이 상수 시간에 수행 횟수를 계산하고 결과를 출력하기 때문입니다.
