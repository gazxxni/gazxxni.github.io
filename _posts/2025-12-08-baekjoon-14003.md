## 문제
수열 A가 주어졌을 때, 가장 긴 증가하는 부분 수열을 구하는 프로그램을 작성하시오.

예를 들어, 수열 A = {10, 20, 10, 30, 20, 50} 인 경우에 가장 긴 증가하는 부분 수열은 A = {10, 20, 10, 30, 20, 50} 이고, 길이는 4이다.

---

## 입력
첫째 줄에 수열 A의 크기 N (1 ≤ N ≤ 1,000,000)이 주어진다.

둘째 줄에는 수열 A를 이루고 있는 Ai가 주어진다. (-1,000,000,000 ≤ Ai ≤ 1,000,000,000)

---

## 출력
첫째 줄에 수열 A의 가장 긴 증가하는 부분 수열의 길이를 출력한다.

둘째 줄에는 정답이 될 수 있는 가장 긴 증가하는 부분 수열을 출력한다.

## 풀이

### 풀이 핵심 로직
가장 긴 증가하는 부분 수열(LIS)을 찾기 위해 이분 탐색과 역추적을 활용합니다. `tails` 배열을 사용하여 각 길이에 대한 최소 끝값을 저장하고, `prev_idx` 배열을 통해 각 원소의 이전 원소 인덱스를 추적하여 LIS를 복원합니다.

### 동작 과정
1. **입력**: 수열 A = {10, 20, 10, 30, 20, 50}의 길이인 N과 A의 원소들을 입력받습니다.
2. **초기화**: `tails`와 `tails_pos` 배열을 초기화하고, `prev_idx` 배열을 -1로 초기화합니다.
3. **이분 탐색**: 각 원소를 `tails` 배열에 이분 탐색을 통해 삽입할 위치를 찾습니다.
   - 예를 들어, 첫 번째 원소 10을 `tails`에 추가하여 `tails = [10]`이 됩니다.
   - 두 번째 원소 20은 `tails`의 끝에 추가되어 `tails = [10, 20]`이 됩니다.
   - 세 번째 원소 10은 10보다 크거나 같은 최소값을 찾아 10으로 대체되어 `tails = [10, 20]`가 됩니다.
   - 이 과정을 반복하며 `tails`를 업데이트합니다.
4. **역추적**: `tails_pos`로부터 가장 긴 부분 수열의 길이와 마지막 원소의 인덱스를 찾아 `prev_idx`를 따라가며 원소들을 추출합니다. 
   - 최종적으로 `seq`에는 {10, 20, 30, 50}이 담기고, 이를 역순으로 출력합니다.
5. **출력**: 가장 긴 증가하는 부분 수열의 길이와 원소들을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)입니다. 이는 각 원소에 대해 이분 탐색을 수행하는 데 O(log N)의 시간이 걸리기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

# tails[k] = 길이 (k+1)인 증가 부분수열의 최소 끝값
# tails_pos[k] = 위 tails[k]를 만든 인덱스
tails = []
tails_pos = []

# prev_idx[i] = arr[i]가 LIS에 포함될 때 바로 이전 원소의 인덱스
prev_idx = [-1] * n

def search(a, x):
    lo, hi = 0, len(a)
    while lo < hi:
        mid = (lo + hi) // 2
        if a[mid] < x:
            lo = mid + 1
        else:
            hi = mid
    return lo

for i, x in enumerate(arr):
    j = search(tails, x)

    # j-1 길이의 꼬리에 이어 붙이므로, 경로의 마지막 인덱스를 prev로 기록
    prev_idx[i] = tails_pos[j - 1] if j > 0 else -1

    if j == len(tails):  # 길이 확장
        tails.append(x)
        tails_pos.append(i)
    else:    # 꼬리값 갱신
        tails[j] = x
        tails_pos[j] = i

# 실가장 긴 길이의 마지막 인덱스부터 prev_idx를 따라 역추적
lis_len = len(tails)
k = tails_pos[-1] if tails_pos else -1

seq = []
while k != -1:
    seq.append(arr[k])
    k = prev_idx[k]
seq.reverse()

print(lis_len)
print(*seq)
```
