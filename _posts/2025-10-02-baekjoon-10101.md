## 문제
창영이는 삼각형의 종류를 잘 구분하지 못한다. 따라서 프로그램을 이용해 이를 외우려고 한다.

삼각형의 세 각을 입력받은 다음, 


	세 각의 크기가 모두 60이면, Equilateral
	세 각의 합이 180이고, 두 각이 같은 경우에는 Isosceles
	세 각의 합이 180이고, 같은 각이 없는 경우에는 Scalene
	세 각의 합이 180이 아닌 경우에는 Error


를 출력하는 프로그램을 작성하시오.

---

## 입력
총 3개의 줄에 걸쳐 삼각형의 각의 크기가 주어진다. 모든 정수는 0보다 크고, 180보다 작다.

---

## 출력
문제의 설명에 따라 Equilateral, Isosceles, Scalene, Error 중 하나를 출력한다.

---

## 풀이

### 풀이 핵심 로직
삼각형의 세 각을 입력받아, 이 각들이 이루는 삼각형의 종류를 판별하는 문제이다. 각의 합이 180도인지 확인하고, 각의 크기를 비교하여 Equilateral, Isosceles, Scalene, 또는 Error를 출력한다. 각이 모두 60도인 경우에는 Equilateral로, 두 각이 같은 경우에는 Isosceles로, 세 각이 모두 다르고 합이 180도인 경우에는 Scalene으로 판단한다.

### 동작 과정
1. 세 개의 각(a, b, c)을 입력받는다.
2. 첫 번째 조건으로 모든 각이 60도인지를 확인하여, 참이면 'Equilateral'을 출력한다.
   - 예시 입력: 60, 60, 60 → 출력: Equilateral
3. 두 번째 조건으로 세 각의 합(a + b + c)이 180도인지 확인한다.
   - 예시 입력: 50, 50, 80 → 출력: Isosceles (두 각이 같은 경우)
   - 예시 입력: 30, 60, 90 → 출력: Scalene (모든 각이 다름)
4. 만약 합이 180도가 아니라면 'Error'를 출력한다.
   - 예시 입력: 100, 100, 100 → 출력: Error (합이 300도)

### 시간 복잡도
O(1) - 입력받은 각의 수가 고정되어 있고, 단순 비교 및 연산만 수행되므로 상수 시간 복잡도를 갖는다.

## 코드
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
c = int(input())


if a == b == c == 60:
    print('Equilateral')

elif a + b + c == 180:
    if a == b and b != c:
        print('Isosceles')
    elif a == c and b != c:
        print('Isosceles')
    elif b == c and a != c:
        print('Isosceles')
    else:
        print('Scalene')

else:
    print('Error')
```
