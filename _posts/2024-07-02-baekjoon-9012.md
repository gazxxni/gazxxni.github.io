## 풀이

### 풀이 핵심 로직
이 코드는 주어진 괄호 문자열이 올바른 괄호 쌍으로 이루어져 있는지를 확인하는 문제입니다. 스택 자료구조를 사용하여 '('가 나타날 때는 스택에 추가하고, ')'가 나타날 때는 스택에서 '('를 제거합니다. 마지막에 스택이 비어있으면 모든 괄호가 짝이 맞는 것이므로 "YES"를 출력하고, 그렇지 않으면 "NO"를 출력합니다.

### 동작 과정
1. 입력으로 테스트 케이스의 수 `t`를 받습니다.
2. 각 테스트 케이스마다 빈 리스트 `stack`을 생성합니다.
3. 괄호 문자열을 입력받아 하나씩 처리합니다:
   - '('를 만나면 `stack`에 추가합니다.
   - ')'를 만나면 `stack`이 비어있지 않다면 마지막 '('를 제거하고, 비어있다면 "NO"를 출력하고 반복문을 종료합니다.
4. 문자열을 모두 처리한 후, `stack`이 비어있다면 "YES"를 출력하고, 비어있지 않다면 "NO"를 출력합니다.

**예시 입력:**
```
3
(())
(()()
(()))
```
**단계별 설명:**
- 첫 번째 테스트 케이스 `(())`:
  - '(' → 스택: ['(']
  - '(' → 스택: ['(', '(']
  - ')' → 스택: ['(']
  - ')' → 스택: []
  - 스택 비어있음 → 출력 "YES"
  
- 두 번째 테스트 케이스 `(()()`:
  - '(' → 스택: ['(']
  - '(' → 스택: ['(', '(']
  - ')' → 스택: ['(']
  - '(' → 스택: ['(', '(']
  - ')' → 스택: ['(']
  - 스택 비어있지 않음 → 출력 "NO"

- 세 번째 테스트 케이스 `(()))`:
  - '(' → 스택: ['(']
  - '(' → 스택: ['(', '(']
  - ')' → 스택: ['(']
  - ')' → 스택: []
  - ')' → 스택 비어있음 → 출력 "NO"

### 시간 복잡도
O(n) - 각 문자열을 한 번 탐색하며 스택의 push와 pop 연산은 O(1) 시간에 수행되므로, 전체적으로 입력된 괄호 문자열의 길이에 비례하는 시간 복잡도를 가집니다.

## 코드
```python
import sys

t=int(input())

for i in range(t):
    stack = []
    a=input()
    for j in a:
        if j == '(':
            stack.append(j)
        elif j == ')':
            if stack:
                stack.pop()
            else: 
                print("NO")
                break
    else: 
        if not stack: 
            print("YES")
        else:
            print("NO")
```
