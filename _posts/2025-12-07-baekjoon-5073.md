## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이를 통해 삼각형의 종류를 판단하기 위해 먼저 세 변의 길이를 정렬합니다. 그 후, 가장 긴 변이 나머지 두 변의 합보다 크거나 같은지 확인하여 삼각형의 유효성을 판단하고, 각 변의 길이에 따라 Equilateral, Isosceles, Scalene 중 하나로 분류합니다.

### 동작 과정
1. 입력으로 세 개의 양의 정수 a, b, c를 읽어옵니다. 예를 들어, 입력이 `6 3 2`인 경우.
2. 입력받은 세 변을 정렬하여 a ≤ b ≤ c로 만듭니다. 이 경우 정렬 후에는 a=2, b=3, c=6이 됩니다.
3. c가 a + b(즉, 6 ≥ 2 + 3)보다 크거나 같은지 확인합니다. 이 조건이 참이므로 "Invalid"를 출력합니다.
4. 다음 입력으로 `3 3 3`을 받으면 정렬 후 a=3, b=3, c=3이 되고, 모든 변이 같으므로 "Equilateral"을 출력합니다.
5. 또 다른 입력으로 `5 5 2`를 받으면 a=2, b=5, c=5가 되고, 두 변이 같으므로 "Isosceles"를 출력합니다.
6. 마지막으로 `4 5 6`을 입력받으면 a=4, b=5, c=6이 되고, 세 변이 모두 다르므로 "Scalene"을 출력합니다.
7. 입력이 `0 0 0`일 때는 프로그램이 종료됩니다.

### 시간 복잡도
O(1) - 입력되는 세 변의 길이의 수는 항상 세 개이므로, 정렬 및 조건 확인에 소요되는 시간은 상수 시간입니다.

---

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
