## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
삼각형의 성질을 이용하여 세 변의 길이에 따라 삼각형의 종류를 판별한다. 주어진 세 변의 길이가 삼각형의 조건을 만족하는지 확인하고, 맞다면 Equilateral, Isosceles, Scalene 중 하나로 분류한다. 그렇지 않으면 "Invalid"를 출력한다.

### 동작 과정
1. 사용자로부터 세 개의 정수 a, b, c를 입력받는다.
2. 입력된 값이 모두 0이면 프로그램을 종료한다.
3. 입력된 세 변의 길이를 정렬하여 a, b, c로 할당한다. (a ≤ b ≤ c)
4. 가장 긴 변 c가 나머지 두 변 a와 b의 합보다 크거나 같으면 "Invalid"를 출력한다.
5. 세 변이 모두 같으면 "Equilateral"을 출력한다.
6. 두 변이 같고, 하나가 다르면 "Isosceles"를 출력한다.
7. 세 변이 모두 다르면 "Scalene"을 출력한다.

예시 입력과 출력:
- 입력: `6 3 2` → 출력: `Invalid`
- 입력: `3 3 3` → 출력: `Equilateral`
- 입력: `5 5 2` → 출력: `Isosceles`
- 입력: `3 4 5` → 출력: `Scalene`

### 시간 복잡도
O(1) - 각 입력에 대해 상수 시간 내에 연산을 수행하므로, 입력의 수에 관계없이 시간 복잡도는 상수이다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
