## 풀이

### 풀이 핵심 로직
이 문제는 큐(Queue) 자료구조의 기본적인 동작을 구현하는 것입니다. Python의 `collections.deque`를 사용하여 큐를 구현하며, 입력된 명령에 따라 큐에 원소를 추가하거나 제거하고, 큐의 상태를 확인하는 여러 기능을 제공합니다.

### 동작 과정
1. 입력된 명령의 수 `n`을 읽어들입니다.
2. `for` 루프를 통해 각 명령을 읽어들이고, 해당 명령에 따라 큐를 조작합니다.
   - `push x`: 정수 `x`를 큐의 뒤에 추가합니다.
   - `pop`: 큐의 앞에서 정수를 제거하고 그 값을 출력합니다. 큐가 비어있으면 `-1`을 출력합니다.
   - `size`: 큐에 있는 원소의 개수를 출력합니다.
   - `empty`: 큐가 비어있으면 `1`, 그렇지 않으면 `0`을 출력합니다.
   - `front`: 큐의 앞에 있는 값을 출력하고, 큐가 비어있으면 `-1`을 출력합니다.
   - `back`: 큐의 뒤에 있는 값을 출력하고, 큐가 비어있으면 `-1`을 출력합니다.

예시 입력:
```
5
push 1
push 2
front
back
pop
```

단계별 설명:
- 첫 번째 명령 `push 1` 실행: 큐는 `[1]`이 됩니다.
- 두 번째 명령 `push 2` 실행: 큐는 `[1, 2]`가 됩니다.
- 세 번째 명령 `front` 실행: 큐의 앞 원소인 `1`을 출력합니다.
- 네 번째 명령 `back` 실행: 큐의 뒤 원소인 `2`를 출력합니다.
- 다섯 번째 명령 `pop` 실행: 큐의 앞 원소인 `1`을 제거하고 `1`을 출력합니다. 이제 큐는 `[2]`가 됩니다.

### 시간 복잡도
모든 명령의 시간 복잡도는 O(1)입니다. 각 명령은 큐의 기본 동작을 수행하며, `deque`의 메서드들은 상수 시간에 실행됩니다. 따라서 전체 명령을 처리하는 데 걸리는 시간은 O(n)입니다.

## 코드
```python
import sys
from collections import deque
input = sys.stdin.readline
n = int(input())
q = deque()

for _ in range(n):
    command = input().split()
    
    a = command[0]
    
    if a == 'push':
        b = int(command[1])
        q.append(b)
        
    elif a == 'pop':
        if q:
            print(q.popleft())
        else:
            print(-1)
            
    elif a == 'size':
        print(len(q))
        
    elif a == 'empty':
        if not q:
            print(1)
        else:
            print(0)
            
    elif a == 'front':
        if q:
            print(q[0])
        else:
            print(-1)
            
    elif a == 'back':
        if q:
            print(q[-1])
        else:
            print(-1)
```
