## 문제
재용이는 최신 컴퓨터 10대를 가지고 있다. 어느 날 재용이는 많은 데이터를 처리해야 될 일이 생겨서 각 컴퓨터에 1번부터 10번까지의 번호를 부여하고, 10대의 컴퓨터가 다음과 같은 방법으로 데이터들을 처리하기로 하였다.

1번 데이터는 1번 컴퓨터, 2번 데이터는 2번 컴퓨터, 3번 데이터는 3번 컴퓨터, ... ,

10번 데이터는 10번 컴퓨터, 11번 데이터는 1번 컴퓨터, 12번 데이터는 2번 컴퓨터, ...

총 데이터의 개수는 항상 ab개의 형태로 주어진다. 재용이는 문득 마지막 데이터가 처리될 컴퓨터의 번호가 궁금해졌다. 이를 수행해주는 프로그램을 작성하라.

## 풀이

### 풀이 핵심 로직
문제는 주어진 수 \( a^b \)의 마지막 자리 숫자가 어느 컴퓨터에서 처리되는지를 구하는 것이다. 10대의 컴퓨터가 1번부터 10번까지 번호가 부여되어 있으므로, \( a^b \mod 10 \)의 결과값에 따라 마지막 데이터가 처리되는 컴퓨터의 번호를 결정할 수 있다. 각 숫자의 마지막 자리수는 주기성을 가지므로, 이를 활용하여 효율적으로 계산할 수 있다.

### 동작 과정
1. 입력으로 주어지는 \( T \)개의 테스트 케이스에 대해 각각 \( a \)와 \( b \)를 읽는다.
2. \( a \)의 마지막 자리수 (즉, \( a \mod 10 \))를 계산하여 `base`에 저장한다.
3. `base`가 0이면 마지막 자리수는 10(10번 컴퓨터)으로 출력한다.
4. `base`가 1, 5, 6인 경우는 항상 그 숫자 자체가 마지막 자리수이므로 그대로 출력한다.
5. `base`가 4 또는 9인 경우, \( b \)를 2로 나눈 나머지에 따라 출력값이 결정된다.
6. 나머지가 0이면 \( base^2 \mod 10 \), 나머지가 1이면 \( base \mod 10 \)을 출력한다.
7. `base`가 2, 3, 7, 8인 경우는 \( b \)를 4로 나눈 나머지에 따라 출력값이 결정된다. 나머지가 0일 때는 \( base^4 \mod 10 \), 나머지가 1, 2, 3일 때는 각각 \( base^1, base^2, base^3 \mod 10 \)을 출력한다.

**예시 입력**:
```
3
1 3
2 3
3 5
```
**예시 동작 과정**:
- 첫 번째 테스트 케이스: \( a = 1, b = 3 \) → `base` = 1 → 출력: 1
- 두 번째 테스트 케이스: \( a = 2, b = 3 \) → `base` = 2 → \( b \mod 4 = 3 \) → 출력: \( 2^3 \mod 10 = 8 \)
- 세 번째 테스트 케이스: \( a = 3, b = 5 \) → `base` = 3 → \( b \mod 4 = 1 \) → 출력: \( 3^1 \mod 10 = 3 \)

### 시간 복잡도
각 테스트 케이스에 대해 상수 시간 \( O(1) \)이 소요되므로 전체 시간 복잡도는 \( O(T) \)이다. 여기서 \( T \)는 테스트 케이스의 수이다.

## 코드
```python
import sys

t = int(sys.stdin.readline())

for _ in range(t):
    a, b = map(int, sys.stdin.readline().split())
    
    base = a % 10
    
    if base == 0:
        print(10)
    elif base in [1, 5, 6]:
        print(base)
    elif base in [4, 9]:
        remain = b % 2
        if remain == 0:
            print((base ** 2) % 10)
        else:
            print(base)
    else:
        remain = b % 4
        if remain == 0:
            print((base ** 4) % 10)
        else:
            print((base ** remain) % 10)
```
