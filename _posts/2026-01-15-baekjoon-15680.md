## 문제
연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.

이를 출력하는 프로그램을 작성해보도록 하자.

---

## 입력
첫째 줄에 N이 주어진다. (N = 0 또는 1)

---

## 출력
N = 0일 경우: 연세대학교의 영문명을 출력한다.
	N = 1일 경우: 연세대학교의 슬로건을 출력한다.


대소문자 구별에 주의하도록 하자.

## 풀이

### 풀이 핵심 로직
문제는 입력 값에 따라 두 가지 문자열 중 하나를 출력하는 간단한 조건문을 요구합니다. 입력 값이 0이면 "YONSEI"를 출력하고, 1이면 "Leading the Way to the Future"를 출력합니다. 이 문제는 조건문을 활용하여 주어진 입력에 맞는 출력을 결정하는 기본적인 로직입니다.

### 동작 과정
1. 프로그램이 시작되면 사용자로부터 정수 N을 입력 받습니다.
2. 입력된 N의 값이 0인지 1인지 확인합니다.
   - 예를 들어, N이 0이라면 "YONSEI"를 출력합니다.
   - N이 1이라면 "Leading the Way to the Future"를 출력합니다.
3. 이 과정을 통해 조건에 맞는 출력을 수행합니다.

예시 입력과 출력:
- 입력: `0`
  - 출력: `YONSEI`
- 입력: `1`
  - 출력: `Leading the Way to the Future`

### 시간 복잡도
이 문제의 시간 복잡도는 O(1)입니다. 입력 값에 따라 조건문을 통해 즉시 결정을 내리기 때문에 입력 크기와 관계없이 일정한 시간을 소요합니다.

---

## 코드
```python
n = int(input())

if n == 0:
    print("YONSEI")
else:
    print("Leading the Way to the Future")
```
