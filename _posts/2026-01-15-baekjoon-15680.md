## 문제
연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.

이를 출력하는 프로그램을 작성해보도록 하자.

---

## 입력
첫째 줄에 N이 주어진다. (N = 0 또는 1)

---

## 출력
N = 0일 경우: 연세대학교의 영문명을 출력한다.
	N = 1일 경우: 연세대학교의 슬로건을 출력한다.


대소문자 구별에 주의하도록 하자.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 문제는 입력값 N에 따라 두 가지 문자열 중 하나를 출력하는 간단한 조건문 기반의 구현 문제입니다. N이 0이면 "YONSEI"를 출력하고, N이 1이면 "Leading the Way to the Future"를 출력하는 방식으로 처리합니다.

### 동작 과정
1. 프로그램이 실행되면 사용자로부터 정수 N을 입력받습니다.
2. 입력받은 N의 값에 따라 조건문을 사용하여 다음과 같은 동작을 수행합니다:
   - 만약 N이 0이라면 "YONSEI"를 출력합니다.
   - 만약 N이 1이라면 "Leading the Way to the Future"를 출력합니다.
   
예시 입력:
- N = 0일 경우: 
  - 입력: 0
  - 출력: YONSEI
- N = 1일 경우:
  - 입력: 1
  - 출력: Leading the Way to the Future

### 시간 복잡도
이 문제의 시간 복잡도는 O(1)입니다. 입력값에 따라 조건문을 통해 상수 시간 내에 결과를 출력하기 때문입니다.

---

## 코드
```python
n = int(input())

if n == 0:
    print("YONSEI")
else:
    print("Leading the Way to the Future")
```
