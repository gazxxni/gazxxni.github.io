## 풀이

### 풀이 핵심 로직
이 문제는 연속적인 K일의 온도의 합을 구하는 문제로, 슬라이딩 윈도우 기법을 통해 해결할 수 있습니다. 처음 K일의 온도를 합한 후, 이후의 일자에서 K일의 합을 구할 때는 이전의 첫 날 온도를 빼고 새로운 마지막 날의 온도를 더하는 방식으로 효율적으로 계산합니다. 이를 통해 모든 K일의 연속 합을 구할 수 있습니다.

### 동작 과정
1. 입력에서 N(총 일수)과 K(연속적인 일수)를 읽어옵니다.
2. N일 동안의 온도를 리스트로 받아옵니다.
3. 첫 K일의 온도 합을 계산합니다.
   - 예시: 온도가 `3 -2 -4 -9 0 3 7 13 8 -3`일 때, 첫 5일의 합은 `3 + (-2) + (-4) + (-9) + 0 = -12`입니다.
4. 이후부터는 슬라이딩 윈도우를 이용하여 계산합니다.
   - 예를 들어, 5일의 합을 구한 후, 6일째부터는 `-12 - 3 + 7 = -8`로 계산합니다.
5. 이 과정을 반복하며 최대값을 업데이트합니다.
   - 각 단계에서 새로운 온도를 더하고, 가장 오래된 온도를 빼는 방식으로 계산합니다.
6. 최종적으로 최대 합을 출력합니다.

예시 입력:
```
10 5
3 -2 -4 -9 0 3 7 13 8 -3
```
이 경우, 온도의 합을 계산하면서 다음과 같은 값을 얻습니다:
- 초기 합: 3 - 2 - 4 - 9 + 0 = -12
- 2번째 합: -12 - 3 + 7 = -8
- 3번째 합: -8 - (-2) + 13 = 3
- ...
- 마지막 합: 3 + 7 + 13 + 8 - 9 = 22 (최대값)

최종 출력: `22`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. N일 동안의 온도를 한 번만 순회하면서 슬라이딩 윈도우 기법을 적용하기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
