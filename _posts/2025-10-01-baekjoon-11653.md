## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수 N을 소수의 곱으로 표현하는 과정입니다. 이 문제에서는 2부터 시작하여 N의 제곱근까지의 모든 소수를 검사하면서 N을 나누고, 나머지가 1이 될 때까지 반복하여 소인수를 찾습니다.

### 동작 과정
1. **입력 받기**: 정수 N을 입력받습니다.
2. **2로 나누기**: N이 짝수일 때, 2로 나누어서 나올 때마다 2를 출력합니다. 이 과정을 N이 더 이상 짝수가 아닐 때까지 반복합니다.
   - 예: N = 24라면, 출력: 2, 2, 2 (2로 세 번 나누어짐)
3. **홀수로 나누기**: 3부터 N의 제곱근까지의 홀수를 차례로 검사합니다. 각 홀수 i에 대해 N이 i로 나누어 떨어지는 동안 i를 출력하고 N을 i로 나누어줍니다.
   - 예: N = 24, i = 3인 경우, N은 3으로 나누어 떨어지지 않으므로 다음 홀수인 5로 이동합니다. i = 5도 마찬가지. 
   - i = 2는 이미 처리했으므로, i = 3에서 N은 3으로 나누어 떨어지지 않습니다.
   - 이후 i = 5도 나누어 떨어지지 않으므로, 다음 홀수를 검사합니다.
4. **남은 수 출력**: 마지막으로 N이 1보다 큰 경우, 그 N 자체가 소수이므로 출력합니다.
   - 예: N이 7로 남았다면, 7을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(√N)입니다. 이는 N의 소인수분해를 위해 N의 제곱근까지 반복적으로 나누기를 수행하기 때문입니다. 각 소수에 대해 나누는 과정은 상수 시간에 이루어지므로 전체적인 복잡도는 O(√N)입니다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
