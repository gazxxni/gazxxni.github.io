## 풀이

### 풀이 핵심 로직
이 코드는 주어진 정수 \( n \)의 소인수 분해를 수행하는 알고리즘입니다. 먼저, 2로 나누어지는 만큼 2를 출력하고, 이후 홀수 소수들로 \( n \)을 나누면서 소인수를 출력합니다. 마지막으로 \( n \)이 1보다 크다면, 그 자체가 소수이므로 출력합니다.

### 동작 과정
예를 들어, 입력이 `60`인 경우를 살펴보겠습니다.

1. **초기 입력**: `n = 60`
2. **2로 나누기**: 
   - `60 % 2 == 0` → 출력 `2`, \( n \)을 30으로 업데이트
   - `30 % 2 == 0` → 출력 `2`, \( n \)을 15로 업데이트
   - `15 % 2 != 0` → 2로 나누는 과정을 종료

3. **홀수 소수로 나누기**: 
   - \( i = 3 \) (홀수 소수)
     - `15 % 3 == 0` → 출력 `3`, \( n \)을 5로 업데이트
     - `5 % 3 != 0` → 3으로 나누는 과정을 종료
   - \( i = 5 \)
     - `5 % 5 == 0` → 출력 `5`, \( n \)을 1로 업데이트
     - `1 % 5 != 0` → 5로 나누는 과정을 종료

4. **마지막 체크**: \( n = 1 \)이므로 추가 출력 없이 종료합니다.

결과적으로 출력된 값은 `2`, `2`, `3`, `5`입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(\sqrt{n}) \)입니다. \( n \)이 2로 나누어질 때는 상수 시간이 걸리며, 이후 홀수 소수로 나누는 과정은 \( \sqrt{n} \)까지 반복되므로 전체적으로 \( O(\sqrt{n}) \)의 복잡도를 가집니다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
