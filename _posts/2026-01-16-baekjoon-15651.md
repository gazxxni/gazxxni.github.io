## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 M개를 고른 수열
	같은 수를 여러 번 골라도 된다.

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
이 문제는 백트래킹 기법을 사용하여 자연수 N에서 M개의 수를 고르는 모든 조합을 생성하는 것입니다. 같은 숫자를 여러 번 선택할 수 있기 때문에, 현재 선택한 수를 다시 선택할 수 있도록 재귀적으로 호출하며, M개의 수를 선택했을 때 결과를 출력합니다.

### 동작 과정
1. 입력으로 N과 M이 주어지면, N=3과 M=2라고 가정합니다.
2. 빈 리스트 `arr`를 선언하여 선택한 수를 저장합니다.
3. 재귀 함수 `aa()`를 호출하여 수열을 생성합니다.
   - 함수가 호출될 때마다 리스트의 길이가 M에 도달하면 현재 리스트를 출력합니다.
   - 그렇지 않으면 1부터 N까지의 숫자를 반복하여, 숫자를 `arr`에 추가하고 재귀적으로 `aa()`를 호출합니다.
   - 재귀 호출이 끝난 후에는 `arr`에 추가한 숫자를 제거하여 다음 조합을 위해 준비합니다.
4. 이 과정을 모든 가능한 조합에 대해 반복합니다.
5. 최종적으로는 다음과 같은 출력이 이루어집니다:
   ```
   1 1
   1 2
   1 3
   2 1
   2 2
   2 3
   3 1
   3 2
   3 3
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^M)입니다. M개의 수를 선택할 때 각 자리에서 N개의 선택이 가능하므로, 총 경우의 수는 N을 M번 곱한 값이 됩니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
