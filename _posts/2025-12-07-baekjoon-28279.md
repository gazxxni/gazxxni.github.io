## 문제
정수를 저장하는 덱을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 여덟 가지이다.


	1 X: 정수 X를 덱의 앞에 넣는다. (1 ≤ X ≤ 100,000)
	2 X: 정수 X를 덱의 뒤에 넣는다. (1 ≤ X ≤ 100,000)
	3: 덱에 정수가 있다면 맨 앞의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	4: 덱에 정수가 있다면 맨 뒤의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	5: 덱에 들어있는 정수의 개수를 출력한다.
	6: 덱이 비어있으면 1, 아니면 0을 출력한다.
	7: 덱에 정수가 있다면 맨 앞의 정수를 출력한다. 없다면 -1을 대신 출력한다.
	8: 덱에 정수가 있다면 맨 뒤의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 덱(deque) 자료구조를 사용하여 정수를 저장하고, 주어진 명령에 따라 해당 작업을 수행하는 것입니다. 각 명령은 덱의 앞이나 뒤에서 원소를 추가하거나 제거하고, 상태를 출력하는 등의 작업을 수행합니다. Python의 `collections` 모듈에 포함된 `deque`를 사용하면 이러한 작업을 효율적으로 처리할 수 있습니다.

### 동작 과정
주어진 예시 입력을 통해 단계별로 설명하겠습니다.

1. **입력 예시**:
   ```
   7
   1 10
   2 20
   3
   5
   6
   7
   8
   ```

2. **단계별 처리**:
   - **1 10**: 정수 10을 덱의 앞에 추가합니다. 덱 상태: `[10]`
   - **2 20**: 정수 20을 덱의 뒤에 추가합니다. 덱 상태: `[10, 20]`
   - **3**: 덱의 앞에서 정수 10을 제거하고 출력합니다. 출력: `10`, 덱 상태: `[20]`
   - **5**: 덱에 있는 정수의 개수를 출력합니다. 출력: `1`
   - **6**: 덱이 비어있지 않으므로 `0`을 출력합니다. 출력: `0`
   - **7**: 덱의 앞에 있는 정수 20을 출력합니다. 출력: `20`
   - **8**: 덱의 뒤에 있는 정수 20을 출력합니다. 출력: `20`

### 시간 복잡도
각 명령에 대한 시간 복잡도는 다음과 같습니다:
- 삽입 (append, appendleft): O(1)
- 삭제 (pop, popleft): O(1)
- 크기 확인 (len): O(1)
- 비어있는지 확인: O(1)
- 인덱스 접근: O(1)

따라서 총 N개의 명령을 처리하는 데 걸리는 시간 복잡도는 O(N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline
from collections import deque

n = int(input())
q = deque()

for _ in range(n):
    command = input().split()
    
    a = int(command[0])
    
    if a == 1:
        q.appendleft(command[1])
        
    elif a == 2:
        q.append(command[1])
        
    elif a == 3:
        if q:
            print(q.popleft())
        else:
            print(-1)
            
    elif a == 4:
        if q:
            print(q.pop())
        else:
            print(-1)
            
    elif a == 5:
        print(len(q))
        
    elif a == 6:
        if q:
            print(0)
        else:
            print(1)
            
    elif a == 7:
        if q:
            print(q[0])
        else:
            print(-1)
            
    elif a == 8:
        if q:
            print(q[-1])
        else:
            print(-1)
```
