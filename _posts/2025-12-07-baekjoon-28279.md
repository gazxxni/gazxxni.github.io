## 문제
정수를 저장하는 덱을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 여덟 가지이다.


	1 X: 정수 X를 덱의 앞에 넣는다. (1 ≤ X ≤ 100,000)
	2 X: 정수 X를 덱의 뒤에 넣는다. (1 ≤ X ≤ 100,000)
	3: 덱에 정수가 있다면 맨 앞의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	4: 덱에 정수가 있다면 맨 뒤의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	5: 덱에 들어있는 정수의 개수를 출력한다.
	6: 덱이 비어있으면 1, 아니면 0을 출력한다.
	7: 덱에 정수가 있다면 맨 앞의 정수를 출력한다. 없다면 -1을 대신 출력한다.
	8: 덱에 정수가 있다면 맨 뒤의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 덱(deque) 자료구조를 사용하여 정수의 추가, 삭제, 조회와 관련된 여러 가지 명령을 처리하는 것입니다. 덱은 양쪽 끝에서 삽입과 삭제가 가능한 자료구조로, Python의 `collections` 모듈에서 제공하는 `deque`를 활용하여 효율적으로 구현할 수 있습니다.

### 동작 과정
주어진 명령을 차례대로 처리하는 과정은 다음과 같습니다:

1. **입력 처리**: 첫 번째 줄에서 명령의 수 N을 입력받고, 다음 N줄에서 각각의 명령을 입력받습니다.
2. **명령 실행**:
   - `1 X`: 정수 X를 덱의 앞에 추가합니다.
   - `2 X`: 정수 X를 덱의 뒤에 추가합니다.
   - `3`: 덱의 앞에서 정수를 제거하고 출력합니다. 덱이 비어있으면 -1을 출력합니다.
   - `4`: 덱의 뒤에서 정수를 제거하고 출력합니다. 덱이 비어있으면 -1을 출력합니다.
   - `5`: 덱에 있는 정수의 개수를 출력합니다.
   - `6`: 덱이 비어있으면 1, 아니면 0을 출력합니다.
   - `7`: 덱의 앞 정수를 출력합니다. 비어있으면 -1을 출력합니다.
   - `8`: 덱의 뒤 정수를 출력합니다. 비어있으면 -1을 출력합니다.

예시 입력:
```
8
1 10
2 20
3
4
5
6
7
8
```
처리 결과:
```
10
20
1
0
10
20
```

### 시간 복잡도
각 명령에 대해 덱의 연산은 평균적으로 O(1) 시간 복잡도를 가집니다. 따라서 전체 명령 N개를 처리하는 데 걸리는 시간은 O(N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline
from collections import deque

n = int(input())
q = deque()

for _ in range(n):
    command = input().split()
    
    a = int(command[0])
    
    if a == 1:
        q.appendleft(command[1])
        
    elif a == 2:
        q.append(command[1])
        
    elif a == 3:
        if q:
            print(q.popleft())
        else:
            print(-1)
            
    elif a == 4:
        if q:
            print(q.pop())
        else:
            print(-1)
            
    elif a == 5:
        print(len(q))
        
    elif a == 6:
        if q:
            print(0)
        else:
            print(1)
            
    elif a == 7:
        if q:
            print(q[0])
        else:
            print(-1)
            
    elif a == 8:
        if q:
            print(q[-1])
        else:
            print(-1)
```
