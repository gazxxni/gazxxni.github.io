## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 수행 시간은 입력 크기 \( n \)에 따라 \( n^2 \)로 증가합니다. 이는 이중 루프를 통해 모든 쌍을 처리하는 방식으로, 수행 횟수를 계산할 수 있습니다. 또한, 최고차항의 차수는 2로, 다항식의 형태가 \( n^2 \)에 해당하기 때문에 이를 출력하면 됩니다.

### 동작 과정
1. 입력 값 \( n \)을 받습니다. 예를 들어, 입력이 5인 경우 \( n = 5 \)입니다.
2. 수행 횟수를 계산하여 \( n^2 \)을 출력합니다. 즉, \( 5^2 = 25 \)를 출력합니다.
3. 다항식의 최고차항의 차수는 2이므로 2를 출력합니다.

예시 입력:
```
5
```
예시 출력:
```
25
2
```

### 시간 복잡도
시간 복잡도는 \( O(n^2) \)입니다. 이는 이중 루프를 통해 모든 입력 쌍을 고려하기 때문입니다.
