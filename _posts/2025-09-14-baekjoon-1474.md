## 문제
세준이는 N개의 영어 단어를 이용해 길이가 M인 새로운 단어를 만들려고 한다. 새로운 단어는 N개의 단어를 순서대로 이어 붙이고, 각 단어의 사이에 _을 넣어서 만든다. 이렇게 만든 새로운 단어의 길이가 M이 아닌 경우 _를 추가해서 길이가 M이 되게 만들어야 한다.


	_는 단어와 단어 사이에만 추가할 수 있다. 따라서, 새로운 단어는 _으로 시작하거나, _로 끝날 수 없다.
	단어와 단어 사이에 있는 _의 개수는 모두 같아야 한다.
	
		모두 같게 만드는 것이 불가능한 경우 단어와 단어 사이에 있는 _의 개수의 최댓값과 최솟값의 차이는 1이 되어야 한다.
	
	


새로운 단어 중 사전 순으로 가장 앞서는 단어를 구해보자.

---

## 입력
첫째 줄에 N과 M이 주어진다. 둘째 줄부터 N개의 줄에 단어가 한 줄에 하나씩 주어진다.

---

## 출력
첫째 줄에 사전 순으로 가장 앞서는 단어를 출력한다.

알파벳 대문자, 소문자, 밑 줄의 순서는 'A'

## 풀이

### 풀이 핵심 로직
이 문제의 핵심은 N개의 단어를 이어붙이면서 각 단어 사이에 들어가는 `_`의 개수를 균등하게 나누어 길이 M을 만들고, 사전 순으로 가장 앞서는 결과를 도출하는 것입니다. 이를 위해 `_`의 개수를 균등하게 나누거나 차이가 1인 경우를 고려하여 최적의 문자열을 생성합니다.

### 동작 과정
1. **입력 처리**: 먼저 N과 M을 입력받고, N개의 단어를 리스트에 저장합니다.
2. **기본 길이 계산**: 모든 단어의 길이를 합산하여 총 길이 `total`을 계산합니다.
3. **남은 길이 계산**: 목표 길이 M에서 `total`을 빼서 `_`의 총 개수를 구합니다.
4. **균등 분배**: `greedy` 함수를 통해 남은 `_`의 개수를 N-1개의 구간에 균등하게 분배합니다. 각 구간에 들어가는 `_`의 개수를 정하고, 남은 개수는 사전 순을 고려하여 적절히 분배합니다.
5. **결과 생성**: 첫 번째 단어를 결과 리스트에 추가한 후, 각 단어 사이에 `_`를 삽입하며 나머지 단어를 결과 리스트에 추가합니다.
6. **출력**: 최종적으로 완성된 문자열을 출력합니다.

예시 입력:
```
3 14
abc
ab
cde
```
- `total` = 9 (abc + ab + cde)
- 남은 길이 = 14 - 9 = 5
- `_`의 개수 = 5 (단어 사이에 2개, 2개, 1개로 나누어짐)
- 결과: "abc__ab_cde" (사전 순으로 가장 앞서는 문자열)

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 단어의 길이 합산 및 `_`의 개수 계산, 결과 문자열 생성 등의 작업이 모두 N에 비례하는 시간 내에 이루어지기 때문입니다.

## 코드
```python
n, m = map(int, input().split())
arr = [input().rstrip() for _ in range(n)]

def greedy(a, b):
    aa = a // b
    bb = a % b

    result = [aa] * b
    for i in range(b):
        if ord(arr[i + 1][0]) >= 97 and bb > 0:
            result[i] += 1
            bb -= 1
    
    for i in range(b - 1, 0, -1):
        if result[i] == aa and bb > 0:
            result[i] += 1
            bb -= 1

    result2 = ["_" * x for x in result]
    return result2

total = 0
for i in range(n):
    total += len(arr[i])

result = greedy(m - total, n- 1)

ans = [arr.pop(0)]
for i in range(n - 1):
    ans.append(result[i])
    ans.append(arr[i])

print(*ans, sep = '')
```
