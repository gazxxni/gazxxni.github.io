## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제는 원형 구조로 배치된 풍선들을 터뜨리는 순서를 계산하는 것입니다. 각 풍선은 자신이 터질 때 이동할 방향과 거리를 명시하는 값을 가지고 있으며, 이미 터진 풍선은 이동 시 제외해야 합니다. 이를 위해 덱(Deque) 자료구조를 활용하여 풍선의 인덱스와 값을 저장하고, 터진 풍선의 인덱스를 기록하면서 적절히 회전하여 다음 풍선을 선택합니다.

### 동작 과정
1. **입력 처리**: N개의 풍선 수와 각 풍선의 종이에 적힌 값을 입력받습니다.
   - 예시 입력: 
     ```
     5
     3 2 1 -3 -1
     ```
   - 이 경우, 각 풍선의 값은 `[3, 2, 1, -3, -1]`입니다.

2. **덱 초기화**: 각 풍선의 인덱스와 값을 튜플 형태로 덱에 추가합니다.
   - 초기 덱: `[(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)]`

3. **풍선 터뜨리기**: 덱에서 풍선을 하나씩 터뜨리면서 해당 풍선의 값을 읽고, 그 값만큼 이동합니다.
   - 첫 번째 풍선(인덱스 0, 값 3)을 터뜨립니다.
   - 현재 덱: `[(1, 2), (2, 1), (3, -3), (4, -1)]`
   - 3만큼 오른쪽으로 이동하여 인덱스 2의 풍선(값 1)을 터뜨립니다.
   - 현재 덱: `[(1, 2), (3, -3), (4, -1)]`
   - 이 과정을 반복하며 풍선의 인덱스를 기록합니다.

4. **결과 출력**: 터진 풍선의 순서를 출력합니다.
   - 최종 출력: `1 3 5 2 4`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 각 풍선을 터뜨릴 때마다 덱의 크기가 줄어들어 이동해야 하는 횟수가 N에 비례하여 최대 O(N) 시간이 소요되고, 전체 N개의 풍선을 터뜨리므로 N * O(N) = O(N^2)입니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
