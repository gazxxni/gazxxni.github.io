## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 원형으로 배치된 풍선을 처리하기 위해 덱(Deque) 자료구조를 사용하는 것입니다. 덱을 이용해 현재 터진 풍선의 인덱스를 저장하고, 풍선 안의 숫자에 따라 다음 터질 풍선의 인덱스를 계산할 수 있습니다. 이 과정을 반복하여 모든 풍선을 터트리면, 최종적으로 터진 풍선의 번호를 나열할 수 있습니다.

### 동작 과정
1. **입력:** 
   - N = 5
   - arr = [3, 2, 1, -3, -1]

2. **초기 설정:**
   - 덱에 각 풍선의 인덱스와 종이에 적힌 숫자를 저장합니다. 
   - 덱의 초기 상태: `deque([(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)])`
   
3. **첫 번째 풍선 터뜨리기:** 
   - 1번 풍선 (인덱스 0, 숫자 3) 터뜨림.
   - 덱에서 제거하고, 결과에 추가: `ans = [1]`
   - 다음 풍선으로 이동: 오른쪽으로 3칸 이동 (터진 풍선 제외). 
   - 새로운 덱 상태: `deque([(1, 2), (2, 1), (3, -3), (4, -1)])` → `deque([(2, 1), (3, -3), (4, -1), (1, 2)])`
   
4. **두 번째 풍선 터뜨리기:**
   - 새로 선택된 3번 풍선 (인덱스 2, 숫자 1) 터뜨림.
   - 결과에 추가: `ans = [1, 3]`
   - 다음 풍선으로 이동: 오른쪽으로 1칸 이동.
   - 새로운 덱 상태: `deque([(3, -3), (4, -1), (1, 2)])`
   
5. **세 번째 풍선 터뜨리기:**
   - 4번 풍선 (인덱스 3, 숫자 -3) 터뜨림.
   - 결과에 추가: `ans = [1, 3, 4]`
   - 다음 풍선으로 이동: 왼쪽으로 3칸 이동.
   - 새로운 덱 상태: `deque([(1, 2), (2, 1)])`
   
6. **네 번째 풍선 터뜨리기:**
   - 2번 풍선 (인덱스 1, 숫자 2) 터뜨림.
   - 결과에 추가: `ans = [1, 3, 4, 2]`
   - 다음 풍선으로 이동: 오른쪽으로 2칸 이동.
   - 새로운 덱 상태: `deque([(2, 1)])`
   
7. **마지막 풍선 터뜨리기:**
   - 5번 풍선 (인덱스 4, 숫자 -1) 터뜨림.
   - 결과에 추가: `ans = [1, 3, 4, 2, 5]`
   - 덱이 비어있으므로 종료.

8. **결과 출력:**
   - 최종 결과: `1 3 4 2 5`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각각의 풍선을 한 번씩 터뜨리고, 덱의 회전 연산이 최악의 경우 O(N)까지 걸릴 수 있지만, 전체적으로는 N개의 풍선을 순회하면서 처리하기 때문에 O(N)으로 간주할 수 있습니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
