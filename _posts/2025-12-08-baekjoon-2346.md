## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 풍선들을 원형으로 관리하기 위해 덱(deque)을 사용하여 효율적으로 터뜨릴 풍선을 선택하고, 해당 풍선의 종이에 적힌 수만큼 이동하는 것입니다. 매번 터진 풍선의 인덱스를 기록하고, 이동 방향에 따라 덱을 회전시켜서 다음 풍선을 쉽게 찾을 수 있습니다.

### 동작 과정
예를 들어, 입력값이 다음과 같다고 가정합니다:
```
5
3 2 1 -3 -1
```
1. 처음에 1번 풍선(종이: 3)을 터뜨립니다. 현재 덱 상태: `[(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)]`
   - 터진 풍선 번호: `1`
   - 종이에 적힌 숫자 `3`으로 오른쪽으로 3칸 이동합니다.
   
2. 현재 위치에서 오른쪽으로 3칸 이동하면 4번 풍선(종이: -3)에 도착합니다. 덱 상태: `[(1, 2), (2, 1), (3, -3), (4, -1)]`
   - 터진 풍선 번호: `4`
   - 종이에 적힌 숫자 `-3`으로 왼쪽으로 3칸 이동합니다.
   
3. 현재 위치에서 왼쪽으로 3칸 이동하면 5번 풍선(종이: -1)에 도착합니다. 덱 상태: `[(1, 2), (2, 1), (4, -1)]`
   - 터진 풍선 번호: `5`
   - 종이에 적힌 숫자 `-1`으로 왼쪽으로 1칸 이동합니다.

4. 현재 위치에서 왼쪽으로 1칸 이동하면 2번 풍선(종이: 2)에 도착합니다. 덱 상태: `[(1, 2), (2, 1)]`
   - 터진 풍선 번호: `2`
   - 종이에 적힌 숫자 `2`으로 오른쪽으로 2칸 이동합니다.

5. 현재 위치에서 오른쪽으로 2칸 이동하면 3번 풍선(종이: 1)에 도착합니다. 덱 상태: `[(2, 1)]`
   - 터진 풍선 번호: `3`
   - 종이에 적힌 숫자 `1`으로 오른쪽으로 1칸 이동합니다.

6. 현재 위치에서 오른쪽으로 1칸 이동하면 덱이 비게 되어 종료됩니다.

최종적으로 터진 풍선의 순서는: `1 4 5 2 3`입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 풍선을 터뜨리고 회전하는 과정에서 각각 최대 N번의 연산이 발생할 수 있으므로, 전체적으로 N개의 풍선을 처리하는 데 O(N) 시간이 소요됩니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
