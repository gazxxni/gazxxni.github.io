## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘은 입력 배열 A의 크기 n을 기반으로 세 개의 서로 다른 요소를 선택하는 조합의 개수를 계산합니다. 조합의 개수는 nC3 (n choose 3)으로 계산되며, 이는 n(n-1)(n-2)/6으로 표현됩니다. 여기서 최고차항의 차수는 3입니다.

### 동작 과정
입력으로 n이 주어졌다고 가정합시다. 예를 들어 n = 5일 경우, 다음과 같은 단계가 진행됩니다:

1. 조합의 개수 계산:
   - n = 5일 때, nC3 = 5 * 4 * 3 / 6 = 10
   - 따라서, 첫 번째 줄의 출력은 10입니다.

2. 최고차항의 차수 출력:
   - 조합의 개수는 n^3에 비례하므로, 최고차항의 차수는 3입니다.
   - 따라서, 두 번째 줄의 출력은 3입니다.

즉, 입력이 5일 경우 출력은 다음과 같습니다:
```
10
3
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 n에 대한 계산만 수행되며, 반복문이나 재귀 호출이 없기 때문입니다.

## 코드
```python
n = int(input())

print(int((n * (n - 1) * (n - 2)) / 6))
print(3)
```
