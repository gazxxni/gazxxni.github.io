[https://www.acmicpc.net/problem/1018](https://www.acmicpc.net/problem/1018)

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 보드에서 모든 가능한 8x8 체스판을 검사하여, 각 체스판이 'B' 또는 'W'로 시작할 때 필요한 칸 수를 계산하는 것이다. 이 과정을 통해 최소한의 수정이 필요한 체스판을 찾아내는 것이다.

### 동작 과정
1. **입력 받기**: 먼저 보드의 크기 `n`과 `m`을 입력받고, `n`개의 줄을 통해 보드를 리스트 형태로 저장한다.
   
2. **체스판 탐색**: 모든 가능한 8x8 체스판의 시작 좌표 `(i, j)`를 탐색한다. 이 범위는 `n-7`과 `m-7`으로 설정하여, 8x8 체스판이 보드의 범위 내에 있도록 한다.

3. **수정할 칸 수 계산**:
   - 두 개의 카운터 `a1`과 `a2`를 초기화 한다. `a1`은 'B'로 시작하는 체스판에서 수정해야 할 칸의 수, `a2`는 'W'로 시작하는 체스판에서 수정해야 할 칸의 수를 저장한다.
   - 8x8 체스판 내의 각 칸 `(a, b)`를 검사하면서, `(a + b) % 2`에 따라 해당 칸이 어떤 색이어야 하는지를 판단하고, 실제 보드와 비교하여 수정해야 할 칸 수를 증가시킨다.

4. **결과 저장**: 모든 8x8 체스판에 대해 수정해야 할 칸 수를 `result` 리스트에 추가한다.

5. **최소값 출력**: 최종적으로 `result` 리스트에서 최소값을 찾아 출력한다. 이는 수정이 가장 적게 필요한 경우를 의미한다.

#### 예시 입력
```
8 8
WBWBWBWB
BWBWBWBW
WBWBWBWB
BWBWBWBW
WBWBWBWB
BWBWBWBW
WBWBWBWB
BWBWBWBW
```

#### 단계별 설명
- 입력된 보드는 완벽한 체스판이다.
- 모든 가능한 8x8 체스판의 시작 좌표 `(i, j)`를 탐색하였을 때, 해당하는 모든 경우에서 수정할 필요가 없는 것으로 계산된다.
- 결과적으로 `a1`과 `a2`가 모두 0이 되어, `result`에 0이 추가되고, 출력은 0이 된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n * m)입니다. 이는 보드의 모든 위치를 검사하는 과정에서, 각 8x8 체스판을 만들기 위해 최대 64회 (8x8) 반복하는 작업을 수행하기 때문입니다. 따라서 전체적으로 n과 m에 비례하는 복잡도를 가진다.

## 코드
```python
import sys

n, m = map(int, input().split())

board = []  # 입력받은 보드
result = []  # 결과 값

for _ in range(n):
    board.append(input())

# 8x8 체스판을 만들기 위한 시작 좌표 (i, j)를 탐색
for i in range(n-7):  # 세로 범위
    for j in range(m-7):  # 가로 범위
        a1 = 0  # 'B'로 시작하는 체스판에서 칠해야 할 칸의 수
        a2 = 0  # 'W'로 시작하는 체스판에서 칠해야 할 칸의 수

        for a in range(i, i+8):
            for b in range(j, j+8):
               
                if (a+b) % 2 == 0:   # 좌표의 합이 짝수: 시작 칸과 같은 색이어야 함
                    if board[a][b] != 'B':  # 'B'로 시작하는 경우: 짝수 좌표에 'B'가 있어야 함
                        a1 += 1    # 다르면 칠해야 하므로 a1 증가
                    if board[a][b] != 'W':  # 'W'로 시작하는 경우: 짝수 좌표에 'W'가 있어야 함
                        a2 += 1    # 다르면 칠해야 하므로 a2 증가
                else:
                    # 좌표의 합이 홀수: 시작 칸과 반대 색이어야 함
                    if board[a][b] != 'W':  # 'B'로 시작하는 경우: 홀수 좌표에 'W'가 있어야 함
                        a1 += 1   # 다르면 칠해야 하므로 a1 증가
                    if board[a][b] != 'B':  # 'W'로 시작하는 경우: 홀수 좌표에 'B'가 있어야 함
                        a2 += 1  # 다르면 칠해야 하므로 a2 증가

        result.append(a1)
        result.append(a2)

print(min(result))

```
