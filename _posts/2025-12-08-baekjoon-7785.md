## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 직원의 출입 정보를 사용하여 현재 회사에 있는 직원의 목록을 관리하는 것입니다. 각 직원의 이름과 출입 상태(enter 또는 leave)를 기록하고, 현재 회사에 있는 직원은 "enter" 상태인 직원만 남기고, 이를 사전 순의 역순으로 정렬하여 출력합니다.

### 동작 과정
1. **입력 처리**: 먼저, 로그의 수 `n`을 입력받고, `defaultdict`를 사용하여 직원의 출입 상태를 기록할 딕셔너리를 초기화합니다.
2. **출입 기록 처리**: 각 출입 기록을 순차적으로 읽어서 직원의 이름과 출입 상태를 확인합니다. 
   - "enter"인 경우, 해당 직원의 출입 상태를 1 증가시킵니다.
   - "leave"인 경우, 해당 직원의 출입 상태를 1 감소시킵니다.
3. **현재 직원 확인**: 딕셔너리에서 값이 1인 직원들만 추출하여 현재 회사에 있는 직원 목록을 만듭니다.
4. **정렬 및 출력**: 추출한 직원 목록을 사전 순의 역순으로 정렬하여 출력합니다.

**예시 입력**:
```
4
Alice enter
Bob enter
Alice leave
Charlie enter
```
**예시 단계별 설명**:
- `Alice`가 들어오면 `dic`에 `{'Alice': 1}`이 됩니다.
- `Bob`이 들어오면 `dic`에 `{'Alice': 1, 'Bob': 1}`이 됩니다.
- `Alice`가 나가면 `dic`에 `{'Alice': 0, 'Bob': 1}`이 됩니다.
- `Charlie`가 들어오면 `dic`에 `{'Alice': 0, 'Bob': 1, 'Charlie': 1}`이 됩니다.
- 최종적으로 `dic`에서 값이 1인 직원은 `Bob`과 `Charlie`이며, 이들을 사전 순의 역순으로 정렬하면 `Charlie`, `Bob`이 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. 이는 n개의 출입 기록을 처리하는 O(n) 시간과, 최종적으로 직원 이름을 정렬하는 O(k log k) 시간(여기서 k는 현재 회사에 있는 직원 수) 때문입니다. 그러나 k는 n보다 작거나 같으므로, 전체 복잡도를 O(n log n)으로 볼 수 있습니다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
