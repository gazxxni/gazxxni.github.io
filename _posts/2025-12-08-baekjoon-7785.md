## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 직원의 출입 기록을 통해 현재 회사에 있는 직원의 이름을 추적하는 것이다. 출근 시에는 해당 직원의 상태를 '회사에 있음'으로 표시하고, 퇴근 시에는 '회사에 없음'으로 표시한다. 최종적으로 회사에 있는 직원의 이름을 사전 순의 역순으로 정렬하여 출력한다.

### 동작 과정
1. 입력으로 첫 번째 줄에 직원 출입 로그의 수 `n`을 읽는다.
2. `defaultdict`를 사용하여 직원의 출입 상태를 저장할 딕셔너리를 초기화한다.
3. `n`개의 로그를 반복하여, 직원 이름과 출입 상태("enter" 또는 "leave")를 읽는다.
   - 만약 로그가 "enter"라면 해당 직원의 상태를 1 증가시킨다.
   - 만약 로그가 "leave"라면 해당 직원의 상태를 1 감소시킨다.
4. 로그를 모두 처리한 후, 상태 값이 1인 직원들을 결과 리스트에 추가한다.
5. 결과 리스트를 사전 순 역순으로 정렬한 후, 각 직원 이름을 출력한다.

**예시 입력:**
```
4
Alice enter
Bob enter
Alice leave
Charlie enter
```
**단계별 설명:**
- 첫 번째 로그 "Alice enter": Alice의 상태가 1이 됨.
- 두 번째 로그 "Bob enter": Bob의 상태가 1이 됨.
- 세 번째 로그 "Alice leave": Alice의 상태가 0이 됨.
- 네 번째 로그 "Charlie enter": Charlie의 상태가 1이 됨.
- 최종적으로 상태가 1인 직원은 Bob과 Charlie이며, 이들을 사전 순 역순으로 정렬하면 Charlie, Bob이 된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)이다. 로그를 처리하는 부분은 O(n)이고, 최종적으로 사전 순 역순으로 정렬하는 과정이 O(m log m) (m은 현재 회사에 있는 직원 수, 최악의 경우 n과 같다)이다. 따라서 전체적인 시간 복잡도는 O(n log n)이다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
