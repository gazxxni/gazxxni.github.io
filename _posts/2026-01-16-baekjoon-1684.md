## 문제
정수 N을 정수 D로 나눴을 때의 몫을 Q, 나머지를 R이라고 하면 항등식 R = N - Q×D가 성립한다.

n개의 정수로 된 수열이 있을 때, 모든 정수를 한 정수 D로 나눴을 때 나머지가 같아지는 경우가 있다. 그리고 수열에 따라서는 이러한 정수 D가 여러 개 존재할 수 있다.

n개의 정수로 된 수열이 주어졌을 때, 가장 큰 D를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 n(1 ≤ n ≤ 1,000)이 주어진다. 다음 줄에는 절댓값이 1,000,000을 넘지 않는 n개의 정수들이 주어진다.

---

## 출력
첫째 줄에 가장 큰 D를 출력한다. 항상 가장 큰 D가 존재하는 경우만 입력으로 주어진다.

## 풀이

### 풀이 핵심 로직
주어진 수열의 모든 원소를 어떤 정수 D로 나누었을 때 나머지가 같아지기 위해서는, 모든 원소 간의 차이가 D로 나누어 떨어져야 한다. 즉, 차이의 최대공약수(GCD)를 구하면 가장 큰 D를 찾을 수 있다.

### 동작 과정
1. 입력으로 주어진 n개의 정수에서 서로 다른 값의 리스트를 만든다.
2. 이 리스트의 원소들 간의 차이를 구한다.
   - 예를 들어, 주어진 수열이 [3, 6, 9]라면, 차이는 |6-3|=3, |9-3|=6, |9-6|=3이므로, 차이는 [3, 6, 3]이 된다.
3. 이 차이들의 최대공약수를 계산한다.
   - 위의 예에서, GCD(3, 6, 3) = 3이므로, 가장 큰 D는 3이 된다.
4. 결과를 출력한다.

### 시간 복잡도
주어진 수열의 원소 간 차이를 계산하는 과정에서 O(n^2)의 시간복잡도가 발생하지만, n은 최대 1000으로 제한되어 있으므로 효율적으로 처리할 수 있다. GCD 계산은 O(log(max(a, b)))의 시간복잡도를 가지므로, 최종적인 시간복잡도는 O(n^2 * log(max(a, b)))이다.

## 코드
```python
import sys
input = sys.stdin.readline

'''
두 수의 차가 D로 나누어 떨어진다
(aᵢ - aⱼ) % D = 0 -> 성립해야함

따라서 |a₂ - a₁|, |a₃ - a₁|, ..., |aₙ - a₁|
이 숫자들의 최대공약수를 구해야함
'''

n = int(input())
arr = list(map(int, input().split()))

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
        
    return a

brr = list(set(arr))
crr = []

for i in range(len(brr) - 1):
    for j in range(i + 1, len(brr)):
        crr.append(abs(brr[j] - brr[i])) 
        
ans = crr[0]
for i in range(1, len(crr)): 
    ans = gcd(ans, crr[i])
    
print(ans)
```
