## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
팩토리얼은 주어진 정수 N의 모든 양의 정수의 곱으로 정의됩니다. N의 팩토리얼(N!)은 N * (N-1) * (N-2) * ... * 1로 계산됩니다. 특히, 0!은 1로 정의되며, 재귀 함수를 사용하여 N!을 계산할 수 있습니다.

### 동작 과정
1. 프로그램이 시작되면 사용자로부터 정수 N을 입력받습니다.
2. `fac`라는 재귀 함수를 정의합니다. 이 함수는 인자로 받은 N이 1일 경우 1을 반환하고, 그 외의 경우 N과 N-1의 팩토리얼을 곱한 값을 반환합니다.
3. 입력받은 N이 0일 경우, 팩토리얼 값은 1이므로 특별히 1을 출력합니다.
4. 그렇지 않은 경우, `fac` 함수를 호출하여 N!을 계산하고 결과를 출력합니다.

예를 들어, N이 5일 경우:
- fac(5) 호출 → 5 * fac(4)
- fac(4) 호출 → 4 * fac(3)
- fac(3) 호출 → 3 * fac(2)
- fac(2) 호출 → 2 * fac(1)
- fac(1) 호출 → 1 반환
- 이 값을 차례로 곱하여 최종적으로 120을 출력합니다.

### 시간 복잡도
O(N) - N의 크기에 따라 재귀 호출이 N번 이루어지므로 시간 복잡도는 O(N)입니다.

---

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
