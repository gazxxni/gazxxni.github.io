## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 N부터 1까지의 모든 정수를 곱한 값으로 정의됩니다. 재귀 함수를 이용하여 N이 1일 때 1을 반환하고, 그 외의 경우에는 N과 N-1의 팩토리얼 값을 곱하여 결과를 구합니다. N이 0일 때는 특별히 1로 정의되어 있습니다.

### 동작 과정
1. 입력으로 정수 N을 받습니다. 예를 들어 N = 5인 경우를 가정합니다.
2. `fac` 함수가 호출되며, N이 5이므로 `fac(5)`가 실행됩니다.
3. `fac(5)`는 5와 `fac(4)`의 곱을 반환합니다.
   - `fac(4)`는 4와 `fac(3)`의 곱을 반환합니다.
     - `fac(3)`는 3과 `fac(2)`의 곱을 반환합니다.
       - `fac(2)`는 2와 `fac(1)`의 곱을 반환합니다.
         - `fac(1)`은 1을 반환합니다.
4. 이제 반환된 값을 차례대로 계산합니다:
   - `fac(2)`는 2 * 1 = 2
   - `fac(3)`는 3 * 2 = 6
   - `fac(4)`는 4 * 6 = 24
   - `fac(5)`는 5 * 24 = 120
5. 따라서, 출력 결과는 120입니다.

### 시간 복잡도
시간 복잡도는 O(N)입니다. 재귀 호출이 N번 발생하기 때문에 N에 비례하는 시간 복잡도를 가집니다.

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
