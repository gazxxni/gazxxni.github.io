## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 M개를 고른 수열
	같은 수를 여러 번 골라도 된다.

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
이 문제는 백트래킹을 이용하여 해결할 수 있습니다. 주어진 N과 M에 대해 1부터 N까지의 자연수를 M개 선택하는 모든 조합을 구하는 것이 목표입니다. 이때, 같은 수를 여러 번 사용할 수 있으므로 재귀적으로 수열을 생성하고, 깊이가 M에 도달하면 결과를 출력합니다.

### 동작 과정
1. **입력**: N과 M을 입력받습니다. 예를 들어, 입력이 `3 2`일 경우, N은 3, M은 2입니다.
2. **재귀 함수** `aa()`를 정의합니다.
   - 현재 수열의 길이가 M에 도달하면, 수열을 출력합니다.
   - 그렇지 않으면 1부터 N까지의 수를 반복하여 수열에 추가하고, 재귀 호출을 통해 다음 깊이로 진행합니다.
   - 재귀 호출 후에는 마지막에 추가한 수를 제거하여 다음 수를 추가할 준비를 합니다.
3. **결과 출력**: 수열이 완성되면 결과를 출력합니다.

예시로, 입력이 `3 2`일 경우, 다음과 같은 단계로 동작합니다:
- `arr`이 비어있는 상태에서 시작
- `1`을 추가한 후, 다시 `1`을 추가 -> [1, 1] 출력
- `1`을 추가한 후, `2`를 추가 -> [1, 2] 출력
- `1`을 추가한 후, `3`을 추가 -> [1, 3] 출력
- `2`를 추가한 후, 후속 수를 추가하여 반복
- 이 과정을 반복하여 모든 조합을 탐색하게 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^M)입니다. M개의 수열을 만들 때, 각 위치에 대해 N개의 선택이 가능하기 때문입니다. M이 최대 7, N이 최대 7인 경우, 최악의 경우의 수는 7^7 = 823543입니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
