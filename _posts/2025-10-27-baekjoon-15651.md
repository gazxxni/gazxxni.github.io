[https://www.acmicpc.net/problem/15651](https://www.acmicpc.net/problem/15651)

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 숫자 `n`에 대해 1부터 `n`까지의 숫자를 중복을 허용하여 `m`개 선택하는 조합을 출력하는 문제입니다. 핵심 아이디어는 재귀를 사용하여 현재 선택한 숫자를 배열에 추가하고, 배열의 길이가 `m`에 도달할 때마다 배열을 출력하는 방식입니다.

### 동작 과정
예를 들어, 입력이 `3 2`라면, `n=3`이고 `m=2`입니다. 아래는 이 코드의 단계별 설명입니다.

1. **초기화**: `arr`라는 빈 리스트를 정의하고, `aa()`라는 재귀 함수를 호출합니다.
2. **재귀 호출**:
   - 처음 `aa()`가 호출되면 `arr`의 길이가 `m`(2)인지 체크합니다. 현재는 빈 리스트이므로 조건을 만족하지 않습니다.
   - `for` 루프가 시작되며, `i`는 1부터 `n`(3)까지 반복됩니다.
   - 첫 번째 반복 (`i=1`): `arr`에 1을 추가하고 `aa()`를 재귀 호출합니다.
     - 두 번째 호출에서 `arr`의 길이는 1이므로 다시 `for` 루프가 반복됩니다.
     - `i=1`: `arr`에 1을 추가하고 `aa()`를 재귀 호출합니다. (현재 `arr`은 `[1, 1]`)
       - `arr`의 길이가 2이므로, `[1, 1]`을 출력합니다.
     - `i=2`: `arr`에 2를 추가하고 `aa()`를 재귀 호출합니다. (현재 `arr`은 `[1, 2]`)
       - `arr`의 길이가 2이므로, `[1, 2]`을 출력합니다.
     - `i=3`: `arr`에 3을 추가하고 `aa()`를 재귀 호출합니다. (현재 `arr`은 `[1, 3]`)
       - `arr`의 길이가 2이므로, `[1, 3]`을 출력합니다.
   - 이후, `arr`에서 마지막 숫자(1)를 제거합니다.
   - `i=2`: `arr`에 2를 추가하고 `aa()`를 재귀 호출합니다. (현재 `arr`은 `[2]`)
     - 이와 같은 방식으로 `[2, 1]`, `[2, 2]`, `[2, 3]`을 출력합니다.
   - 마지막으로, `i=3`일 때 `[3, 1]`, `[3, 2]`, `[3, 3]`을 출력합니다.

결과적으로 출력되는 조합은:
```
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \(O(n^m)\)입니다. `n`은 선택할 수 있는 숫자의 개수이고, `m`은 선택할 숫자의 개수입니다. 각 숫자 선택 시 모든 가능한 조합을 탐색해야 하므로, 모든 경우의 수는 `n`의 `m`제곱에 비례합니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
