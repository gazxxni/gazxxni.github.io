## 풀이

### 풀이 핵심 로직
이 문제는 주어진 점수 리스트에서 상위 15%와 하위 15%의 점수를 제외한 평균 점수를 계산하는 문제이다. 이를 위해 먼저 리스트를 정렬한 후, 상위와 하위 15%에 해당하는 점수를 제외하고 남은 점수들의 평균을 구한다. 평균을 구할 때는 반올림 규칙을 적용하여 최종 결과를 출력한다.

### 동작 과정
1. 입력으로 주어진 점수의 개수 `n`을 읽는다.
2. 점수가 0이 아닌 경우에만 다음 단계를 진행한다.
3. `n`개의 점수를 리스트에 저장한다.
4. `nn`을 계산하여 15%에 해당하는 점수의 개수를 정수로 반올림한다.
5. 점수 리스트를 정렬한다.
6. 만약 `nn`이 0보다 크면 상위 15%와 하위 15%를 제외한 나머지 점수들의 평균을 계산한다. 이를 위해 리스트에서 `nn`만큼 앞과 뒤의 점수를 제외하고, 남은 점수들의 합을 그 개수로 나누어 평균을 구한다.
7. `nn`이 0이면 모든 점수를 사용하여 평균을 계산한다.
8. 최종적으로 계산한 평균을 반올림하여 출력한다.
9. 점수가 0인 경우, 0을 출력한다.

예시 입력:
```
5
1
2
3
4
5
```
- `n`은 5, 점수 리스트는 [1, 2, 3, 4, 5]이다.
- `nn`은 `my_round(5 * 0.15)`, 즉 `my_round(0.75)`이 되어 1이 된다.
- 리스트를 정렬하면 변하지 않고 [1, 2, 3, 4, 5]이다.
- 상위 1개와 하위 1개 점수를 제외하면 남은 점수는 [2, 3, 4]이고, 이들의 평균은 (2 + 3 + 4) / 3 = 3이 된다.
- 최종적으로 3을 반올림하여 출력한다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)이다. 점수 리스트를 정렬하는 데 O(n log n) 시간이 소요되기 때문이다. 나머지 연산들은 O(n) 시간에 수행된다.

## 코드
```python
import sys

def my_round(val):
    if val-int(val)>=0.5:
        return int(val)+1
    else:
        return int(val)

n=int(sys.stdin.readline().strip())

if n:
    level=[]
    for _ in range(n):
        level.append(int(sys.stdin.readline().strip()))

    nn=my_round(n*0.15)
    level.sort()

    if nn>0:
        print(my_round(sum(level[nn:-nn])/len(level[nn:-nn])))
    else:
        print(my_round(sum(level)/len(level)))

else:
    print(0)

    
```
