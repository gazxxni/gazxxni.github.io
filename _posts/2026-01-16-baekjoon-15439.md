## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
상의와 하의의 색상이 서로 다르다는 조건을 만족하는 조합의 수를 구하기 위해, 상의의 색상으로 선택할 수 있는 N가지 중 한 가지를 선택한 후, 하의의 색상은 상의와 다른 색상 N-1가지 중에서 선택할 수 있습니다. 따라서 총 가능한 조합의 수는 \( N \times (N - 1) \)이 됩니다.

### 동작 과정
1. 입력으로 주어진 N을 읽습니다. 예를 들어, N = 3일 경우, 상의와 하의의 색상은 다음과 같습니다:
   - 상의: 1, 2, 3
   - 하의: 1, 2, 3
2. 상의의 색상을 선택할 수 있는 경우의 수는 3가지입니다:
   - 상의가 색상 1일 때: 하의는 색상 2 또는 3 (2가지)
   - 상의가 색상 2일 때: 하의는 색상 1 또는 3 (2가지)
   - 상의가 색상 3일 때: 하의는 색상 1 또는 2 (2가지)
3. 따라서 총 조합의 수는 \( 3 \times 2 = 6 \)입니다.
4. 이 과정에서 모든 가능한 (상의, 하의) 조합을 고려하여 서로 다른 색상의 조합만을 셉니다.

### 시간 복잡도
시간 복잡도는 \( O(1) \)입니다. 입력 N을 읽고, 상수 시간에 조합 수를 계산하는 단일 수식을 수행하기 때문입니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
