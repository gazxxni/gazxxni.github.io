## 풀이

### 풀이 핵심 로직
이 문제는 주어진 수를 스택에 추가하고, 0이 입력될 경우 스택에서 가장 최근에 추가된 수를 제거하는 방식으로 해결합니다. 최종적으로 스택에 남아 있는 모든 수의 합을 구하여 출력합니다. 스택을 사용하여 LIFO(Last In First Out) 특성을 활용하는 것이 핵심입니다.

### 동작 과정
예를 들어, 입력이 다음과 같다고 가정해 보겠습니다:
```
5
1
2
0
3
0
```
1. 첫 번째 입력 `5`는 스택의 크기를 결정합니다.
2. 두 번째 입력 `1`이 들어오면 스택에 `1`을 추가합니다. (stack = [1])
3. 세 번째 입력 `2`가 들어오면 스택에 `2`를 추가합니다. (stack = [1, 2])
4. 네 번째 입력 `0`이 들어오면 스택에서 가장 최근에 추가된 `2`를 제거합니다. (stack = [1])
5. 다섯 번째 입력 `3`이 들어오면 스택에 `3`을 추가합니다. (stack = [1, 3])
6. 마지막 입력 `0`이 들어오면 스택에서 `3`을 제거합니다. (stack = [1])
7. 스택에 남아 있는 수는 `1`뿐이므로, 최종적으로 `1`의 합을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 여기서 n은 입력의 수(k)입니다. 각 입력에 대해 스택에 추가하거나 제거하는 연산은 O(1)로 수행되므로, 전체 입력에 대해 선형 시간 복잡도를 가집니다.

## 코드
```python
import sys

k=int(input())
stack=[]

for i in range(k):
    a=int(input())

    if a==0:
        stack.pop()
    else:
        stack.append(a)

print(sum(stack))


```
