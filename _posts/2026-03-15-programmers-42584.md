## 문제
초 단위로 기록된 주식가격이 담긴 배열 prices가 매개변수로 주어질 때, 가격이 떨어지지 않은 기간은 몇 초인지를 return 하도록 solution 함수를 완성하세요.

제한사항


prices의 각 가격은 1 이상 10,000 이하인 자연수입니다.
prices의 길이는 2 이상 100,000 이하입니다.


입출력 예

        
prices
return


        
[1, 2, 3, 2, 3]
[4, 3, 1, 1, 0]


      
입출력 예 설명


1초 시점의 ₩1은 끝까지 가격이 떨어지지 않았습니다.
2초 시점의 ₩2은 끝까지 가격이 떨어지지 않았습니다.
3초 시점의 ₩3은 1초뒤에 가격이 떨어집니다. 따라서 1초간 가격이 떨어지지 않은 것으로 봅니다.
4초 시점의 ₩2은 1초간 가격이 떨어지지 않았습니다.
5초 시점의 ₩3은 0초간 가격이 떨어지지 않았습니다.

## 풀이
### 풀이 핵심 로직
이 문제의 핵심은 각 주식 가격이 떨어지지 않은 시간(초)을 계산하는 것입니다. 스택을 사용하여 현재 주식 가격이 이전 가격보다 떨어지는 경우를 추적하고, 떨어지지 않은 기간을 세어 결과 리스트에 저장합니다. 스택에는 (인덱스, 가격) 쌍을 저장하여 가격이 떨어지는 시점을 효과적으로 확인할 수 있습니다.

### 동작 과정
1. **초기화**: 주식 가격의 개수를 `n`으로 설정하고, 결과를 저장할 리스트 `answer`를 0으로 초기화합니다. 스택을 사용하여 (인덱스, 가격) 쌍을 저장할 준비를 합니다.
  
2. **첫 번째 가격부터 순회**: 인덱스가 1부터 `n-1`까지의 가격을 순회하면서 아래 작업을 수행합니다.
   - 스택의 top 요소(가장 최근에 저장된 가격)와 현재 가격을 비교합니다.
   - 현재 가격이 스택의 top 가격보다 낮다면, 가격이 떨어진 것이므로 스택에서 pop하고 해당 가격이 떨어지기까지의 시간을 계산하여 `answer`에 저장합니다.
   - 현재 가격을 스택에 추가합니다.

3. **마무리 작업**: 모든 가격을 순회한 후에도 스택에 남아있는 요소들은 마지막 가격까지 떨어지지 않았으므로, 남아있는 인덱스에 대해 남은 초를 계산하여 `answer`에 저장합니다.

4. **결과 반환**: 최종적으로 각 주식 가격에 대한 떨어지지 않은 초를 담은 리스트 `answer`를 반환합니다.

#### 예시 입력으로 단계별 설명
- 예를 들어, 입력이 `[1, 2, 3, 2, 3]`인 경우:
  - **초 0** (가격 1): 끝까지 떨어지지 않음 → `answer[0]` = 4
  - **초 1** (가격 2): 끝까지 떨어지지 않음 → `answer[1]` = 3
  - **초 2** (가격 3): 1초 후에 가격 2로 떨어짐 → `answer[2]` = 1
  - **초 3** (가격 2): 1초 후에 가격 3으로 떨어지지 않음 → `answer[3]` = 1
  - **초 4** (가격 3): 마지막 가격이므로 떨어지지 않음 → `answer[4]` = 0

최종적으로 `answer`는 `[4, 3, 1, 1, 0]`이 됩니다.

### 시간 복잡도
이 알고리즘은 각 가격을 한 번씩만 스택에 푸시하고 팝하기 때문에 시간 복잡도는 O(n)입니다. 여기서 n은 주식 가격의 개수입니다.

## 코드
```python
def solution(prices):
    n = len(prices)
    answer = [0] * n
    stack = [(0, prices[0])]
    
    for i, v in enumerate(prices[1:n], start=1):
        while stack and stack[-1][1] > v:
            answer[stack[-1][0]] = i - stack[-1][0]
            stack.pop(-1)
        stack.append((i, v))
        
    while stack:
        answer[stack[-1][0]] = (n - 1) - stack[-1][0]
        stack.pop(-1)
        
    return answer
```
