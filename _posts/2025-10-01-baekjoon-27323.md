## 풀이

### 풀이 핵심 로직
문제는 두 개의 정수 A와 B를 입력받아 이들의 곱을 출력하는 것입니다. 핵심 아이디어는 입력된 두 정수를 곱한 결과를 계산하고 출력하는 간단한 연산으로, 기본적인 입출력과 산술 연산에 대한 이해가 필요합니다.

### 동작 과정
1. 사용자로부터 첫 번째 정수 A를 입력받습니다. 예를 들어, A = 3.
2. 사용자로부터 두 번째 정수 B를 입력받습니다. 예를 들어, B = 4.
3. 두 정수 A와 B를 곱합니다. 즉, 3 * 4 = 12.
4. 결과인 12를 출력합니다.

입력 예시:
```
3
4
```
출력:
```
12
```

### 시간 복잡도
이 알고리즘은 상수 시간에 수행되므로, 시간 복잡도는 O(1)입니다. 입력된 두 정수의 곱을 계산하는 연산은 입력의 크기에 관계없이 항상 같은 시간이 소요됩니다.

## 코드
```python
a = int(input())
b = int(input())

print(a * b)
```
