## 문제
지난 밤 겨울 숲에는 눈이 많이 내렸다. 당신은 숲의 주민들을 위해 눈이 오지 않는 동안 모든 집 앞의 눈을 치우고자 한다.

당신은 1분에 한 번씩 두 집을 선택해서 두 집 앞의 눈을 각각 1만큼 치우거나, 한 집을 선택해서 그 집 앞의 눈을 1만큼 치울 수 있다.

모든 집 앞의 눈을 전부 치울 때까지 걸리는 최소 시간은 얼마일까?

---

## 입력
첫 줄에 집의 수를 의미하는 정수 N$N$ (1≤N≤100$1 \leq N \leq 100$)이 주어진다.

다음 줄에는 각각의 집 앞에 쌓여 있는 눈의 양을 나타내는 정수 ai$a_{i}$ (1≤ai≤2000$1 \leq a_{i} \leq 2000$)이 주어진다.

---

## 출력
모든 집 앞의 눈을 치우는 데 최소 몇 분이 걸리는지를 출력한다. 24시간(1440분)이 넘게 걸릴 경우 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 눈의 양이 가장 많은 두 집을 선택하여 동시에 눈을 치우는 것이다. 이를 통해 최소한의 시간 안에 모든 집 앞의 눈을 제거할 수 있으며, 정렬을 통해 매번 가장 많은 눈이 쌓인 두 집을 쉽게 선택할 수 있다.

### 동작 과정
1. **입력**: 집의 수 N과 각 집 앞의 눈의 양을 입력받는다.
   - 예시 입력: 
     ```
     3
     5 3 8
     ```
2. **정렬**: 눈의 양을 내림차순으로 정렬한다. 현재 상태는 [8, 5, 3]이 된다.
3. **눈 치우기**:
   - 첫 번째 반복:
     - 가장 많은 눈이 쌓인 두 집을 선택: 8(집 1)과 5(집 2) 
     - 각각 5만큼 눈을 치우고 집 2의 눈은 0이 된다. 현재 상태는 [3, 3, 0]이 되고, 총 시간은 5분이 소요된다.
   - 두 번째 반복:
     - 현재 상태를 다시 정렬하여 [3, 3, 0]이 되고, 다시 3과 3을 선택.
     - 각각 3만큼 눈을 치우고, 더 이상 눈이 없는 집은 제거. 총 시간은 8분이 된다.
   - 반복이 끝나면 모든 집의 눈이 치워진다.
4. **결과 출력**: 최종 시간이 1440분을 초과하지 않으면 총 시간을 출력한다. 예시의 경우 최종 시간 8분이므로 출력은 8이 된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)이다. 매 반복마다 배열을 정렬하는 데 O(N log N)의 시간이 소요되며, 눈을 치우는 과정은 O(N)으로, 정렬과 반복의 조합으로 인해 전체 시간 복잡도는 O(N log N)으로 나타난다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

cnt = 0

while True:
    if len(arr) == 1:
        break
    
    arr.sort(reverse=True)
    arr[0] -= arr[1]
    cnt += arr[1]
    arr.pop(1)
    
cnt += arr[0]
if cnt > 1440:
    print(-1)
else:
    print(cnt)
```
