## 문제
지난 밤 겨울 숲에는 눈이 많이 내렸다. 당신은 숲의 주민들을 위해 눈이 오지 않는 동안 모든 집 앞의 눈을 치우고자 한다.

당신은 1분에 한 번씩 두 집을 선택해서 두 집 앞의 눈을 각각 1만큼 치우거나, 한 집을 선택해서 그 집 앞의 눈을 1만큼 치울 수 있다.

모든 집 앞의 눈을 전부 치울 때까지 걸리는 최소 시간은 얼마일까?

---

## 입력
첫 줄에 집의 수를 의미하는 정수 N$N$ (1≤N≤100$1 \leq N \leq 100$)이 주어진다.

다음 줄에는 각각의 집 앞에 쌓여 있는 눈의 양을 나타내는 정수 ai$a_{i}$ (1≤ai≤2000$1 \leq a_{i} \leq 2000$)이 주어진다.

---

## 출력
모든 집 앞의 눈을 치우는 데 최소 몇 분이 걸리는지를 출력한다. 24시간(1440분)이 넘게 걸릴 경우 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심은 눈을 치우는 최적의 방법을 찾는 것입니다. 두 집을 동시에 선택하여 눈을 치우는 것이 가장 효율적이므로, 눈의 양이 많은 집을 우선적으로 선택하여 최대한 많은 눈을 동시에 치우는 방식으로 진행합니다. 이를 위해 집 앞 눈의 양을 정렬하여 가장 많은 눈이 쌓인 집 두 곳을 선택합니다.

### 동작 과정
예시 입력: 
```
5
3 1 4 1 5
```
1. 처음 집 앞의 눈의 양은 `[3, 1, 4, 1, 5]`입니다.
2. 배열을 내림차순으로 정렬하면 `[5, 4, 3, 1, 1]`이 됩니다.
3. 가장 많은 눈이 쌓인 두 집(5와 4)을 선택하여 각각 1만큼 눈을 치우면 `[4, 3, 3, 1, 1]`이 됩니다. 이 과정에서 1분이 소요됩니다.
4. 다시 배열을 정렬하면 `[4, 3, 3, 1, 1]`입니다.
5. 같은 방법으로 반복하여 눈을 치우는 과정을 계속 진행합니다.
6. 모든 집 앞 눈이 0이 될 때까지 반복하며, 총 소요된 시간을 계산합니다.
7. 만약 소요된 시간이 1440분을 넘는다면 `-1`을 출력합니다.

최종적으로 모든 집의 눈을 치우면 경과 시간을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2 log N)입니다. 각 반복문에서 배열을 정렬하는데 O(N log N)의 시간이 소요되고, 최악의 경우 N번 반복할 수 있기 때문입니다. 하지만, N이 최대 100이므로 이 시간 복잡도는 실용적인 범위 내에 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

cnt = 0

while True:
    if len(arr) == 1:
        break
    
    arr.sort(reverse=True)
    arr[0] -= arr[1]
    cnt += arr[1]
    arr.pop(1)
    
cnt += arr[0]
if cnt > 1440:
    print(-1)
else:
    print(cnt)
```
