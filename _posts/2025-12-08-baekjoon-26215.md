## 문제
지난 밤 겨울 숲에는 눈이 많이 내렸다. 당신은 숲의 주민들을 위해 눈이 오지 않는 동안 모든 집 앞의 눈을 치우고자 한다.

당신은 1분에 한 번씩 두 집을 선택해서 두 집 앞의 눈을 각각 1만큼 치우거나, 한 집을 선택해서 그 집 앞의 눈을 1만큼 치울 수 있다.

모든 집 앞의 눈을 전부 치울 때까지 걸리는 최소 시간은 얼마일까?

---

## 입력
첫 줄에 집의 수를 의미하는 정수 N$N$ (1≤N≤100$1 \leq N \leq 100$)이 주어진다.

다음 줄에는 각각의 집 앞에 쌓여 있는 눈의 양을 나타내는 정수 ai$a_{i}$ (1≤ai≤2000$1 \leq a_{i} \leq 2000$)이 주어진다.

---

## 출력
모든 집 앞의 눈을 치우는 데 최소 몇 분이 걸리는지를 출력한다. 24시간(1440분)이 넘게 걸릴 경우 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 각 집 앞의 눈을 효율적으로 치우기 위해 가장 많은 눈이 쌓인 두 집을 선택하여 동시에 눈을 치우는 것입니다. 이를 통해 눈이 가장 많이 쌓인 집부터 우선적으로 처리하여 최소한의 시간을 소모하는 방식으로 진행합니다. 만약 모든 집의 눈을 치우는 데 걸리는 시간이 24시간을 넘는다면 -1을 출력합니다.

### 동작 과정
1. 입력으로 주어진 집의 수 N과 각 집 앞의 눈의 양을 리스트로 받습니다.
2. 리스트를 내림차순으로 정렬하여 가장 많은 눈이 쌓인 집을 먼저 선택합니다.
3. 두 집을 선택하여 각 집 앞의 눈을 동시에 치우는 과정에서, 두 번째로 많은 눈이 쌓인 집의 눈의 양만큼 첫 번째 집의 눈이 줄어들고, 그 집의 눈이 모두 치워지면 리스트에서 제거합니다.
4. 이러한 과정을 모든 집의 눈이 치워질 때까지 반복합니다.
5. 최종적으로 걸린 시간을 계산하여 1440분을 초과하면 -1을 출력하고, 그렇지 않으면 시간을 출력합니다.

#### 예시 입력
```
5
5 3 2 6 1
```
- 초기 상태: [5, 3, 2, 6, 1]
- 1분 후: [4, 2, 2, 5, 1] (6과 5를 선택하여 동시에 치움)
- 2분 후: [3, 2, 2, 4, 1]
- 3분 후: [2, 2, 2, 3, 1]
- 4분 후: [1, 2, 2, 2, 1]
- 5분 후: [0, 2, 2, 2, 0]
- 6분 후: [0, 1, 1, 1, 0]
- 7분 후: [0, 0, 1, 0, 0]
- 8분 후: [0, 0, 0, 0, 0]
- 총 소요 시간: 8분

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 각 반복에서 리스트를 정렬하는 데 O(N log N)의 시간이 소요되고, 모든 집의 눈을 치울 때까지 반복하는 과정에서 최대 N번의 반복이 필요합니다. 따라서 최악의 경우 O(N^2)로 계산됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

cnt = 0

while True:
    if len(arr) == 1:
        break
    
    arr.sort(reverse=True)
    arr[0] -= arr[1]
    cnt += arr[1]
    arr.pop(1)
    
cnt += arr[0]
if cnt > 1440:
    print(-1)
else:
    print(cnt)
```
