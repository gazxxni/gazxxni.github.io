## 풀이

### 풀이 핵심 로직
이 문제는 2xN 크기의 직사각형을 1x2와 2x1 타일로 채우는 경우의 수를 구하는 문제입니다. 이를 해결하기 위해 동적 프로그래밍(Dynamic Programming) 기법을 사용하여 각 길이 N에 대한 경우의 수를 이전 두 길이의 경우의 수를 이용해 점진적으로 계산합니다. 즉, `f(n) = f(n-1) + f(n-2)`의 점화식을 사용하여 문제를 해결합니다.

### 동작 과정
1. **입력**: 사용자로부터 N 값을 입력받습니다. 예를 들어, N이 4라고 가정합니다.
2. **초기화**: 
   - `arr[1]`을 1로 설정합니다. (1x2 타일 하나로만 구성 가능한 경우)
   - `arr[2]`을 2로 설정합니다. (1x2 타일 두 개 또는 2x1 타일 하나로 구성 가능한 경우)
3. **동적 프로그래밍 테이블 채우기**:
   - N이 3 이상인 경우, 반복문을 통해 `arr[i]`를 계산합니다. 예를 들어, 
     - `arr[3]` = `arr[2]` + `arr[1]` = 2 + 1 = 3
     - `arr[4]` = `arr[3]` + `arr[2]` = 3 + 2 = 5
4. **출력**: 마지막으로 `arr[n]` 값을 10007로 나눈 나머지를 출력합니다. 예를 들어, N이 4일 때, `arr[4]`는 5이므로 5를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 이는 N까지의 경우의 수를 계산하기 위해 한 번의 반복문을 사용하기 때문입니다.

---

## 코드
```python
import sys

n=int(input())

arr = [0] * 1001
arr[1] = 1
arr[2] = 2

for i in range(3,1001):
    arr[i] = arr[i-1] + arr[i-2]

print(arr[n]%10007)


```
