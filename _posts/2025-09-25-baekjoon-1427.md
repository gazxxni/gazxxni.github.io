## 풀이

### 풀이 핵심 로직
주어진 정수를 내림차순으로 정렬하여 출력하는 문제입니다. 입력된 숫자를 각 자리수로 분리한 후, 이를 정렬하여 가장 큰 자릿수부터 출력하는 방식으로 해결합니다. 이 과정에서 문자열로 입력을 받고, 각 문자를 정수로 변환하여 리스트에 저장한 뒤, 정렬합니다.

### 동작 과정
1. **입력 받기**: `input()` 함수를 사용하여 숫자를 문자열로 입력받습니다. 예를 들어, 입력이 `2143`이라면 `n`은 `"2143"`가 됩니다.
2. **문자열을 리스트로 변환**: `map(int, n)`을 사용하여 문자열의 각 문자를 정수로 변환하고, 이를 리스트로 만듭니다. 이 경우 `a`는 `[2, 1, 4, 3]`이 됩니다.
3. **정렬**: `a.sort(reverse=True)`를 통해 리스트를 내림차순으로 정렬합니다. 정렬 후 `a`는 `[4, 3, 2, 1]`이 됩니다.
4. **출력**: `print(*a, sep='')`를 사용하여 정렬된 리스트를 공백 없이 출력합니다. 최종 출력은 `4321`이 됩니다.

### 시간 복잡도
- 입력을 처리하는 데 O(N) (N은 숫자의 자릿수)
- 리스트를 정렬하는 데 O(N log N)
- 따라서 전체 시간 복잡도는 O(N log N)입니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
