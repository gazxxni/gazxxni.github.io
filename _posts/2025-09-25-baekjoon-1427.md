## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 자연수 N의 각 자리수를 분리한 뒤, 이를 내림차순으로 정렬하여 새로운 수를 형성하는 것이 핵심 아이디어입니다. 이를 위해 문자열로 입력받은 N을 리스트로 변환한 후, 정렬 기능을 이용하여 자리수를 정렬합니다.

### 동작 과정
1. 입력으로 주어진 수 N을 문자열로 받아옵니다. 예를 들어, 입력이 `2143`이라면 문자열로 `'2143'`가 됩니다.
2. 이 문자열을 각 자리수의 정수로 변환하여 리스트로 만듭니다. 즉, `['2', '1', '4', '3']`에서 각 숫자를 정수로 변환하여 `[2, 1, 4, 3]`가 됩니다.
3. 이 리스트를 내림차순으로 정렬합니다. 정렬 후의 리스트는 `[4, 3, 2, 1]`이 됩니다.
4. 최종적으로 정렬된 리스트의 숫자를 문자열로 합쳐서 출력합니다. 결과는 `4321`이 됩니다.

예시 입력과 출력:
- 입력: `2143`
- 출력: `4321`

### 시간 복잡도
입력받은 수 N의 자리수 개수를 k라고 할 때, 정렬의 시간 복잡도는 O(k log k)입니다. 따라서 최악의 경우 k는 10(최대 1,000,000,000의 자리수)이고, 전체 시간 복잡도는 O(10 log 10)으로 상수 시간에 가까워 매우 효율적입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
