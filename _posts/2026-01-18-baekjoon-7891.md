## 문제 링크
[https://www.acmicpc.net/problem/7891](https://www.acmicpc.net/problem/7891)

---

## 문제
Given two integers, calculate and output their sum.

---

## 입력
The input contains several test cases. The first line contains and integer t (t ≤ 100) denoting the number of test cases. Then t tests follow, each of them consisiting of two space separated integers x and y (−109 ≤ x, y ≤ 109).

---

## 출력
For each test case output output the sum of the corresponding integers.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

## 풀이

### 풀이 핵심 로직
주어진 문제는 여러 개의 테스트 케이스에서 두 정수의 합을 구하는 것입니다. 입력으로 주어진 정수 쌍에 대해 단순히 덧셈 연산을 수행하고 결과를 출력하면 됩니다.

### 동작 과정
1. 첫 줄에서 테스트 케이스의 수 `t`를 입력받습니다.
2. 반복문을 통해 `t`번 만큼 다음 단계를 수행합니다.
   - 각 테스트 케이스에 대해 두 개의 정수 `x`와 `y`를 공백으로 구분하여 입력받습니다.
   - 입력받은 두 정수를 더한 결과를 출력합니다.

예시 입력:
```
3
1 2
-3 5
1000000000 -1000000000
```

단계별 설명:
- 첫 번째 줄 `3`은 테스트 케이스의 수입니다.
- 첫 번째 테스트 케이스 `1 2`의 합은 `3`입니다.
- 두 번째 테스트 케이스 `-3 5`의 합은 `2`입니다.
- 세 번째 테스트 케이스 `1000000000 -1000000000`의 합은 `0`입니다.
- 최종 출력은 `3`, `2`, `0`이 됩니다.

### 시간 복잡도
주어진 입력의 수에 따라 선형적으로 증가하는 시간 복잡도를 가지므로, 시간 복잡도는 O(t)입니다. 여기서 t는 테스트 케이스의 수입니다.

---

## 코드
```python
t = int(input())

for _ in range(t):
    a, b = map(int, input().split())
    
    print(a + b)
```
