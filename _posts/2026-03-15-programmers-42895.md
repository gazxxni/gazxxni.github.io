## 문제
아래와 같이 5와 사칙연산만으로 12를 표현할 수 있습니다.

12 = 5 + 5 + (5 / 5) + (5 / 5)
12 = 55 / 5 + 5 / 5
12 = (55 + 5) / 5

5를 사용한 횟수는 각각 6,5,4 입니다. 그리고 이중 가장 작은 경우는 4입니다.
이처럼 숫자 N과 number가 주어질 때, N과 사칙연산만 사용해서 표현 할 수 있는 방법 중 N 사용횟수의 최솟값을 return 하도록 solution 함수를 작성하세요.

제한사항


N은 1 이상 9 이하입니다.
number는 1 이상 32,000 이하입니다.
수식에는 괄호와 사칙연산만 가능하며 나누기 연산에서 나머지는 무시합니다.
최솟값이 8보다 크면 -1을 return 합니다.


입출력 예

        
N
number
return

## 풀이
### 풀이 핵심 로직
이 문제는 주어진 숫자 N을 이용하여 사칙연산과 괄호를 통해 주어진 number를 표현하는 방법 중 N의 사용 횟수를 최소화하는 것을 목표로 합니다. 이를 위해 동적 계획법(Dynamic Programming)을 사용하여 각 단계에서 사용할 수 있는 모든 수를 집합에 저장하고, 이를 조합하여 결과를 도출합니다.

### 동작 과정
1. **초기 설정**: N이 number와 같으면 즉시 1을 반환합니다. 그 외의 경우에는 1부터 8까지의 N 사용 횟수에 대해 가능한 숫자 조합을 저장할 리스트 `dp`를 초기화합니다.

2. **N의 반복 사용**: 각 인덱스 i에 대해 `N`을 i번 이어붙인 숫자를 계산하여 `dp[i]`에 추가합니다. 예를 들어, N이 5일 경우 `dp[1]`에는 5, `dp[2]`에는 55, `dp[3]`에는 555 등을 추가합니다.

3. **조합 생성**: 각 `dp[i]`를 구성하기 위해 이전 단계의 조합을 사용합니다. 두 개의 하위 집합 `dp[j]`와 `dp[i-j]`를 선택하여 각각의 수를 a, b라고 할 때, `a + b`, `a - b`, `a * b`, `a // b`를 계산하여 `dp[i]`에 추가합니다.

4. **결과 확인**: 현재 단계에서 number가 포함되어 있는지 체크하고, 포함되어 있다면 사용한 N의 최소 횟수 i를 반환합니다.

5. **결과 반환**: 모든 조합을 시도한 후에도 number를 만들 수 없다면 -1을 반환합니다.

### 예시
- **입력**: N = 5, number = 12
- **단계**:
    - dp[1] = {5}
    - dp[2] = {55, 10}
    - dp[3] = {555, 15, 25, 50, 5, 0, ...}
    - dp[4] = {12, 20, ...} (여기서 12를 만들었기 때문에 반환: 4)

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^3)입니다. N은 최대 8이므로 실제로는 상수 시간으로 처리될 수 있습니다. 각 단계에서 가능한 조합을 모두 시도하기 때문에, 그에 따라 중첩 반복문이 사용됩니다. 

코드:
```python
def solution(N, number):
    if N == number:
        return 1
    
    dp = [set() for _ in range(9)]    
    
    for i in range(1, 9):
        dp[i].add(int(str(N) * i))  # N을 i번 붙인 수 추가
    
    for i in range(1, 9):
        for j in range(1, i):
            for a in dp[j]:
                for b in dp[i - j]:
                    dp[i].add(a + b)
                    dp[i].add(a - b)
                    dp[i].add(a * b)
                    if b != 0:
                        dp[i].add(a // b)

        if number in dp[i]:
            return i
    
    return -1
```
