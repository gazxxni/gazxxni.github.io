## 문제
다음과 같은 다각형 모양 지형에서 캐릭터가 아이템을 줍기 위해 이동하려 합니다.



지형은 각 변이 x축, y축과 평행한 직사각형이 겹쳐진 형태로 표현하며, 캐릭터는 이 다각형의 둘레(굵은 선)를 따라서 이동합니다. 

만약 직사각형을 겹친 후 다음과 같이 중앙에 빈 공간이 생기는 경우, 다각형의 가장 바깥쪽 테두리가 캐릭터의 이동 경로가 됩니다.



단, 서로 다른 두 직사각형의 x축 좌표 또는 y축 좌표가 같은 경우는 없습니다.



즉, 위 그림처럼 서로 다른 두 직사각형이 꼭짓점에서 만나거나, 변이 겹치는 경우 등은 없습니다.

다음 그림과 같이 지형이 2개 이상으로 분리된 경우도 없습니다.



한 직사각형이 다른 직사각형 안에 완전히 포함되는 경우 또한 없습니다.



지형을 나타내는 직사각형이 담긴 2차원 배열 rectangle, 초기 캐릭터의 위치 characterX, characterY, 아이템의 위치 itemX, itemY가 solution 함수의 매개변수로 주어질 때, 캐릭터가 아이템을 줍기 위해 이동해야 하는 가장 짧은 거리를 return 하도록 solution 함수를 완성해주세요.

## 풀이
### 풀이 핵심 로직
이 문제는 주어진 직사각형의 테두리를 따라 이동하는 경로를 찾는 문제입니다. 캐릭터와 아이템의 위치는 직사각형의 테두리 위에 있으며, BFS(너비 우선 탐색)를 통해 최단 경로를 찾습니다. 직사각형의 내부는 이동할 수 없으므로, 테두리 부분을 `1`로 표시하고 내부는 `0`으로 표시하여 BFS를 수행합니다.

### 동작 과정
1. 주어진 직사각형의 좌표를 2배로 확대하여 `board` 배열을 생성합니다. 이는 각 직사각형의 테두리를 `1`로 표시하고 내부는 `0`으로 표시하여 BFS에서 테두리만 탐색할 수 있도록 합니다.
2. 캐릭터의 시작 위치와 아이템의 목표 위치도 2배로 확대하여 설정합니다.
3. BFS를 수행하여 캐릭터의 위치에서 시작하여 아이템의 위치까지 이동합니다. 이때, 현재 위치에서 상하좌우로 이동하며 테두리(`1`)인 경우만 이동할 수 있도록 합니다.
4. BFS 큐에서 아이템의 위치에 도달할 경우, 이동한 거리(초기 거리의 절반)를 반환합니다.

#### 예시 입력
- `rectangle = [[1,1,7,4],[3,2,5,5],[4,3,6,9],[2,6,8,8]]`
- `characterX = 1`, `characterY = 3`
- `itemX = 7`, `itemY = 8`

**단계별 설명:**
- 캐릭터의 시작 위치는 (1, 3) → (2, 6)로 이동하여 테두리를 따라 아이템 위치 (7, 8)로 이동합니다.
- BFS를 통해 (1, 3)에서 시작하여, 테두리(1)로 이동하며 목표 위치에 도달할 때까지 이동합니다.
- 최단 거리인 17을 반환합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 `O(N^2)`입니다. 여기서 `N`은 주어진 직사각형의 최대 크기(50x50)을 고려할 때 최대 10000개의 셀을 탐색해야 할 수 있습니다. BFS는 각 셀을 한 번만 방문하므로 이 복잡도를 가집니다.

## 코드
```python
from collections import deque

def solution(rectangle, characterX, characterY, itemX, itemY):
    answer = 0
    directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]
    board = [[-1] * 102 for _ in range(102)]
    
    n = len(board)
    for r in rectangle:
        x1, y1, x2, y2 = map(lambda x: x*2, r)

        for i in range(x1, x2 + 1):
            for j in range(y1, y2 + 1):
                if x1 < i < x2 and y1 < j < y2:
                    board[i][j] = 0

                elif board[i][j] != 0:
                    board[i][j] = 1
    
    start_x, start_y = characterX * 2, characterY * 2
    target_x, target_y = itemX * 2, itemY * 2
    
    q = deque([(start_x, start_y, 0)])
    visited = set()
    visited.add((start_x, start_y))
    
    
    while q:
        cx, cy, t = q.popleft()
        
        if cx == target_x and cy == target_y:
            return t // 2

        for dx, dy in directions:
            nx = cx + dx
            ny = cy + dy
            
            if 0 <= nx < n and 0 <= ny < n and (nx, ny) not in visited and board[nx][ny] == 1:
                visited.add((nx, ny))
                q.append((nx, ny, t+1))
    
    
    
    return -1
```
