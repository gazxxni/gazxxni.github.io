## 문제
2차원 좌표 평면 위의 두 선분 L1, L2가 주어졌을 때, 두 선분이 교차하는지 아닌지 구해보자. 한 선분의 끝 점이 다른 선분이나 끝 점 위에 있는 것도 교차하는 것이다.

L1의 양 끝 점은 (x1, y1), (x2, y2), L2의 양 끝 점은 (x3, y3), (x4, y4)이다.

---

## 입력
첫째 줄에 L1의 양 끝 점 x1, y1, x2, y2가, 둘째 줄에 L2의 양 끝 점 x3, y3, x4, y4가 주어진다.

---

## 출력
L1과 L2가 교차하면 1, 아니면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 두 선분이 서로 교차하는지 판단하기 위해 'CCW'(Counter Clockwise) 기법을 사용하는 것입니다. CCW는 세 점이 이루는 방향을 판별하여 두 선분의 상대적 위치를 확인하고, 이를 통해 선분이 교차하는지, 일직선에 있는지, 또는 한 선분의 끝 점이 다른 선분 위에 있는지를 판단합니다.

### 동작 과정
1. **입력**: 두 선분의 끝 점을 입력받습니다. 예를 들어, L1의 끝 점 (1, 1)과 (3, 3), L2의 끝 점 (2, 2)와 (4, 4)라고 가정합니다.
2. **CCW 계산**: 각 선분에 대해 CCW 값을 계산합니다.
   - `o1 = ccw(1, 1, 3, 3, 2, 2)`
   - `o2 = ccw(1, 1, 3, 3, 4, 4)`
   - `o3 = ccw(2, 2, 4, 4, 1, 1)`
   - `o4 = ccw(2, 2, 4, 4, 3, 3)`
3. **교차 여부 판단**:
   - 모든 CCW 값이 0인 경우: 두 선분이 일직선에 놓여 있는 상황입니다. 이 경우, 두 선분이 겹치는지 확인하여 교차 여부를 판단합니다.
   - CCW 값 중 적어도 하나는 0이 아닌 경우: 두 선분이 서로 교차하는지 판단합니다. `o1 * o2 <= 0` 및 `o3 * o4 <= 0` 조건을 만족하면 두 선분이 교차합니다.
4. **출력**: 교차하면 1, 아니면 0을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 선분의 교차 여부를 판단하기 위해 상수 시간 내에 모든 계산을 완료할 수 있기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

def ccw(xa, ya, xb, yb, xc, yc):
    cross = (xb - xa) * (yc - ya) - (yb - ya) * (xc - xa)
    return cross

x1, y1, x2, y2 = map(int, input().split())
x3, y3, x4, y4 = map(int, input().split())

o1 = ccw(x1, y1, x2, y2, x3, y3)
o2 = ccw(x1, y1, x2, y2, x4, y4)
o3 = ccw(x3, y3, x4, y4, x1, y1)
o4 = ccw(x3, y3, x4, y4, x2, y2)

if o1 == 0 and o2 == 0 and o3 == 0 and o4 == 0:
    if (x1, y1) > (x2, y2):
        x1, x2 = x2, x1
        y1, y2 = y2, y1
    if (x3, y3) > (x4, y4):
        x3, x4 = x4, x3
        y3, y4 = y4, y3

    
    if max(min(x1,x2), min(x3,x4)) <= min(max(x1,x2), max(x3,x4)):
        if max(min(y1,y2), min(y3,y4)) <= min(max(y1,y2), max(y3,y4)):
            print(1)
            exit()
        else:
            print(0)
            exit()
    else:
        print(0)
        exit()

if o1 * o2 <= 0 and o3 * o4 <= 0:
    print(1)
    exit()
else:
    print(0)
    exit()

```
