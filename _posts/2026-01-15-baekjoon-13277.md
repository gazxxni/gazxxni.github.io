## 문제
두 정수 A와 B가 주어졌을 때, 두 수의 곱을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 A와 B가 주어진다. 두 정수는 0보다 크거나 같은 정수이며, 0을 제외한 정수는 0으로 시작하지 않으며, 수의 앞에 불필요한 0이 있는 경우도 없다. 또한, 수의 길이는 300,000자리를 넘지 않는다.

---

## 출력
두 수의 곱을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 큰 정수를 입력받아 곱셈을 수행한 후 결과를 출력하는 것입니다. Python의 `int` 타입은 임의의 정밀도를 지원하므로, 입력받은 두 수를 정수형으로 변환한 후 곱셈을 수행하고 결과를 출력하면 됩니다.

### 동작 과정
1. 첫 번째 줄에서 두 개의 정수 A와 B를 입력받습니다. 예를 들어, 입력이 `123456789012345678901234567890 987654321098765432109876543210`이라고 가정합니다.
2. `map(int, input().split())`를 사용하여 입력받은 문자열을 정수형으로 변환합니다. 여기서 A는 `123456789012345678901234567890`, B는 `987654321098765432109876543210`이 됩니다.
3. 두 수를 곱한 결과를 `a * b`를 통해 계산하고, 이를 `print` 함수를 사용하여 출력합니다.
4. 출력된 결과는 두 수의 곱으로, 예를 들어 `121932631137021795226185032733622923332236639404063348100112745464000`이 됩니다.

### 시간 복잡도
곱셈 연산의 시간 복잡도는 O(n^2)입니다. 그러나 Python의 `int` 타입은 최적화된 알고리즘을 사용하여 큰 수 연산을 처리하므로, 실제 성능은 이론적인 복잡도보다 빠를 수 있습니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a * b)
```
