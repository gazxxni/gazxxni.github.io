## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 정수 `n`에 대해 2의 `n` 제곱을 계산하는 것입니다. 이는 `n`개의 비트를 이용하여 표현할 수 있는 모든 조합의 수를 의미합니다. 즉, `n`개의 비트를 갖는 경우의 수는 2^n입니다.

### 동작 과정
1. 입력을 받습니다. 예를 들어, `n = 3`이라고 가정합니다.
2. `print(2 ** n)`을 통해 2의 3제곱을 계산합니다. 이 경우, 2 ** 3 = 8입니다.
3. 결과인 8이 출력됩니다. 이는 3개의 비트로 표현 가능한 모든 조합의 수를 나타냅니다.

예시 입력:
```
3
```
예시 출력:
```
8
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 이는 입력에 관계없이 고정된 시간 내에 결과를 계산하기 때문입니다.

---

## 코드
```python
n = int(input())

print(2 ** n)
```
