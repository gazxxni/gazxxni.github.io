## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 문자열의 서로 다른 부분 문자열을 구하기 위해, 문자열의 모든 가능한 길이의 부분 문자열을 생성하고, 이를 집합(set) 자료형에 저장합니다. 집합은 중복을 허용하지 않기 때문에, 최종적으로 집합의 크기를 세어 서로 다른 부분 문자열의 개수를 구합니다.

### 동작 과정
1. 입력 문자열 `s`가 "ababc"인 경우를 고려해봅시다.
2. 문자열의 길이(5)에 따라 길이가 1부터 5까지의 부분 문자열을 생성합니다.
   - 길이 1: `a`, `b`, `a`, `b`, `c` (중복 제거 후: `a`, `b`, `c`)
   - 길이 2: `ab`, `ba`, `ab`, `bc` (중복 제거 후: `ab`, `ba`, `bc`)
   - 길이 3: `aba`, `bab`, `abc` (중복 제거 후: `aba`, `bab`, `abc`)
   - 길이 4: `abab`, `babc` (중복 제거 후: `abab`, `babc`)
   - 길이 5: `ababc` (중복 제거 후: `ababc`)
3. 이렇게 생성된 모든 부분 문자열을 집합에 추가합니다.
4. 최종적으로 집합의 크기를 출력하면 서로 다른 부분 문자열의 개수를 알 수 있습니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 여기서 N은 문자열의 길이입니다. 이는 이중 루프를 통해 부분 문자열을 생성하는 과정에서 발생하며, 각 부분 문자열 생성 시 O(1) 시간에 추가할 수 있습니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
