## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    i = ⌊n / 2⌋;
    return A[i]; # 코드1
}

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘은 주어진 배열 A의 중간 원소를 반환하는 간단한 알고리즘으로, 항상 한 번의 연산(인덱스 접근)을 수행합니다. 따라서 수행 횟수는 항상 1이며, 이는 입력의 크기 n에 관계없이 일정합니다. 최고차항의 차수는 다항식으로 표현할 수 없으므로 0이 됩니다.

### 동작 과정
1. 입력으로 주어진 n을 읽습니다. 예를 들어, n = 5라고 가정합니다.
2. MenOfPassion 알고리즘에서 i는 ⌊n / 2⌋로 계산됩니다. 여기서 n이 5일 경우 i는 2가 됩니다.
3. 알고리즘은 A[2]를 반환하는데, 이는 중간 원소에 대한 단일 접근을 나타냅니다.
4. 수행 횟수는 1회로, 다항식의 최고차항 차수는 0입니다.

### 시간 복잡도
O(1)
