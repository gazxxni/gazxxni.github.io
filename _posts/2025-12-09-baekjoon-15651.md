## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 M개를 고른 수열
	같은 수를 여러 번 골라도 된다.

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
이 문제는 1부터 N까지의 자연수를 M개 고르는 모든 조합을 구하는 문제로, 중복을 허용하기 때문에 각 수를 여러 번 선택할 수 있습니다. 백트래킹 기법을 사용하여 현재 선택된 수열의 길이가 M에 도달할 때까지 재귀적으로 수를 추가하고, 길이가 M에 도달하면 해당 조합을 출력합니다.

### 동작 과정
다음은 입력 예시 `N = 3`, `M = 2`에 대한 단계별 설명입니다.

1. `arr` 리스트를 초기화하여 선택된 수를 저장합니다.
2. `aa()` 함수가 호출되면, 현재 `arr`의 길이가 M인지 확인합니다. 길이가 M이라면 `arr`를 출력합니다.
3. 그렇지 않으면, 1부터 N까지의 숫자(1, 2, 3)를 반복하면서 각 숫자를 `arr`에 추가합니다.
4. 숫자를 추가한 후 재귀적으로 `aa()` 함수를 호출하여 다음 숫자를 추가할 준비를 합니다.
5. 재귀 호출이 끝난 후, 마지막에 추가한 숫자를 `arr`에서 제거하여 다음 조합을 위해 상태를 복원합니다.
6. 이 과정이 반복되어 모든 가능한 조합이 출력됩니다.

예를 들어, 입력이 `3 2`일 경우, 출력은 다음과 같습니다:
```
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^M)입니다. 각 호출에서 N개의 선택지를 가지며, M번의 깊이로 재귀 호출이 이루어지기 때문입니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
