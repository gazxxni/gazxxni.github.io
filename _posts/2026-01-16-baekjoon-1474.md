## 문제
세준이는 N개의 영어 단어를 이용해 길이가 M인 새로운 단어를 만들려고 한다. 새로운 단어는 N개의 단어를 순서대로 이어 붙이고, 각 단어의 사이에 _을 넣어서 만든다. 이렇게 만든 새로운 단어의 길이가 M이 아닌 경우 _를 추가해서 길이가 M이 되게 만들어야 한다.


	_는 단어와 단어 사이에만 추가할 수 있다. 따라서, 새로운 단어는 _으로 시작하거나, _로 끝날 수 없다.
	단어와 단어 사이에 있는 _의 개수는 모두 같아야 한다.
	
		모두 같게 만드는 것이 불가능한 경우 단어와 단어 사이에 있는 _의 개수의 최댓값과 최솟값의 차이는 1이 되어야 한다.
	
	


새로운 단어 중 사전 순으로 가장 앞서는 단어를 구해보자.

---

## 입력
첫째 줄에 N과 M이 주어진다. 둘째 줄부터 N개의 줄에 단어가 한 줄에 하나씩 주어진다.

---

## 출력
첫째 줄에 사전 순으로 가장 앞서는 단어를 출력한다.

알파벳 대문자, 소문자, 밑 줄의 순서는 'A'

## 풀이

### 풀이 핵심 로직
이 문제는 N개의 단어를 `_`로 구분하여 하나의 문자열로 만드는 과정에서, 전체 길이가 M이 되도록 적절한 수의 `_`를 추가하는 문제입니다. 각 단어 사이에 추가하는 `_`의 개수는 모두 같아야 하며, 불가능한 경우에는 최댓값과 최솟값의 차이가 1이 되어야 합니다. 사전 순으로 가장 앞서는 문자열을 만들기 위해서는 가능한 한 적은 수의 `_`를 추가하고, 단어를 정렬하여 이들을 올바른 순서로 이어 붙여야 합니다.

### 동작 과정
1. **입력 처리**: N과 M을 입력받고, N개의 단어를 리스트에 저장합니다.
2. **단어 총 길이 계산**: 모든 단어의 길이를 합산하여 `total`을 구합니다.
3. **필요한 밑줄 수 계산**: 새로운 단어의 길이를 M으로 맞추기 위해 필요한 `_`의 총 개수를 계산합니다. 이 값은 `m - total`입니다.
4. **균등하게 `_` 분배**: 
   - 필요한 `_`의 수를 단어 간의 개수(N-1)로 나누어 각 간격에 들어갈 `_`의 기본 개수 `aa`를 구합니다.
   - 나머지를 `bb`로 저장하여, 남은 `_`를 사전 순을 고려하여 각 간격에 분배합니다.
5. **사전 순으로 단어 결합**: 첫 번째 단어를 저장한 후, 단어와 `_`를 차례로 결합하여 최종 결과를 만듭니다.
6. **출력**: 최종 결과 문자열을 출력합니다.

**예시 입력**:
```
3 20
hello
world
python
```

**단계별 설명**:
- N은 3, M은 20, 각 단어의 길이는 5, 5, 6으로 총 길이는 16입니다.
- 필요한 `_`의 개수는 `20 - 16 = 4`.
- `_`를 4개의 간격에 균등하게 추가하기 위해 `4 // 2 = 2`로 각 간격에 2개의 `_`를 추가하고, 남은 0개는 없으므로 균일하게 분배합니다.
- 최종적으로 "hello__world__python"의 형태로 단어들이 결합됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. N개의 단어를 읽고, 길이를 계산하며, 최종적으로 출력하는 과정이 포함되어 있습니다.

## 코드
```python
n, m = map(int, input().split())
arr = [input().rstrip() for _ in range(n)]

def greedy(a, b):
    aa = a // b
    bb = a % b

    result = [aa] * b
    for i in range(b):
        if ord(arr[i + 1][0]) >= 97 and bb > 0:
            result[i] += 1
            bb -= 1
    
    for i in range(b - 1, 0, -1):
        if result[i] == aa and bb > 0:
            result[i] += 1
            bb -= 1

    result2 = ["_" * x for x in result]
    return result2

total = 0
for i in range(n):
    total += len(arr[i])

result = greedy(m - total, n- 1)

ans = [arr.pop(0)]
for i in range(n - 1):
    ans.append(result[i])
    ans.append(arr[i])

print(*ans, sep = '')
```
