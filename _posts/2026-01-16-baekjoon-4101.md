## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
입력으로 주어진 두 정수 a와 b를 비교하여, a가 b보다 큰 경우 "Yes"를 출력하고, 그렇지 않은 경우 "No"를 출력하는 문제입니다. 입력이 "0 0"일 때까지 반복하며, 이를 통해 여러 테스트 케이스를 처리할 수 있습니다.

### 동작 과정
1. 사용자가 두 개의 정수 a와 b를 입력합니다.
   - 예시 입력: `5 3`
   - a = 5, b = 3
2. a와 b를 비교합니다. 
   - 5 > 3 이므로 "Yes"를 출력합니다.
3. 다음 테스트 케이스를 입력받습니다.
   - 예시 입력: `2 4`
   - a = 2, b = 4
4. a와 b를 비교합니다.
   - 2 > 4 이므로 "No"를 출력합니다.
5. 입력이 "0 0"이 될 때까지 이 과정을 반복합니다.

최종적으로 "0 0"이 입력되면 프로그램이 종료됩니다.

### 시간 복잡도
이 프로그램의 시간 복잡도는 O(N)입니다. 여기서 N은 입력된 테스트 케이스의 수를 의미합니다. 각 테스트 케이스마다 상수 시간 O(1)으로 비교 및 출력을 수행하기 때문입니다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
