## 문제
부산사이버대학교 학생 성택이는 엄마의 의뢰를 받아 주어진 문자열이 현관문 비밀번호에 사용 가능한지 알아내야 한다. 성택이는 공부해야 하므로 우리가 도와주자!

사용할 수 있는 비밀번호의 규칙은 다음과 같다.


	비밀번호는 6자리 이상 9자리 이하여야 한다.


예를 들어, 123124는 올바른 비밀번호이고, 1202727161은 잘못된 비밀번호이다. 문자열이 주어졌을 때 현관문 비밀번호로 사용할 수 있는지 판단하자.

---

## 입력
첫째 줄에 문자열의 총개수 N$N$이 주어진다.

둘째 줄부터 N$N$개의 줄에 걸쳐 숫자, 영어 대소문자로만 구성된 문자열이 주어진다.

---

## 출력
줄마다 사용할 수 있는 비밀번호면 yes, 그렇지 않으면 no를 출력한다.

---

## 풀이

### 풀이 핵심 로직
주어진 비밀번호 문자열의 길이를 확인하여, 비밀번호가 6자리 이상 9자리 이하인지 판단한다. 조건을 만족하면 "yes", 그렇지 않으면 "no"를 출력한다.

### 동작 과정
1. 먼저, 첫 번째 줄에서 입력된 총 문자열의 개수 N을 읽는다.
2. 이어서 N개의 문자열을 한 줄씩 읽으면서, 각 문자열의 길이를 계산한다.
3. 문자열의 길이가 6 이상 9 이하인 경우 "yes"를 출력하고, 그렇지 않은 경우 "no"를 출력한다.

**예시 입력:**
```
3
123456
abcdefg
1234567890
```

**단계별 설명:**
- 첫 번째 문자열 "123456"의 길이는 6이므로 "yes" 출력.
- 두 번째 문자열 "abcdefg"의 길이는 7이므로 "yes" 출력.
- 세 번째 문자열 "1234567890"의 길이는 10이므로 "no" 출력.

**출력:**
```
yes
yes
no
```

### 시간 복잡도
O(N) - N개의 문자열을 각각 검사하는 과정이므로, 입력된 문자열 개수에 비례하는 선형 시간 복잡도를 가진다.

---

## 코드
```python
for _ in range(int(input())):
    password = len(input())
    print("yes") if 6 <= password and password <= 9 else print("no")

```
