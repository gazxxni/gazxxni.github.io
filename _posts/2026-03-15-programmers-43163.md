## 문제
두 개의 단어 begin, target과 단어의 집합 words가 있습니다. 아래와 같은 규칙을 이용하여 begin에서 target으로 변환하는 가장 짧은 변환 과정을 찾으려고 합니다.
1. 한 번에 한 개의 알파벳만 바꿀 수 있습니다.
2. words에 있는 단어로만 변환할 수 있습니다.

예를 들어 begin이 "hit", target가 "cog", words가 ["hot","dot","dog","lot","log","cog"]라면 "hit" -&gt; "hot" -&gt; "dot" -&gt; "dog" -&gt; "cog"와 같이 4단계를 거쳐 변환할 수 있습니다.

두 개의 단어 begin, target과 단어의 집합 words가 매개변수로 주어질 때, 최소 몇 단계의 과정을 거쳐 begin을 target으로 변환할 수 있는지 return 하도록 solution 함수를 작성해주세요.

제한사항


각 단어는 알파벳 소문자로만 이루어져 있습니다.
각 단어의 길이는 3 이상 10 이하이며 모든 단어의 길이는 같습니다.
words에는 3개 이상 50개 이하의 단어가 있으며 중복되는 단어는 없습니다.
begin과 target은 같지 않습니다.
변환할 수 없는 경우에는 0를 return 합니다.

## 풀이
### 풀이 핵심 로직
이 문제는 BFS(너비 우선 탐색)를 사용하여 시작 단어에서 목표 단어로 변환하는 최단 경로를 찾는 문제입니다. BFS는 그래프의 최단 경로를 찾기에 적합하며, 주어진 단어 목록에서 한 번에 한 글자만 바꾸어 가능한 단어를 탐색합니다.

### 동작 과정
1. **초기화**: BFS를 위한 큐를 초기화하고, 방문한 단어를 기록할 `visited` 세트를 만듭니다. 큐에는 `(변환 단계, 현재 단어)` 형태로 시작 단어와 함께 0을 넣습니다.
2. **BFS 반복**:
   - 큐에서 첫 번째 요소를 꺼내 현재 단어와 변환 단계를 확인합니다.
   - 현재 단어가 목표 단어와 같다면 그 단계 수를 반환합니다.
   - 현재 단어와 한 글자만 다른 단어를 찾아서, 아직 방문하지 않은 단어라면 큐에 추가하고 방문 목록에 등록합니다.
3. **단어 변환**: 단어 목록을 순회하며, 현재 단어와 비교하여 한 글자만 다른 단어를 찾습니다.
4. **종료 조건**: 큐가 비어버리면 목표 단어에 도달할 수 없다는 뜻이므로 0을 반환합니다.

**예시 입력**:
- `begin`: "hit"
- `target`: "cog"
- `words`: ["hot", "dot", "dog", "lot", "log", "cog"]

**단계별 설명**:
1. 시작: 큐에 (0, "hit") 추가.
2. "hit"에서 "hot"으로 변환 (1단계). 큐에 (1, "hot") 추가.
3. "hot"에서 "dot" 및 "lot"으로 변환 (2단계). 큐에 (2, "dot"), (2, "lot") 추가.
4. "dot"에서 "dog"으로 변환 (3단계). 큐에 (3, "dog") 추가.
5. "dog"에서 "cog"으로 변환 (4단계). 큐에 (4, "cog") 추가.
6. "cog"에 도달하므로 4를 반환.

### 시간 복잡도
- 최악의 경우 모든 단어를 탐색해야 하므로, 시간 복잡도는 O(N^2)입니다. 여기서 N은 단어 목록의 크기입니다. 각 단어는 최대 10글자이므로, 비교하는 데 O(1) 시간이 걸립니다.

## 코드
```python
from collections import deque

def solution(begin, target, words):
    def bfs(t, start):
        q = deque([(t, start)])
        visited = set()
        visited.add(start)
        
        while q:
            cur_t, cur_word = q.popleft()
            
            if cur_word == target:
                return cur_t
            
            for word in words:
                diff = [a for a, b in zip(cur_word, word) if a != b]
                if len(diff) == 1 and word not in visited:
                    visited.add(word)
                    q.append((cur_t+1, word))
        
        return 0
    
    return bfs(0, begin) 


```
