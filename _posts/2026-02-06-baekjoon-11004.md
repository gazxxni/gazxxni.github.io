## 문제
수 N개 A1, A2, ..., AN이 주어진다. A를 오름차순 정렬했을 때, 앞에서부터 K번째 있는 수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 5,000,000)과 K (1 ≤ K ≤ N)이 주어진다.

둘째에는 A1, A2, ..., AN이 주어진다. (-109 ≤ Ai ≤ 109)

---

## 출력
A를 정렬했을 때, 앞에서부터 K번째 있는 수를 출력한다.

---

## 풀이

### 풀이 핵심 로직
주어진 N개의 수를 오름차순으로 정렬한 뒤, K번째 수를 출력하는 문제이다. Python의 내장 `sort()` 함수를 사용하면 효율적으로 정렬을 수행할 수 있으며, 정렬된 리스트에서 K번째 원소는 인덱스 K-1에 위치하므로 이를 출력하면 된다.

### 동작 과정
1. **입력 받기**: 첫 줄에서 N과 K를 입력받고, 둘째 줄에서 N개의 수를 리스트 형태로 입력받는다.
   - 예시 입력: 
     ```
     5 2
     3 1 2 5 4
     ```
2. **정렬**: 입력받은 리스트를 오름차순으로 정렬한다.
   - 정렬된 리스트: `[1, 2, 3, 4, 5]`
3. **K번째 수 출력**: 정렬된 리스트에서 K번째 수를 찾아 출력한다. K는 2이므로 인덱스 1의 수를 출력한다.
   - 출력: `2`

### 시간 복잡도
정렬의 시간 복잡도는 O(N log N)이다. 주어진 N의 범위가 최대 5,000,000이므로, 이 방법은 실질적으로 수행 가능한 범위 내에 있다.

---

## 코드
```python
n, k = map(int, input().split())
arr = list(map(int, input().split()))
arr.sort()
print(arr[k-1])
```
