## 문제
배열 array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수를 구하려 합니다.

예를 들어 array가 [1, 5, 2, 6, 3, 7, 4], i = 2, j = 5, k = 3이라면


array의 2번째부터 5번째까지 자르면 [5, 2, 6, 3]입니다.
1에서 나온 배열을 정렬하면 [2, 3, 5, 6]입니다.
2에서 나온 배열의 3번째 숫자는 5입니다.


배열 array, [i, j, k]를 원소로 가진 2차원 배열 commands가 매개변수로 주어질 때, commands의 모든 원소에 대해 앞서 설명한 연산을 적용했을 때 나온 결과를 배열에 담아 return 하도록 solution 함수를 작성해주세요.

제한사항


array의 길이는 1 이상 100 이하입니다.
array의 각 원소는 1 이상 100 이하입니다.
commands의 길이는 1 이상 50 이하입니다.
commands의 각 원소는 길이가 3입니다.


입출력 예

        
array
commands
return


        
[1, 5, 2, 6, 3, 7, 4]
[[2, 5, 3], [4, 4, 1], [1, 7, 3]]
[5, 6, 3]


      
입출력 예 설명

[1, 5, 2, 6, 3, 7, 4]를 2번째부터 5번째까지 자른 후 정렬합니다. [2, 3, 5, 6]의 세 번째 숫자는 5입니다.
[1, 5, 2, 6, 3, 7, 4]를 4번째부터 4번째까지 자른 후 정렬합니다. [6]의 첫 번째 숫자는 6입니다.
[1, 5, 2, 6, 3, 7, 4]를 1번째부터 7번째까지 자릅니다. [1, 2, 3, 4, 5, 6, 7]의 세 번째 숫자는 3입니다.

## 풀이
### 풀이 핵심 로직
주어진 배열에서 각 명령어에 대해 특정 범위를 자르고, 자른 배열을 정렬한 후, k번째 요소를 찾아 반환하는 방식으로 문제를 해결합니다. 이를 위해 각 명령어를 순회하며 배열의 슬라이스를 사용하고, 정렬을 적용하여 필요한 값을 추출합니다.

### 동작 과정
1. **입력 배열과 명령어 배열을 준비합니다.** 예를 들어, `array`는 `[1, 5, 2, 6, 3, 7, 4]`이고, `commands`는 `[[2, 5, 3], [4, 4, 1], [1, 7, 3]]`입니다.
2. **각 명령어를 순회합니다.** 명령어는 `[i, j, k]`의 형태로, 배열의 `i`번째부터 `j`번째까지의 부분 배열을 정의합니다.
3. **부분 배열을 자릅니다.** `array[i-1:j]`를 사용하여 인덱스를 조정한 후, 부분 배열을 생성합니다.
   - 첫 번째 명령어 `[2, 5, 3]`의 경우, `array[1:5]`는 `[5, 2, 6, 3]`입니다.
4. **부분 배열을 정렬합니다.** 생성한 부분 배열에 대해 `sort()`를 호출합니다. 
   - 정렬된 결과는 `[2, 3, 5, 6]`입니다.
5. **k번째 요소를 추출합니다.** 정렬된 배열에서 `k-1` 인덱스를 사용하여 원하는 값을 찾습니다. 
   - `k=3`이므로 인덱스 `2`의 값은 `5`입니다.
6. **모든 명령어에 대해 위 과정을 반복하여 결과를 리스트에 추가합니다.**
7. **최종 결과 리스트를 반환합니다.** 

위 과정을 거쳐 최종적으로 `[5, 6, 3]`이 반환됩니다.

### 시간 복잡도
- 각 명령어에 대해 부분 배열을 자르고 정렬하는 과정이 포함됩니다. 부분 배열의 최대 크기는 100이므로 정렬하는 데 최악의 경우 O(N log N)의 시간이 걸립니다. 
- 명령어의 수가 최대 50개이므로 전체 시간 복잡도는 O(M * N log N)입니다. 여기서 M은 명령어의 수, N은 배열의 최대 길이입니다.

## 코드
```python
def solution(array, commands):
    answer = []
    for row in commands:
        arr = array[row[0] - 1:row[1]]
        arr.sort()
        answer.append(arr[row[2] - 1])
    
    return answer
```
