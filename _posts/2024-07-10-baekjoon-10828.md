## 풀이

### 풀이 핵심 로직
문제는 스택 자료구조를 구현하는 것으로, 주어진 명령어에 따라 스택의 동작을 수행합니다. 주어진 명령어는 `push`, `pop`, `size`, `empty`, `top`이 있으며, 각각의 명령어에 대해 스택의 상태를 조작하거나 정보를 출력합니다.

### 동작 과정
1. 먼저, 입력받은 정수 `n`만큼 반복하며 각 명령어를 처리합니다.
2. 각 명령어는 다음과 같은 기능을 수행합니다:
   - `push x`: 스택에 x를 추가합니다.
   - `pop`: 스택에서 가장 위의 요소를 제거하고 그 값을 출력합니다. 스택이 비어있으면 -1을 출력합니다.
   - `size`: 스택에 현재 저장된 요소의 개수를 출력합니다.
   - `empty`: 스택이 비어있으면 1, 그렇지 않으면 0을 출력합니다.
   - `top`: 스택의 가장 위에 있는 요소를 출력합니다. 스택이 비어있으면 -1을 출력합니다.

예시 입력:
```
5
push 1
push 2
top
pop
size
```

단계별 설명:
- 첫 번째 명령어 `push 1`이 실행되어 스택은 `[1]`이 됩니다.
- 두 번째 명령어 `push 2`가 실행되어 스택은 `[1, 2]`가 됩니다.
- 세 번째 명령어 `top`이 실행되어 스택의 최상단 요소인 `2`가 출력됩니다.
- 네 번째 명령어 `pop`이 실행되어 스택에서 `2`가 제거되고 출력됩니다. 현재 스택은 `[1]`입니다.
- 마지막 명령어 `size`가 실행되어 스택의 크기인 `1`이 출력됩니다.

### 시간 복잡도
모든 명령어는 O(1) 시간 복잡도로 수행됩니다. 따라서 전체 알고리즘의 시간 복잡도는 O(n)입니다.

## 코드
```python
import sys

n = int(sys.stdin.readline())

stack=[]
for i in range(n):
    command = sys.stdin.readline().split()

    if command[0]=='push':
        stack.append(command[1])
    elif command[0]=='pop':
        if len(stack)==0:
            print(-1)
        else:
            print(stack.pop())
    elif command[0] == 'size':
        print(len(stack))
    elif command[0] == 'empty':
        if len(stack)==0:
            print(1)
        else:
            print(0)
    elif command[0] == 'top':
        if len(stack)==0:
            print(-1)
        else:
            print(stack[-1])
```
