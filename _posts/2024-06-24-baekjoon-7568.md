## 풀이

### 풀이 핵심 로직
이 문제는 주어진 사람들의 몸무게와 키를 기준으로 각 사람의 "등수"를 계산하는 문제입니다. 각 사람의 높이와 몸무게를 비교하여 더 높은 키와 몸무게를 가진 사람이 있으면 해당 사람의 등수를 증가시킵니다. 최종적으로 각 사람의 등수를 출력합니다.

### 동작 과정
1. **입력**: 첫 줄에 사람의 수 `n`을 입력받고, 이후 `n`개의 줄에서 각각의 사람의 키와 몸무게를 입력받아 리스트 `a`에 튜플 형식으로 저장합니다.
   - 예시 입력:
     ```
     5
     55 185
     58 183
     88 186
     60 175
     46 155
     ```
   - 리스트 `a`는 다음과 같이 구성됩니다:
     ```
     a = [(55, 185), (58, 183), (88, 186), (60, 175), (46, 155)]
     ```

2. **등수 계산**: 각 사람 `i`에 대하여 이중 루프를 사용해 다른 모든 사람 `j`와 비교합니다. 만약 `i`의 키와 몸무게가 모두 `j`보다 작으면 `i`의 등수를 1 증가시킵니다.
   - 예를 들어, 첫 번째 사람 `(55, 185)`를 기준으로 `j`를 비교하면:
     - `(58, 183)`, `(88, 186)`, `(60, 175)`는 키와 몸무게가 모두 더 크므로 등수가 3으로 증가합니다.
     - `(46, 155)`는 더 작으므로 등수는 증가하지 않습니다.
   - 최종적으로 `(55, 185)`의 등수는 3이 됩니다.

3. **출력**: 모든 사람에 대해 계산이 완료되면 각 사람의 등수를 공백으로 구분하여 출력합니다.
   - 최종 출력: `2 1 1 3 5`

### 시간 복잡도
이 알고리즘은 두 개의 중첩 루프를 사용하므로 시간 복잡도는 O(n^2)입니다. 여기서 `n`은 사람의 수를 의미합니다.

---

## 코드
```python
import sys

n=int(input())
a=[]

for _ in range(n):
    x,y=map(int,input().split())
    a.append((x,y))

for i in a:
    rank=1
    for j in a:
        if i[0]<j[0] and i[1]<j[1]:
            rank+=1
    print(rank,end=' ')
```
