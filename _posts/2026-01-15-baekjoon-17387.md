## 문제
2차원 좌표 평면 위의 두 선분 L1, L2가 주어졌을 때, 두 선분이 교차하는지 아닌지 구해보자. 한 선분의 끝 점이 다른 선분이나 끝 점 위에 있는 것도 교차하는 것이다.

L1의 양 끝 점은 (x1, y1), (x2, y2), L2의 양 끝 점은 (x3, y3), (x4, y4)이다.

---

## 입력
첫째 줄에 L1의 양 끝 점 x1, y1, x2, y2가, 둘째 줄에 L2의 양 끝 점 x3, y3, x4, y4가 주어진다.

---

## 출력
L1과 L2가 교차하면 1, 아니면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
선분 L1과 L2의 교차 여부를 판단하기 위해, 각 선분의 끝점들이 상대적으로 어느 방향에 위치하는지를 나타내는 CCW(Counter Clockwise) 함수를 사용합니다. 이 함수는 세 점의 위치에 따라 0(일직선), 양수(반시계 방향), 음수(시계 방향)로 값을 반환합니다. 이를 통해 두 선분이 서로 교차하는지, 혹은 한 선분이 다른 선분의 끝점 위에 있는지를 확인합니다.

### 동작 과정
1. **입력:** L1의 끝점 (x1, y1), (x2, y2) 및 L2의 끝점 (x3, y3), (x4, y4)를 입력받습니다.
2. **CCW 계산:** 각 선분에 대해 CCW 함수를 호출하여 o1, o2, o3, o4 값을 계산합니다.
   - o1 = CCW(L1의 첫 점, L1의 두 번째 점, L2의 첫 점)
   - o2 = CCW(L1의 첫 점, L1의 두 번째 점, L2의 두 번째 점)
   - o3 = CCW(L2의 첫 점, L2의 두 번째 점, L1의 첫 점)
   - o4 = CCW(L2의 첫 점, L2의 두 번째 점, L1의 두 번째 점)
3. **공통선분 확인:** 만약 o1, o2, o3, o4가 모두 0이라면 두 선분은 일직선상에 있습니다. 이 경우 선분의 겹침 여부를 확인합니다.
   - x좌표와 y좌표의 범위를 비교하여 두 선분이 겹치는지를 판단합니다.
4. **교차 여부 판별:** o1과 o2의 곱, o3과 o4의 곱이 모두 0 이하라면 두 선분이 교차합니다. 그렇지 않으면 교차하지 않습니다.

**예시 입력:**
```
0 0 2 2
1 1 3 3
```
- L1은 (0, 0)에서 (2, 2)로, L2는 (1, 1)에서 (3, 3)로 주어집니다.
- CCW 함수에 각 점을 대입하여 o1, o2, o3, o4를 계산하고, 모두 0이므로 두 선분은 일직선에 있습니다.
- 겹치는 범위를 확인하여 두 선분이 교차한다고 출력합니다. 출력: `1`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력된 점의 수가 고정되어 있으므로, CCW 함수 호출과 비교 연산이 상수 시간에 수행됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

def ccw(xa, ya, xb, yb, xc, yc):
    cross = (xb - xa) * (yc - ya) - (yb - ya) * (xc - xa)
    return cross

x1, y1, x2, y2 = map(int, input().split())
x3, y3, x4, y4 = map(int, input().split())

o1 = ccw(x1, y1, x2, y2, x3, y3)
o2 = ccw(x1, y1, x2, y2, x4, y4)
o3 = ccw(x3, y3, x4, y4, x1, y1)
o4 = ccw(x3, y3, x4, y4, x2, y2)

if o1 == 0 and o2 == 0 and o3 == 0 and o4 == 0:
    if (x1, y1) > (x2, y2):
        x1, x2 = x2, x1
        y1, y2 = y2, y1
    if (x3, y3) > (x4, y4):
        x3, x4 = x4, x3
        y3, y4 = y4, y3

    
    if max(min(x1,x2), min(x3,x4)) <= min(max(x1,x2), max(x3,x4)):
        if max(min(y1,y2), min(y3,y4)) <= min(max(y1,y2), max(y3,y4)):
            print(1)
            exit()
        else:
            print(0)
            exit()
    else:
        print(0)
        exit()

if o1 * o2 <= 0 and o3 * o4 <= 0:
    print(1)
    exit()
else:
    print(0)
    exit()

```
