## 문제
체스판은 8×8크기이고, 검정 칸과 하얀 칸이 번갈아가면서 색칠되어 있다. 가장 왼쪽 위칸 (0,0)은 하얀색이다. 체스판의 상태가 주어졌을 때, 하얀 칸 위에 말이 몇 개 있는지 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄부터 8개의 줄에 체스판의 상태가 주어진다. ‘.’은 빈 칸이고, ‘F’는 위에 말이 있는 칸이다.

---

## 출력
첫째 줄에 문제의 정답을 출력한다.

## 성능 요약

## 풀이

### 풀이 핵심 로직
체스판은 8x8 크기로, 가장 왼쪽 위 칸(0,0)은 하얀색입니다. 하얀 칸의 위치는 (i, j)의 인덱스가 모두 짝수이거나 모두 홀수일 때입니다. 입력받은 체스판에서 하얀 칸 위에 존재하는 'F'의 개수를 세어 출력합니다.

### 동작 과정
1. 입력으로 8개의 줄을 받아 체스판의 상태를 리스트로 저장합니다.
2. 이중 for 루프를 통해 모든 칸을 확인합니다:
   - 첫 번째 루프에서 0부터 7까지의 행 인덱스(i)를 순회합니다.
   - 두 번째 루프에서 0부터 7까지의 열 인덱스(j)를 순회합니다.
3. 각각의 칸이 하얀 칸인지 확인합니다:
   - (i % 2 == 0 and j % 2 == 0) 또는 (i % 2 == 1 and j % 2 == 1) 조건을 만족하면 하얀 칸입니다.
4. 해당 칸이 'F'인지 확인하고, 맞다면 카운트를 증가시킵니다.
5. 모든 칸을 확인한 후 최종 카운트를 출력합니다.

예를 들어, 다음과 같은 입력이 주어진다면:
```
F.F.F.F.F.
.F.F.F.F.F
F.F.F.F.F.
.F.F.F.F.F
F.F.F.F.F.
.F.F.F.F.F
F.F.F.F.F.
.F.F.F.F.F
```
위 코드는 하얀 칸 위에 총 32개의 'F'가 있으므로 출력은 `32`가 됩니다.

### 시간 복잡도
O(1) - 체스판의 크기가 고정되어 있으므로, 항상 64개의 칸을 검사하는 시간이 일정합니다.

---

## 코드
```python
graph = [input().rstrip() for _ in range(8)]

cnt = 0
for i in range(8):
    for j in range(8):
        if (i % 2 == 0 and j % 2 == 0) or (i % 2 == 1 and j % 2 == 1):
            if graph[i][j] == 'F':
                cnt += 1
                
print(cnt)
```
