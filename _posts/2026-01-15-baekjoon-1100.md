## 문제
체스판은 8×8크기이고, 검정 칸과 하얀 칸이 번갈아가면서 색칠되어 있다. 가장 왼쪽 위칸 (0,0)은 하얀색이다. 체스판의 상태가 주어졌을 때, 하얀 칸 위에 말이 몇 개 있는지 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄부터 8개의 줄에 체스판의 상태가 주어진다. ‘.’은 빈 칸이고, ‘F’는 위에 말이 있는 칸이다.

---

## 출력
첫째 줄에 문제의 정답을 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

---

## 풀이

### 풀이 핵심 로직
체스판의 각 칸의 색깔은 좌표의 인덱스에 따라 결정된다. 짝수 인덱스의 합이 짝수일 때(즉, (i + j) % 2 == 0) 해당 칸은 하얀색이다. 주어진 체스판에서 'F'가 있는 하얀 칸의 개수를 세기 위해, 모든 칸을 탐색하면서 이 조건을 만족하는지 확인하고, 'F'가 있는 경우 카운트를 증가시킨다.

### 동작 과정
1. 입력으로 8개의 문자열을 받아 체스판의 상태를 나타내는 8x8 배열을 만든다.
2. 2중 for문을 사용하여 각 칸의 좌표 (i, j)를 탐색한다.
3. (i + j) % 2 == 0을 통해 해당 칸이 하얀색인지 확인한다.
4. 하얀색 칸에 'F'가 있는 경우 카운트를 1 증가시킨다.
5. 모든 칸을 탐색한 후 최종 카운트를 출력한다.

**예시 입력:**
```
........
.F...... 
........
..F..... 
........
F....... 
........
........
```
- 첫 번째 줄은 모두 빈 칸이므로 카운트에 영향을 주지 않는다.
- 두 번째 줄에서 (1, 1) 위치에 'F'가 있으므로 카운트 1 증가.
- 네 번째 줄에서 (3, 2) 위치에 'F'가 있으므로 카운트 2 증가.
- 여섯 번째 줄에서 (5, 0) 위치에 'F'가 있으므로 카운트 3 증가.
- 그 외의 줄은 하얀색 칸에 'F'가 없음.

**최종 카운트:** 3

### 시간 복잡도
O(1) - 체스판의 크기가 고정되어 있으므로 항상 64개의 칸을 탐색하므로 시간 복잡도는 상수 시간이다.

---

## 코드
```python
graph = [input().rstrip() for _ in range(8)]

cnt = 0
for i in range(8):
    for j in range(8):
        if (i % 2 == 0 and j % 2 == 0) or (i % 2 == 1 and j % 2 == 1):
            if graph[i][j] == 'F':
                cnt += 1
                
print(cnt)
```
