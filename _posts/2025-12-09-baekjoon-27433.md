## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 재귀를 사용하여 N! (N 팩토리얼)을 계산하는 방식으로 해결할 수 있습니다. 팩토리얼은 N이 0 또는 1일 때 1이고, 그 외의 경우 N!은 N * (N-1)!로 정의됩니다. 이를 재귀 호출을 통해 구현합니다.

### 동작 과정
1. 입력으로 N이 주어집니다. 예를 들어, N = 5라고 가정합시다.
2. `factorial` 함수가 호출되면서 n=5로 시작합니다.
3. n이 1보다 크므로, `factorial(4) * 5`를 계산하기 위해 `factorial(4)`를 호출합니다.
4. 이 과정은 n이 1에 도달할 때까지 반복되며, n=1일 경우 1을 반환합니다.
5. 이후 각 호출이 종료되면서 반환되는 값들을 이용해 계산이 이루어집니다. 즉, `factorial(2)`는 2, `factorial(3)`은 6, `factorial(4)`는 24, `factorial(5)`는 120이 됩니다.
6. 최종적으로 5! = 120이 출력됩니다.

### 시간 복잡도
이 재귀 함수의 시간 복잡도는 O(N)입니다. 각 n에 대해 한 번씩 호출되므로 N에 비례하여 실행 시간이 증가합니다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
