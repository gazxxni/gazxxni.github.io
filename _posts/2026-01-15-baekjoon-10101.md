## 문제
창영이는 삼각형의 종류를 잘 구분하지 못한다. 따라서 프로그램을 이용해 이를 외우려고 한다.

삼각형의 세 각을 입력받은 다음, 


	세 각의 크기가 모두 60이면, Equilateral
	세 각의 합이 180이고, 두 각이 같은 경우에는 Isosceles
	세 각의 합이 180이고, 같은 각이 없는 경우에는 Scalene
	세 각의 합이 180이 아닌 경우에는 Error


를 출력하는 프로그램을 작성하시오.

---

## 입력
총 3개의 줄에 걸쳐 삼각형의 각의 크기가 주어진다. 모든 정수는 0보다 크고, 180보다 작다.

---

## 출력
문제의 설명에 따라 Equilateral, Isosceles, Scalene, Error 중 하나를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심은 삼각형의 내각의 합과 각의 크기를 통해 삼각형의 종류를 구분하는 것입니다. 주어진 세 각이 모두 60도이면 정삼각형(Equilateral)이고, 각의 합이 180도이며 두 각이 같은 경우에는 이등변삼각형(Isosceles), 모든 각이 다르면 삼각형(Skalene), 그렇지 않으면 오류(Error)를 출력합니다.

### 동작 과정
1. 사용자로부터 세 개의 각 `a`, `b`, `c`를 입력받습니다.
   - 예시 입력: 
     ```
     60
     60
     60
     ```

2. 먼저, 세 각이 모두 60인지 확인합니다.
   - `a == b == c == 60`인 경우, 출력: `Equilateral`

3. 다음으로, 각의 합이 180인지 확인합니다.
   - `a + b + c == 180`인 경우, 두 각이 같은지를 확인합니다.
     - `a == b` 및 `b != c`인 경우, 출력: `Isosceles`
     - `a == c` 및 `b != c`인 경우, 출력: `Isosceles`
     - `b == c` 및 `a != c`인 경우, 출력: `Isosceles`
     - 모든 각이 다르면, 출력: `Scalene`

4. 만약 위 조건을 모두 만족하지 않으면, 출력: `Error`

   - 예시 입력:
     ```
     60
     70
     50
     ```
     - 각의 합이 180이므로, 각의 크기가 모두 다르기 때문에 `Scalene`을 출력합니다.

   - 예시 입력:
     ```
     30
     30
     30
     ```
     - 각의 합이 90이므로 `Error`를 출력합니다.

### 시간 복잡도
이 알고리즘은 입력의 크기에 비례하지 않고, 상수 시간에 수행되므로 시간 복잡도는 O(1)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
c = int(input())


if a == b == c == 60:
    print('Equilateral')

elif a + b + c == 180:
    if a == b and b != c:
        print('Isosceles')
    elif a == c and b != c:
        print('Isosceles')
    elif b == c and a != c:
        print('Isosceles')
    else:
        print('Scalene')

else:
    print('Error')
```
