## 문제
당신은 N개의 조약돌을 가지고 있다. 이 조약돌을 좌표평면의 격자점 위에 아무렇게나 떨어뜨렸다. 격자점이란, x좌표와 y좌표 모두가 정수인 지점을 말한다. 이를테면 (1, 1)이나 (0, -9)는 격자점이며, (-2, 3.5)이나 (π, 7.14)는 격자점이 아니다.

모든 조약돌을 포함하는 가장 작은 직사각형을 생각할 수 있다. 예를 들어 세 개의 조약돌을 (2,4), (4, 8), (5,5)에 떨어뜨렸다면, 이 세 조약돌을 모두 포함하는 가장 작은 직사각형은 가로 3, 세로 4인 직사각형이다. 이 경우 직사각형의 둘레는 14가 된다. 직사각형의 가로와 세로 길이는 반드시 1 이상이어야 한다.

조약돌의 개수 N이 주어졌을 때, 조약돌을 좌표평면의 격자점에 적절히 떨어뜨려서 모든 조약돌을 포함하는 직사각형의 둘레를 최소로 하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 조약돌의 개수 N(1 ≤ n ≤ 500,000,000)이 주어진다.

---

## 출력
첫째 줄에 최소 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 조약돌의 개수 N을 포함하는 직사각형의 최소 둘레를 구하기 위해, 조약돌을 가로(i)와 세로(j)로 나누어 배치하는 것입니다. 최소 둘레를 만들기 위해 가로와 세로의 길이를 가능한 한 비슷하게 맞추는 것이 중요하며, 이를 위해 i를 N의 제곱근에 가깝게 설정하고 j는 N을 i로 나눈 몫으로 계산합니다.

### 동작 과정
1. 입력으로 조약돌의 개수 N을 받습니다.
2. 가로 길이(i)를 1부터 시작하여 N보다 작은 i의 제곱을 찾습니다. 즉, \(i * i < N\) 조건을 만족하는 최대의 i를 찾습니다.
3. 세로 길이(j)를 \(j = \lceil \frac{N}{i} \rceil\)로 계산합니다.
4. 조약돌의 개수가 1에서 4 사이인 경우, 둘레는 항상 4로 고정됩니다. 나머지 경우, 둘레는 \(2 \times (i + j - 2)\)로 계산됩니다.
5. 결과를 출력합니다.

예를 들어, N이 10인 경우:
- i는 4로 설정됩니다 (4 * 4 < 10).
- j는 \(j = \lceil \frac{10}{4} \rceil = 3\)으로 계산됩니다.
- 최종적으로 둘레는 \(2 \times (4 + 3 - 2) = 2 \times 5 = 10\)이 됩니다.

### 시간 복잡도
O(√N) - i의 값을 찾기 위해 최대 N의 제곱근까지 반복하므로 시간 복잡도는 제곱근에 비례합니다.

## 코드
```python
import sys, math
input = sys.stdin.readline

n = int(input())
i = 1

while i * i < n:
    i += 1

j = math.ceil(n / i)

if 1 <= n <= 4:
    print(4)
else:
    print(2 * (i + j - 2))
```
