## 문제 링크
[https://www.acmicpc.net/problem/1251](https://www.acmicpc.net/problem/1251)

---

## 풀이

### 풀이 핵심 로직
문자열을 세 부분으로 나누고 각 부분을 뒤집어 새로운 문자열을 생성하는 방법을 사용한다. 가능한 모든 나누기 조합을 통해 생성된 문자열들을 정렬한 후, 가장 알파벳 순서가 앞서는 문자열을 찾는다.

### 동작 과정
1. 입력으로 주어진 문자열을 `arr` 리스트에 저장한다.
2. 첫 번째 나누는 지점 `i`를 1부터 `len(arr) - 1`까지 반복한다.
3. 두 번째 나누는 지점 `j`를 `i + 1`부터 `len(arr)`까지 반복한다.
4. 각 반복마다 문자열을 세 부분으로 나눈다:
   - `a`: 문자열의 처음부터 `i`번째까지
   - `b`: `i`번째부터 `j`번째까지
   - `c`: `j`번째부터 끝까지
5. 각 부분 `a`, `b`, `c`를 뒤집고, 이를 합쳐 새로운 문자열을 생성하여 `tmp` 리스트에 추가한다.
6. `tmp` 리스트에 있는 모든 문자열을 정렬한 후, 가장 첫 번째 문자열(가장 알파벳 순서가 앞서는 문자열)을 출력한다.

#### 예시 입력: `abcde`
- `i=1, j=2`: `a = ['a'], b = ['b'], c = ['c', 'd', 'e']` -> `['a', 'b', 'e', 'd', 'c']`
- `i=1, j=3`: `a = ['a'], b = ['b', 'c'], c = ['d', 'e']` -> `['a', 'c', 'b', 'e', 'd']`
- `i=1, j=4`: `a = ['a'], b = ['b', 'c', 'd'], c = ['e']` -> `['a', 'd', 'c', 'b', 'e']`
- `i=2, j=3`: `a = ['a', 'b'], b = ['c'], c = ['d', 'e']` -> `['b', 'a', 'c', 'e', 'd']`
- `i=2, j=4`: `a = ['a', 'b'], b = ['c', 'd'], c = ['e']` -> `['b', 'a', 'd', 'c', 'e']`
- `i=3, j=4`: `a = ['a', 'b', 'c'], b = ['d'], c = ['e']` -> `['c', 'b', 'a', 'd', 'e']`
- 최종적으로 `tmp` 리스트에는 여러 개의 문자열이 저장되고, 이들을 정렬하여 가장 앞서는 문자열을 출력.

### 시간 복잡도
O(N^3)
- 이 알고리즘은 문자열 길이 `N`에 대해 이중 반복문을 사용하므로, 문자열을 나누고 뒤집는 과정에서 O(N) 시간이 추가로 소요된다. 최종적으로 O(N^3)의 시간 복잡도를 가진다.

## 코드
```python
import sys

arr = list(input())
ans = []
tmp = []

# 첫 번째 나누는 지점 i를 1부터 len(arr) - 1까지 반복
for i in range(1, len(arr) - 1):
    # 두 번째 나누는 지점 j를 i+1부터 len(arr)까지 반복
    for j in range(i + 1, len(arr)):
        # 문자열을 세 부분으로 나누기
        a = arr[:i]   
        b = arr[i:j] 
        c = arr[j:] 

        a.reverse()
        b.reverse()
        c.reverse()

        tmp.append(a + b + c)

# tmp에 있는 각 리스트 요소를 문자열로 변환하여 ans 리스트에 추가
for i in tmp:
    ans.append(''.join(i))

print(sorted(ans)[0])

```
