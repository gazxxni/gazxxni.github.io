## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 입력된 세 개의 문자열 중에서 'Fizz', 'Buzz', 'FizzBuzz'가 아닌 숫자를 찾아 그 값을 기반으로 주어진 규칙에 따라 결과를 출력하는 것입니다. 각 입력에 대해 3부터 1까지의 가중치를 더하여 최종 값을 계산하고, 이 값이 3과 5의 배수인지에 따라 'Fizz', 'Buzz', 'FizzBuzz' 또는 그 숫자를 출력합니다.

### 동작 과정
1. 입력을 3회 받습니다. 각 입력은 'Fizz', 'Buzz', 'FizzBuzz' 중 하나이거나 숫자일 수 있습니다.
2. 세 번의 반복에서, 만약 입력이 'Fizz', 'Buzz', 'FizzBuzz'가 아닌 경우에는 해당 입력을 정수로 변환하고 현재 반복의 인덱스 `i`를 더하여 `n`을 계산합니다.
   - 예를 들어, 입력이 '1', 'Fizz', '3'이라면:
     - 1회차: x = '1' → `n` = 1 + 3 = 4
     - 2회차: x = 'Fizz' → 무시
     - 3회차: x = '3' → `n` = 4 (이전 값) + 1 = 5
3. 최종적으로 `n`이 3의 배수인지, 5의 배수인지, 혹은 두 경우 모두인지 검사하여 적절한 문자열 혹은 숫자를 출력합니다.
   - `n`이 5일 경우, 5 % 3 != 0, 5 % 5 == 0 이므로 'Buzz'를 출력합니다.

### 시간 복잡도
이 알고리즘은 입력을 3번만 처리하므로 시간 복잡도는 O(1)입니다.

---

## 코드
```python
import sys

for i in range(3, 0, -1):
    x = input()
    if x not in ['Fizz', 'Buzz', 'FizzBuzz']:
        n = int(x) + i

if n%3==0 and n%5==0:
    print('FizzBuzz')
elif n%3==0:
    print('Fizz')
elif n%5==0:
    print('Buzz')
else:
    print(n)
```
