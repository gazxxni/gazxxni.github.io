## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 수행 시간을 계산하기 위해, 주어진 입력 크기 \( n \)에 대해 코드1의 수행 횟수를 수학적으로 도출합니다. 이 알고리즘은 이중 반복문을 사용하여 특정 작업을 수행하며, 각 반복문의 범위에 따라 수행 횟수를 계산할 수 있습니다. 최종적으로 수행 횟수를 다항식으로 표현하고, 최고차항의 차수를 결정합니다.

### 동작 과정
1. 입력으로 \( n \)이 주어집니다. 예를 들어 \( n = 5 \)라면, 배열의 크기는 5입니다.
2. 코드1의 수행 횟수는 다음과 같이 계산됩니다:
   \[
   \text{수행 횟수} = n \times (n - 1) - (n - 1) \times \left(\frac{n}{2}\right)
   \]
   위 식은 각 반복문이 수행하는 횟수를 기반으로 생성된 수식입니다.
3. 예를 들어 \( n = 5 \)일 때, 수행 횟수는:
   \[
   5 \times (5 - 1) - (5 - 1) \times \left(\frac{5}{2}\right) = 20 - 10 = 10
   \]
4. 수행 횟수를 출력한 후, 다항식으로 나타낼 때 최고차항의 차수를 확인합니다. 이 경우, 다항식의 차수는 2입니다.
5. 따라서 출력은 다음과 같습니다:
   ```
   10
   2
   ```

### 시간 복잡도
수행 횟수 계산은 \( O(1) \)의 시간 복잡도를 가집니다. 입력 크기 \( n \)에 따라 수행 횟수를 직접 계산할 수 있으므로 효율적입니다.

## 코드
```python
n = int(input())

print(int(n * (n - 1) - (n - 1) * (n / 2)))
print(2)
```
