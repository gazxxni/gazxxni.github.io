## 문제
한수는 지금 (x, y)에 있다. 직사각형은 각 변이 좌표축에 평행하고, 왼쪽 아래 꼭짓점은 (0, 0), 오른쪽 위 꼭짓점은 (w, h)에 있다. 직사각형의 경계선까지 가는 거리의 최솟값을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 x, y, w, h가 주어진다.

---

## 출력
첫째 줄에 문제의 정답을 출력한다.

## 풀이

### 풀이 핵심 로직
문제에서 주어진 점 (x, y)와 직사각형의 경계선까지의 거리를 계산하여 최솟값을 구하는 것이 핵심이다. 직사각형의 경계에 도달하기 위해 (x, y)에서 각각 왼쪽, 오른쪽, 위쪽, 아래쪽으로의 거리를 계산하고, 이 중에서 가장 짧은 거리를 선택하면 된다.

### 동작 과정
1. **입력 받기**: `x`, `y`, `w`, `h`를 입력받는다. 여기서 (0, 0)은 직사각형의 왼쪽 아래 꼭짓점, (w, h)는 오른쪽 위 꼭짓점이다.
   - 예: 입력이 `6 2 10 3`일 때, (x, y) = (6, 2), (w, h) = (10, 3)이다.
   
2. **거리 계산**:
   - 왼쪽 경계까지의 거리: `c = abs(x - 0) = x`
   - 오른쪽 경계까지의 거리: `a = abs(x - w) = w - x`
   - 아래 경계까지의 거리: `d = abs(y - 0) = y`
   - 위 경계까지의 거리: `b = abs(y - h) = h - y`
   - 예: 위의 입력에서, `c = 6`, `a = 4`, `d = 2`, `b = 1`이다.

3. **최솟값 계산**: 계산된 거리들 중에서 최소값을 찾는다. 
   - 예: `min(a, b, c, d)`에서 `min(4, 1, 6, 2)`의 결과는 `1`이다.

4. **결과 출력**: 최소 거리인 `1`을 출력한다.

### 시간 복잡도
이 알고리즘은 상수 시간 복잡도 O(1)을 가진다. 입력된 값을 바탕으로 단순한 수학 연산을 수행하므로 입력 크기에 따라 성능이 달라지지 않는다.

---

## 코드
```python
x, y, w, h = map(int, input().split())

a = abs(x - w)
b = abs(y - h)
c = abs(x)
d = abs(y)

print(min(a, b, c, d))
```
