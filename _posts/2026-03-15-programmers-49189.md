## 문제
n개의 노드가 있는 그래프가 있습니다. 각 노드는 1부터 n까지 번호가 적혀있습니다. 1번 노드에서 가장 멀리 떨어진 노드의 갯수를 구하려고 합니다. 가장 멀리 떨어진 노드란 최단경로로 이동했을 때 간선의 개수가 가장 많은 노드들을 의미합니다.

노드의 개수 n, 간선에 대한 정보가 담긴 2차원 배열 vertex가 매개변수로 주어질 때, 1번 노드로부터 가장 멀리 떨어진 노드가 몇 개인지를 return 하도록 solution 함수를 작성해주세요.

제한사항


노드의 개수 n은 2 이상 20,000 이하입니다.
간선은 양방향이며 총 1개 이상 50,000개 이하의 간선이 있습니다.
vertex 배열 각 행 [a, b]는 a번 노드와 b번 노드 사이에 간선이 있다는 의미입니다.


입출력 예

        
n
vertex
return

## 풀이
### 풀이 핵심 로직
이 문제는 그래프에서 최단 경로를 찾는 BFS (너비 우선 탐색) 알고리즘을 사용하여 1번 노드에서 가장 멀리 떨어진 노드를 찾는 것입니다. BFS를 통해 각 노드까지의 거리를 계산한 후, 가장 큰 거리 값을 가진 노드의 개수를 세어 결과를 반환합니다.

### 동작 과정
1. **그래프 구성**: 입력으로 주어진 간선 정보를 바탕으로 인접 리스트 형태의 그래프를 구성합니다.
   - 예를 들어, `n = 6`이고 `vertex = [[3, 6], [4, 3], [3, 2], [1, 3], [1, 2], [2, 4], [5, 2]]`일 때, 그래프는 다음과 같이 구성됩니다:
     ```
     1: [2, 3]
     2: [1, 4, 5]
     3: [1, 2, 4, 6]
     4: [3, 2]
     5: [2]
     6: [3]
     ```
   
2. **BFS 실행**: 1번 노드에서 시작하여 BFS를 수행하여 각 노드까지의 최단 거리를 기록합니다. 
   - 시작 노드(1번)에서 인접한 노드(2번, 3번)를 큐에 추가하고, 거리를 1로 설정합니다. 
   - 이후 큐에서 노드를 꺼내어 인접한 노드를 계속 탐색하고 거리를 업데이트합니다.

3. **최대 거리 노드 개수 계산**: BFS가 완료되면, 각 노드까지의 거리를 담은 리스트에서 최대 거리 값(max distance)을 찾고, 이 최대 거리와 동일한 거리를 가진 노드의 개수를 세어 반환합니다.
   - 예를 들어, 1번 노드에서 가장 멀리 떨어진 노드는 4, 5, 6번 노드이며, 이들은 모두 거리 2에 위치합니다. 따라서 반환 값은 3입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(V + E)입니다. 여기서 V는 노드의 개수(n), E는 간선의 개수입니다. BFS는 모든 노드를 한 번씩 방문하고, 모든 간선을 한 번씩 탐색하므로 이 시간 복잡도가 발생합니다.

## 코드
```python
from collections import deque

def solution(n, edge):
    answer = 0
    
    def bfs(graph, start):
        distance = [-1] * (n+1)
        q = deque([start])
        distance[start] = 0
        
        while q:
            cur = q.popleft()
            
            for next in graph[cur]:
                if distance[next] == -1:
                    distance[next] = distance[cur] + 1
                    q.append(next)
        
        return distance
        
    graph = {i:[] for i in range(1, n+1)}
    for u, v in edge:
        graph[u].append(v)
        graph[v].append(u)
    
    distances = bfs(graph, 1)
    return distances.count(max(distances))
```
