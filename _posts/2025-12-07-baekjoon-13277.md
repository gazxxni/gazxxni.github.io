## 문제
두 정수 A와 B가 주어졌을 때, 두 수의 곱을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 A와 B가 주어진다. 두 정수는 0보다 크거나 같은 정수이며, 0을 제외한 정수는 0으로 시작하지 않으며, 수의 앞에 불필요한 0이 있는 경우도 없다. 또한, 수의 길이는 300,000자리를 넘지 않는다.

---

## 출력
두 수의 곱을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 매우 큰 정수 A와 B를 입력받아 그 곱을 출력하는 문제입니다. 파이썬은 내장된 `int` 타입을 사용하여 자동으로 큰 수를 처리할 수 있으므로, 두 수를 정수로 변환한 후 곱셈을 수행하고 결과를 출력하면 됩니다.

### 동작 과정
1. 입력을 통해 두 개의 수 A와 B를 문자열로 받습니다.
   - 예시 입력: `123456789012345678901234567890 987654321098765432109876543210`
2. 이 두 수를 정수형으로 변환합니다.
   - A = 123456789012345678901234567890
   - B = 987654321098765432109876543210
3. 변환된 두 수를 곱합니다.
   - 곱셈 결과: `1219326311370217952261850327336229233322360640`
4. 결과를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n * m)입니다. 여기서 n과 m은 두 수 A와 B의 자리수입니다. 하지만 파이썬의 정수 곱셈은 내부적으로 최적화되어 있어서 실제 실행 시간은 매우 빠르게 처리됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a * b)
```
