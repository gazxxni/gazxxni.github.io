## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 수행 시간은 입력 크기 \( n \)에 따라 결정됩니다. 이 알고리즘은 단순히 \( n \)을 입력받아 그 값을 반환하고, 수행 횟수는 \( n \)과 같으므로 첫 번째 출력은 \( n \)입니다. 또한, 반복문이 없고 단일 연산으로 이루어져 있으므로 수행 횟수는 1차 다항식으로 표현됩니다.

### 동작 과정
1. **입력**: 사용자가 \( n \) 값을 입력합니다. 예를 들어, 사용자가 `5`를 입력한다고 가정합니다.
2. **수행 횟수 출력**: `print(n)`을 통해 입력받은 \( n \)의 값을 그대로 출력합니다. 예를 들어, `5`가 출력됩니다.
3. **다항식 차수 출력**: 코드의 구조상 알고리즘의 수행 횟수는 항상 1이므로, 다항식으로 나타낼 때 최고차항의 차수는 1이 됩니다. 따라서 `1`을 출력합니다.

### 시간 복잡도
이 알고리즘은 O(1) 시간 복잡도를 가집니다. 입력을 받고, 간단한 출력을 수행하는 과정이므로 입력 크기에 관계없이 상수 시간이 소요됩니다.
