## 문제 링크
[https://www.acmicpc.net/problem/14500](https://www.acmicpc.net/problem/14500)

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 DFS(깊이 우선 탐색)를 통해 모든 가능한 테트로미노 모양을 탐색하면서 최대 합을 계산하는 것입니다. 또한, 'ㅗ' 모양과 같은 특정 형태는 별도로 함수로 처리하여 모든 경우의 수를 고려합니다.

### 동작 과정
1. **입력**: n과 m을 입력받아 n x m 크기의 그래프를 구성합니다.
   예시 입력:
   ```
   3 3
   1 2 3
   4 5 6
   7 8 9
   ```

2. **DFS 호출**: 각 셀에 대해 DFS를 시작하여 4칸을 선택하면서 가능한 모든 조합을 탐색합니다.
   - 예를 들어, (0, 0)에서 시작하여 1(0, 0) → 2(0, 1) → 5(1, 1) → 6(1, 2)로 이동하여 합은 1 + 2 + 5 + 6 = 14가 됩니다.

3. **'ㅗ' 모양 처리**: 각 셀에 대해 인접한 4개의 셀을 고려하여 'ㅗ' 모양의 합을 계산합니다.
   - 예를 들어, (1, 1)에서 주변 값 1, 2, 4, 6을 고려하여 가장 작은 1을 제외하고 합을 계산하여 2 + 4 + 6 + 5 = 17로 갱신합니다.

4. **최대값 갱신**: DFS와 'ㅗ' 모양 처리를 통해 계산된 최대값을 `max_a`에 저장합니다.

5. **결과 출력**: 모든 탐색이 끝난 후 `max_a`를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n * m * 4)입니다. 각 셀에 대해 DFS를 호출하는데, DFS는 최대 4번의 인접 셀을 탐색하기 때문입니다. 따라서 전체 시간 복잡도는 O(n * m)입니다.

## 코드
```python
import sys

n, m = map(int, input().split())
graph = [list(map(int, input().split())) for _ in range(n)]
visited = [[False] * m for _ in range(n)]

dx = [1, -1, 0, 0] 
dy = [0, 0, 1, -1]  

max_a = 0  # 최대값을 저장할 변수

def dfs(x, y, tmp, cnt):
    global max_a

    if cnt == 4:  # 테트로미노 4칸을 모두 선택했으면 최대값 갱신
        max_a = max(max_a, tmp)
        return
    
    for i in range(4):
        nx = x + dx[i]  
        ny = y + dy[i]  

        # 경계 조건과 방문 여부 체크
        if nx < 0 or nx >= m or ny < 0 or ny >= n or visited[ny][nx]:
            continue
        
        # 다음 좌표 방문 처리 및 DFS 호출
        visited[ny][nx] = True
        dfs(nx, ny, tmp + graph[ny][nx], cnt + 1)
        visited[ny][nx] = False  # DFS 종료 후 방문 해제 (백트래킹)


# 'ㅗ' 모양 처리 함수
def fy(x, y):
    global max_a
    tmp = graph[y][x]  # 현재 위치의 값
    arr = []  # 주변 4개의 값 저장

    for i in range(4):
        nx = x + dx[i]  
        ny = y + dy[i]  

        # 경계 조건 체크
        if nx < 0 or nx >= m or ny < 0 or ny >= n:
            continue
        arr.append(graph[ny][nx])  # 유효한 값만 추가

    length = len(arr)

    # 주변 값이 4개일 경우, 가장 작은 값을 제외하고 계산
    if length == 4:
        arr.sort(reverse=True)  # 내림차순 정렬
        arr.pop()               # 가장 작은 값 제거
        max_a = max(max_a, sum(arr) + graph[y][x])

    # 주변 값이 3개일 경우, 그대로 계산
    elif length == 3:
        max_a = max(max_a, sum(arr) + graph[y][x])
    return

for i in range(n):
    for j in range(m):
        # DFS로 테트로미노 4칸 합 계산
        visited[i][j] = True
        dfs(j, i, graph[i][j], 1)
        visited[i][j] = False
        fy(j, i)

print(max_a)

```
