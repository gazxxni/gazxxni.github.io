## 문제
조규현과 백승환은 터렛에 근무하는 직원이다. 하지만 워낙 존재감이 없어서 인구수는 차지하지 않는다. 다음은 조규현과 백승환의 사진이다.



이석원은 조규현과 백승환에게 상대편 마린(류재명)의 위치를 계산하라는 명령을 내렸다. 조규현과 백승환은 각각 자신의 터렛 위치에서 현재 적까지의 거리를 계산했다.

조규현의 좌표 (x1,y1)$(x_1, y_1)$와 백승환의 좌표 (x2,y2)$(x_2, y_2)$가 주어지고, 조규현이 계산한 류재명과의 거리 r1$r_1$과 백승환이 계산한 류재명과의 거리 r2$r_2$가 주어졌을 때, 류재명이 있을 수 있는 좌표의 수를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T$T$가 주어진다. 각 테스트 케이스는 다음과 같이 이루어져 있다.

한 줄에 공백으로 구분 된 여섯 정수 x1$x_1$, y1$y_1$, r1$r_1$, x2$x_2$, y2$y_2$, r2$r_2$가 주어진다.

---

## 출력
각 테스트 케이스마다 류재명이 있을 수 있는 위치의 수를 출력한다. 만약 류재명이 있을 수 있는 위치의 개수가 무한대일 경우에는 −1$-1$ 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 원이 서로 교차하거나 겹치는 조건을 판단하여, 두 원의 중심과 반지름을 이용해 교차점의 개수를 계산하는 문제입니다. 주어진 두 점과 반지름을 기준으로 원의 위치 관계를 분석하여, 교차점의 수를 구하는 방식으로 해결합니다.

### 동작 과정
1. **입력 예시**: 
    - 입력: `1` (테스트 케이스 수)
    - 입력: `0 0 1 0 2 1` (조규현의 위치 `(0, 0)`, 반지름 `1`, 백승환의 위치 `(0, 2)`, 반지름 `1`)

2. **거리 계산**:
    - 두 점 사이의 거리: 
      \[
      \text{distance} = \sqrt{(0 - 0)^2 + (0 - 2)^2} = 2
      \]

3. **조건 검사**:
    - 첫 번째 조건: 두 중심점이 동일하고 반지름이 같으면 무한대의 점이 존재함 (`distance == 0` and `r1 == r2`).
    - 두 번째 조건: 두 원이 접하는 경우 (`abs(r1 - r2) == distance` or `r1 + r2 == distance`).
    - 세 번째 조건: 두 원이 교차하는 경우 (`abs(r1 - r2) < distance < (r1 + r2)`).
    - 네 번째 조건: 두 원이 서로 떨어져 있는 경우 (`distance >= (r1 + r2)` or `distance <= abs(r1 - r2)`).

4. **출력**:
    - 위 조건에 따라 결과를 출력합니다. 예를 들어, 위의 입력에서는 두 원이 교차하지 않으므로 `0`을 출력합니다.

### 시간 복잡도
이 알고리즘은 입력을 한 번씩 읽고, 몇 가지 수학적 연산을 수행하기 때문에 시간 복잡도는 \(O(1)\)입니다. 전체 테스트 케이스에 대해 \(O(T)\)가 되므로 최종 시간 복잡도는 \(O(T)\)입니다.

## 코드
```python
import math

t = int(input())

for _ in range(t):
    x1, y1, r1, x2, y2, r2 = map(int, input().split())
    distance = math.sqrt((x1 - x2) ** 2 + (y1 - y2) ** 2)
    
    if distance == 0 and r1 == r2 :  
        print(-1)
    elif abs(r1 - r2) == distance or r1 + r2 == distance: 
        print(1)
    elif abs(r1 - r2) < distance < (r1 + r2) :  
        print(2)
    else:
        print(0) 
```
