## 문제
H-Index는 과학자의 생산성과 영향력을 나타내는 지표입니다. 어느 과학자의 H-Index를 나타내는 값인 h를 구하려고 합니다. 위키백과1에 따르면, H-Index는 다음과 같이 구합니다.

어떤 과학자가 발표한 논문 n편 중, h번 이상 인용된 논문이 h편 이상이고 나머지 논문이 h번 이하 인용되었다면 h의 최댓값이 이 과학자의 H-Index입니다.

어떤 과학자가 발표한 논문의 인용 횟수를 담은 배열 citations가 매개변수로 주어질 때, 이 과학자의 H-Index를 return 하도록 solution 함수를 작성해주세요.

제한사항


과학자가 발표한 논문의 수는 1편 이상 1,000편 이하입니다.
논문별 인용 횟수는 0회 이상 10,000회 이하입니다.


입출력 예

        
citations
return


        
[3, 0, 6, 1, 5]
3


      
입출력 예 설명

이 과학자가 발표한 논문의 수는 5편이고, 그중 3편의 논문은 3회 이상 인용되었습니다. 그리고 나머지 2편의 논문은 3회 이하 인용되었기 때문에 이 과학자의 H-Index는 3입니다.

## 풀이
### 풀이 핵심 로직
H-Index를 계산하기 위해, 주어진 인용 횟수를 내림차순으로 정렬한 후, 각 인용 횟수와 그 인용 횟수 이상의 논문 수를 비교합니다. 만약 현재 인용 횟수가 논문 수보다 작아지면, 그 이전 인용 횟수가 H-Index의 최댓값이 됩니다.

### 동작 과정
1. 입력된 `citations` 배열을 내림차순으로 정렬합니다.
   - 예시: `citations = [3, 0, 6, 1, 5]` → 정렬 후 `[6, 5, 3, 1, 0]`
2. 정렬된 배열을 순회하며, 각 인용 횟수와 그 인용 횟수 이상인 논문의 수를 비교합니다.
   - `i = 0`: 인용 횟수 `6`, 논문 수 `1` (6 >= 1)
   - `i = 1`: 인용 횟수 `5`, 논문 수 `2` (5 >= 2)
   - `i = 2`: 인용 횟수 `3`, 논문 수 `3` (3 >= 3) → H-Index 후보
   - `i = 3`: 인용 횟수 `1`, 논문 수 `4` (1 < 4) → H-Index 결정
3. H-Index는 `3`으로 결정되며, 이를 반환합니다.

### 시간 복잡도
- 정렬 과정이 O(n log n)으로 가장 시간이 많이 걸리며, 이후에 배열을 한 번 순회하는 O(n) 시간이 필요합니다.
- 따라서 전체 시간 복잡도는 O(n log n)입니다.

## 코드
```python
def solution(citations):
    citations.sort(reverse=True)
    for i, v in enumerate(citations):
        if v < i + 1:
            return i
    return len(citations)
```
