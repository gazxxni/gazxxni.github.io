## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

## 성능 요약

## 풀이

### 풀이 핵심 로직
문자열 S의 서로 다른 부분 문자열의 개수를 구하기 위해, 모든 가능한 길이의 부분 문자열을 생성하고 이를 집합(set)에 추가합니다. 집합은 중복된 값을 허용하지 않기 때문에, 최종적으로 집합의 크기를 구하면 서로 다른 부분 문자열의 개수를 알 수 있습니다.

### 동작 과정
1. 입력 문자열 S가 "ababc"라고 가정합니다.
2. 부분 문자열의 길이를 1부터 S의 길이까지 증가시키면서 반복합니다.
3. 각 길이에 대해, 문자열의 모든 시작 인덱스에서 부분 문자열을 생성합니다.
   - 예를 들어, 길이가 1일 때: "a", "b", "a", "b", "c" → 집합에 추가 → { "a", "b", "c" }
   - 길이가 2일 때: "ab", "ba", "ab", "bc" → 집합에 추가 → { "a", "b", "c", "ab", "ba", "bc" }
   - 길이가 3일 때: "aba", "bab", "abc" → 집합에 추가 → { "a", "b", "c", "ab", "ba", "bc", "aba", "bab", "abc" }
   - 길이가 4일 때: "abab", "babc" → 집합에 추가 → { "a", "b", "c", "ab", "ba", "bc", "aba", "bab", "abc", "abab", "babc" }
   - 길이가 5일 때: "ababc" → 집합에 추가 → { "a", "b", "c", "ab", "ba", "bc", "aba", "bab", "abc", "abab", "babc", "ababc" }
4. 모든 부분 문자열을 집합에 추가한 후, 집합의 크기를 출력합니다. 이 경우, 최종적으로 12가 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 여기서 N은 문자열 S의 길이입니다. 이는 두 개의 중첩 루프 때문인데, 외부 루프는 부분 문자열의 길이를 결정하고, 내부 루프는 해당 길이에 대해 가능한 시작 인덱스를 탐색하며 부분 문자열을 생성하기 때문입니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
