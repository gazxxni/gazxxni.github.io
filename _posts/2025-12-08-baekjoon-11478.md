## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
문자열 S의 서로 다른 부분 문자열을 찾기 위해, 두 개의 반복문을 사용하여 모든 가능한 길이의 부분 문자열을 생성하고, 이를 집합(set)에 저장합니다. 집합은 중복된 값을 허용하지 않기 때문에, 최종적으로 집합의 크기를 통해 서로 다른 부분 문자열의 개수를 쉽게 구할 수 있습니다.

### 동작 과정
1. 입력 문자열 S가 "ababc"라고 가정합니다.
2. 첫 번째 반복문(i)은 부분 문자열의 길이를 1부터 S의 길이(5)까지 변화시킵니다.
   - i = 1: 부분 문자열은 "a", "b", "a", "b", "c" (집합에 추가 후: {'a', 'b', 'c'})
   - i = 2: 부분 문자열은 "ab", "ba", "ab", "bc" (집합에 추가 후: {'a', 'b', 'c', 'ab', 'ba', 'bc'})
   - i = 3: 부분 문자열은 "aba", "bab", "abc" (집합에 추가 후: {'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc'})
   - i = 4: 부분 문자열은 "abab", "babc" (집합에 추가 후: {'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc', 'abab', 'babc'})
   - i = 5: 부분 문자열은 "ababc" (집합에 추가 후: {'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc', 'abab', 'babc', 'ababc'})
3. 반복이 끝나면, 집합에는 서로 다른 부분 문자열이 모두 저장되며, 그 크기를 출력합니다. 이 예시의 경우 12개의 서로 다른 부분 문자열이 존재합니다.

### 시간 복잡도
최악의 경우, 모든 부분 문자열을 생성하는 데 O(n^2)의 시간이 소요됩니다. 여기서 n은 문자열 S의 길이입니다. 따라서 전체 시간 복잡도는 O(n^2)입니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
