## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제는 원형으로 배열된 풍선들을 터뜨리는 과정에서 발생하는 인덱스 이동을 효율적으로 처리하기 위해 덱(deque) 자료구조를 사용합니다. 각 풍선에는 터뜨린 후 이동할 방향과 거리가 기록되어 있으며, 이를 통해 다음에 터뜨릴 풍선의 인덱스를 결정합니다. 덱의 `rotate` 메소드를 활용하여 빠르게 인덱스를 조정하고, 터진 풍선을 기록한 후 결과를 출력합니다.

### 동작 과정
입력이 다음과 같다고 가정합니다.
```
5
3 2 1 -3 -1
```
1. 처음에 1번 풍선을 터뜨립니다. (풍선: 3 2 1 -3 -1)
   - 터진 풍선 번호: 1
   - 터진 풍선의 종이에는 3이 적혀 있으므로, 오른쪽으로 3번째 풍선을 찾습니다.
2. 현재 풍선 상태는 (2, 1, -3, -1)이므로, 오른쪽으로 3번째 풍선을 터뜨립니다. 4번 풍선 (-3)을 터뜨립니다.
   - 터진 풍선 번호: 4
   - 다음 이동은 왼쪽으로 3칸 이동합니다.
3. 현재 풍선 상태는 (2, 1, -1)이므로, 왼쪽으로 3번째 풍선을 터뜨립니다. 5번 풍선 (-1)을 터뜨립니다.
   - 터진 풍선 번호: 5
   - 다음 이동은 왼쪽으로 1칸 이동합니다.
4. 현재 풍선 상태는 (2, 1)에서 왼쪽으로 1칸 이동하여 3번 풍선을 터뜨립니다.
   - 터진 풍선 번호: 3
   - 다음 이동은 오른쪽으로 2칸 이동합니다.
5. 현재 풍선 상태는 (2)이고, 오른쪽으로 2칸 이동하여 2번 풍선을 터뜨립니다.
   - 터진 풍선 번호: 2

결과적으로 터진 풍선의 순서는 1, 4, 5, 3, 2가 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 풍선을 터뜨리는 과정에서 최대 N번의 회전이 발생하지만, 각 풍선은 한 번만 터지므로 전체적으로 N개의 풍선을 처리하는 데 필요한 시간은 선형적입니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
