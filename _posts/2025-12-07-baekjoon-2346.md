## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제는 풍선을 터뜨리는 순서를 결정하기 위해 deque를 사용하여 원형으로 연결된 풍선의 인덱스를 관리합니다. 각 풍선은 터뜨릴 때 그 안의 값에 따라 이동할 방향과 거리로 다음 풍선을 결정합니다. deque의 `rotate` 메소드를 통해 쉽게 원형 구조를 구현할 수 있으며, 이미 터진 풍선은 deque에서 제거되고, 순서대로 터진 풍선의 번호를 기록합니다.

### 동작 과정
1. **입력**: 예를 들어, N=5이고 풍선의 값이 [3, 2, 1, -3, -1]일 때, 초기 deque의 상태는 다음과 같습니다:
   ```
   deque([(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)])
   ```
   여기서 각 튜플은 (풍선 번호, 종이에 적힌 값)입니다.

2. **1번 풍선 터트리기**: 처음으로 1번 풍선을 터뜨립니다. 현재 deque에서 1번 풍선은 (0, 3)입니다. 이를 터뜨리고 결과 리스트에 추가합니다.
   ```
   ans = [1]
   ```

3. **이동**: 3만큼 오른쪽으로 이동하므로, deque를 2칸 회전합니다.
   ```
   deque([(2, 1), (3, -3), (4, -1), (1, 2)])
   ```

4. **2번 풍선 터트리기**: 2번 풍선을 터뜨리고 결과 리스트에 추가한 후, 1의 값을 사용하여 1칸 오른쪽으로 이동합니다.
   ```
   ans = [1, 3]
   ```

5. **반복**: 이 과정을 반복하여 마지막 풍선까지 터뜨립니다. 최종적으로 터진 풍선의 번호는 [1, 3, 5, 2, 4]가 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 풍선은 한 번씩만 터지고, deque의 각 연산은 O(1) 또는 O(N)으로 수행될 수 있으나, 전체적으로 N개의 풍선에 대해 작업을 수행하므로 효율적으로 처리됩니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
