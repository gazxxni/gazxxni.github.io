## 문제
2020년에 학교로 복학한 주형이는 월세를 마련하기 위해서 군 적금을 깨고 복리 투자를 하려고 한다.

주형이가 하려는 투자에는 3가지 방법의 투자 방식이 있다. 


	1년마다 5%의 이율을 얻는 투자 (A)
	3년마다 20%의 이율을 얻는 투자 (B)
	5년마다 35%의 이율을 얻는 투자 (C)


투자를 할 때에는 다음과 같은 주의점이 있다.


	투자의 기한(1년, 3년, 5년)을 채우는 시점에 이율이 반영되며, 그 사이에는 돈이 늘어나지 않는다.
	투자 방식은 매년 바꿀 수 있다.
	매번 이율은 소수점 이하를 버림 해서 받는다.


예를 들어서, 지금 가진 돈이 11111원이면, A 방식이면 1년 후에 555원, B 방식이면 3년 후에 2,222원, C 방식이면 5년 후에 3,888원을 이자로 받을 수 있다. 만약 C 방식으로 투자했지만 4년이 지난 시점이라면 받을 수 있는 이자는 0원이다.

주형이의 초기 비용이 H원일 때, Y년이 지난 시점에 가장 많은 금액을 얻을 수 있는 투자 패턴을 분석하고 그 금액을 출력하자.

---

## 입력
첫째 줄에 초기 비용 H와 투자 기간 Y가 주어진다.

모든 입력은 정수로 주어진다.

---

## 출력
가장 많은 이득을 얻었을 때의 총 자산을 소수점을 모두 버리고 정수로 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 주어진 투자 방식에 따라 주어진 기간 동안 최대 자산을 얻는 방법을 찾는 것입니다. 다이나믹 프로그래밍을 사용하여 각 연도마다 가능한 최대 자산을 계산하고, 이전의 최대 자산에 이율을 반영하여 갱신해 나갑니다. 이를 통해 각 연도마다 가장 최적화된 선택을 할 수 있습니다.

### 동작 과정
예시 입력으로 `H = 11111`, `Y = 6`을 가정해 보겠습니다.

1. 초기 자산 `dp[0]`은 `11111`입니다.
2. 1년 후(`i = 1`):
   - A 방식: `dp[0] * 1.05 = 11111 * 1.05 = 11666` (소수점 버림: 11666)
   - B 방식, C 방식은 해당 기간에 적용할 수 없으므로 `dp[1] = 11666`입니다.
   
3. 2년 후(`i = 2`):
   - A 방식: `dp[1] * 1.05 = 11666 * 1.05 = 12299` (소수점 버림: 12299)
   - B, C 방식 적용 불가: `dp[2] = 12299`입니다.
   
4. 3년 후(`i = 3`):
   - A 방식: `dp[2] * 1.05 = 12299 * 1.05 = 12914` (소수점 버림: 12914)
   - B 방식: `dp[0] * 1.2 = 11111 * 1.2 = 13333` (소수점 버림: 13333)
   - C 방식 적용 불가: `dp[3] = max(12914, 13333) = 13333`입니다.
   
5. 4년 후(`i = 4`):
   - A 방식: `dp[3] * 1.05 = 13333 * 1.05 = 13999` (소수점 버림: 13999)
   - B 방식: `dp[1] * 1.2 = 11666 * 1.2 = 13999` (소수점 버림: 13999)
   - C 방식 적용 불가: `dp[4] = max(13999, 13999) = 13999`입니다.
   
6. 5년 후(`i = 5`):
   - A 방식: `dp[4] * 1.05 = 13999 * 1.05 = 14669` (소수점 버림: 14669)
   - B 방식: `dp[2] * 1.2 = 12299 * 1.2 = 14758` (소수점 버림: 14758)
   - C 방식: `dp[0] * 1.35 = 11111 * 1.35 = 14999` (소수점 버림: 14999)
   - `dp[5] = max(14669, 14758, 14999) = 14999`입니다.
   
7. 6년 후(`i = 6`):
   - A 방식: `dp[5] * 1.05 = 14999 * 1.05 = 15748` (소수점 버림: 15748)
   - B 방식: `dp[3] * 1.2 = 13333 * 1.2 = 15999` (소수점 버림: 15999)
   - C 방식: `dp[1] * 1.35 = 11666 * 1.35 = 15799` (소수점 버림: 15799)
   - `dp[6] = max(15748, 15999, 15799) = 15999`입니다.

최종적으로 `dp[6]`의 값인 15999가 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(Y)입니다. Y는 투자 기간으로, 매년마다 이전의 결과를 바탕으로 최대 자산을 계산하는 방식이기 때문입니다.

## 코드
```python
h,y = map(int,input().split())
dp = [0 for _ in range(y+1)]
dp[0] = h

for i in range(1,y+1):
	if i >= 5:
		dp[i] = int(max(dp[i-1]*1.05, dp[i-3]*1.2, dp[i-5]*1.35))
	elif i >= 3:
		dp[i] = int(max(dp[i-1]*1.05, dp[i-3]*1.2))
	else:
		dp[i] = int(dp[i-1]*1.05)

print(dp[y])
```
