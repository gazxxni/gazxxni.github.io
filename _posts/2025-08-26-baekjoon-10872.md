## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
문제는 주어진 정수 N의 팩토리얼(N!)을 구하는 것이다. 팩토리얼은 N이 0일 때 1로 정의되며, N이 1 이상일 경우 N! = N × (N-1)!의 재귀적 정의를 따른다. 이를 통해 간단한 재귀 함수를 구현하여 팩토리얼 값을 구할 수 있다.

### 동작 과정
1. 사용자가 정수 N을 입력한다. 예를 들어, N=5라고 가정하자.
2. `fac`라는 재귀 함수를 호출하여 N의 팩토리얼을 계산한다.
   - fac(5)을 호출: 5 × fac(4)
   - fac(4)을 호출: 4 × fac(3)
   - fac(3)을 호출: 3 × fac(2)
   - fac(2)을 호출: 2 × fac(1)
   - fac(1)을 호출: 1 (기저 사례)
3. 이제 계산이 시작된다:
   - fac(2) = 2 × 1 = 2
   - fac(3) = 3 × 2 = 6
   - fac(4) = 4 × 6 = 24
   - fac(5) = 5 × 24 = 120
4. 최종적으로 120이 출력된다.

### 시간 복잡도
시간 복잡도는 O(N)이다. N이 최대 12이므로, 재귀 호출이 N번 발생하며 각 호출에서 상수 시간의 작업이 이루어진다.

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
