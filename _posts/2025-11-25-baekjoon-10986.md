## 문제
수 N개 A1, A2, ..., AN이 주어진다. 이때, 연속된 부분 구간의 합이 M으로 나누어 떨어지는 구간의 개수를 구하는 프로그램을 작성하시오.

즉, Ai + ... + Aj (i ≤ j) 의 합이 M으로 나누어 떨어지는 (i, j) 쌍의 개수를 구해야 한다.

---

## 입력
첫째 줄에 N과 M이 주어진다. (1 ≤ N ≤ 106, 2 ≤ M ≤ 103)

둘째 줄에 N개의 수 A1, A2, ..., AN이 주어진다. (0 ≤ Ai ≤ 109)

---

## 출력
첫째 줄에 연속된 부분 구간의 합이 M으로 나누어 떨어지는 구간의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제에서 요구하는 연속된 부분 구간의 합이 M으로 나누어 떨어진다는 것은, 두 누적합의 나머지가 같다는 것을 의미합니다. 즉, `prefix_sum[j] % M == prefix_sum[i-1] % M`이 성립할 때, 해당 구간의 합이 M으로 나누어 떨어집니다. 이를 통해 나머지를 카운트하여 가능한 구간의 쌍을 계산할 수 있습니다.

### 동작 과정
1. 주어진 입력값을 읽어 `N`과 `M`, 그리고 수열 `A`를 저장합니다.
2. `remainder_count` 배열을 초기화하여 각 나머지의 개수를 저장할 준비를 합니다. 여기서 `remainder_count[0]`은 초기값으로 1로 설정합니다. 이는 누적합이 처음부터 M으로 나누어 떨어지는 경우를 대비합니다.
3. `current_sum` 변수를 통해 누적합을 계산하며, 각 원소를 더한 후 그 값을 M으로 나눈 나머지를 구합니다.
4. 해당 나머지를 `remainder_count` 배열에서 카운트합니다.
5. 마지막으로, 각 나머지가 2개 이상일 경우 그 조합을 통해 구간 쌍의 수를 계산합니다. `cnt * (cnt - 1) // 2`의 공식을 사용하여 조합의 수를 더합니다.

예시 입력:
```
5 3
1 2 3 1 2
```
- 누적합 계산:
  - 1: (1) -> 1 % 3 = 1
  - 2: (1+2) -> 3 % 3 = 0
  - 3: (1+2+3) -> 6 % 3 = 0
  - 4: (1+2+3+1) -> 7 % 3 = 1
  - 5: (1+2+3+1+2) -> 9 % 3 = 0
- 나머지 배열 `remainder_count`는 [3, 2, 0]이 되어, 나머지가 0인 경우는 3개, 나머지가 1인 경우는 2개입니다.
- 0인 경우에서 구간 쌍의 수는 `3 * (3 - 1) // 2 = 3`이고, 1인 경우는 `2 * (2 - 1) // 2 = 1`이므로 총합은 3 + 1 = 4입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 수열을 한 번 순회하면서 누적합과 나머지를 계산하기 때문입니다. 공간 복잡도는 O(M)으로, 나머지를 저장하기 위한 배열의 크기에 비례합니다.

## 코드
```python
import sys
input = sys.stdin.readline

'''
1. 구간 합이 M으로 나누어 떨어진다는 것은 
(prefix_sum[j] - prefix_sum[i-1]) % M == 0이라는 뜻

2. 이 식을 이항하면 prefix_sum[j] % M == prefix_sum[i-1] % M

3. 즉, 누적합을 M으로 나눈 나머지가 같은 인덱스들을 찾으면 됨
'''
n, m = map(int, input().split())
arr = list(map(int, input().split()))

# 1. 나머지를 카운트할 배열 또는 딕셔너리 준비
remainder_count = [0] * m

# 2. 누적합 변수 초기화
current_sum = 0

# 3. 나머지가 0인 경우를 미리 카운트
remainder_count[0] = 1 

# 4. 수열을 순회하면서 누적합과 나머지 계산
for x in arr:
    current_sum += x
    r = current_sum % m
    remainder_count[r] += 1

ans = 0
for cnt in remainder_count:
    # 해당 나머지를 가진 누적합의 개수가 2개 이상일 때만 쌍을 만들 수 있음
    if cnt > 1:
        ans += (cnt * (cnt - 1)) // 2

print(ans)
```
