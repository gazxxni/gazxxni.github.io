## 문제
부산사이버대학교 학생 성택이는 엄마의 의뢰를 받아 주어진 문자열이 현관문 비밀번호에 사용 가능한지 알아내야 한다. 성택이는 공부해야 하므로 우리가 도와주자!

사용할 수 있는 비밀번호의 규칙은 다음과 같다.


	비밀번호는 6자리 이상 9자리 이하여야 한다.


예를 들어, 123124는 올바른 비밀번호이고, 1202727161은 잘못된 비밀번호이다. 문자열이 주어졌을 때 현관문 비밀번호로 사용할 수 있는지 판단하자.

---

## 입력
첫째 줄에 문자열의 총개수 N$N$이 주어진다.

둘째 줄부터 N$N$개의 줄에 걸쳐 숫자, 영어 대소문자로만 구성된 문자열이 주어진다.

---

## 출력
줄마다 사용할 수 있는 비밀번호면 yes, 그렇지 않으면 no를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 문자열의 길이를 확인하여 비밀번호의 유효성을 검사합니다. 비밀번호는 6자리 이상 9자리 이하여야 하므로, 입력된 문자열의 길이가 이 범위 내에 있는지를 조건문을 통해 판단합니다. 조건이 만족하면 "yes", 그렇지 않으면 "no"를 출력합니다.

### 동작 과정
1. 첫 번째 줄에서 입력되는 총 문자열의 개수 N을 읽어옵니다.
2. 다음 N개의 줄에 대해 반복하며 각 비밀번호 문자열을 읽어옵니다.
3. 각 문자열의 길이를 측정합니다.
4. 길이가 6 이상 9 이하인 경우 "yes"를 출력하고, 그렇지 않은 경우 "no"를 출력합니다.

예시 입력:
```
3
123456
abcd
1234567890
```
- 첫 번째 비밀번호 "123456"는 6자리이므로 "yes" 출력.
- 두 번째 비밀번호 "abcd"는 4자리이므로 "no" 출력.
- 세 번째 비밀번호 "1234567890"은 10자리이므로 "no" 출력.

출력 결과:
```
yes
no
no
```

### 시간 복잡도
O(N) - N은 입력 문자열의 개수입니다. 각 문자열에 대해 길이를 측정하는 작업은 O(1) 시간 안에 수행되므로 전체 시간 복잡도는 입력 문자열의 개수에 비례합니다.

---

## 코드
```python
for _ in range(int(input())):
    password = len(input())
    print("yes") if 6 <= password and password <= 9 else print("no")

```
