## 문제
부산사이버대학교 학생 성택이는 엄마의 의뢰를 받아 주어진 문자열이 현관문 비밀번호에 사용 가능한지 알아내야 한다. 성택이는 공부해야 하므로 우리가 도와주자!

사용할 수 있는 비밀번호의 규칙은 다음과 같다.


	비밀번호는 6자리 이상 9자리 이하여야 한다.


예를 들어, 123124는 올바른 비밀번호이고, 1202727161은 잘못된 비밀번호이다. 문자열이 주어졌을 때 현관문 비밀번호로 사용할 수 있는지 판단하자.

---

## 입력
첫째 줄에 문자열의 총개수 N$N$이 주어진다.

둘째 줄부터 N$N$개의 줄에 걸쳐 숫자, 영어 대소문자로만 구성된 문자열이 주어진다.

---

## 출력
줄마다 사용할 수 있는 비밀번호면 yes, 그렇지 않으면 no를 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 주어진 문자열의 길이를 체크하여 비밀번호의 규칙을 만족하는지를 판단하는 것입니다. 비밀번호는 6자리 이상 9자리 이하이어야 하므로, 문자열의 길이를 측정하고 그 길이가 이 범위에 포함되는지를 확인하면 됩니다.

### 동작 과정
1. 입력 받은 첫 번째 줄에서 총 문자열의 개수 \( N \)을 읽어옵니다.
2. 이후 \( N \)개의 문자열을 차례로 읽어옵니다.
3. 각 문자열에 대해 길이를 계산합니다.
4. 길이가 6 이상 9 이하인 경우 "yes"를 출력하고, 그렇지 않은 경우 "no"를 출력합니다.

예시 입력:
```
3
123456
abcdefg
abcdefghij
```
단계별 설명:
- 첫 번째 입력 "123456"의 길이는 6이므로 "yes"를 출력.
- 두 번째 입력 "abcdefg"의 길이는 7이므로 "yes"를 출력.
- 세 번째 입력 "abcdefghij"의 길이는 10이므로 "no"를 출력.

출력:
```
yes
yes
no
```

### 시간 복잡도
입력 문자열의 개수를 \( N \)이라고 할 때, 각 문자열의 길이를 체크하는 작업은 \( O(1) \) 연산이므로 전체 시간 복잡도는 \( O(N) \)입니다.

---

## 코드
```python
for _ in range(int(input())):
    password = len(input())
    print("yes") if 6 <= password and password <= 9 else print("no")

```
