## 문제
때는 2040년, 이민혁은 우주에 자신만의 왕국을 만들었다. 왕국은 N개의 행성으로 이루어져 있다. 민혁이는 이 행성을 효율적으로 지배하기 위해서 행성을 연결하는 터널을 만들려고 한다.


	행성은 3차원 좌표위의 한 점으로 생각하면 된다. 두 행성 A(xA, yA, zA)와 B(xB, yB, zB)를 터널로 연결할 때 드는 비용은 min(|xA-xB|, |yA-yB|, |zA-zB|)이다.


	민혁이는 터널을 총 N-1개 건설해서 모든 행성이 서로 연결되게 하려고 한다. 이때, 모든 행성을 터널로 연결하는데 필요한 최소 비용을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 행성의 개수 N이 주어진다. (1 ≤ N ≤ 100,000) 다음 N개 줄에는 각 행성의 x, y, z좌표가 주어진다. 좌표는 -109보다 크거나 같고, 109보다 작거나 같은 정수이다. 한 위치에 행성이 두 개 이상 있는 경우는 없다.

---

## 출력
첫째 줄에 모든 행성을 터널로 연결하는데 필요한 최소 비용을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 최소 스패닝 트리(MST) 문제로, 주어진 N개의 행성을 연결하는 최적의 터널 비용을 계산해야 합니다. 비용은 두 행성 간의 최소 차원 거리로 정의되며, 이를 위해 각 차원별로 정렬하여 인접한 행성들 간의 비용을 간편하게 계산할 수 있습니다. Kruskal의 알고리즘을 사용하여 이러한 간선들을 선택하여 MST를 만들고, 그 비용을 합산하여 최종 결과를 도출합니다.

### 동작 과정
1. **입력**: 예를 들어, N=4인 경우 다음과 같은 4개의 행성 좌표가 주어졌다고 가정합니다.
   ```
   (1, 2, 3)
   (3, 2, 1)
   (6, 5, 4)
   (5, 6, 7)
   ```
   
2. **행성 좌표 정리**: 각 차원(x, y, z)별로 행성의 좌표와 인덱스를 묶어서 리스트를 생성합니다.
   - `xx = [(1, 0), (3, 1), (5, 3), (6, 2)]`
   - `yy = [(2, 0), (2, 1), (5, 3), (6, 2)]`
   - `zz = [(1, 1), (3, 0), (4, 2), (7, 3)]`

3. **비용 계산**: 각 차원에서 인접한 행성 간의 비용을 계산합니다. 예를 들어, x좌표에서:
   - (1, 0)과 (3, 1) 간의 비용: `min(|1-3|, |2-2|, |3-1|) = 2`
   - (3, 1)과 (5, 3) 간의 비용: `2`
   - (5, 3)과 (6, 2) 간의 비용: `1`
   이와 같은 방식으로 모든 차원에서 비용을 계산하여 간선 리스트를 생성합니다.

4. **간선 정렬**: 모든 간선을 비용 기준으로 정렬합니다.

5. **Kruskal 알고리즘 적용**: 정렬된 간선 리스트를 순회하면서, 간선의 양 끝에 있는 행성이 서로 연결되어 있지 않은 경우에만 그 간선을 선택하고, 선택된 간선의 비용을 합산합니다. 이를 통해 MST를 구성합니다.

6. **최종 결과 출력**: MST의 총 비용을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)입니다. 주된 시간 소모는 행성을 정렬하는 부분과 간선을 정렬하는 부분에서 발생하며, 각각 O(N log N)입니다. Kruskal의 알고리즘에서 간선의 수는 최대 3N-3이므로, 전체적으로 O(N log N) 복잡도를 유지하게 됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

def find(parent, x):
    if parent[x] != x:
        parent[x] = find(parent, parent[x])
    return parent[x]

def union(parent, a, b):
    a = find(parent, a)
    b = find(parent, b)
    
    if a < b:
        parent[b] = a
    else:
        parent[a] = b
        
xx, yy, zz = [], [], []
for i in range(n):
    x, y, z = map(int, input().split())
    xx.append((x, i))
    yy.append((y, i))
    zz.append((z, i))
    
xx.sort()
yy.sort()
zz.sort()

edge = []
for cur in xx, yy, zz:
    for i in range(1, n):
        w1, a = cur[i - 1]
        w2, b = cur[i]
        edge.append((abs(w1 - w2), a, b))
        
edge.sort()

parent = [i for i in range(n)]

def krustal():
    ans = 0
    
    for cost, a, b in edge:
        if find(parent, a) != find(parent, b):
            union(parent, a, b)
            ans += cost
    
    return ans

print(krustal())
```
