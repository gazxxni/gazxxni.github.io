## 문제
때는 2040년, 이민혁은 우주에 자신만의 왕국을 만들었다. 왕국은 N개의 행성으로 이루어져 있다. 민혁이는 이 행성을 효율적으로 지배하기 위해서 행성을 연결하는 터널을 만들려고 한다.


	행성은 3차원 좌표위의 한 점으로 생각하면 된다. 두 행성 A(xA, yA, zA)와 B(xB, yB, zB)를 터널로 연결할 때 드는 비용은 min(|xA-xB|, |yA-yB|, |zA-zB|)이다.


	민혁이는 터널을 총 N-1개 건설해서 모든 행성이 서로 연결되게 하려고 한다. 이때, 모든 행성을 터널로 연결하는데 필요한 최소 비용을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 행성의 개수 N이 주어진다. (1 ≤ N ≤ 100,000) 다음 N개 줄에는 각 행성의 x, y, z좌표가 주어진다. 좌표는 -109보다 크거나 같고, 109보다 작거나 같은 정수이다. 한 위치에 행성이 두 개 이상 있는 경우는 없다.

---

## 출력
첫째 줄에 모든 행성을 터널로 연결하는데 필요한 최소 비용을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 각 행성 간의 터널 연결 비용을 최소화하기 위해 Kruskal 알고리즘을 사용하여 최소 스패닝 트리를 구성하는 것입니다. 각 행성의 3차원 좌표를 활용하여 x, y, z 축에 따라 정렬한 후, 인접한 행성들 간의 비용을 계산하여 간선 리스트를 생성합니다. 이후 이 간선 리스트를 기반으로 Kruskal 알고리즘을 통해 최소 비용을 구합니다.

### 동작 과정
1. **입력 받기:** N개의 행성의 좌표를 입력받습니다.
2. **정렬:** x, y, z 좌표를 각각 기준으로 행성을 정렬합니다. 각 축에서 인접한 행성들 간의 거리 차이를 간선의 비용으로 삼습니다.
3. **간선 리스트 생성:** 정렬된 각 축에 대해 인접한 행성의 좌표 차이를 계산하여 간선 리스트를 만듭니다. 이때 비용은 `min(|xA-xB|, |yA-yB|, |zA-zB|)`입니다.
4. **Kruskal 알고리즘 수행:** 간선을 비용이 낮은 순서로 정렬한 후, Union-Find 자료구조를 통해 사이클이 생기지 않도록 간선을 추가하면서 최소 비용을 계산합니다.
5. **결과 출력:** 모든 행성을 연결하는 데 필요한 최소 비용을 출력합니다.

**예시 입력:**
```
5
0 0 0
2 2 2
3 3 3
5 5 5
1 1 1
```
- 입력된 5개의 행성의 좌표를 기반으로 x, y, z 축에 따라 정렬합니다.
- 각 축에서 인접한 행성 간의 비용을 계산하여 간선 리스트를 생성합니다.
- Kruskal 알고리즘을 통해 최종적으로 모든 행성을 연결하는 최소 비용을 계산합니다.

**예시 출력:**
```
4
```

### 시간 복잡도
- 간선 생성 및 정렬 단계에서 O(N log N)의 시간 복잡도를 가집니다.
- Kruskal 알고리즘의 경우 간선을 정렬하는 데 O(E log E) 소요되며, E는 생성된 간선의 수입니다. 최악의 경우 O(N^2)이 될 수 있지만, 실제로는 O(N log N)으로 간선 수를 줄일 수 있습니다.
- 따라서 전체적인 시간 복잡도는 O(N log N) 입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

def find(parent, x):
    if parent[x] != x:
        parent[x] = find(parent, parent[x])
    return parent[x]

def union(parent, a, b):
    a = find(parent, a)
    b = find(parent, b)
    
    if a < b:
        parent[b] = a
    else:
        parent[a] = b
        
xx, yy, zz = [], [], []
for i in range(n):
    x, y, z = map(int, input().split())
    xx.append((x, i))
    yy.append((y, i))
    zz.append((z, i))
    
xx.sort()
yy.sort()
zz.sort()

edge = []
for cur in xx, yy, zz:
    for i in range(1, n):
        w1, a = cur[i - 1]
        w2, b = cur[i]
        edge.append((abs(w1 - w2), a, b))
        
edge.sort()

parent = [i for i in range(n)]

def krustal():
    ans = 0
    
    for cost, a, b in edge:
        if find(parent, a) != find(parent, b):
            union(parent, a, b)
            ans += cost
    
    return ans

print(krustal())
```
