## 풀이

### 풀이 핵심 로직
주어진 문제는 피보나치 수열의 n번째 항을 계산하는 함수와 그 과정에서 호출되는 함수의 수를 세는 문제이다. 이 코드는 재귀를 사용하여 n번째 피보나치 수를 계산하고, 동시에 호출된 재귀 함수의 수를 count하는 방법으로, 피보나치 수는 n-2의 형태로 직접 계산하여 반환한다.

### 동작 과정
1. 사용자로부터 정수 n을 입력받는다.
2. `fib(n)` 함수를 호출하여 n번째 피보나치 수를 계산한다:
   - n이 1 또는 2일 경우 1을 반환한다.
   - 그 외의 경우, `fib(n-1)`과 `fib(n-2)`의 결과를 더하여 반환한다.
3. `fibonacci(n)` 함수를 호출하여 n번째 피보나치 수를 계산하기 위한 재귀 호출 수를 n-2로 반환한다.
4. 최종적으로 두 개의 결과를 출력한다: n번째 피보나치 수와 재귀 호출 수.

예시:
- 입력: `5`
  - `fib(5)` 호출 → `fib(4)`과 `fib(3)` 호출
  - `fib(4)` 호출 → `fib(3)`과 `fib(2)` 호출
  - `fib(3)` 호출 → `fib(2)`와 `fib(1)` 호출
  - 결과적으로 `fib(5)`는 5를 반환하고, `fibonacci(5)`는 3을 반환한다.
  
출력: `5 3`

### 시간 복잡도
- `fib(n)`의 재귀 호출은 중복 계산이 많기 때문에 지수 시간 복잡도 O(2^n)을 가진다.
- `fibonacci(n)`는 O(1)로 상수 시간 복잡도를 가진다.
- 최종적으로 전체 시간 복잡도는 O(2^n)이다.

## 코드
```python
def fib(n):
    if n == 1 or n == 2:
        return 1
    else:
        return fib(n - 1) + fib(n - 2)
    
def fibonacci(n):
    return n-2
    
n = int(input())
print(fib(n), fibonacci(n))
```
