## 풀이

### 풀이 핵심 로직
이 문제는 2개의 스티커 배열에서 스티커를 선택하는 방식으로 최댓값을 구하는 동적 프로그래밍 문제이다. 인접한 스티커를 선택할 수 없기 때문에, 각 스티커를 선택할 때 이전 스티커와의 관계를 고려하여 최댓값을 갱신하는 방식으로 해결한다.

### 동작 과정
1. 입력을 통해 테스트 케이스 수 `t`와 각 테스트 케이스의 스티커 배열 크기 `n` 및 스티커 배열 `arr`을 읽는다.
2. DP 테이블 `dp`를 초기화하여 첫 번째 스티커의 값을 설정한다.
3. 스티커가 1개일 경우, 두 스티커 중 큰 값을 출력하고 다음 테스트 케이스로 넘어간다.
4. 스티커가 2개일 경우, 두 번째 스티커를 선택할 때의 최댓값을 계산하여 출력하고 다음 테스트 케이스로 넘어간다.
5. 스티커가 3개 이상일 경우, 2번째 스티커부터 마지막 스티커까지 반복하며 각 스티커를 선택할 때의 최댓값을 DP 테이블에 저장한다. 이 때, 이전 스티커를 선택한 경우와 선택하지 않은 경우를 고려하여 최댓값을 갱신한다.
6. 마지막으로, 두 줄의 마지막 스티커 중 큰 값을 출력한다.

예시 입력:
```
2
5
50 10 100 20 40
30 50 70 10 60
```
- 첫 번째 케이스에서, 첫 번째 스티커의 값은 그대로 두고, 두 번째 스티커부터 각 스티커에 대해 최댓값을 계산하여 `dp`를 갱신한다.
- 최종적으로 `dp[0][4]`와 `dp[1][4]` 중 큰 값을 출력하게 된다.

### 시간 복잡도
O(n) - 입력을 처리하고 DP 테이블을 갱신하는 과정에서 각 스티커를 한 번씩만 방문하기 때문에 선형 시간 복잡도를 가진다.

## 코드
```python
import sys
input = sys.stdin.readline 

t = int(input()) 

for _ in range(t):
    n = int(input())  
    arr = [list(map(int, input().split())) for _ in range(2)]  

    dp = [[0] * n for _ in range(2)]
    dp[0][0] = arr[0][0] 
    dp[1][0] = arr[1][0]  

    if n == 1:
        print(max(dp[0][0], dp[1][0]))
        continue

    dp[0][1] = arr[0][1] + arr[1][0]  # 두 번째 열에서 위쪽 + 아래쪽 첫 번째 칸과 더함
    dp[1][1] = arr[1][1] + arr[0][0]  # 두 번째 열에서 아래쪽 + 위쪽 첫 번째 칸과 더함

    if n == 2:
        print(max(dp[0][1], dp[1][1]))
        continue

    for i in range(2, n):
        # 이전 대각선 값(dp[1][i-1]) 또는 두 칸 전 대각선 값(dp[1][i-2]) 중 최댓값을 더함
        dp[0][i] = max(dp[1][i - 1], dp[1][i - 2]) + arr[0][i]

        # 이전 대각선 값(dp[0][i-1]) 또는 두 칸 전 대각선 값(dp[0][i-2]) 중 최댓값을 더함
        dp[1][i] = max(dp[0][i - 1], dp[0][i - 2]) + arr[1][i]

    print(max(dp[0][n - 1], dp[1][n - 1]))






"""
arr -> O(n)
dp -> O(n)
for문 -> O(n)
"""
```
