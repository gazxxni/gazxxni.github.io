## 풀이

### 풀이 핵심 로직
이 문제는 주어진 숫자들을 절댓값 기준으로 우선순위를 두어 처리하는 문제입니다. 숫자가 0일 경우, 현재 힙에서 절댓값이 가장 작은 숫자를 출력하고, 절댓값이 같을 경우에는 음수를 먼저 출력합니다. 이를 위해 파이썬의 `heapq` 모듈을 사용하여 최소 힙을 구현합니다.

### 동작 과정
1. **입력 처리**: 먼저, 정수 n을 읽어 들여 n개의 숫자를 처리할 준비를 합니다.
2. **숫자 삽입**: 각 숫자를 읽어들여, 만약 그 숫자가 0이 아닐 경우, (절댓값, 원래 값) 형태로 힙에 삽입합니다. 이때 `heapq.heappush`를 사용하여 힙에 삽입합니다.
3. **숫자 출력**: 숫자가 0일 경우, 힙이 비어있지 않다면 `heapq.heappop`을 이용해 절댓값이 가장 작은 원래 값을 꺼내 출력합니다. 만약 힙이 비어 있다면 0을 출력합니다.
4. **예시 흐름**:
   - 입력: `3`, `1`, `-1`, `0`
   - 처리:
     - `1` 삽입 -> 힙: `[(1, 1)]`
     - `-1` 삽입 -> 힙: `[(1, 1), (1, -1)]`
     - `0` 입력 -> 힙에서 `1`을 꺼내 출력 (`-1`은 절댓값이 같지만 음수가 먼저 출력됨)

### 시간 복잡도
- 각 삽입과 삭제 연산은 O(log k) 시간복잡도를 가집니다. 여기서 k는 힙의 크기입니다. 따라서 n개의 숫자를 모두 처리하는 데 걸리는 시간 복잡도는 O(n log n)입니다.

## 코드
```python
import heapq
import sys

input = sys.stdin.readline
heap = []

n = int(input())

for _ in range(n):
    num = int(input())
    
    if num != 0:
        # (절댓값, 원래 값) 형태로 힙에 삽입
        heapq.heappush(heap, (abs(num), num))
    else:
        if heap:
            # 절댓값이 가장 작은 값을 꺼냄, 절댓값이 같으면 음수가 먼저 나옴
            print(heapq.heappop(heap)[1])
        else:
            print(0)

```
