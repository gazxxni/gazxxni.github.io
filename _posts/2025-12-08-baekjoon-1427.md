## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 수 N의 각 자리수를 분리하여 리스트에 저장한 후, 리스트를 내림차순으로 정렬하여 결과를 출력하는 방식으로 문제를 해결한다. 이 과정에서 각 자리수를 정수로 변환하고 정렬한 뒤, 다시 문자열로 합쳐서 출력한다.

### 동작 과정
1. 입력된 수 N을 문자열로 읽어온다. 예를 들어, 입력이 `2143`일 경우, 문자열 `"2143"`으로 처리된다.
2. 각 자리수를 리스트로 변환한다. 이때 `map(int, n)`을 사용하여 각 문자 ('2', '1', '4', '3')를 정수로 변환하여 리스트 `[2, 1, 4, 3]`를 만든다.
3. `sort(reverse=True)` 메서드를 사용하여 리스트를 내림차순으로 정렬한다. 정렬 후 리스트는 `[4, 3, 2, 1]`이 된다.
4. 정렬된 리스트를 출력하기 위해 `print(*a, sep='')`를 사용하여 리스트의 요소를 공백 없이 연결한 문자열로 출력한다. 최종 출력은 `4321`이 된다.

### 시간 복잡도
주어진 수 N의 자리수 개수는 최대 10개이므로, 리스트의 정렬에는 O(d log d)의 시간이 필요하다. 여기서 d는 N의 자리수 개수(최대 10)이다. 따라서 전체 시간 복잡도는 O(1)로 간주할 수 있다, 즉 상수 시간 내에 해결된다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
