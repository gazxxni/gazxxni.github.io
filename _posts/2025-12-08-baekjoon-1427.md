## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 자연수의 각 자리수를 분리하여 내림차순으로 정렬하는 문제입니다. 이 문제는 문자열로 입력된 수를 각 자리수로 나눈 후, 정렬하여 다시 문자열로 합치는 방식으로 해결할 수 있습니다.

### 동작 과정
1. 입력으로 주어진 수 N을 문자열로 읽어옵니다.
   - 예시: 입력 `2143`
2. 문자열을 각 자리수로 나누어 리스트로 변환합니다.
   - 변환 결과: `[2, 1, 4, 3]`
3. 리스트를 내림차순으로 정렬합니다.
   - 정렬 결과: `[4, 3, 2, 1]`
4. 정렬된 리스트를 다시 문자열로 변환하여 출력합니다.
   - 출력: `4321`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(d log d)입니다. 여기서 d는 입력 수 N의 자리수입니다. 자리수를 정렬하는 데 필요한 시간은 log d이므로, 전체적으로는 O(d log d)로 볼 수 있습니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
