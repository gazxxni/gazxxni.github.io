## 문제
정수를 저장하는 스택을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 다섯 가지이다.


	1 X: 정수 X를 스택에 넣는다. (1 ≤ X ≤ 100,000)
	2: 스택에 정수가 있다면 맨 위의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	3: 스택에 들어있는 정수의 개수를 출력한다.
	4: 스택이 비어있으면 1, 아니면 0을 출력한다.
	5: 스택에 정수가 있다면 맨 위의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
스택은 후입선출(LIFO) 구조를 가지며, 주어진 명령에 따라 정수를 추가하거나 제거하는 기본적인 연산을 수행한다. 파이썬의 리스트를 활용하여 스택을 구현하고, 각 명령에 따라 적절한 리스트 메소드를 사용하여 스택의 상태를 변경하거나 정보를 출력한다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 명령의 수 `N`을 입력받고, 이후 `N`개의 명령을 처리한다.
2. **명령 처리**:
   - **명령 1 X**: `X`를 스택에 추가 (`stack.append(b)`).
   - **명령 2**: 스택에서 최상단 요소를 제거하고 출력. 스택이 비어 있으면 `-1` 출력 (`stack.pop()`).
   - **명령 3**: 스택의 현재 크기를 출력 (`len(stack)`).
   - **명령 4**: 스택이 비어 있으면 `1`, 아니면 `0` 출력.
   - **명령 5**: 스택의 최상단 요소를 출력. 비어 있을 경우 `-1` 출력.

예시 입력:
```
5
1 10
1 20
2
3
4
```
- 명령 1: `10`을 스택에 추가 → 스택: `[10]`
- 명령 1: `20`을 스택에 추가 → 스택: `[10, 20]`
- 명령 2: 스택에서 최상단 `20`을 제거하고 출력 → 출력: `20`
- 명령 3: 스택의 크기 출력 → 출력: `1`
- 명령 4: 스택이 비어 있지 않으므로 `0` 출력 → 출력: `0`

### 시간 복잡도
모든 명령은 `O(1)`의 시간 복잡도를 가지므로 총 `N`개의 명령을 처리하는 전체 시간 복잡도는 `O(N)`이다.

## 코드
```python
import sys

n = int(sys.stdin.readline())
stack = []

for _ in range(n):
    command = sys.stdin.readline().split()
    
    a = int(command[0])
    
    if a == 1:
        b = int(command[1])
        stack.append(b)
        
    elif a == 2:
        if stack:
            print(stack.pop())
        else:
            print(-1)
            
    elif a == 3:
        print(len(stack))
        
    elif a == 4:
        if not stack:
            print(1)
        else:
            print(0)
            
    elif a == 5:
        if stack:
            print(stack[-1])
        else:
            print(-1)
```
