## 문제
45656이란 수를 보자.

이 수는 인접한 모든 자리의 차이가 1이다. 이런 수를 계단 수라고 한다.

N이 주어질 때, 길이가 N이면서 0부터 9까지 숫자가 모두 등장하는 계단 수가 총 몇 개 있는지 구하는 프로그램을 작성하시오. 0으로 시작하는 수는 계단수가 아니다.

---

## 입력
첫째 줄에 N이 주어진다. N은 1보다 크거나 같고, 100보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 정답을 1,000,000,000으로 나눈 나머지를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
계단 수는 인접한 자리의 차이가 1인 수로 정의된다. 이를 해결하기 위해 다이나믹 프로그래밍을 사용하며, 각 숫자의 자리수, 숫자, 그리고 0부터 9까지의 숫자가 등장했는지를 비트마스킹을 통해 표현하여 길이 N의 계단 수를 계산한다. 최종적으로 모든 숫자가 적어도 한 번 등장하는 경우의 수를 모아서 결과를 도출한다.

### 동작 과정
1. **초기화**: 길이 1의 계단 수는 1~9로 시작할 수 있으므로 `dp[1][j][1 << j] = 1`로 설정한다. 여기서 `j`는 숫자, `1 << j`는 현재 숫자 `j`가 등장했음을 비트로 나타낸다.
   
2. **상태 전이**:
   - 길이 `i`의 경우를 고려하여 각 숫자 `j`와 비트 마스크 `k`에 대해 다음 자리수로 이동할 수 있는 경우를 계산한다.
   - `j-1`로 이동할 경우와 `j+1`로 이동할 경우를 각각 처리하며, 새로운 숫자와 비트 마스크를 업데이트한다.
   
3. **결과 계산**: 길이 `n`의 계단 수에서 모든 숫자가 등장한 경우를 찾기 위해 `dp[n][j][(1 << 10) - 1]`을 모두 합산하여 최종 결과를 도출하며, 1,000,000,000으로 나눈 나머지를 출력한다.

**예시 입력**: N = 3
- 길이 1의 계단 수: 1, 2, 3, 4, 5, 6, 7, 8, 9
- 길이 2의 계단 수: 12, 21, 23, 32, 34, 43, ..., 89
- 길이 3의 계단 수: 123, 321, 234, 432, ..., 987
- 각 경우에서 비트 마스크를 업데이트하며 모든 숫자가 포함된 경우를 찾아 최종적으로 합산.

### 시간 복잡도
O(N * 10 * 1024) = O(N * 10^3)  
여기서 N은 숫자의 길이, 10은 가능한 숫자의 수, 1024는 비트 마스킹을 통한 상태 수이다.

## 코드
```python
import sys
input = sys.stdin.readline

MOD = 1_000_000_000
n = int(input())
dp = [[[0] * 1024 for _ in range(10)] for _ in range(n + 1)]

# 길이 1의 계단 수 초기화
for j in range(1, 10):
    dp[1][j][1 << j] = 1

for i in range(1, n):
    for j in range(10):
        for k in range(1 << 10):
            if dp[i][j][k] == 0:
                continue
            
            # Case 1: 다음 숫자가 j-1일 경우
            if j > 0:
                new_digit = j - 1
                new_mask = k | (1 << new_digit)
                dp[i + 1][new_digit][new_mask] = (dp[i + 1][new_digit][new_mask] + dp[i][j][k]) % MOD
            
            # Case 2: 다음 숫자가 j+1일 경우
            if j < 9:
                new_digit = j + 1
                new_mask = k | (1 << new_digit)
                dp[i + 1][new_digit][new_mask] = (dp[i + 1][new_digit][new_mask] + dp[i][j][k]) % MOD

result = 0
for j in range(10):
    result = (result + dp[n][j][(1 << 10) - 1]) % MOD
    
print(result)
```
