## 문제
조규현과 백승환은 터렛에 근무하는 직원이다. 하지만 워낙 존재감이 없어서 인구수는 차지하지 않는다. 다음은 조규현과 백승환의 사진이다.



이석원은 조규현과 백승환에게 상대편 마린(류재명)의 위치를 계산하라는 명령을 내렸다. 조규현과 백승환은 각각 자신의 터렛 위치에서 현재 적까지의 거리를 계산했다.

조규현의 좌표 (x1,y1)$(x_1, y_1)$와 백승환의 좌표 (x2,y2)$(x_2, y_2)$가 주어지고, 조규현이 계산한 류재명과의 거리 r1$r_1$과 백승환이 계산한 류재명과의 거리 r2$r_2$가 주어졌을 때, 류재명이 있을 수 있는 좌표의 수를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T$T$가 주어진다. 각 테스트 케이스는 다음과 같이 이루어져 있다.

한 줄에 공백으로 구분 된 여섯 정수 x1$x_1$, y1$y_1$, r1$r_1$, x2$x_2$, y2$y_2$, r2$r_2$가 주어진다.

---

## 출력
각 테스트 케이스마다 류재명이 있을 수 있는 위치의 수를 출력한다. 만약 류재명이 있을 수 있는 위치의 개수가 무한대일 경우에는 −1$-1$ 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 점과 그 점에서의 원의 위치를 고려하여 류재명이 존재할 수 있는 위치의 수를 구하는 문제입니다. 두 점에서 각각의 반지름을 가진 원이 어떻게 서로 교차하는지를 파악하여, 류재명이 있을 수 있는 가능성을 판단합니다. 원이 서로 동심원인 경우, 무한대의 교차점이 존재할 수 있으며, 원의 위치와 반지름에 따라 0, 1, 2개의 교차점이 생길 수 있습니다.

### 동작 과정
1. 입력으로 주어진 테스트 케이스의 개수를 받습니다.
2. 각 테스트 케이스에 대해 두 점의 좌표와 반지름을 입력받습니다.
3. 두 점 사이의 유클리드 거리를 계산합니다.
4. 다음과 같은 조건을 통해 류재명의 위치 수를 판단합니다:
   - 두 점의 위치가 같고 반지름도 같으면, 원이 동일하여 무한대의 교차점이 생기므로 -1을 출력합니다.
   - 두 원의 반지름 차와 거리, 두 반지름의 합과 거리가 일치하는 경우는 각각 1개의 교차점이 존재합니다.
   - 두 원의 반지름 차가 거리보다 작고, 거리보다 두 반지름의 합이 큰 경우는 2개의 교차점이 존재합니다.
   - 그 외의 경우에는 교차점이 존재하지 않으므로 0을 출력합니다.

예시 입력: 
```
1
0 0 1 0 2 1
```
- 두 점 (0, 0)과 (0, 2) 사이의 거리 계산: 2
- 두 원의 반지름 합: 1 + 1 = 2
- 반지름 차: |1 - 1| = 0
- 조건에 따라 1개의 교차점이 존재하므로 출력: 1

### 시간 복잡도
이 알고리즘은 각 테스트 케이스에 대해 상수 시간 O(1) 내에 수행되므로, 전체 시간 복잡도는 O(T)입니다. 여기서 T는 테스트 케이스의 개수입니다.

## 코드
```python
import math

t = int(input())

for _ in range(t):
    x1, y1, r1, x2, y2, r2 = map(int, input().split())
    distance = math.sqrt((x1 - x2) ** 2 + (y1 - y2) ** 2)
    
    if distance == 0 and r1 == r2 :  
        print(-1)
    elif abs(r1 - r2) == distance or r1 + r2 == distance: 
        print(1)
    elif abs(r1 - r2) < distance < (r1 + r2) :  
        print(2)
    else:
        print(0) 
```
