## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 MenOfPassion 알고리즘에서 특정 코드의 수행 횟수를 계산하는 것입니다. 주어진 입력 크기 `n`에 대해, `n`개의 요소 중 3개를 선택하는 조합의 수를 구해야 하며, 이는 수학적으로 `C(n, 3) = n * (n - 1) * (n - 2) / 6`으로 표현됩니다. 이 값을 계산하고, 다항식으로 나타낼 경우 최고차항의 차수는 3이므로 이를 출력합니다.

### 동작 과정
1. 입력으로 `n`을 받습니다. 예를 들어, `n = 5`라고 가정합시다.
2. `C(5, 3)`을 계산합니다:
   - `C(5, 3) = 5 * (5 - 1) * (5 - 2) / 6`
   - `C(5, 3) = 5 * 4 * 3 / 6 = 10`
3. 따라서 코드1의 수행 횟수는 10입니다.
4. 다항식으로 나타낼 경우, 최고차항은 3이므로 3을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 크기에 관계없이 단순히 수학적 계산을 수행하기 때문에 입력 크기와 무관한 상수 시간에 완료됩니다.
