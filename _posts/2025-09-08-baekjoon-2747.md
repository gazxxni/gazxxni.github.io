## 풀이

### 풀이 핵심 로직
이 문제는 피보나치 수열을 계산하는 문제로, 피보나치 수열은 각 수가 이전 두 수의 합으로 정의됩니다. 동적 프로그래밍(Dynamic Programming) 기법을 사용하여, 이전에 계산된 값을 저장함으로써 중복 계산을 피하고 효율적으로 n번째 피보나치 수를 구합니다.

### 동작 과정
1. 사용자로부터 정수 n을 입력받습니다.
2. 길이가 n+1인 dp 리스트를 생성하고, dp[0]을 0으로, dp[1]을 1로 초기화합니다.
3. 2부터 n까지 반복하며, 각 dp[i]에 대해 dp[i-1]과 dp[i-2]의 합을 계산하여 저장합니다.
   - 예를 들어, n이 5일 경우:
     - dp[2] = dp[1] + dp[0] = 1 + 0 = 1
     - dp[3] = dp[2] + dp[1] = 1 + 1 = 2
     - dp[4] = dp[3] + dp[2] = 2 + 1 = 3
     - dp[5] = dp[4] + dp[3] = 3 + 2 = 5
4. 마지막으로 dp[n]을 출력하여 n번째 피보나치 수를 반환합니다.

### 시간 복잡도
O(n) - n까지의 피보나치 수를 계산하기 위해 n번의 반복을 수행하기 때문에 시간 복잡도는 선형적입니다.

---

## 코드
```python
n = int(input())

dp = [0] * (n + 1)
dp[0] = 0
if n >= 1:
    dp[1] = 1

for i in range(2, n + 1):
    dp[i] = dp[i - 1] + dp[i - 2]

print(dp[n])

```
