## 문제
학교 근처 편의점에 새 초콜릿이 들어왔다. 이 초콜릿은 막대 모양이고, 각 막대는 정사각형 N개로 이루어져 있다. 초콜릿의 크기(정사각형의 개수)는 항상 2의 제곱 형태이다. 즉, 1, 2, 4, 8, 16, ...개의 정사각형으로 이루어져 있다.

상근이는 점심식사로 초콜릿을 먹는다. 이때, 적어도 K개 정사각형을 먹어야 남은 수업을 졸지 않고 버틸 수 있다. 상근이의 친구 선영이도 초콜릿을 좋아한다. 선영이는 초콜릿은 돈을 주고 사기 아깝다고 생각하기 때문에, 상근이가 주는 초콜릿만 먹는다.

상근이는 막대 초콜릿를 하나 산 다음에, 정확하게 K개 정사각형이 되도록 초콜릿을 쪼갠다. K개는 자신이 먹고 남는 것은 선영이에게 준다.

막대 초콜릿은 나누기 조금 어렵게 되어 있어서, 항상 가운데로만 쪼개진다. 즉, 정사각형이 D개 있는 막대는 D/2개 막대 두 조각으로 쪼개진다.

K개 정사각형을 만들기 위해서, 최소 몇 번 초콜릿을 쪼개야 하는지와 사야하는 가장 작은 초콜릿의 크기를 구하는 프로그램을 작성하시오. 상근이는 초콜릿을 하나만 살 수 있다. 꼭 한 조각이 K개일 필요는 없고, 여러 조각에 있는 정사각형을 합쳤을 때 K개이면 된다.

---

## 입력
첫째 줄에 K가 주어진다. (1 ≤ K ≤ 1,000,000)

---

## 출력
첫째 줄에는 상근이가 구매해야하는 가장 작은 초콜릿의 크기와 최소 몇 번 쪼개야 하는지를 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 K개 이상의 정사각형을 만들기 위해 필요한 최소한의 초콜릿 크기와 쪼개는 횟수를 구하는 것입니다. 초콜릿의 크기는 항상 2의 제곱형태이므로, K보다 크거나 같은 최소의 2의 제곱수를 찾고, 그 초콜릿을 K개 정사각형으로 나누기 위한 쪼개기 횟수를 계산합니다.

### 동작 과정
1. **초콜릿 크기 찾기**: K의 값에 대해, K보다 크거나 같은 최소의 2의 제곱수를 찾습니다. 이를 위해 a를 1로 초기화하고, a가 K보다 작을 동안 2배씩 증가시킵니다.
   - 예를 들어, K=20인 경우:
     - a=1 (1개), a=2 (2개), a=4 (4개), a=8 (8개), a=16 (16개) → a=32 (32개)에서 종료.
     - 따라서, 가장 작은 초콜릿 크기는 32입니다.

2. **초콜릿 쪼개기**: 초기 크기 a에서 시작하여, K보다 클 경우 K에서 a를 빼고, K가 a보다 작을 경우 a를 반으로 나누고 쪼개기 횟수를 증가시킵니다.
   - 위 예시에서:
     - K=20, a=32에서 시작.
     - 32 >= 20 → K=20 - 32 = -12 (여기서는 a를 줄이지 않고 계속 진행)
     - a=16으로 나눈 후 K=20 - 16 = 4.
     - a=8으로 나눈 후 K=4 - 8 = -4.
     - a=4로 나눈 후 K=0 (모든 정사각형을 만들었으므로 종료).

3. **결과 출력**: 초콜릿의 최소 크기와 쪼개기 횟수를 출력합니다.
   - 최종적으로 초콜릿 크기(32)와 쪼개기 횟수(3)를 출력합니다.

### 시간 복잡도
O(log K) - K의 크기에 따라 초콜릿의 크기를 찾는 과정은 로그 시간에 걸쳐 진행됩니다.

## 코드
```python
k = int(input())

cnt = 0
a = 1
while True:
  if a >= k:
      break
  
  a *= 2
 

cnt2 = 0
ans = a
while k > 0:
    if k >= a:
        k -= a
    else:
        a //= 2
        cnt2 += 1
print(ans, cnt2)
```
