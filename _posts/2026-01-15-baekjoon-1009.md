## 문제
재용이는 최신 컴퓨터 10대를 가지고 있다. 어느 날 재용이는 많은 데이터를 처리해야 될 일이 생겨서 각 컴퓨터에 1번부터 10번까지의 번호를 부여하고, 10대의 컴퓨터가 다음과 같은 방법으로 데이터들을 처리하기로 하였다.

1번 데이터는 1번 컴퓨터, 2번 데이터는 2번 컴퓨터, 3번 데이터는 3번 컴퓨터, ... ,

10번 데이터는 10번 컴퓨터, 11번 데이터는 1번 컴퓨터, 12번 데이터는 2번 컴퓨터, ...

총 데이터의 개수는 항상 ab개의 형태로 주어진다. 재용이는 문득 마지막 데이터가 처리될 컴퓨터의 번호가 궁금해졌다. 이를 수행해주는 프로그램을 작성하라.

## 풀이

### 풀이 핵심 로직
이 문제는 특정 숫자의 거듭제곱의 마지막 자리 숫자를 찾는 문제로, 10개의 컴퓨터에 데이터를 분산 처리하는 방식입니다. 주어진 `a`의 `b` 거듭제곱을 계산할 때, 마지막 자리 숫자만 필요하므로 `a`를 10으로 나눈 나머지를 이용하여 패턴을 찾고, 이를 통해 `b`에 따라 적절한 마지막 자리 숫자를 결정합니다.

### 동작 과정
1. 입력으로 주어진 `a`와 `b`를 읽습니다.
2. `a`의 마지막 자리 숫자를 구하기 위해 `base = a % 10`을 계산합니다.
3. 마지막 자리 숫자에 따른 패턴을 확인하여 결과를 결정합니다:
   - 만약 `base`가 0이면, 결과는 항상 10입니다.
   - `base`가 1, 5, 6이면, 마지막 자리 숫자는 항상 자기 자신입니다.
   - `base`가 4 또는 9일 경우, `b`의 홀짝 여부에 따라서 결과가 달라집니다.
   - 나머지 경우(2, 3, 7, 8)에서는 `b % 4`를 이용하여 마지막 자리 숫자를 결정합니다.
4. 모든 테스트 케이스에 대해 이 과정을 반복하고 결과를 출력합니다.

**예시 입력**:
```
3
1 10
2 5
3 3
```

**단계별 설명**:
- **1 10**: `1`의 `10` 제곱은 `1`이므로 결과는 `1`.
- **2 5**: `2`의 `5` 제곱은 `32`, 마지막 자리 숫자는 `2`이므로 결과는 `2`.
- **3 3**: `3`의 `3` 제곱은 `27`, 마지막 자리 숫자는 `7`이므로 결과는 `3`.

**출력**:
```
1
2
7
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(T)입니다. 여기서 T는 테스트 케이스의 수입니다. 각 테스트 케이스에 대해 상수 시간 내에 결과를 계산할 수 있기 때문입니다.

## 코드
```python
import sys

t = int(sys.stdin.readline())

for _ in range(t):
    a, b = map(int, sys.stdin.readline().split())
    
    base = a % 10
    
    if base == 0:
        print(10)
    elif base in [1, 5, 6]:
        print(base)
    elif base in [4, 9]:
        remain = b % 2
        if remain == 0:
            print((base ** 2) % 10)
        else:
            print(base)
    else:
        remain = b % 4
        if remain == 0:
            print((base ** 4) % 10)
        else:
            print((base ** remain) % 10)
```
