## 문제
서강대학교 컴퓨터공학과 실습실 R912호에는 현재 N개의 창문이 있고 또 N명의 사람이 있다. 1번째 사람은 1의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다.  2번째 사람은 2의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다. 이러한 행동을 N번째 사람까지 진행한 후 열려 있는 창문의 개수를 구하라. 단, 처음에 모든 창문은 닫혀 있다.

예를 들어 현재 3개의 창문이 있고 3명의 사람이 있을 때,


	1번째 사람은 1의 배수인 1,2,3번 창문을 연다. (1, 1, 1)
	2번째 사람은 2의 배수인 2번 창문을 닫는다. (1, 0, 1)
	3번째 사람은 3의 배수인 3번 창문을 닫는다. (1, 0, 0)


결과적으로 마지막에 열려 있는 창문의 개수는 1개 이다.

---

## 입력
첫 번째 줄에는 창문의 개수와 사람의 수 N(1 ≤ N ≤ 2,100,000,000)이 주어진다.

---

## 출력
마지막에 열려 있는 창문의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 창문이 열리고 닫히는 과정을 수학적으로 분석하는 것이 핵심입니다. 각 창문은 그 번호의 약수 개수만큼 열리고 닫히는데, 홀수 개의 약수가 있는 경우에만 창문이 열려 있습니다. 홀수 개의 약수는 완전 제곱수에서만 발생하므로, N개의 창문 중 완전 제곱수의 개수만큼의 창문이 열려 있게 됩니다. 따라서, 열려 있는 창문의 개수는 N의 제곱근을 취한 정수 부분과 같습니다.

### 동작 과정
1. **입력**: N = 3일 때, 창문과 사람의 수는 각각 3입니다.
2. **1번째 사람**: 1의 배수인 1, 2, 3번 창문을 열기 때문에 (1, 1, 1)로 상태가 변경됩니다.
3. **2번째 사람**: 2의 배수인 2번 창문을 닫기 때문에 (1, 0, 1)로 상태가 변경됩니다.
4. **3번째 사람**: 3의 배수인 3번 창문을 닫기 때문에 (1, 0, 0)으로 최종 상태가 됩니다.
5. **결과 확인**: 마지막에 열려 있는 창문은 1개입니다.

이 과정을 통해 N = 3일 때 열려 있는 창문의 개수는 1개임을 알 수 있습니다.

### 시간 복잡도
O(1) - 입력 N에 대해 상수 시간 내에 결과를 계산할 수 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
print(int(n ** 0.5))
```
