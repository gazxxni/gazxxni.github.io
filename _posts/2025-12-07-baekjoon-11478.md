## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

## 성능 요약

## 풀이

### 풀이 핵심 로직
문자열 S의 서로 다른 부분 문자열을 구하기 위해, 모든 가능한 길이의 부분 문자열을 생성하여 집합(set)에 추가합니다. 집합은 중복된 값을 허용하지 않기 때문에, 최종적으로 집합의 크기를 세면 서로 다른 부분 문자열의 개수를 구할 수 있습니다.

### 동작 과정
1. 입력: 예를 들어, 문자열 S가 "ababc"라고 가정합니다.
2. 부분 문자열 생성:
   - 길이가 1인 부분 문자열: `a`, `b`, `a`, `b`, `c` → 집합에 추가 → `{'a', 'b', 'c'}`
   - 길이가 2인 부분 문자열: `ab`, `ba`, `ab`, `bc` → 집합에 추가 → `{'a', 'b', 'c', 'ab', 'ba', 'bc'}`
   - 길이가 3인 부분 문자열: `aba`, `bab`, `abc` → 집합에 추가 → `{'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc'}`
   - 길이가 4인 부분 문자열: `abab`, `babc` → 집합에 추가 → `{'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc', 'abab', 'babc'}`
   - 길이가 5인 부분 문자열: `ababc` → 집합에 추가 → `{'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc', 'abab', 'babc', 'ababc'}`
3. 최종 집합의 크기를 계산하여 서로 다른 부분 문자열의 개수를 출력합니다. 이 예시에서는 12가 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 이는 n이 문자열 S의 길이일 때, 각 길이 i(1부터 n까지)에 대해 최대 n-i+1개의 부분 문자열을 생성해야 하기 때문입니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
