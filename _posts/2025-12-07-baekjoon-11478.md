## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
문자열 S의 서로 다른 부분 문자열을 구하기 위해, 길이가 1부터 S의 길이까지의 모든 부분 문자열을 생성하고 이를 집합(set)에 저장합니다. 집합은 중복된 값을 허용하지 않으므로 서로 다른 부분 문자열의 개수를 쉽게 카운트할 수 있습니다.

### 동작 과정
1. 입력으로 문자열 "ababc"를 받습니다.
2. 부분 문자열의 길이를 1부터 시작하여 최대 S의 길이(5)까지 반복합니다.
   - 길이 1: 'a', 'b', 'c'가 생성됩니다. (현재 집합: {'a', 'b', 'c'})
   - 길이 2: 'ab', 'ba', 'ab', 'bc'가 생성됩니다. (현재 집합: {'a', 'b', 'c', 'ab', 'ba', 'bc'})
   - 길이 3: 'aba', 'bab', 'abc'가 생성됩니다. (현재 집합: {'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc'})
   - 길이 4: 'abab', 'babc'가 생성됩니다. (현재 집합: {'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc', 'abab', 'babc'})
   - 길이 5: 'ababc'가 생성됩니다. (현재 집합: {'a', 'b', 'c', 'ab', 'ba', 'bc', 'aba', 'bab', 'abc', 'abab', 'babc', 'ababc'})
3. 최종적으로 집합에 저장된 서로 다른 부분 문자열의 개수는 12개입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 이는 n이 문자열의 길이일 때, 각 길이에 대해 모든 가능한 시작 인덱스를 반복하므로 n(n+1)/2의 복잡도를 가지기 때문입니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
