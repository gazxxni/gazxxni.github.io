## 문제
오늘도 서준이는 점근적 표기 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

알고리즘의 소요 시간을 나타내는 O-표기법(빅-오)을 다음과 같이 정의하자.

O(g(n)) = {f(n) | 모든 n ≥ n0에 대하여 f(n) ≤ c × g(n)인 양의 상수 c와 n0가 존재한다}

이 정의는 실제 O-표기법(https://en.wikipedia.org/wiki/Big_O_notation)과 다를 수 있다.

함수 f(n) = a1n + a0, 양의 정수 c, n0가 주어질 경우 O(n) 정의를 만족하는지 알아보자.

---

## 입력
첫째 줄에 함수 f(n)을 나타내는 정수 a1, a0가 주어진다. (0 ≤ |ai| ≤ 100)

다음 줄에 양의 정수 c가 주어진다. (1 ≤ c ≤ 100)

다음 줄에 양의 정수 n0가 주어진다. (1 ≤ n0 ≤ 100)

---

## 출력
f(n), c, n0가 O(n) 정의를 만족하면 1, 아니면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 함수 \( f(n) = a_1 n + a_0 \)이 주어진 상수 \( c \)와 \( n_0 \)에 대해 점근적 표기 O(n)의 정의를 만족하는지를 판단하는 것입니다. 즉, 모든 \( n \geq n_0 \)에 대해 \( f(n) \leq c \times n \)이 성립하는지를 확인해야 합니다. 이를 위해, \( f(n) \)의 값과 \( g(n) = c \times n \)의 값을 비교하여 조건을 만족하는지를 확인합니다.

### 동작 과정
1. 입력으로 \( a_1, a_0 \)가 주어집니다. 예를 들어, \( a_1 = 2, a_0 = 3 \)이라고 가정하겠습니다.
2. 다음으로 상수 \( c \)가 주어집니다. 예를 들어, \( c = 5 \)라고 가정합니다.
3. 마지막으로 \( n_0 \)가 주어집니다. 예를 들어, \( n_0 = 10 \)이라고 가정합니다.
4. 이제 \( n \)을 \( n_0 \)부터 시작하여 증가시키며 \( f(n) \)와 \( g(n) \)를 비교합니다.
   - \( n = 10 \)일 때, \( f(10) = 2 \times 10 + 3 = 23 \)이고 \( g(10) = 5 \times 10 = 50 \)입니다. 
   - \( 23 \leq 50 \)이므로 조건을 만족합니다.
5. \( a_1 \leq c \) 조건도 확인합니다. 여기서 \( 2 \leq 5 \)이므로 이 조건도 만족합니다.
6. 따라서 출력은 1입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 값에 따라 계산이 고정된 수의 연산으로 이루어지기 때문입니다.

## 코드
```python
a1, a0 = map(int, input().split())
c = int(input())
n = int(input())

f = a1 * n + a0
g = c * n

if f <= g and  a1 <= c:
    print(1)

else:
    print(0)
```
