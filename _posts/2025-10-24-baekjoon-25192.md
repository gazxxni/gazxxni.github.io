## 풀이

### 풀이 핵심 로직
이 문제는 여러 사용자가 채팅방에 들어와서 메시지를 보낼 때, 'ENTER'라는 메시지가 들어오면 현재 채팅방에 있는 사용자 수를 세고, 다음 'ENTER'가 올 때까지 새로운 메시지를 추가하는 구조입니다. 각 'ENTER' 메시지가 들어올 때마다 현재 사용자 수를 기록하고, 이후 사용자 목록을 초기화하여 다음 세션을 시작합니다.

### 동작 과정
1. **입력**: 예를 들어, 입력이 다음과 같다고 가정합니다.
   ```
   7
   A
   B
   ENTER
   C
   D
   ENTER
   E
   ```
   
2. **초기화**: `cnt`는 0으로 초기화하고, 사용자 목록을 담을 `arr`는 빈 집합으로 초기화합니다.

3. **1st Iteration**: `A` 입력 -> `arr`에 `A` 추가 (`arr` = {'A'}).
   
4. **2nd Iteration**: `B` 입력 -> `arr`에 `B` 추가 (`arr` = {'A', 'B'}).
   
5. **3rd Iteration**: `ENTER` 입력 -> `cnt`에 `arr`의 크기(2)를 추가하고, `arr` 초기화 (`cnt` = 2, `arr` = {}).
   
6. **4th Iteration**: `C` 입력 -> `arr`에 `C` 추가 (`arr` = {'C'}).
   
7. **5th Iteration**: `D` 입력 -> `arr`에 `D` 추가 (`arr` = {'C', 'D'}).
   
8. **6th Iteration**: `ENTER` 입력 -> `cnt`에 `arr`의 크기(2)를 추가하고, `arr` 초기화 (`cnt` = 4, `arr` = {}).
   
9. **7th Iteration**: `E` 입력 -> `arr`에 `E` 추가 (`arr` = {'E'}).
   
10. **마무리**: 모든 입력을 처리한 후, 최종적으로 `cnt`에 `arr`의 크기(1)를 추가하여 (`cnt` = 5) 출력.

최종 출력: `5`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 각 입력에 대해 한 번씩 처리하며, 집합에 원소를 추가하는 작업은 평균적으로 O(1) 시간에 수행됩니다. 따라서 전체적으로 n개의 입력을 처리하는데 O(n)의 시간이 소요됩니다.

---

## 코드
```python
n = int(input())

cnt = 0
arr = set()
for _ in range(n):
    a = input().rstrip()
    if a == 'ENTER':
        cnt += len(arr)
        arr = set()
        continue
    
    arr.add(a)

cnt += len(arr)
print(cnt)
```
