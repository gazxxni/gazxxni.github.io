[https://www.acmicpc.net/problem/4949](https://www.acmicpc.net/problem/4949)

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 문자열에서 괄호의 짝이 맞는지를 확인하는 문제입니다. 스택 자료구조를 활용하여 여는 괄호(`(`, `[`)는 스택에 추가하고, 닫는 괄호(`)`, `]`)가 나올 때 스택의 마지막 원소와 비교하여 짝이 맞는지 확인합니다. 모든 문자를 처리한 후 스택이 비어 있다면 괄호가 짝이 맞는 것이고, 비어 있지 않다면 짝이 맞지 않는 것입니다.

### 동작 과정
1. 입력 문자열이 `.`인 경우 반복문을 종료합니다.
2. 각 문자열에 대해 빈 스택을 생성합니다.
3. 문자열의 각 문자를 순회하며:
   - 여는 괄호(`(`, `[`)는 스택에 추가합니다.
   - 닫는 괄호(`)`, `]`)가 나오면:
     - 스택이 비어있지 않고, 스택의 마지막 원소가 짝이 맞는 여는 괄호라면 스택에서 그 여는 괄호를 제거합니다.
     - 그렇지 않으면 해당 닫는 괄호를 스택에 추가하고 반복문을 종료합니다.
4. 문자열의 모든 문자를 처리한 후, 스택이 비어 있으면 "yes"를 출력하고, 그렇지 않으면 "no"를 출력합니다.

예시 입력:
```
[([])]
```
- 첫 문자 `[`가 여는 괄호이므로 스택에 추가: `['[']`
- 두 번째 문자 `(`가 여는 괄호이므로 스택에 추가: `['[', '(']`
- 세 번째 문자 `[`가 여는 괄호이므로 스택에 추가: `['[', '(', '[']`
- 네 번째 문자 `]`가 나오면 스택의 마지막 원소는 `[`이므로 짝이 맞아 스택에서 제거: `['[', '(']`
- 다섯 번째 문자 `)`가 나오면 스택의 마지막 원소는 `(`이므로 짝이 맞아 스택에서 제거: `['[']`
- 여섯 번째 문자 `]`가 나오면 스택의 마지막 원소는 `[`이므로 짝이 맞아 스택에서 제거: `[]`
- 최종적으로 스택이 비어 있으므로 "yes"를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 여기서 n은 입력 문자열의 길이입니다. 각 문자를 한 번씩만 검사하고 스택에 대한 연산(추가 및 제거)은 상수 시간에 이루어지므로 전체적으로 선형 시간 복잡도를 가집니다.

## 코드
```python
import sys

while True:
    a=input()
    stack =[]

    if a==".":
        break

    for i in a:
        if i=='[' or i == '(':
            stack.append(i)
        elif i == ']':
            if len(stack) !=0 and stack[-1] == '[':
                stack.pop()
            else:
                stack.append(']')
                break
        elif i==')':
            if len(stack)!=0 and stack[-1]=='(':
                stack.pop()
            else:
                stack.append(')')
                break
        
    if len(stack)==0:
        print('yes')
    else:
        print('no')


```
