## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 수 N의 각 자리수를 분리하여 리스트로 만든 후, 해당 리스트를 내림차순으로 정렬하여 결과를 출력하는 방법이다. 문자열 형태로 입력받은 N의 각 문자를 정수로 변환하고, 이를 정렬한 뒤 다시 조합하여 최종 결과를 만들어낸다.

### 동작 과정
1. **입력 처리**: 첫째 줄에 주어진 수 N을 문자열로 입력받고, 공백을 제거하여 `n`에 저장한다.
   - 예시 입력: `2143`
   
2. **자리수 분리**: `map(int, n)`을 사용하여 각 자리수를 정수로 변환하고, 이를 리스트 `a`에 저장한다.
   - `a`의 결과: `[2, 1, 4, 3]`
   
3. **정렬**: 리스트 `a`를 `sort(reverse=True)`를 통해 내림차순으로 정렬한다.
   - 정렬된 `a`: `[4, 3, 2, 1]`
   
4. **출력**: `print(*a, sep='')`를 사용하여 리스트의 요소를 공백 없이 이어붙여 출력한다.
   - 최종 출력: `4321`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(m log m)이다. 여기서 m은 N의 자리수의 수이다. 정렬 연산에 소요되는 시간 때문이며, 최대 N이 1,000,000,000인 경우 m은 10이므로 실질적으로 매우 빠르게 실행된다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
