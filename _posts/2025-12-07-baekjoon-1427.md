## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 수 N의 각 자리수를 분리하여 리스트에 저장한 후, 이 리스트를 내림차순으로 정렬합니다. 정렬된 자리수를 다시 하나의 숫자로 합쳐 출력하는 방식으로 문제를 해결합니다.

### 동작 과정
1. 입력으로 주어진 수 N을 문자열로 받아옵니다. 예를 들어, 입력이 `2143`이라면 `n = '2143'`가 됩니다.
2. 이 문자열을 각 자리수를 정수형으로 변환하여 리스트 `a`에 저장합니다. 이 경우 `a = [2, 1, 4, 3]`가 됩니다.
3. 리스트 `a`를 내림차순으로 정렬합니다. 정렬 후 `a`는 `[4, 3, 2, 1]`이 됩니다.
4. 정렬된 리스트의 요소를 하나의 문자열로 합쳐서 출력합니다. 최종 출력은 `4321`입니다.

### 시간 복잡도
입력된 수 N의 자리수 수를 d라고 할 때, 리스트를 정렬하는 데 필요한 시간 복잡도는 O(d log d)입니다. N의 최대값인 1,000,000,000에서 d는 최대 10이므로, 이 문제의 시간 복잡도는 O(10 log 10)으로 상수 시간에 가까운 효율성을 가집니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
