## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 정수 x가 나타내는 위치를 찾기 위해 각 대각선(line)을 순차적으로 탐색하면서 x의 값을 줄여 나가는 것입니다. 각 대각선의 항목 수(line)를 계산하여 x가 어느 대각선에 위치하는지 파악한 후, 해당 대각선에서의 분자와 분모를 결정합니다.

### 동작 과정
1. 입력으로 x가 주어집니다. 예를 들어, x = 14라고 가정합니다.
2. `line`을 1로 초기화하고, while 루프를 통해 x가 현재 대각선의 항목 수(line)보다 큰 동안 x에서 line을 빼며 line을 증가시킵니다.
   - 1st iteration: x(14) > line(1) → x = 14 - 1 = 13, line = 2
   - 2nd iteration: x(13) > line(2) → x = 13 - 2 = 11, line = 3
   - 3rd iteration: x(11) > line(3) → x = 11 - 3 = 8, line = 4
   - 4th iteration: x(8) > line(4) → x = 8 - 4 = 4, line = 5
   - 5th iteration: x(4) > line(5) → x = 4 - 5 = -1, line = 6 (loop 종료)
3. 이제 line은 6이 되고, x는 -1입니다. 이는 x가 6번째 대각선에 속함을 의미합니다.
4. line이 짝수이므로, 분자 a는 x(= -1)이고, 분모 b는 line - x + 1(= 6 - (-1) + 1 = 8)입니다.
5. 최종적으로 a와 b를 출력하여 결과는 "1/8"이 됩니다.

### 시간 복잡도
이 알고리즘은 while 루프를 통해 line의 값을 증가시키며 x를 줄여 나가므로, x가 1일 때부터 시작하여 대각선의 개수에 따라 선형적으로 진행됩니다. 따라서 시간 복잡도는 O(√x)입니다.

## 코드
```python
import sys

input = sys.stdin.readline

x = int(input())

line = 1

# 짝수 라인 : 분모가 1씩 늘어나고 분자가 1씩 줄어듦
# 홀수 라인 : 분자가 1씩 늘어나고 분모가 1씩 줄어듦
# 각 줄에 있는 항목의 개수(line)를 빼면서 x가 어느 줄에 속하는지 찾음
while x > line:
    x -= line  # 해당 줄의 수만큼 x에서 빼줌
    line += 1  # 다음 줄로 넘어감

if line % 2 == 0:
    a = x  # 분자
    b = line - x + 1  # 분모

elif line % 2 == 1:
    a = line - x + 1  # 분자
    b = x  # 분모

print(f'{a}/{b}')

```
