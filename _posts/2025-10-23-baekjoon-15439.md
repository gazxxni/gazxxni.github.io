## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 베라가 상의와 하의를 서로 다른 색상으로 조합하는 방법의 수를 계산하는 것입니다. 상의와 하의의 색상이 서로 다르므로, 상의 색상으로 선택한 후 하의 색상은 선택한 상의 색상과 다른 색상 중에서 선택해야 합니다. 이에 따라 상의의 선택 수(N)와 하의 선택 수(N-1)를 곱하여 조합의 수를 구할 수 있습니다.

### 동작 과정
1. N을 입력받습니다. (예: N = 3)
2. 상의는 3가지 색상(1, 2, 3) 중 하나를 선택할 수 있습니다.
3. 상의 색상을 선택한 후, 해당 색상과 다른 색상을 가진 하의 중에서 선택해야 하므로 하의 선택할 수 있는 색상은 N-1개입니다.
4. 예를 들어, 상의 색상을 1로 선택하면 하의는 2 또는 3을 선택할 수 있어 2가지 경우가 가능합니다.
5. 따라서, 가능한 조합의 수는 N * (N - 1)입니다.
6. 결과적으로, N = 3일 경우, 가능한 조합의 수는 3 * (3 - 1) = 6입니다.

### 시간 복잡도
이 문제의 시간 복잡도는 O(1)입니다. 입력값 N이 주어지면 단순한 산술 연산을 통해 결과를 도출하므로, 입력 크기에 관계없이 항상 일정한 시간에 실행됩니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
