## 풀이

### 풀이 핵심 로직
이 문제에서는 나이와 이름으로 구성된 여러 명의 정보를 입력받아 나이를 기준으로 정렬한 후 출력하는 것이 목표입니다. 리스트에 나이와 이름을 저장한 후, `sorted` 함수를 사용하여 나이를 기준으로 정렬하고, 최종적으로 정렬된 리스트를 출력합니다.

### 동작 과정
1. **입력 받기**: 5명의 나이와 이름을 다음과 같이 입력한다고 가정합니다.
   ```
   5
   21 Junkyu
   21 Dohyun
   20 Sunyoung
   20 Jisoo
   22 Jiwon
   ```
   첫 번째 줄에는 5가 입력되어, n이 5로 설정됩니다.

2. **리스트 구성**: for 루프를 통해 각 줄의 정보를 읽고, 나이를 정수로 변환하여 리스트 `a`에 `[나이, 이름]` 형태로 추가합니다.
   - 리스트 `a`는 다음과 같이 구성됩니다:
     ```
     [[21, 'Junkyu'], [21, 'Dohyun'], [20, 'Sunyoung'], [20, 'Jisoo'], [22, 'Jiwon']]
     ```

3. **정렬**: `sorted` 함수를 사용하여 리스트 `a`를 나이를 기준으로 정렬합니다. 정렬 후 리스트는 다음과 같습니다:
   ```
   [[20, 'Sunyoung'], [20, 'Jisoo'], [21, 'Junkyu'], [21, 'Dohyun'], [22, 'Jiwon']]
   ```

4. **출력**: 정렬된 리스트를 순회하며 각 요소를 출력하여 최종 결과를 얻습니다:
   ```
   20 Sunyoung
   20 Jisoo
   21 Junkyu
   21 Dohyun
   22 Jiwon
   ```

### 시간 복잡도
이 코드의 시간 복잡도는 O(n log n)입니다. 여기서 n은 입력으로 주어진 사람의 수입니다. 이는 `sorted` 함수의 정렬 알고리즘이 평균적으로 O(n log n)의 시간 복잡도를 가지기 때문입니다. 입력과 출력 과정에서 O(n)의 시간이 추가로 소요되지만, 전체적으로는 정렬 과정이 지배적입니다.

## 코드
```python
import sys

n=int(input())
a=[]

for _ in range(n):
    x, y = input().split()
    a.append([int(x),y])

sorted(a,key=lambda x : x[0])

for i in a:
    print(i[0],i[1])
```
