## 문제
위와 같은 삼각형의 꼭대기에서 바닥까지 이어지는 경로 중, 거쳐간 숫자의 합이 가장 큰 경우를 찾아보려고 합니다. 아래 칸으로 이동할 때는 대각선 방향으로 한 칸 오른쪽 또는 왼쪽으로만 이동 가능합니다. 예를 들어 3에서는 그 아래칸의 8 또는 1로만 이동이 가능합니다.

삼각형의 정보가 담긴 배열 triangle이 매개변수로 주어질 때, 거쳐간 숫자의 최댓값을 return 하도록 solution 함수를 완성하세요.

제한사항


삼각형의 높이는 1 이상 500 이하입니다.
삼각형을 이루고 있는 숫자는 0 이상 9,999 이하의 정수입니다.


입출력 예

        
triangle
result


        
[[7], [3, 8], [8, 1, 0], [2, 7, 4, 4], [4, 5, 2, 6, 5]]
30

## 풀이
### 풀이 핵심 로직
이 문제는 동적 계획법을 사용하여 해결할 수 있습니다. 각 단계에서 현재 위치의 값을 이전 단계에서 가능한 최대 합으로 업데이트하며, 이를 통해 바닥까지 도달할 때의 최대 합을 구합니다. 구한 최대 합은 항상 해당 위치에서 가능한 경로의 최대 합으로 구성됩니다.

### 동작 과정
주어진 삼각형을 예로 들어 단계별로 설명하겠습니다.

입력: 
```
triangle = [[7], 
            [3, 8], 
            [8, 1, 0], 
            [2, 7, 4, 4], 
            [4, 5, 2, 6, 5]]
```

1. **첫 번째 단계** (i=1):
   - triangle[1][0] = 3 → triangle[1][0] + triangle[0][0] = 3 + 7 = 10
   - triangle[1][1] = 8 → triangle[1][1] + triangle[0][0] = 8 + 7 = 15
   - 결과: `[[7], [10, 15], ...]`

2. **두 번째 단계** (i=2):
   - triangle[2][0] = 8 → triangle[2][0] + triangle[1][0] = 8 + 10 = 18
   - triangle[2][1] = 1 → triangle[2][1] + max(triangle[1][0], triangle[1][1]) = 1 + max(10, 15) = 16
   - triangle[2][2] = 0 → triangle[2][2] + triangle[1][1] = 0 + 15 = 15
   - 결과: `[[7], [10, 15], [18, 16, 15], ...]`

3. **세 번째 단계** (i=3):
   - triangle[3][0] = 2 → triangle[3][0] + triangle[2][0] = 2 + 18 = 20
   - triangle[3][1] = 7 → triangle[3][1] + max(triangle[2][0], triangle[2][1]) = 7 + max(18, 16) = 25
   - triangle[3][2] = 4 → triangle[3][2] + max(triangle[2][1], triangle[2][2]) = 4 + max(16, 15) = 20
   - triangle[3][3] = 4 → triangle[3][3] + triangle[2][2] = 4 + 15 = 19
   - 결과: `[[7], [10, 15], [18, 16, 15], [20, 25, 20, 19], ...]`

4. **네 번째 단계** (i=4):
   - triangle[4][0] = 4 → triangle[4][0] + triangle[3][0] = 4 + 20 = 24
   - triangle[4][1] = 5 → triangle[4][1] + max(triangle[3][0], triangle[3][1]) = 5 + max(20, 25) = 30
   - triangle[4][2] = 2 → triangle[4][2] + max(triangle[3][1], triangle[3][2]) = 2 + max(25, 20) = 27
   - triangle[4][3] = 6 → triangle[4][3] + max(triangle[3][2], triangle[3][3]) = 6 + max(20, 19) = 26
   - triangle[4][4] = 5 → triangle[4][4] + triangle[3][3] = 5 + 19 = 24
   - 결과: `[[7], [10, 15], [18, 16, 15], [20, 25, 20, 19], [24, 30, 27, 26, 24]]`

마지막 단계에서 가장 큰 값은 30입니다. 따라서, 반환값은 30입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 여기서 n은 삼각형의 높이입니다. 각 레벨을 순회하면서 각 레벨의 요소를 한 번씩 업데이트하기 때문에 이와 같은 복잡도가 발생합니다.

## 코드
```python
def solution(triangle):
    answer = 0
    
    for i in range(1, len(triangle)):
        for j in range(1, len(triangle[i]) - 1):
            triangle[i][j] += max(triangle[i-1][j], triangle[i-1][j-1])
        
        triangle[i][0] += triangle[i-1][0]
        triangle[i][-1] += triangle[i-1][-1]
        
    return max(triangle[-1])
```
