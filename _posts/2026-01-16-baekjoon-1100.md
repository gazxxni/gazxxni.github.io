## 문제
체스판은 8×8크기이고, 검정 칸과 하얀 칸이 번갈아가면서 색칠되어 있다. 가장 왼쪽 위칸 (0,0)은 하얀색이다. 체스판의 상태가 주어졌을 때, 하얀 칸 위에 말이 몇 개 있는지 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄부터 8개의 줄에 체스판의 상태가 주어진다. ‘.’은 빈 칸이고, ‘F’는 위에 말이 있는 칸이다.

---

## 출력
첫째 줄에 문제의 정답을 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
체스판은 8x8 크기로, 좌표 (i, j)에서 하얀 칸은 (i + j) 가 짝수인 경우입니다. 주어진 체스판 상태에서 하얀 칸에 놓인 말('F')의 개수를 세기 위해 이 조건을 이용하여 이중 반복문을 통해 모든 칸을 확인합니다.

### 동작 과정
1. 입력으로 주어진 8줄의 체스판 상태를 리스트 형태로 저장합니다.
2. 이중 반복문을 사용하여 각 칸의 좌표 (i, j)를 확인합니다.
   - 외부 반복문은 행을 기준으로 0부터 7까지 반복 (i).
   - 내부 반복문은 열을 기준으로 0부터 7까지 반복 (j).
3. 현재 칸이 하얀 칸인지 확인하기 위해 (i + j) % 2 == 0 조건을 체크합니다.
4. 하얀 칸이라면 해당 칸의 값이 'F'인지를 확인하고, 맞다면 카운트를 증가시킵니다.
5. 모든 칸을 검사한 후 카운트된 값을 출력합니다.

예시 입력:
```
........
.F...... 
........
.F..F...
........
........
.F......
........
```
- 1행: 하얀 칸 없음
- 2행: (1, 1)에 'F' 있음
- 3행: 하얀 칸 없음
- 4행: (3, 1), (3, 5)에 'F' 있음
- 5행: 하얀 칸 없음
- 6행: (5, 1)에 'F' 있음
- 7행: 하얀 칸 없음
- 8행: 하얀 칸 없음

결과적으로 하얀 칸 위에 'F'가 4개 존재하므로 출력은 4입니다.

### 시간 복잡도
O(1) - 체스판의 크기가 고정되어 있기 때문에 입력의 크기에 상관없이 항상 상수 시간에 계산됩니다.

---

## 코드
```python
graph = [input().rstrip() for _ in range(8)]

cnt = 0
for i in range(8):
    for j in range(8):
        if (i % 2 == 0 and j % 2 == 0) or (i % 2 == 1 and j % 2 == 1):
            if graph[i][j] == 'F':
                cnt += 1
                
print(cnt)
```
