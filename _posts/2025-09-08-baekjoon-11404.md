## 풀이

### 풀이 핵심 로직
이 문제는 주어진 도시와 버스 노선 정보를 바탕으로 모든 도시 쌍 간의 최단 경로를 구하는 문제이다. 다익스트라 알고리즘을 이용하여 각 도시를 시작점으로 하여 최단 거리를 계산하고, 이를 통해 모든 도시 쌍의 최단 거리를 구한다. 각 도시에서 시작하여 도달 가능한 다른 도시까지의 최단 거리를 업데이트하면서 진행한다.

### 동작 과정
1. **입력 예시**: 
   ```
   4
   7
   1 2 4
   1 3 2
   2 3 5
   3 2 1
   3 4 3
   2 4 1
   4 1 7
   ```

2. **초기화**:
   - 도시의 수 `n`은 4, 버스의 수 `m`은 7이다.
   - 거리 배열 `distance`는 `INF`로 초기화된 5x5 행렬(0부터 4까지)로 생성되며, `distance[i][i]`는 0으로 설정된다.

3. **그래프 구성**:
   - 각 버스 노선 정보를 읽어들여 `graph`에 저장한다.
   - 예를 들어, 1번 도시에서 2번 도시로 가는 비용이 4라면 `graph[1].append((2, 4))`로 추가된다.

4. **다익스트라 알고리즘 실행**:
   - 각 도시를 출발점으로 설정하고 `dijkstra` 함수를 호출한다.
   - `dijkstra` 함수 내부에서는 우선순위 큐를 사용하여 현재 도시에서 연결된 도시로의 최단 거리를 계산하고, 이를 `distance` 배열에 업데이트한다.
   - 예를 들어, 1번 도시에서 시작하여 2번 도시로 가는 최소 비용을 계산한 후, 3번 도시로의 경로를 탐색하여 업데이트한다.

5. **결과 출력**:
   - 모든 도시 쌍의 최단 거리를 출력한다. 만약 도달할 수 없는 도시가 있다면 0을 출력한다.

### 시간 복잡도
각 도시를 시작점으로 하여 다익스트라 알고리즘을 수행하므로, 전체 시간 복잡도는 O(n * (m + n log n))이다. 여기서 n은 도시의 수, m은 버스의 수이며, 다익스트라 알고리즘의 경우 우선순위 큐를 사용하여 최단 경로를 찾기 때문에 n log n의 복잡도를 가진다.

## 코드
```python
# 다익스트라

import heapq

n = int(input())
m = int(input())
INF = 1e8

graph = [[] for _ in range(n+1)] 

distance = [[INF] * (n+1) for _ in range(n+1)]

for _ in range(m):
    u, v, w = map(int, input().split()) 
    graph[u].append((v, w))  

def dijkstra(start):
    q = []
    heapq.heappush(q, (0, start))
    distance[start][start] = 0  

    while q:
        dist, now = heapq.heappop(q)

        if distance[start][now] < dist:
            continue

        for i in graph[now]:  
            cost = dist + i[1]
            if cost < distance[start][i[0]]:  
                distance[start][i[0]] = cost  
                heapq.heappush(q, (cost, i[0]))

for i in range(1, n+1): 
    dijkstra(i)

for i in range(1, n+1):
    for j in range(1, n+1):
        if distance[i][j] == INF:
            print(0, end=" ")
        else:
            print(distance[i][j], end=" ")
    print()
```
