## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
문제는 주어진 정수 N의 팩토리얼(N!)을 계산하는 것입니다. 팩토리얼은 자연수 N에 대해 N! = N × (N-1) × ... × 1로 정의되며, N이 0일 경우 0!은 1로 정의됩니다. 재귀 함수를 사용하여 N의 팩토리얼을 계산하는 방식으로 구현했습니다.

### 동작 과정
1. 입력으로 정수 N이 주어집니다. 예를 들어, N = 5라고 가정합니다.
2. `factorial(5)`가 호출됩니다. N은 5이므로, 5는 1보다 크므로 else 구문으로 넘어갑니다.
3. `factorial(4)`가 호출되고, 이 과정이 반복됩니다:
   - `factorial(4)` → `factorial(3)`
   - `factorial(3)` → `factorial(2)`
   - `factorial(2)` → `factorial(1)`
   - `factorial(1)` → `1` (base case, N이 1이므로 1 반환)
4. 이제 반환된 값을 이용하여 역으로 계산합니다:
   - `factorial(2)` → `1 * 2 = 2`
   - `factorial(3)` → `2 * 3 = 6`
   - `factorial(4)` → `6 * 4 = 24`
   - `factorial(5)` → `24 * 5 = 120`
5. 최종적으로 120이 출력됩니다.

### 시간 복잡도
재귀 호출의 깊이는 N에 비례하므로 시간 복잡도는 O(N)입니다. 각 호출마다 상수 시간의 곱셈 연산이 수행되기 때문에 전체적으로 O(N)입니다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
