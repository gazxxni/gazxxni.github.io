## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 0 또는 1일 때 1이며, 그 외의 경우 N!은 N과 (N-1)!의 곱으로 정의된다. 재귀적으로 이 정의를 활용하여 N!을 계산하는 함수를 작성한다.

### 동작 과정
1. 입력으로 주어진 정수 N을 받는다. 예를 들어, N이 5일 경우.
2. `factorial` 함수가 호출되고 N이 5로 들어간다.
3. N이 5일 때, 함수는 5와 factorial(4)을 곱하기 위해 factorial(4)을 호출한다.
4. 같은 방식으로 factorial(4) → factorial(3) → factorial(2) → factorial(1) → factorial(0)까지 호출된다.
5. factorial(0)은 1을 반환하고, 이 값을 바탕으로 factorial(1) → factorial(2) → factorial(3) → factorial(4) → factorial(5) 순서로 결과가 차례로 계산되어 최종적으로 120이 반환된다.

### 시간 복잡도
O(N) - N의 크기에 비례하여 재귀 호출이 N번 발생하므로 시간 복잡도는 선형이다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
