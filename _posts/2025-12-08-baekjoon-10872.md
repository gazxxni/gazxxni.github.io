## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 N부터 1까지의 모든 정수를 곱한 결과로 정의됩니다. 이를 재귀적으로 구현할 수 있으며, 기저 사례로 N이 0일 때는 1을 반환하고, 그 외의 경우에는 N과 (N-1)!을 곱하여 결과를 반환합니다.

### 동작 과정
1. 입력으로 정수 N을 받습니다.
2. N이 0일 경우 1을 출력합니다.
3. 그렇지 않으면, `fac` 함수를 호출하여 N!을 계산합니다.
   - 예를 들어 N = 5일 경우:
     - `fac(5)`가 호출되어 `5 * fac(4)`을 계산합니다.
     - `fac(4)`는 `4 * fac(3)`을 호출하고,
     - `fac(3)`는 `3 * fac(2)`을 호출하며,
     - `fac(2)`는 `2 * fac(1)`을 호출합니다.
     - `fac(1)`에서 1을 반환하고, 이 값을 차례로 곱해 올라가면서 최종적으로 `5 * 4 * 3 * 2 * 1`의 결과인 120을 반환합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 이는 N이 커짐에 따라 호출되는 재귀 함수의 깊이가 N이기 때문입니다. 각 호출마다 상수 시간 내에 계산이 이루어지므로, 전체적으로 O(N)의 시간 복잡도를 가집니다.

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
