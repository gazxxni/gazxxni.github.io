## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 N과 그보다 작은 모든 양의 정수를 곱한 값입니다. 이 문제에서는 재귀 함수를 사용하여 N!을 계산합니다. N이 0일 경우 팩토리얼 값은 1로 정의되며, 그 외의 경우에는 N을 N-1의 팩토리얼과 곱하여 결과를 구합니다.

### 동작 과정
1. 입력을 통해 정수 N을 받습니다. 예를 들어, N=5라고 가정합니다.
2. `fac`라는 재귀 함수를 정의합니다. 이 함수는 N이 1일 경우 1을 반환하고, 그 외의 경우 N과 `fac(n-1)`의 곱을 반환합니다.
3. N이 0인 경우, 직접 1을 출력합니다. 
4. N이 0이 아닐 경우, `fac` 함수를 호출하여 N!을 계산하고 그 결과를 출력합니다.
   - 예시 입력 N=5의 경우:
     - `fac(5)` 호출 → 5 * `fac(4)`
     - `fac(4)` 호출 → 4 * `fac(3)`
     - `fac(3)` 호출 → 3 * `fac(2)`
     - `fac(2)` 호출 → 2 * `fac(1)`
     - `fac(1)` 호출 → 1 반환
     - 결과적으로, `fac(2)` → 2, `fac(3)` → 6, `fac(4)` → 24, `fac(5)` → 120이 되어 최종적으로 120을 출력합니다.

### 시간 복잡도
O(N) - 재귀 호출이 N번 발생하므로, 시간 복잡도는 N에 비례합니다.

---

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
