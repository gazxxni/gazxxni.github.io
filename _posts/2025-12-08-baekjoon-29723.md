## 문제
올해 고3인 브실이는 세계 최고의 명문 대학 브실대학(브론즈실버대학)에 가기 위해서 자신의 현재 점수를 토대로 입시 전략을 세우려고 한다. 브실대학에서는 특정 과목들의 성적의 합을 통해 서류 전형의 합격여부를 결정한다고 한다. 그러나 브실대학에서는 어떤 과목이 서류 평가에 반영되는지 모두 알려주지 않고 일부만 알려주는 사악한 학교다. 브실대학에서 요구하는 과목 수와 반영된다고 공개된 과목들이 주어질 때, 브실이가 얻을 수 있는 최소 점수와 최대 점수를 구해보자.

단, 공개된 과목과 비공개된 과목은 브실이가 수강한 과목에 모두 포함되어 있으며, 과목은 중복되지 않는다.

---

## 입력
첫 번째 줄에 브실이가 수강한 과목 수 N$N$과 브실대학에서 요구하는 과목 수 M$M$, 그리고 브실대학에서 공개한 과목 수 K$K$가 공백으로 구분되어 주어진다. (1≤K≤M≤N≤10000)$(1 \le K \le M \le N \le 10\,000)$ 

그다음 N$N$줄에 걸쳐 브실이가 수강한 과목 이름 si$s_i$과 정수 점수 pi$p_i$가 공백으로 구분되어 주어진다. si$s_i$는 영어 소문자로만 이루어져 있다. (3≤|si|≤20;$(3 \le |s_i| \le 20;$ 0≤pi≤100)$0 \le p_i \le 100)$ 

그다음 K$K$줄에 걸쳐 브실대학에서 공개한 과목 이름 ti$t_i$가 주어진다. ti$t_i$는 영어 소문자로만 이루어져 있다. (3≤|ti|≤20)$(3 \le |t_i| \le 20)$

---

## 출력
브실이가 얻을 수 있는 최소 점수와 최대 점수를 공백으로 구분하여 출력한다.

## 풀이

### 풀이 핵심 로직
브실이가 수강한 과목의 점수를 관리하기 위해 해시맵(딕셔너리)을 사용하여 과목 이름과 점수를 저장합니다. 공개된 과목의 점수를 합산하여 기본 점수를 구한 후, 비공식 과목에서 남은 과목들 중 최소 M-K 과목의 점수를 더해 최소 점수를, 최대 M-K 과목의 점수를 더해 최대 점수를 계산합니다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 수강한 과목 수 \(N\), 요구하는 과목 수 \(M\), 공개된 과목 수 \(K\)를 입력받습니다. 그 후 \(N\)줄에 걸쳐 과목 이름과 점수를 입력받아 해시맵에 저장합니다.
   - 예시 입력:
     ```
     5 3 2
     math 80
     english 75
     physics 90
     chemistry 85
     biology 60
     english
     chemistry
     ```
   
2. **공개된 과목 처리**: 공개된 과목들의 점수를 합산하여 `ans` 변수에 저장하고, 해당 과목은 해시맵에서 삭제합니다.
   - 공개된 과목: `english(75)`, `chemistry(85)` → `ans = 75 + 85 = 160`

3. **비공식 과목 정렬**: 남은 과목 점수를 오름차순으로 정렬합니다.
   - 남은 과목: `math(80)`, `physics(90)`, `biology(60)` → 정렬 결과: `[(biology, 60), (math, 80), (physics, 90)]`

4. **최소 및 최대 점수 계산**:
   - 남은 과목 수 \(M-K = 3-2=1\) → 최소 점수는 가장 낮은 점수(60) 추가, 최대 점수는 가장 높은 점수(90) 추가.
   - 최소 점수: `min_ans = ans + 60 = 160 + 60 = 220`
   - 최대 점수: `max_ans = ans + 90 = 160 + 90 = 250`

5. **결과 출력**: 최종적으로 최소 점수와 최대 점수를 출력합니다.
   - 출력: `220 250`

### 시간 복잡도
- 입력 처리: \(O(N)\)
- 공개 과목 처리: \(O(K)\)
- 정렬: \(O((N-K) \log (N-K))\)
- 최종 점수 계산: \(O(M-K)\)
- 전체 시간 복잡도는 \(O(N \log N)\)입니다.

## 코드
```python
from collections import defaultdict

n, m, k = map(int, input().split())
dic = defaultdict(int)

for _ in range(n):
    s, p = map(str, input().split())
    dic[s] = int(p)

ans = 0
for _ in range(k):
    t = input().rstrip()
    ans += dic[t]
    
    dic.pop(t)

dic = sorted(dic.items(), key=lambda item: item[1])
m -= k

max_ans = ans
min_ans = ans
if m > 0:
    a = dic[:m]
    min_ans += sum(item[1] for item in a)
    
    b = dic[-m:]
    max_ans += sum(item[1] for item in b)
    
print(min_ans, max_ans)

```
