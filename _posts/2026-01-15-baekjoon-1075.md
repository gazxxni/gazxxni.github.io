## 문제
두 정수 N과 F가 주어진다. 지민이는 정수 N의 가장 뒤 두 자리를 적절히 바꿔서 N을 F로 나누어 떨어지게 만들려고 한다. 만약 가능한 것이 여러 가지이면, 뒤 두 자리를 가능하면 작게 만들려고 한다.

예를 들어, N=275이고, F=5이면, 답은 00이다. 200이 5로 나누어 떨어지기 때문이다. N=1021이고, F=11이면, 정답은 01인데, 1001이 11로 나누어 떨어지기 때문이다.

---

## 입력
첫째 줄에 N, 둘째 줄에 F가 주어진다. N은 100보다 크거나 같고, 2,000,000,000보다 작거나 같은 자연수이다. F는 100보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 마지막 두 자리를 모두 출력한다. 한자리이면 앞에 0을 추가해서 두 자리로 만들어야 한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 정수 N의 가장 뒤 두 자리를 변경하여 N이 F로 나누어 떨어지도록 만드는 것입니다. N의 마지막 두 자리를 00부터 99까지 반복적으로 변경하면서, 해당 값이 F로 나누어 떨어지는 경우를 찾습니다. 가장 먼저 찾은 값이 최소값이므로, 탐색을 종료합니다.

### 동작 과정
1. 입력으로 N=275와 F=5를 받습니다.
2. N의 마지막 두 자리를 제외한 나머지(여기서는 200)를 구합니다.
3. 200에 0부터 99까지의 값을 더하여 탐색합니다.
   - 200 + 0 = 200 → 200 % 5 = 0 (나누어 떨어짐) → 출력: 00
   - 탐색을 종료합니다.
   
다른 예시로 N=1021과 F=11을 고려해 보겠습니다.
1. 입력으로 N=1021와 F=11을 받습니다.
2. N의 마지막 두 자리를 제외한 나머지(여기서는 1000)를 구합니다.
3. 1000에 0부터 99까지의 값을 더하여 탐색합니다.
   - 1000 + 0 = 1000 → 1000 % 11 ≠ 0
   - 1000 + 1 = 1001 → 1001 % 11 = 0 (나누어 떨어짐) → 출력: 01
   - 탐색을 종료합니다.

### 시간 복잡도
이 알고리즘은 마지막 두 자리를 0부터 99까지 탐색하므로, 시간 복잡도는 O(100)으로 상수 시간인 O(1)로 볼 수 있습니다.

---

## 코드
```python
n = int(input())
f = int(input())

base_n = (n // 100) * 100

for i in range(100):
    if (base_n + i) % f == 0:
        print(f"{i:02d}")
        break
```
