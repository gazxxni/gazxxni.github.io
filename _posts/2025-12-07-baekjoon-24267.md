## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘은 n개의 요소에서 3개의 요소를 선택하여 조합을 만드는 과정에서 발생하는 수행 횟수를 계산하는 문제입니다. 조합의 수는 nC3로 표현되며, 이는 `n! / (3! * (n - 3)!)`로 계산되며, 이를 간단히 하면 `(n * (n - 1) * (n - 2)) / 6`이 됩니다. 또한, 다항식의 최고차항 차수는 3입니다.

### 동작 과정
1. 입력으로 n을 받습니다. 예를 들어, n이 5일 경우:
2. MenOfPassion 알고리즘의 수행 횟수를 계산하는 식에 n=5를 대입합니다:
   - 수행 횟수 = `(5 * (5 - 1) * (5 - 2)) / 6`
   - 수행 횟수 = `(5 * 4 * 3) / 6 = 10`
3. 두 번째 출력으로 최고차항의 차수를 출력합니다. 이 문제에서는 항상 3이므로 3을 출력합니다.

예시 입력:
```
5
```
예시 출력:
```
10
3
```

### 시간 복잡도
이 알고리즘은 입력 n에 대해 O(1)의 시간 복잡도를 가집니다. 즉, 상수 시간에 수행되며, 수행 횟수 계산을 위한 기본적인 산술 연산만을 포함하므로 매우 효율적입니다.

## 코드
```python
n = int(input())

print(int((n * (n - 1) * (n - 2)) / 6))
print(3)
```
