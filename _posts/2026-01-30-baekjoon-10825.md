## 문제
도현이네 반 학생 N명의 이름과 국어, 영어, 수학 점수가 주어진다. 이때, 다음과 같은 조건으로 학생의 성적을 정렬하는 프로그램을 작성하시오.


	국어 점수가 감소하는 순서로
	국어 점수가 같으면 영어 점수가 증가하는 순서로
	국어 점수와 영어 점수가 같으면 수학 점수가 감소하는 순서로
	모든 점수가 같으면 이름이 사전 순으로 증가하는 순서로 (단, 아스키 코드에서 대문자는 소문자보다 작으므로 사전순으로 앞에 온다.)

---

## 입력
첫째 줄에 도현이네 반의 학생의 수 N (1 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 한 줄에 하나씩 각 학생의 이름, 국어, 영어, 수학 점수가 공백으로 구분해 주어진다. 점수는 1보다 크거나 같고, 100보다 작거나 같은 자연수이다. 이름은 알파벳 대소문자로 이루어진 문자열이고, 길이는 10자리를 넘지 않는다.

---

## 출력
문제에 나와있는 정렬 기준으로 정렬한 후 첫째 줄부터 N개의 줄에 걸쳐 각 학생의 이름을 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 주어진 학생들의 점수를 다양한 기준에 따라 정렬하는 것이다. 국어 점수는 감소 순서로, 영어 점수는 증가 순서로, 수학 점수는 다시 감소 순서로, 마지막으로 이름은 사전 순으로 정렬해야 하므로, 이를 위해 파이썬의 `sort()` 메서드와 `lambda` 함수를 활용하여 정렬 기준을 정의하였다.

### 동작 과정
1. **입력 받기**: 먼저 학생 수 N을 입력받고, N명의 학생 정보를 리스트에 저장한다. 각 학생의 정보는 이름과 점수(국어, 영어, 수학)를 포함한다.
   - 예시 입력:
     ```
     3
     Jang 90 80 85
     Kim 90 90 80
     Lee 80 90 85
     ```

2. **데이터 구조**: 각 학생의 정보를 튜플 형태로 `(이름, 국어 점수, 영어 점수, 수학 점수)`로 저장하여 리스트에 추가한다. 
   - 예시 리스트:
     ```
     arr = [
         ('Jang', 90, 80, 85),
         ('Kim', 90, 90, 80),
         ('Lee', 80, 90, 85)
     ]
     ```

3. **정렬**: `sort()` 메서드를 사용하여 위에서 정의한 정렬 기준에 따라 리스트를 정렬한다. 정렬 기준은 다음과 같다:
   - 국어 점수는 내림차순 (음수로 변환)
   - 영어 점수는 오름차순
   - 수학 점수는 내림차순 (음수로 변환)
   - 이름은 사전 순 (기본적으로 오름차순)
   - 정렬 예시 결과:
     ```
     arr = [
         ('Kim', 90, 90, 80),
         ('Jang', 90, 80, 85),
         ('Lee', 80, 90, 85)
     ]
     ```

4. **출력**: 정렬된 리스트에서 각 학생의 이름만 출력한다.
   - 출력 결과:
     ```
     Kim
     Jang
     Lee
     ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)이다. 여기서 N은 학생의 수로, `sort()` 메서드를 사용하여 N개의 학생을 정렬하는 과정에서 발생하는 시간 복잡도이다.

## 코드
```python
n = int(input())
arr = []

for _ in range(n):
    data = input().split()
    name = data[0]
    ko, en, ma = map(int, data[1:])
    
    arr.append((name, ko, en, ma))
    
arr.sort(key=lambda x:(-x[1], x[2], -x[3], x[0]))

for i in arr:
    print(i[0])
```
