## 문제
정휘는 후배들이 재귀 함수를 잘 다루는 재귀의 귀재인지 알아보기 위해 재귀 함수와 관련된 문제를 출제하기로 했다.

팰린드롬이란, 앞에서부터 읽었을 때와 뒤에서부터 읽었을 때가 같은 문자열을 말한다. 팰린드롬의 예시로 AAA, ABBA, ABABA 등이 있고, 팰린드롬이 아닌 문자열의 예시로 ABCA, PALINDROME 등이 있다.

어떤 문자열이 팰린드롬인지 판별하는 문제는 재귀 함수를 이용해 쉽게 해결할 수 있다. 아래 코드의 isPalindrome 함수는 주어진 문자열이 팰린드롬이면 1, 팰린드롬이 아니면 0을 반환하는 함수다.

#include

## 입력
첫째 줄에 테스트케이스의 개수 T$T$가 주어진다. (1≤T≤1000$1 \leq T \leq 1\,000$)

둘째 줄부터 T$T$개의 줄에 알파벳 대문자로 구성된 문자열 S$S$가 주어진다. (1≤|S|≤1000$1 \leq \vert S\vert \leq 1\,000$)

---

## 출력
각 테스트케이스마다, isPalindrome 함수의 반환값과 recursion 함수의 호출 횟수를 한 줄에 공백으로 구분하여 출력한다.

## 풀이

### 풀이 핵심 로직
문자열이 팰린드롬인지 확인하기 위해 재귀 함수를 사용하여 문자열의 양 끝에서부터 중앙으로 진행하며 문자가 같은지 비교합니다. 문자열의 앞쪽과 뒤쪽 문자가 일치하면 재귀적으로 다음 문자 쌍을 비교하고, 불일치할 경우 팰린드롬이 아니라고 판단합니다. 이 과정에서 호출된 재귀 함수의 횟수를 카운트하여 반환합니다.

### 동작 과정
1. 입력된 문자열 `S`를 기준으로 재귀 함수를 호출합니다. 예를 들어, 문자열 `S = "ABBA"`라고 가정합니다.
2. `recursion` 함수가 호출될 때, `l`은 0, `r`은 3(문자열의 길이 - 1)으로 설정됩니다.
3. 첫 번째 호출: `recursion("ABBA", 0, 3, 1)` - `S[0]`와 `S[3]` 비교 → 같음, 재귀 호출: `recursion("ABBA", 1, 2, 2)`
4. 두 번째 호출: `recursion("ABBA", 1, 2, 2)` - `S[1]`과 `S[2]` 비교 → 같음, 재귀 호출: `recursion("ABBA", 2, 1, 3)`
5. 세 번째 호출: `recursion("ABBA", 2, 1, 3)` - `l(2)`가 `r(1)`보다 크므로 팰린드롬으로 판단하고 `1`을 반환, 호출 횟수는 `3`.
6. 최종적으로 `isPalindrome` 함수가 `1 3`을 반환하게 되어, 출력합니다.

이 과정을 다른 문자열에 대해서도 반복하여 각 테스트 케이스에 대해 결과를 출력합니다.

### 시간 복잡도
재귀 호출의 깊이는 문자열의 길이에 비례하므로, 최악의 경우 O(n)입니다. 따라서 전체 시간 복잡도는 O(T * n)입니다. 여기서 T는 테스트 케이스의 수, n은 문자열의 최대 길이입니다.

## 코드
```python
def recursion(s, l, r, cnt):
    if l >= r:
        return 1, cnt
    elif s[l] != s[r]:
        return 0, cnt
    else:
        return recursion(s, l + 1, r - 1, cnt + 1)

def isPalindrome(s):
    return recursion(s, 0, len(s) - 1, 1)

t = int(input())
for _ in range(t):
    s = input().rstrip()
    print(*isPalindrome(s))

```
