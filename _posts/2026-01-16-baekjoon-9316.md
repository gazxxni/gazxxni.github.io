## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

---

## 풀이

### 풀이 핵심 로직
주어진 입력 N에 따라 1부터 N까지의 숫자를 순회하면서 각 줄에 "Hello World, Judge i!" 형식의 문자열을 출력하는 것이 핵심 아이디어입니다. 이를 위해 반복문을 사용하여 각 줄의 번호를 출력 문자열에 포함시킵니다.

### 동작 과정
1. 사용자로부터 정수 N을 입력받습니다. (예: N = 3)
2. 반복문을 사용하여 i를 1부터 N까지 증가시킵니다.
   - i = 1일 때: "Hello World, Judge 1!" 출력
   - i = 2일 때: "Hello World, Judge 2!" 출력
   - i = 3일 때: "Hello World, Judge 3!" 출력
3. N이 3인 경우, 최종 출력은 다음과 같습니다:
   ```
   Hello World, Judge 1!
   Hello World, Judge 2!
   Hello World, Judge 3!
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 이는 N개의 줄을 출력하기 위해 N번의 반복이 필요하기 때문입니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
