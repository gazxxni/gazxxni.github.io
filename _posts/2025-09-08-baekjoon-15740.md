## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 A와 B (-1010000 ≤ A, B ≤ 1010000)가 주어진다.

---

## 출력
첫째 줄에 A+B를 출력한다.

---

## 풀이

### 풀이 핵심 로직
문제는 두 개의 정수 A와 B를 입력받아 이들의 합을 출력하는 것입니다. A와 B는 매우 큰 수일 수 있으므로 파이썬의 기본 정수형(int)은 임의의 정밀도를 지원하기 때문에 이를 활용하여 간단히 계산합니다.

### 동작 과정
1. **입력 받기**: 첫 줄에서 A와 B를 공백으로 구분하여 입력받습니다. 예를 들어, 입력이 `12345678901234567890 98765432109876543210`이라면, A는 `12345678901234567890`, B는 `98765432109876543210`이 됩니다.
2. **정수 변환**: 입력받은 A와 B를 `map(int, ...)`을 사용하여 정수형으로 변환합니다.
3. **덧셈 수행**: A와 B를 더하여 결과를 계산합니다. 이 예시에서는 `12345678901234567890 + 98765432109876543210`이 되어 `111111111011111111100`입니다.
4. **출력**: 최종 결과를 출력합니다.

### 시간 복잡도
이 문제의 시간 복잡도는 O(n)입니다. 여기서 n은 A와 B의 자릿수의 최대 길이를 의미합니다. 하지만, 파이썬의 정수 덧셈은 상수 시간에 수행될 수도 있으므로 실제로는 매우 빠르게 동작합니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a + b)
```
