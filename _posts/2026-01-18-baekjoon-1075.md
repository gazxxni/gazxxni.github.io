## 문제
두 정수 N과 F가 주어진다. 지민이는 정수 N의 가장 뒤 두 자리를 적절히 바꿔서 N을 F로 나누어 떨어지게 만들려고 한다. 만약 가능한 것이 여러 가지이면, 뒤 두 자리를 가능하면 작게 만들려고 한다.

예를 들어, N=275이고, F=5이면, 답은 00이다. 200이 5로 나누어 떨어지기 때문이다. N=1021이고, F=11이면, 정답은 01인데, 1001이 11로 나누어 떨어지기 때문이다.

---

## 입력
첫째 줄에 N, 둘째 줄에 F가 주어진다. N은 100보다 크거나 같고, 2,000,000,000보다 작거나 같은 자연수이다. F는 100보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 마지막 두 자리를 모두 출력한다. 한자리이면 앞에 0을 추가해서 두 자리로 만들어야 한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 수 N의 마지막 두 자리를 조정하여 N을 F로 나누어 떨어지도록 만드는 것입니다. N의 마지막 두 자리를 변경할 수 있는 모든 경우를 고려하면서, F로 나누어 떨어지는 가장 작은 두 자리 수를 찾습니다.

### 동작 과정
1. **입력값 읽기**: N과 F를 입력받습니다.
   - 예시: N=275, F=5
2. **기본 값 계산**: N의 마지막 두 자리를 0으로 만들기 위해 N을 100으로 나눈 후 100을 곱합니다. 이렇게 하면 N의 뒤 두 자리를 무시한 값이 구해집니다.
   - 예시: `base_n = (275 // 100) * 100` → `base_n = 200`
3. **모든 경우의 수 탐색**: 0부터 99까지의 모든 정수를 더하여 `base_n`에 추가합니다. 이때, 각 경우에 대해 결과가 F로 나누어 떨어지는지를 확인합니다.
   - 반복문을 이용하여 `for i in range(100)`을 진행합니다.
4. **조건 검사**: 각 경우에 대해 `(base_n + i) % F == 0`인 경우를 찾습니다. 조건을 만족하는 i를 발견하면, 해당 값을 두 자리 문자열로 포맷팅하여 출력합니다.
   - 예시: `i = 0`일 때, `200 % 5 == 0`이므로 출력: `00`
5. **출력**: 첫 번째로 만족하는 i를 발견하면 종료합니다.

### 시간 복잡도
O(1) - 반복문이 항상 100번만 수행되므로, 입력과 무관하게 상수 시간 복잡도를 가집니다.

---

## 코드
```python
n = int(input())
f = int(input())

base_n = (n // 100) * 100

for i in range(100):
    if (base_n + i) % f == 0:
        print(f"{i:02d}")
        break
```
