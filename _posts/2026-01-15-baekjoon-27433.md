## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 N과 N-1!을 곱한 값으로 정의됩니다. 이 문제에서는 재귀 함수를 사용하여 N!을 계산합니다. 기본적으로 0!과 1!은 1로 정의되며, 그 이상의 값에 대해서는 N! = N × (N-1)!로 계산합니다.

### 동작 과정
1. 입력으로 주어진 정수 N을 읽습니다. 예를 들어, N = 5라고 가정합니다.
2. `factorial` 함수를 호출하여 N!을 계산합니다.
   - factorial(5) 호출:
     - 5는 1보다 크므로, factorial(4) * 5 호출.
   - factorial(4) 호출:
     - 4는 1보다 크므로, factorial(3) * 4 호출.
   - factorial(3) 호출:
     - 3은 1보다 크므로, factorial(2) * 3 호출.
   - factorial(2) 호출:
     - 2는 1보다 크므로, factorial(1) * 2 호출.
   - factorial(1) 호출:
     - 1은 1보다 크지 않으므로 1 반환.
3. 이제 각 호출이 반환되기 시작합니다:
   - factorial(2)는 1 * 2 = 2 반환.
   - factorial(3)는 2 * 3 = 6 반환.
   - factorial(4)는 6 * 4 = 24 반환.
   - factorial(5)는 24 * 5 = 120 반환.
4. 최종적으로 120이 출력됩니다.

### 시간 복잡도
시간 복잡도는 O(N)입니다. N에 대한 재귀 호출이 N번 발생하기 때문입니다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
