## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
팩토리얼은 주어진 정수 N에 대해 N! = N * (N-1)!로 정의됩니다. 이를 재귀적으로 구현하여 N이 0이나 1일 때는 1을 반환하고, 그 외에는 N과 (N-1)!의 곱을 계산하여 반환합니다. 이 방식은 간결하지만, N이 작기 때문에 재귀 호출로 인한 성능 문제는 발생하지 않습니다.

### 동작 과정
1. 예를 들어, N = 5일 경우, `factorial(5)`가 호출됩니다.
2. `factorial(5)`는 `factorial(4) * 5`를 호출합니다.
3. `factorial(4)`는 `factorial(3) * 4`를 호출하고, 이 과정은 N이 1이 될 때까지 계속됩니다.
4. 최종적으로 `factorial(1)`이 호출되면 1을 반환합니다.
5. 이 값이 역으로 계산되어, `factorial(2)`는 2, `factorial(3)`은 6, `factorial(4)`는 24, `factorial(5)`는 120을 반환하게 됩니다.
6. 따라서, 5!의 결과인 120이 출력됩니다.

### 시간 복잡도
시간 복잡도는 O(N)입니다. 재귀 호출이 N번 발생하므로, N이 20 이하의 값일 때 충분히 효율적입니다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
