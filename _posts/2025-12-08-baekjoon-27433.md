## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 0보다 크거나 같은 정수 N에 대해 N부터 1까지의 모든 정수를 곱한 결과입니다. 재귀 함수를 이용하여 N이 1 이하일 때 1을 반환하고, 그 외의 경우 N과 N-1의 팩토리얼을 곱하여 결과를 계산합니다.

### 동작 과정
예시 입력으로 N = 5를 사용할 경우, 단계별 설명은 다음과 같습니다:

1. `factorial(5)` 호출
   - N이 1보다 크므로 `factorial(4) * 5`로 진행
2. `factorial(4)` 호출
   - N이 1보다 크므로 `factorial(3) * 4`로 진행
3. `factorial(3)` 호출
   - N이 1보다 크므로 `factorial(2) * 3`로 진행
4. `factorial(2)` 호출
   - N이 1보다 크므로 `factorial(1) * 2`로 진행
5. `factorial(1)` 호출
   - N이 1이므로 1을 반환
6. `factorial(2)`는 `1 * 2`를 계산하여 2를 반환
7. `factorial(3)`는 `2 * 3`을 계산하여 6을 반환
8. `factorial(4)`는 `6 * 4`를 계산하여 24를 반환
9. `factorial(5)`는 `24 * 5`를 계산하여 120을 반환

최종적으로 `print(factorial(5))`는 120을 출력합니다.

### 시간 복잡도
재귀 호출의 깊이가 N에 비례하므로, 이 알고리즘의 시간 복잡도는 O(N)입니다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
