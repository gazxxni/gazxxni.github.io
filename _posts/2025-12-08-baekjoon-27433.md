## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 주어진 정수 N에 대해 N! (팩토리얼)을 계산하여 출력하는 것이다. 팩토리얼은 N이 0인 경우 1로 정의되며, N이 1 이상인 경우 N!은 N과 (N-1)!의 곱으로 정의된다. 이를 재귀적으로 계산하여 결과를 도출할 수 있다.

### 동작 과정
1. 사용자가 정수 N을 입력한다. 예를 들어, N = 5라고 가정하자.
2. `factorial` 함수가 호출되며, N의 값이 5로 전달된다.
3. 함수 내부에서 N이 1보다 큰 경우 재귀 호출이 이루어진다. 즉, `factorial(5)`는 `factorial(4) * 5`로 계산된다.
4. 이 과정은 N이 0 또는 1에 도달할 때까지 반복된다:
   - `factorial(4)` → `factorial(3) * 4`
   - `factorial(3)` → `factorial(2) * 3`
   - `factorial(2)` → `factorial(1) * 2`
   - `factorial(1)` → 1 (재귀의 기저 사례)
5. 각 재귀 호출이 완료되면서 곱셈이 수행되고, 최종적으로 5! = 120이 계산된다.
6. 결과가 출력된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)이다. N에 대해 재귀 호출이 N번 발생하고, 각 호출에서 상수 시간의 곱셈 연산이 이루어지기 때문이다.

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
