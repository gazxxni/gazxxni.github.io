## 입력

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 온도 수열에서 연속적인 K일 동안의 온도 합이 최대가 되는 값을 찾는 문제입니다. 슬라이딩 윈도우 기법을 사용하여 초기 K일의 합을 계산한 후, 다음 날부터는 가장 오래된 날의 온도를 빼고 새로운 날의 온도를 더하는 방식으로 O(N) 시간 복잡도로 해결할 수 있습니다.

### 동작 과정
1. **입력 예시**: 
   - N = 10, K = 3
   - 온도 수열: [3, -2, -4, -9, 0, 3, 7, 13, 8, -3]

2. **초기 K일의 온도 합 계산**:
   - 첫 3일의 온도 합: `3 + (-2) + (-4) = -3`
   - 이 값을 `ans`에 저장: `ans = -3`

3. **슬라이딩 윈도우 적용**:
   - 4일째 (i=3): `ss = ss - arr[0] + arr[3]` → `ss = -3 - 3 - 9 = -9`, `ans = max(-3, -9) = -3`
   - 5일째 (i=4): `ss = ss - arr[1] + arr[4]` → `ss = -9 - (-2) + 0 = -7`, `ans = max(-3, -7) = -3`
   - 6일째 (i=5): `ss = ss - arr[2] + arr[5]` → `ss = -7 - (-4) + 3 = 0`, `ans = max(-3, 0) = 0`
   - 7일째 (i=6): `ss = ss - arr[3] + arr[6]` → `ss = 0 - (-9) + 7 = 16`, `ans = max(0, 16) = 16`
   - 8일째 (i=7): `ss = ss - arr[4] + arr[7]` → `ss = 16 - 0 + 13 = 29`, `ans = max(16, 29) = 29`
   - 9일째 (i=8): `ss = ss - arr[5] + arr[8]` → `ss = 29 - 3 + 8 = 34`, `ans = max(29, 34) = 34`
   - 10일째 (i=9): `ss = ss - arr[6] + arr[9]` → `ss = 34 - 7 - 3 = 24`, `ans = max(34, 24) = 34`

4. **최종 결과 출력**: `34`

### 시간 복잡도
O(N) - 온도 수열의 길이에 대해 한 번만 순회하므로 시간 복잡도는 O(N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
