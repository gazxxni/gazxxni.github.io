## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 정수 N에 대해 1부터 N까지의 숫자를 사용하여 각 줄마다 "Hello World, Judge i!"라는 형식의 문자열을 출력하는 간단한 구현 문제입니다. 반복문을 사용하여 각 숫자에 대한 문자열을 생성하고 출력합니다.

### 동작 과정
1. 사용자로부터 정수 N을 입력받습니다.
2. 1부터 N까지의 숫자에 대해 반복문을 실행합니다.
3. 각 반복에서 현재 숫자 i를 사용하여 "Hello World, Judge i!"라는 문자열을 출력합니다.

예시 입력:
```
3
```
예시 동작 과정:
- N = 3이므로, 반복문이 1부터 3까지 실행됩니다.
- i = 1일 때, "Hello World, Judge 1!" 출력
- i = 2일 때, "Hello World, Judge 2!" 출력
- i = 3일 때, "Hello World, Judge 3!" 출력

최종 출력:
```
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

### 시간 복잡도
O(N) - N번의 반복을 수행하므로 시간 복잡도는 선형입니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
