## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 숫자 N에 대해 1부터 N까지의 숫자를 순회하면서 각 숫자에 맞는 인사 메시지를 출력하는 간단한 구현 문제입니다. 포맷에 맞춰 문자열을 출력하기 위해 f-string을 사용하여 가독성을 높였습니다.

### 동작 과정
1. 입력으로 N을 받습니다. 예를 들어, 입력이 `3`이라면 N은 3입니다.
2. 1부터 N까지 반복문을 실행합니다.
   - i=1일 때: `Hello World, Judge 1!` 출력
   - i=2일 때: `Hello World, Judge 2!` 출력
   - i=3일 때: `Hello World, Judge 3!` 출력
3. 최종적으로 3개의 인사 메시지가 출력됩니다.

예시 입력: 
```
3
```
출력 결과:
```
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

### 시간 복잡도
O(N) - 입력으로 주어진 N만큼의 줄을 순회하며 출력하기 때문에, 시간 복잡도는 N에 비례합니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
