[https://www.acmicpc.net/problem/11050](https://www.acmicpc.net/problem/11050)

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 n과 k에 대해 이항 계수 \(\binom{n}{k}\)를 계산하는 문제입니다. 이항 계수는 조합의 수를 나타내며, 수학적으로 다음과 같이 정의됩니다: \(\binom{n}{k} = \frac{n!}{(n-k)! \cdot k!}\). 코드에서는 재귀를 이용하여 팩토리얼을 계산한 후, 이항 계수를 구하는 방식으로 구현되었습니다.

### 동작 과정
예를 들어, 입력이 `5 2`일 경우를 살펴보겠습니다.

1. **입력 처리**: `n`과 `k`를 입력받습니다. 여기서는 `n = 5`와 `k = 2`입니다.
2. **팩토리얼 계산**:
   - `factorial(5)` 호출:
     - `5 * factorial(4)` 호출:
       - `4 * factorial(3)` 호출:
         - `3 * factorial(2)` 호출:
           - `2 * factorial(1)` 호출:
             - `factorial(1)`은 1 반환.
           - 따라서 `factorial(2)`는 `2 * 1 = 2` 반환.
         - `factorial(3)`는 `3 * 2 = 6` 반환.
       - `factorial(4)`는 `4 * 6 = 24` 반환.
     - `factorial(5)`는 `5 * 24 = 120` 반환.
   - `factorial(3)` 호출 (n-k = 3):
     - `factorial(3)`는 이미 계산한 대로 `6`을 반환.
   - `factorial(2)` 호출 (k = 2):
     - `factorial(2)`는 이미 계산한 대로 `2`를 반환.
3. **이항 계수 계산**:
   - \(\binom{5}{2} = \frac{120}{6 \cdot 2} = \frac{120}{12} = 10\).
4. **결과 출력**: 최종적으로 10이 출력됩니다.

### 시간 복잡도
이 코드의 시간 복잡도는 \(O(n)\)입니다. 이는 팩토리얼을 계산하기 위해 최대 n번의 재귀 호출이 발생하기 때문입니다. 따라서 입력값 n에 비례하는 시간 복잡도를 가집니다.

## 코드
```python
import sys

n,k=map(int,input().split())

def factorial(n):
  if n == 0 or n == 1:
    return 1
  else:
    return n * factorial(n-1)
  
print(factorial(n) // (factorial(n-k) * factorial(k)))
```
