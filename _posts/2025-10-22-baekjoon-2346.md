## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제는 원형으로 배치된 풍선들을 터뜨리는 규칙을 기반으로 하고 있습니다. 각 풍선은 자신이 가진 수만큼 이동하여 다음 풍선을 터뜨리며, 이미 터진 풍선은 무시합니다. 이를 효율적으로 관리하기 위해 `deque` 자료구조를 사용하여 풍선의 순서를 유지하고, 터진 풍선을 쉽게 제거할 수 있도록 합니다.

### 동작 과정
예를 들어, 입력으로 `5`와 `3, 2, 1, -3, -1`이 주어진 경우를 살펴보겠습니다.

1. 처음에 1번 풍선을 터뜨립니다. 이 풍선의 종이에 적힌 수는 `3`입니다. 따라서, 오른쪽으로 `3`칸 이동합니다. 현재 남아있는 풍선은 `[1, 2, 3, 4, 5]`이고, 오른쪽으로 `3`칸 이동하면 `4`번 풍선을 터뜨립니다.
2. `4`번 풍선을 터뜨리면, 그 종이에 있는 수는 `-3`입니다. 이 값을 이용하여 왼쪽으로 `3`칸 이동합니다. 현재 남아있는 풍선은 `[2, 3, 5]`이고, 왼쪽으로 `3`칸 이동하면 `2`번 풍선을 터뜨립니다.
3. `2`번 풍선에서는 `2`가 적혀있으므로, 오른쪽으로 `2`칸 이동하여 `5`번 풍선을 터뜨립니다.
4. `5`번 풍선의 종이에 있는 수는 `-1`입니다. 왼쪽으로 `1`칸 이동하면 `3`번 풍선을 터뜨립니다.
5. 마지막으로 `3`번 풍선을 터뜨리면서 과정이 종료됩니다.

결과적으로 터진 풍선의 순서는 `1 4 2 5 3`이 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 풍선이 한 번씩만 터지므로, 전체적으로 N번의 작업이 이루어지며, `deque`의 회전 및 제거 작업이 평균적으로 O(1) 시간이 걸립니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
