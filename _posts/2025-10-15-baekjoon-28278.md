## 풀이

### 풀이 핵심 로직
이 문제는 스택 자료구조를 구현하여 주어진 명령에 따라 스택의 상태를 조작하고 결과를 출력하는 것입니다. 각 명령은 스택에 대한 다양한 연산(푸시, 팝, 크기, 비어있는지 확인, 최상단 원소 확인)을 수행하며, 이들 각각에 대해 적절한 결과를 출력합니다.

### 동작 과정
입력이 다음과 같다고 가정해 보겠습니다:
```
5
1 10
1 20
2
3
4
```
1. `5`는 총 명령의 수를 나타냅니다.
2. `1 10`: 스택에 `10`을 푸시합니다. (스택 상태: `[10]`)
3. `1 20`: 스택에 `20`을 푸시합니다. (스택 상태: `[10, 20]`)
4. `2`: 스택에서 최상단 원소인 `20`을 팝합니다. `20`이 출력됩니다. (스택 상태: `[10]`)
5. `3`: 현재 스택의 크기를 출력합니다. `1`이 출력됩니다. (스택 상태: `[10]`)
6. `4`: 스택이 비어 있지 않으므로 `0`이 출력됩니다. (스택 상태: `[10]`)

### 시간 복잡도
각 명령은 O(1)의 시간 복잡도를 가지므로, 전체 명령 수 n에 대해 시간 복잡도는 O(n)입니다.

## 코드
```python
import sys

n = int(sys.stdin.readline())
stack = []

for _ in range(n):
    command = sys.stdin.readline().split()
    
    a = int(command[0])
    
    if a == 1:
        b = int(command[1])
        stack.append(b)
        
    elif a == 2:
        if stack:
            print(stack.pop())
        else:
            print(-1)
            
    elif a == 3:
        print(len(stack))
        
    elif a == 4:
        if not stack:
            print(1)
        else:
            print(0)
            
    elif a == 5:
        if stack:
            print(stack[-1])
        else:
            print(-1)
```
