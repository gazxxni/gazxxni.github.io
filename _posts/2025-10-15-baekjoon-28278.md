## 문제
정수를 저장하는 스택을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 다섯 가지이다.


	1 X: 정수 X를 스택에 넣는다. (1 ≤ X ≤ 100,000)
	2: 스택에 정수가 있다면 맨 위의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	3: 스택에 들어있는 정수의 개수를 출력한다.
	4: 스택이 비어있으면 1, 아니면 0을 출력한다.
	5: 스택에 정수가 있다면 맨 위의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 스택 자료구조를 사용하여 주어진 명령을 처리하는 문제입니다. 스택은 LIFO(Last In, First Out) 구조로, 가장 최근에 추가된 데이터가 가장 먼저 제거됩니다. 명령에 따라 정수를 추가하거나 제거하고, 스택의 상태를 확인하는 기능을 구현합니다.

### 동작 과정
주어진 입력 예시를 따라 단계별로 설명해 보겠습니다.

입력:
```
7
1 10
1 20
3
4
5
2
2
```

1. **1 10**: 스택에 10을 추가합니다. 현재 스택: [10]
2. **1 20**: 스택에 20을 추가합니다. 현재 스택: [10, 20]
3. **3**: 스택에 있는 정수의 개수를 출력합니다. 출력: 2
4. **4**: 스택이 비어있는지 확인합니다. 비어있지 않으므로 출력: 0
5. **5**: 스택의 맨 위 정수를 출력합니다. 현재 맨 위 정수는 20이므로 출력: 20
6. **2**: 스택에서 맨 위의 정수를 제거하고 출력합니다. 20이 제거되므로 출력: 20. 현재 스택: [10]
7. **2**: 다시 스택에서 맨 위의 정수를 제거하고 출력합니다. 10이 제거되므로 출력: 10. 현재 스택: []

### 시간 복잡도
각 명령은 O(1)의 시간 복잡도로 처리됩니다. 따라서 N개의 명령을 처리하는 전체 시간 복잡도는 O(N)입니다. 이 문제는 최대 1,000,000개의 명령을 처리해야 하므로, O(N) 시간 복잡도로 효율적으로 해결할 수 있습니다.

## 코드
```python
import sys

n = int(sys.stdin.readline())
stack = []

for _ in range(n):
    command = sys.stdin.readline().split()
    
    a = int(command[0])
    
    if a == 1:
        b = int(command[1])
        stack.append(b)
        
    elif a == 2:
        if stack:
            print(stack.pop())
        else:
            print(-1)
            
    elif a == 3:
        print(len(stack))
        
    elif a == 4:
        if not stack:
            print(1)
        else:
            print(0)
            
    elif a == 5:
        if stack:
            print(stack[-1])
        else:
            print(-1)
```
