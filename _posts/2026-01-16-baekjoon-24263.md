## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 주요 작업은 입력 크기 n에 따라 단순히 n번의 반복을 수행하는 것입니다. 따라서 수행 횟수는 n이 되고, 이 알고리즘의 복잡도는 O(n)입니다. 최고차항의 차수는 1로, 다항식으로 나타낼 수 있습니다.

### 동작 과정
1. 입력으로 n의 값이 주어진다고 가정합시다. 예를 들어, n = 5라고 하겠습니다.
2. 코드1에서는 n만큼의 반복이 일어나므로 수행 횟수는 5회가 됩니다.
3. 따라서 첫 번째 출력은 5가 되고, 코드1의 수행 횟수를 다항식으로 표현했을 때 최고차항의 차수는 1이므로 두 번째 출력은 1이 됩니다.

예시 입력/출력:
- 입력: 5
- 출력:
```
5
1
```

### 시간 복잡도
O(n)
