## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
문제에서 주어진 MenOfPassion 알고리즘은 두 개의 코드 블록이 있으며, 이 코드 블록의 수행 횟수를 분석해야 합니다. 주어진 n에 대해 수행 횟수는 n^2으로 계산되며, 이 경우 최고차항의 차수는 2입니다. 따라서, 출력은 n^2과 2가 됩니다.

### 동작 과정
1. 입력으로 n이 주어진다고 가정합니다. 예를 들어, n = 5라고 합시다.
2. 코드1의 수행 횟수는 n^2이므로, 5^2 = 25가 됩니다.
3. 최고차항의 차수는 2이므로, 두 번째 출력은 2가 됩니다.
4. 최종적으로 출력은 다음과 같습니다:
   ```
   25
   2
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 n에 따라 수행 횟수와 차수를 계산하는 과정은 상수 시간에 이루어지기 때문입니다.
