## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 1부터 N까지의 모든 수를 이어서 썼을 때의 자릿수를 계산하는 것입니다. 자릿수를 계산하기 위해, 각 자릿수의 범위를 기준으로 자릿수의 개수를 누적하여 더해가는 방식을 사용합니다. 즉, 1자리, 2자리, 3자리 수의 개수를 계산한 후, N의 자릿수에 대한 계산을 수행합니다.

### 동작 과정
1. **입력 처리**: N을 입력받고, N의 자릿수를 계산합니다. 예를 들어, N이 23이라면 `len(n)`은 2가 됩니다.
2. **자릿수 별 계산**:
   - 1자리 수 (1~9): 9개의 수, 각 수가 1자리이므로 `9 * 1 = 9` 자릿수
   - 2자리 수 (10~99): 최대 90개의 수, 각 수가 2자리이므로 `90 * 2 = 180` 자릿수
   - 3자리 수 (100~999): 최대 900개의 수, 각 수가 3자리이므로 `900 * 3 = 2700` 자릿수
   - 위의 방식으로 반복하여, 자릿수가 l-1인 수를 계산합니다.
3. **N의 자릿수 추가**: 마지막으로, N의 자릿수에 해당하는 수를 계산합니다. 예를 들어, N이 23일 경우 (10~23)의 숫자는 14개이고, 각 숫자는 2자리이므로 `14 * 2 = 28` 자릿수 추가됩니다.
4. **결과 출력**: 모든 자릿수를 합산하여 최종 결과를 출력합니다.

예시 입력:
```
23
```
- 1자리 수 (1~9): 9
- 2자리 수 (10~23): (23 - 10 + 1) * 2 = 14 * 2 = 28
- 총 자릿수 = 9 + 28 = 37

최종 출력: `37`

### 시간 복잡도
O(log N) - N의 자릿수에 비례하여 반복문이 실행되므로, 입력 값 N에 대한 로그 복잡도를 가집니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
