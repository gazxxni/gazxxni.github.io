## 문제
창영이는 삼각형의 종류를 잘 구분하지 못한다. 따라서 프로그램을 이용해 이를 외우려고 한다.

삼각형의 세 각을 입력받은 다음, 


	세 각의 크기가 모두 60이면, Equilateral
	세 각의 합이 180이고, 두 각이 같은 경우에는 Isosceles
	세 각의 합이 180이고, 같은 각이 없는 경우에는 Scalene
	세 각의 합이 180이 아닌 경우에는 Error


를 출력하는 프로그램을 작성하시오.

---

## 입력
총 3개의 줄에 걸쳐 삼각형의 각의 크기가 주어진다. 모든 정수는 0보다 크고, 180보다 작다.

---

## 출력
문제의 설명에 따라 Equilateral, Isosceles, Scalene, Error 중 하나를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 세 각의 크기를 입력받아, 삼각형의 종류를 결정하는 프로그램을 작성한다. 각의 합이 180도인지 확인한 후, 각각의 조건에 따라 Equilateral, Isosceles, Scalene, 또는 Error을 출력한다.

### 동작 과정
1. 세 각 \( a, b, c \)를 입력받는다.
2. 먼저 세 각이 모두 60도인지 확인하여, 이 경우에는 'Equilateral'을 출력한다.
3. 다음으로, 세 각의 합이 180도인지 확인한다. 
   - 만약 두 각이 같은 경우(예: \( a = b \neq c \))라면 'Isosceles'을 출력한다.
   - 세 각이 모두 다르다면 'Scalene'을 출력한다.
4. 마지막으로, 세 각의 합이 180도가 아니라면 'Error'를 출력한다.

예시 입력:
```
60
60
60
```
- 첫 번째 조건을 만족하므로 출력: `Equilateral`

예시 입력:
```
70
70
40
```
- 세 각의 합은 180이지만 두 각이 같으므로 출력: `Isosceles`

예시 입력:
```
30
60
90
```
- 세 각의 합은 180이며 모두 다르므로 출력: `Scalene`

예시 입력:
```
100
40
50
```
- 세 각의 합이 180이 아니므로 출력: `Error`

### 시간 복잡도
- 입력을 받고 조건을 체크하는 과정은 각각 O(1) 연산이므로 전체 시간 복잡도는 O(1)이다.

## 코드
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
c = int(input())


if a == b == c == 60:
    print('Equilateral')

elif a + b + c == 180:
    if a == b and b != c:
        print('Isosceles')
    elif a == c and b != c:
        print('Isosceles')
    elif b == c and a != c:
        print('Isosceles')
    else:
        print('Scalene')

else:
    print('Error')
```
