## 문제
창영이는 삼각형의 종류를 잘 구분하지 못한다. 따라서 프로그램을 이용해 이를 외우려고 한다.

삼각형의 세 각을 입력받은 다음, 


	세 각의 크기가 모두 60이면, Equilateral
	세 각의 합이 180이고, 두 각이 같은 경우에는 Isosceles
	세 각의 합이 180이고, 같은 각이 없는 경우에는 Scalene
	세 각의 합이 180이 아닌 경우에는 Error


를 출력하는 프로그램을 작성하시오.

---

## 입력
총 3개의 줄에 걸쳐 삼각형의 각의 크기가 주어진다. 모든 정수는 0보다 크고, 180보다 작다.

---

## 출력
문제의 설명에 따라 Equilateral, Isosceles, Scalene, Error 중 하나를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 세 각을 통해 삼각형의 종류를 판단하는 문제이다. 각의 합이 180도인지 확인한 후, 각의 크기와 개수를 비교하여 삼각형의 종류를 결정한다. 세 각이 모두 60도이면 정삼각형(Equilateral), 두 각이 같고 합이 180도이면 이등변삼각형(Isosceles), 모든 각이 다르고 합이 180도이면 부등변삼각형(Scalene), 그렇지 않으면 에러(Error)를 출력한다.

### 동작 과정
1. 사용자로부터 세 개의 각을 입력받는다.
    - 예시 입력: 
        ```
        60
        60
        60
        ```

2. 입력된 세 각이 모두 60도인지 확인한다.
    - 이 경우, 출력: `Equilateral`

3. 각의 합이 180도인지 확인한다.
    - 다음 예시 입력: 
        ```
        70
        70
        40
        ```
    - 두 각(70도)이 같고 합이 180도이므로, 출력: `Isosceles`

4. 모든 각이 다르고, 합이 180도인지 확인한다.
    - 다음 예시 입력:
        ```
        60
        70
        50
        ```
    - 모든 각이 다르고 합이 180도이므로, 출력: `Scalene`

5. 각의 합이 180도가 아닌 경우를 처리한다.
    - 다음 예시 입력:
        ```
        90
        45
        50
        ```
    - 각의 합이 185도이므로, 출력: `Error`

### 시간 복잡도
O(1) - 입력받는 각의 개수가 상수이므로, 처리 시간은 입력의 수에 따라 변하지 않는다.

## 코드
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
c = int(input())


if a == b == c == 60:
    print('Equilateral')

elif a + b + c == 180:
    if a == b and b != c:
        print('Isosceles')
    elif a == c and b != c:
        print('Isosceles')
    elif b == c and a != c:
        print('Isosceles')
    else:
        print('Scalene')

else:
    print('Error')
```
