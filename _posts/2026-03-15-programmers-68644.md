## 문제
정수 배열 numbers가 주어집니다. numbers에서 서로 다른 인덱스에 있는 두 개의 수를 뽑아 더해서 만들 수 있는 모든 수를 배열에 오름차순으로 담아 return 하도록 solution 함수를 완성해주세요.



제한사항


numbers의 길이는 2 이상 100 이하입니다.


numbers의 모든 수는 0 이상 100 이하입니다.





입출력 예

        
numbers
result


        
[2,1,3,4,1]
[2,3,4,5,6,7]


[5,0,2,7]
[2,5,7,9,12]


      


입출력 예 설명

입출력 예 #1


2 = 1 + 1 입니다. (1이 numbers에 두 개 있습니다.)
3 = 2 + 1 입니다.
4 = 1 + 3 입니다.
5 = 1 + 4 = 2 + 3 입니다.
6 = 2 + 4 입니다.
7 = 3 + 4 입니다.
따라서 [2,3,4,5,6,7] 을 return 해야 합니다.


입출력 예 #2


2 = 0 + 2 입니다.
5 = 5 + 0 입니다.
7 = 0 + 7 = 5 + 2 입니다.
9 = 2 + 7 입니다.
12 = 5 + 7 입니다.
따라서 [2,5,7,9,12] 를 return 해야 합니다.

## 풀이
### 풀이 핵심 로직
주어진 정수 배열 `numbers`에서 서로 다른 인덱스에 있는 두 수를 선택하여 그 합을 구한 후, 모든 가능한 합을 중복 없이 저장하기 위해 `set` 자료구조를 사용합니다. 마지막으로, 이 합들을 정렬하여 리스트로 반환합니다.

### 동작 과정
1. 입력 배열 `numbers` = [2, 1, 3, 4, 1]이라고 가정합니다.
2. 두 개의 중첩된 루프를 사용하여 모든 서로 다른 인덱스 조합 `(i, j)`를 탐색합니다. 여기서 `i`는 첫 번째 수의 인덱스이고, `j`는 두 번째 수의 인덱스입니다. 두 인덱스가 같지 않도록 확인합니다.
3. 각 조합에 대해, `numbers[i] + numbers[j]`의 값을 계산하여 `set`에 추가합니다.
   - 예: `numbers[0] + numbers[1]` = 2 + 1 = 3, `set`에 추가
   - 예: `numbers[0] + numbers[2]` = 2 + 3 = 5, `set`에 추가
   - 이 과정을 모든 인덱스 조합에 대해 반복합니다.
4. 최종적으로, `set`에 저장된 값을 리스트로 변환하고 오름차순으로 정렬합니다.
5. 정렬된 리스트를 반환합니다.

예시 입력 [2, 1, 3, 4, 1]에 대해 수행한 결과는 다음과 같습니다:
- 가능한 합: 2, 3, 4, 5, 6, 7
- 반환되는 결과: [2, 3, 4, 5, 6, 7]

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 여기서 n은 `numbers`의 길이입니다. 중첩된 루프를 통해 각 숫자 쌍을 한 번씩 확인하므로, 최악의 경우 n(n-1)번의 합을 계산하게 됩니다.

## 코드
```python
def solution(numbers):
    answer = set()
    n = len(numbers)
    for i in range(n):
        for j in range(n):
            if i != j:
                answer.add(numbers[i] + numbers[j])
    answer = list(answer)
    answer.sort()
    return answer
```
