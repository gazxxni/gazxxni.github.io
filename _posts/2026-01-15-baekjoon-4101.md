## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 두 개의 양의 정수 a와 b에 대해 a가 b보다 큰지를 비교하여 결과를 출력하는 문제입니다. 입력이 "0 0"일 때까지 반복적으로 두 수를 입력받고, a가 b보다 크면 "Yes", 그렇지 않으면 "No"를 출력합니다.

### 동작 과정
1. 프로그램이 시작되면 무한 루프를 통해 입력을 계속 받습니다.
2. 사용자가 두 정수 a와 b를 입력합니다. 예를 들어, 첫 번째 입력이 "5 3"이라면 a는 5, b는 3이 됩니다.
3. a와 b를 비교하여 a > b가 참이면 "Yes"를 출력하고, 그렇지 않으면 "No"를 출력합니다.
4. 이 과정을 반복하다가 "0 0"이 입력되면 루프를 종료하고 프로그램이 종료됩니다.
5. 예시 입력 및 출력:
   - 입력: "5 3" → 출력: "Yes"
   - 입력: "2 4" → 출력: "No"
   - 입력: "0 0" → 프로그램 종료

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 각 테스트 케이스에 대해 상수 시간 내에 비교와 출력을 수행하기 때문입니다. 그러나 전체 입력에 대한 처리 시간은 입력받는 테스트 케이스의 수에 따라 달라지므로, 최악의 경우 O(n)으로 표현할 수 있습니다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
