## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 풀이

### 풀이 핵심 로직
이 문제는 두 양의 정수를 입력받아 첫 번째 수가 두 번째 수보다 큰지를 판단하는 간단한 구현 문제입니다. 입력이 "0 0"일 때까지 반복하여 각 테스트 케이스에 대해 "Yes" 또는 "No"를 출력합니다.

### 동작 과정
1. 프로그램이 시작되면 무한 루프에 들어갑니다.
2. 각 반복에서 사용자로부터 두 개의 정수를 입력받습니다.
3. 입력받은 두 정수가 모두 0인지 확인합니다. 이 경우 루프를 종료합니다.
4. 첫 번째 수가 두 번째 수보다 큰 경우 "Yes"를 출력하고, 그렇지 않은 경우 "No"를 출력합니다.
5. 이 과정을 입력이 "0 0"일 때까지 반복합니다.

예시 입력: 
```
3 2
4 5
0 0
```
- 첫 번째 입력인 `3 2`에서 `3 > 2`가 참이므로 "Yes" 출력.
- 두 번째 입력인 `4 5`에서 `4 > 5`가 거짓이므로 "No" 출력.
- 마지막 입력인 `0 0`에서 루프 종료.

### 시간 복잡도
이 문제의 시간 복잡도는 O(n)입니다. 여기서 n은 입력된 테스트 케이스의 개수입니다. 각 테스트 케이스에 대해 상수 시간 O(1) 내에 작업을 수행하므로 n개의 테스트 케이스에 대해 O(n) 시간이 소요됩니다.

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
