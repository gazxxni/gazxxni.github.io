## 풀이

### 풀이 핵심 로직
문제는 주어진 세 변을 이용해 삼각형을 만들 수 있는지 여부와 그에 따른 가장 긴 변의 길이를 찾는 것입니다. 삼각형의 성질에 따라 두 변의 합이 나머지 한 변보다 커야 합니다. 이를 통해 가능한 삼각형의 둘레를 계산하고, 삼각형을 만들 수 없는 경우와 특별한 경우를 처리합니다.

### 동작 과정
1. **입력 처리**: 세 변의 길이 `a`, `b`, `c`를 입력받습니다.
   - 예시 입력: `3 4 5`
   
2. **정렬**: 입력받은 변의 길이를 오름차순으로 정렬합니다. 이때 `a`, `b`, `c`는 각각 가장 짧은 변에서 가장 긴 변으로 배치됩니다.
   - 정렬 후: `a = 3`, `b = 4`, `c = 5`
   
3. **삼각형 검증**:
   - `a + b <= c` 조건을 확인하여 삼각형을 만들 수 있는지 판단합니다. 이 경우 삼각형을 만들 수 없으므로 둘레는 `(a + b) * 2 - 1`로 계산됩니다.
   - 예시: `3 + 4 > 5` 이므로 이 조건은 성립하지 않습니다.
   
4. **모든 변이 같은 경우**: `a == b and b == c` 조건을 확인하여 삼각형의 모든 변이 같은지 체크합니다. 이 경우 둘레는 `a * 3`으로 계산됩니다.
   - 예시: 이 경우는 성립하지 않습니다.
   
5. **삼각형의 둘레 계산**: 위의 두 조건이 모두 성립하지 않으면, 세 변의 합이 삼각형의 둘레가 됩니다.
   - 예시: `3 + 4 + 5 = 12`가 출력됩니다.

### 시간 복잡도
O(1) - 입력의 길이는 고정되어 있으며, 정렬도 상수 시간에 처리되기 때문에 시간 복잡도는 상수 시간입니다.

---

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
