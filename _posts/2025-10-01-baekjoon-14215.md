## 문제
영선이는 길이가 a, b, c인 세 막대를 가지고 있고, 각 막대의 길이를 마음대로 줄일 수 있다.

영선이는 세 막대를 이용해서 아래 조건을 만족하는 삼각형을 만들려고 한다.


	각 막대의 길이는 양의 정수이다
	세 막대를 이용해서 넓이가 양수인 삼각형을 만들 수 있어야 한다.
	삼각형의 둘레를 최대로 해야 한다.


a, b, c가 주어졌을 때, 만들 수 있는 가장 큰 둘레를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 100)가 주어진다.

---

## 출력
첫째 줄에 만들 수 있는 가장 큰 삼각형의 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 주어진 세 막대 길이로 삼각형을 만들 수 있는 조건을 만족하면서, 둘레를 최대화하는 것입니다. 삼각형의 성립 조건인 두 변의 합이 다른 변보다 커야 하며, 이를 만족하지 못할 경우 막대를 줄여서 최대한의 둘레를 만들어야 합니다.

### 동작 과정
1. 입력받은 세 막대의 길이를 `a`, `b`, `c`로 정의하고, 이를 정렬합니다. 이때 `a <= b <= c`가 되도록 합니다.
2. 삼각형의 성립 조건을 확인합니다:
   - 만약 `a + b <= c`이면, 삼각형을 만들 수 없으므로, 두 막대의 길이의 합에 1을 빼서 최대 둘레를 계산합니다. 이때, 홀수로 만들기 위해 결과값을 `(a + b) * 2 - 1`로 설정합니다.
   - 만약 세 막대의 길이가 모두 같다면 (`a == b == c`) 최대 둘레는 `3 * a`입니다.
   - 그 외의 경우에는 `a + b + c`가 최대 둘레입니다.

#### 예시 입력으로 단계별 설명
- 입력: `3 4 5`
  - 정렬 후: `a = 3`, `b = 4`, `c = 5`
  - 조건 확인: `3 + 4 > 5`, 삼각형을 만들 수 있음.
  - 출력: `3 + 4 + 5 = 12`

- 입력: `1 1 3`
  - 정렬 후: `a = 1`, `b = 1`, `c = 3`
  - 조건 확인: `1 + 1 <= 3`, 삼각형을 만들 수 없음.
  - 출력: `(1 + 1) * 2 - 1 = 3`

### 시간 복잡도
O(1) - 주어진 세 개의 수에 대해 정렬 및 간단한 조건문으로 처리되므로 상수 시간 복잡도를 가집니다.

---

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
