## 문제
N$N$개의 정점으로 이루어진 트리의 모든 정점 쌍에 대하여, 두 정점 사이의 거리의 합을 최소화하시오. 정점에는 1$1$부터 N$N$까지 번호가 매겨져 있다.

즉, 정점 i$i$와 정점 j$j$ 사이의 거리를 dist(i,j)$\textrm {dist}(i,\, j)$라 할 때  ∑1≤ij≤Ndist(i,j)$\sum_{1 \leq i 가 최소인 트리를 구성해야 한다.

---

## 입력
첫째 줄에 정점의 개수 N$N$이 주어진다. (2≤N≤100000)$(2\leq N \leq 100\,000)$

---

## 출력
첫째 줄에 모든 정점 쌍에 대하여 두 정점 사이 거리의 합의 최솟값을 출력한다.

둘째 줄부터 N−1$N-1$개의 줄에 걸쳐, 트리를 이루는 간선의 정보를 출력한다. 각 줄에 간선이 연결하는 두 정점의 번호를 공백을 사이에 두고 출력한다.

조건을 만족하는 트리가 여럿이라면 아무거나 하나 출력한다.

## 풀이

### 풀이 핵심 로직
트리의 모든 정점 쌍 간의 거리를 최소화하기 위해, 한 정점을 중심에 두고 나머지 정점들을 연결하는 구조를 사용합니다. 이를 통해 중심 정점과의 거리는 1로 설정하고, 서로 다른 정점 간의 거리는 2로 설정하여 총 거리를 최소화합니다. 이 구조는 항상 모든 정점 쌍 간의 거리 합을 최소로 유지합니다.

### 동작 과정
1. 입력으로 정점의 개수 \( N \)을 받습니다.
2. 모든 정점 쌍의 거리 합을 계산하는 공식은 \( (N-1)^2 \)입니다. 이는 중심 정점과 각 정점 간의 거리 합과 나머지 정점들 간의 거리 합을 계산한 결과입니다.
3. 중심 정점은 1로 설정하고, 1과 나머지 정점들(2부터 N까지)을 연결하는 간선을 출력합니다.
4. 출력은 거리 합과 각 간선 정보입니다.

예시 입력:
```
5
```
예시 출력:
```
16
1 2
1 3
1 4
1 5
```
- 여기서, 정점 1과 나머지 정점들(2, 3, 4, 5) 간의 거리는 1이며, 나머지 정점들 간의 거리는 2입니다. 
- 모든 정점 쌍 간의 거리 합은 \( (5-1)^2 = 16 \)입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(N) \)입니다. 입력을 읽고, 간선을 출력하는 과정에서 N번의 작업이 필요하기 때문입니다.

## 코드
```python
'''
1. 가운데 정점과 모든 정점과의 거리 1
2. 나머지 정점들끼리의 거리 2

n개 일때
1. n-1
2. (n-2 + n-3 + n-4 + ... + 1) * 2
    = (n-2)(n-1)
    
1, 2번은 더하면 (n-1) ** 2
'''

n = int(input())

print((n-1) ** 2)
for i in range(2, n+1):
    print(1, i)
```
