## 문제
오늘도 서준이는 병합 정렬 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

N개의 서로 다른 양의 정수가 저장된 배열 A가 있다. 병합 정렬로 배열 A를 오름차순 정렬할 경우 배열 A에 K 번째 저장되는 수를 구해서 우리 서준이를 도와주자.

크기가 N인 배열에 대한 병합 정렬 의사 코드는 다음과 같다.

merge_sort(A[p..r]) { # A[p..r]을 오름차순 정렬한다.
    if (p < r) then {
        q <- ⌊(p + r) / 2⌋;       # q는 p, r의 중간 지점
        merge_sort(A, p, q);      # 전반부 정렬
        merge_sort(A, q + 1, r);  # 후반부 정렬
        merge(A, p, q, r);        # 병합
    }
}

---

## 입력
첫째 줄에 배열 A의 크기 N(5 ≤ N ≤ 500,000), 저장 횟수 K(1 ≤ K ≤ 108)가 주어진다.

다음 줄에 서로 다른 배열 A의 원소 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 109)

---

## 출력
배열 A에 K 번째 저장 되는 수를 출력한다. 저장 횟수가 K 보다 작으면 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
병합 정렬을 사용하여 주어진 배열을 정렬하면서, 정렬 과정 중 배열에 저장되는 횟수를 카운트하여 K 번째 저장되는 값을 찾아낸다. 이를 위해 병합 과정에서 배열의 특정 인덱스에 값을 저장할 때마다 카운트를 증가시키고, K와 비교하여 K 번째 저장되는 값을 기록한다.

### 동작 과정
예시 입력으로 N=5, K=3, 배열 A=[5, 4, 3, 2, 1]을 가정하자.

1. **초기 상태**: A = [5, 4, 3, 2, 1]
2. **병합 정렬 시작**: A[0..4]를 정렬하기 위해 중간 인덱스 q를 2로 설정.
3. **왼쪽 부분 정렬**: A[0..2] = [5, 4, 3]을 정렬.
   - q를 1로 설정하여 A[0..1]과 A[2]로 나누고, A[0..1]을 정렬.
   - A[0]과 A[1] 비교 후 A[0] (5)와 A[1] (4)을 병합하여 A[0..1] = [4, 5]가 된다.
   - A[0..2]에 A[2] (3)를 병합하면 A[0..2] = [3, 4, 5]가 된다.
4. **오른쪽 부분 정렬**: A[3..4] = [2, 1]을 정렬.
   - A[3] (2)와 A[4] (1)을 병합하여 A[3..4] = [1, 2]가 된다.
5. **전체 병합**: A[0..2]와 A[3..4]를 병합할 때 저장 횟수를 카운트하면서 최종 배열 A는 [1, 2, 3, 4, 5]가 된다.
   - 저장 과정에서 1, 2, 3번째 저장이 이뤄지며, 3번째 저장 시 A[2] (3)가 K번째 저장되는 값이 된다.

최종적으로 출력은 3이 된다.

### 시간 복잡도
병합 정렬의 시간 복잡도는 O(N log N)이다. 여기서 N은 배열의 크기이다.

## 코드
```python
def merge_sort(A, p, r):
    if p < r:
        q = (p + r) // 2
        merge_sort(A, p, q)
        merge_sort(A, q + 1, r)
        merge(A, p, q, r)

def merge(A, p, q, r):
    global cnt, ans
    
    i = p
    j = q + 1
    t = 0
    tmp = [0] * (r - p + 1)
    
    while i <= q and j <= r:
        if A[i] <= A[j]:
            tmp[t] = A[i]
            i += 1
        else:
            tmp[t] = A[j]
            j += 1
        t += 1
        
    while i <= q:
        tmp[t] = A[i]
        i += 1
        t += 1
        
    while j <= r:
        tmp[t] = A[j]
        j += 1
        t += 1
        
    i = p
    t = 0
    
    while i <= r:
        A[i] = tmp[t]
        cnt += 1
        
        if cnt == k:
            ans = A[i]
            
        i += 1
        t += 1

n, k = map(int, input().split())
arr = list(map(int, input().split()))

cnt = 0
ans = -1

merge_sort(arr, 0, n - 1)

print(ans)
```
