## 문제
때는 2040년, 이민혁은 우주에 자신만의 왕국을 만들었다. 왕국은 N개의 행성으로 이루어져 있다. 민혁이는 이 행성을 효율적으로 지배하기 위해서 행성을 연결하는 터널을 만들려고 한다.


	행성은 3차원 좌표위의 한 점으로 생각하면 된다. 두 행성 A(xA, yA, zA)와 B(xB, yB, zB)를 터널로 연결할 때 드는 비용은 min(|xA-xB|, |yA-yB|, |zA-zB|)이다.


	민혁이는 터널을 총 N-1개 건설해서 모든 행성이 서로 연결되게 하려고 한다. 이때, 모든 행성을 터널로 연결하는데 필요한 최소 비용을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 행성의 개수 N이 주어진다. (1 ≤ N ≤ 100,000) 다음 N개 줄에는 각 행성의 x, y, z좌표가 주어진다. 좌표는 -109보다 크거나 같고, 109보다 작거나 같은 정수이다. 한 위치에 행성이 두 개 이상 있는 경우는 없다.

---

## 출력
첫째 줄에 모든 행성을 터널로 연결하는데 필요한 최소 비용을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 최소 스패닝 트리(MST)를 찾는 문제로, 행성 간의 터널 비용을 최소화해야 합니다. 비용은 행성 간의 x, y, z 좌표의 차이 중 최소값으로 정의됩니다. 이를 위해 각 좌표 축별로 행성을 정렬한 후, 인접한 행성 쌍 사이의 비용을 계산하여 그래프의 간선으로 추가하고, 크루스칼 알고리즘을 사용하여 MST를 구성합니다.

### 동작 과정
1. **입력 예시**: 4개의 행성이 주어지고, 각 행성의 좌표는 다음과 같습니다.
   ```
   1 1 1
   2 2 2
   3 3 3
   6 6 6
   ```

2. **좌표 정렬**: x, y, z 좌표 각각에 대해 행성을 정렬합니다.
   - x 정렬: `[(1, 0), (2, 1), (3, 2), (6, 3)]`
   - y 정렬: `[(1, 0), (2, 1), (3, 2), (6, 3)]`
   - z 정렬: `[(1, 0), (2, 1), (3, 2), (6, 3)]`

3. **간선 생성**: 각 축별로 인접한 행성 쌍 사이의 비용을 계산하여 간선을 생성합니다.
   - x축 간선: 
     - (1, 0) - (2, 1) : 비용 = 1
     - (2, 1) - (3, 2) : 비용 = 1
     - (3, 2) - (6, 3) : 비용 = 3
   - y축과 z축의 간선도 동일하게 생성됩니다.

4. **간선 정렬**: 모든 간선을 비용 기준으로 정렬합니다.
   ```
   [(1, 0, 1), (1, 1, 2), (1, 0, 1), (1, 1, 2), (3, 2, 3)]
   ```

5. **크루스칼 알고리즘 실행**: 비용이 낮은 간선부터 선택하여 MST를 구성합니다. 사이클이 발생하지 않도록 유니온 파인드(Union-Find) 자료구조를 사용하여 연결합니다.
   - 선택된 간선: (0, 1), (1, 2), (2, 3) -> 총 비용 = 3

6. **결과 출력**: 모든 행성을 연결하는 최소 비용은 3입니다.

### 시간 복잡도
- 정렬 단계: O(N log N) (각 축마다 정렬 3회)
- 간선 생성: O(N) (N-1개의 간선 생성)
- 크루스칼 알고리즘: O(E log E) (E는 간선의 개수, 최악의 경우 3N)
  
따라서 전체 시간 복잡도는 O(N log N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

def find(parent, x):
    if parent[x] != x:
        parent[x] = find(parent, parent[x])
    return parent[x]

def union(parent, a, b):
    a = find(parent, a)
    b = find(parent, b)
    
    if a < b:
        parent[b] = a
    else:
        parent[a] = b
        
xx, yy, zz = [], [], []
for i in range(n):
    x, y, z = map(int, input().split())
    xx.append((x, i))
    yy.append((y, i))
    zz.append((z, i))
    
xx.sort()
yy.sort()
zz.sort()

edge = []
for cur in xx, yy, zz:
    for i in range(1, n):
        w1, a = cur[i - 1]
        w2, b = cur[i]
        edge.append((abs(w1 - w2), a, b))
        
edge.sort()

parent = [i for i in range(n)]

def krustal():
    ans = 0
    
    for cost, a, b in edge:
        if find(parent, a) != find(parent, b):
            union(parent, a, b)
            ans += cost
    
    return ans

print(krustal())
```
