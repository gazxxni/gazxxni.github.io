## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 두 분수의 합을 계산하기 위해, 먼저 두 분수의 분모의 최소 공배수(LCM)를 구합니다. 그 후, 두 분수의 분자를 동일한 분모로 변환하여 합을 구한 다음, 최종 결과를 기약분수 형태로 만들기 위해 최대 공약수(GCD)로 나누어줍니다.

### 동작 과정
1. 입력:
   - 첫 번째 분수: 1/2 (즉, A=1, B=2)
   - 두 번째 분수: 1/3 (즉, C=1, D=3)

2. 각 분수를 입력받아 변수에 저장합니다:
   ```python
   a, b = 1, 2  # 첫 번째 분수
   c, d = 1, 3  # 두 번째 분수
   ```

3. GCD 함수를 통해 두 분모 (B=2, D=3)의 최소 공배수를 계산합니다:
   - LCM(2, 3) = 6

4. 두 분수를 동일한 분모로 변환합니다:
   - `aa = n // b = 6 // 2 = 3`
   - `bb = n // d = 6 // 3 = 2`

5. 변환된 분모를 사용하여 두 분수의 분자를 더합니다:
   - `cc = aa * a + bb * c = 3 * 1 + 2 * 1 = 3 + 2 = 5`

6. 결과 분수는 5/6입니다. 이 상태에서 GCD를 구하여 기약분수로 변환합니다:
   - `dd = gcd(5, 6) = 1`
   - 기약분수는 `5 // 1`과 `6 // 1` 이므로 결과는 5/6입니다.

7. 최종 출력:
   - `print(int(cc // dd), int(n // dd))`는 `5 6`을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(log(min(B, D)))입니다. 이는 GCD를 계산하는 데 사용되는 유클리드 호제법의 시간 복잡도에 기반합니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
