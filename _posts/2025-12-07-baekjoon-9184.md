## 문제
재귀 호출만 생각하면 신이 난다! 아닌가요?

다음과 같은 재귀함수 w(a, b, c)가 있다.

if a  20 or b > 20 or c > 20, then w(a, b, c) returns:
    w(20, 20, 20)

if a 

위의 함수를 구현하는 것은 매우 쉽다. 하지만, 그대로 구현하면 값을 구하는데 매우 오랜 시간이 걸린다. (예를 들면, a=15, b=15, c=15)

a, b, c가 주어졌을 때, w(a, b, c)를 출력하는 프로그램을 작성하시오.

---

## 입력
입력은 세 정수 a, b, c로 이루어져 있으며, 한 줄에 하나씩 주어진다. 입력의 마지막은 -1 -1 -1로 나타내며, 세 정수가 모두 -1인 경우는 입력의 마지막을 제외하면 없다.

---

## 출력
입력으로 주어진 각각의 a, b, c에 대해서, w(a, b, c)를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 재귀 함수 `w(a, b, c)`의 결과를 효율적으로 계산하기 위해 메모이제이션 기법을 사용합니다. 주어진 `a`, `b`, `c` 값이 특정 조건을 만족할 때, 재귀 호출을 통해 이전에 계산된 값을 저장하여 중복 호출을 방지합니다. 이로 인해 계산 속도를 크게 향상시킬 수 있습니다.

### 동작 과정
1. **입력 처리:** 사용자로부터 `a`, `b`, `c` 값이 입력됩니다. 입력이 `-1 -1 -1`이 되면 종료합니다.
2. **조건 검사 및 반환값 결정:**
   - `a`, `b`, `c` 중 하나라도 0 이하이면 1을 반환합니다.
   - `a`, `b`, `c` 중 하나라도 20을 초과하면 `w(20, 20, 20)`을 호출합니다.
   - 이전에 계산된 값이 존재하면 해당 값을 반환합니다.
3. **재귀 호출:** 
   - 만약 `a < b < c`일 경우, `w(a, b, c)`는 `w(a, b, c-1) + w(a, b-1, c-1) - w(a, b-1, c)`로 정의됩니다.
   - 그 외의 경우에는 `w(a, b, c)`는 `w(a-1, b, c) + w(a-1, b-1, c) + w(a-1, b, c-1) - w(a-1, b-1, c-1)`으로 정의됩니다.
4. **출력:** 각 입력에 대해 계산된 결과를 출력합니다.

예시 입력:
```
1 2 3
2 3 4
10 20 20
15 15 15
-1 -1 -1
```
출력 결과:
```
w(1, 2, 3) = 2
w(2, 3, 4) = 3
w(10, 20, 20) = 1048576
w(15, 15, 15) = 43758
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. `a`, `b`, `c`의 값이 0 이상 20 이하로 제한되어 있고, 각 값에 대해 최대 21x21x21의 메모리 공간을 사용하여 결과를 저장하기 때문에, 모든 입력에 대해 상수 시간 내에 결과를 반환할 수 있습니다.

## 코드
```python
def w(a, b, c):
    if a <= 0 or b <= 0 or c <= 0:
        return 1
    
    if a > 20 or b > 20 or c > 20:
        return w(20, 20, 20)
    
    if dp[a][b][c]:
        return dp[a][b][c]
    
    if a < b < c:
        dp[a][b][c] = w(a, b, c - 1) + w(a, b - 1, c- 1) - w(a, b - 1, c)
        return dp[a][b][c]
    
    dp[a][b][c] = w(a - 1, b, c) + w(a - 1, b - 1, c) + w(a - 1, b, c - 1) - w(a - 1, b - 1, c - 1)
    return dp[a][b][c]

dp = [[[0] * 21 for _ in range(21)] for _ in range(21)]

while True:
    a, b, c = map(int, input().split())

    if a == -1 and b == -1 and c == -1:
        exit()
    
    print(f'w({a}, {b}, {c}) = {w(a, b, c)}')
```
