## 풀이

### 풀이 핵심 로직
이 문제는 연속적인 K일 간의 온도 합이 최대가 되는 경우를 찾는 문제로, 슬라이딩 윈도우 기법을 사용하여 효율적으로 해결할 수 있습니다. 처음 K일의 합을 계산한 후, 이후 날마다 슬라이딩 윈도우를 이동하면서 이전 K일의 온도를 빼고 새로운 온도를 더하여 현재 합을 갱신하며 최대 합을 찾아냅니다.

### 동작 과정
1. 입력으로 N과 K를 받습니다. 예를 들어, N=10, K=3이라고 가정합니다.
2. 다음으로 N개의 온도를 입력받습니다. 예를 들어, 온도 배열이 `[3, -2, -4, -9, 0, 3, 7, 13, 8, -3]`라고 합니다.
3. 처음 K일(즉, 처음 3일)의 온도 합을 계산합니다. `3 + (-2) + (-4) = -3`입니다.
4. 이 값을 초기 최대값으로 설정합니다.
5. 이제 K일 이후의 날부터 시작하여 다음 과정을 반복합니다:
   - 슬라이딩 윈도우를 이동하여, 이전 K일의 첫 번째 온도(3일 전의 온도)를 빼고, 현재 날의 온도를 더합니다.
   - 예를 들어, 4일째(온도 -9)부터 시작하면, `-3 - 3 + (-9) = -9`가 됩니다.
   - 이 과정을 N까지 반복하며 매번 최대값을 갱신합니다.
6. 모든 반복이 끝난 후, 최대 합을 출력합니다.

### 시간 복잡도
이 알고리즘은 배열을 한 번만 순회하므로 시간 복잡도는 O(N)입니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
