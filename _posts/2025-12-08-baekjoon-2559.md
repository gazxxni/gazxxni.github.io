## 입력
첫째 줄에는 두 개의 정수 N과 K가 한 개의 공백을 사이에 두고 순서대로 주어진다. 첫 번째 정수 N은 온도를 측정한 전체 날짜의 수이다. N은 2 이상 100,000 이하이다. 두 번째 정수 K는 합을 구하기 위한 연속적인 날짜의 수이다. K는 1과 N 사이의 정수이다. 둘째 줄에는 매일 측정한 온도를 나타내는 N개의 정수가 빈칸을 사이에 두고 주어진다. 이 수들은 모두 -100 이상 100 이하이다.

## 풀이

### 풀이 핵심 로직
이 문제는 연속된 K일 동안의 온도 합이 가장 큰 값을 찾는 문제로, 슬라이딩 윈도우 기법을 사용하여 O(N) 시간 복잡도로 해결할 수 있습니다. 처음 K일의 온도 합을 계산한 후, 다음 날부터는 이전 K일의 합에서 가장 오래된 날의 온도를 빼고 새로운 날의 온도를 더하는 방식으로 연속 합을 효율적으로 업데이트합니다.

### 동작 과정
1. 입력으로 온도 측정 일수 N과 연속 일수 K를 받습니다.
2. 온도 수열을 리스트로 저장합니다.
3. 처음 K일의 합을 계산하여 `ss`에 저장하고, 이를 `ans`에 초기값으로 설정합니다.
4. K일 이후부터 N일 까지 반복하며:
   - 현재의 K일 합 `ss`에서 K일 전의 온도를 빼고, 새로운 날의 온도를 더하여 업데이트합니다.
   - 업데이트된 `ss`가 이전 최대값 `ans`보다 크면, `ans`를 업데이트합니다.
5. 모든 반복이 끝난 후, `ans`를 출력합니다.

예시 입력:
```
10 3
3 -2 -4 -9 0 3 7 13 8 -3
```

단계별 설명:
- 처음 K=3일 동안의 온도 합: 3 + (-2) + (-4) = -3
- 다음으로 K=3일 동안의 온도 합을 업데이트:
  - i=3: -3 - 3 + (-9) = -9 (ans는 -3)
  - i=4: -9 - (-2) + 0 = -7 (ans는 -3)
  - i=5: -7 - (-4) + 3 = 0 (ans는 -3)
  - i=6: 0 - 9 + 7 = -2 (ans는 -3)
  - i=7: -2 - 0 + 13 = 11 (ans는 11)
  - i=8: 11 - 3 + 8 = 16 (ans는 16)
  - i=9: 16 - 7 - 3 = 6 (ans는 16)
  
최종적으로 최대 합은 16입니다.

### 시간 복잡도
O(N) - N일의 온도 데이터를 한 번만 순회하기 때문에, 시간 복잡도는 선형입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
