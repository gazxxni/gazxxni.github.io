## 입력

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 온도 수열에서 연속적인 K일 동안의 온도 합이 최대가 되는 값을 구하는 문제로, 슬라이딩 윈도우 기법을 활용하여 효율적으로 해결할 수 있습니다. 초기 K일 동안의 합을 계산한 후, 다음 날부터는 이전의 합에서 K일 전의 온도를 빼고 현재 온도를 더하는 방식으로 연속적인 K일의 합을 구합니다.

### 동작 과정
1. **입력 받기**: N(온도 측정 일수)과 K(연속적인 일수)를 입력받고, N개의 온도를 리스트로 저장합니다.
2. **초기 합 계산**: 첫 K일 동안의 온도 합을 계산합니다. 예를 들어, 주어진 입력이 `3 -2 -4 -9 0 3 7 13 8 -3`이고 K가 3이라면, 초기 합은 `3 - 2 - 4 = -3`입니다.
3. **슬라이딩 윈도우 적용**: 이후 각 날마다 K일 전의 온도를 빼고 현재 날의 온도를 더하여 새로운 K일 간의 합을 계산합니다. 이 과정에서 최대 합을 갱신합니다.
   - 예를 들어, K가 3일 경우 첫 번째 슬라이딩 후의 합은 `-3 - (-9) + 0 = 6`이 되고, 이 과정을 반복하여 최대 합을 찾아냅니다.
4. **결과 출력**: 최종적으로 최대 합을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 이는 초기 K일 합을 계산하는 O(K)와 이후 K일 간의 합을 O(N-K) 번 계산하는 과정이 포함되어 있어, 전체적으로 선형 시간 복잡도를 가집니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
