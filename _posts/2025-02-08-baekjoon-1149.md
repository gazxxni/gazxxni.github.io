## 풀이

### 풀이 핵심 로직
이 문제는 동적 프로그래밍을 활용하여 최소 비용으로 집을 칠하는 방법을 찾는 것입니다. 각 집을 세 가지 색 중 하나로 칠할 때, 이전 집과 같은 색을 칠할 수 없으므로, 각 집의 최소 비용을 이전 집들의 색상 비용을 참고하여 갱신합니다. 이를 통해 최종적으로 n번째 집을 칠하는 최소 비용을 계산합니다.

### 동작 과정
예를 들어, 다음과 같은 입력이 주어진다고 가정해 보겠습니다:
```
3
26 40 83
49 60 57
13 89 99
```
1. **입력 처리**: `n`은 3이고, `arr` 리스트는 다음과 같이 초기화됩니다:
   ```
   arr = [
       [26, 40, 83],
       [49, 60, 57],
       [13, 89, 99]
   ]
   ```
   
2. **첫 번째 집 (i=0)**: 이미 주어진 비용이므로 변경하지 않습니다.

3. **두 번째 집 (i=1)**:
   - 1번 집의 1색과 2색을 고려하여 0색의 비용을 갱신:
     ```
     arr[1][0] = min(arr[0][1], arr[0][2]) + arr[1][0] = min(40, 83) + 49 = 89
     ```
   - 1번 집의 0색과 2색을 고려하여 1색의 비용을 갱신:
     ```
     arr[1][1] = min(arr[0][0], arr[0][2]) + arr[1][1] = min(26, 83) + 60 = 86
     ```
   - 1번 집의 0색과 1색을 고려하여 2색의 비용을 갱신:
     ```
     arr[1][2] = min(arr[0][0], arr[0][1]) + arr[1][2] = min(26, 40) + 57 = 83
     ```
   - 업데이트된 `arr`은 다음과 같습니다:
     ```
     arr = [
         [26, 40, 83],
         [89, 86, 83],
         [13, 89, 99]
     ]
     ```

4. **세 번째 집 (i=2)**:
   - 2번 집의 1색과 2색을 고려하여 0색의 비용을 갱신:
     ```
     arr[2][0] = min(arr[1][1], arr[1][2]) + arr[2][0] = min(86, 83) + 13 = 96
     ```
   - 2번 집의 0색과 2색을 고려하여 1색의 비용을 갱신:
     ```
     arr[2][1] = min(arr[1][0], arr[1][2]) + arr[2][1] = min(89, 83) + 89 = 172
     ```
   - 2번 집의 0색과 1색을 고려하여 2색의 비용을 갱신:
     ```
     arr[2][2] = min(arr[1][0], arr[1][1]) + arr[2][2] = min(89, 86) + 99 = 185
     ```
   - 최종 업데이트된 `arr`은 다음과 같습니다:
     ```
     arr = [
         [26, 40, 83],
         [89, 86, 83],
         [96, 172, 185]
     ]
     ```

5. **최종 결과**: `arr[n-1]`의 최소값을 출력하여 최소 비용을 확인합니다.
   ```
   print(min(arr[2][0], arr[2][1], arr[2][2]))  # Output: 96
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 각 집마다 상수 시간(3번의 연산)으로 갱신하므로, 총 n개의 집에 대해 O(n) 시간이 소요됩니다.

## 코드
```python
import sys

n = int(input())
arr = [list(map(int, input().split())) for _ in range(n)]

for i in range(1, n):
    arr[i][0] = min(arr[i-1][1], arr[i-1][2]) + arr[i][0]
    arr[i][1] = min(arr[i-1][0], arr[i-1][2]) + arr[i][1]
    arr[i][2] = min(arr[i-1][0], arr[i-1][1]) + arr[i][2]

print(min(arr[n-1][0], arr[n-1][1], arr[n-1][2]))
```
