## 풀이

### 풀이 핵심 로직
문제는 주어진 `n`에 대해 `n^3` 값을 계산하고, 알고리즘의 시간 복잡도를 출력하는 것입니다. 여기서 `n`은 입력으로 주어지는 자연수이며, 알고리즘의 시간 복잡도는 항상 O(1)로 주어져 있습니다. 따라서, `n`의 세제곱을 계산하고, 상수 값을 출력하는 매우 간단한 구조입니다.

### 동작 과정
1. 사용자로부터 정수 `n`을 입력받습니다. (예: `2`)
2. `n`의 세제곱을 계산하여 출력합니다. (예: `2^3 = 8`)
3. 알고리즘의 시간 복잡도를 상수 `3`으로 출력합니다. (여기서는 고정된 값으로, 문제의 요구사항에 따라 다를 수 있음)

입력 예시:
```
2
```
출력:
```
8
3
```

### 시간 복잡도
O(1)  
이 알고리즘은 입력 크기에 관계없이 일정한 시간 내에 실행되므로 상수 시간 복잡도를 가집니다.

---

## 코드
```python
n = int(input())
print(n ** 3)
print(3)
```
