## 문제
연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.

이를 출력하는 프로그램을 작성해보도록 하자.

---

## 입력
첫째 줄에 N이 주어진다. (N = 0 또는 1)

---

## 출력
N = 0일 경우: 연세대학교의 영문명을 출력한다.
	N = 1일 경우: 연세대학교의 슬로건을 출력한다.


대소문자 구별에 주의하도록 하자.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
문제는 입력 값 N에 따라 두 가지 문자열 중 하나를 출력하는 간단한 조건문 구조를 요구한다. N이 0일 경우 "YONSEI"를 출력하고, N이 1일 경우 "Leading the Way to the Future"를 출력하면 된다.

### 동작 과정
1. 사용자로부터 입력을 받는다. 이 입력은 정수 N으로, 0 또는 1의 값을 가진다.
2. 입력된 N의 값이 0인지 확인한다. 
   - 만약 0이라면 "YONSEI"를 출력한다.
3. N의 값이 1인지 확인한다.
   - 만약 1이라면 "Leading the Way to the Future"를 출력한다.

예시 입력 및 출력:
- 입력: `0`
  - 출력: `YONSEI`
  
- 입력: `1`
  - 출력: `Leading the Way to the Future`

### 시간 복잡도
이 문제의 시간 복잡도는 O(1)이다. 입력된 값을 조건문으로 확인하고 해당 결과를 출력하는 과정이 상수 시간에 이루어지기 때문이다.

---

## 코드
```python
n = int(input())

if n == 0:
    print("YONSEI")
else:
    print("Leading the Way to the Future")
```
