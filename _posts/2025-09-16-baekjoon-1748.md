## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 1부터 N까지의 모든 수를 이어 썼을 때 총 자릿수를 계산하는 문제입니다. 각 자리 수에 따라 수의 개수가 어떻게 변화하는지를 이용하여, 1자리 수, 2자리 수, 3자리 수 등의 자릿수의 합을 계산하여 최종 결과를 도출합니다.

### 동작 과정
1. **입력 처리**: N을 입력받고, N의 자릿수를 구합니다. 예를 들어, N이 23이라면 `l`은 2가 됩니다.
2. **1자리 수의 자릿수 계산**: 1부터 9까지의 수는 총 9개로, 이들의 자릿수는 9 * 1 = 9입니다.
3. **2자리 수의 자릿수 계산**: 10부터 99까지의 수는 총 90개로, 이들의 자릿수는 90 * 2 = 180입니다.
4. **l-1 자리수 처리**: N의 자릿수보다 하나 작은 자리수의 수들(즉, 1자리와 2자리 수)을 모두 합산합니다.
5. **N에 포함된 l자리 수 처리**: N이 l자리 수일 경우, `10^(l-1)`부터 N까지의 모든 수의 자릿수를 계산하여 최종 자릿수에 더합니다. 예를 들어, N이 23일 경우 10부터 23까지의 수를 계산합니다: (23 - 10 + 1) * 2 = 28.
6. **결과 출력**: 총 자릿수를 계산하여 출력합니다.

예시 입력:
- N = 23
    - `l` = 2 (N의 자릿수)
    - 1자리 수: 9 * 1 = 9
    - 2자리 수: 90 * 2 = 180
    - 10부터 23까지의 2자리 수: (23 - 10 + 1) * 2 = 28
    - 총 자릿수 = 9 + 180 + 28 = 217

최종 출력: 217

### 시간 복잡도
O(log N) - N의 자릿수를 계산하는 과정이 log N에 비례하기 때문에, 전체적인 시간 복잡도는 O(log N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
