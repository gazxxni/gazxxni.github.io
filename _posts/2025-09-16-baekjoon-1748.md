## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 1부터 N까지의 수를 각 자릿수에 따라 그룹화하여 전체 자리 수를 계산하는 것입니다. 각 자릿수에 대해 작성되는 수의 개수를 파악하고, 이를 바탕으로 자릿수를 누적하여 최종 결과를 도출합니다.

### 동작 과정
예를 들어, N이 23이라고 가정해 보겠습니다.

1. **자릿수 계산**:
   - 1자리 수 (1~9): 총 9개, 각 1자리 → 9 * 1 = 9자리
   - 2자리 수 (10~23): 총 14개 (10~23), 각 2자리 → 14 * 2 = 28자리

2. **결과 누적**:
   - 1자리 수의 자릿수: 9
   - 2자리 수의 자릿수: 28
   - 따라서, 전체 자릿수 = 9 + 28 = 37

3. **코드 구현**:
   - 입력받은 N에 대해 자릿수를 계산하는 반복문을 작성.
   - 각 자릿수 그룹에 대해 자릿수 계산을 수행하고, 마지막으로 N의 자릿수에 해당하는 수를 추가합니다.

### 시간 복잡도
이 알고리즘은 N의 자릿수를 기준으로 반복하기 때문에 O(log N)입니다. N이 최대 100,000,000일 경우, 자릿수는 최대 8이므로, 최대 8회 반복합니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
