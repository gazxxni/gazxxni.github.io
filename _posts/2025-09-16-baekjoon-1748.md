## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 수 `n`까지의 모든 수에서 각 숫자가 차지하는 자리수의 합을 구하는 것입니다. 자리수가 1부터 `l-1`까지인 수들은 규칙적으로 증가하므로, 이들을 한 번에 계산할 수 있습니다. 그리고 `l`자리 수인 경우에는 `n`까지의 수들을 따로 계산하여 최종 결과에 추가합니다.

### 동작 과정
1. 입력값으로 주어진 `n`을 문자열로 읽고, 그 길이 `l`을 구합니다.
   - 예시: `n = "123"`
   - `l = 3`

2. `1`자리 수부터 `l-1`자리 수까지의 각 자리수의 합을 계산합니다. 이 수들은 1부터 9, 10부터 99, ..., 100부터 999까지의 수들입니다.
   - `ans`는 0으로 초기화됩니다.
   - `for i in range(1, l)` 반복문을 통해 각 자리수 `i`에 대해 다음을 계산합니다:
     - `9 * (10 ** (i - 1)) * i`는 `i`자리 수의 개수(9 * 10^(i-1))에 각 수의 자리수(i)를 곱하여 더합니다.
     - 예를 들어, `i = 1`일 때는 `9 * 1`, `i = 2`일 때는 `9 * 10 * 2`, `i = 3`일 때는 `9 * 100 * 3`이 계산됩니다.
   - 이 과정을 통해 `ans`는 `9 * (10 ** (i - 1)) * i`의 합으로 업데이트됩니다.
   - 최종적으로 `ans`는 0에서 시작하여 모든 자리수의 합을 포함하게 됩니다.

3. `l`자리 수인 경우, `n`과 가장 가까운 아래의 `l-1`자리 수를 계산하여 그 범위 내에서의 자리수 합을 추가합니다.
   - `ans += (int(n) - (10 ** (l - 1)) + 1) * l`를 통해 `n`까지의 수에서 자리수 합을 추가합니다.
   - 예를 들어 `n = 123`인 경우, `10^(3-1) = 100`이므로 `ans += (123 - 100 + 1) * 3`이 계산되어 `ans`에 추가됩니다.

4. 최종적으로 계산된 `ans`를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(log n)입니다. 이는 `n`의 자리수를 기준으로 계산하기 때문이며, 최대 10자리의 수에 대해서도 상수 시간 내에 처리할 수 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
