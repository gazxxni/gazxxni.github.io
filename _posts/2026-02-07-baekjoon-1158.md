## 문제
요세푸스 문제는 다음과 같다.

1번부터 N번까지 N명의 사람이 원을 이루면서 앉아있고, 양의 정수 K(≤ N)가 주어진다. 이제 순서대로 K번째 사람을 제거한다. 한 사람이 제거되면 남은 사람들로 이루어진 원을 따라 이 과정을 계속해 나간다. 이 과정은 N명의 사람이 모두 제거될 때까지 계속된다. 원에서 사람들이 제거되는 순서를 (N, K)-요세푸스 순열이라고 한다. 예를 들어 (7, 3)-요세푸스 순열은 이다.

N과 K가 주어지면 (N, K)-요세푸스 순열을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N과 K가 빈 칸을 사이에 두고 순서대로 주어진다. (1 ≤ K ≤ N ≤ 5,000)

---

## 출력
예제와 같이 요세푸스 순열을 출력한다.

## 풀이

### 풀이 핵심 로직
요세푸스 문제는 원형 큐를 사용하여 K번째 사람을 제거하는 과정을 시뮬레이션하는 방식으로 해결할 수 있다. Python의 `deque` 자료구조를 활용하여 O(1)의 시간 복잡도로 양쪽에서 원소를 추가하고 제거할 수 있으므로 효과적으로 문제를 해결할 수 있다.

### 동작 과정
1. 입력된 N과 K에 따라 1부터 N까지의 숫자를 가진 리스트를 생성한다.
2. 이 리스트를 `deque` 형태로 변환하여 원형 큐를 구현한다.
3. 인덱스를 사용하여 K번째 사람을 제거할 때까지 큐에서 원소를 순환시킨다:
   - K번째 사람의 인덱스가 될 때까지 큐의 앞에서 원소를 꺼내 뒤에 다시 추가한다.
   - K번째 사람을 제거하고 그 사람을 결과 리스트에 추가한다.
4. 모든 사람이 제거될 때까지 이 과정을 반복한다.
5. 최종적으로 결과 리스트를 포맷에 맞춰 출력한다.

예시 입력: `7 3`

- 초기 큐: `[1, 2, 3, 4, 5, 6, 7]`
- 1단계: 3번째(3) 제거 → `[1, 2, 4, 5, 6, 7]` → 결과: `[3]`
- 2단계: 3번째(6) 제거 → `[1, 2, 4, 5, 7]` → 결과: `[3, 6]`
- 3단계: 3번째(2) 제거 → `[1, 4, 5, 7]` → 결과: `[3, 6, 2]`
- 4단계: 3번째(7) 제거 → `[1, 4, 5]` → 결과: `[3, 6, 2, 7]`
- 5단계: 3번째(5) 제거 → `[1, 4]` → 결과: `[3, 6, 2, 7, 5]`
- 6단계: 3번째(1) 제거 → `[4]` → 결과: `[3, 6, 2, 7, 5, 1]`
- 마지막 단계: 3번째(4) 제거 → `[]` → 결과: `[3, 6, 2, 7, 5, 1, 4]`

최종 출력: `<3, 6, 2, 7, 5, 1, 4>`

### 시간 복잡도
- 전체 과정에서 각 사람을 제거하는 데 O(K)만큼의 시간이 걸리며, 최악의 경우 N번 반복하므로 시간 복잡도는 O(N*K)이다. 그러나 K가 고정되어 있거나 N에 비해 작은 경우에는 사실상 O(N)으로 간주할 수 있다.

## 코드
```python
from collections import deque

n, k = map(int, input().split())
arr = [i for i in range(1, n + 1)]
q = deque(arr)
ans = []
idx = 0

while True:
    if not q:
        break
    
    if idx == k - 1:
        ans.append(q.popleft())
        idx = 0
        
    else:
        q.append(q.popleft())
        idx += 1
        
print("<" + ", ".join(map(str, ans)) + ">")

```
