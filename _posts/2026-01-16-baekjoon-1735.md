## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
두 분수를 합산하기 위해 공통 분모를 찾아야 합니다. 이를 위해 두 분모의 최소 공배수를 구하고, 각 분수를 이 공통 분모로 변환한 후, 분자를 더합니다. 마지막으로, 결과 분수를 기약분수 형태로 나타내기 위해 최대공약수로 나누어줍니다.

### 동작 과정
입력이 다음과 같다고 가정합니다:
```
1 2
1 3
```
1. 첫 번째 분수 A/B는 1/2이고, 두 번째 분수 C/D는 1/3입니다.
2. 두 분모 2와 3의 최소 공배수를 구합니다: lcm(2, 3) = 6
3. 첫 번째 분수를 공통 분모로 변환합니다: 
   - 1/2는 (1 * 3) / (2 * 3) = 3/6
4. 두 번째 분수를 공통 분모로 변환합니다: 
   - 1/3는 (1 * 2) / (3 * 2) = 2/6
5. 두 분수를 더합니다: 3/6 + 2/6 = (3 + 2) / 6 = 5/6
6. 결과 분수 5/6의 기약분수를 구하기 위해 최대공약수(gcd)를 계산합니다: gcd(5, 6) = 1
7. 최종 결과는 (5 / 1, 6 / 1) = 5 6입니다.

### 시간 복잡도
이 알고리즘의 주요 부분은 최대공약수를 구하는 과정과 최소공배수를 계산하는 것이므로, 두 수의 크기와 관계없이 O(log(min(a, b)))의 시간 복잡도를 가집니다. 전체 과정은 이 두 계산에 의해 지배되므로, 전체 시간 복잡도는 O(log(n))입니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
