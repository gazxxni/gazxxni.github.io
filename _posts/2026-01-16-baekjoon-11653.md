## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수 N을 소수들의 곱으로 표현하는 과정입니다. 이 문제에서는 N을 2부터 시작하여 모든 소수로 나누어가며, 나누어떨어질 경우 해당 소수를 출력하고 N을 갱신하는 방식으로 진행합니다. N이 1이 아닐 경우 마지막에 남은 N도 출력합니다.

### 동작 과정
1. **입력**: N이 주어진다. 예를 들어 N = 28.
2. **2로 나누기**: N이 2로 나누어지는 동안 2를 출력하고 N을 2로 나눈다.
   - N은 28 -> 출력: 2, N = 28 / 2 = 14
   - N은 14 -> 출력: 2, N = 14 / 2 = 7
   - N은 7 -> 나누어 떨어지지 않음
3. **홀수 소수로 나누기**: 3부터 시작하여 N의 제곱근까지 홀수로 반복하면서 N이 해당 소수로 나누어지는지 체크한다.
   - i = 3: N(7) % 3 != 0
   - i = 5: N(7) % 5 != 0
   - i = 7: N(7) % 7 == 0 -> 출력: 7, N = 7 / 7 = 1
4. **마지막 체크**: N이 1보다 큰 경우 남은 N을 출력한다. (여기서는 N=1이므로 출력 없음)
5. **결과**: 출력된 소인수는 2, 7이다.

### 시간 복잡도
O(√N) - N의 소인수분해는 최악의 경우 N의 제곱근까지 반복하여 나누어지는 과정을 수행하므로, 시간 복잡도는 O(√N)입니다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
