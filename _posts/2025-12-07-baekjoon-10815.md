## 문제
숫자 카드는 정수 하나가 적혀져 있는 카드이다. 상근이는 숫자 카드 N개를 가지고 있다. 정수 M개가 주어졌을 때, 이 수가 적혀있는 숫자 카드를 상근이가 가지고 있는지 아닌지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 상근이가 가지고 있는 숫자 카드의 개수 N(1 ≤ N ≤ 500,000)이 주어진다. 둘째 줄에는 숫자 카드에 적혀있는 정수가 주어진다. 숫자 카드에 적혀있는 수는 -10,000,000보다 크거나 같고, 10,000,000보다 작거나 같다. 두 숫자 카드에 같은 수가 적혀있는 경우는 없다.

셋째 줄에는 M(1 ≤ M ≤ 500,000)이 주어진다. 넷째 줄에는 상근이가 가지고 있는 숫자 카드인지 아닌지를 구해야 할 M개의 정수가 주어지며, 이 수는 공백으로 구분되어져 있다. 이 수도 -10,000,000보다 크거나 같고, 10,000,000보다 작거나 같다

---

## 출력
첫째 줄에 입력으로 주어진 M개의 수에 대해서, 각 수가 적힌 숫자 카드를 상근이가 가지고 있으면 1을, 아니면 0을 공백으로 구분해 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 숫자 카드의 정보를 해시맵(딕셔너리)에 저장하여, 상근이가 가지고 있는 카드인지 여부를 O(1)의 시간 복잡도로 빠르게 확인할 수 있다. 입력으로 주어진 M개의 수에 대해 각각 상근이가 가진 카드인지 체크하고, 결과를 리스트에 저장한 후 한 번에 출력한다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 상근이가 가진 카드의 개수 N과 카드에 적힌 숫자들을 입력받는다. 두 번째 줄에서는 확인하고자 하는 M개의 숫자를 입력받는다.
2. **해시맵 생성**: 상근이가 가진 카드의 숫자를 키로 하고, 임의의 값을 넣어 해시맵을 생성한다. 이는 카드에 적힌 숫자가 존재하는지를 효율적으로 확인하기 위함이다.
3. **카드 존재 여부 확인**: M개의 숫자 각각에 대해 해시맵에서 해당 숫자의 존재 여부를 체크한다. 존재하면 1, 존재하지 않으면 0을 결과 리스트에 추가한다.
4. **출력**: 결과 리스트의 내용을 공백으로 구분하여 출력한다.

예시 입력:
```
5
6 3 2 10 -10
8
10 9 -5 2 3 4 5 -10
```
단계별 설명:
- 첫 번째 줄에서 N=5를 읽고, 카드 목록을 [6, 3, 2, 10, -10]으로 저장한다.
- 두 번째 줄에서 M=8을 읽고, 확인할 숫자 목록을 [10, 9, -5, 2, 3, 4, 5, -10]으로 저장한다.
- 해시맵을 생성하여 {6: 0, 3: 0, 2: 0, 10: 0, -10: 0} 형태로 만든다.
- 각 숫자에 대해 존재 여부를 확인하여 결과는 [1, 0, 0, 1, 1, 0, 0, 1]이 된다.
- 마지막으로 결과 리스트를 출력하면 `1 0 0 1 1 0 0 1`이 된다.

### 시간 복잡도
O(N + M) - N개의 카드 정보를 해시맵에 저장하는 데 O(N), M개의 숫자 존재 여부를 확인하는 데 O(M)의 시간이 소요된다.

## 코드
```python
n = int(input())
arr = list(map(int, input().split()))
m = int(input())
brr = list(map(int, input().split()))

dic = {}
for i in range(len(arr)):
    dic[arr[i]] = 0

ans = []
for i in range(m):
    if brr[i] not in dic:
        ans.append(0)
    else:
        ans.append(1)
        
print(*ans)
```
