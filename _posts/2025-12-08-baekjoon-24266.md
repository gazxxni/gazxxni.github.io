## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 수행 시간은 입력 크기 \( n \)의 세제곱에 비례합니다. 즉, 알고리즘의 수행 횟수는 \( n^3 \)이며, 이 경우 최고차항의 차수는 3입니다. 따라서 출력은 수행 횟수와 최고차항의 차수를 각각 구분하여 제공해야 합니다.

### 동작 과정
1. 입력: 사용자로부터 정수 \( n \)을 입력받습니다. 예를 들어, 입력이 5라면, \( n = 5 \)입니다.
2. 수행 횟수 계산: 코드1의 수행 횟수는 \( n^3 \)이므로, \( 5^3 = 125 \)가 됩니다.
3. 최고차항의 차수 출력: 다항식의 최고차항의 차수는 3이므로, 이를 출력합니다.

예시 입력:
```
5
```
예시 출력:
```
125
3
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(1) \)입니다. 주어진 입력 \( n \)에 대한 계산은 상수 시간 내에 수행됩니다.
