## 문제 링크
[https://www.acmicpc.net/problem/10610](https://www.acmicpc.net/problem/10610)

## 풀이

### 풀이 핵심 로직
문제는 주어진 숫자들로 만들 수 있는 가장 큰 30의 배수인 숫자를 찾는 것입니다. 30의 배수는 두 가지 조건을 만족해야 합니다: 마지막 숫자가 0이어야 하고, 모든 숫자의 합이 3의 배수여야 합니다. 이 두 조건을 체크한 후, 가능한 숫자들을 내림차순으로 정렬하여 가장 큰 수를 출력합니다.

### 동작 과정
1. 입력을 받아서 각 자릿수를 리스트 `n`에 저장합니다.
   - 예시 입력: `102`
   - `n` 리스트: `['1', '0', '2']`
   
2. 숫자를 정수형으로 변환하여 `arr` 리스트에 저장하고, 각 숫자의 합을 `cnt`에 저장합니다.
   - `arr` 리스트: `[1, 0, 2]`
   - `cnt` 값: `3` (1 + 0 + 2)

3. `arr`에 0이 포함되어 있는지 확인합니다.
   - 0이 포함되어 있으므로 다음 단계로 진행합니다.

4. `cnt`가 3의 배수인지 확인합니다.
   - `3 % 3 == 0` 이므로 조건을 만족합니다.

5. `arr`을 내림차순으로 정렬하여 가장 큰 수를 만듭니다.
   - 정렬된 `arr`: `[2, 1, 0]`

6. 정렬된 리스트를 출력합니다.
   - 출력: `210`

7. 만약 0이 없거나 `cnt`가 3의 배수가 아니면 -1을 출력합니다.

### 시간 복잡도
코드의 시간 복잡도는 O(n log n)입니다. 여기서 n은 입력 숫자의 자릿수입니다. 이는 리스트를 정렬하는 데 필요한 시간 복잡도입니다. 나머지 연산(합 계산, 조건 체크)은 O(n)으로 수행되므로 전체적인 시간 복잡도는 O(n log n)으로 정리됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = list(input().rstrip())
cnt = 0
arr = []

for i in n:
    arr.append(int(i))

if 0 in arr:
    
    for i in arr:
        cnt += i
        
    if cnt % 3 == 0:
        arr.sort(reverse=True)
        
        print(*arr, sep='')
        
    else:
        print(-1)
    
    
else:
    print(-1)
```
