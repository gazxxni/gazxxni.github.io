## 문제
2차원 좌표 평면 위의 두 선분 L1, L2가 주어졌을 때, 두 선분이 교차하는지 아닌지 구해보자. 한 선분의 끝 점이 다른 선분이나 끝 점 위에 있는 것도 교차하는 것이다.

L1의 양 끝 점은 (x1, y1), (x2, y2), L2의 양 끝 점은 (x3, y3), (x4, y4)이다.

---

## 입력
첫째 줄에 L1의 양 끝 점 x1, y1, x2, y2가, 둘째 줄에 L2의 양 끝 점 x3, y3, x4, y4가 주어진다.

---

## 출력
L1과 L2가 교차하면 1, 아니면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 두 선분이 교차하는지 판단하기 위해, 각 선분의 끝점에 대한 위치 관계를 계산합니다. 이를 위해 CCW(Counter Clock Wise) 함수를 사용하여 선분의 방향성을 확인하고, 결과를 바탕으로 교차 여부를 결정합니다. 또한, 두 선분이 일직선 상에 있을 경우, 선분의 끝점이 서로의 구간에 포함되는지를 검사하여 교차 여부를 판별합니다.

### 동작 과정
1. **입력 처리**: 두 선분 L1의 끝점 (x1, y1)와 (x2, y2), L2의 끝점 (x3, y3)와 (x4, y4)를 입력받습니다.
   
2. **CCW 함수 정의**: CCW 함수는 세 점 (xa, ya), (xb, yb), (xc, yc)에 대해 벡터의 외적을 계산하여 방향성을 판단합니다. 이 값이 0이면 일직선, 0보다 크면 반시계 방향, 0보다 작으면 시계 방향입니다.

3. **방향성 계산**: 각 선분의 끝점에 대한 CCW 값을 계산합니다:
   - o1 = CCW(x1, y1, x2, y2, x3, y3)
   - o2 = CCW(x1, y1, x2, y2, x4, y4)
   - o3 = CCW(x3, y3, x4, y4, x1, y1)
   - o4 = CCW(x3, y3, x4, y4, x2, y2)

4. **일직선 판별**: 모든 CCW 값이 0인 경우 두 선분이 일직선에 놓여있다고 판단합니다. 이 경우, 선분의 끝점들이 서로의 구간에 포함되는지 확인합니다.

5. **교차 여부 판단**: 
   - 만약 o1와 o2가 서로 다른 부호를 가지며 (즉, o1 * o2 <= 0), o3와 o4도 마찬가지라면 선분이 교차한다고 판단합니다.
   - 그렇지 않으면 교차하지 않는다고 판단합니다.

6. **결과 출력**: 교차하면 1을, 그렇지 않으면 0을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력된 좌표에 대한 고정된 수의 계산만 수행하기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

def ccw(xa, ya, xb, yb, xc, yc):
    cross = (xb - xa) * (yc - ya) - (yb - ya) * (xc - xa)
    return cross

x1, y1, x2, y2 = map(int, input().split())
x3, y3, x4, y4 = map(int, input().split())

o1 = ccw(x1, y1, x2, y2, x3, y3)
o2 = ccw(x1, y1, x2, y2, x4, y4)
o3 = ccw(x3, y3, x4, y4, x1, y1)
o4 = ccw(x3, y3, x4, y4, x2, y2)

if o1 == 0 and o2 == 0 and o3 == 0 and o4 == 0:
    if (x1, y1) > (x2, y2):
        x1, x2 = x2, x1
        y1, y2 = y2, y1
    if (x3, y3) > (x4, y4):
        x3, x4 = x4, x3
        y3, y4 = y4, y3

    
    if max(min(x1,x2), min(x3,x4)) <= min(max(x1,x2), max(x3,x4)):
        if max(min(y1,y2), min(y3,y4)) <= min(max(y1,y2), max(y3,y4)):
            print(1)
            exit()
        else:
            print(0)
            exit()
    else:
        print(0)
        exit()

if o1 * o2 <= 0 and o3 * o4 <= 0:
    print(1)
    exit()
else:
    print(0)
    exit()

```
