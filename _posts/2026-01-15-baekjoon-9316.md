## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 문제는 단순한 반복문을 사용하여 출력 형식을 맞추는 것입니다. N개의 테스트 케이스에 대해 "Hello World, Judge i!" 형식의 문자열을 출력하는 문제로, i는 1부터 N까지의 숫자를 나타냅니다. 이를 위해 for 루프를 사용하여 각 줄에 대해 필요한 문자열을 출력합니다.

### 동작 과정
1. 첫째 줄에 정수 N이 입력됩니다. 예를 들어 N = 3이라고 가정합니다.
2. for 루프를 통해 i는 1부터 N(3)까지 반복합니다.
   - i = 1일 때: "Hello World, Judge 1!" 출력
   - i = 2일 때: "Hello World, Judge 2!" 출력
   - i = 3일 때: "Hello World, Judge 3!" 출력
3. 최종적으로 총 3줄의 출력이 이루어집니다.

입력 예시:
```
3
```
출력 결과:
```
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

### 시간 복잡도
이 프로그램의 시간 복잡도는 O(N)입니다. 이는 N의 크기만큼 반복문이 실행되기 때문입니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
