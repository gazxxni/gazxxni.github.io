## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 정수 N에 대해 1부터 N까지의 숫자에 맞춰서 "Hello World, Judge i!"라는 문자열을 출력하는 것입니다. 문자열에서 i는 현재 반복의 인덱스에 해당하며, 이를 통해 각 테스트 케이스에 대해 고유한 인사를 할 수 있습니다.

### 동작 과정
1. 입력으로 N을 받습니다. 예를 들어, N이 3일 경우를 생각해보겠습니다.
2. 반복문을 사용하여 1부터 N까지 반복합니다.
   - i = 1: "Hello World, Judge 1!" 출력
   - i = 2: "Hello World, Judge 2!" 출력
   - i = 3: "Hello World, Judge 3!" 출력
3. 최종적으로 3줄의 인사가 출력됩니다:
   ```
   Hello World, Judge 1!
   Hello World, Judge 2!
   Hello World, Judge 3!
   ```

### 시간 복잡도
시간 복잡도는 O(N)입니다. N번의 반복을 통해 각 인사 문자열을 출력하므로, 입력값 N에 비례하여 수행 시간이 증가합니다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
