## 문제
수 N개 A1, A2, ..., AN이 주어진다. 이때, 연속된 부분 구간의 합이 M으로 나누어 떨어지는 구간의 개수를 구하는 프로그램을 작성하시오.

즉, Ai + ... + Aj (i ≤ j) 의 합이 M으로 나누어 떨어지는 (i, j) 쌍의 개수를 구해야 한다.

---

## 입력
첫째 줄에 N과 M이 주어진다. (1 ≤ N ≤ 106, 2 ≤ M ≤ 103)

둘째 줄에 N개의 수 A1, A2, ..., AN이 주어진다. (0 ≤ Ai ≤ 109)

---

## 출력
첫째 줄에 연속된 부분 구간의 합이 M으로 나누어 떨어지는 구간의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 연속된 부분 구간의 합이 M으로 나누어 떨어지는 경우를 찾는 문제입니다. 누적합을 이용하여 각 인덱스에서의 누적합을 M으로 나눈 나머지를 계산하고, 같은 나머지를 가진 누적합의 개수를 세어 그 조합으로 구간의 개수를 구합니다.

### 동작 과정
1. 입력으로 주어진 N과 M, 그리고 수열을 읽어옵니다.
2. `remainder_count` 배열을 생성하여 M으로 나눈 나머지의 개수를 기록합니다. 이 배열의 크기는 M입니다.
3. 누적합을 계산하면서 각 단계에서의 누적합을 M으로 나눈 나머지를 구합니다.
4. 이 나머지가 0인 경우는 특별히 카운트하고, 나머지가 같은 경우의 수를 `remainder_count`에서 증가시킵니다.
5. 모든 나머지에 대해 조합을 계산하여 최종적으로 구간의 개수를 구합니다.

예시 입력:
```
5 3
1 2 3 1 2
```

단계별 설명:
- N=5, M=3, 수열 = [1, 2, 3, 1, 2]
- 초기 상태에서 `remainder_count`는 [1, 0, 0] (나머지가 0인 경우 1로 카운트)
- 누적합 계산:
  - i=0: 현재 합 = 1, 나머지 1 → `remainder_count` = [1, 1, 0]
  - i=1: 현재 합 = 3, 나머지 0 → `remainder_count` = [2, 1, 0]
  - i=2: 현재 합 = 6, 나머지 0 → `remainder_count` = [3, 1, 0]
  - i=3: 현재 합 = 7, 나머지 1 → `remainder_count` = [3, 2, 0]
  - i=4: 현재 합 = 9, 나머지 0 → `remainder_count` = [4, 2, 0]
- 모든 나머지에 대해 조합을 계산:
  - 나머지 0에서 4개의 경우가 있으므로 (4 * 3) / 2 = 6
  - 나머지 1에서 2개의 경우가 있으므로 (2 * 1) / 2 = 1
- 총 구간의 개수는 6 + 1 = 7.

### 시간 복잡도
O(N) - 수열을 한 번 순회하면서 누적합과 나머지를 계산하고, 나머지 개수를 세기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

'''
1. 구간 합이 M으로 나누어 떨어진다는 것은 
(prefix_sum[j] - prefix_sum[i-1]) % M == 0이라는 뜻

2. 이 식을 이항하면 prefix_sum[j] % M == prefix_sum[i-1] % M

3. 즉, 누적합을 M으로 나눈 나머지가 같은 인덱스들을 찾으면 됨
'''
n, m = map(int, input().split())
arr = list(map(int, input().split()))

# 1. 나머지를 카운트할 배열 또는 딕셔너리 준비
remainder_count = [0] * m

# 2. 누적합 변수 초기화
current_sum = 0

# 3. 나머지가 0인 경우를 미리 카운트
remainder_count[0] = 1 

# 4. 수열을 순회하면서 누적합과 나머지 계산
for x in arr:
    current_sum += x
    r = current_sum % m
    remainder_count[r] += 1

ans = 0
for cnt in remainder_count:
    # 해당 나머지를 가진 누적합의 개수가 2개 이상일 때만 쌍을 만들 수 있음
    if cnt > 1:
        ans += (cnt * (cnt - 1)) // 2

print(ans)
```
