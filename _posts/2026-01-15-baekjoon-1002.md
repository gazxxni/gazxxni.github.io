## 문제
조규현과 백승환은 터렛에 근무하는 직원이다. 하지만 워낙 존재감이 없어서 인구수는 차지하지 않는다. 다음은 조규현과 백승환의 사진이다.



이석원은 조규현과 백승환에게 상대편 마린(류재명)의 위치를 계산하라는 명령을 내렸다. 조규현과 백승환은 각각 자신의 터렛 위치에서 현재 적까지의 거리를 계산했다.

조규현의 좌표 (x1,y1)$(x_1, y_1)$와 백승환의 좌표 (x2,y2)$(x_2, y_2)$가 주어지고, 조규현이 계산한 류재명과의 거리 r1$r_1$과 백승환이 계산한 류재명과의 거리 r2$r_2$가 주어졌을 때, 류재명이 있을 수 있는 좌표의 수를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T$T$가 주어진다. 각 테스트 케이스는 다음과 같이 이루어져 있다.

한 줄에 공백으로 구분 된 여섯 정수 x1$x_1$, y1$y_1$, r1$r_1$, x2$x_2$, y2$y_2$, r2$r_2$가 주어진다.

---

## 출력
각 테스트 케이스마다 류재명이 있을 수 있는 위치의 수를 출력한다. 만약 류재명이 있을 수 있는 위치의 개수가 무한대일 경우에는 −1$-1$ 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 원이 주어졌을 때, 두 원의 교차점 개수를 계산하는 문제입니다. 두 원의 중심 간 거리와 반지름을 이용해 원의 위치 관계를 판단하여 교차점의 개수를 출력합니다. 중심 간 거리가 0이고 두 원의 반지름이 같을 경우 교차점이 무한대가 됩니다.

### 동작 과정
1. **입력 받기**: 먼저 테스트 케이스 개수 T를 입력받고, 각 테스트 케이스에 대해 두 원의 중심 좌표와 반지름을 입력받습니다.
2. **거리 계산**: 두 원의 중심 간 거리 `distance`를 계산합니다. 이는 피타고라스의 정리를 활용하여 다음과 같이 계산할 수 있습니다:
   \[
   distance = \sqrt{(x1 - x2)^2 + (y1 - y2)^2}
   \]
3. **조건 체크**:
   - **무한대의 교차점**: 두 원의 중심이 같고 반지름이 동일한 경우 (`distance == 0` and `r1 == r2`), 교차점이 무한대므로 -1을 출력합니다.
   - **접점**: 두 원이 외접하거나 내접하는 경우 (`abs(r1 - r2) == distance` or `r1 + r2 == distance`), 접점이 하나이므로 1을 출력합니다.
   - **교차점이 두 개**: 두 원이 서로 교차하는 경우 (`abs(r1 - r2) < distance < (r1 + r2)`), 교차점이 2개이므로 2를 출력합니다.
   - **교차점 없음**: 그 외의 경우, 교차점이 없으므로 0을 출력합니다.

**예시 입력**: 
```
1
0 0 1 0 2 1
```
**단계별 설명**:
- 첫 번째 줄에서 `t = 1`로 테스트 케이스가 하나임을 알 수 있습니다.
- 두 번째 줄에서 두 원의 정보가 `(0, 0, 1)`과 `(0, 2, 1)`로 주어집니다.
- 두 원의 중심 간 거리 `distance`는 `2`로 계산됩니다.
- 두 원의 반지름의 합 `r1 + r2`는 `2`이고, 두 원의 반지름 차의 절대값 `abs(r1 - r2)`는 `0`입니다.
- 조건을 확인하면 접점이 하나인 경우에 해당하므로 출력으로 `1`이 나옵니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(T)입니다. 각 테스트 케이스에 대해 상수 시간 내에 처리가 가능하므로, 전체적으로 T개의 테스트 케이스에 대해 선형적으로 처리됩니다.

## 코드
```python
import math

t = int(input())

for _ in range(t):
    x1, y1, r1, x2, y2, r2 = map(int, input().split())
    distance = math.sqrt((x1 - x2) ** 2 + (y1 - y2) ** 2)
    
    if distance == 0 and r1 == r2 :  
        print(-1)
    elif abs(r1 - r2) == distance or r1 + r2 == distance: 
        print(1)
    elif abs(r1 - r2) < distance < (r1 + r2) :  
        print(2)
    else:
        print(0) 
```
