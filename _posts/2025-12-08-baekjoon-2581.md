## 문제
자연수 M과 N이 주어질 때 M이상 N이하의 자연수 중 소수인 것을 모두 골라 이들 소수의 합과 최솟값을 찾는 프로그램을 작성하시오.

예를 들어 M=60, N=100인 경우 60이상 100이하의 자연수 중 소수는 61, 67, 71, 73, 79, 83, 89, 97 총 8개가 있으므로, 이들 소수의 합은 620이고, 최솟값은 61이 된다.

---

## 입력
입력의 첫째 줄에 M이, 둘째 줄에 N이 주어진다.

M과 N은 10,000이하의 자연수이며, M은 N보다 작거나 같다.

---

## 출력
M이상 N이하의 자연수 중 소수인 것을 모두 찾아 첫째 줄에 그 합을, 둘째 줄에 그 중 최솟값을 출력한다. 

단, M이상 N이하의 자연수 중 소수가 없을 경우는 첫째 줄에 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 범위 [M, N] 내의 소수를 찾기 위해 에라토스테네스의 체 알고리즘을 사용하여 1부터 N까지의 소수를 미리 구합니다. 이후 M 이상 N 이하의 소수를 필터링하여 이들의 합과 최솟값을 계산합니다. 소수가 없을 경우 -1을 출력합니다.

### 동작 과정
1. **입력:** M과 N의 값을 입력받습니다. 예를 들어, M=60, N=100.
2. **소수 판별:** 1부터 N(100)까지의 자연수에 대해 소수인지 여부를 판단하기 위해, 배열 `arr`을 사용하여 에라토스테네스의 체를 구현합니다.
   - `arr[i]`가 True이면 i는 소수입니다.
   - 2부터 √N까지의 수를 기준으로 소수를 제거합니다.
3. **소수 필터링:** M부터 N까지의 수 중에서 소수인 경우를 찾아, 그 합을 `cnt`에 저장하고, 최솟값을 `ans`에 저장합니다.
   - 예를 들어, 60, 61, 67, 71, ..., 97까지의 소수를 찾아 합산하면 620이 되고, 최솟값은 61이 됩니다.
4. **출력:** 소수가 없다면 -1을 출력하고, 소수가 있다면 합과 최솟값을 각각 출력합니다.

### 시간 복잡도
소수 판별을 위한 에라토스테네스의 체 알고리즘은 O(N log log N)의 시간 복잡도를 가집니다. 이후 M에서 N까지 소수를 필터링하는 과정은 O(N-M)입니다. 따라서 전체 시간 복잡도는 O(N log log N)으로 표현할 수 있습니다.

## 코드
```python
m = int(input())
n = int(input())

arr = [True] * 10001
arr[0], arr[1] = False, False

for i in range(2, int(n ** 0.5) + 1): 
    if arr[i]:
        for j in range(i * i, n + 1, i):
            arr[j] = False
             
cnt = 0
ans = 0
for i in range(m, n + 1):
    if arr[i]:
        cnt += i
        if ans == 0:
            ans = i

if ans == 0:
    print(-1)
else:
    print(cnt)
    print(ans)
```
