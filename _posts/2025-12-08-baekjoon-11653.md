## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 정수 N을 소인수분해하기 위해, 먼저 2로 나누어 N의 짝수 소인수를 처리하고, 그 후 홀수 소인수를 처리하기 위해 3부터 시작하여 N의 제곱근까지의 모든 홀수에 대해 나누어 떨어지는지 확인한다. 최종적으로 남은 N이 1보다 크면 그것도 소인수로 출력한다.

### 동작 과정
예시 입력으로 `N = 60`을 고려해보자.
1. `n`을 2로 나누어 소인수 2를 찾는다:
   - 60 % 2 == 0 → 2 출력, `n = 60 // 2 = 30`
   - 30 % 2 == 0 → 2 출력, `n = 30 // 2 = 15`
   - 15 % 2 != 0 → 2로 더 이상 나눌 수 없음
2. 이제 홀수 소인수인 3부터 시작:
   - 15 % 3 == 0 → 3 출력, `n = 15 // 3 = 5`
   - 5 % 3 != 0 → 3으로 더 이상 나눌 수 없음
3. 다음 홀수인 5를 확인:
   - 5 % 5 == 0 → 5 출력, `n = 5 // 5 = 1`
4. `n`이 1보다 크지 않으므로 종료.

출력 결과:
```
2
2
3
5
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(√N)이다. 이는 N의 소인수분해를 위해 N의 제곱근까지의 수를 검사하기 때문이며, 특히 짝수 소인수 2의 처리를 포함하면 더 빠를 수 있다.

---

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
