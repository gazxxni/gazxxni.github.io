## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수 N을 소수의 곱으로 표현하는 과정입니다. 이 문제에서는 N을 2부터 시작해 가능한 소수로 나누고, 그 과정을 반복하여 N이 1이 될 때까지 소인수를 출력합니다. 특히, 2는 유일한 짝수 소수이므로 먼저 처리하고, 그 이후에는 홀수 소수만으로 나누는 방식으로 효율성을 높입니다.

### 동작 과정
1. **입력**: 예를 들어, N = 60이라고 가정합니다.
2. **2로 나누기**: 
   - 60 % 2 == 0이므로 2를 출력하고, N을 60 // 2 = 30으로 업데이트.
   - 30 % 2 == 0이므로 2를 다시 출력하고, N을 30 // 2 = 15로 업데이트.
   - 15는 2로 나누어지지 않으므로 이 과정을 종료.
3. **홀수 소수로 나누기**:
   - i = 3부터 시작하여, 15 % 3 == 0이므로 3을 출력하고 N을 15 // 3 = 5로 업데이트.
   - 다음 i = 5로 이동하여 5 % 5 == 0이므로 5를 출력하고 N을 5 // 5 = 1로 업데이트.
4. **마무리**: N이 1이 되었으므로 종료합니다. 출력된 소인수는 2, 2, 3, 5입니다.

### 시간 복잡도
최악의 경우, N이 소수인 경우까지 모든 소수를 검사해야 하므로, 시간 복잡도는 O(√N)입니다. 이는 N을 나눌 수 있는 모든 소수에 대해 나누기를 수행하는 과정이기 때문입니다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
