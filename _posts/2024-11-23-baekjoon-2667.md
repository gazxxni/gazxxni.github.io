[https://www.acmicpc.net/problem/2667](https://www.acmicpc.net/problem/2667)

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 2차원 격자에서 연결된 집(1)의 개수를 찾아 단지의 수와 각 단지에 포함된 집의 수를 세는 문제입니다. BFS(너비 우선 탐색)를 사용하여 각 집에서 시작하여 인접한 집들을 탐색하며 단지를 구분하고, 집의 수를 세어서 결과를 저장합니다.

### 동작 과정
1. **입력 처리**: 먼저, 입력받은 2차원 배열을 `input_arr`에 저장합니다. `n`은 배열의 크기입니다.
   
2. **BFS 함수 정의**: BFS를 통해 집이 연결된 단지를 탐색합니다. 큐를 사용하여 현재 집의 위치를 저장하고, 상하좌우로 인접한 집을 탐색합니다. 탐색한 집은 `visited` 배열에 표시하여 중복 탐색을 방지합니다. 단지 내 집의 수를 세어 반환합니다.

3. **단지 탐색**: 이중 반복문을 통해 `input_arr`의 각 요소를 순회합니다. 집(1)이 있고 아직 방문하지 않은 곳이면 새로운 단지를 발견한 것입니다. 이 경우, `vilage`를 증가시키고 BFS를 호출하여 해당 단지의 집 수를 세어 `vilage_cnt` 리스트에 추가합니다.

4. **결과 출력**: 탐색이 끝난 후 단지의 수와 각 단지의 집 수를 정렬하여 출력합니다.

**예시 입력**:
```
7
0110100
0111100
0010000
0000000
0101110
0111010
0000000
```

**단계별 설명**:
- 첫 번째 반복에서 (0, 1) 위치의 집을 발견하고 BFS가 시작되어 (0, 1) → (1, 1) → (1, 2) → (1, 3)으로 연결된 집들을 탐색하여 첫 번째 단지의 집 수를 5로 세고, `vilage_cnt`에 추가합니다.
- 두 번째 반복에서 (4, 1) 위치의 집을 발견하고 BFS가 시작되어 (4, 1) → (4, 2) → (4, 3) → (5, 2) → (5, 1)로 연결된 집들을 탐색하여 두 번째 단지의 집 수를 7로 세고 `vilage_cnt`에 추가합니다.
- 모든 집을 탐색한 후, 최종적으로 단지 수 3개와 각 단지의 집 수를 정렬하여 출력합니다.

### 시간 복잡도
BFS를 사용하여 모든 집을 한 번씩 방문하므로 시간 복잡도는 O(N^2)입니다. 여기서 N은 격자의 한 변의 길이로, 입력 크기와 비례합니다.

## 코드
```python
from collections import deque

dx = [-1, 1, 0, 0]
dy = [0, 0, -1, 1]

n = int(input())

input_arr = [list(map(int, input().strip())) for _ in range(n)]

vilage_cnt = []  # 각 단지의 집 수를 저장할 리스트
visited = [[False] * n for _ in range(n)]
vilage = 0  # 단지 수

def BFS(x, y):
    queue = deque()
    queue.append((x,y))
    visited[x][y] = True
    cnt = 1   # 현재 단지에 속하는 집의 수

    while queue:
        cur_x, cur_y = queue.popleft()

        for i in range(4):     # 상하좌우 탐색
            nx = cur_x + dx[i]
            ny = cur_y + dy[i]

            # 배열을 벗어나지 않고, 집(1)이고 아직 방문하지 않은 곳이면 탐색
            if 0 <= nx < n and 0 <= ny < n and input_arr[nx][ny] == 1 and not visited[nx][ny]:
                visited[nx][ny] = True
                queue.append((nx, ny))
                cnt += 1

    return cnt

for i in range(n):
    for j in range(n):
        if input_arr[i][j] == 1 and not visited[i][j]:  # 집이 있고 방문하지 않은 경우
            vilage += 1    # 새로운 단지를 찾았으므로 단지 수 증가
            house_cnt = BFS(i, j)  # BFS로 단지 내 집의 수를 구함
            vilage_cnt.append(house_cnt)

vilage_cnt.sort()

print(vilage)

for cnt in vilage_cnt:
    print(cnt)


```
