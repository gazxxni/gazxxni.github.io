## 문제
1부터 n까지 번호가 붙어있는 n명의 사람이 영어 끝말잇기를 하고 있습니다. 영어 끝말잇기는 다음과 같은 규칙으로 진행됩니다. 


1번부터 번호 순서대로 한 사람씩 차례대로 단어를 말합니다. 
마지막 사람이 단어를 말한 다음에는 다시 1번부터 시작합니다. 
앞사람이 말한 단어의 마지막 문자로 시작하는 단어를 말해야 합니다. 
이전에 등장했던 단어는 사용할 수 없습니다. 
한 글자인 단어는 인정되지 않습니다. 


다음은 3명이 끝말잇기를 하는 상황을 나타냅니다. 

tank → kick → know → wheel → land → dream → mother → robot → tank

위 끝말잇기는 다음과 같이 진행됩니다.


1번 사람이 자신의 첫 번째 차례에 tank를 말합니다.
2번 사람이 자신의 첫 번째 차례에 kick을 말합니다.
3번 사람이 자신의 첫 번째 차례에 know를 말합니다.
1번 사람이 자신의 두 번째 차례에 wheel을 말합니다.
(계속 진행)


끝말잇기를 계속 진행해 나가다 보면, 3번 사람이 자신의 세 번째 차례에 말한 tank 라는 단어는 이전에 등장했던 단어이므로 탈락하게 됩니다. 

사람의 수 n과 사람들이 순서대로 말한 단어 words 가 매개변수로 주어질 때, 가장 먼저 탈락하는 사람의 번호와 그 사람이 자신의 몇 번째 차례에 탈락하는지를 구해서 return 하도록 solution 함수를 완성해주세요.

제한 사항


끝말잇기에 참여하는 사람의 수 n은 2 이상 10 이하의 자연수입니다.
words는 끝말잇기에 사용한 단어들이 순서대로 들어있는 배열이며, 길이는 n 이상 100 이하입니다.
단어의 길이는 2 이상 50 이하입니다.
모든 단어는 알파벳 소문자로만 이루어져 있습니다.
끝말잇기에 사용되는 단어의 뜻(의미)은 신경 쓰지 않으셔도 됩니다.
정답은 [ 번호, 차례 ] 형태로 return 해주세요.
만약 주어진 단어들로 탈락자가 생기지 않는다면, [0, 0]을 return 해주세요.

## 풀이
### 풀이 핵심 로직
끝말잇기 문제를 해결하기 위해, 단어를 차례대로 확인하며 이전에 사용된 단어와 마지막 단어의 규칙을 검증합니다. 각 단어가 사용되었는지 여부를 확인하기 위해 세트(set)를 사용하고, 현재 단어의 첫 글자가 이전 단어의 마지막 글자와 일치하는지 확인합니다. 만약 규칙에 어긋나는 경우 해당 플레이어의 번호와 차례를 반환합니다.

### 동작 과정
1. `words` 배열의 첫 번째 단어를 초기 단어로 설정하고, 이 단어를 기록합니다.
2. 각 단어를 순회하며:
   - 현재 플레이어의 번호를 계산합니다.
   - 이전 단어의 마지막 글자와 현재 단어의 첫 글자가 일치하는지 확인합니다.
   - 만약 일치하지 않으면 해당 플레이어의 번호와 차례를 반환합니다.
   - 만약 일치하면, 그 단어가 이전에 사용된 단어인지 확인합니다.
     - 사용된 단어라면 해당 플레이어의 번호와 차례를 반환합니다.
     - 사용되지 않았다면 현재 단어를 이전 단어로 설정하고, 사용된 단어 목록에 추가합니다.
3. 모든 단어를 검증한 후 탈락자가 없으면 [0, 0]을 반환합니다.

**예시 입력:**
- `n = 3`
- `words = ["tank", "kick", "know", "wheel", "land", "dream", "mother", "robot", "tank"]`

**단계별 설명:**
1. 첫 번째 단어 "tank"를 1번 플레이어가 사용.
2. 두 번째 단어 "kick"를 2번 플레이어가 사용.
3. 세 번째 단어 "know"를 3번 플레이어가 사용.
4. 네 번째 단어 "wheel"을 1번 플레이어가 사용.
5. 계속 진행하다가 3번 플레이어가 "tank"를 사용했을 때, 이전에 사용된 단어이므로 3번 플레이어가 탈락하게 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(m), 여기서 m은 `words`의 길이입니다. 각 단어를 한 번씩 확인하며, 세트에 단어를 추가하거나 확인하는 작업은 평균 O(1)의 시간 복잡도를 가집니다.

### 최종 코드
```python
def solution(n, words):
    used_words = set()  # 사용된 단어를 저장할 세트
    last_word = words[0]
    used_words.add(last_word)

    for i in range(1, len(words)):
        current_word = words[i]
        player_number = (i % n) + 1  # 1부터 n까지의 번호
        turn = (i // n) + 1  # 몇 번째 차례인지

        # 규칙 체크
        if current_word in used_words or current_word[0] != last_word[-1]:
            return [player_number, turn]
        
        # 규칙에 맞다면, 단어 추가 및 업데이트
        used_words.add(current_word)
        last_word = current_word

    return [0, 0]  # 탈락자가 없을 경우
``` 

이 코드는 주어진 문제의 요구사항을 충족하며, 각 조건을 명확하게 처리합니다.
