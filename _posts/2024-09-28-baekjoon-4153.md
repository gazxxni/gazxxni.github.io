## 풀이

### 풀이 핵심 로직
이 문제는 피타고라스의 정리를 이용하여 주어진 세 개의 숫자가 직각삼각형의 변의 길이인지 확인하는 문제입니다. 입력된 세 변의 길이를 정렬한 후, 가장 큰 변을 빗변으로 가정하고 나머지 두 변의 제곱의 합이 빗변의 제곱과 같은지를 확인합니다.

### 동작 과정
1. 입력을 통해 세 개의 정수(변의 길이)를 받습니다. 입력이 "0 0 0"인 경우 반복문을 종료합니다.
   - 예시 입력: `3 4 5`
2. 받은 세 변을 정렬합니다. 정렬 결과: `[3, 4, 5]`
3. 피타고라스의 정리를 적용하여 \(3^2 + 4^2 = 5^2\)를 계산합니다: \(9 + 16 = 25\) (참)
4. 조건이 참이므로 "right"을 출력합니다.
5. 다른 예시 입력: `5 12 13`
   - 정렬 후: `[5, 12, 13]`
   - 조건 확인: \(5^2 + 12^2 = 13^2\) (참)
   - 출력: "right"
6. 입력이 "0 0 0"이 되면 반복문을 종료합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \(O(1)\)입니다. 각 반복에서 세 개의 숫자를 정렬하는데 상수 시간(최대 3개의 수 정렬)과 몇 개의 산술 연산이 필요하기 때문에 전체적으로 상수 시간에 해당합니다.

## 코드
```python
import sys

while True:
    a=list(map(int,input().split()))

    if a[0]==0 & a[1]==0 & a[2]==0:
        break
    a.sort()

    if a[0]**2+a[1]**2==a[2]**2:
        print("right")
    else:
        print("wrong")

```
