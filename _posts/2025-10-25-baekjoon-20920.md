## 문제
화은이는 이번 영어 시험에서 틀린 문제를 바탕으로 영어 단어 암기를 하려고 한다. 그 과정에서 효율적으로 영어 단어를 외우기 위해 영어 단어장을 만들려 하고 있다. 화은이가 만들고자 하는 단어장의 단어 순서는 다음과 같은 우선순위를 차례로 적용하여 만들어진다.


	자주 나오는 단어일수록 앞에 배치한다.
	해당 단어의 길이가 길수록 앞에 배치한다.
	알파벳 사전 순으로 앞에 있는 단어일수록 앞에 배치한다


 M$M$보다 짧은 길이의 단어의 경우 읽는 것만으로도 외울 수 있기 때문에 길이가 M$M$이상인 단어들만 외운다고 한다. 화은이가 괴로운 영단어 암기를 효율적으로 할 수 있도록 단어장을 만들어 주자.

---

## 입력
첫째 줄에는 영어 지문에 나오는 단어의 개수 N$N$과 외울 단어의 길이 기준이 되는 M$M$이 공백으로 구분되어 주어진다. (1≤N≤100000$1 \leq N \leq 100\,000$, 1≤M≤10$1 \leq M \leq 10$)

둘째 줄부터 N+1$N+1$번째 줄까지 외울 단어를 입력받는다. 이때의 입력은 알파벳 소문자로만 주어지며 단어의 길이는 10$10$을 넘지 않는다.

단어장에 단어가 반드시 1개 이상 존재하는 입력만 주어진다.

---

## 출력
화은이의 단어장에 들어 있는 단어를 단어장의 앞에 위치한 단어부터 한 줄에 한 단어씩 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
화은이가 외우고자 하는 단어 목록에서 길이가 M 이상인 단어를 추출한 후, 각 단어의 등장 횟수를 세어 자주 나오는 단어를 먼저 정렬합니다. 이후 길이와 사전 순으로 정렬하여 최종적으로 출력합니다. 이를 위해 `defaultdict`를 사용하여 단어의 출현 빈도를 기록하고, 정렬할 때는 여러 기준을 동시에 고려합니다.

### 동작 과정
1. **입력 받기**:
   - 첫째 줄에서 단어의 개수 N과 길이 기준 M을 입력받습니다.
   - N개의 단어를 입력받으면서, 길이가 M 이상인 단어만 `dic`에 등장 횟수를 기록합니다.

2. **단어 등장 횟수 정리**:
   - `dic`에 저장된 단어와 그 횟수를 `dic2`라는 새로운 `defaultdict`로 옮깁니다. 여기서는 키를 등장 횟수, 값은 해당 횟수를 가진 단어 리스트로 설정합니다.

3. **정렬**:
   - `dic2`의 각 리스트에 대해 단어를 길이 기준(내림차순)과 사전 순(오름차순)으로 정렬합니다.
   - 마지막으로 `dic2`의 키(등장 횟수)를 기준으로 내림차순으로 정렬하여 최종 리스트를 만듭니다.

4. **출력**:
   - 정렬된 리스트를 순회하며 각 단어를 출력합니다.

예시 입력:
```
7 5
apple
banana
peach
apple
banana
peach
grape
```

- `N`은 7, `M`은 5입니다.
- 입력받은 단어 중 길이가 5 이상인 단어는 `["banana", "peach", "apple"]`입니다.
- 등장 횟수는 `{"banana": 2, "peach": 2, "apple": 2}`입니다.
- 이 단어들을 등장 횟수(2) 기준으로 정렬하고, 그 다음 길이와 사전 순으로 정렬합니다. 결과는 `["apple", "banana", "peach"]`가 됩니다.

### 시간 복잡도
주어진 코드의 시간 복잡도는 O(N log N)입니다. 
- 단어를 입력받고 세는 과정은 O(N)입니다.
- 정렬 과정에서 각 단어 리스트를 정렬하는 데 O(K log K) 소요되며, K는 각 등장 횟수에 해당하는 단어의 수입니다. 최종적으로 등장 횟수 기준으로 정렬하는 데도 O(K log K) 소요됩니다.
- 하지만 이 모든 과정을 통틀어 N개의 단어를 기준으로 정렬하므로 O(N log N)으로 정리할 수 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline
from collections import defaultdict

n, m = map(int, input().split())

dic = defaultdict(int)
for _ in range(n):
    a = input().rstrip()
    if len(a) >= m:
        dic[a] += 1
    
dic2 = defaultdict(list)
for k, v in dic.items():
    dic2[v].append(k)
    
for k, v in dic2.items():
    v.sort(key=lambda x: (-len(x), x))
    
a = sorted(dic2.items(), key=lambda x:x[0], reverse=True)

for i, j in a:
    for k in j:
        print(k)
```
