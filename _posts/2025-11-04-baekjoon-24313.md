## 문제
오늘도 서준이는 점근적 표기 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

알고리즘의 소요 시간을 나타내는 O-표기법(빅-오)을 다음과 같이 정의하자.

O(g(n)) = {f(n) | 모든 n ≥ n0에 대하여 f(n) ≤ c × g(n)인 양의 상수 c와 n0가 존재한다}

이 정의는 실제 O-표기법(https://en.wikipedia.org/wiki/Big_O_notation)과 다를 수 있다.

함수 f(n) = a1n + a0, 양의 정수 c, n0가 주어질 경우 O(n) 정의를 만족하는지 알아보자.

---

## 입력
첫째 줄에 함수 f(n)을 나타내는 정수 a1, a0가 주어진다. (0 ≤ |ai| ≤ 100)

다음 줄에 양의 정수 c가 주어진다. (1 ≤ c ≤ 100)

다음 줄에 양의 정수 n0가 주어진다. (1 ≤ n0 ≤ 100)

---

## 출력
f(n), c, n0가 O(n) 정의를 만족하면 1, 아니면 0을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 함수 f(n) = a1n + a0와 상수 c, n0에 대해 O(n)의 정의를 만족하는지 확인하기 위해, n이 n0 이상일 때 f(n) ≤ c * n이 성립하는지를 검증한다. 이를 위해 n0에서의 f(n)과 c * n의 값을 비교하고, a1이 c보다 작거나 같은지도 확인하여 조건을 만족하는지 판단한다.

### 동작 과정
1. **입력 예시**: a1 = 2, a0 = 3, c = 5, n0 = 10
2. **계산 과정**:
   - f(n) = 2n + 3
   - n = n0 = 10일 때, f(10) = 2 * 10 + 3 = 20 + 3 = 23
   - g(n) = c * n = 5 * 10 = 50
3. **비교**:
   - f(10) = 23 ≤ 50 = g(10) → 조건 1 만족
   - a1 = 2 ≤ c = 5 → 조건 2 만족
4. **결과**: 두 조건을 모두 만족하므로 1을 출력.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)이다. 입력값에 상관없이 상수 시간에 결과를 계산하기 때문이다.

---

## 코드
```python
a1, a0 = map(int, input().split())
c = int(input())
n = int(input())

f = a1 * n + a0
g = c * n

if f <= g and  a1 <= c:
    print(1)

else:
    print(0)
```
