## 문제 링크
[https://www.acmicpc.net/problem/32929](https://www.acmicpc.net/problem/32929)

## 문제
UOS 문자열이란 UOSUOSUOSU...와 같이 UOS가 무한히 반복되어 나타나는 문자열이다. 양의 정수 x$x$가 주어질 때 UOS 문자열의 x$x$번째 문자를 구하여라.

## 입력
첫 번째 줄에 x$x$가 주어진다. (1≤x≤109)$(1 \leq x \leq 10^9)$

## 출력
UOS 문자열의 x$x$번째 문자를 출력한다.

## 풀이
### 풀이 핵심 로직
UOS 문자열은 'UOS'가 무한히 반복되는 구조입니다. 따라서, x번째 문자를 찾기 위해서는 x를 3으로 나눈 나머지를 이용하여 해당 인덱스를 결정할 수 있습니다. 나머지가 1이면 'U', 2면 'O', 0이면 'S'에 해당합니다.

### 동작 과정
1. 입력된 x 값이 1부터 10^9까지의 범위에 있는 양의 정수라고 가정합니다.
2. UOS 문자열의 길이는 3이므로, x를 3으로 나눈 나머지를 계산합니다.
   - 예를 들어, x = 1일 경우, `1 % 3`는 1이므로 1번째 문자인 'U'를 출력합니다.
   - x = 2일 경우, `2 % 3`는 2이므로 2번째 문자인 'O'를 출력합니다.
   - x = 3일 경우, `3 % 3`는 0이므로 3번째 문자인 'S'를 출력합니다.
3. 나머지가 0일 경우에는 3번째 문자를 출력하기 위해 인덱스를 조정합니다. 따라서, x % 3이 0일 때는 'S'를 출력하도록 합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 x에 대해 상수 시간 내에 결과를 계산할 수 있기 때문입니다.

## 코드
```python
s = 'UOS'
x = int(input())
print(s[x%3-1])
```
