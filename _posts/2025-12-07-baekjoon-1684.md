## 문제
정수 N을 정수 D로 나눴을 때의 몫을 Q, 나머지를 R이라고 하면 항등식 R = N - Q×D가 성립한다.

n개의 정수로 된 수열이 있을 때, 모든 정수를 한 정수 D로 나눴을 때 나머지가 같아지는 경우가 있다. 그리고 수열에 따라서는 이러한 정수 D가 여러 개 존재할 수 있다.

n개의 정수로 된 수열이 주어졌을 때, 가장 큰 D를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 n(1 ≤ n ≤ 1,000)이 주어진다. 다음 줄에는 절댓값이 1,000,000을 넘지 않는 n개의 정수들이 주어진다.

---

## 출력
첫째 줄에 가장 큰 D를 출력한다. 항상 가장 큰 D가 존재하는 경우만 입력으로 주어진다.

## 풀이

### 풀이 핵심 로직
주어진 n개의 정수에서 같은 나머지를 가지기 위해서는 모든 정수의 차가 어떤 정수 D로 나누어 떨어져야 합니다. 따라서, 이러한 차이들의 최대공약수(GCD)를 구하면, 그 GCD가 가장 큰 D가 됩니다. 즉, 각 수의 차이들로부터 GCD를 계산하여 가장 큰 D를 찾습니다.

### 동작 과정
1. 입력으로 주어진 n개의 정수를 리스트로 받습니다.
2. 리스트의 각 요소의 차이를 구하기 위해, 중복을 제거한 리스트를 만듭니다.
3. 두 수의 차이를 구하고, 이 차이들의 GCD를 계산합니다.
4. GCD를 반복적으로 계산하여 최종적으로 가장 큰 D를 구합니다.

예시 입력:
```
5
1 2 3 4 5
```
- 중복을 제거한 리스트는 `[1, 2, 3, 4, 5]`입니다.
- 차이 리스트는 `|2-1|, |3-1|, |4-1|, |5-1|, |3-2|, |4-2|, |5-2|, |4-3|, |5-3|, |5-4|`로, 계산하면 `[1, 2, 3, 4, 1, 2, 3, 1, 2, 1]`입니다.
- 이 값들의 GCD를 계산하면 `1`이 나오고, 이는 주어진 수열의 모든 정수가 같은 나머지를 가지기 위한 가장 큰 D입니다.

### 시간 복잡도
주어진 수열의 차이를 구하기 위해 O(n^2) 시간이 소요되며, GCD를 계산하는 과정은 최대 n-1번의 연산이 들어가므로 전체 시간 복잡도는 O(n^2)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

'''
두 수의 차가 D로 나누어 떨어진다
(aᵢ - aⱼ) % D = 0 -> 성립해야함

따라서 |a₂ - a₁|, |a₃ - a₁|, ..., |aₙ - a₁|
이 숫자들의 최대공약수를 구해야함
'''

n = int(input())
arr = list(map(int, input().split()))

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
        
    return a

brr = list(set(arr))
crr = []

for i in range(len(brr) - 1):
    for j in range(i + 1, len(brr)):
        crr.append(abs(brr[j] - brr[i])) 
        
ans = crr[0]
for i in range(1, len(crr)): 
    ans = gcd(ans, crr[i])
    
print(ans)
```
