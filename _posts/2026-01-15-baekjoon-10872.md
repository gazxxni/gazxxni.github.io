## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 N과 N-1, N-2, ..., 1을 곱한 결과로 정의됩니다. 이 문제에서는 재귀 함수를 사용하여 N의 팩토리얼을 계산합니다. 특히, N이 0일 때는 1로 정의되므로 이를 따로 처리합니다.

### 동작 과정
1. 사용자가 입력한 정수 N을 받습니다. 예를 들어, N이 5라면, 프로그램은 N의 팩토리얼인 5!을 계산해야 합니다.
2. `fac`라는 재귀 함수를 정의합니다. 이 함수는 다음과 같이 동작합니다:
   - 만약 N이 1이면 1을 반환합니다 (기저 사례).
   - 그 외의 경우, N과 `fac(n - 1)`의 결과를 곱합니다.
3. 만약 입력값이 0이면, 팩토리얼 값은 1로 정의되므로 바로 1을 출력합니다.
4. 그렇지 않으면, `fac` 함수를 호출하여 N의 팩토리얼 값을 계산하고 출력합니다.

예를 들어, N이 5인 경우:
- fac(5) 호출
  - fac(5) = 5 * fac(4)
    - fac(4) = 4 * fac(3)
      - fac(3) = 3 * fac(2)
        - fac(2) = 2 * fac(1)
          - fac(1) = 1 (기저 사례)
        - fac(2) = 2 * 1 = 2
      - fac(3) = 3 * 2 = 6
    - fac(4) = 4 * 6 = 24
  - fac(5) = 5 * 24 = 120
- 최종적으로 120이 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. N이 증가할수록 재귀 호출이 N번 일어나기 때문입니다.

---

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
