## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
팩토리얼은 주어진 정수 N에 대해 N! = N × (N-1) × (N-2) × ... × 1로 정의됩니다. N이 0일 경우, 0!은 1로 정의되므로 이를 별도로 처리하고, 나머지 경우에는 재귀 함수를 사용하여 N!을 계산합니다.

### 동작 과정
1. 입력으로 정수 N을 받습니다. 예를 들어, 입력이 5인 경우 N = 5입니다.
2. `fac` 함수가 호출되어 N이 1이 아닐 경우, N과 `fac(N-1)`의 곱을 반환합니다. 
   - fac(5) 호출: 5 * fac(4)
   - fac(4) 호출: 4 * fac(3)
   - fac(3) 호출: 3 * fac(2)
   - fac(2) 호출: 2 * fac(1)
   - fac(1) 호출: 1 (재귀 종료)
3. 각 호출 스택이 해제되며, 다음과 같은 계산이 이루어집니다:
   - fac(2) = 2 * 1 = 2
   - fac(3) = 3 * 2 = 6
   - fac(4) = 4 * 6 = 24
   - fac(5) = 5 * 24 = 120
4. 최종적으로 120이 출력됩니다.

### 시간 복잡도
O(N): N에 대해 재귀적으로 호출되므로, N이 클수록 호출 횟수가 증가합니다. 하지만 N의 최대값이 12로 제한되어 있어, 실제로는 수행 시간에 큰 차이가 없습니다.

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
