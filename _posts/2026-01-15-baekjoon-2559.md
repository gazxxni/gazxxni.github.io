## 입력

## 풀이

### 풀이 핵심 로직
이 문제는 슬라이딩 윈도우(Sliding Window) 기법을 활용하여 해결할 수 있습니다. 첫 K일의 온도의 합을 먼저 계산한 후, 이후의 날들에 대해서는 이전 K일의 합에서 가장 오래된 날의 온도를 빼고 새로운 날의 온도를 더하는 방식으로 연속적인 K일의 합을 효율적으로 갱신합니다. 이 과정을 통해 전체 N일의 온도 배열을 한 번만 순회하므로 최적의 성능을 확보할 수 있습니다.

### 동작 과정
1. 입력으로 주어진 온도 배열을 읽어온다. 예를 들어, N=10, K=3이고 온도 배열이 `[3, -2, -4, -9, 0, 3, 7, 13, 8, -3]`라고 가정하겠다.
2. 처음 K일의 합인 첫 3일의 온도 합을 계산한다: `3 + (-2) + (-4) = -3`.
3. 이 값을 `ans` 변수에 저장한다.
4. 그 다음 날부터 K일의 합을 갱신한다. 예를 들어, 4일째(인덱스 3)부터 시작하여:
   - 4일째 합: `-3 - 3 + 0 = -6` (이전 합에서 1일 전의 온도인 -3을 빼고, 새로운 온도인 0을 더함)
   - 5일째 합: `-6 - (-2) + 3 = -5`
   - 6일째 합: `-5 - (-4) + 7 = 6`
   - 7일째 합: `6 - (-9) + 13 = 28`
   - 8일째 합: `28 - 0 + 8 = 36`
   - 9일째 합: `36 - 3 + (-3) = 30`
5. 각 단계에서 구한 K일의 합을 `ans`와 비교하여 최대값을 갱신한다. 최종적으로 모든 K일의 합 중 최대값을 출력한다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. N일의 온도를 한 번만 순회하면서 K일의 합을 효율적으로 갱신하기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
