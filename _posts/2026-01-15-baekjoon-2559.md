## 입력

## 풀이

### 풀이 핵심 로직
주어진 온도 수열에서 K일 동안의 온도의 합을 최대화하기 위해 슬라이딩 윈도우 기법을 사용합니다. 처음 K일의 합을 계산한 후, 이후의 날마다 이전 K일의 합에서 가장 먼저 들어간 날의 온도를 빼고, 새로 추가된 날의 온도를 더해서 새로운 합을 구합니다. 이 과정을 반복하면서 최대 합을 갱신합니다.

### 동작 과정
주어진 입력이 다음과 같다고 가정합시다:
```
10 3
3 -2 -4 -9 0 3 7 13 8 -3
```
1. **입력 파싱**: N=10, K=3, 온도 수열 = [3, -2, -4, -9, 0, 3, 7, 13, 8, -3]
2. **초기 K일의 합 계산**: 처음 3일의 합 (3 + -2 + -4) = -3
3. **최대 합 초기화**: ans = -3
4. **슬라이딩 윈도우 시작**:
   - i=3 (날짜 4): 합 = -3 - 3 (첫 날) + -9 = -9 → ans = max(-3, -9) = -3
   - i=4 (날짜 5): 합 = -9 - -2 (첫 날) + 0 = -7 → ans = max(-3, -7) = -3
   - i=5 (날짜 6): 합 = -7 - -4 (첫 날) + 3 = 0 → ans = max(-3, 0) = 0
   - i=6 (날짜 7): 합 = 0 - -9 (첫 날) + 7 = 16 → ans = max(0, 16) = 16
   - i=7 (날짜 8): 합 = 16 - 0 (첫 날) + 13 = 29 → ans = max(16, 29) = 29
   - i=8 (날짜 9): 합 = 29 - 3 (첫 날) + 8 = 34 → ans = max(29, 34) = 34
   - i=9 (날짜 10): 합 = 34 - 7 (첫 날) + -3 = 24 → ans = max(34, 24) = 34
5. **결과 출력**: 최종 ans = 34

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. N개의 온도 수를 한 번만 순회하면서 K일의 합을 계산하므로 효율적입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
