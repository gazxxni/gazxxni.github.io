## 문제
오늘도 서준이는 점근적 표기 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

알고리즘의 소요 시간을 나타내는 O-표기법(빅-오)을 다음과 같이 정의하자.

O(g(n)) = {f(n) | 모든 n ≥ n0에 대하여 f(n) ≤ c × g(n)인 양의 상수 c와 n0가 존재한다}

이 정의는 실제 O-표기법(https://en.wikipedia.org/wiki/Big_O_notation)과 다를 수 있다.

함수 f(n) = a1n + a0, 양의 정수 c, n0가 주어질 경우 O(n) 정의를 만족하는지 알아보자.

---

## 입력
첫째 줄에 함수 f(n)을 나타내는 정수 a1, a0가 주어진다. (0 ≤ |ai| ≤ 100)

다음 줄에 양의 정수 c가 주어진다. (1 ≤ c ≤ 100)

다음 줄에 양의 정수 n0가 주어진다. (1 ≤ n0 ≤ 100)

---

## 출력
f(n), c, n0가 O(n) 정의를 만족하면 1, 아니면 0을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 함수 f(n) = a1n + a0가 O(n) 정의를 만족하기 위해서는 두 가지 조건을 충족해야 합니다. 첫째, n이 n0 이상일 때 f(n) ≤ c × n이 성립해야 하고, 둘째, a1이 c보다 작거나 같아야 합니다. 이 두 조건이 모두 만족하면 1을 출력하고, 그렇지 않으면 0을 출력합니다.

### 동작 과정
1. 입력으로 a1, a0, c, n0 값을 받습니다.
   - 예를 들어, a1 = 2, a0 = 3, c = 5, n0 = 10이라고 가정합니다.
2. n0를 기준으로 f(n)과 g(n)을 계산합니다:
   - f(n0) = a1 * n0 + a0 = 2 * 10 + 3 = 23
   - g(n0) = c * n0 = 5 * 10 = 50
3. 두 번째 조건을 확인합니다:
   - a1 ≤ c → 2 ≤ 5, 이 조건은 만족합니다.
4. 첫 번째 조건을 확인합니다:
   - f(n0) ≤ g(n0) → 23 ≤ 50, 이 조건도 만족합니다.
5. 두 조건이 모두 만족하므로, 1을 출력합니다.

### 시간 복잡도
이 코드의 시간 복잡도는 O(1)입니다. 입력 값을 읽고 조건을 확인하는 연산이 상수 시간 내에 이루어지기 때문입니다.

---

## 코드
```python
a1, a0 = map(int, input().split())
c = int(input())
n = int(input())

f = a1 * n + a0
g = c * n

if f <= g and  a1 <= c:
    print(1)

else:
    print(0)
```
