## 입력

## 풀이

### 풀이 핵심 로직
문제는 주어진 온도 수열에서 연속적인 K일 동안의 온도 합이 최대가 되는 값을 찾는 것입니다. 이를 위해 슬라이딩 윈도우 기법을 사용하여 처음 K일의 합을 계산한 후, 다음 날부터는 이전의 합에서 K일 전의 값을 빼고 현재 값을 더하는 방식으로 연속적인 K일의 합을 효율적으로 계산합니다.

### 동작 과정
주어진 입력 예시를 통해 단계별로 설명하겠습니다.

1. **입력**: 
   ```
   10 5
   3 -2 -4 -9 0 3 7 13 8 -3
   ```
   - N = 10 (온도 측정일수)
   - K = 5 (연속적으로 합을 구할 일수)
   - 온도 배열: `arr = [3, -2, -4, -9, 0, 3, 7, 13, 8, -3]`

2. **처음 K일의 합 계산**: 
   - `ss = sum(arr[:5])` → `ss = 3 - 2 - 4 - 9 + 0 = -12`
   - 최대값 초기화: `ans = ss` → `ans = -12`

3. **슬라이딩 윈도우 시작**:
   - 인덱스 5부터 시작하여 N까지 반복합니다.
   - 각 반복에서 현재 K일의 합을 업데이트하고, 최대값을 비교합니다.

   - **인덱스 5**:
     ```
     ss = ss - arr[0] + arr[5] → ss = -12 - 3 + 3 = -12
     ans = max(ans, ss) → ans = -12
     ```

   - **인덱스 6**:
     ```
     ss = ss - arr[1] + arr[6] → ss = -12 - (-2) + 7 = -7
     ans = max(ans, ss) → ans = -7
     ```

   - **인덱스 7**:
     ```
     ss = ss - arr[2] + arr[7] → ss = -7 - (-4) + 13 = 10
     ans = max(ans, ss) → ans = 10
     ```

   - **인덱스 8**:
     ```
     ss = ss - arr[3] + arr[8] → ss = 10 - (-9) + 8 = 27
     ans = max(ans, ss) → ans = 27
     ```

   - **인덱스 9**:
     ```
     ss = ss - arr[4] + arr[9] → ss = 27 - 0 - 3 = 24
     ans = max(ans, ss) → ans = 27
     ```

4. **결과 출력**:
   - 최종적으로 `print(ans)` → `27`이 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 슬라이딩 윈도우 방식 덕분에 배열을 한 번만 순회하면서 K일의 합을 효율적으로 계산할 수 있기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
arr = list(map(int, input().split()))

ss = sum(arr[:k])
ans = ss

for i in range(k, n):
    ss = ss - arr[i - k] + arr[i]
    ans = max(ans, ss)

print(ans)
```
