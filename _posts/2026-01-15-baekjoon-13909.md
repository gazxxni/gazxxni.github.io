## 문제
서강대학교 컴퓨터공학과 실습실 R912호에는 현재 N개의 창문이 있고 또 N명의 사람이 있다. 1번째 사람은 1의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다.  2번째 사람은 2의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다. 이러한 행동을 N번째 사람까지 진행한 후 열려 있는 창문의 개수를 구하라. 단, 처음에 모든 창문은 닫혀 있다.

예를 들어 현재 3개의 창문이 있고 3명의 사람이 있을 때,


	1번째 사람은 1의 배수인 1,2,3번 창문을 연다. (1, 1, 1)
	2번째 사람은 2의 배수인 2번 창문을 닫는다. (1, 0, 1)
	3번째 사람은 3의 배수인 3번 창문을 닫는다. (1, 0, 0)


결과적으로 마지막에 열려 있는 창문의 개수는 1개 이다.

---

## 입력
첫 번째 줄에는 창문의 개수와 사람의 수 N(1 ≤ N ≤ 2,100,000,000)이 주어진다.

---

## 출력
마지막에 열려 있는 창문의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 각 창문이 몇 번 열리고 닫히는지를 이해하는 것입니다. 각 창문 `i`는 약수의 개수만큼 상태가 바뀌는데, 홀수 개의 약수를 가지는 경우(즉, 창문이 열린 상태로 남는 경우)는 정사각형 수에 해당합니다. 따라서, `N`개의 창문 중에서 열려 있는 창문의 개수는 `N`의 제곱근을 구한 값의 정수 부분입니다.

### 동작 과정
1. **초기 상태**: 모든 창문은 닫혀 있습니다.
2. **1번째 사람**: 1의 배수인 모든 창문(1, 2, ..., N)을 열기 시작합니다. 결과적으로 모든 창문이 열립니다.
3. **2번째 사람**: 2의 배수인 창문(2, 4, 6, ...)을 닫습니다. 짝수 번째 창문들이 닫히고, 홀수 번째 창문들은 유지됩니다.
4. **3번째 사람**: 3의 배수인 창문(3, 6, 9, ...)을 닫습니다. 이때 닫히는 창문은 3, 6, 9 등입니다.
5. 이 과정을 N번째 사람까지 반복합니다.

결과적으로, 창문 `k`는 `k`의 약수의 개수만큼 상태를 변경합니다. 홀수 개의 약수를 가지는 경우는 `k`가 완전 제곱수일 때만 발생하므로, 열려 있는 창문의 개수는 `N` 이하의 완전 제곱수의 개수와 같습니다.

예를 들어, `N = 3`일 때, 열려 있는 창문은 1개(창문 1)입니다. `N = 10`일 때는 3개(1, 4, 9)입니다.

### 시간 복잡도
O(1) - 입력을 받고 제곱근을 계산하는 과정은 상수 시간에 이루어집니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
print(int(n ** 0.5))
```
