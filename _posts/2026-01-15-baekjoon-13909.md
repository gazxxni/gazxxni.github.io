## 문제
서강대학교 컴퓨터공학과 실습실 R912호에는 현재 N개의 창문이 있고 또 N명의 사람이 있다. 1번째 사람은 1의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다.  2번째 사람은 2의 배수 번째 창문을 열려 있으면 닫고 닫혀 있으면 연다. 이러한 행동을 N번째 사람까지 진행한 후 열려 있는 창문의 개수를 구하라. 단, 처음에 모든 창문은 닫혀 있다.

예를 들어 현재 3개의 창문이 있고 3명의 사람이 있을 때,


	1번째 사람은 1의 배수인 1,2,3번 창문을 연다. (1, 1, 1)
	2번째 사람은 2의 배수인 2번 창문을 닫는다. (1, 0, 1)
	3번째 사람은 3의 배수인 3번 창문을 닫는다. (1, 0, 0)


결과적으로 마지막에 열려 있는 창문의 개수는 1개 이다.

---

## 입력
첫 번째 줄에는 창문의 개수와 사람의 수 N(1 ≤ N ≤ 2,100,000,000)이 주어진다.

---

## 출력
마지막에 열려 있는 창문의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 각 창문이 열리거나 닫히는 횟수가 그 창문의 배수에 해당하는 사람 수에 의해 결정된다는 것입니다. 각 창문은 약수의 개수만큼 상태가 변경되는데, 홀수 개의 약수를 가지는 경우(즉, 완전 제곱수인 경우)에만 마지막에 열려 있게 됩니다. 따라서, 열려 있는 창문의 개수는 N 이하의 완전 제곱수의 개수로 줄여져, 이는 `int(n ** 0.5)`로 계산할 수 있습니다.

### 동작 과정
예를 들어 N = 3일 때:

1. **초기 상태**: 모든 창문은 닫혀 있습니다. (0, 0, 0)
   
2. **1번째 사람** (1의 배수): 모든 창문을 열기 (1, 1, 1)
   
3. **2번째 사람** (2의 배수): 2번 창문을 닫기 (1, 0, 1)
   
4. **3번째 사람** (3의 배수): 3번 창문을 닫기 (1, 0, 0)

최종적으로 열려 있는 창문은 1개입니다. N = 3의 경우, 완전 제곱수는 1(1^2), 4(2^2)까지 존재하므로 총 1개입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력으로 주어진 N에 대해 단순하게 제곱근을 계산하는 것이기 때문입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
print(int(n ** 0.5))
```
