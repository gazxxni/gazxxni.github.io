## 풀이

### 풀이 핵심 로직
이 문제는 주어진 숫자 \( n \)에 대해 두 개의 값을 출력하는 간단한 문제입니다. 첫 번째 출력은 입력받은 \( n \) 자체이며, 두 번째 출력은 항상 1입니다. 이 문제는 입력 크기와 무관하게 고정된 출력을 요구합니다.

### 동작 과정
1. 사용자가 입력으로 정수를 입력합니다. 예를 들어, 사용자가 `5`를 입력했다고 가정합니다.
2. 프로그램은 `n` 변수에 5를 저장합니다.
3. 첫 번째 출력으로 `5`를 출력합니다.
4. 두 번째 출력으로 `1`을 출력합니다.
   
   최종 출력 결과:
   ```
   5
   1
   ```

### 시간 복잡도
이 프로그램의 시간 복잡도는 \( O(1) \)입니다. 입력 크기와 관계없이 항상 두 개의 출력을 생성하므로, 시간 복잡도는 상수 시간입니다.

## 코드
```python
n = int(input())
print(n)
print(1)
```
