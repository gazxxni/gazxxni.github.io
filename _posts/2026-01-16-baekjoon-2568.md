## 문제
두 전봇대 A와 B 사이에 하나 둘씩 전깃줄을 추가하다 보니 전깃줄이 서로 교차하는 경우가 발생하였다. 합선의 위험이 있어 이들 중 몇 개의 전깃줄을 없애 전깃줄이 교차하지 않도록 만들려고 한다.

예를 들어, 과 같이 전깃줄이 연결되어 있는 경우 A의 1번 위치와 B의 8번 위치를 잇는 전깃줄, A의 3번 위치와 B의 9번 위치를 잇는 전깃줄, A의 4번 위치와 B의 1번 위치를 잇는 전깃줄을 없애면 남아있는 모든 전깃줄이 서로 교차하지 않게 된다. 





전깃줄이 전봇대에 연결되는 위치는 전봇대 위에서부터 차례대로 번호가 매겨진다. 전깃줄의 개수와 전깃줄들이 두 전봇대에 연결되는 위치의 번호가 주어질 때, 남아있는 모든 전깃줄이 서로 교차하지 않게 하기 위해 없애야 하는 최소 개수의 전깃줄을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에는 두 전봇대 사이의 전깃줄의 개수가 주어진다. 전깃줄의 개수는 100,000 이하의 자연수이다. 둘째 줄부터 한 줄에 하나씩 전깃줄이 A전봇대와 연결되는 위치의 번호와 B전봇대와 연결되는 위치의 번호가 차례로 주어진다. 위치의 번호는 500,000 이하의 자연수이고, 같은 위치에 두 개 이상의 전깃줄이 연결될 수 없다.

---

## 출력
첫째 줄에 남아있는 모든 전깃줄이 서로 교차하지 않게 하기 위해 없애야 하는 전깃줄의 최소 개수를 출력한다. 둘째 줄부터 한 줄에 하나씩 없애야 하는 전깃줄의 A전봇대에 연결되는 위치의 번호를 오름차순으로 출력한다. 만약 답이 두 가지 이상이라면 그 중 하나를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 주어진 전깃줄들이 교차하지 않도록 최소한의 전깃줄을 제거하는 것입니다. 이를 위해 가장 긴 증가하는 부분 수열(LIS) 개념을 활용하여, 전깃줄의 B전봇대 위치들을 기준으로 정렬한 후, LIS를 찾아 교차하지 않는 전깃줄의 최대 개수를 구합니다. 제거해야 할 전깃줄의 수는 전체 전깃줄 수에서 LIS의 길이를 뺀 값이 됩니다.

### 동작 과정
1. **입력 및 정렬**: 전깃줄의 개수 `n`을 입력받고, 각 전깃줄의 A전봇대와 B전봇대의 연결 위치를 튜플로 저장한 후, A전봇대의 위치 기준으로 정렬합니다. 이때, 동일한 A전봇대의 위치는 B전봇대 위치를 기준으로 정렬하여 순서를 유지합니다.

2. **LIS 계산**: B전봇대 위치들만 추출하여 LIS를 찾습니다. 이 과정에서 이진 탐색을 활용하여 `tails` 리스트를 유지합니다. `tails` 리스트는 현재까지의 LIS의 마지막 원소들을 저장하는 리스트입니다.

3. **전깃줄 제거 결정**: LIS의 길이를 구한 후, 전체 전깃줄 수에서 LIS의 길이를 뺀 값을 계산하여 제거해야 할 전깃줄의 수를 결정합니다. 이후 어떤 전깃줄을 제거해야 하는지 파악하기 위해, `prev_idx` 배열을 사용하여 제거할 전깃줄의 인덱스를 추적합니다.

4. **결과 출력**: 제거해야 할 전깃줄의 A전봇대 위치를 오름차순으로 정렬하여 출력합니다.

예를 들어, 다음과 같은 입력이 주어질 때:
```
8
1 8
3 9
4 1
9 2
6 3
7 4
2 5
5 6
```
1. 전깃줄을 A전봇대 위치 기준으로 정렬하면:
```
(1, 8), (2, 5), (3, 9), (4, 1), (5, 6), (6, 3), (7, 4), (9, 2)
```
2. B전봇대 위치들을 추출하여 LIS를 계산하면, 최대 교차하지 않는 전깃줄 수를 찾을 수 있습니다.
3. 최종적으로 제거해야 할 전깃줄의 A전봇대 위치를 출력합니다.

### 시간 복잡도
이 알고리즘은 전깃줄을 정렬하는 데 O(n log n), 그리고 LIS를 찾기 위한 이진 탐색을 포함한 반복 과정에서 O(n log n)의 시간이 소요됩니다. 따라서 전체 시간 복잡도는 O(n log n)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = []
for _ in range(n):
    a, b = map(int, input().split())
    arr.append((a, b))
arr.sort()
brr = []
for a, b in arr:
    brr.append(b)

tails = []
tails_pos = []

prev_idx = [-1] * n

def search(a, x):
    lo, hi = 0, len(a)
    while lo < hi:
        mid = (lo + hi) // 2
        if a[mid] < x:
            lo = mid + 1
        else:
            hi = mid
    return lo

for i, x in enumerate(brr):
    j = search(tails, x)

    prev_idx[i] = tails_pos[j - 1] if j > 0 else -1

    if j == len(tails): 
        tails.append(x)
        tails_pos.append(i)
    else:    
        tails[j] = x
        tails_pos[j] = i

lis_len = len(tails)
k = tails_pos[-1] if tails_pos else -1

keep = [False] * n
k = tails_pos[-1] if tails_pos else -1
while k != -1:
    keep[k] = True
    k = prev_idx[k]
    
ans = [arr[i][0] for i in range(n) if not keep[i]]
ans.sort()
        
print(n - lis_len)
for a in ans:
    print(a)
```
