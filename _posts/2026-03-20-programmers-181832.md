## 풀이
### 풀이 핵심 로직
이 문제의 핵심은 주어진 n x n 배열에 1부터 n^2까지의 정수를 시계방향으로 나선형으로 배치하는 것입니다. 이를 위해 배열을 생성하고, 이동 방향을 정의한 후, 각 숫자를 배열에 순서대로 채워 넣습니다. 이동 중에 경계나 이미 채워진 칸에 부딪히면 방향을 바꿉니다.

### 동작 과정
1. **초기화**: n x n 크기의 배열을 0으로 초기화합니다.
2. **방향 정의**: 시계방향으로 이동하기 위해 오른쪽, 아래, 왼쪽, 위의 방향을 배열로 정의합니다.
3. **숫자 채우기**: 1부터 n^2까지 반복하여 현재 위치에 숫자를 채웁니다.
4. **위치 업데이트**: 현재 방향으로 한 칸 이동한 후, 만약 이동한 위치가 배열의 경계를 넘거나 이미 숫자가 채워져 있다면 방향을 바꿉니다.
5. **반복**: 이 과정을 n^2번 반복하며 완성된 배열을 반환합니다.

예를 들어, n=4인 경우 다음과 같은 과정을 거칩니다:
- 처음 위치 (0, 0)에서 1을 채우고 오른쪽으로 이동 (0, 1) -> 2를 채우고 (0, 2) -> 3을 채우고 (0, 3) -> 4를 채운 후 아래로 이동 (1, 3) -> 5를 채우고 (2, 3) -> 6을 채우고 (3, 3) -> 7을 채운 후 왼쪽으로 이동 (3, 2) -> 8을 채우고 (3, 1) -> 9을 채우고 (3, 0) -> 10을 채운 후 위로 이동 (2, 0) -> 11을 채우고 (1, 0) -> 12을 채운 후 오른쪽으로 이동 (1, 1) -> 13을 채우고 (1, 2) -> 14을 채운 후 아래로 이동 (2, 2) -> 15을 채우고 (2, 3) -> 16을 채웁니다.

이런 식으로 배열의 모든 칸이 채워집니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. n x n 배열의 모든 칸에 대해 1번씩 숫자를 채워야 하므로, n이 커질수록 연산량은 n^2에 비례하게 증가합니다.

## 코드
```python
def solution(n):
    answer = [[0] * n for _ in range(n)]

    directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]
    d = 0
    r, c = 0, 0
    for i in range(1, n * n + 1):
        answer[r][c] = i
        
        dr, dc = directions[d]
        nr = r + dr
        nc = c + dc
        
        if nc < 0 or nr < 0 or nc > n - 1 or nr > n - 1 or answer[nr][nc] != 0:
            d = (d + 1) % 4
            dr, dc = directions[d]
            nr = r + dr
            nc = c + dc
        
        r, c = nr, nc
        
    return answer
```
