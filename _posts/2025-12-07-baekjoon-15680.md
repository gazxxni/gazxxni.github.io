## 문제
연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.

이를 출력하는 프로그램을 작성해보도록 하자.

---

## 입력
첫째 줄에 N이 주어진다. (N = 0 또는 1)

---

## 출력
N = 0일 경우: 연세대학교의 영문명을 출력한다.
	N = 1일 경우: 연세대학교의 슬로건을 출력한다.


대소문자 구별에 주의하도록 하자.

## 풀이

### 풀이 핵심 로직
이 문제는 입력된 정수 N의 값에 따라 두 가지 문자열 중 하나를 출력하는 간단한 구현 문제입니다. N이 0일 경우 "YONSEI"를, N이 1일 경우 "Leading the Way to the Future"를 출력합니다.

### 동작 과정
1. 사용자로부터 입력을 받아 N 값을 정수형으로 변환합니다.
2. 조건문을 사용하여 N의 값에 따라 출력할 내용을 결정합니다.
   - N이 0이면 "YONSEI"를 출력합니다.
   - N이 1이면 "Leading the Way to the Future"를 출력합니다.

예시 입력과 출력 과정:
- 입력: `0`
  - 조건문에서 N이 0이므로 "YONSEI"가 출력됩니다.
- 입력: `1`
  - 조건문에서 N이 1이므로 "Leading the Way to the Future"가 출력됩니다.

### 시간 복잡도
이 문제는 단순한 조건문과 출력으로 구성되어 있으므로 시간 복잡도는 O(1)입니다.

---

## 코드
```python
n = int(input())

if n == 0:
    print("YONSEI")
else:
    print("Leading the Way to the Future")
```
