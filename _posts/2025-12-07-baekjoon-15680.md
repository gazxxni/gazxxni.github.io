## 문제
연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.

이를 출력하는 프로그램을 작성해보도록 하자.

---

## 입력
첫째 줄에 N이 주어진다. (N = 0 또는 1)

---

## 출력
N = 0일 경우: 연세대학교의 영문명을 출력한다.
	N = 1일 경우: 연세대학교의 슬로건을 출력한다.


대소문자 구별에 주의하도록 하자.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 입력 값에 따라 두 가지 문자열 중 하나를 출력하는 매우 간단한 구현 문제입니다. 입력으로 주어진 정수 N이 0이면 "YONSEI"를 출력하고, 1이면 "Leading the Way to the Future"를 출력합니다. 조건문을 사용하여 N의 값을 판단하고 해당하는 문자열을 출력하면 됩니다.

### 동작 과정
1. 프로그램이 시작되면 사용자로부터 정수 N을 입력받습니다.
2. 입력된 N이 0일 경우, "YONSEI"를 출력합니다.
3. 입력된 N이 1일 경우, "Leading the Way to the Future"를 출력합니다.

예시 입력:
- N = 0일 때: 출력은 "YONSEI"
- N = 1일 때: 출력은 "Leading the Way to the Future"

### 시간 복잡도
이 프로그램의 시간 복잡도는 O(1)입니다. 입력이 주어지고 조건문을 통해 출력이 이루어지기 때문에 입력의 크기와 관계없이 항상 일정한 시간 내에 실행됩니다.

---

## 코드
```python
n = int(input())

if n == 0:
    print("YONSEI")
else:
    print("Leading the Way to the Future")
```
