## 문제
두 정수 A와 B가 주어졌을 때, 두 수의 곱을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 A와 B가 주어진다. 두 정수는 0보다 크거나 같은 정수이며, 0을 제외한 정수는 0으로 시작하지 않으며, 수의 앞에 불필요한 0이 있는 경우도 없다. 또한, 수의 길이는 300,000자리를 넘지 않는다.

---

## 출력
두 수의 곱을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 큰 정수를 곱하는 문제입니다. Python의 정수 타입은 임의의 정밀도를 지원하므로, 입력받은 두 수를 정수형으로 변환한 후 곱셈 연산을 수행하면 매우 큰 수에 대해서도 정확한 결과를 얻을 수 있습니다.

### 동작 과정
1. 입력으로 두 개의 정수 A와 B를 공백으로 구분하여 받습니다. 예를 들어, 입력이 `12345678901234567890 98765432109876543210`이라면 A는 `12345678901234567890`, B는 `98765432109876543210`입니다.
2. `map(int, input().split())`을 사용하여 A와 B를 각각 정수형으로 변환합니다.
3. `a * b`를 통해 A와 B의 곱을 계산합니다.
4. 계산된 결과를 출력합니다. 이 예시에서는 `1219326311370217952237463801111263526900`이 출력됩니다.

### 시간 복잡도
이 연산의 시간 복잡도는 O(n)입니다. 여기서 n은 두 숫자 중 더 긴 수의 자릿수에 비례합니다. Python의 정수 곱셈은 Karatsuba 알고리즘과 같은 효율적인 알고리즘을 사용하여 수행되므로 대규모 정수 곱셈에서도 비교적 빠르게 처리됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a * b)
```
