## 문제
두 정수 A와 B가 주어졌을 때, 두 수의 곱을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 A와 B가 주어진다. 두 정수는 0보다 크거나 같은 정수이며, 0을 제외한 정수는 0으로 시작하지 않으며, 수의 앞에 불필요한 0이 있는 경우도 없다. 또한, 수의 길이는 300,000자리를 넘지 않는다.

---

## 출력
두 수의 곱을 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 두 개의 큰 정수 A와 B의 곱을 계산하는 것이며, Python의 `int` 타입은 임의의 크기를 가질 수 있어 큰 수 연산을 안전하게 처리할 수 있다. 따라서 입력받은 두 수를 정수형으로 변환한 뒤 곱셈을 수행하고 결과를 출력하면 된다.

### 동작 과정
1. 입력으로 주어진 두 정수 A와 B를 한 줄로 받아온다.
2. `map` 함수를 사용하여 A와 B를 정수형으로 변환한다.
3. 두 수 A와 B를 곱하여 결과를 계산한다.
4. 결과를 출력한다.

예시 입력:
```
123456789012345678901234567890 987654321098765432109876543210
```

예시 동작 과정:
- A = 123456789012345678901234567890
- B = 987654321098765432109876543210
- 계산: `A * B` = 121932631137021795226185032733622923332236095766640000000000000000000000000000000000000000000000000000
- 출력 결과: 121932631137021795226185032733622923332236095766640000000000000000000000000000000000000000000000000000

### 시간 복잡도
이 문제는 두 수의 곱을 계산하는 것이므로, 곱셈 연산의 시간 복잡도는 O(n * m)이다. 여기서 n과 m은 각각 A와 B의 자리수를 나타낸다. 하지만 Python의 `int` 타입은 내부적으로 최적화된 알고리즘을 사용하므로 실제 수행 시간은 매우 빠르다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a * b)
```
