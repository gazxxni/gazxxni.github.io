## 풀이

### 풀이 핵심 로직
이 문제는 삼각형 별 패턴을 재귀적으로 생성하는 방식으로 해결합니다. 기본 패턴(3개의 행과 5개의 열)에서 시작하여, 각 단계에서 이전 패턴을 두 배로 확장하고, 이를 상하로 결합하여 더 큰 패턴을 만듭니다. 이때 중앙 정렬을 위해 공백을 추가하고, 하단은 이전 패턴을 좌우로 붙여 새로운 패턴을 형성합니다.

### 동작 과정
1. **입력**: 사용자가 원하는 크기 `n`(3의 거듭제곱)을 입력합니다. 예를 들어, `n = 12`라고 가정합니다.
2. **재귀 호출**: `draw_star(12)`가 호출되면, 내부적으로 `draw_star(6)`이 호출되고, 다시 `draw_star(3)`이 호출됩니다.
   - `draw_star(3)`은 기본 패턴인 `["  *  ", " * * ", "*****"]`를 반환합니다.
3. **위쪽 패턴 생성**: `draw_star(6)`에서는 `draw_star(3)`의 각 행 앞뒤에 `3`개의 공백을 추가하여 중앙 정렬된 상단 패턴을 생성합니다.
   - 결과: 
     ```
     ["    *    ", "   * *   ", "  *****  "]
     ```
4. **아래쪽 패턴 생성**: 하단은 `draw_star(3)`의 각 행을 좌우로 붙여서 생성합니다.
   - 결과:
     ```
     ["***** *****", " * * * * *", "  *   *   "]
     ```
5. **패턴 결합**: 위쪽과 아래쪽 패턴을 결합하여 최종 패턴을 완성합니다.
6. **출력**: 최종적으로 완성된 별 패턴을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 각 단계에서 n이 2로 나누어지고, n이 3의 거듭제곱이기 때문에, 전체적으로 log(n) 단계에서 O(n) 작업을 수행하게 됩니다. 즉, 최종적으로 O(n) 시간 복잡도를 가지게 됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

def draw_star(n):
    if n == 3:
        return ["  *  ", " * * ", "*****"]
    
    # 재귀 호출: n/2 크기의 별 패턴을 가져옵니다.
    prev = draw_star(n // 2)
    
    # 윗부분: 이전 패턴의 각 행 앞뒤에 n/2 개의 공백을 추가하여 중앙 정렬합니다.
    top = [(" " * (n // 2)) + line + (" " * (n // 2)) for line in prev]
    
    # 아랫부분: 이전 패턴의 각 행을 좌우로 붙입니다.
    bottom = [line + " " + line for line in prev]
    
    # 윗부분과 아랫부분을 합쳐서 현재 크기의 별 패턴을 완성합니다.
    return top + bottom

result = draw_star(n)
for line in result:
    print(line)
```
