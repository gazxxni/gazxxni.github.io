## 문제
정수를 저장하는 스택을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 다섯 가지이다.


	1 X: 정수 X를 스택에 넣는다. (1 ≤ X ≤ 100,000)
	2: 스택에 정수가 있다면 맨 위의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	3: 스택에 들어있는 정수의 개수를 출력한다.
	4: 스택이 비어있으면 1, 아니면 0을 출력한다.
	5: 스택에 정수가 있다면 맨 위의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 스택 자료구조를 구현하고, 주어진 명령어를 처리하는 방식으로 해결할 수 있다. 스택은 LIFO(Last In, First Out) 구조이므로, 가장 최근에 추가된 요소를 먼저 제거하거나 조회하는 특성을 활용하여 명령을 처리한다.

### 동작 과정
입력이 주어지면, 각 명령어를 순차적으로 처리한다. 예를 들어, 다음과 같은 입력이 주어졌다고 가정해 보자:

```
7
1 10
1 20
3
4
2
5
2
```

1. 첫 번째 명령 `1 10`: 스택에 10을 추가한다. 스택 상태: [10]
2. 두 번째 명령 `1 20`: 스택에 20을 추가한다. 스택 상태: [10, 20]
3. 세 번째 명령 `3`: 스택에 들어있는 정수의 개수를 출력한다. 출력: `2`
4. 네 번째 명령 `4`: 스택이 비어있지 않으므로 0을 출력한다. 출력: `0`
5. 다섯 번째 명령 `2`: 스택에서 맨 위의 정수를 제거하고 출력한다. 20이 제거되므로 출력: `20`. 스택 상태: [10]
6. 여섯 번째 명령 `5`: 스택의 맨 위 정수를 출력한다. 출력: `10`
7. 일곱 번째 명령 `2`: 스택에서 맨 위의 정수를 제거하고 출력한다. 10이 제거되므로 출력: `10`. 스택 상태: []

### 시간 복잡도
모든 명령어는 O(1) 시간 복잡도로 처리할 수 있으므로, 전체 명령어 N개를 처리하는 데 걸리는 시간 복잡도는 O(N)이다.

## 코드
```python
import sys

n = int(sys.stdin.readline())
stack = []

for _ in range(n):
    command = sys.stdin.readline().split()
    
    a = int(command[0])
    
    if a == 1:
        b = int(command[1])
        stack.append(b)
        
    elif a == 2:
        if stack:
            print(stack.pop())
        else:
            print(-1)
            
    elif a == 3:
        print(len(stack))
        
    elif a == 4:
        if not stack:
            print(1)
        else:
            print(0)
            
    elif a == 5:
        if stack:
            print(stack[-1])
        else:
            print(-1)
```
