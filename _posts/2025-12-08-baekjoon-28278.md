## 문제
정수를 저장하는 스택을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 다섯 가지이다.


	1 X: 정수 X를 스택에 넣는다. (1 ≤ X ≤ 100,000)
	2: 스택에 정수가 있다면 맨 위의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	3: 스택에 들어있는 정수의 개수를 출력한다.
	4: 스택이 비어있으면 1, 아니면 0을 출력한다.
	5: 스택에 정수가 있다면 맨 위의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 스택 자료 구조를 구현하고, 주어진 명령에 따라 스택의 상태를 변경하거나 정보를 출력하는 것입니다. 스택의 기본 연산인 푸시(push), 팝(pop), 크기 확인(size), 비어있는지 여부 확인(is_empty), 최상위 요소 확인(peek)을 수행하며 효율적으로 처리합니다.

### 동작 과정
1. 첫째 줄에 명령의 수 \( N \)이 주어집니다.
2. 다음 \( N \)줄에는 명령이 주어집니다. 명령은 다음과 같은 형식입니다:
   - `1 X`: 정수 \( X \)를 스택에 추가합니다.
   - `2`: 스택에서 정수를 하나 제거하고 출력합니다. 스택이 비어있으면 -1을 출력합니다.
   - `3`: 스택의 크기를 출력합니다.
   - `4`: 스택이 비어있으면 1, 아니면 0을 출력합니다.
   - `5`: 스택의 최상위 요소를 출력합니다. 비어있으면 -1을 출력합니다.

**예시 입력:**
```
7
1 10
1 20
3
5
2
3
4
```

**단계별 설명:**
- **1 10**: 스택에 10 추가 → 스택: [10]
- **1 20**: 스택에 20 추가 → 스택: [10, 20]
- **3**: 스택의 크기 출력 → 2
- **5**: 스택의 최상위 요소 출력 → 20
- **2**: 스택에서 최상위 요소 제거(20) → 출력: 20 → 스택: [10]
- **3**: 스택의 크기 출력 → 1
- **4**: 스택이 비어있지 않으므로 0 출력 → 출력: 0

### 시간 복잡도
모든 명령은 \( O(1) \)의 시간 복잡도를 가집니다. 따라서 전체 명령을 처리하는 시간 복잡도는 \( O(N) \)입니다. 여기서 \( N \)은 주어진 명령의 수입니다.

## 코드
```python
import sys

n = int(sys.stdin.readline())
stack = []

for _ in range(n):
    command = sys.stdin.readline().split()
    
    a = int(command[0])
    
    if a == 1:
        b = int(command[1])
        stack.append(b)
        
    elif a == 2:
        if stack:
            print(stack.pop())
        else:
            print(-1)
            
    elif a == 3:
        print(len(stack))
        
    elif a == 4:
        if not stack:
            print(1)
        else:
            print(0)
            
    elif a == 5:
        if stack:
            print(stack[-1])
        else:
            print(-1)
```
