## 문제
네 점이 주어졌을 때, 네 점을 이용해 정사각형을 만들 수 있는지 없는지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 네 줄로 이루어져 있으며, 점의 좌표가 한 줄에 하나씩 주어진다. 점의 좌표는 -100,000보다 크거나 같고, 100,000보다 작거나 같은 정수이다. 같은 점이 두 번 이상 주어지지 않는다.

---

## 출력
각 테스트 케이스마다 입력으로 주어진 네 점을 이용해서 정사각형을 만들 수 있으면 1을, 없으면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
정사각형을 형성하기 위한 조건은 네 개의 점 사이의 거리 패턴이 특정해야 한다는 것입니다. 즉, 4개의 점에서 두 가지의 서로 다른 거리를 가지며, 이 중 하나는 변의 길이(4번 나타나야 함)이고 다른 하나는 대각선의 길이(2번 나타나야 함)입니다. 이를 통해 주어진 네 점이 정사각형을 만들 수 있는지를 판단할 수 있습니다.

### 동작 과정
1. **입력 처리**: 테스트 케이스의 개수를 입력받고, 각 테스트 케이스마다 4개의 점의 좌표를 입력받아 리스트에 저장합니다.
   
2. **거리 계산**: 두 점 사이의 유클리드 거리를 제곱하여 계산합니다. (거리 제곱을 사용하여 루트를 제거하고 정수 비교를 쉽게 합니다.)

3. **거리 빈도수 카운트**: 두 점 사이의 거리를 계산하여 사전(`defaultdict`)을 사용해 각 거리의 빈도수를 기록합니다. (예: 같은 거리에서 점을 선택할 때 발생하는 조합 수)

4. **정사각형 조건 체크**: 사전에서 거리와 그 빈도수를 확인하여,
   - 변의 길이가 4번 나타나고,
   - 대각선의 길이가 2번 나타나면 정사각형이 형성될 수 있다고 판단하여 1을 출력합니다.
   - 그렇지 않으면 0을 출력합니다.

예시 입력으로 다음과 같은 점들이 주어졌다고 가정합니다:
```
0 0
0 1
1 0
1 1
```
이 경우, 각 점 사이의 거리 제곱은 1 (변의 길이), 2 (대각선의 길이)로 나타납니다. 이 패턴을 통해 변이 4번, 대각선이 2번 나타나므로 정사각형을 만들 수 있음을 확인합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(T * N^2)입니다. 여기서 T는 테스트 케이스의 수, N은 각 테스트 케이스에서 점의 수(4)에 해당합니다. 따라서 N은 상수이므로 실제 시간 복잡도는 O(T)로 간주할 수 있습니다.

## 코드
```python
from collections import defaultdict
t = int(input())

for _ in range(t):
    arr = []
    
    for i in range(4):
        x, y = map(int, input().split())
        arr.append((x, y))
    
    dic = defaultdict(int)
    for i in range(4):
        for j in range(4):
            if i != j:
                a = (arr[i][0] - arr[j][0]) ** 2 + (arr[i][1] - arr[j][1]) ** 2
                dic[a] += 1
    
    aa = False
    bb = False  
    for k, v in dic.items():
        if v == 8:
            aa = True
        
        elif v == 4:
            bb = True
            
    if aa and bb:
        print(1)
        
    else:
        print(0)
```
