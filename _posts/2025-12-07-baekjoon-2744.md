## 문제
영어 소문자와 대문자로 이루어진 단어를 입력받은 뒤, 대문자는 소문자로, 소문자는 대문자로 바꾸어 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 영어 소문자와 대문자로만 이루어진 단어가 주어진다. 단어의 길이는 최대 100이다.

---

## 출력
첫째 줄에 입력으로 주어진 단어에서 대문자는 소문자로, 소문자는 대문자로 바꾼 단어를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 88 ms

## 풀이

### 풀이 핵심 로직
주어진 문자열의 각 문자를 확인하여 대문자인 경우 소문자로, 소문자인 경우 대문자로 변환하는 방식으로 해결합니다. Python의 문자열 메서드인 `isupper()`, `lower()`, `isupper()`를 사용하여 문자의 대소문자를 판별하고 변환합니다.

### 동작 과정
1. 입력으로 "HelloWorld"라는 문자열을 받습니다.
2. 첫 번째 문자 'H'는 대문자이므로 소문자 'h'로 변환해 출력합니다.
3. 두 번째 문자 'e'는 소문자이므로 대문자 'E'로 변환해 출력합니다.
4. 같은 방식으로 문자열의 모든 문자를 처리하여 최종적으로 "hELLOwORLD"라는 결과를 출력합니다.

### 시간 복잡도
O(n) - n은 입력 문자열의 길이입니다. 각 문자를 한 번씩 확인하고 변환하기 때문입니다.

---

## 코드
```python
for i in input():
    if i.isupper():
        print(i.lower(), end='')
    else:
        print(i.upper(), end='')
```
