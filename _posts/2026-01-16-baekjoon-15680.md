## 문제
연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.

이를 출력하는 프로그램을 작성해보도록 하자.

---

## 입력
첫째 줄에 N이 주어진다. (N = 0 또는 1)

---

## 출력
N = 0일 경우: 연세대학교의 영문명을 출력한다.
	N = 1일 경우: 연세대학교의 슬로건을 출력한다.


대소문자 구별에 주의하도록 하자.

---

## 풀이

### 풀이 핵심 로직
문제는 입력 값 N에 따라 두 가지 문자열 중 하나를 출력하는 간단한 조건문을 사용하는 것이다. N이 0일 경우 "YONSEI"를 출력하고, N이 1일 경우 "Leading the Way to the Future"를 출력하도록 구성한다.

### 동작 과정
1. 사용자로부터 입력을 받는다. 예를 들어, 사용자가 0을 입력한다고 가정하자.
2. `if` 문을 사용하여 N이 0인지 1인지 확인한다.
   - N이 0일 경우, "YONSEI"를 출력한다.
   - N이 1일 경우, "Leading the Way to the Future"를 출력한다.
3. 만약 0을 입력했다면, 프로그램은 "YONSEI"를 출력하게 된다.

예시 입력 및 출력:
- 입력: `0`
  - 출력: `YONSEI`
- 입력: `1`
  - 출력: `Leading the Way to the Future`

### 시간 복잡도
이 문제의 시간 복잡도는 O(1)이다. 입력 값에 따라 조건을 검사하고 해당 문자열을 출력하는 작업은 상수 시간에 수행된다.

---

## 코드
```python
n = int(input())

if n == 0:
    print("YONSEI")
else:
    print("Leading the Way to the Future")
```
