## 풀이

### 풀이 핵심 로직
이 문제는 n개의 방에서 스위치를 켜고 끄는 작업을 통해 최종적으로 켜져 있는 방의 개수를 구하는 문제입니다. 방의 스위치는 자신의 번호와 같은 배수로만 작동하며, 결과적으로 마지막에 켜져 있는 방은 제곱수 방들로 한정됩니다. 따라서, 최종적으로 켜져 있는 방의 수는 1부터 n까지의 제곱수의 개수, 즉 n의 제곱근을 구하면 됩니다.

### 동작 과정
1. 입력으로 n이 주어집니다. 예를 들어, n = 10이라고 가정합시다.
2. 각 방의 스위치는 그 방 번호의 약수의 개수에 따라 상태가 결정됩니다. 홀수 개의 약수를 가지는 방, 즉 제곱수의 방만 최종적으로 켜집니다.
3. 10 이하의 제곱수는 1, 4, 9로 총 3개입니다.
4. 따라서, n = 10일 때 최종적으로 켜져 있는 방의 개수는 `int(10 ** 0.5)`로 계산하여 3이 됩니다.

### 시간 복잡도
O(1) - 입력 n에 대해 상수 시간에 결과를 반환하기 때문에 시간 복잡도는 O(1)입니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
print(int(n ** 0.5))
```
