## 문제
0과 1로만 이루어진 수를 이진수라 한다. 이러한 이진수 중 특별한 성질을 갖는 것들이 있는데, 이들을 이친수(pinary number)라 한다. 이친수는 다음의 성질을 만족한다.


	이친수는 0으로 시작하지 않는다.
	이친수에서는 1이 두 번 연속으로 나타나지 않는다. 즉, 11을 부분 문자열로 갖지 않는다.


예를 들면 1, 10, 100, 101, 1000, 1001 등이 이친수가 된다. 하지만 0010101이나 101101은 각각 1, 2번 규칙에 위배되므로 이친수가 아니다.

N(1 ≤ N ≤ 90)이 주어졌을 때, N자리 이친수의 개수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N이 주어진다.

---

## 출력
첫째 줄에 N자리 이친수의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
이친수는 0으로 시작하지 않으며, 두 개의 1이 연속으로 나타날 수 없습니다. 이러한 규칙으로 인해 N자리 이친수를 세는 방법은 N-1자리 이친수와 N-2자리 이친수를 조합하여 구할 수 있습니다. 이친수의 개수는 다음과 같은 재귀 관계로 정의됩니다: `dp[n] = dp[n-1] + dp[n-2]`. 여기서 `dp[n-1]`은 N자리에서 0으로 끝나는 이친수의 개수를, `dp[n-2]`는 1로 끝나는 이친수의 개수를 의미합니다.

### 동작 과정
1. **입력:** N이 입력된다. 예를 들어, N = 5.
2. **초기화:** `dp[1]`을 1로 설정하여 1자리 이친수(1) 개수를 초기화한다.
3. **재귀 관계 적용:** 
   - N = 2일 때, `dp[2] = dp[1] + dp[0]` (dp[0]은 0으로 간주) → `dp[2] = 1 + 0 = 1`.
   - N = 3일 때, `dp[3] = dp[2] + dp[1]` → `dp[3] = 1 + 1 = 2`.
   - N = 4일 때, `dp[4] = dp[3] + dp[2]` → `dp[4] = 2 + 1 = 3`.
   - N = 5일 때, `dp[5] = dp[4] + dp[3]` → `dp[5] = 3 + 2 = 5`.
4. **출력:** 최종적으로 `dp[5]`의 값을 출력하면 5자리 이친수의 개수인 5가 출력된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 이는 N까지의 값을 계산하기 위해 단순히 한 번의 반복문을 사용하기 때문입니다.

---

## 코드
```python
n = int(input())

dp = [0] * (n + 1)
dp[1] = 1


for i in range(2, n+1):
    dp[i] = dp[i-2] + dp[i-1]

print(dp[n])
```
