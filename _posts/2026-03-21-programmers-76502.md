## 문제
다음 규칙을 지키는 문자열을 올바른 괄호 문자열이라고 정의합니다.


(), [], {} 는 모두 올바른 괄호 문자열입니다.
만약 A가 올바른 괄호 문자열이라면, (A), [A], {A} 도 올바른 괄호 문자열입니다. 예를 들어, [] 가 올바른 괄호 문자열이므로, ([]) 도 올바른 괄호 문자열입니다.
만약 A, B가 올바른 괄호 문자열이라면, AB 도 올바른 괄호 문자열입니다. 예를 들어, {} 와 ([]) 가 올바른 괄호 문자열이므로, {}([]) 도 올바른 괄호 문자열입니다.


대괄호, 중괄호, 그리고 소괄호로 이루어진 문자열 s가 매개변수로 주어집니다. 이 s를 왼쪽으로 x (0 ≤ x &lt; (s의 길이)) 칸만큼 회전시켰을 때 s가 올바른 괄호 문자열이 되게 하는 x의 개수를 return 하도록 solution 함수를 완성해주세요.

## 풀이
### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 문자열을 회전시켜가며 올바른 괄호 문자열인지 확인하는 것입니다. 문자열을 회전시키면 원래의 괄호 구성은 달라지므로, 각 회전마다 괄호의 짝이 맞는지를 확인하기 위해 스택을 사용합니다. 올바른 괄호 문자열일 경우 스택이 비어 있어야 하며, 이를 통해 유효한 회전의 개수를 세어 출력합니다.

### 동작 과정
1. 주어진 문자열 `s`를 두 번 이어붙여서 `new`라는 문자열을 생성합니다. 이는 회전된 문자열을 쉽게 접근하기 위함입니다.
2. 문자열의 길이 `n`을 계산합니다.
3. 문자열의 각 인덱스 `i`에 대해, `new[i:i+n]` 부분 문자열을 순회하면서 올바른 괄호 문자열인지 판단합니다.
4. 스택을 사용하여 괄호의 짝을 맞춥니다. 스택의 최상단 괄호와 현재 문자가 짝을 이루는 경우 pop하여 제거하고, 그렇지 않으면 현재 문자를 스택에 추가합니다.
5. 모든 문자를 확인한 후 스택이 비어 있으면 올바른 괄호 문자열이라는 의미이므로, 카운트를 증가시킵니다.
6. 모든 회전을 확인한 후 유효한 회전의 개수를 반환합니다.

**예시 입력 설명**
- 입력: `s = "[](){}"`
  - 회전 0: `[](){}` (O)
  - 회전 1: `](){}[` (X)
  - 회전 2: `(){}[]` (O)
  - 회전 3: `){}[](` (X)
  - 회전 4: `{}`[]() (O)
  - 회전 5: `}[](){` (X)
  - 결과: 3개의 올바른 괄호 문자열이 있음.
  
- 입력: `s = "}]()[{"`
  - 회전 0: `}]()[{` (X)
  - 회전 1: `]()[{}` (X)
  - 회전 2: `()[{}` (O)
  - 회전 3: `)[{}` (X)
  - 회전 4: `[{}]()` (O)
  - 회전 5: `{}`[()] (X)
  - 결과: 2개의 올바른 괄호 문자열이 있음.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 문자열의 길이가 n일 때, n개의 회전을 각각 n번의 문자열 탐색으로 확인하므로 O(n * n) = O(n^2)입니다.

## 코드
```python
def solution(s):
    answer = 0
    new = s * 2
    n = len(s)
    for i in range(n):
        stack = [new[i]]
        for word in new[i+1:i+n]:
            if stack:
                if stack[-1] == '(' and word == ')':
                    stack.pop(-1)
                elif stack[-1] == '[' and word == ']':
                    stack.pop(-1)
                elif stack[-1] == '{' and word == '}':
                    stack.pop(-1)
                else:
                    stack.append(word)
                    
            else:
                if word in ('{', '[', '('):
                    stack.append(word)
                
            
        
        if not stack:
            answer += 1
            
    
        
    return answer
```
