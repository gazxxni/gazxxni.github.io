## 문제
셀프 넘버는 1949년 인도 수학자 D.R. Kaprekar가 이름 붙였다. 양의 정수 n에 대해서 d(n)을 n과 n의 각 자리수를 더하는 함수라고 정의하자. 예를 들어, d(75) = 75+7+5 = 87이다.

양의 정수 n이 주어졌을 때, 이 수를 시작해서 n, d(n), d(d(n)), d(d(d(n))), ...과 같은 무한 수열을 만들 수 있다. 

예를 들어, 33으로 시작한다면 다음 수는 33 + 3 + 3 = 39이고, 그 다음 수는 39 + 3 + 9 = 51, 다음 수는 51 + 5 + 1 = 57이다. 이런식으로 다음과 같은 수열을 만들 수 있다.

33, 39, 51, 57, 69, 84, 96, 111, 114, 120, 123, 129, 141, ...

n을 d(n)의 생성자라고 한다. 위의 수열에서 33은 39의 생성자이고, 39는 51의 생성자, 51은 57의 생성자이다. 생성자가 한 개보다 많은 경우도 있다. 예를 들어, 101은 생성자가 2개(91과 100) 있다. 

생성자가 없는 숫자를 셀프 넘버라고 한다. 100보다 작은 셀프 넘버는 총 13개가 있다. 1, 3, 5, 7, 9, 20, 31, 42, 53, 64, 75, 86, 97

10000보다 작거나 같은 셀프 넘버를 한 줄에 하나씩 출력하는 프로그램을 작성하시오.

---

## 입력
입력은 없다.

10,000보다 작거나 같은 셀프 넘버를 한 줄에 하나씩 증가하는 순서로 출력한다.

## 풀이

### 풀이 핵심 로직
셀프 넘버를 찾기 위해, 먼저 1부터 10,000까지의 모든 숫자에 대해 생성자를 찾아 해당 숫자가 생성자인지를 판단합니다. 이를 위해, 각 숫자에 대해 d(n) 값을 계산하여, 해당 값이 1부터 10,000 사이의 숫자인 경우에 생성자로 표시합니다. 마지막으로, 생성자가 아닌 숫자들(셀프 넘버)을 출력합니다.

### 동작 과정
1. **초기화**: 길이가 10001인 배열 `arr`을 생성하고, 0번 인덱스는 True로, 나머지는 False로 초기화합니다. (arr[i]가 True면 i는 생성자가 있다는 의미)
   
2. **생성자 찾기**: 1부터 10,000까지의 각 숫자에 대해 반복합니다.
   - 현재 숫자를 `number`라고 할 때, `d(number)`를 계산합니다.
   - `d(number)`는 `number`와 `number`의 각 자리수의 합을 더한 값입니다. 이를 반복하여 계산합니다.
   - 이 과정을 통해 계산된 `d(number)`가 10,000 이하이면, `arr[d(number)]`를 True로 설정합니다. (즉, `number`가 `d(number)`의 생성자임을 기록)

3. **셀프 넘버 출력**: 모든 숫자에 대해, `arr` 배열이 False인 인덱스(셀프 넘버)를 출력합니다.

예시:
- 숫자 33을 시작으로 d(33) = 39, d(39) = 51, d(51) = 57과 같이 생성자를 찾고, 이 과정에서 39, 51, 57의 생성자 배열을 갱신합니다.
- 최종적으로 arr 배열에서 False인 인덱스를 출력하여 셀프 넘버를 나열합니다.

### 시간 복잡도
O(N * M) (N은 10,000, M은 각 숫자의 자리수의 수, 즉 최대 4)  
여기서, N은 1부터 10,000까지의 숫자 개수, M은 각 숫자의 자리 수(최대 4자리 수)로 각 숫자에 대해 최대 4번의 연산이 필요합니다.

## 코드
```python
arr = [True] + [False] * 10000

for number in range(1, 10001):
    while True:
        str_num = str(number)
        new_num = number
        for i in range(len(str_num)):
            new_num += int(str_num[i])

        number = new_num
        
        if number > 10000:
            break
        
        arr[number] = True

for i in range(10001):
    if not arr[i]:
        print(i)
```
