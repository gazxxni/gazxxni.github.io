## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
상의와 하의의 색상이 서로 다르다는 조건에서, N개의 상의 중에서 하나를 선택하고, 선택한 상의와 색상이 다른 N-1개의 하의 중에서 하나를 선택하는 조합을 계산합니다. 따라서, 가능한 조합의 수는 N (상의 개수) 곱하기 (N-1) (상의와 색상이 다른 하의 개수)가 됩니다.

### 동작 과정
예를 들어, N이 3인 경우를 살펴보겠습니다. 상의와 하의의 색상은 다음과 같습니다:
- 상의: 1, 2, 3
- 하의: 1, 2, 3

각 상의에 대해 하의를 선택할 수 있는 경우는 다음과 같습니다:
1. 상의 1을 선택할 때: 하의는 2 또는 3 (2가지)
2. 상의 2를 선택할 때: 하의는 1 또는 3 (2가지)
3. 상의 3을 선택할 때: 하의는 1 또는 2 (2가지)

따라서, 총 조합의 수는 2 + 2 + 2 = 6입니다. 이 결과는 3 * (3 - 1) = 3 * 2 = 6으로도 확인할 수 있습니다.

### 시간 복잡도
O(1) - 입력 N에 관계없이 상수 시간 내에 결과를 계산할 수 있습니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
