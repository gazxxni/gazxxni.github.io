## 문제
때는 2040년, 이민혁은 우주에 자신만의 왕국을 만들었다. 왕국은 N개의 행성으로 이루어져 있다. 민혁이는 이 행성을 효율적으로 지배하기 위해서 행성을 연결하는 터널을 만들려고 한다.


	행성은 3차원 좌표위의 한 점으로 생각하면 된다. 두 행성 A(xA, yA, zA)와 B(xB, yB, zB)를 터널로 연결할 때 드는 비용은 min(|xA-xB|, |yA-yB|, |zA-zB|)이다.


	민혁이는 터널을 총 N-1개 건설해서 모든 행성이 서로 연결되게 하려고 한다. 이때, 모든 행성을 터널로 연결하는데 필요한 최소 비용을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 행성의 개수 N이 주어진다. (1 ≤ N ≤ 100,000) 다음 N개 줄에는 각 행성의 x, y, z좌표가 주어진다. 좌표는 -109보다 크거나 같고, 109보다 작거나 같은 정수이다. 한 위치에 행성이 두 개 이상 있는 경우는 없다.

---

## 출력
첫째 줄에 모든 행성을 터널로 연결하는데 필요한 최소 비용을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 최소 스패닝 트리를 구성하는 문제로, 행성을 연결하는 터널의 비용을 최소화해야 합니다. 두 행성을 연결할 때의 비용은 세 좌표 중 두 점의 차의 최소값으로 정의되어 있기 때문에, 각 좌표별로 행성을 정렬한 후 인접한 행성들 간의 비용을 계산하여 간선을 생성합니다. 이후 크루스칼 알고리즘을 사용하여 최소 스패닝 트리를 구합니다.

### 동작 과정
1. **입력 처리**: 행성의 개수 N을 입력받고, 각 행성의 x, y, z 좌표를 리스트에 저장합니다.
   
2. **좌표별 정렬**: x, y, z 좌표를 기준으로 각각 정렬합니다. 정렬된 리스트를 통해 인접한 행성들 간의 비용을 계산합니다.
   - 예를 들어, 행성이 (1, 2, 3), (4, 5, 6), (7, 8, 9)로 주어졌다면, x좌표로 정렬했을 때 (1, 2, 3) -> (4, 5, 6) 간의 비용은 `min(|1-4|, |2-5|, |3-6|) = 1`입니다.

3. **간선 생성**: 각 정렬된 리스트에서 인접한 두 점끼리 간선을 생성하고 그 비용을 기록합니다. 이렇게 해서 총 3 * (N-1)개의 간선을 생성합니다.

4. **간선 정렬**: 생성된 간선을 비용 기준으로 오름차순 정렬합니다.

5. **크루스칼 알고리즘 실행**: 
   - 부모 노드를 초기화하고, 정렬된 간선을 하나씩 확인하여 사이클을 형성하지 않을 경우에만 간선을 추가합니다.
   - 추가한 간선의 비용을 누적하여 최소 비용을 계산합니다.

6. **결과 출력**: 최종적으로 모든 행성을 연결하는 데 필요한 최소 비용을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)입니다. 각 좌표에 대해 정렬을 수행하는 데 O(N log N)의 시간이 소요되며, 간선 생성 및 크루스칼 알고리즘에서의 연산도 O(N log N)으로 이루어지기 때문에 전체 복잡도는 O(N log N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

def find(parent, x):
    if parent[x] != x:
        parent[x] = find(parent, parent[x])
    return parent[x]

def union(parent, a, b):
    a = find(parent, a)
    b = find(parent, b)
    
    if a < b:
        parent[b] = a
    else:
        parent[a] = b
        
xx, yy, zz = [], [], []
for i in range(n):
    x, y, z = map(int, input().split())
    xx.append((x, i))
    yy.append((y, i))
    zz.append((z, i))
    
xx.sort()
yy.sort()
zz.sort()

edge = []
for cur in xx, yy, zz:
    for i in range(1, n):
        w1, a = cur[i - 1]
        w2, b = cur[i]
        edge.append((abs(w1 - w2), a, b))
        
edge.sort()

parent = [i for i in range(n)]

def krustal():
    ans = 0
    
    for cost, a, b in edge:
        if find(parent, a) != find(parent, b):
            union(parent, a, b)
            ans += cost
    
    return ans

print(krustal())
```
