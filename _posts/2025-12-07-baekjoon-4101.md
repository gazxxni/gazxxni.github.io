## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 두 양의 정수 \( a \)와 \( b \)에 대해 \( a \)가 \( b \)보다 큰지를 판단하는 문제입니다. 입력이 여러 개의 테스트 케이스로 구성되어 있으며, 각 케이스의 결과에 따라 "Yes" 또는 "No"를 출력합니다. 입력의 마지막은 "0 0"으로 표시되며, 이 경우 프로그램을 종료합니다.

### 동작 과정
1. 사용자로부터 두 정수를 입력받습니다.
2. 입력받은 두 정수가 모두 0이면 루프를 종료합니다.
3. \( a \)가 \( b \)보다 큰 경우 "Yes"를 출력하고, 그렇지 않은 경우 "No"를 출력합니다.
4. 이 과정을 반복하여 모든 테스트 케이스를 처리합니다.

**예시 입력:**
```
3 2
1 4
0 0
```

**단계별 설명:**
- 첫 번째 입력 "3 2": \( a = 3 \), \( b = 2 \)이므로 \( 3 > 2 \)이므로 "Yes" 출력.
- 두 번째 입력 "1 4": \( a = 1 \), \( b = 4 \)이므로 \( 1 > 4 \)가 아니므로 "No" 출력.
- 마지막 입력 "0 0": 입력 종료 조건이므로 루프를 종료합니다.

### 시간 복잡도
이 알고리즘은 각 테스트 케이스에 대해 상수 시간 \( O(1) \)의 연산을 수행하므로 전체 시간 복잡도는 \( O(n) \)이며, 여기서 \( n \)은 입력된 테스트 케이스의 개수입니다.

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
