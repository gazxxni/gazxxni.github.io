## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 두 양의 정수 a와 b에 대해 a가 b보다 큰지를 판단하는 문제입니다. 입력이 여러 개의 테스트 케이스로 주어지므로, 반복문을 사용하여 각 케이스에 대해 조건을 체크합니다. 마지막에 "0 0"이 입력되면 종료합니다.

### 동작 과정
1. 입력을 받기 위해 무한 루프를 시작합니다.
2. 각 테스트 케이스마다 두 개의 정수를 입력받습니다.
3. 입력받은 값이 모두 0인 경우 루프를 종료합니다.
4. 첫 번째 수 a가 두 번째 수 b보다 큰지 비교하여 결과에 따라 "Yes" 또는 "No"를 출력합니다.

예시 입력:
```
3 2
5 5
6 7
0 0
```
- 첫 번째 줄에서 3과 2를 비교하여 3 > 2 이므로 "Yes" 출력.
- 두 번째 줄에서 5와 5를 비교하여 5 <= 5 이므로 "No" 출력.
- 세 번째 줄에서 6과 7을 비교하여 6 < 7 이므로 "No" 출력.
- 마지막으로 "0 0" 입력으로 루프 종료.

### 시간 복잡도
각 테스트 케이스에 대해 상수 시간 O(1) 내에 비교 및 출력이 이루어지므로 전체 시간 복잡도는 O(n)입니다. 여기서 n은 입력된 테스트 케이스의 수입니다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
