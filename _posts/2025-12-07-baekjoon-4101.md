## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 두 양의 정수를 입력받아 첫 번째 수가 두 번째 수보다 큰지를 비교하는 것이다. 입력이 "0 0"일 때 프로그램을 종료하고, 그 외의 경우에는 조건에 따라 "Yes" 또는 "No"를 출력한다.

### 동작 과정
1. 프로그램은 무한 루프를 시작하여 사용자로부터 두 정수 `a`와 `b`를 입력받는다.
2. 입력받은 값이 "0 0"일 경우 루프를 종료한다.
3. 그렇지 않은 경우, `a`가 `b`보다 큰지 비교한다.
   - 만약 `a > b`이면 "Yes"를 출력하고,
   - 그렇지 않으면 "No"를 출력한다.
4. 이 과정을 반복하여 여러 테스트 케이스를 처리한다.

**예시 입력**:
```
3 2
5 5
8 10
0 0
```
**단계별 설명**:
- 첫 번째 입력 `3 2`: `3 > 2`이므로 "Yes" 출력.
- 두 번째 입력 `5 5`: `5 > 5`가 아니므로 "No" 출력.
- 세 번째 입력 `8 10`: `8 > 10`이 아니므로 "No" 출력.
- 마지막 입력 `0 0`: 루프 종료.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)이다. 각 테스트 케이스는 상수 시간 내에 처리되며, 입력의 개수에 따라 전체 실행 시간은 입력의 개수에 비례한다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
