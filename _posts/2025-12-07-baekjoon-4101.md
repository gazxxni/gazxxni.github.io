## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 풀이

### 풀이 핵심 로직
문제의 핵심은 두 개의 양의 정수 a와 b를 비교하여 a가 b보다 큰지 여부를 판단하는 것이다. 입력이 0 0이 될 때까지 반복적으로 두 수를 입력받아 비교하고, 결과에 따라 "Yes" 또는 "No"를 출력한다.

### 동작 과정
1. 프로그램이 시작되면 무한 루프를 통해 사용자로부터 두 개의 정수를 입력받는다.
2. 입력받은 두 수 a와 b가 모두 0이면 루프를 종료한다.
3. 그렇지 않으면 a와 b를 비교한다.
   - 만약 a > b이면 "Yes"를 출력하고,
   - 그렇지 않으면 "No"를 출력한다.
4. 이 과정을 반복하여 여러 테스트 케이스를 처리한다.

예시 입력:
```
1 2
2 1
3 3
0 0
```
예시 출력:
```
No
Yes
No
```
- 첫 번째 케이스에서 1은 2보다 작으므로 "No" 출력.
- 두 번째 케이스에서 2는 1보다 크므로 "Yes" 출력.
- 세 번째 케이스에서 3은 3과 같으므로 "No" 출력.
- 마지막 줄 "0 0"이 입력되면 프로그램 종료.

### 시간 복잡도
O(n) - n은 입력받은 테스트 케이스의 수. 각 테스트 케이스마다 상수 시간 내에 비교와 출력을 수행하기 때문에 전체적으로 선형 시간 복잡도를 가진다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
