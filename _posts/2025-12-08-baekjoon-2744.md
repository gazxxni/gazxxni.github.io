## 문제
영어 소문자와 대문자로 이루어진 단어를 입력받은 뒤, 대문자는 소문자로, 소문자는 대문자로 바꾸어 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 영어 소문자와 대문자로만 이루어진 단어가 주어진다. 단어의 길이는 최대 100이다.

---

## 출력
첫째 줄에 입력으로 주어진 단어에서 대문자는 소문자로, 소문자는 대문자로 바꾼 단어를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 88 ms

## 풀이

### 풀이 핵심 로직
주어진 문자열에서 각 문자를 하나씩 확인하여, 대문자는 소문자로, 소문자는 대문자로 변환하는 방식으로 문제를 해결한다. Python의 문자열 메서드 `isupper()`와 `islower()`를 활용하여 각 문자의 대소문자를 판별하고, 각각의 변환 메서드인 `lower()`와 `upper()`를 사용하여 변환된 문자를 출력한다.

### 동작 과정
1. 예시 입력: `HelloWorld`
2. 첫 번째 문자 'H'는 대문자이므로, `lower()` 메서드를 호출하여 'h'로 변환하고 출력한다.
3. 두 번째 문자 'e'는 소문자이므로, `upper()` 메서드를 호출하여 'E'로 변환하고 출력한다.
4. 나머지 문자들에 대해서도 같은 과정을 반복한다:
   - 'l' -> 'L'
   - 'l' -> 'L'
   - 'o' -> 'O'
   - 'W' -> 'w'
   - 'o' -> 'O'
   - 'r' -> 'R'
   - 'l' -> 'L'
   - 'd' -> 'D'
5. 최종적으로 출력되는 문자열은 `hELLOwORLD`가 된다.

### 시간 복잡도
O(n) 
- 여기서 n은 입력 문자열의 길이이다. 각 문자를 한 번씩 확인하고 변환하기 때문에 시간 복잡도는 문자열의 길이에 비례한다.

---

## 코드
```python
for i in input():
    if i.isupper():
        print(i.lower(), end='')
    else:
        print(i.upper(), end='')
```
