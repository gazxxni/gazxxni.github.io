## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    i = ⌊n / 2⌋;
    return A[i]; # 코드1
}

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘은 입력 배열 A에서 n의 절반에 해당하는 인덱스의 값을 반환하는 간단한 알고리즘입니다. 이 알고리즘에서 코드1은 단순히 배열 A의 특정 인덱스에 접근하는 단일 연산으로, 수행 횟수는 항상 1이 됩니다. 또한, 이 연산은 입력 크기 n에 관계없이 일정하므로 다항식으로 나타낼 경우 최고차항의 차수는 0입니다.

### 동작 과정
1. 입력으로 n이 주어질 때, 예를 들어 n = 5라고 가정합니다.
2. MenOfPassion 알고리즘을 실행하면 `i = ⌊5 / 2⌋`가 되어 i는 2가 됩니다.
3. 코드1은 `A[i]`를 반환하는데, 이 과정은 단 한번의 접근으로 이루어지므로 수행 횟수는 1회입니다.
4. 따라서 첫 번째 출력은 1입니다.
5. 이어서 코드1의 수행 횟수는 다항식으로 표현할 수 있으며, 최고차항의 차수는 0입니다. 따라서 두 번째 출력은 0이 됩니다.

### 시간 복잡도
O(1) - 알고리즘의 수행 시간이 입력의 크기 n에 상관없이 일정하므로 상수 시간 복잡도를 가집니다.
