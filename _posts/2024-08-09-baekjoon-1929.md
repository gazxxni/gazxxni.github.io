## 문제 링크
[https://www.acmicpc.net/problem/1929](https://www.acmicpc.net/problem/1929)

---

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 범위 [m, n] 내의 모든 소수를 찾는 것이다. 각 수 i에 대해 2부터 i의 제곱근까지의 수로 나누어 떨어지는지 확인하여 소수를 판별한다. 만약 나누어 떨어지지 않으면 i는 소수이다.

### 동작 과정
1. **입력 받기**: 사용자가 두 정수 m과 n을 입력한다. 예를 들어, `m=10`, `n=30`.
2. **범위 반복**: `for i in range(m, n+1)`를 사용하여 m부터 n까지의 모든 수를 반복한다.
   - i가 1인 경우는 소수가 아니므로 `continue`로 넘어간다.
3. **소수 판별**:
   - `for j in range(2, int(i**0.5)+1)`를 통해 2부터 i의 제곱근까지의 수 j에 대해 i를 나눈다.
   - 만약 i를 j로 나누어 떨어지는 경우가 발견되면 `break`로 종료하고, i는 소수가 아님.
   - 모든 j에 대해 나누어 떨어지지 않으면 `else` 블록이 실행되어 i를 출력한다.
4. **결과 출력**: 소수인 i를 출력한다. 위의 예시에서는 11, 13, 17, 19, 23, 29가 출력된다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n√n)이다. n은 주어진 범위의 크기(m과 n의 차이)이며, 각 수 i에 대해 제곱근까지의 수로 나누는 과정을 수행하기 때문에 √n의 반복이 필요하다.

---

## 코드
```python
import sys

m,n=map(int,input().split())

for i in range(m,n+1):
    if i==1:
        continue
    for j in range(2,int(i**0.5)+1):
        if i%j==0:
            break
    else:
        print(i)

        
```
