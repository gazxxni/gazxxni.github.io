## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이를 입력받고, 이를 정렬한 후, 가장 긴 변이 나머지 두 변의 합보다 크거나 같은지 확인하여 삼각형을 형성할 수 있는지를 판단합니다. 이후 세 변의 길이에 따라 삼각형의 종류를 출력합니다.

### 동작 과정
1. 입력으로 세 변의 길이 \(a\), \(b\), \(c\)를 받습니다.
2. 입력이 "0 0 0"이면 프로그램을 종료합니다.
3. 세 변의 길이를 오름차순으로 정렬합니다. 이때 \(a \leq b \leq c\)가 되도록 합니다.
4. 삼각형의 조건을 확인합니다:
   - 가장 긴 변 \(c\)가 나머지 두 변 \(a\)와 \(b\)의 합보다 크거나 같다면 "Invalid"를 출력합니다.
5. 삼각형의 종류를 확인합니다:
   - 모든 변의 길이가 같으면 "Equilateral"을 출력합니다.
   - 두 변의 길이만 같으면 "Isosceles"을 출력합니다.
   - 세 변의 길이가 모두 다르면 "Scalene"을 출력합니다.

예시 입력:
```
6 3 2
5 5 5
7 8 9
3 3 5
0 0 0
```
출력:
```
Invalid
Equilateral
Scalene
Isosceles
```

- 첫 번째 입력 `6, 3, 2`는 삼각형을 형성할 수 없으므로 "Invalid".
- 두 번째 입력 `5, 5, 5`는 모든 변이 같아서 "Equilateral".
- 세 번째 입력 `7, 8, 9`는 모두 다른 길이로 "Scalene".
- 네 번째 입력 `3, 3, 5`는 두 변이 같아서 "Isosceles".

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \(O(1)\)입니다. 각 입력에 대해 정렬은 상수 시간 내에 수행되며, 조건 체크 및 출력도 상수 시간이 소요되기 때문입니다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
