## 풀이

### 풀이 핵심 로직
문제는 주어진 문자열의 해시 값을 계산하는 것입니다. 해시 값은 각 문자에 대해 특정 값을 부여하고, 이를 제곱수로 가중하여 합산하는 방식으로 계산됩니다. 주어진 문자열의 각 문자는 'a'부터 'z'까지 1부터 26까지의 값을 가지며, 이를 특정 진수(여기서는 31)로 가중하여 최종 해시 값을 구합니다.

### 동작 과정
1. 입력으로 문자열의 길이 `l`과 문자열 `a`를 받습니다.
2. 각각의 문자의 ASCII 값을 이용하여 'a'는 1, 'b'는 2, ..., 'z'는 26의 값을 갖도록 변환합니다. 이는 `ord(a[i]) - 96`을 통해 이루어집니다.
3. 각 문자의 변환된 값에 `31`의 제곱을 곱하여 누적합을 구합니다. 이 때, `i`는 해당 문자의 인덱스입니다.
4. 모든 문자의 가중합을 구한 후, `1234567891`로 나눈 나머지를 출력합니다.
   
예시 입력:
```
3
abc
```
- `l`은 3, `a`는 "abc"입니다.
- 각 문자의 변환: 
  - 'a' -> 1 (31^0 = 1) → 1 * 1 = 1
  - 'b' -> 2 (31^1 = 31) → 2 * 31 = 62
  - 'c' -> 3 (31^2 = 961) → 3 * 961 = 2883
- 누적합: 1 + 62 + 2883 = 2946
- 최종값: 2946 % 1234567891 = 2946

### 시간 복잡도
O(n), 여기서 n은 문자열의 길이입니다. 이는 문자열의 각 문자에 대해 한 번씩만 연산을 수행하기 때문입니다.

## 코드
```python
import sys

l=int(input())
m=1234567891
r=31
a=input()
answer=0
for i in range(len(a)):
   num=ord(a[i])-96
   answer+=num*(r**i)
print(answer%m)
```
