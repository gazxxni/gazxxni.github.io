## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
풍선 터뜨리기 문제는 원형 구조에서 풍선을 터뜨리는 순서를 결정하는 문제입니다. 각 풍선의 값에 따라 이동할 방향과 거리를 조정하여 다음 풍선을 결정합니다. 이를 위해 덱(deque)을 사용하여 효율적으로 원형 구조를 구현하고, 풍선 터지기를 관리합니다.

### 동작 과정
1. **입력 예시**: 
   ```
   5
   3 2 1 -3 -1
   ```
   - 풍선의 수는 5개이며, 각각의 풍선 안에는 3, 2, 1, -3, -1이 적혀있습니다.

2. **초기 상태**:
   - 풍선 배열: [(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)]
   - 여기서 (인덱스, 값)의 형태로 풍선 정보를 저장합니다.

3. **1번 풍선 터지기**:
   - 첫 번째 풍선을 터뜨리면 값은 3입니다. 
   - 다음으로 오른쪽으로 3번째 풍선(4번 풍선)을 터뜨리기 위해 `q.rotate(-(3 - 1))`를 수행합니다.
   - 현재 상태: [(1, 2), (2, 1), (3, -3), (4, -1), (0, 3)] → 터진 풍선 순서: [1]

4. **4번 풍선 터지기**:
   - 4번 풍선을 터뜨리면 값은 -1입니다.
   - 왼쪽으로 1번째 풍선(3번 풍선)을 터뜨립니다.
   - 현재 상태: [(1, 2), (2, 1), (3, -3), (0, 3)] → 터진 풍선 순서: [1, 4]

5. **3번 풍선 터지기**:
   - 3번 풍선을 터뜨리면 값은 -3입니다.
   - 왼쪽으로 3번째 풍선(1번 풍선)을 터뜨립니다.
   - 현재 상태: [(1, 2), (0, 3)] → 터진 풍선 순서: [1, 4, 3]

6. **2번 풍선 터지기**:
   - 2번 풍선을 터뜨리면 값은 2입니다.
   - 오른쪽으로 2번째 풍선(0번 풍선)을 터뜨립니다.
   - 현재 상태: [(0, 3)] → 터진 풍선 순서: [1, 4, 3, 2]

7. **0번 풍선 터지기**:
   - 마지막으로 0번 풍선을 터뜨리면 값은 3입니다.
   - 오른쪽으로 3번째 풍선(터진 풍선 없음) → 현재 상태: [] → 터진 풍선 순서: [1, 4, 3, 2, 5]

8. **최종 결과**: 
   ```
   1 4 3 2 5
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 풍선을 한 번씩 터뜨리고, 각 터뜨림에 대해 덱의 회전 연산이 포함되지만, 전체적으로 N개의 풍선에 대한 작업을 수행하므로 선형 시간 복잡도를 가집니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
