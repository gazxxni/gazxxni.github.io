## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제는 원형으로 배치된 풍선을 터뜨리는 문제로, 각 풍선은 터뜨릴 때마다 그 안에 적힌 숫자만큼 이동하여 다음 풍선을 터뜨립니다. 데크(Deque)를 사용하여 풍선의 상태를 관리하고, 이미 터진 풍선은 deque에서 제거하는 방식으로 문제를 해결합니다.

### 동작 과정
1. **입력 예시**: N = 5 (풍선 개수), 풍선 안의 숫자 = [3, 2, 1, -3, -1]
2. **초기 상태**: Deque에 (번호, 종이 값) 형태로 풍선 정보를 저장합니다.
   - q = [(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)]
3. **1번 풍선 터뜨리기**: 1번 풍선(0번 인덱스)을 터뜨리면, 3이 적혀 있어 오른쪽으로 3칸 이동 (1, 2, 3번 풍선).
   - 터진 풍선: 1
   - 이동 후 q = [(1, 2), (2, 1), (3, -3), (4, -1)]
4. **2번 풍선 터뜨리기**: 2번 풍선(0번 인덱스)을 터뜨리면, 2가 적혀 있어 오른쪽으로 2칸 이동 (3, 4번 풍선).
   - 터진 풍선: 1, 2
   - 이동 후 q = [(2, 1), (3, -3), (4, -1)]
5. **3번 풍선 터뜨리기**: 3번 풍선(0번 인덱스)을 터뜨리면, 1이 적혀 있어 오른쪽으로 1칸 이동 (4번 풍선).
   - 터진 풍선: 1, 2, 3
   - 이동 후 q = [(3, -3), (4, -1)]
6. **4번 풍선 터뜨리기**: 4번 풍선(0번 인덱스)을 터뜨리면, -3이 적혀 있어 왼쪽으로 3칸 이동 (2, 1번 풍선).
   - 터진 풍선: 1, 2, 3, 4
   - 이동 후 q = [(4, -1), (2, 1)]
7. **5번 풍선 터뜨리기**: 5번 풍선(0번 인덱스)을 터뜨리면, -1이 적혀 있어 왼쪽으로 1칸 이동 (4번 풍선).
   - 터진 풍선: 1, 2, 3, 4, 5
   - 이동 후 q = [(2, 1)]
8. **마지막 풍선 터뜨리기**: 2번 풍선(0번 인덱스)을 터뜨리면, 1이 적혀 있어 오른쪽으로 1칸 이동 (없음).
   - 최종 결과: 1, 2, 3, 4, 5

위 과정을 통해 터진 풍선의 순서가 1, 2, 3, 4, 5임을 확인할 수 있습니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 풍선을 한번씩만 터뜨리기 때문에 N개의 풍선에 대해 선형 시간에 처리됩니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
