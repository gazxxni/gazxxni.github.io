## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이에 따라 삼각형의 유형을 판단하는 문제입니다. 삼각형의 성질에 따라, 가장 긴 변의 길이(c)가 나머지 두 변(a, b)의 합보다 작아야 하며, 이를 통해 "Invalid" 여부를 판단합니다. 그 후 변의 길이에 따라 Equilateral, Isosceles, Scalene 중 하나로 분류합니다.

### 동작 과정
1. **입력**: 세 변의 길이 a, b, c를 입력받습니다.
2. **종료 조건 확인**: 입력된 값이 모두 0이면 종료합니다.
3. **정렬**: a, b, c를 정렬하여 c가 가장 긴 변이 되도록 합니다.
4. **유효성 검사**: c가 a + b보다 크거나 같으면 "Invalid"를 출력합니다.
5. **삼각형 유형 판단**:
   - a, b, c가 모두 같으면 "Equilateral"을 출력합니다.
   - 두 변이 같고 하나가 다르면 "Isosceles"를 출력합니다.
   - 세 변이 모두 다르면 "Scalene"을 출력합니다.

**예시 입력**:
```
6 3 2
3 3 3
4 4 5
2 3 4
0 0 0
```

**단계별 설명**:
- **입력 (6, 3, 2)**: 정렬 후 (2, 3, 6) → 6 >= 2 + 3 (Invalid)
- **입력 (3, 3, 3)**: 정렬 후 (3, 3, 3) → a == b == c (Equilateral)
- **입력 (4, 4, 5)**: 정렬 후 (4, 4, 5) → a == b (Isosceles)
- **입력 (2, 3, 4)**: 정렬 후 (2, 3, 4) → 모두 다름 (Scalene)
- **입력 (0, 0, 0)**: 종료 조건 충족

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 변의 길이를 입력받고 비교하는 작업은 상수 시간에 수행되며, 정렬도 상수 개수의 원소(3개)에 대해서만 수행되므로 전체적으로 O(1)입니다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
