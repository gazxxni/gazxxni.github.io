## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이를 기반으로 삼각형의 종류를 결정하기 위해, 가장 긴 변이 나머지 두 변의 합보다 크거나 같은지를 확인하여 삼각형이 유효한지 판단합니다. 그런 다음 세 변의 길이에 따라 Equilateral, Isosceles, Scalene으로 분류합니다.

### 동작 과정
1. **입력 처리**: 
   - 사용자로부터 세 개의 변의 길이를 입력받습니다. 
   - 입력이 `0 0 0`이면 프로그램을 종료합니다.

2. **변 정렬**: 
   - 입력된 세 변을 정렬하여 가장 긴 변을 쉽게 찾을 수 있도록 합니다. 예를 들어, 입력이 `6, 3, 2`라면 정렬 후 `(2, 3, 6)`이 됩니다.

3. **삼각형 유효성 검사**: 
   - 가장 긴 변 `c`가 나머지 두 변 `a`와 `b`의 합보다 크거나 같으면 "Invalid"를 출력합니다. (예: `6 >= 2 + 3`)

4. **삼각형 종류 판별**:
   - 모든 변의 길이가 같으면 "Equilateral"을 출력합니다.
   - 두 변의 길이가 같고 하나는 다르면 "Isosceles"을 출력합니다.
   - 세 변의 길이가 모두 다르면 "Scalene"을 출력합니다.

5. **결과 출력**: 
   - 각 입력에 대해 적절한 결과를 출력합니다.

예시 입력으로는 다음과 같은 상황이 있을 수 있습니다:
- 입력: `6 3 2` → 출력: `Invalid`
- 입력: `3 3 3` → 출력: `Equilateral`
- 입력: `5 5 3` → 출력: `Isosceles`
- 입력: `4 5 6` → 출력: `Scalene`

### 시간 복잡도
이 알고리즘은 입력된 세 변을 정렬하는 과정이 포함되어 있으며, 정렬은 O(1) (상수 시간)로 처리됩니다(세 개의 원소만 정렬). 따라서 전체 시간 복잡도는 O(1)입니다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
