## 문제
재귀 호출만 생각하면 신이 난다! 아닌가요?

다음과 같은 재귀함수 w(a, b, c)가 있다.

if a  20 or b > 20 or c > 20, then w(a, b, c) returns:
    w(20, 20, 20)

if a 

위의 함수를 구현하는 것은 매우 쉽다. 하지만, 그대로 구현하면 값을 구하는데 매우 오랜 시간이 걸린다. (예를 들면, a=15, b=15, c=15)

a, b, c가 주어졌을 때, w(a, b, c)를 출력하는 프로그램을 작성하시오.

---

## 입력
입력은 세 정수 a, b, c로 이루어져 있으며, 한 줄에 하나씩 주어진다. 입력의 마지막은 -1 -1 -1로 나타내며, 세 정수가 모두 -1인 경우는 입력의 마지막을 제외하면 없다.

---

## 출력
입력으로 주어진 각각의 a, b, c에 대해서, w(a, b, c)를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 입력값 `a`, `b`, `c`에 대해 재귀적으로 정의된 함수 `w(a, b, c)`의 값을 계산하는 문제입니다. 재귀 호출을 통해 계산하면서, 이전에 계산한 값을 저장하여 중복 계산을 방지하는 메모이제이션 기법을 사용합니다. 이를 통해 시간 복잡도를 줄이고 효율적으로 결과를 도출할 수 있습니다.

### 동작 과정
1. **입력 처리**: `-1 -1 -1`이 입력으로 주어질 때까지 반복해서 `a`, `b`, `c` 값을 입력받습니다.
2. **조건 처리**:
   - `a`, `b`, `c` 중 하나라도 0 이하이면 `1`을 반환합니다.
   - `a`, `b`, `c` 중 하나라도 20을 초과하면, `w(a, b, c)`는 `w(20, 20, 20)`을 호출하여 값을 계산합니다.
3. **메모이제이션**: 이미 계산한 값이 저장되어 있다면, 해당 값을 그대로 반환합니다.
4. **재귀 호출**:
   - 만약 `a < b < c`이면, 다음과 같이 값을 계산합니다:
     ```
     dp[a][b][c] = w(a, b, c - 1) + w(a, b - 1, c - 1) - w(a, b - 1, c)
     ```
   - 그 외의 경우에는 다음과 같이 값을 계산합니다:
     ```
     dp[a][b][c] = w(a - 1, b, c) + w(a - 1, b - 1, c) + w(a - 1, b, c - 1) - w(a - 1, b - 1, c - 1)
     ```
5. **결과 출력**: 계산된 `w(a, b, c)` 값을 출력합니다.

**예시 입력**:
```
1 2 3
2 3 4
10 20 30
-1 -1 -1
```
**출력**:
```
w(1, 2, 3) = 5
w(2, 3, 4) = 6
w(10, 20, 20) = 1048576
```

### 시간 복잡도
메모이제이션을 사용하여 각 `(a, b, c)` 조합에 대해 최대 21x21x21 = 9261개의 상태를 저장하므로, 시간 복잡도는 O(1)로 상수 시간에 값을 조회할 수 있습니다. 그러나 재귀 호출의 깊이가 최대 20까지 갈 수 있으므로 실질적인 계산량은 O(1)로 제한됩니다.

## 코드
```python
def w(a, b, c):
    if a <= 0 or b <= 0 or c <= 0:
        return 1
    
    if a > 20 or b > 20 or c > 20:
        return w(20, 20, 20)
    
    if dp[a][b][c]:
        return dp[a][b][c]
    
    if a < b < c:
        dp[a][b][c] = w(a, b, c - 1) + w(a, b - 1, c- 1) - w(a, b - 1, c)
        return dp[a][b][c]
    
    dp[a][b][c] = w(a - 1, b, c) + w(a - 1, b - 1, c) + w(a - 1, b, c - 1) - w(a - 1, b - 1, c - 1)
    return dp[a][b][c]

dp = [[[0] * 21 for _ in range(21)] for _ in range(21)]

while True:
    a, b, c = map(int, input().split())

    if a == -1 and b == -1 and c == -1:
        exit()
    
    print(f'w({a}, {b}, {c}) = {w(a, b, c)}')
```
