## 문제
수열 A가 주어졌을 때, 가장 긴 증가하는 부분 수열을 구하는 프로그램을 작성하시오.

예를 들어, 수열 A = {10, 20, 10, 30, 20, 50} 인 경우에 가장 긴 증가하는 부분 수열은 A = {10, 20, 10, 30, 20, 50} 이고, 길이는 4이다.

---

## 입력
첫째 줄에 수열 A의 크기 N (1 ≤ N ≤ 1,000,000)이 주어진다.

둘째 줄에는 수열 A를 이루고 있는 Ai가 주어진다. (-1,000,000,000 ≤ Ai ≤ 1,000,000,000)

---

## 출력
첫째 줄에 수열 A의 가장 긴 증가하는 부분 수열의 길이를 출력한다.

둘째 줄에는 정답이 될 수 있는 가장 긴 증가하는 부분 수열을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 가장 긴 증가하는 부분 수열(LIS)을 구하는 문제로, 이분 탐색을 활용하여 O(N log N) 시간 복잡도로 해결할 수 있다. 각 원소에 대해 현재까지의 LIS를 구성할 수 있는 가장 작은 끝값을 유지하는 리스트 `tails`를 만들어, 새로운 원소가 들어올 때 이 리스트를 업데이트하여 LIS의 길이를 계산한다. 또한, 역추적을 통해 실제 LIS를 구성할 수 있다.

### 동작 과정
1. 입력으로 주어진 수열 A의 크기 N과 원소들을 받는다.
2. `tails` 리스트와 `tails_pos` 리스트를 초기화한다. `tails`는 현재까지의 LIS의 각 길이에 대한 최소 끝값을, `tails_pos`는 해당 끝값을 만든 원소의 인덱스를 저장한다.
3. `prev_idx` 리스트를 사용하여 각 원소가 LIS의 일부일 때 그 이전 원소의 인덱스를 저장한다.
4. 각 원소를 순회하면서 이분 탐색을 통해 `tails` 리스트에서 현재 원소가 들어갈 위치를 찾는다.
5. 찾은 위치에 따라 `tails`를 업데이트하고, `prev_idx`를 갱신한다.
6. `tails`의 길이가 LIS의 길이가 된다. 마지막 인덱스에서 시작하여 `prev_idx`를 따라가며 실제 LIS를 추출한다.
7. 최종적으로 LIS의 길이와 원소들을 출력한다.

예를 들어, 수열 A = {10, 20, 10, 30, 20, 50}인 경우:
- 10을 추가: `tails` = [10]
- 20을 추가: `tails` = [10, 20]
- 10을 추가: `tails` = [10, 20] (20의 자리를 10으로 대체)
- 30을 추가: `tails` = [10, 20, 30]
- 20을 추가: `tails` = [10, 20, 30] (30의 자리를 20으로 대체)
- 50을 추가: `tails` = [10, 20, 30, 50] (길이 확장)

이때, `tails`의 길이는 4이므로 LIS의 길이는 4이고, 역추적을 통해 실제 LIS가 {10, 20, 30, 50}임을 알 수 있다.

### 시간 복잡도
O(N log N) - N은 수열의 크기이며, log N은 이분 탐색의 시간 복잡도이다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

# tails[k] = 길이 (k+1)인 증가 부분수열의 최소 끝값
# tails_pos[k] = 위 tails[k]를 만든 인덱스
tails = []
tails_pos = []

# prev_idx[i] = arr[i]가 LIS에 포함될 때 바로 이전 원소의 인덱스
prev_idx = [-1] * n

def search(a, x):
    lo, hi = 0, len(a)
    while lo < hi:
        mid = (lo + hi) // 2
        if a[mid] < x:
            lo = mid + 1
        else:
            hi = mid
    return lo

for i, x in enumerate(arr):
    j = search(tails, x)

    # j-1 길이의 꼬리에 이어 붙이므로, 경로의 마지막 인덱스를 prev로 기록
    prev_idx[i] = tails_pos[j - 1] if j > 0 else -1

    if j == len(tails):  # 길이 확장
        tails.append(x)
        tails_pos.append(i)
    else:    # 꼬리값 갱신
        tails[j] = x
        tails_pos[j] = i

# 실가장 긴 길이의 마지막 인덱스부터 prev_idx를 따라 역추적
lis_len = len(tails)
k = tails_pos[-1] if tails_pos else -1

seq = []
while k != -1:
    seq.append(arr[k])
    k = prev_idx[k]
seq.reverse()

print(lis_len)
print(*seq)
```
