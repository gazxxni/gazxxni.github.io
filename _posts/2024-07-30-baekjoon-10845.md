## 풀이

### 풀이 핵심 로직
이 문제는 큐(Queue)의 기능을 구현하는 문제이다. 주어진 명령어에 따라 큐의 원소를 추가, 삭제하고, 큐의 상태를 확인하는 작업을 수행한다. `push`, `pop`, `size`, `empty`, `front`, `back`의 각 명령어에 대해 적절한 큐의 동작을 구현하였다.

### 동작 과정
예시 입력:
```
7
push 1
push 2
front
back
size
pop
empty
```

1. `push 1`: 큐에 1을 추가. 큐 상태: `[1]`
2. `push 2`: 큐에 2를 추가. 큐 상태: `[1, 2]`
3. `front`: 큐의 가장 앞에 있는 원소(1)를 출력. 출력: `1`
4. `back`: 큐의 가장 뒤에 있는 원소(2)를 출력. 출력: `2`
5. `size`: 큐의 크기(2)를 출력. 출력: `2`
6. `pop`: 큐에서 가장 앞의 원소(1)를 제거하고 출력. 큐 상태: `[2]`, 출력: `1`
7. `empty`: 큐가 비어있는지 확인. 비어있지 않으므로 출력: `0`

### 시간 복잡도
모든 명령어의 시간 복잡도는 O(1)이다. 하지만 `pop` 명령에서 `pop(0)`을 사용하였기 때문에 큐의 원소를 제거할 때 O(n)의 시간이 소요된다. 따라서 최악의 경우 전체 시간 복잡도는 O(n^2)이다.

## 코드
```python
import sys

n = int(sys.stdin.readline().strip())

stack=[]

for i in range(n):
    cmd=sys.stdin.readline().strip().split()

    if cmd[0]=='push':
        stack.append(cmd[1])
    elif cmd[0]=='pop':
        if len(stack)!=0:
            print(stack.pop(0))
        else:
            print(-1)
    elif cmd[0]=='size':
        print(len(stack))
    elif cmd[0]=='empty':
        if len(stack)==0:
            print(1)
        else:
            print(0)
    elif cmd[0]=='front':
        if len(stack)!=0:
            print(stack[0])
        else:
            print(-1)
    elif cmd[0]=='back':
        if len(stack)!=0:
            print(stack[-1])
        else:
            print(-1)

            
```
