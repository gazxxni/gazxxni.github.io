## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 M개를 고른 수열
	같은 수를 여러 번 골라도 된다.

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
주어진 문제는 1부터 N까지의 자연수 중에서 M개의 수를 선택하여 수열을 만드는 문제로, 같은 수를 여러 번 선택할 수 있습니다. 이를 해결하기 위해 백트래킹 기법을 사용하여 재귀적으로 수열을 생성하고, 수열의 길이가 M에 도달했을 때 결과를 출력합니다.

### 동작 과정
1. 예를 들어, N=3, M=2인 경우를 생각해보겠습니다.
2. 함수 `aa()`를 호출하면, 현재 수열인 `arr`의 길이가 M인지 확인합니다.
3. 길이가 M이면 현재 수열을 출력하고 함수 호출을 종료합니다.
4. 그렇지 않다면, 1부터 N까지의 숫자를 반복하며:
   - 현재 숫자를 `arr`에 추가합니다.
   - `aa()`를 재귀적으로 호출하여 다음 숫자를 추가합니다.
   - 수열이 돌아오는 과정에서 마지막으로 추가한 숫자를 제거하여 다음 숫자를 추가할 수 있도록 합니다.
5. 이 과정을 통해 모든 가능한 수열을 생성하고 출력합니다.

예시 입력:
```
3 2
```
출력:
```
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^M)입니다. 각각의 레벨에서 N개의 숫자를 선택할 수 있으며, 깊이는 M이기 때문에 최악의 경우 모든 조합을 생성하게 됩니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
