## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 M개를 고른 수열
	같은 수를 여러 번 골라도 된다.

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 백트래킹(Backtracking) 기법을 사용하여 1부터 N까지의 자연수 중에서 M개의 수를 선택하는 모든 조합을 생성하는 것입니다. 각 수열에 대해 재귀적으로 숫자를 추가하고, 수열의 길이가 M에 도달하면 출력한 후, 마지막에 추가한 숫자를 제거하여 다음 숫자를 추가하는 방식으로 모든 경우를 탐색합니다.

### 동작 과정
1. **입력**: 사용자로부터 두 개의 자연수 N과 M을 입력받습니다. 예를 들어, 입력이 `3 1`이라면 N은 3, M은 1입니다.
2. **초기화**: 빈 리스트 `arr`를 생성하여 수열을 저장합니다.
3. **재귀 함수 호출**: 
   - `aa()` 함수가 호출되면 현재 수열의 길이가 M인지 확인합니다.
   - 길이가 M인 경우, 현재 수열을 출력합니다.
   - 그렇지 않은 경우, 1부터 N까지의 숫자를 반복하면서, 수열에 숫자를 추가하고 다시 `aa()` 함수를 호출합니다.
   - 호출이 끝난 후에는 마지막에 추가된 숫자를 수열에서 제거하여 다음 숫자를 추가할 수 있도록 합니다.
4. **출력**: 모든 가능한 수열이 출력됩니다. 예를 들어, `N=3`이고 `M=1`인 경우, 출력은 다음과 같습니다:
   ```
   1
   2
   3
   ```
   `N=3`이고 `M=2`인 경우, 출력은 다음과 같습니다:
   ```
   1 1
   1 2
   1 3
   2 1
   2 2
   2 3
   3 1
   3 2
   3 3
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^M)입니다. 이는 각 자리수에 대해 N개의 선택지가 있으며, 이러한 선택이 M번 반복되기 때문입니다. 따라서, 최악의 경우 N=7, M=7일 때 7^7 = 823543개의 수열이 생성될 수 있습니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
