## 문제
지난 밤 겨울 숲에는 눈이 많이 내렸다. 당신은 숲의 주민들을 위해 눈이 오지 않는 동안 모든 집 앞의 눈을 치우고자 한다.

당신은 1분에 한 번씩 두 집을 선택해서 두 집 앞의 눈을 각각 1만큼 치우거나, 한 집을 선택해서 그 집 앞의 눈을 1만큼 치울 수 있다.

모든 집 앞의 눈을 전부 치울 때까지 걸리는 최소 시간은 얼마일까?

---

## 입력
첫 줄에 집의 수를 의미하는 정수 N$N$ (1≤N≤100$1 \leq N \leq 100$)이 주어진다.

다음 줄에는 각각의 집 앞에 쌓여 있는 눈의 양을 나타내는 정수 ai$a_{i}$ (1≤ai≤2000$1 \leq a_{i} \leq 2000$)이 주어진다.

---

## 출력
모든 집 앞의 눈을 치우는 데 최소 몇 분이 걸리는지를 출력한다. 24시간(1440분)이 넘게 걸릴 경우 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
눈을 치우는 과정에서는 항상 가장 눈이 많이 쌓인 두 집을 선택하여 동시에 눈을 치우는 것이 가장 효율적입니다. 이를 통해 눈을 치우는 시간을 최소화할 수 있습니다. 집의 수가 1개일 경우에는 그 집 앞의 눈을 혼자 치워야 하므로 별도의 처리가 필요합니다.

### 동작 과정
1. **입력 예시**: 
   ```
   5
   4 2 3 6 1
   ```
   - 집의 수 \( N = 5 \)
   - 각 집 앞의 눈의 양: [4, 2, 3, 6, 1]

2. **첫 번째 반복**:
   - 배열을 내림차순으로 정렬: [6, 4, 3, 2, 1]
   - 가장 많은 눈을 가진 두 집(6, 4) 선택
   - 각각 4만큼 눈을 치운다: [2, 4, 3, 2, 1]
   - 카운터 업데이트: `cnt = 4`

3. **두 번째 반복**:
   - 배열: [4, 3, 2, 2, 1] (정렬)
   - 가장 많은 눈을 가진 두 집(4, 3) 선택
   - 각각 3만큼 눈을 치운다: [1, 4, 2, 2, 1]
   - 카운터 업데이트: `cnt = 7`

4. **세 번째 반복**:
   - 배열: [4, 2, 2, 1, 1] (정렬)
   - 가장 많은 눈을 가진 두 집(4, 2) 선택
   - 각각 2만큼 눈을 치운다: [2, 2, 2, 1, 1]
   - 카운터 업데이트: `cnt = 9`

5. **이후 반복**:
   - 계속해서 위 과정을 반복하여 집 앞의 눈을 모두 치우고, 최종적으로 카운터를 업데이트합니다.
   - 모든 집의 눈이 없어지면 반복을 종료하고, 최종 카운터 값을 반환합니다.

6. **마지막 확인**:
   - 만약 `cnt` 값이 1440을 초과하면 `-1`을 출력하고, 그렇지 않으면 `cnt` 값을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(N^2 \log N) \)입니다. 
1. 매 반복마다 배열을 정렬하는데 \( O(N \log N) \)의 시간이 소요되며,
2. 눈을 치우는 횟수는 각 집의 눈의 양에 따라 달라지므로 최악의 경우 \( O(N) \)의 반복을 수행할 수 있습니다.
3. 결과적으로, 최악의 경우 \( O(N^2 \log N) \)의 시간 복잡도를 가집니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

cnt = 0

while True:
    if len(arr) == 1:
        break
    
    arr.sort(reverse=True)
    arr[0] -= arr[1]
    cnt += arr[1]
    arr.pop(1)
    
cnt += arr[0]
if cnt > 1440:
    print(-1)
else:
    print(cnt)
```
