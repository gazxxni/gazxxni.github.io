## 문제
왕비를 피해 일곱 난쟁이들과 함께 평화롭게 생활하고 있던 백설공주에게 위기가 찾아왔다. 일과를 마치고 돌아온 난쟁이가 일곱 명이 아닌 아홉 명이었던 것이다.

아홉 명의 난쟁이는 모두 자신이 "백설 공주와 일곱 난쟁이"의 주인공이라고 주장했다. 뛰어난 수학적 직관력을 가지고 있던 백설공주는, 다행스럽게도 일곱 난쟁이의 키의 합이 100이 됨을 기억해 냈다.

아홉 난쟁이의 키가 주어졌을 때, 백설공주를 도와 일곱 난쟁이를 찾는 프로그램을 작성하시오.

---

## 입력
아홉 개의 줄에 걸쳐 난쟁이들의 키가 주어진다. 주어지는 키는 100을 넘지 않는 자연수이며, 아홉 난쟁이의 키는 모두 다르며, 가능한 정답이 여러 가지인 경우에는 아무거나 출력한다.

---

## 출력
일곱 난쟁이의 키를 오름차순으로 출력한다. 일곱 난쟁이를 찾을 수 없는 경우는 없다.

## 풀이

### 풀이 핵심 로직
이 문제는 9명의 난쟁이 중에서 2명의 난쟁이를 제외하고 나머지 7명의 키의 합이 100이 되어야 하는 조건을 만족해야 합니다. 따라서, 모든 난쟁이의 키를 더한 후 100을 빼면, 그 결과는 제외해야 할 두 난쟁이의 키의 합과 일치합니다. 이 두 난쟁이를 찾기 위해 이중 반복문을 사용하여 가능한 조합을 모두 확인합니다.

### 동작 과정
1. 난쟁이들의 키를 입력받아 리스트에 저장합니다.
   ```
   입력: 20
           7
           23
           19
           10
           15
           25
           8
           13
   ```
2. 입력된 키를 오름차순으로 정렬합니다.
   ```
   정렬된 배열: [7, 8, 10, 13, 15, 19, 20, 23, 25]
   ```
3. 모든 난쟁이의 키의 합을 계산합니다.
   ```
   배열 합: 7 + 8 + 10 + 13 + 15 + 19 + 20 + 23 + 25 = 120
   ```
4. 120에서 100을 빼서 제외해야 할 두 난쟁이의 키의 합을 구합니다.
   ```
   제외할 두 난쟁이의 키의 합: 120 - 100 = 20
   ```
5. 이중 반복문을 사용하여 두 난쟁이를 찾습니다. 
   - 첫 번째 반복문은 `i`를 0부터 8까지, 두 번째 반복문은 `j`를 `i + 1`부터 8까지 진행합니다.
   - 두 난쟁이의 키의 합이 20인 경우를 찾아 해당 난쟁이의 키를 `fake1`과 `fake2`에 저장합니다.
6. 최종적으로 나머지 7명의 난쟁이를 출력합니다. 
   ```
   출력: 7
           8
           10
           13
           15
           19
           23
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 여기서 N은 난쟁이의 수(9개)로 고정되어 있으므로, 상수 시간 내에 실행된다고 볼 수 있습니다.

---

## 코드
```python
arr = [int(input()) for _ in range(9)]
arr.sort()

sum_arr = sum(arr)
target = sum_arr - 100

fake1 = 0
fake2 = 0

for i in range(len(arr)):
    for j in range(i + 1, len(arr)):
        if arr[i] + arr[j] == target:
            fake1 = arr[i]
            fake2 = arr[j]
            break
    if fake1 != 0:
        break

for i in arr:
    if i != fake1 and i != fake2:
        print(i)
```
