## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 두 집합의 차집합을 이용하여 서로 다른 원소의 개수를 구하는 것입니다. 두 집합 `arr`과 `brr`의 차집합을 구한 후, 각 집합에서 다른 집합에 속하지 않는 원소의 수를 더하여 최종적으로 서로 다른 원소의 개수를 계산합니다.

### 동작 과정
1. 첫 번째 입력으로 두 정수 `a`와 `b`를 받아옵니다. 이는 각각 첫 번째 집합과 두 번째 집합의 원소 개수를 나타냅니다.
2. 두 번째 줄에서 `arr` 집합을 입력받고, 세 번째 줄에서 `brr` 집합을 입력받습니다.
3. `arr - brr`를 통해 `arr`에만 있는 원소의 집합 `crr`을 생성합니다.
4. `brr - arr`을 통해 `brr`에만 있는 원소의 집합 `drr`을 생성합니다.
5. 두 집합의 길이를 더하여 서로 다른 원소의 총 개수를 출력합니다.

예시 입력:
```
3 5
1 2 3
2 3 4 5 6
```

단계별 설명:
- 첫 번째 줄에서 `a`는 3, `b`는 5로 설정됩니다.
- 두 번째 줄에서 원소 `1, 2, 3`이 포함된 집합 `arr`이 생성됩니다.
- 세 번째 줄에서 원소 `2, 3, 4, 5, 6`이 포함된 집합 `brr`이 생성됩니다.
- `arr - brr`을 계산하면 `crr = {1}`이 되고, `brr - arr`을 계산하면 `drr = {4, 5, 6}`이 됩니다.
- 두 집합의 크기를 더하면 `len(crr) + len(drr) = 1 + 3 = 4`가 되어 최종 출력값은 `4`입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n + m)입니다. 여기서 n과 m은 각각 `arr`과 `brr`의 원소 개수입니다. 집합의 생성 및 차집합 연산은 평균적으로 O(n)과 O(m)의 시간 복잡도를 가지므로 전체적으로 O(n + m)입니다.

## 코드
```python
a, b = map(int, input().split())
arr = set(list(map(int, input().split())))
brr = set(list(map(int, input().split())))

crr = arr - brr
drr = brr - arr

print(len(crr) + len(drr))
```
