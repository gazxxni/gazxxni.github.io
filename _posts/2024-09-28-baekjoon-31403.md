## 풀이

### 풀이 핵심 로직
문제의 핵심 아이디어는 입력받은 세 개의 문자열 A, B, C를 이용하여 두 가지 계산을 수행하는 것입니다. 첫 번째 계산은 A와 B를 정수로 변환한 후 더하고, C를 정수로 변환하여 빼는 것입니다. 두 번째 계산은 A와 B를 문자열로 합친 뒤 정수로 변환하고, C를 정수로 변환하여 다시 빼는 것입니다.

### 동작 과정
1. 입력 예시:
   ```
   A = "123"
   B = "456"
   C = "789"
   ```
   
2. 첫 번째 출력 계산:
   - `int(A) + int(B) - int(C)` 계산:
     - `int(A) = 123`
     - `int(B) = 456`
     - `int(C) = 789`
     - 결과: `123 + 456 - 789 = -210`
   
3. 두 번째 출력 계산:
   - `int(A + B) - int(C)` 계산:
     - `A + B = "123456"` (문자열 결합)
     - `int(A + B) = 123456`
     - `int(C) = 789`
     - 결과: `123456 - 789 = 122667`

4. 최종 출력:
   ```
   -210
   122667
   ```

### 시간 복잡도
이 코드의 시간 복잡도는 O(n)입니다. 여기서 n은 입력 문자열의 길이입니다. 각 문자열 A, B, C를 정수로 변환하거나 결합하는 과정에서 문자열의 길이에 비례하는 시간이 소요되기 때문입니다.

---

## 코드
```python
import sys

A = input()
B = input()
C = input()

print(int(A) + int(B) - int(C))
print(int(A + B) - int(C))
```
