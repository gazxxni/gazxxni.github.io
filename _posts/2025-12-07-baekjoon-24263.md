## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 핵심은 입력 크기 n에 대해 단순히 n 번의 작업을 수행하는 것입니다. 이 알고리즘에서 가장 중요한 작업의 수행 횟수는 n으로, 이는 입력의 크기에 비례합니다. 따라서 수행 횟수는 n이 됩니다. 또한, 이 다항식의 최고차항의 차수는 1입니다.

### 동작 과정
1. 입력으로 n(1 ≤ n ≤ 500,000)을 받습니다.
2. 첫 번째 출력으로 n을 출력합니다. 이는 알고리즘의 수행 횟수를 나타냅니다.
3. 두 번째 출력으로 1을 출력합니다. 이는 다항식에서 최고차항의 차수를 나타냅니다.

예를 들어, 입력이 5일 경우:
- n = 5
- 첫 번째 줄에 5를 출력합니다.
- 두 번째 줄에 1을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력 크기 n에 관계없이 수행되는 작업이 상수 시간에 끝나기 때문입니다.
