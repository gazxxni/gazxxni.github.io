## 문제
네 점이 주어졌을 때, 네 점을 이용해 정사각형을 만들 수 있는지 없는지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 네 줄로 이루어져 있으며, 점의 좌표가 한 줄에 하나씩 주어진다. 점의 좌표는 -100,000보다 크거나 같고, 100,000보다 작거나 같은 정수이다. 같은 점이 두 번 이상 주어지지 않는다.

---

## 출력
각 테스트 케이스마다 입력으로 주어진 네 점을 이용해서 정사각형을 만들 수 있으면 1을, 없으면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 네 점이 정사각형의 꼭짓점인지 판단하기 위해, 각 두 점 사이의 거리의 제곱을 계산하여 그 결과를 카운트한다. 정사각형의 경우, 4개의 변의 길이가 같고, 2개의 대각선의 길이가 같아야 하므로, 거리의 제곱을 사용하여 각각의 경우를 세어 8개의 변과 2개의 대각선이 존재하는지를 확인한다.

### 동작 과정
1. 테스트 케이스의 개수 `T`를 입력받는다.
2. 각 테스트 케이스마다 4개의 점의 좌표를 입력받아 리스트 `arr`에 저장한다.
3. 두 점 사이의 거리의 제곱을 계산하여 사전 `dic`에 저장하며, 각 거리 값의 빈도를 센다. 
   - 예를 들어, 점 A(0, 0), B(1, 0), C(0, 1), D(1, 1) 이라면:
     - A와 B의 거리 제곱: (0-1)² + (0-0)² = 1
     - A와 C의 거리 제곱: (0-0)² + (0-1)² = 1
     - A와 D의 거리 제곱: (0-1)² + (0-1)² = 2
     - B와 C의 거리 제곱: (1-0)² + (0-1)² = 2
     - B와 D의 거리 제곱: (1-1)² + (0-1)² = 1
     - C와 D의 거리 제곱: (0-1)² + (1-1)² = 1
   - 이 경우, 거리 제곱을 계산하면 1이 4회, 2가 2회 등장한다.
4. 사전 `dic`를 통해 빈도를 체크하여, 8개의 변의 길이(같은 거리 값)와 2개의 대각선의 길이(같은 거리 값)가 각각 존재하는지 확인한다.
5. 조건을 만족하면 1을 출력하고, 그렇지 않으면 0을 출력한다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)이다. 각 테스트 케이스에서 점의 수가 고정되어 있으므로, 거리 계산의 횟수는 항상 일정하다.

## 코드
```python
from collections import defaultdict
t = int(input())

for _ in range(t):
    arr = []
    
    for i in range(4):
        x, y = map(int, input().split())
        arr.append((x, y))
    
    dic = defaultdict(int)
    for i in range(4):
        for j in range(4):
            if i != j:
                a = (arr[i][0] - arr[j][0]) ** 2 + (arr[i][1] - arr[j][1]) ** 2
                dic[a] += 1
    
    aa = False
    bb = False  
    for k, v in dic.items():
        if v == 8:
            aa = True
        
        elif v == 4:
            bb = True
            
    if aa and bb:
        print(1)
        
    else:
        print(0)
```
