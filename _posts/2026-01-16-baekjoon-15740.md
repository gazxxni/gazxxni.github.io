## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 A와 B (-1010000 ≤ A, B ≤ 1010000)가 주어진다.

---

## 출력
첫째 줄에 A+B를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 96 ms

---

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 큰 수를 처리할 수 있는 파이썬의 기본 자료형인 `int`를 사용하여 두 개의 정수를 입력받고, 이를 더한 값을 출력하는 것입니다. 파이썬은 임의 정밀도를 지원하므로, 주어진 범위의 정수도 문제 없이 처리할 수 있습니다.

### 동작 과정
1. `sys.stdin.readline`을 사용하여 입력을 받습니다. 이는 입력 속도를 높이기 위해 사용됩니다.
2. 입력된 문자열을 `split()` 메소드를 통해 공백 기준으로 나누고, `map(int, ...)`를 사용하여 두 개의 정수로 변환합니다.
3. 변환된 두 정수 `a`와 `b`를 더하여 결과를 출력합니다.

예를 들어, 입력으로 `123456789123456789 987654321987654321`이 주어지면, 
- 첫 번째 단계에서 `input()`으로부터 문자열 `'123456789123456789 987654321987654321'`을 읽어옵니다.
- `split()`을 통해 `['123456789123456789', '987654321987654321']`로 나누고,
- `map(int, ...)`을 사용하여 각각 `123456789123456789`와 `987654321987654321`이라는 정수로 변환합니다.
- 마지막으로 두 정수를 더하여 `1111111111111111110`을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 여기서 N은 입력받은 수의 자릿수입니다. 파이썬의 정수 덧셈 연산은 내부적으로 자릿수에 비례하는 시간 복잡도를 가지므로, 입력의 크기에 따라 연산 시간이 결정됩니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a + b)
```
