## 문제
오늘도 서준이는 병합 정렬 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

N개의 서로 다른 양의 정수가 저장된 배열 A가 있다. 병합 정렬로 배열 A를 오름차순 정렬할 경우 배열 A에 K 번째 저장되는 수를 구해서 우리 서준이를 도와주자.

크기가 N인 배열에 대한 병합 정렬 의사 코드는 다음과 같다.

merge_sort(A[p..r]) { # A[p..r]을 오름차순 정렬한다.
    if (p < r) then {
        q <- ⌊(p + r) / 2⌋;       # q는 p, r의 중간 지점
        merge_sort(A, p, q);      # 전반부 정렬
        merge_sort(A, q + 1, r);  # 후반부 정렬
        merge(A, p, q, r);        # 병합
    }
}

---

## 입력
첫째 줄에 배열 A의 크기 N(5 ≤ N ≤ 500,000), 저장 횟수 K(1 ≤ K ≤ 108)가 주어진다.

다음 줄에 서로 다른 배열 A의 원소 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 109)

---

## 출력
배열 A에 K 번째 저장 되는 수를 출력한다. 저장 횟수가 K 보다 작으면 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 병합 정렬을 수행하면서 K번째 저장된 값을 기록하는 문제가입니다. 병합 과정에서 배열을 정렬할 때마다 카운터를 증가시키고, K에 해당하는 카운터 값을 찾으면 해당 값을 저장합니다. 병합 정렬의 재귀 구조를 활용하여 각 단계에서 값을 저장하는 방식으로 해결합니다.

### 동작 과정
예시 입력:
```
5 3
5 3 1 4 2
```

1. 처음 입력된 배열은 `[5, 3, 1, 4, 2]` 입니다. 우리는 이 배열을 병합 정렬로 정렬하면서 K=3에 해당하는 저장 횟수를 찾습니다.
   
2. `merge_sort` 함수 호출로 시작합니다. 배열을 반으로 나누어 `[5, 3]`와 `[1, 4, 2]`로 나누고 각각을 정렬합니다.
   
3. `[5, 3]`은 다시 `[5]`와 `[3]`으로 나누어지며, 각각 정렬 후 병합 과정에서 `[3, 5]`로 정렬됩니다. 이 과정에서 저장 횟수를 카운트합니다. `cnt`가 1이 되고, 2가 되며, 3이 될 때 `[3, 5]`에서 3이 저장되므로 `ans`에 3이 저장됩니다.
   
4. `[1, 4, 2]`는 `[1]`과 `[4, 2]`로 나누어지고, `[4, 2]`는 다시 `[4]`와 `[2]`로 나누어 정렬됩니다. 이 과정에서도 저장 횟수가 증가합니다.
   
5. 최종적으로 병합된 배열은 `[1, 2, 3, 4, 5]`가 되고, K=3에 해당하는 3이 `ans`에 저장되어 출력됩니다.

### 시간 복잡도
병합 정렬의 시간 복잡도는 O(N log N)입니다. 여기서 N은 배열의 크기입니다. 추가적으로, K에 따라 저장 횟수를 체크하는 과정이 있으나 이는 각각의 병합 과정에서 진행되므로 전체 복잡도에 큰 영향을 미치지 않습니다.

## 코드
```python
def merge_sort(A, p, r):
    if p < r:
        q = (p + r) // 2
        merge_sort(A, p, q)
        merge_sort(A, q + 1, r)
        merge(A, p, q, r)

def merge(A, p, q, r):
    global cnt, ans
    
    i = p
    j = q + 1
    t = 0
    tmp = [0] * (r - p + 1)
    
    while i <= q and j <= r:
        if A[i] <= A[j]:
            tmp[t] = A[i]
            i += 1
        else:
            tmp[t] = A[j]
            j += 1
        t += 1
        
    while i <= q:
        tmp[t] = A[i]
        i += 1
        t += 1
        
    while j <= r:
        tmp[t] = A[j]
        j += 1
        t += 1
        
    i = p
    t = 0
    
    while i <= r:
        A[i] = tmp[t]
        cnt += 1
        
        if cnt == k:
            ans = A[i]
            
        i += 1
        t += 1

n, k = map(int, input().split())
arr = list(map(int, input().split()))

cnt = 0
ans = -1

merge_sort(arr, 0, n - 1)

print(ans)
```
