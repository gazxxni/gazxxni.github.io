## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 1부터 N까지의 모든 수를 이어서 썼을 때, 그 결과의 자릿수를 계산하는 것입니다. 자릿수의 계산은 각 자릿수 범위별로 나누어 생각하면 간편해집니다. 1자리 수, 2자리 수, ..., N의 자릿수까지의 합을 계산하여 최종적으로 결과를 도출합니다.

### 동작 과정
1. 입력으로 주어진 N의 자리수를 구합니다. 예를 들어, N = 23일 경우, 23은 2자리 수이므로 `l = 2`가 됩니다.
2. 1자리 수(1~9)에서 총 자릿수는 9개이고, 각 수는 1자리이므로 `9 * 1 = 9` 자릿수를 차지합니다.
3. 2자리 수(10~99)에서 총 자릿수는 90개이고, 각 수는 2자리이므로 `90 * 2 = 180` 자릿수를 차지합니다.
4. N이 3자리 수(100~999) 이상이라면, 3자리 수의 자릿수도 계산해야 하지만, N이 23이므로 이 부분은 건너뜁니다.
5. 마지막으로 2자리 수(10~23)의 자릿수는 `(23 - 10 + 1) * 2 = 28` 자릿수를 차지합니다.
6. 총 자릿수는 9 + 180 + 28로 계산하여 결과를 출력합니다.

예시:
- N = 23일 경우:
  - 1자리 수: 9
  - 2자리 수: 28
  - 총 합: 9 + 28 = 37

### 시간 복잡도
O(log N) - N의 자리수를 구하는 과정에서 로그 시간 복잡도를 가지며, 일정 범위 반복문을 통해 자릿수를 계산합니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
