## 문제
오늘도 서준이는 병합 정렬 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

N개의 서로 다른 양의 정수가 저장된 배열 A가 있다. 병합 정렬로 배열 A를 오름차순 정렬할 경우 배열 A에 K 번째 저장되는 수를 구해서 우리 서준이를 도와주자.

크기가 N인 배열에 대한 병합 정렬 의사 코드는 다음과 같다.

merge_sort(A[p..r]) { # A[p..r]을 오름차순 정렬한다.
    if (p < r) then {
        q <- ⌊(p + r) / 2⌋;       # q는 p, r의 중간 지점
        merge_sort(A, p, q);      # 전반부 정렬
        merge_sort(A, q + 1, r);  # 후반부 정렬
        merge(A, p, q, r);        # 병합
    }
}

---

## 입력
첫째 줄에 배열 A의 크기 N(5 ≤ N ≤ 500,000), 저장 횟수 K(1 ≤ K ≤ 108)가 주어진다.

다음 줄에 서로 다른 배열 A의 원소 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 109)

---

## 출력
배열 A에 K 번째 저장 되는 수를 출력한다. 저장 횟수가 K 보다 작으면 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 병합 정렬을 사용해 배열을 정렬하는 과정에서 특정 K번째 저장되는 수를 찾는 것이다. 병합 정렬의 과정에서 원소를 배열에 저장할 때마다 카운트를 증가시키고, 그 카운트가 K에 도달하면 해당 원소를 기록하여 최종적으로 출력한다.

### 동작 과정
예를 들어, 입력으로 주어진 배열이 `[4, 2, 5, 3, 1]`이고 K가 `3`이라고 가정하자. 병합 정렬의 과정은 다음과 같다:

1. **초기 배열**: `[4, 2, 5, 3, 1]`
2. **분할 단계**:
   - `[4, 2, 5]`와 `[3, 1]`으로 나눈다.
   - `[4, 2]`와 `[5]`, `[3]`와 `[1]`로 각각 나눈다.
   - 최종적으로 `[4]`, `[2]`, `[5]`, `[3]`, `[1]`으로 나누어진다.

3. **정렬 및 병합 단계**:
   - `[4]`와 `[2]`를 병합하여 `[2, 4]` 저장 (1회 저장)
   - `[2, 4]`와 `[5]`를 병합하여 `[2, 4, 5]` 저장 (2회 저장)
   - `[3]`와 `[1]`을 병합하여 `[1, 3]` 저장 (3회 저장)
   - `[2, 4, 5]`와 `[1, 3]`을 병합하여 `[1, 2, 3, 4, 5]` 저장 (4회 저장)
   
   이 과정 중 3번째 저장 시에 `1`이 저장되므로, 출력값은 `1`이 된다.

### 시간 복잡도
병합 정렬의 시간 복잡도는 O(N log N)이다. 여기서 N은 배열의 크기이고, log N은 배열을 반으로 나누는 단계에서 발생하는 깊이이다. 이 문제에서는 추가적으로 K번째 저장을 찾기 위해 정렬 및 저장 과정을 모두 수행하므로, 전체 시간 복잡도는 여전히 O(N log N)으로 유지된다.

## 코드
```python
def merge_sort(A, p, r):
    if p < r:
        q = (p + r) // 2
        merge_sort(A, p, q)
        merge_sort(A, q + 1, r)
        merge(A, p, q, r)

def merge(A, p, q, r):
    global cnt, ans
    
    i = p
    j = q + 1
    t = 0
    tmp = [0] * (r - p + 1)
    
    while i <= q and j <= r:
        if A[i] <= A[j]:
            tmp[t] = A[i]
            i += 1
        else:
            tmp[t] = A[j]
            j += 1
        t += 1
        
    while i <= q:
        tmp[t] = A[i]
        i += 1
        t += 1
        
    while j <= r:
        tmp[t] = A[j]
        j += 1
        t += 1
        
    i = p
    t = 0
    
    while i <= r:
        A[i] = tmp[t]
        cnt += 1
        
        if cnt == k:
            ans = A[i]
            
        i += 1
        t += 1

n, k = map(int, input().split())
arr = list(map(int, input().split()))

cnt = 0
ans = -1

merge_sort(arr, 0, n - 1)

print(ans)
```
