## 풀이

### 풀이 핵심 로직
주어진 문제는 정수를 입력받고 이를 정렬하여 출력하는 것이다. 핵심 아이디어는 리스트를 사용하여 입력된 정수를 저장한 후, Python의 내장 `sort()` 메서드를 이용하여 이를 정렬하고 출력하는 것이다.

### 동작 과정
1. 첫 번째 줄에서 정수 `n`을 입력받는다. 이는 입력할 정수의 개수를 나타낸다.
   - 예시 입력: `5`
   
2. 그 다음 `n`개의 정수를 입력받아 리스트 `arr`에 저장한다.
   - 예시 입력: `5`, `3`, `8`, `1`, `2`
   - 리스트 상태: `arr = [5, 3, 8, 1, 2]`
   
3. `arr.sort()`를 호출하여 리스트를 오름차순으로 정렬한다.
   - 정렬 후 리스트 상태: `arr = [1, 2, 3, 5, 8]`
   
4. 마지막으로 정렬된 리스트의 각 요소를 한 줄씩 출력한다.
   - 출력 결과:
     ```
     1
     2
     3
     5
     8
     ```

### 시간 복잡도
입력된 정수의 개수를 `n`이라고 할 때, 정렬 알고리즘의 시간 복잡도는 O(n log n)이다. 따라서 이 코드의 전체 시간 복잡도는 O(n log n)이다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
