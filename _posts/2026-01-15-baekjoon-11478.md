## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
문자열 S의 서로 다른 부분 문자열을 구하기 위해, 모든 가능한 부분 문자열을 생성하고 이를 집합(set)에 추가하여 중복을 제거합니다. 이렇게 하면 최종적으로 서로 다른 부분 문자열의 개수를 쉽게 세어볼 수 있습니다.

### 동작 과정
1. 입력 문자열 S가 주어집니다. 예를 들어, S = "ababc"라고 가정합니다.
2. 부분 문자열의 길이를 1부터 S의 길이까지 증가시키며 반복합니다.
3. 각 길이에 대해, 문자열 S의 모든 시작 위치에서 부분 문자열을 추출하여 집합에 추가합니다.
   - 예를 들어, 길이가 1일 경우: `{'a', 'b', 'c'}`
   - 길이가 2일 경우: `{'ab', 'ba', 'bc'}`
   - 길이가 3일 경우: `{'aba', 'bab', 'abc'}` 
   - 길이가 4일 경우: `{'abab', 'babc'}`
   - 길이가 5일 경우: `{'ababc'}`
4. 최종적으로 집합에 저장된 서로 다른 부분 문자열의 개수를 세어 출력합니다. 이 경우, 서로 다른 부분 문자열의 개수는 12개입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. n은 문자열 S의 길이로, 각 길이에 대해 최대 n개의 시작 위치에서 부분 문자열을 추출하기 때문입니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
