## 문제
정휘는 후배들이 재귀 함수를 잘 다루는 재귀의 귀재인지 알아보기 위해 재귀 함수와 관련된 문제를 출제하기로 했다.

팰린드롬이란, 앞에서부터 읽었을 때와 뒤에서부터 읽었을 때가 같은 문자열을 말한다. 팰린드롬의 예시로 AAA, ABBA, ABABA 등이 있고, 팰린드롬이 아닌 문자열의 예시로 ABCA, PALINDROME 등이 있다.

어떤 문자열이 팰린드롬인지 판별하는 문제는 재귀 함수를 이용해 쉽게 해결할 수 있다. 아래 코드의 isPalindrome 함수는 주어진 문자열이 팰린드롬이면 1, 팰린드롬이 아니면 0을 반환하는 함수다.

#include

## 입력
첫째 줄에 테스트케이스의 개수 T$T$가 주어진다. (1≤T≤1000$1 \leq T \leq 1\,000$)

둘째 줄부터 T$T$개의 줄에 알파벳 대문자로 구성된 문자열 S$S$가 주어진다. (1≤|S|≤1000$1 \leq \vert S\vert \leq 1\,000$)

---

## 출력
각 테스트케이스마다, isPalindrome 함수의 반환값과 recursion 함수의 호출 횟수를 한 줄에 공백으로 구분하여 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 문자열이 팰린드롬인지 판단하는 재귀 함수를 구현하는 것이다. 팰린드롬은 양쪽 끝에서부터 문자를 비교하여, 모든 문자가 일치하는지 확인함으로써 판단할 수 있다. 이 과정에서 재귀 호출을 통해 문자열의 해당 인덱스들을 계속해서 비교한다.

### 동작 과정
예를 들어, 입력 문자열이 "ABBA"인 경우의 동작 과정을 살펴보자.

1. `isPalindrome("ABBA")` 호출
   - `recursion("ABBA", 0, 3, 1)` 호출 (l=0, r=3, cnt=1)
   - 비교: `s[0]` ("A") == `s[3]` ("A") → 일치, 재귀 호출
2. `recursion("ABBA", 1, 2, 2)` 호출 (l=1, r=2, cnt=2)
   - 비교: `s[1]` ("B") == `s[2]` ("B") → 일치, 재귀 호출
3. `recursion("ABBA", 2, 1, 3)` 호출 (l=2, r=1, cnt=3)
   - l(2) >= r(1) → 종료 조건 만족, 1과 호출 횟수 3 반환

따라서, "ABBA"는 팰린드롬이며, 반환값은 (1, 3)이다.

반면, 입력 문자열이 "ABC"인 경우는 다음과 같이 진행된다.

1. `isPalindrome("ABC")` 호출
   - `recursion("ABC", 0, 2, 1)` 호출 (l=0, r=2, cnt=1)
   - 비교: `s[0]` ("A") != `s[2]` ("C") → 불일치, 0과 호출 횟수 1 반환

결과적으로 "ABC"는 팰린드롬이 아니며, 반환값은 (0, 1)이다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)이다. 팰린드롬 검사는 문자열의 길이에 비례하여 문자를 한 번씩 비교하기 때문에, 최악의 경우 모든 문자에 대해 비교를 수행하게 된다.

## 코드
```python
def recursion(s, l, r, cnt):
    if l >= r:
        return 1, cnt
    elif s[l] != s[r]:
        return 0, cnt
    else:
        return recursion(s, l + 1, r - 1, cnt + 1)

def isPalindrome(s):
    return recursion(s, 0, len(s) - 1, 1)

t = int(input())
for _ in range(t):
    s = input().rstrip()
    print(*isPalindrome(s))

```
