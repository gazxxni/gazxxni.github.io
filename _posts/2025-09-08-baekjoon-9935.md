[https://www.acmicpc.net/problem/9935](https://www.acmicpc.net/problem/9935)

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 스택을 사용하여 문자열을 처리하는 것입니다. 주어진 문자열에서 연속적으로 발생하는 폭탄 문자열을 찾아 제거하는 방식으로, 문자열을 한 글자씩 스택에 추가하며 마지막 부분에서 폭탄 문자열이 있는지 확인합니다. 만약 폭탄 문자열이 발견되면 해당 부분을 스택에서 제거합니다.

### 동작 과정
1. 입력으로 주어진 문자열 `s`와 폭탄 문자열 `bomb`을 읽습니다.
2. `s`의 각 문자를 하나씩 스택에 추가합니다.
3. 스택의 마지막 문자가 폭탄 문자열의 마지막 문자와 같을 때, 스택의 마지막 `bomb_len` 만큼의 문자를 잘라내어 폭탄 문자열과 비교합니다.
4. 만약 스택의 마지막 `bomb_len` 길이의 문자열이 폭탄 문자열과 같다면 해당 부분을 스택에서 제거합니다.
5. 모든 문자를 처리한 후, 스택에 남아있는 문자열을 출력합니다. 만약 스택이 비어있다면 'FRULA'를 출력합니다.

#### 예시 입력
```
mirkov
bomb
```
- 초기 상태: `stack = []`
- `'m'` 추가 -> `stack = ['m']`
- `'i'` 추가 -> `stack = ['m', 'i']`
- `'r'` 추가 -> `stack = ['m', 'i', 'r']`
- `'k'` 추가 -> `stack = ['m', 'i', 'r', 'k']`
- `'o'` 추가 -> `stack = ['m', 'i', 'r', 'k', 'o']`
- `'v'` 추가 -> `stack = ['m', 'i', 'r', 'k', 'o', 'v']`
- 최종적으로 스택에 남은 문자열: `'mirkov'`, 폭탄 문자열이 없으므로 출력: `mirkov`

#### 예시 입력
```
mirkovbomb
bomb
```
- 초기 상태: `stack = []`
- `'m', 'i', 'r', 'k', 'o', 'v', 'b', 'o', 'm', 'b'`를 순서대로 추가.
- 마지막 부분에서 폭탄 문자열 `'bomb'`이 발견되어 제거.
- 최종적으로 스택에 남은 문자열: `'mirkov'`, 출력: `mirkov`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 여기서 n은 입력 문자열의 길이입니다. 각 문자를 스택에 한 번 추가하고, 폭탄 문자열을 확인하기 위해서는 최대 `bomb_len` 만큼의 비교를 수행하기 때문에, 전체적으로 선형 시간 복잡도로 처리됩니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

s = input().strip()
bomb = input().strip()
bomb_len = len(bomb)
stack = []

for i in range(len(s)):
    stack.append(s[i])
    if s[i] == bomb[-1] and ''.join(stack[-bomb_len:]) == bomb:
        del stack[-bomb_len:]

print(''.join(stack) if stack else 'FRULA')

```
