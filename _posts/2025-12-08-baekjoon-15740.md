## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 A와 B (-1010000 ≤ A, B ≤ 1010000)가 주어진다.

---

## 출력
첫째 줄에 A+B를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 96 ms

---

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 큰 정수 A와 B를 입력받아 그 합을 출력하는 기본적인 수학 문제입니다. Python은 내장된 `int` 자료형이 임의의 정밀도를 지원하므로, 매우 큰 숫자도 문제없이 처리할 수 있습니다. 따라서 입력받은 값을 정수로 변환한 뒤, 단순히 더한 결과를 출력하면 됩니다.

### 동작 과정
1. 입력을 받기 위해 `sys.stdin.readline`을 사용하여 한 줄의 문자열을 읽습니다.
2. 입력된 문자열을 공백을 기준으로 분리한 후, `map(int, ...)`을 사용하여 두 개의 정수 A와 B로 변환합니다.
3. A와 B를 더한 후, 결과를 출력합니다.

예시 입력:
```
123456789012345678901234567890 987654321098765432109876543210
```
단계별 설명:
- `input()`을 통해 문자열 `"123456789012345678901234567890 987654321098765432109876543210"`를 읽습니다.
- 이 문자열을 공백으로 나누어 두 개의 부분 `"123456789012345678901234567890"`와 `"987654321098765432109876543210"`로 분리합니다.
- `map(int, ...)`을 통해 두 문자열을 정수로 변환하여 A와 B에 각각 할당합니다.
- A + B를 계산하여 결과인 `1111111110111111111011111111100`를 출력합니다.

### 시간 복잡도
O(1) - 입력되는 두 숫자의 크기에 관계없이, 고정된 수의 덧셈 연산만 수행하므로 시간 복잡도는 상수 시간입니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a + b)
```
