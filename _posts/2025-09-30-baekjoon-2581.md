## 문제
자연수 M과 N이 주어질 때 M이상 N이하의 자연수 중 소수인 것을 모두 골라 이들 소수의 합과 최솟값을 찾는 프로그램을 작성하시오.

예를 들어 M=60, N=100인 경우 60이상 100이하의 자연수 중 소수는 61, 67, 71, 73, 79, 83, 89, 97 총 8개가 있으므로, 이들 소수의 합은 620이고, 최솟값은 61이 된다.

---

## 입력
입력의 첫째 줄에 M이, 둘째 줄에 N이 주어진다.

M과 N은 10,000이하의 자연수이며, M은 N보다 작거나 같다.

---

## 출력
M이상 N이하의 자연수 중 소수인 것을 모두 찾아 첫째 줄에 그 합을, 둘째 줄에 그 중 최솟값을 출력한다. 

단, M이상 N이하의 자연수 중 소수가 없을 경우는 첫째 줄에 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 범위 내의 소수를 찾기 위해 에라토스테네스의 체를 사용하여 소수를 효율적으로 판별하고, M 이상 N 이하의 소수를 모두 찾아 합과 최솟값을 계산하는 방식으로 해결합니다. 소수를 찾은 후, 해당 소수가 존재하는지 확인하고, 존재할 경우 합과 최솟값을 출력합니다.

### 동작 과정
1. **입력 받기**: 두 자연수 M과 N을 입력받습니다.
   - 예시: M = 60, N = 100
2. **소수 판별 배열 초기화**: 길이 10001의 배열을 생성하여 (0~10000) 모든 수를 소수로 초기화합니다. 0과 1은 소수가 아니므로 False로 설정합니다.
   - 배열 상태: `arr[0] = False`, `arr[1] = False`, `arr[2] = True`, ..., `arr[10000] = True`
3. **에라토스테네스의 체**: 2부터 N의 제곱근까지의 수를 반복하며, 해당 수가 소수일 경우 그 배수를 False로 설정하여 소수를 판별합니다.
   - 예시: 2, 3, 5 등 소수를 만나면 그 배수들을 False로 설정.
4. **소수 찾기**: M부터 N까지의 범위에서 소수를 찾아 합을 계산하고 최솟값을 찾습니다.
   - 예시: M=60, N=100 범위에서 소수는 61, 67, 71, 73, 79, 83, 89, 97로 확인.
5. **결과 출력**: 소수가 없으면 -1을 출력하고, 소수가 존재하면 합과 최솟값을 출력합니다.
   - 예시 출력: 
     ```
     620
     61
     ```

### 시간 복잡도
이 알고리즘의 주요 부분인 에라토스테네스의 체는 O(N log log N) 시간 복잡도를 가지며, 소수를 찾고 합과 최솟값을 계산하는 과정은 O(N-M)입니다. 따라서 전체 시간 복잡도는 O(N log log N)으로, 주어진 입력 범위 내에서 매우 효율적으로 동작합니다.

## 코드
```python
m = int(input())
n = int(input())

arr = [True] * 10001
arr[0], arr[1] = False, False

for i in range(2, int(n ** 0.5) + 1): 
    if arr[i]:
        for j in range(i * i, n + 1, i):
            arr[j] = False
             
cnt = 0
ans = 0
for i in range(m, n + 1):
    if arr[i]:
        cnt += i
        if ans == 0:
            ans = i

if ans == 0:
    print(-1)
else:
    print(cnt)
    print(ans)
```
