## 풀이

### 풀이 핵심 로직
이 문제는 2xN 크기의 타일을 1x2와 2x1 크기의 타일로 채우는 방법의 수를 구하는 문제입니다. 점화식은 `dp[n] = dp[n-1] + 2 * dp[n-2]`로 설정되어 있으며, 이는 n-1 크기의 경우에 1x2 타일을 추가하거나 n-2 크기의 경우에 2x1 타일 두 개를 추가하는 경우를 고려합니다. 최종 결과는 10007로 나눈 나머지를 출력합니다.

### 동작 과정
1. 입력으로 n을 받아옵니다.
2. dp 배열을 생성하고, dp[1]과 dp[2]의 값을 초기화합니다.
   - dp[1] = 1 (1x2 타일로만 채우는 경우)
   - dp[2] = 3 (1x2 타일 2개 또는 2x1 타일 1개 두 개로 채우는 경우)
3. for 루프를 통해 3부터 1000까지의 dp 값을 계산합니다:
   - dp[3] = dp[2] + 2 * dp[1] = 3 + 2 * 1 = 5
   - dp[4] = dp[3] + 2 * dp[2] = 5 + 2 * 3 = 11
   - dp[5] = dp[4] + 2 * dp[3] = 11 + 2 * 5 = 21
   - ...
4. n번째 dp 값을 구한 후, 10007로 나눈 나머지를 출력합니다.

예를 들어, n이 4인 경우:
- dp[1] = 1
- dp[2] = 3
- dp[3] = 5 (dp[2] + 2 * dp[1])
- dp[4] = 11 (dp[3] + 2 * dp[2])
- 결과 출력: 11 % 10007 = 11

### 시간 복잡도
O(n) - n에 대해 선형적으로 dp 배열을 계산하므로 시간 복잡도는 O(n)입니다.

## 코드
```python
import sys

n=int(input())

arr = [0] * 1001
arr[1] = 1
arr[2] = 3

for i in range(3,1001):
    arr[i] = arr[i-1] + 2*arr[i-2]

print(arr[n]%10007)


```
