## 문제
자연수 x를 y로 변환하려고 합니다. 사용할 수 있는 연산은 다음과 같습니다.

x에 3을 곱합니다.

## 풀이
### 풀이 핵심 로직
이 문제는 BFS(너비 우선 탐색)를 사용하여 자연수 x에서 y로 변환하는 최소 연산 횟수를 찾는 것입니다. x에 n을 더하거나, 2 또는 3을 곱하는 세 가지 연산을 통해 y에 도달할 수 있는 경로를 탐색하면서, 가장 적은 연산 횟수를 계산합니다.

### 동작 과정
1. **초기화**: BFS를 위한 큐를 생성하여 시작점인 x와 현재 연산 횟수를 0으로 설정합니다. 또한, 방문한 숫자를 기록할 집합을 초기화합니다.
   
2. **반복 탐색**:
   - 큐에서 현재 숫자와 연산 횟수를 꺼냅니다.
   - 현재 숫자가 목표 숫자 y와 같으면, 현재 연산 횟수를 반환합니다.
   - 현재 숫자에서 가능한 다음 숫자(현재 숫자 + n, 현재 숫자 * 2, 현재 숫자 * 3)를 계산합니다.
   - 이 다음 숫자가 목표 숫자 y보다 작거나 같고, 방문하지 않은 숫자라면 큐에 추가하고 방문 기록에 추가합니다.
   
3. **종료 조건**: 큐가 비어도 목표 숫자 y에 도달하지 못한 경우 -1을 반환합니다.

**예시 입력에 대한 단계별 설명**:
- 입력: `x = 10, y = 40, n = 5`
  - 시작: 큐에 (10, 0) 추가
  - (10, 0)에서 다음 숫자로 (15, 1), (20, 1), (30, 1) 추가
  - (15, 1)에서 다음 숫자로 (20, 2), (30, 2), (45, 2) 추가 (20은 이미 방문했으므로 추가하지 않음)
  - (20, 1)에서 다음 숫자로 (25, 2), (40, 2) 추가 (40이 목표값이므로 2를 반환)
  
이 과정에서 최소 연산 횟수인 2를 찾게 됩니다.

### 시간 복잡도
BFS는 각 정점(숫자)을 한 번만 방문하기 때문에 최악의 경우 O(V + E)의 시간 복잡도를 가집니다. 여기서 V는 가능한 숫자의 개수, E는 각 숫자에서 가능한 다음 숫자의 개수입니다. 문제의 조건에서 y는 최대 1,000,000이므로, BFS 탐색의 시간 복잡도는 O(y)로 볼 수 있습니다.

## 코드
```python
from collections import deque

def bfs(start, target, n):
    q = deque([(start, 0)])
    visited = {start}

    while q:
        cur, cnt = q.popleft()
        
        if cur == target:
            return cnt
        
        for nex in [cur + n, cur * 2, cur * 3]:
            if nex <= target and nex not in visited:
                q.append((nex, cnt+1))
                visited.add(nex)

    return -1
    
def solution(x, y, n):
    return bfs(x, y, n)
```
