## 문제
세 점이 주어졌을 때, 축에 평행한 직사각형을 만들기 위해서 필요한 네 번째 점을 찾는 프로그램을 작성하시오.

---

## 입력
세 점의 좌표가 한 줄에 하나씩 주어진다. 좌표는 1보다 크거나 같고, 1000보다 작거나 같은 정수이다.

---

## 출력
직사각형의 네 번째 점의 좌표를 출력한다.

---

## 성능 요약
메모리: 110064 KB, 시간: 104 ms

---

## 풀이

### 풀이 핵심 로직
문제에서 주어진 세 점을 이용해 축에 평행한 직사각형의 네 번째 점의 좌표를 찾아야 합니다. 주어진 세 점의 x좌표와 y좌표를 각각 카운트하여, 홀수 개의 좌표가 남는 x와 y 값을 찾으면 됩니다. 홀수 개의 좌표는 네 번째 점의 좌표가 됩니다.

### 동작 과정
1. 세 점의 좌표를 입력으로 받습니다. 예를 들어, 입력이 다음과 같다고 합시다:
   ```
   1 1
   2 2
   1 2
   ```
   
2. 각 x좌표와 y좌표의 출현 횟수를 기록합니다. 위의 예시에서는 다음과 같이 카운트됩니다:
   - x좌표: {1: 2, 2: 1}
   - y좌표: {1: 1, 2: 2}

3. 이제 x좌표에서 출현 횟수가 홀수인 값을 찾습니다. 위의 경우 x좌표 2가 홀수이므로, x = 2가 됩니다.
4. y좌표에서도 같은 방식으로 홀수인 값을 찾습니다. y좌표 1이 홀수이므로, y = 1이 됩니다.

5. 최종적으로 찾은 네 번째 점의 좌표는 (2, 1)로 출력됩니다.

### 시간 복잡도
이 알고리즘은 각 점을 입력받아 두 개의 딕셔너리에 각각 x좌표와 y좌표를 카운트하는 과정이므로, 입력 점의 수가 3으로 고정되어 있습니다. 따라서, 시간 복잡도는 O(1)입니다.

---

## 코드
```python
from collections import defaultdict
dic_x = defaultdict(int)
dic_y = defaultdict(int)

for _ in range(3):
    a, b = map(int, input().split())
    
    dic_x[a] += 1
    dic_y[b] += 1
    
for k, v in dic_x.items():
    if v % 2 == 1:
        x = k
for k, v in dic_y.items():
    if v % 2 == 1:
        y = k
           
print(x, y)
```
