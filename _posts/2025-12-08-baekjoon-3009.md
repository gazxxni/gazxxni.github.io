## 문제
세 점이 주어졌을 때, 축에 평행한 직사각형을 만들기 위해서 필요한 네 번째 점을 찾는 프로그램을 작성하시오.

---

## 입력
세 점의 좌표가 한 줄에 하나씩 주어진다. 좌표는 1보다 크거나 같고, 1000보다 작거나 같은 정수이다.

---

## 출력
직사각형의 네 번째 점의 좌표를 출력한다.

---

## 성능 요약
메모리: 110064 KB, 시간: 104 ms

---

## 풀이

### 풀이 핵심 로직
주어진 세 점의 x좌표와 y좌표 각각의 빈도를 세어, 짝수가 아닌 좌표를 찾아서 직사각형의 네 번째 점을 결정합니다. 세 점 중에서 각 좌표가 홀수 번 등장하는 좌표가 직사각형의 네 번째 점이 됩니다.

### 동작 과정
1. 입력으로 주어진 세 점의 좌표를 읽어옵니다.
2. `dic_x`와 `dic_y`라는 두 개의 딕셔너리를 생성하여 각 x좌표와 y좌표의 등장 횟수를 기록합니다.
3. 첫 번째 점 (1, 2)을 읽어와서 `dic_x`와 `dic_y`에 각각 1과 2의 카운트를 1 증가시킵니다.
4. 두 번째 점 (1, 3)을 읽어와서 `dic_x`의 1을 2로, `dic_y`의 3을 1로 증가시킵니다.
5. 세 번째 점 (2, 3)을 읽어와서 `dic_x`의 2를 1로, `dic_y`의 3을 2로 증가시킵니다.
6. `dic_x`에서 등장 횟수가 홀수인 x좌표 (여기서는 2)를 찾고, `dic_y`에서 등장 횟수가 홀수인 y좌표 (여기서는 2)를 찾습니다.
7. 따라서, 네 번째 점의 좌표 (2, 3)를 출력합니다.

입력 예시:
```
1 2
1 3
2 3
```
출력:
```
2 2
```

### 시간 복잡도
O(1) - 입력 점의 개수가 항상 3개로 고정되어 있으므로, 수행 시간은 상수 시간에 해당합니다.

---

## 코드
```python
from collections import defaultdict
dic_x = defaultdict(int)
dic_y = defaultdict(int)

for _ in range(3):
    a, b = map(int, input().split())
    
    dic_x[a] += 1
    dic_y[b] += 1
    
for k, v in dic_x.items():
    if v % 2 == 1:
        x = k
for k, v in dic_y.items():
    if v % 2 == 1:
        y = k
           
print(x, y)
```
