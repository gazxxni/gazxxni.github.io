## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 자연수 N의 각 자리수를 내림차순으로 정렬하기 위해, 먼저 N을 문자열로 입력받고 각 자리수를 리스트로 변환합니다. 그 후, 리스트를 내림차순으로 정렬하고, 정렬된 리스트를 다시 문자열로 변환하여 출력합니다.

### 동작 과정
1. 입력 예시: `2143`
2. `input().strip()`를 통해 입력받은 문자열 `N`을 `2143`으로 얻습니다.
3. `map(int, n)`를 통해 각 자리수를 정수로 변환하여 리스트로 만들면 `[2, 1, 4, 3]`이 됩니다.
4. `sort(reverse=True)`를 통해 리스트를 내림차순으로 정렬하면 `[4, 3, 2, 1]`이 됩니다.
5. `print(*a, sep='')`를 통해 리스트의 요소를 공백 없이 출력하면 최종 결과 `4321`이 됩니다.

### 시간 복잡도
입력된 숫자의 자리수 N에 대해 O(d log d)의 시간 복잡도를 가집니다. 여기서 d는 N의 자리수입니다. 정렬 과정이 가장 시간이 많이 걸리므로, d는 최대 10 (1,000,000,000까지)입니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
