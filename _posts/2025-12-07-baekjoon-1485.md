## 문제
네 점이 주어졌을 때, 네 점을 이용해 정사각형을 만들 수 있는지 없는지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 네 줄로 이루어져 있으며, 점의 좌표가 한 줄에 하나씩 주어진다. 점의 좌표는 -100,000보다 크거나 같고, 100,000보다 작거나 같은 정수이다. 같은 점이 두 번 이상 주어지지 않는다.

---

## 출력
각 테스트 케이스마다 입력으로 주어진 네 점을 이용해서 정사각형을 만들 수 있으면 1을, 없으면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
정사각형의 성질을 이용하여 문제를 해결합니다. 네 점이 정사각형을 이루려면, 점들 간의 거리가 특정한 패턴을 따라야 합니다. 즉, 4개의 변의 길이가 동일하고, 2개의 대각선의 길이도 동일해야 하며, 대각선의 길이는 변의 길이보다 길어야 합니다. 이를 위해 각 점들 간의 거리의 제곱을 계산하고, 거리의 빈도를 통해 정사각형 여부를 판단합니다.

### 동작 과정
1. 입력으로 주어진 테스트 케이스의 수 T를 읽습니다.
2. 각 테스트 케이스마다 4개의 점을 리스트에 저장합니다.
3. 두 점 간의 거리 제곱을 계산하여 딕셔너리에 저장합니다. 이 때, 동일한 거리 제곱이 몇 번 발생하는지를 카운트합니다.
4. 딕셔너리를 순회하면서, 거리가 8번(정사각형의 변) 발생한 경우와 4번(정사각형의 대각선) 발생한 경우를 찾습니다.
5. 두 조건이 모두 만족되면 1을 출력하고, 그렇지 않으면 0을 출력합니다. 

예시 입력:
```
1
0 0
0 1
1 0
1 1
```
- 거리 제곱 계산: 
  - (0,0)과 (0,1) -> 1
  - (0,0)과 (1,0) -> 1
  - (0,0)과 (1,1) -> 2
  - (0,1)과 (1,0) -> 2
  - (0,1)과 (1,1) -> 1
  - (1,0)과 (1,1) -> 1
- 최종적으로 거리 제곱의 빈도는 {1: 8, 2: 2} 이므로, 8번의 변과 4번의 대각선이 조건을 만족하여 정사각형을 만들 수 있습니다. 따라서 출력은 1입니다.

### 시간 복잡도
O(1) - 입력 점의 수가 항상 4개로 고정되어 있으므로, 거리를 계산하는 데 필요한 연산 수는 상수입니다.

## 코드
```python
from collections import defaultdict
t = int(input())

for _ in range(t):
    arr = []
    
    for i in range(4):
        x, y = map(int, input().split())
        arr.append((x, y))
    
    dic = defaultdict(int)
    for i in range(4):
        for j in range(4):
            if i != j:
                a = (arr[i][0] - arr[j][0]) ** 2 + (arr[i][1] - arr[j][1]) ** 2
                dic[a] += 1
    
    aa = False
    bb = False  
    for k, v in dic.items():
        if v == 8:
            aa = True
        
        elif v == 4:
            bb = True
            
    if aa and bb:
        print(1)
        
    else:
        print(0)
```
