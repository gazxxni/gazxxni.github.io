## 문제
네 점이 주어졌을 때, 네 점을 이용해 정사각형을 만들 수 있는지 없는지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 네 줄로 이루어져 있으며, 점의 좌표가 한 줄에 하나씩 주어진다. 점의 좌표는 -100,000보다 크거나 같고, 100,000보다 작거나 같은 정수이다. 같은 점이 두 번 이상 주어지지 않는다.

---

## 출력
각 테스트 케이스마다 입력으로 주어진 네 점을 이용해서 정사각형을 만들 수 있으면 1을, 없으면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
정사각형을 만들기 위해서는 네 점 사이의 거리 관계가 중요합니다. 네 점에서 서로 다른 두 종류의 거리를 확인하여, 한 종류는 4개(변의 길이)이고 다른 종류는 2개(대각선의 길이)여야 합니다. 이때 변의 길이는 동일하고 대각선의 길이는 변의 길이의 √2배가 되어야 합니다.

### 동작 과정
1. 입력으로 주어진 테스트 케이스 수 만큼 반복합니다.
2. 각 테스트 케이스에서 네 점의 좌표를 리스트에 저장합니다.
3. 두 점 사이의 거리의 제곱을 계산하여, 각 거리의 발생 횟수를 딕셔너리에 저장합니다.
4. 발생 횟수에 따라 두 가지 조건을 확인합니다:
   - 4개의 변의 길이(발생 횟수 4)
   - 2개의 대각선의 길이(발생 횟수 2)
5. 조건이 만족하면 1을 출력하고, 그렇지 않으면 0을 출력합니다.

예시 입력:
```
2
0 0
1 0
1 1
0 1
0 0
1 0
1 2
0 2
```

단계별 설명:
- 첫 번째 테스트 케이스에서 입력한 네 점은 정사각형을 이루므로, 각 점 사이의 거리 계산 결과 4개의 변과 2개의 대각선이 확인됩니다.
- 두 번째 테스트 케이스에서는 네 점이 정사각형이 아니므로, 조건을 만족하지 않아 0을 출력합니다.

### 시간 복잡도
이 알고리즘은 각 테스트 케이스마다 점들 간의 거리를 계산하기 위해 O(1) 시간 복잡도를 가진 6개의 거리 계산을 수행합니다. 전체적으로 T개의 테스트 케이스가 주어지므로, 전체 시간 복잡도는 O(T)입니다.

## 코드
```python
from collections import defaultdict
t = int(input())

for _ in range(t):
    arr = []
    
    for i in range(4):
        x, y = map(int, input().split())
        arr.append((x, y))
    
    dic = defaultdict(int)
    for i in range(4):
        for j in range(4):
            if i != j:
                a = (arr[i][0] - arr[j][0]) ** 2 + (arr[i][1] - arr[j][1]) ** 2
                dic[a] += 1
    
    aa = False
    bb = False  
    for k, v in dic.items():
        if v == 8:
            aa = True
        
        elif v == 4:
            bb = True
            
    if aa and bb:
        print(1)
        
    else:
        print(0)
```
