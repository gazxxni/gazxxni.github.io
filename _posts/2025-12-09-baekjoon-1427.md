## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 자연수 N의 각 자리수를 분리하여 리스트로 만든 후, 이 리스트를 내림차순으로 정렬합니다. 정렬된 리스트의 원소를 다시 합쳐서 최종 결과를 출력합니다.

### 동작 과정
1. 입력으로 주어진 자연수 N을 문자열로 읽어옵니다.
2. 각 자리수를 정수로 변환하여 리스트에 저장합니다. 예를 들어, 입력이 `2143`이라면 리스트는 `[2, 1, 4, 3]`가 됩니다.
3. 리스트를 내림차순으로 정렬합니다. 이 경우 `[4, 3, 2, 1]`로 정렬됩니다.
4. 정렬된 리스트의 원소를 공백 없이 이어붙여서 하나의 문자열로 만듭니다. 최종 출력은 `4321`이 됩니다.

예시 입력: `2143`
- 리스트 변환: `[2, 1, 4, 3]`
- 정렬: `[4, 3, 2, 1]`
- 출력: `4321`

### 시간 복잡도
정렬 알고리즘의 시간 복잡도는 O(n log n)입니다. 여기서 n은 입력된 수의 자리수(최대 10자리)입니다. 따라서 최악의 경우에도 이 문제는 O(10 log 10)로 매우 효율적으로 처리됩니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
