## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수를 소수의 곱으로 표현하는 과정이다. 주어진 정수 N을 2부터 시작하여, 각 후보 소수로 나누어가며 소인수를 찾는다. 이 과정은 N이 소수보다 작아질 때까지 반복되며, 최종적으로 남은 N이 소수일 경우 이를 출력한다.

### 동작 과정
입력으로 정수 N이 주어진다고 가정하자. 예를 들어, N = 60 일 때, 소인수분해 과정을 단계별로 설명하면 다음과 같다:

1. **N = 60**: 먼저 2로 나누어보면, 60 % 2 == 0 이므로 2를 출력하고 N을 30으로 갱신한다.
   - 출력: `2`
2. **N = 30**: 다시 2로 나누어보면, 30 % 2 == 0 이므로 2를 출력하고 N을 15로 갱신한다.
   - 출력: `2`
3. **N = 15**: 2로는 나누어지지 않으므로, 3으로 나누어보자. 15 % 3 == 0 이므로 3을 출력하고 N을 5로 갱신한다.
   - 출력: `3`
4. **N = 5**: 3으로는 나누어지지 않지만, N이 5로 남았으므로 5는 소수이다. 따라서 5를 출력한다.
   - 출력: `5`

최종 출력은 다음과 같다:
```
2
2
3
5
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(√N)이다. 이는 N을 나누는 과정에서 각 소수 후보를 2부터 √N까지 시도하기 때문이며, 각 시도에 대해 최대 O(log N) 번의 나누기를 수행하게 된다.

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
