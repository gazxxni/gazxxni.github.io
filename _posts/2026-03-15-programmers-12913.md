## 문제
땅따먹기 게임을 하려고 합니다. 땅따먹기 게임의 땅(land)은 총 N행 4열로 이루어져 있고, 모든 칸에는 점수가 쓰여 있습니다. 1행부터 땅을 밟으며 한 행씩 내려올 때, 각 행의 4칸 중 한 칸만 밟으면서 내려와야 합니다. 단, 땅따먹기 게임에는 한 행씩 내려올 때, 같은 열을 연속해서 밟을 수 없는 특수 규칙이 있습니다. 

예를 들면, 

| 1 | 2 | 3 | 5 |

| 5 | 6 | 7 | 8 |

| 4 | 3 | 2 | 1 |

로 땅이 주어졌다면, 1행에서 네번째 칸 (5)를 밟았으면, 2행의 네번째 칸 (8)은 밟을 수 없습니다. 

마지막 행까지 모두 내려왔을 때, 얻을 수 있는 점수의 최대값을 return하는 solution 함수를 완성해 주세요. 위 예의 경우, 1행의 네번째 칸 (5), 2행의 세번째 칸 (7), 3행의 첫번째 칸 (4) 땅을 밟아 16점이 최고점이 되므로 16을 return 하면 됩니다.

제한사항


행의 개수 N : 100,000 이하의 자연수
열의 개수는 4개이고, 땅(land)은 2차원 배열로 주어집니다.
점수 : 100 이하의 자연수


입출력 예

        
land
answer


        
[[1,2,3,5],[5,6,7,8],[4,3,2,1]]
16


      
입출력 예 설명

입출력 예 #1

## 풀이
### 풀이 핵심 로직
이 문제는 동적 프로그래밍을 사용하여 해결할 수 있습니다. 각 행에서 선택할 수 있는 점수는 이전 행에서 선택한 점수와 중복되지 않도록 관리해야 하므로, 현재 행의 각 열의 점수는 이전 행의 최대 점수에서 해당 열의 점수를 더한 값으로 갱신됩니다. 이를 통해 마지막 행까지 내려왔을 때의 최대 점수를 계산합니다.

### 동작 과정
1. **초기화**: 첫 번째 행의 점수를 `prev` 리스트에 복사합니다.
   - 예시 입력: `land = [[1,2,3,5],[5,6,7,8],[4,3,2,1]]`
   - `prev = [1, 2, 3, 5]`

2. **행 반복**: 두 번째 행부터 시작하여 각 행을 반복합니다. 현재 행의 점수는 이전 행의 최대 점수(동일 열 제외)를 더하여 계산합니다.
   - 2행에서
     - 1열: `5 + max(2, 3, 5) = 5 + 5 = 10`
     - 2열: `6 + max(1, 3, 5) = 6 + 5 = 11`
     - 3열: `7 + max(1, 2, 5) = 7 + 5 = 12`
     - 4열: `8 + max(1, 2, 3) = 8 + 3 = 11`
   - `cur` 리스트는 `[10, 11, 12, 11]`이 됩니다.
   - `prev`를 `cur`로 업데이트합니다.

3. **최종 행 처리**: 마지막 행을 처리하고, `prev` 리스트에 남아있는 값들 중 최대값을 반환합니다.
   - 3행에서
     - 1열: `4 + max(11, 12, 11) = 4 + 12 = 16`
     - 2열: `3 + max(10, 12, 11) = 3 + 12 = 15`
     - 3열: `2 + max(10, 11, 11) = 2 + 11 = 13`
     - 4열: `1 + max(10, 11, 12) = 1 + 12 = 13`
   - `prev` 리스트는 `[16, 15, 13, 13]`이 되고, 이 중 최대값인 16을 반환합니다.

### 시간 복잡도
- 이 알고리즘의 시간 복잡도는 O(N)입니다. 각 행에 대해 4개의 열을 처리하는데, 이 과정에서 4개의 최대값을 찾는 작업은 상수 시간 O(4)로 처리되므로 전체적으로 N행을 순회하는 O(N) 복잡도를 가집니다. 

### 최종 코드
```python
def solution(land):
    prev = land[0][:]
    
    for r in range(1, len(land)):
        cur = [0] * (len(land[0]))  # 현재 행의 점수 초기화
        
        for c in range(len(prev)):
            # 이전 행의 최대값을 제외한 채로 현재 열의 점수를 더함
            cur[c] = land[r][c] + max(prev[k] for k in range(len(prev)) if k != c)
        
        prev = cur  # 현재 행의 점수를 이전 행으로 업데이트
    
    return max(prev)  # 마지막 행의 최대 점수를 반환
```

이 코드는 주어진 문제를 효율적으로 해결하며, 각 행을 순회하면서 최대 점수를 계산합니다.
