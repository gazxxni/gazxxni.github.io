## 문제
영선이는 길이가 a, b, c인 세 막대를 가지고 있고, 각 막대의 길이를 마음대로 줄일 수 있다.

영선이는 세 막대를 이용해서 아래 조건을 만족하는 삼각형을 만들려고 한다.


	각 막대의 길이는 양의 정수이다
	세 막대를 이용해서 넓이가 양수인 삼각형을 만들 수 있어야 한다.
	삼각형의 둘레를 최대로 해야 한다.


a, b, c가 주어졌을 때, 만들 수 있는 가장 큰 둘레를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 100)가 주어진다.

---

## 출력
첫째 줄에 만들 수 있는 가장 큰 삼각형의 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 세 막대의 길이 \(a\), \(b\), \(c\)를 이용하여 삼각형을 만들기 위해서는 삼각형의 조건을 만족해야 한다. 이 조건은 두 막대의 길이의 합이 나머지 한 막대의 길이보다 커야 한다. 최댓값을 구하기 위해, 막대의 길이를 줄여가면서 가장 큰 둘레를 계산한다. 만약 \(a + b \leq c\)가 성립한다면, \(c\)를 줄여서 \(a + b - 1\)의 형태로 최대 둘레를 구할 수 있다.

### 동작 과정
1. 입력: \(a, b, c\)의 값을 읽어온다. 예를 들어, \(a = 3\), \(b = 4\), \(c = 5\)라고 가정하자.
2. 막대의 길이를 정렬하여 \(a \leq b \leq c\)로 설정한다. 위의 경우 정렬 후 \(a = 3\), \(b = 4\), \(c = 5\)가 된다.
3. 조건을 확인한다:
   - \(a + b \leq c\) (즉, \(3 + 4 \leq 5\))가 성립하지 않으므로, 다음 조건으로 넘어간다.
   - \(a = b = c\) (즉, 모든 막대가 같음)도 성립하지 않으므로, 마지막으로 일반적인 경우로 넘어가 `print(a + b + c)`를 실행하여 \(3 + 4 + 5 = 12\)를 출력한다.
4. 만약 \(a + b \leq c\)였다면, 두 막대를 줄여서 \(2(a + b) - 1\)의 형태로 출력했을 것이다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \(O(1)\)이다. 주어진 세 개의 숫자에 대해서 정렬과 조건 체크를 하기 때문에 상수 시간 내에 결과를 도출할 수 있다.

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
