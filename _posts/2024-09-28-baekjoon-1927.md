## 문제 링크
[https://www.acmicpc.net/problem/1927](https://www.acmicpc.net/problem/1927)

## 풀이

### 풀이 핵심 로직
이 문제는 "최소 힙" 자료구조를 사용하여 주어진 수를 관리하는 방식으로 해결합니다. 입력된 수가 0이 아닐 경우 해당 수를 힙에 추가하고, 0일 경우 힙에서 가장 작은 수를 제거하고 출력합니다. 힙이 비어 있을 경우 0을 출력하도록 처리합니다.

### 동작 과정
예를 들어, 입력이 다음과 같다고 가정하겠습니다:
```
5
1
2
0
3
0
0
```

1. 첫 번째 입력 `5`는 처리할 수의 개수입니다.
2. 두 번째 입력 `1`이 들어오면, 힙에 `1`이 추가됩니다. (`heap = [1]`)
3. 세 번째 입력 `2`가 들어오면, 힙에 `2`가 추가됩니다. (`heap = [1, 2]`)
4. 네 번째 입력 `0`이 들어오면, 힙에서 가장 작은 값인 `1`이 제거되고 출력됩니다. (`heap = [2]`, 출력: `1`)
5. 다섯 번째 입력 `3`이 들어오면, 힙에 `3`이 추가됩니다. (`heap = [2, 3]`)
6. 여섯 번째 입력 `0`이 들어오면, 힙에서 가장 작은 값인 `2`가 제거되고 출력됩니다. (`heap = [3]`, 출력: `2`)
7. 마지막 입력 `0`이 들어오면, 힙에서 가장 작은 값인 `3`이 제거되고 출력됩니다. (`heap = []`, 출력: `3`)
8. 다음 `0`이 들어오면, 힙이 비어 있으므로 `0`이 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. 여기서 n은 입력된 수의 개수입니다. `heapq.heappush`와 `heapq.heappop` 연산은 각각 O(log k)의 시간 복잡도를 가지며, k는 힙의 현재 크기입니다. 최악의 경우, 모든 입력이 0이 아닐 때 힙에 추가해야 하므로 O(n log n)의 복잡도를 가진다고 할 수 있습니다.

## 코드
```python
import sys, heapq

input=sys.stdin.readline

n=int(input())
heap=[]

for _ in range(n):
    arr=int(input())
    
    if arr!=0:
        heapq.heappush(heap,arr)  # 최소 힙에 숫자를 넣는 함수
    else:
        try:
            print(heapq.heappop(heap))  # 입력된 숫자가 0이면 가장 작은 값 꺼냄
        except:
            print(0)  # 힙이 비어 있는 경우 0을 출력

        

    
    

```
