## 풀이

### 풀이 핵심 로직
이 문제는 주어진 가정에 따라 최저 비용으로 기타 줄을 구매하는 방법을 결정하는 문제입니다. 주어진 패키지 가격과 개별 가격 중에서 가장 경제적인 조합을 찾아야 하며, 패키지로 사는 것이 개별로 사는 것보다 저렴할 때 적절한 계산을 통해 최소 비용을 도출합니다.

### 동작 과정
1. **입력 처리**: 첫 줄에서 필요한 기타 줄의 개수 `n`과 판매하는 가게의 수 `m`을 입력받습니다. 그 다음, 각 가게에서 패키지 및 개별 가격을 입력받아 `arr` (패키지 가격)와 `brr` (개별 가격) 리스트에 저장합니다.
   
2. **최소 가격 찾기**: `arr`와 `brr`에서 각각 최소값을 구합니다. `aa`는 패키지의 최저 가격, `bb`는 개별 가격의 최저 가격입니다.

3. **비교 및 계산**:
   - 패키지 가격(`aa`)이 개별 가격의 6배(`bb * 6`)보다 작으면 패키지를 사는 것이 유리합니다. 이 경우, 남은 개수(n % 6)에 대해 개별 가격으로 계산할지 패키지를 한 개 더 살지 결정합니다.
   - 그렇지 않은 경우, 모든 줄을 개별 가격으로 구매합니다.

예시 입력:
```
7 3
20 5
30 3
25 6
```
- `n = 7`, `m = 3`
- `arr = [20, 30, 25]`에서 `aa = 20`
- `brr = [5, 3, 6]`에서 `bb = 3`
- `aa < bb * 6` (20 < 18) 조건이 거짓이므로, 모든 줄을 개별 가격으로 구매하는 경우: `n * bb = 7 * 3 = 21`을 출력함.

### 시간 복잡도
이 코드는 `O(m)`의 시간 복잡도를 가집니다. 여기서 `m`은 가게의 수로, 각 가게에서 패키지 및 개별 가격을 입력받는 작업이 있기 때문입니다. 그 외의 연산들은 상수 시간에 수행됩니다.

---

## 코드
```python
n,m = map(int,input().split())

arr = []
brr = []

for _ in range(m):
    a, b = map(int, input().split())
    arr.append(int(a))
    brr.append(int(b))

aa = min(arr)
bb = min(brr)

if aa < bb * 6:
    if aa < (n % 6) * bb:
        print((n // 6) * aa + aa)
    else:
        print((n // 6) * aa + (n % 6) * bb)

elif aa >= bb * 6:
    print(n * bb)
```
