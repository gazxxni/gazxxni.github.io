## 문제
때는 2040년, 이민혁은 우주에 자신만의 왕국을 만들었다. 왕국은 N개의 행성으로 이루어져 있다. 민혁이는 이 행성을 효율적으로 지배하기 위해서 행성을 연결하는 터널을 만들려고 한다.


	행성은 3차원 좌표위의 한 점으로 생각하면 된다. 두 행성 A(xA, yA, zA)와 B(xB, yB, zB)를 터널로 연결할 때 드는 비용은 min(|xA-xB|, |yA-yB|, |zA-zB|)이다.


	민혁이는 터널을 총 N-1개 건설해서 모든 행성이 서로 연결되게 하려고 한다. 이때, 모든 행성을 터널로 연결하는데 필요한 최소 비용을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 행성의 개수 N이 주어진다. (1 ≤ N ≤ 100,000) 다음 N개 줄에는 각 행성의 x, y, z좌표가 주어진다. 좌표는 -109보다 크거나 같고, 109보다 작거나 같은 정수이다. 한 위치에 행성이 두 개 이상 있는 경우는 없다.

---

## 출력
첫째 줄에 모든 행성을 터널로 연결하는데 필요한 최소 비용을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 각 행성을 3차원 좌표계에서 연결하기 위해 최소 비용의 터널을 만드는 것입니다. 두 행성 간의 터널 비용은 x, y, z 좌표의 차이 중 최소값으로 정의되므로, 각 축별로 정렬하여 인접한 행성들 간의 비용을 계산하고 이를 간선으로 만들어 최소 스패닝 트리를 구성하는 방법을 사용합니다.

### 동작 과정
1. **입력 받기**: 먼저 행성의 개수 N을 입력받고, 각 행성의 3차원 좌표를 리스트에 저장합니다.
   
2. **정렬하여 간선 생성**: 
   - x, y, z 좌표 각각에 대해 행성들을 정렬합니다.
   - 정렬된 리스트에서 인접한 행성 쌍 (i-1, i) 사이의 터널 비용을 계산하여 간선 리스트에 추가합니다.
   - 예를 들어, 행성들이 (1, 2, 3), (3, 2, 1), (6, 5, 4)으로 주어졌다면, x좌표 기준으로 정렬 후 (1, 2), (2, 3) 간선의 비용을 각각 계산하여 간선 리스트에 추가합니다. 이 과정을 y와 z 좌표에서도 반복합니다.

3. **간선 정렬**: 모든 간선이 생성된 후, 비용을 기준으로 간선 리스트를 정렬합니다.

4. **크루스칼 알고리즘 적용**: 간선 리스트를 순회하며, 사이클을 형성하지 않도록 유니온-파인드 구조를 이용하여 최소 스패닝 트리를 구성합니다. 이때 선택된 간선의 비용을 누적하여 최종 비용을 계산합니다.

5. **결과 출력**: 최종적으로 모든 행성을 연결하는 데 필요한 최소 비용을 출력합니다.

### 시간 복잡도
- 간선 생성: O(N log N) (각 축별로 N개의 행성을 정렬하므로)
- 간선 정렬: O(E log E) (E는 생성된 간선의 수. 최악의 경우 E는 3N-3이므로 O(N log N))
- 크루스칼 알고리즘: O(E log V) (O(N log N)으로 근사 가능)
- 전체적으로 O(N log N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())

def find(parent, x):
    if parent[x] != x:
        parent[x] = find(parent, parent[x])
    return parent[x]

def union(parent, a, b):
    a = find(parent, a)
    b = find(parent, b)
    
    if a < b:
        parent[b] = a
    else:
        parent[a] = b
        
xx, yy, zz = [], [], []
for i in range(n):
    x, y, z = map(int, input().split())
    xx.append((x, i))
    yy.append((y, i))
    zz.append((z, i))
    
xx.sort()
yy.sort()
zz.sort()

edge = []
for cur in xx, yy, zz:
    for i in range(1, n):
        w1, a = cur[i - 1]
        w2, b = cur[i]
        edge.append((abs(w1 - w2), a, b))
        
edge.sort()

parent = [i for i in range(n)]

def krustal():
    ans = 0
    
    for cost, a, b in edge:
        if find(parent, a) != find(parent, b):
            union(parent, a, b)
            ans += cost
    
    return ans

print(krustal())
```
