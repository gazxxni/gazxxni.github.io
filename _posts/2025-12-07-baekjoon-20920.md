## 문제
화은이는 이번 영어 시험에서 틀린 문제를 바탕으로 영어 단어 암기를 하려고 한다. 그 과정에서 효율적으로 영어 단어를 외우기 위해 영어 단어장을 만들려 하고 있다. 화은이가 만들고자 하는 단어장의 단어 순서는 다음과 같은 우선순위를 차례로 적용하여 만들어진다.


	자주 나오는 단어일수록 앞에 배치한다.
	해당 단어의 길이가 길수록 앞에 배치한다.
	알파벳 사전 순으로 앞에 있는 단어일수록 앞에 배치한다


 M$M$보다 짧은 길이의 단어의 경우 읽는 것만으로도 외울 수 있기 때문에 길이가 M$M$이상인 단어들만 외운다고 한다. 화은이가 괴로운 영단어 암기를 효율적으로 할 수 있도록 단어장을 만들어 주자.

---

## 입력
첫째 줄에는 영어 지문에 나오는 단어의 개수 N$N$과 외울 단어의 길이 기준이 되는 M$M$이 공백으로 구분되어 주어진다. (1≤N≤100000$1 \leq N \leq 100\,000$, 1≤M≤10$1 \leq M \leq 10$)

둘째 줄부터 N+1$N+1$번째 줄까지 외울 단어를 입력받는다. 이때의 입력은 알파벳 소문자로만 주어지며 단어의 길이는 10$10$을 넘지 않는다.

단어장에 단어가 반드시 1개 이상 존재하는 입력만 주어진다.

---

## 출력
화은이의 단어장에 들어 있는 단어를 단어장의 앞에 위치한 단어부터 한 줄에 한 단어씩 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 단어의 출현 빈도, 길이, 그리고 알파벳 순서를 고려하여 단어를 정렬하는 것입니다. 먼저, 주어진 단어들 중 길이가 M 이상인 단어의 출현 빈도를 세고, 그 빈도를 기준으로 단어들을 그룹화한 후, 각 그룹 내에서 길이와 사전 순으로 정렬합니다. 마지막으로, 빈도가 높은 그룹부터 출력합니다.

### 동작 과정
1. **입력 처리**: 첫 번째 줄에서 N(단어 수)과 M(최소 길이)을 입력받습니다. 이어서 N개의 단어를 입력받습니다.
   - 예시 입력:
     ```
     5 3
     apple
     banana
     apple
     grape
     banana
     ```
   
2. **출현 빈도 계산**: 각 단어의 길이가 M 이상인 경우에만 그 단어의 출현 빈도를 `defaultdict`를 사용해 기록합니다. 위의 예시에서는 'apple'이 2회, 'banana'가 2회 출현합니다.

3. **단어 그룹화**: 출현 빈도를 기준으로 단어를 그룹화합니다. 이때, 각 빈도에 해당하는 단어들을 리스트에 저장합니다. 예를 들어, 빈도 2인 단어 목록은 ['apple', 'banana']가 됩니다.

4. **그룹 내 정렬**: 각 빈도 그룹 내의 단어들은 길이가 길수록 먼저 오고, 동일 길이의 단어는 사전 순으로 정렬합니다. 결과적으로 'banana'는 'apple'보다 먼저 오게 됩니다.

5. **최종 정렬**: 빈도를 기준으로 그룹들을 내림차순으로 정렬하여, 가장 자주 등장한 단어부터 출력합니다.

6. **출력**: 정렬된 단어들을 한 줄씩 출력합니다.
   - 최종 출력:
     ```
     banana
     apple
     ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)입니다. 여기서 N은 입력된 단어의 수입니다. 단어의 출현 빈도를 세는 과정은 O(N)이고, 단어 정렬 과정은 O(N log N)입니다. 최종적으로 정렬된 단어를 출력하는 과정도 O(N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline
from collections import defaultdict

n, m = map(int, input().split())

dic = defaultdict(int)
for _ in range(n):
    a = input().rstrip()
    if len(a) >= m:
        dic[a] += 1
    
dic2 = defaultdict(list)
for k, v in dic.items():
    dic2[v].append(k)
    
for k, v in dic2.items():
    v.sort(key=lambda x: (-len(x), x))
    
a = sorted(dic2.items(), key=lambda x:x[0], reverse=True)

for i, j in a:
    for k in j:
        print(k)
```
