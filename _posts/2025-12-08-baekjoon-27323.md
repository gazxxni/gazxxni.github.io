## 문제
정수 A, B 가 주어진다. 세로 길이가 A cm, 가로 길이가 B cm 인 아래와 같은 직사각형의 넓이를 cm2 단위로 구하시오.

---

## 입력
표준 입력에 다음과 같은 형태로 입력이 주어진다.

A
B

---

## 출력
세로 길이가 A cm, 가로 길이가 B cm인 직사각형의 넓이를 cm2 단위로 구하고, 단위 (cm2)를 생략하여 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 88 ms

---

## 풀이

### 풀이 핵심 로직
주어진 두 정수 A와 B는 직사각형의 세로와 가로 길이를 나타낸다. 직사각형의 넓이는 세로 길이와 가로 길이를 곱한 값으로 계산할 수 있다. 따라서, 입력으로 받은 A와 B를 곱하여 직사각형의 넓이를 구한다.

### 동작 과정
1. 사용자로부터 세로 길이 A를 입력받는다. 예를 들어, A = 3.
2. 사용자로부터 가로 길이 B를 입력받는다. 예를 들어, B = 4.
3. A와 B를 곱하여 넓이를 계산한다: 3 cm * 4 cm = 12 cm².
4. 계산된 넓이인 12를 출력한다.

입력 예시:
```
3
4
```
출력 결과:
```
12
```

### 시간 복잡도
O(1) - 입력을 받고 계산하는 과정은 상수 시간에 수행되므로 시간 복잡도는 O(1)이다.

---

## 코드
```python
a = int(input())
b = int(input())

print(a * b)
```
