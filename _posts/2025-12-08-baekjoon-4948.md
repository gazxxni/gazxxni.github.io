## 문제
베르트랑 공준은 임의의 자연수 n에 대하여, n보다 크고, 2n보다 작거나 같은 소수는 적어도 하나 존재한다는 내용을 담고 있다.

이 명제는 조제프 베르트랑이 1845년에 추측했고, 파프누티 체비쇼프가 1850년에 증명했다.

예를 들어, 10보다 크고, 20보다 작거나 같은 소수는 4개가 있다. (11, 13, 17, 19) 또, 14보다 크고, 28보다 작거나 같은 소수는 3개가 있다. (17,19, 23)

자연수 n이 주어졌을 때, n보다 크고, 2n보다 작거나 같은 소수의 개수를 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 케이스는 n을 포함하는 한 줄로 이루어져 있다.

입력의 마지막에는 0이 주어진다.

---

## 출력
각 테스트 케이스에 대해서, n보다 크고, 2n보다 작거나 같은 소수의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 주어진 자연수 \( n \)에 대해 \( n \)보다 크고 \( 2n \)보다 작거나 같은 소수의 개수를 구하는 것이다. 이를 위해 에라토스테네스의 체 알고리즘을 이용해 \( 2n \)까지의 소수를 미리 계산한 후, 주어진 \( n \)에 대해서 해당 구간의 소수 개수를 쉽게 세는 방식으로 해결한다.

### 동작 과정
1. 먼저, \( n \)의 최대 값인 300,000에 대해 에라토스테네스의 체를 이용해 소수를 판별하는 배열을 생성한다. 이 배열은 인덱스가 소수일 때 True 값을 가지며, 나머지는 False로 표시된다.
2. 입력을 받으며, \( n = 0 \)이 입력될 때까지 반복한다.
3. 각 \( n \)에 대해 \( n + 1 \)부터 \( 2n \)까지의 소수 개수를 세기 위해, 미리 계산된 소수 판별 배열을 확인한다.
4. 소수가 발견될 때마다 카운트를 증가시킨 후, 최종적으로 카운트를 출력한다.

예시 입력: 
```
10
13
20
0
```
- 10의 경우: \( 11, 13, 17, 19 \) (4개)
- 13의 경우: \( 17, 19 \) (2개)
- 20의 경우: \( 23, 29 \) (2개)

출력:
```
4
2
5
```

### 시간 복잡도
에라토스테네스의 체를 이용한 소수 판별은 \( O(n \log \log n) \)이며, 각 테스트 케이스에 대해 \( O(n) \)의 시간 복잡도로 소수를 세는 과정이 필요하다. 따라서 전체 시간 복잡도는 \( O(n \log \log n) \)로 볼 수 있다.

## 코드
```python
def prime_numbers(n):
    arr = [True] * (n + 1) 
    arr[0] = arr[1] = False

    for i in range(2, int(n ** 0.5) + 1):
        if arr[i]:
            for j in range(i * i, n + 1, i):
                arr[j] = False
    
    return arr

a = prime_numbers(300000)

while True:
    n = int(input())
    
    if n == 0:
        exit()
        
    cnt = 0
    for i in range(n + 1, 2 * n + 1):
        if a[i] == True:
            cnt += 1
    
    print(cnt)
```
