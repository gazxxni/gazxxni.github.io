## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 문제는 1부터 N까지의 수를 나열했을 때 그 전체 수열의 자릿수를 구하는 것입니다. 이를 위해 각 자리수 범위(1자리, 2자리, ..., N의 자리수)에 따라 자릿수를 계산하여 총합을 구하는 방식으로 문제를 해결합니다. 각 범위에서의 자릿수는 해당 자릿수의 개수에 그 자리수의 크기(1자리면 1, 2자리면 2 등)를 곱하여 구합니다.

### 동작 과정
1. 입력으로 N이 주어집니다. 예를 들어, N = 13이라고 가정합시다.
2. N의 자리수(`l`)를 구합니다. 여기서는 2입니다.
3. 1자리 수(1부터 9까지)에서의 자릿수를 계산합니다:
   - 1자리 수의 개수는 9개 (1~9)
   - 자릿수는 9 * 1 = 9
4. 2자리 수(10부터 13까지)에서의 자릿수를 계산합니다:
   - 2자리 수는 10부터 13까지 총 4개
   - 자릿수는 4 * 2 = 8
5. 최종적으로 1자리 수와 2자리 수의 자릿수를 합칩니다:
   - 총 자릿수 = 9 + 8 = 17
6. 결과적으로 `17`이 출력됩니다.

### 시간 복잡도
O(log N) - N의 자리수를 구하는 과정에서 N의 크기를 기준으로 로그 시간 복잡도를 갖습니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
