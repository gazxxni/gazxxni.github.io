## 문제
1부터 N까지의 수를 이어서 쓰면 다음과 같이 새로운 하나의 수를 얻을 수 있다.


1234567891011121314151617181920212223...


이렇게 만들어진 새로운 수는 몇 자리 수일까? 이 수의 자릿수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 N(1 ≤ N ≤ 100,000,000)이 주어진다.

---

## 출력
첫째 줄에 새로운 수의 자릿수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 1부터 N까지의 숫자를 이어 붙였을 때 총 몇 자리 수가 되는지를 계산하는 문제입니다. 핵심 아이디어는 각 자리 수의 범위를 계산하여 각 범위에 해당하는 숫자들이 차지하는 자릿수를 합산하는 것입니다. 즉, 1자리 수, 2자리 수, ..., n자리 수의 개수를 각각 구하고, 이를 통해 총 자릿수를 구합니다.

### 동작 과정
1. **입력 받기**: N을 입력받습니다. 예를 들어 N = 23이라고 가정합시다.
   
2. **자릿수 계산**:
   - 1자리 수: 1부터 9까지는 총 9개의 수가 있으므로 `9 * 1 = 9` 자리 수입니다.
   - 2자리 수: 10부터 99까지는 총 90개의 수가 있으므로 `90 * 2 = 180` 자리 수입니다. 하지만 N이 23이므로 2자리 수 중 10부터 23까지의 수만 포함됩니다. 따라서 `14 * 2 = 28` 자리 수입니다.
   
3. **최종 계산**:
   - 1자리 수의 자릿수와 2자리 수의 자릿수를 더합니다.
   - 즉, `9 + 28 = 37` 자릿수가 됩니다.

4. **출력**: 최종 결과인 총 자릿수를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(log N)입니다. 이는 N의 자릿수를 계산하는 과정에서 N을 로그 단위로 나누어야 하기 때문입니다. 주어진 N의 범위가 최대 100,000,000이므로 이 알고리즘은 효율적으로 작동합니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = input().rstrip()
l = len(n)

ans = 0
for i in range(1, l):
    ans += 9 * (10 ** (i - 1)) * i
    
ans += (int(n) - (10 ** (l - 1)) + 1) * l
print(ans)
```
