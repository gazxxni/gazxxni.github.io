## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘은 입력 크기 \( n \)에 대해 \( n^3 \)의 수행 횟수를 가지며, 이는 알고리즘의 시간 복잡도를 나타냅니다. 따라서, 출력해야 할 첫 번째 값은 \( n^3 \)이고, 최고차항의 차수는 3으로 일정합니다.

### 동작 과정
1. 주어진 입력 \( n \)을 읽어옵니다. 예를 들어, \( n = 5 \)라고 가정합시다.
2. 수행 횟수를 계산합니다: 
   - \( n^3 = 5^3 = 125 \)입니다.
3. 최고차항의 차수는 항상 3입니다.
4. 결과를 출력합니다:
   - 첫 번째 줄에 125를 출력합니다.
   - 두 번째 줄에 3을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(1) \)입니다. 입력 크기 \( n \)에 따라 수행 횟수를 계산할 때, 단순한 수학 계산만을 수행하기 때문입니다.
