## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 출입 기록을 통해 현재 회사에 있는 직원들의 이름을 파악하는 것입니다. 각 직원의 출근 시 "enter"를, 퇴근 시 "leave"를 기록하며, 현재 사무실에 있는 직원들은 "enter"를 한 후 "leave"를 하지 않은 인원들입니다. 이를 위해 집합 자료구조를 사용하여 직원의 출근 상태를 관리하고, 최종적으로 사전 순의 역순으로 출력합니다.

### 동작 과정
1. **입력 받기**: 먼저, 전체 출입 기록의 수 `n`을 입력받습니다.
2. **직원 상태 관리**: 
   - `dic`이라는 집합을 사용하여 직원의 출근 상태를 관리합니다.
   - 각 기록을 읽어 들이면서 "enter"인 경우 해당 직원의 이름을 `dic`에 추가하고, "leave"인 경우 해당 직원의 이름을 `dic`에서 제거합니다.
3. **현재 인원 리스트 생성**: `dic`에 남아 있는 직원들의 이름을 리스트로 만들어서, 사전 순의 역순으로 정렬합니다.
4. **결과 출력**: 정렬된 리스트를 한 줄씩 출력합니다.

예시 입력:
```
5
Alice enter
Bob enter
Alice leave
Bob leave
Charlie enter
```
- 첫 번째 직원 "Alice"가 들어오고, 두 번째 직원 "Bob"이 들어옵니다.
- "Alice"가 퇴근하고, "Bob"도 퇴근합니다.
- 마지막으로 "Charlie"가 들어오면서 현재 회사에 있는 직원은 "Charlie"입니다.
- 출력은 "Charlie"가 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. 여기서 n은 출입 기록의 수입니다. 직원의 출입 상태를 관리하는 작업은 O(1)로 수행할 수 있지만, 최종 결과를 정렬하는 과정에서 O(n log n)의 복잡도를 발생시키므로 전체 복잡도는 O(n log n)입니다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
