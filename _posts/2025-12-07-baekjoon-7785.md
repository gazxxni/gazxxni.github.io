## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 직원의 출입 기록을 바탕으로 현재 회사에 있는 직원들의 목록을 생성하는 것입니다. 이를 위해 해시맵(딕셔너리)을 사용하여 각 직원의 출근 상태를 기록하고, "enter"일 경우 해당 직원을 추가하고, "leave"일 경우 해당 직원을 제거합니다. 마지막으로, 현재 회사에 있는 직원들을 사전 순의 역순으로 정렬하여 출력합니다.

### 동작 과정
1. 입력으로 주어진 로그의 수 `n`을 읽어옵니다.
2. `n`개의 출입 기록을 순서대로 읽어옵니다.
   - 예를 들어, 입력이 다음과 같다고 가정합니다:
     ```
     5
     Alice enter
     Bob enter
     Alice leave
     Charlie enter
     Bob leave
     ```
3. 입력을 처리하면서, "enter"일 경우 해당 직원의 상태를 딕셔너리에 추가하고, "leave"일 경우 해당 직원의 상태를 제거합니다.
   - 첫 번째 기록인 "Alice enter"에서는 `dic['Alice']`를 1로 설정합니다.
   - 두 번째 기록인 "Bob enter"에서는 `dic['Bob']`을 1로 설정합니다.
   - 세 번째 기록인 "Alice leave"에서는 `dic['Alice']`를 0으로 설정합니다.
   - 네 번째 기록인 "Charlie enter"에서는 `dic['Charlie']`를 1로 설정합니다.
   - 다섯 번째 기록인 "Bob leave"에서는 `dic['Bob']`을 0으로 설정합니다.
4. 모든 기록을 처리한 후, 현재 회사에 있는 직원들은 값이 1인 딕셔너리 항목들로 확인할 수 있습니다.
5. 이 직원들의 이름을 리스트에 추가하고, 사전 순의 역순으로 정렬한 후 출력합니다.
   - 현재 회사에 있는 직원 목록: `['Charlie']`

### 시간 복잡도
입력으로 주어진 출입 기록의 수 `n`에 대해 각 기록을 한 번씩 처리하므로, 전체 시간 복잡도는 O(n)입니다. 이후, 현재 회사에 있는 직원들의 이름을 정렬하는 과정이 추가되므로 최종 시간 복잡도는 O(n log n)입니다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
