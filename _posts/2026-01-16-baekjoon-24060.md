## 문제
오늘도 서준이는 병합 정렬 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

N개의 서로 다른 양의 정수가 저장된 배열 A가 있다. 병합 정렬로 배열 A를 오름차순 정렬할 경우 배열 A에 K 번째 저장되는 수를 구해서 우리 서준이를 도와주자.

크기가 N인 배열에 대한 병합 정렬 의사 코드는 다음과 같다.

merge_sort(A[p..r]) { # A[p..r]을 오름차순 정렬한다.
    if (p < r) then {
        q <- ⌊(p + r) / 2⌋;       # q는 p, r의 중간 지점
        merge_sort(A, p, q);      # 전반부 정렬
        merge_sort(A, q + 1, r);  # 후반부 정렬
        merge(A, p, q, r);        # 병합
    }
}

---

## 입력
첫째 줄에 배열 A의 크기 N(5 ≤ N ≤ 500,000), 저장 횟수 K(1 ≤ K ≤ 108)가 주어진다.

다음 줄에 서로 다른 배열 A의 원소 A1, A2, ..., AN이 주어진다. (1 ≤ Ai ≤ 109)

---

## 출력
배열 A에 K 번째 저장 되는 수를 출력한다. 저장 횟수가 K 보다 작으면 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
병합 정렬 알고리즘을 사용하여 주어진 배열을 정렬하는 동시에, 정렬 과정에서 K번째 저장되는 값을 추적합니다. 정렬이 진행되는 동안 배열의 값을 저장하는 횟수를 카운트하고, 해당 카운트가 K에 도달했을 때 그 값을 기록합니다. 만약 K번째 저장이 이루어지지 않으면 -1을 출력합니다.

### 동작 과정
1. 입력: N = 5, K = 3, 배열 A = [5, 3, 4, 1, 2]
2. `merge_sort`를 시작하여 배열을 두 부분으로 나누고 각각 정렬합니다.
   - 첫 번째 호출: `merge_sort(A, 0, 4)` -> p=0, r=4
   - 중간 지점 q = 2에서 두 개의 부분 배열로 나누어짐: `[5, 3, 4]`와 `[1, 2]`
3. 첫 번째 부분 배열 `[5, 3, 4]`에 대해 다시 `merge_sort` 호출:
   - `merge_sort(A, 0, 2)` -> p=0, r=2, q=1에서 `[5, 3]`와 `[4]`로 나누어짐
4. `[5, 3]`에 대해 다시 나누고 정렬:
   - `merge_sort(A, 0, 1)` -> p=0, r=1, q=0에서 `[5]`와 `[3]`
5. `[5]`와 `[3]`를 병합:
   - `merge` 호출: A = [3, 5, 4, 1, 2] (저장 횟수 1)
6. 다음, `[3, 5]`와 `[4]`를 병합:
   - `merge` 호출: A = [3, 4, 5, 1, 2] (저장 횟수 2)
7. 두 번째 부분 배열 `[1, 2]`에 대해 `merge_sort` 호출:
   - `merge_sort(A, 3, 4)`에서 `[1]`, `[2]`로 나누어짐
8. `[1]`와 `[2]`를 병합:
   - `merge` 호출: A = [3, 4, 5, 1, 2] (저장 횟수 3) -> 이때 K번째 저장 발생, ans = 1
9. 마지막으로 `[3, 4, 5]`와 `[1, 2]`를 병합:
   - `merge` 호출: A = [1, 2, 3, 4, 5] (저장 횟수 4)
10. 최종적으로 K=3에서 저장된 값 1을 출력.

### 시간 복잡도
병합 정렬의 시간 복잡도는 O(N log N)입니다. 여기서 N은 배열의 크기입니다. 추가적으로, K번째 저장을 추적하는 과정은 O(N)으로 이루어지며, 전체적으로 시간 복잡도는 O(N log N)으로 유지됩니다.

## 코드
```python
def merge_sort(A, p, r):
    if p < r:
        q = (p + r) // 2
        merge_sort(A, p, q)
        merge_sort(A, q + 1, r)
        merge(A, p, q, r)

def merge(A, p, q, r):
    global cnt, ans
    
    i = p
    j = q + 1
    t = 0
    tmp = [0] * (r - p + 1)
    
    while i <= q and j <= r:
        if A[i] <= A[j]:
            tmp[t] = A[i]
            i += 1
        else:
            tmp[t] = A[j]
            j += 1
        t += 1
        
    while i <= q:
        tmp[t] = A[i]
        i += 1
        t += 1
        
    while j <= r:
        tmp[t] = A[j]
        j += 1
        t += 1
        
    i = p
    t = 0
    
    while i <= r:
        A[i] = tmp[t]
        cnt += 1
        
        if cnt == k:
            ans = A[i]
            
        i += 1
        t += 1

n, k = map(int, input().split())
arr = list(map(int, input().split()))

cnt = 0
ans = -1

merge_sort(arr, 0, n - 1)

print(ans)
```
