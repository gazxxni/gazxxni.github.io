## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
두 분수 \( A/B \)와 \( C/D \)의 합을 구하기 위해, 두 분수의 분모의 최소공배수(LCM)를 구한 후, 각 분수를 이 LCM로 변환하여 분자를 더한다. 마지막으로, 결과 분수를 기약분수 형태로 만들기 위해 분자와 분모의 최대공약수(GCD)로 나눈다.

### 동작 과정
예를 들어, 입력으로 다음과 같은 두 분수가 주어진다고 가정해 보겠습니다:
```
1 2
1 3
```
1. 첫 번째 분수는 \( A = 1 \), \( B = 2 \)이고, 두 번째 분수는 \( C = 1 \), \( D = 3 \)입니다.
2. 두 분수의 분모 \( B \)와 \( D \)의 LCM을 구합니다:
   - \( \text{lcm}(2, 3) = 6 \)
3. 각 분수를 LCM을 기준으로 변환합니다:
   - \( \frac{1}{2} \)를 \( \frac{3}{6} \)으로 변환 (3 = \( 6/2 \))
   - \( \frac{1}{3} \)를 \( \frac{2}{6} \)으로 변환 (2 = \( 6/3 \))
4. 이제 두 분수를 더합니다:
   - \( \frac{3}{6} + \frac{2}{6} = \frac{5}{6} \)
5. 결과 분수 \( 5/6 \)의 GCD를 구하여 기약분수 형태로 만듭니다:
   - \( \text{gcd}(5, 6) = 1 \)
6. 따라서, 최종 결과는 \( \frac{5 \div 1}{6 \div 1} = \frac{5}{6} \)입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(\log(\min(B, D))) \)입니다. 이는 GCD를 구하는 유클리드 호제법의 복잡도 때문이며, LCM을 계산하는 과정은 O(1) 시간에 수행됩니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
