## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
두 분수 A/B와 C/D의 합을 구하기 위해, 먼저 두 분수의 분모를 같게 만들어야 합니다. 이를 위해 두 분모의 최소공배수(LCM)를 구하고, 각 분수의 분자에 적절한 배수를 곱한 후 합산합니다. 마지막으로, 결과를 기약분수 형태로 표현하기 위해 분자와 분모의 최대공약수(GCD)를 사용하여 약분합니다.

### 동작 과정
1. **입력**: 두 분수를 입력받는다. 예를 들어, 첫 번째 분수는 1/2, 두 번째 분수는 1/3이라고 하자.
   ```
   1 2  (A = 1, B = 2)
   1 3  (C = 1, D = 3)
   ```

2. **최소공배수 계산**: 두 분모 2와 3의 최소공배수를 계산한다.
   ```
   LCM(2, 3) = 6
   ```

3. **분자 변환**: 각 분수를 같은 분모로 변환한다.
   - A/B = 1/2 는 분모를 6으로 맞추면 (1 * 3) / 6 = 3/6
   - C/D = 1/3 은 (1 * 2) / 6 = 2/6

4. **분자 합산**: 두 변환된 분수의 분자를 합친다.
   ```
   3 + 2 = 5
   ```

5. **최종 분수**: 현재 분수는 5/6이다. 기약분수로 만들기 위해 GCD를 구한다.
   ```
   GCD(5, 6) = 1
   ```

6. **약분**: 분자와 분모를 GCD로 나눈다.
   ```
   5 / 1 = 5
   6 / 1 = 6
   ```

7. **출력**: 결과를 출력한다.
   ```
   5 6
   ```

### 시간 복잡도
- 최대공약수(GCD)와 최소공배수(LCM) 계산에 사용되는 유클리드 호제법의 시간 복잡도는 O(log(min(a, b)))입니다. 따라서 전체 알고리즘의 시간 복잡도 또한 O(log(max(B, D)))로 표현할 수 있습니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
