## 문제
Good Bye BOJ, 2021!이 열리는 오늘, 12월 31일은 종서의 생일이다. N$N$ 명의 친구들은 종서에게 생일 선물로 예쁜 케이크를 만들어주려 한다.

여기에서, 예쁜 케이크는 다음과 같은 조건을 만족하는 케이크를 의미한다.


	케이크는 높이가 1$1$이고, 부피가 N$N$인 직육면체 모양이다.
	케이크를 적절히 칼질해서 한 변의 길이가 1$1$인 정육면체 모양 조각 N$N$ 개로 나눌 수 있어야 한다.
	케이크의 옆면에 가로 너비가 1$1$인 직사각형을 이어 붙여 만든 띠를 딱 맞게 두를 수 있어야 한다.
	장식용 띠는 가로 폭이 1$1$인 빨간색, 초록색, 하얀색 직사각형이 순서대로 번갈아 가면서 같은 개수만큼 나와야 한다.


예를 들어, 아래 그림은 N=8$N = 8$인 경우의 예쁜 케이크 중 하나와 그에 사용된 띠를 나타낸다.





아쉽게도 N$N$이 얼마인지에 따라 예쁜 케이크를 만들지 못 할 수도 있다. 종서의 친구들을 위해 부피가 N$N$인 예쁜 케이크를 만들 수 있는지 알려주자.

---

## 입력
첫 번째 줄에 전체 테스트 케이스의 개수를 나타내는 정수 T$T$가 주어진다.

이후 T$T$ 개의 줄에 각각 문제에서 언급한 정수 N$N$이 한 줄에 하나씩 주어진다.

---

## 출력
T$T$ 개의 줄에 걸쳐 한 줄에 하나씩 문제의 답을 출력해야 한다.

부피가 N$N$인 예쁜 케이크를 만들 수 있으면 TAK, 아니면 NIE를 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 주어진 부피 \( N \)에 대해 예쁜 케이크를 만들 수 있는지를 판별하는 것이다. 예쁜 케이크는 다음 조건을 만족해야 한다: 
1. 높이가 1이고 부피가 \( N \)인 직육면체로 나뉘어야 하며,
2. 케이크의 옆면에 부착되는 장식용 띠는 빨간색, 초록색, 하얀색이 번갈아 가며 같은 개수로 이루어져야 한다. 이를 위해 \( N \)이 3의 배수거나 9의 배수일 때 가능하다.

### 동작 과정
1. 입력으로 테스트 케이스 수 \( T \)를 받고, 그 다음 \( T \)개의 \( N \) 값을 입력받는다.
2. 각 \( N \)에 대해 다음을 확인한다:
   - \( N \mod 3 \)의 나머지를 확인한다.
     - 나머지가 1일 경우, 케이크를 만들 수 없으므로 "NIE"를 출력한다.
     - 나머지가 2일 경우, 케이크를 만들 수 있으므로 "TAK"를 출력한다.
     - 나머지가 0일 경우, 추가로 \( N \)이 9의 배수인지 확인한다.
       - 9의 배수일 경우 "TAK"를 출력하고, 그렇지 않으면 "NIE"를 출력한다.
3. 각 결과를 차례대로 출력한다.

예시 입력:
```
3
8
9
10
```
예시 출력:
```
TAK
TAK
NIE
```
- \( N = 8 \): \( 8 \mod 3 = 2 \) → "TAK"
- \( N = 9 \): \( 9 \mod 3 = 0 \) → \( 9 \mod 9 = 0 \) → "TAK"
- \( N = 10 \): \( 10 \mod 3 = 1 \) → "NIE"

### 시간 복잡도
시간 복잡도는 \( O(T) \)로, 각 테스트 케이스에 대해 상수 시간 연산을 수행하기 때문에 선형 시간이 걸린다.

## 코드
```python
import sys
input = sys.stdin.readline

t = int(input())

for _ in range(t):
    n = int(input())
    
    remainder = n % 3
    
    if remainder == 1:
        print('NIE')
    elif remainder == 2:
        print('TAK')
    else:
        if n % 9 == 0:
            print('TAK')
        else:
            print('NIE')
```
