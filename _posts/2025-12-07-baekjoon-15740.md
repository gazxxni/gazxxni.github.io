## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 A와 B (-1010000 ≤ A, B ≤ 1010000)가 주어진다.

---

## 출력
첫째 줄에 A+B를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 96 ms

## 풀이

### 풀이 핵심 로직
문제는 두 개의 정수 A와 B를 입력받아 이들의 합을 출력하는 것입니다. Python에서는 `int` 타입이 무제한 정밀도를 지원하므로, 매우 큰 수의 덧셈을 처리할 수 있습니다. 따라서 기본적인 입력 처리와 덧셈 연산만으로 문제를 해결할 수 있습니다.

### 동작 과정
1. **입력받기**: `input()` 함수를 사용하여 두 정수를 한 줄에서 입력받습니다. 이때 `split()` 함수를 이용하여 공백으로 구분된 두 값을 나누어줍니다.
2. **형 변환**: `map(int, ...)`을 사용하여 입력받은 문자열을 정수형으로 변환합니다.
3. **덧셈 수행**: 변환된 두 정수를 더합니다.
4. **결과 출력**: `print()` 함수를 사용하여 결과를 출력합니다.

예시 입력:
```
123456789123456789 987654321987654321
```
단계별 설명:
- 첫째 줄에서 `123456789123456789`와 `987654321987654321`을 입력받음.
- `map(int, input().split())`에 의해 두 값이 정수형으로 변환되어 각각 `123456789123456789`와 `987654321987654321`이 됨.
- 두 정수를 더하여 `1111111111111111110`이 됨.
- `print()`로 결과를 출력하여, 최종적으로 `1111111111111111110`이 화면에 출력됨.

### 시간 복잡도
O(1)
(입력 크기에 관계없이 덧셈 연산은 항상 일정한 시간에 수행되므로 상수 시간 복잡도를 가집니다.)

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a + b)
```
