## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 A와 B (-1010000 ≤ A, B ≤ 1010000)가 주어진다.

---

## 출력
첫째 줄에 A+B를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 96 ms

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 큰 정수 A와 B를 입력받아 그 합을 계산하여 출력하는 간단한 프로그램을 작성하는 것입니다. Python은 기본적으로 큰 수를 지원하므로, 별도의 라이브러리 없이도 정수의 크기에 상관없이 계산할 수 있습니다.

### 동작 과정
1. 입력: 사용자로부터 두 개의 정수 A와 B를 한 줄에 공백으로 구분하여 입력받습니다. 예를 들어, 입력이 `123456789012345678901234567890 987654321098765432109876543210`이면 A는 `123456789012345678901234567890`, B는 `987654321098765432109876543210`입니다.
2. 처리: 입력받은 두 정수를 정수형으로 변환한 후, A와 B를 더합니다. 이 경우, A + B는 `1111111110111111111011111111100`입니다.
3. 출력: 계산된 결과를 출력합니다.

### 시간 복잡도
이 문제의 시간 복잡도는 O(n)입니다. 여기서 n은 두 정수의 자릿수입니다. 입력과 출력 작업이 각각 O(n)으로 이루어지기 때문에 전체적으로 O(n)의 시간 복잡도를 가집니다.

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a + b)
```
