## 문제
1부터 입력받은 숫자 n 사이에 있는 소수의 개수를 반환하는 함수, solution을 만들어 보세요. 

소수는 1과 자기 자신으로만 나누어지는 수를 의미합니다.
(1은 소수가 아닙니다.)

제한 조건


n은 2이상 1000000이하의 자연수입니다.


입출력 예

        
n
result


        
10
4


5
3


      
입출력 예 설명

입출력 예 #1
1부터 10 사이의 소수는 [2,3,5,7] 4개가 존재하므로 4를 반환

입출력 예 #2
1부터 5 사이의 소수는 [2,3,5] 3개가 존재하므로 3를 반환

## 풀이
### 풀이 핵심 로직
소수를 찾기 위해 에라토스테네스의 체를 사용하여 1부터 n까지의 수 중 소수인 수를 판별합니다. 이 방법은 각 수에 대한 소수 판별을 효율적으로 수행할 수 있어, n이 큰 경우에도 빠르게 소수를 계산할 수 있습니다.

### 동작 과정
1. **is_prime 함수**: 
   - 입력으로 받은 x에 대해, 0부터 x까지의 숫자가 소수인지 여부를 기록하기 위해 `prime` 리스트를 생성합니다. 
   - 0과 1은 소수가 아니므로 `prime[0]`과 `prime[1]`을 `False`로 설정합니다.
   - 2부터 √x까지의 각 수 i에 대해, i가 소수인 경우 i의 배수들을 모두 소수가 아님을 표시합니다.
   - 최종적으로 `prime` 리스트를 반환합니다.

2. **solution 함수**: 
   - `is_prime` 함수를 호출하여 n까지의 소수 정보를 담고 있는 `prime` 리스트를 받습니다.
   - `prime` 리스트에서 `True`의 개수를 세어 소수의 개수를 반환합니다.

예시 입력으로 n = 10인 경우:
- is_prime(10)을 호출하면 `prime` 리스트는 [False, False, True, True, False, True, False, True, False, False, False]가 됩니다. 이 리스트에서 True의 개수는 4개로, 2, 3, 5, 7이 소수임을 나타냅니다.
- 따라서, `solution(10)`은 4를 반환합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log log n)입니다. 에라토스테네스의 체 알고리즘은 n까지의 모든 소수를 구하는 데 효율적이기 때문에, n이 커질수록 다른 방법에 비해 월등한 성능을 보입니다.

## 코드
```python
def is_prime(x):
    prime = [True] * (x + 1)
    prime[0] = prime[1] = False
    
    for i in range(2, int(x ** 0.5) + 1):
        if prime[i]:
            for j in range(i*i, x + 1, i):
                prime[j] = False
                
    return prime
        
        
def solution(n):
    prime = is_prime(n)

    return prime.count(True)
```
