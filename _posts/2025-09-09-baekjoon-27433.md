## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
문제는 주어진 정수 N에 대해 N! (N 팩토리얼)을 계산하는 것이다. 팩토리얼은 N이 1보다 작거나 같을 때 1로 정의되고, 그 외의 경우는 N과 (N-1)!의 곱으로 정의된다. 이 특성을 이용하여 재귀 함수를 통해 쉽게 계산할 수 있다.

### 동작 과정
1. 입력으로 정수 N을 받는다.
   - 예시: N = 5
2. `factorial` 함수가 호출된다:
   - factorial(5) -> 5 * factorial(4)
3. 다시 `factorial` 함수가 호출된다:
   - factorial(4) -> 4 * factorial(3)
4. 계속해서 호출된다:
   - factorial(3) -> 3 * factorial(2)
   - factorial(2) -> 2 * factorial(1)
   - factorial(1) -> 1 (재귀 호출의 종료 조건)
5. 이제 계산이 시작된다:
   - factorial(2) = 2 * 1 = 2
   - factorial(3) = 3 * 2 = 6
   - factorial(4) = 4 * 6 = 24
   - factorial(5) = 5 * 24 = 120
6. 결과적으로 5! = 120이 출력된다.

### 시간 복잡도
O(N) - 재귀 호출이 N번 발생하므로, N의 크기에 비례하는 시간 복잡도를 가진다. 각 호출에서 상수 시간의 곱셈 연산이 수행되기 때문이다.

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
