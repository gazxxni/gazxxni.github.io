## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제에서 주어진 조건에 따르면, N개의 상의와 N개의 하의가 각각 N개의 서로 다른 색상으로 존재합니다. 상의와 하의의 색상이 서로 다르기 위해서는 상의의 색상 중 하나를 선택하고, 해당 색상을 제외한 나머지 하의 색상 중에서 하나를 선택해야 합니다. 따라서 가능한 조합의 수는 N개의 상의 중 하나를 선택하고, N-1개의 하의 색상 중 하나를 선택하는 방식으로 계산할 수 있습니다. 이는 `N * (N - 1)`으로 표현됩니다.

### 동작 과정
1. 입력으로 N을 받는다. 예를 들어, N = 3이라고 가정하자.
2. 상의 색상: 1, 2, 3
3. 하의 색상: 1, 2, 3
4. 상의 색상 1을 선택하면, 하의 색상은 2 또는 3 중 하나를 선택할 수 있다. (2가지)
5. 상의 색상 2를 선택하면, 하의 색상은 1 또는 3 중 하나를 선택할 수 있다. (2가지)
6. 상의 색상 3을 선택하면, 하의 색상은 1 또는 2 중 하나를 선택할 수 있다. (2가지)
7. 따라서 총 조합 수는 3 (상의) * 2 (남은 하의) = 6가지가 된다.

### 시간 복잡도
시간 복잡도는 O(1)이다. 입력 N을 받아서 간단한 산술 연산을 수행하기 때문에 입력 크기와 관계없이 일정한 시간 안에 결과를 도출할 수 있다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
