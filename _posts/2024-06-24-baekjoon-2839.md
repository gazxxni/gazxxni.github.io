## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 3kg 봉지와 5kg 봉지를 조합하여 주어진 무게(n)를 최소한의 봉지 개수로 만들기 위해, 가능한 한 3kg 봉지를 많이 사용하되, 나머지 무게가 5kg 봉지로 나누어 떨어지는 경우에만 해결할 수 있다는 것입니다. 이를 통해 최소의 봉지 개수를 구합니다.

### 동작 과정
1. 입력으로 무게 `n`을 받습니다.
2. 만약 `n`이 5로 나누어 떨어지면, 5kg 봉지만 사용하여 `n // 5`를 출력합니다.
3. 그렇지 않다면, 3kg 봉지를 최대한 사용하기 위해 반복문을 실행합니다.
   - 반복문 내에서 `n`에서 3을 빼고, 봉지 개수를 1 증가시킵니다.
   - 만약 `n`이 5로 나누어 떨어지면, 현재의 봉지 개수에 `n // 5`를 더하여 결과를 출력합니다.
   - 만약 `n`이 1 또는 2가 되면, 더 이상 5kg 봉지로 나누어 떨어지지 않으므로 -1을 출력합니다.
   - 만약 `n`이 0이 되면, 현재 봉지 개수를 출력하고 반복문을 종료합니다.

#### 예시 입력 및 단계별 설명
- 예시 입력: `18`
  - 18은 5로 나누어 떨어지지 않으므로 3kg 봉지 사용 시작.
  - 18 - 3 = 15, 봉지 개수 1
  - 15은 5로 나누어 떨어지므로 총 봉지 개수는 1 (3kg 봉지) + 3 (5kg 봉지) = 4
  - 최종 출력: `4`

- 예시 입력: `4`
  - 4는 5로 나누어 떨어지지 않으므로 3kg 봉지 사용 시작.
  - 4 - 3 = 1, 봉지 개수 1
  - 1은 5로 나누어 떨어지지 않으므로 -1 출력.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n/3)입니다. 이는 n이 3으로 나누어지는 방식으로 줄어들기 때문에, 최악의 경우에서 반복문이 n/3번 실행될 수 있습니다. 하지만, 일반적으로 O(n)으로 표현할 수 있습니다.

## 코드
```python
import sys

n=int(input())

if n%5==0:
    print(n//5)
else:
    a=0
    while n>0:
        n-=3
        a+=1
        if n%5==0:
            a+=n//5
            print(a)
            break
        elif n==1 or n==2:
            print(-1)
        elif n==0:
            print(a)
            break
```
