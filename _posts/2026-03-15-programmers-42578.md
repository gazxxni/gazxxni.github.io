## 풀이
### 풀이 핵심 로직
이 문제는 각 의상의 종류별로 조합할 수 있는 경우의 수를 계산하는 문제입니다. 각 종류의 의상에서 0개(착용하지 않음) 또는 1개(착용) 선택할 수 있으므로, 각 종류의 의상 수에 1을 더하여 곱한 후, 최소 한 개의 의상을 착용해야 하므로 마지막에 1을 빼줍니다.

### 동작 과정
1. **입력 예시**: 
   `clothes = [["yellow_hat", "headgear"], ["blue_sunglasses", "eyewear"], ["green_turban", "headgear"]]`
   
2. **딕셔너리 생성**: 
   의상의 종류를 키로 하고 그 종류에 해당하는 의상의 수를 값으로 하는 딕셔너리 `dic`를 생성합니다.
   - `headgear`의 경우: 2개 (yellow_hat, green_turban)
   - `eyewear`의 경우: 1개 (blue_sunglasses)
   - 결과적으로 `dic`은 `{'headgear': 2, 'eyewear': 1}`이 됩니다.

3. **조합 계산**:
   각 종류별 의상 수에 1을 더한 값을 곱합니다. 
   - `headgear`: 2 + 1 = 3 (yellow_hat, green_turban, 착용하지 않음)
   - `eyewear`: 1 + 1 = 2 (blue_sunglasses, 착용하지 않음)
   
   전체 조합 수는 `3 * 2 = 6`입니다.

4. **최소 의상 착용 조건 적용**:
   최소 한 개의 의상은 착용해야 하므로, 마지막에 1을 빼줍니다.
   - 최종 조합 수: `6 - 1 = 5`

5. **출력**:
   최종적으로 `5`를 리턴합니다.

### 시간 복잡도
이 알고리즘은 의상 목록을 한 번 스캔하여 딕셔너리를 생성하고, 그 딕셔너리를 한 번 더 스캔하여 곱셈을 수행하므로 전체 시간 복잡도는 O(n)입니다. 여기서 n은 의상의 종류 수입니다.

## 코드
```python
def solution(clothes):
    answer = 1
    dic = {}
    
    for k, v in clothes:
        if v in dic:
            dic[v] += 1
        else:
            dic[v] = 1
    
    for k, v in dic.items():
        answer *= (v + 1)
        
    return answer - 1
```
