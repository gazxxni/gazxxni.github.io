## 풀이

### 풀이 핵심 로직
이 문제는 주어진 다각형의 꼭짓점 좌표를 이용하여 면적을 계산하는 것입니다. 면적은 다각형의 꼭짓점을 순서대로 연결하여 구하는 'Shoelace formula(신발끈 공식)'를 사용하여 계산합니다. 이 공식은 좌표의 곱을 이용해 다각형의 면적을 효율적으로 구할 수 있습니다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 다각형의 꼭짓점 개수 `n`을 입력받고, 다음 `n`줄에서 각 꼭짓점의 좌표 `(x, y)`를 입력받아 리스트 `arr`에 저장합니다.
   - 예시 입력:
     ```
     4
     0 0
     4 0
     4 3
     0 4
     ```
   - `arr` 리스트는 `[(0, 0), (4, 0), (4, 3), (0, 4)]`가 됩니다.

2. **면적 계산**: 두 개의 인접한 꼭짓점을 이용하여 면적의 합을 구합니다. 각 꼭짓점 `(x1, y1)`과 다음 꼭짓점 `(x2, y2)`에 대해 `(x1 * y2) - (x2 * y1)`을 계산하여 `ans` 변수에 더합니다. 마지막 꼭짓점은 첫 번째 꼭짓점과 연결되도록 처리하기 위해 인덱스를 `% n`으로 계산합니다.
   - 계산 과정:
     - 1번째 꼭짓점: (0, 0)과 (4, 0) → `0 * 0 - 4 * 0 = 0`
     - 2번째 꼭짓점: (4, 0)과 (4, 3) → `4 * 3 - 4 * 0 = 12`
     - 3번째 꼭짓점: (4, 3)과 (0, 4) → `4 * 4 - 0 * 3 = 16`
     - 4번째 꼭짓점: (0, 4)과 (0, 0) → `0 * 0 - 0 * 4 = 0`
   - 최종적으로 `ans`는 `0 + 12 + 16 + 0 = 28`이 됩니다.

3. **면적 출력**: 구한 `ans`에 절대값을 취하고 2로 나누어 최종 면적을 구한 후 소수점 첫째 자리까지 출력합니다.
   - 면적: `abs(28) / 2 = 14.0` → 출력: `14.0`

### 시간 복잡도
O(n) - 입력받는 점의 개수에 비례하여 면적을 계산하는 과정이 진행되므로, 점의 개수 `n`에 대한 선형 시간 복잡도를 가집니다.

---

## 코드
```python
n = int(input())
arr = []

for _ in range(n):
    x, y = map(int, input().split())
    arr.append((x, y))

ans = 0

for i in range(n):
    x1, y1 = arr[i]
    x2, y2 = arr[(i + 1) % n]
    ans += (x1 * y2) - (x2 * y1)

area = abs(ans) / 2
print(f"{area:.1f}")

```
