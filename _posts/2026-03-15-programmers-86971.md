## 문제
n개의 송전탑이 전선을 통해 하나의 트리 형태로 연결되어 있습니다. 당신은 이 전선들 중 하나를 끊어서 현재의 전력망 네트워크를 2개로 분할하려고 합니다. 이때, 두 전력망이 갖게 되는 송전탑의 개수를 최대한 비슷하게 맞추고자 합니다.

송전탑의 개수 n, 그리고 전선 정보 wires가 매개변수로 주어집니다. 전선들 중 하나를 끊어서 송전탑 개수가 가능한 비슷하도록 두 전력망으로 나누었을 때, 두 전력망이 가지고 있는 송전탑 개수의 차이(절대값)를 return 하도록 solution 함수를 완성해주세요.



제한사항


n은 2 이상 100 이하인 자연수입니다.
wires는 길이가 n-1인 정수형 2차원 배열입니다.


wires의 각 원소는 [v1, v2] 2개의 자연수로 이루어져 있으며, 이는 전력망의 v1번 송전탑과 v2번 송전탑이 전선으로 연결되어 있다는 것을 의미합니다.
1 ≤ v1 &lt; v2 ≤ n 입니다.
전력망 네트워크가 하나의 트리 형태가 아닌 경우는 입력으로 주어지지 않습니다.

## 풀이
### 풀이 핵심 로직
주어진 전력망을 두 개의 부분으로 나누기 위해 각 전선을 끊고, 두 부분의 송전탑 개수를 계산하여 그 차이를 구합니다. BFS(너비 우선 탐색)를 이용해 한 쪽의 송전탑 개수를 세고, 나머지 송전탑 개수는 전체 송전탑 개수에서 세어낸 송전탑 개수를 빼서 구합니다. 이 과정에서 두 부분의 송전탑 개수 차이의 최솟값을 찾아 반환합니다.

### 동작 과정
1. **그래프 구성**: 주어진 송전탑과 전선 정보를 이용해 인접 리스트 형태의 그래프를 만듭니다.
2. **전선 끊기**: 각 전선을 하나씩 끊어보며 두 부분의 송전탑 개수를 세어봅니다.
   - 예를 들어, 입력이 `n = 9`이고 `wires = [[1,3],[2,3],[3,4],[4,5],[4,6],[4,7],[7,8],[7,9]]`일 때 첫 번째 전선인 `[1, 3]`를 끊습니다.
3. **BFS 수행**: 끊어진 전선의 한 쪽 송전탑에서 BFS를 실행하여 그 쪽에 연결된 송전탑 개수를 셉니다. 나머지 송전탑 개수는 전체 송전탑 수에서 BFS로 세어진 송전탑 수를 빼서 계산합니다.
   - 예를 들어, 3번 송전탑에서 시작하여 연결된 송전탑을 세면 6개가 될 수 있습니다. 나머지 송전탑은 3개입니다.
4. **차이 계산**: 두 부분의 송전탑 개수 차이를 구하고, 이 차이가 최소가 되는 경우를 기록합니다.
5. **반복**: 모든 전선에 대해 위의 과정을 반복합니다.
6. **결과 반환**: 구한 최소 차이를 반환합니다.

### 시간 복잡도
- **BFS**를 수행하는 부분은 각 전선이 연결된 송전탑을 모두 탐색해야 하므로 O(n)입니다. 이 과정을 모든 전선에 대해 반복하므로 총 O(n^2)입니다.
- 따라서 이 알고리즘의 시간 복잡도는 O(n^2)입니다.

## 코드
```python
from collections import deque

def solution(n, wires):
    answer = float('inf')
    graph = {i:[] for i in range(1, n+1)}
    for u, v in wires:
        graph[u].append(v)
        graph[v].append(u)
    
    def bfs(start):
        q = deque([start])
        visited = {start}
        cnt = 0
        while q:
            cur = q.popleft()
            cnt += 1
            for next in graph[cur]:
                if next not in visited:
                    visited.add(next)
                    q.append(next)
                    
        return cnt
    
    for u, v in wires:
        graph[u].remove(v)
        graph[v].remove(u)

        cnt1 = bfs(u)
        cnt2 = n - cnt1
        answer = min(answer, abs(cnt1 - cnt2))

        graph[u].append(v)
        graph[v].append(u)
            
    return answer
```
