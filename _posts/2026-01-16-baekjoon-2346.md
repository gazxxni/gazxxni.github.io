## 문제
1번부터 N번까지 N개의 풍선이 원형으로 놓여 있고. i번 풍선의 오른쪽에는 i+1번 풍선이 있고, 왼쪽에는 i-1번 풍선이 있다. 단, 1번 풍선의 왼쪽에 N번 풍선이 있고, N번 풍선의 오른쪽에 1번 풍선이 있다. 각 풍선 안에는 종이가 하나 들어있고, 종이에는 -N보다 크거나 같고, N보다 작거나 같은 정수가 하나 적혀있다. 이 풍선들을 다음과 같은 규칙으로 터뜨린다.

우선, 제일 처음에는 1번 풍선을 터뜨린다. 다음에는 풍선 안에 있는 종이를 꺼내어 그 종이에 적혀있는 값만큼 이동하여 다음 풍선을 터뜨린다. 양수가 적혀 있을 경우에는 오른쪽으로, 음수가 적혀 있을 때는 왼쪽으로 이동한다. 이동할 때에는 이미 터진 풍선은 빼고 이동한다.

예를 들어 다섯 개의 풍선 안에 차례로 3, 2, 1, -3, -1이 적혀 있었다고 하자. 이 경우 3이 적혀 있는 1번 풍선, -3이 적혀 있는 4번 풍선, -1이 적혀 있는 5번 풍선, 1이 적혀 있는 3번 풍선, 2가 적혀 있는 2번 풍선의 순서대로 터지게 된다.

---

## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000)이 주어진다. 다음 줄에는 차례로 각 풍선 안의 종이에 적혀 있는 수가 주어진다. 종이에 0은 적혀있지 않다.

---

## 출력
첫째 줄에 터진 풍선의 번호를 차례로 나열한다.

## 풀이

### 풀이 핵심 로직
이 문제는 원형으로 연결된 풍선들을 터뜨리는 과정을 시뮬레이션하는 문제입니다. 각 풍선은 터질 때마다 그 안에 들어있는 수를 기반으로 다음 터질 풍선의 위치를 결정합니다. 이를 위해 덱(Deque)을 사용하여 풍선의 순서를 효율적으로 관리하고, 현재 터진 풍선은 덱에서 제거하여 이동의 대상에서 제외합니다.

### 동작 과정
1. **입력 예시**: 
   ```
   5
   3 2 1 -3 -1
   ```
   - 첫째 줄의 `5`는 총 5개의 풍선이 있다는 것을 의미합니다.
   - 둘째 줄의 `3 2 1 -3 -1`은 각각의 풍선에 적혀 있는 값입니다.

2. **초기화**:
   - 각 풍선의 인덱스와 종이의 값을 튜플로 묶어서 덱에 추가합니다.
   - 덱의 초기 상태: `[(0, 3), (1, 2), (2, 1), (3, -3), (4, -1)]`

3. **첫 번째 풍선 터트리기**:
   - 첫 번째로 1번 풍선(인덱스 0)을 터뜨립니다. 이 풍선의 값은 `3`입니다.
   - 결과 리스트에 `1`을 추가합니다.
   - 오른쪽으로 `3`번 이동하여 덱을 회전합니다. (이동 후의 덱 상태: `[(1, 2), (2, 1), (3, -3), (4, -1)]`)

4. **두 번째 풍선 터트리기**:
   - 다음으로 2번 풍선(인덱스 1)을 터뜨립니다. 이 풍선의 값은 `2`입니다.
   - 결과 리스트에 `2`를 추가합니다.
   - 오른쪽으로 `2`번 이동하여 덱을 회전합니다. (이동 후의 덱 상태: `[(3, -3), (4, -1), (2, 1)]`)

5. **세 번째 풍선 터트리기**:
   - 4번 풍선(인덱스 3)을 터뜨립니다. 값은 `-3`입니다.
   - 결과 리스트에 `4`를 추가합니다.
   - 왼쪽으로 `3`번 이동하여 덱을 회전합니다. (이동 후의 덱 상태: `[(2, 1), (3, -3), (4, -1)]`)

6. **이 과정을 반복**하여 5번 풍선, 3번 풍선, 1번 풍선을 차례로 터뜨리며 리스트에 추가합니다.

7. **최종 결과**: 터진 풍선의 번호가 차례로 출력됩니다: `1 2 4 5 3`.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 각 풍선을 터뜨릴 때마다 덱의 회전이 발생하는데, 최악의 경우 모든 풍선을 순회해야 하므로 이러한 복잡도가 발생합니다. 하지만 주어진 문제의 N은 최대 1000이므로 효율적으로 처리할 수 있습니다.

## 코드
```python
from collections import deque

n = int(input())
arr = list(map(int, input().split()))
q = deque()
a = 0
for i in arr:
    q.append((a, i))
    a += 1

ans = []
while q:
    idx, paper = q.popleft()
    ans.append(idx + 1)

    if paper > 0:
        q.rotate(-(paper - 1))
    elif paper < 0:
        q.rotate(-paper)

print(*ans)
```
