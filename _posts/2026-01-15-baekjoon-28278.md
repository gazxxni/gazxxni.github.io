## 문제
정수를 저장하는 스택을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 다섯 가지이다.


	1 X: 정수 X를 스택에 넣는다. (1 ≤ X ≤ 100,000)
	2: 스택에 정수가 있다면 맨 위의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	3: 스택에 들어있는 정수의 개수를 출력한다.
	4: 스택이 비어있으면 1, 아니면 0을 출력한다.
	5: 스택에 정수가 있다면 맨 위의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 스택을 구현하고 다양한 명령을 처리하는 프로그램을 작성하는 것이다. 명령은 스택에 정수를 삽입하거나, 삭제하고, 상태를 확인하는 기능을 포함한다. 이를 위해 파이썬의 리스트를 스택으로 활용하여 각 명령을 효율적으로 처리한다.

### 동작 과정
1. 첫째 줄에 명령의 수 N이 주어진다. 예를 들어, N=7일 경우 다음과 같은 명령이 입력될 수 있다:
    ```
    1 5
    1 10
    2
    3
    4
    5
    ```

2. 각 명령에 대한 처리 과정은 다음과 같다:
   - **명령 1 5**: 스택에 5를 넣는다. (스택 상태: [5])
   - **명령 1 10**: 스택에 10을 넣는다. (스택 상태: [5, 10])
   - **명령 2**: 스택에서 가장 위의 요소(10)를 제거하고 출력한다. 출력: 10 (스택 상태: [5])
   - **명령 3**: 스택에 있는 정수의 개수를 출력한다. 출력: 1
   - **명령 4**: 스택이 비어있는지 확인하여 출력한다. 비어있지 않으므로 출력: 0
   - **명령 5**: 스택의 가장 위의 요소(5)를 출력한다. 출력: 5

3. 명령어는 각 줄마다 처리되며, 출력이 요구되는 명령에 대해서는 결과를 즉시 출력한다.

### 시간 복잡도
각 명령은 O(1) 시간 복잡도를 가지므로, 전체 N개의 명령을 처리하는 시간 복잡도는 O(N)이다.

## 코드
```python
import sys

n = int(sys.stdin.readline())
stack = []

for _ in range(n):
    command = sys.stdin.readline().split()
    
    a = int(command[0])
    
    if a == 1:
        b = int(command[1])
        stack.append(b)
        
    elif a == 2:
        if stack:
            print(stack.pop())
        else:
            print(-1)
            
    elif a == 3:
        print(len(stack))
        
    elif a == 4:
        if not stack:
            print(1)
        else:
            print(0)
            
    elif a == 5:
        if stack:
            print(stack[-1])
        else:
            print(-1)
```
