## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이를 입력받아 삼각형의 성질을 판별하는 문제이다. 먼저, 세 변의 길이를 정렬하여 가장 긴 변을 쉽게 찾고, 삼각형의 조건을 만족하는지 확인한 후, 변의 길이에 따라 각각 Equilateral, Isosceles, Scalene, Invalid 중 하나의 결과를 출력한다.

### 동작 과정
1. 세 변의 길이를 입력받는다. 예를 들어, `6 3 2`를 입력받으면 이 줄에서 `0 0 0`이 아닌 경우에만 처리한다.
2. 입력한 변의 길이를 정렬하여 `a`, `b`, `c`로 나눈다. 이때, `c`는 가장 긴 변이 된다.
3. 삼각형의 조건을 확인한다. 가장 긴 변 `c`가 나머지 두 변 `a`와 `b`의 합보다 크거나 같으면 "Invalid"를 출력한다.
4. 세 변의 길이를 비교하여 다음과 같은 결과를 출력한다:
   - 모든 변이 같으면 "Equilateral"
   - 두 변이 같고 한 변이 다르면 "Isosceles"
   - 모든 변이 다르면 "Scalene"

예시 입력:
- `6 3 2` -> 정렬 후 `a=2`, `b=3`, `c=6`. 조건 `c < a + b`를 만족하지 않아 "Invalid" 출력.
- `3 3 3` -> 정렬 후 `a=3`, `b=3`, `c=3`. 모두 같으므로 "Equilateral" 출력.
- `5 5 3` -> 정렬 후 `a=3`, `b=5`, `c=5`. 두 변이 같으므로 "Isosceles" 출력.
- `4 5 6` -> 정렬 후 `a=4`, `b=5`, `c=6`. 모두 다르므로 "Scalene" 출력.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)이다. 입력으로 주어진 세 변의 길이만을 처리하므로 입력의 크기가 고정되어 있다. 각 입력에 대해 일정한 수의 연산만 수행하므로 입력 수에 비례하지 않고 상수 시간에 처리된다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
