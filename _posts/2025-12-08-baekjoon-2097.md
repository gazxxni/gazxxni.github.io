## 문제
당신은 N개의 조약돌을 가지고 있다. 이 조약돌을 좌표평면의 격자점 위에 아무렇게나 떨어뜨렸다. 격자점이란, x좌표와 y좌표 모두가 정수인 지점을 말한다. 이를테면 (1, 1)이나 (0, -9)는 격자점이며, (-2, 3.5)이나 (π, 7.14)는 격자점이 아니다.

모든 조약돌을 포함하는 가장 작은 직사각형을 생각할 수 있다. 예를 들어 세 개의 조약돌을 (2,4), (4, 8), (5,5)에 떨어뜨렸다면, 이 세 조약돌을 모두 포함하는 가장 작은 직사각형은 가로 3, 세로 4인 직사각형이다. 이 경우 직사각형의 둘레는 14가 된다. 직사각형의 가로와 세로 길이는 반드시 1 이상이어야 한다.

조약돌의 개수 N이 주어졌을 때, 조약돌을 좌표평면의 격자점에 적절히 떨어뜨려서 모든 조약돌을 포함하는 직사각형의 둘레를 최소로 하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 조약돌의 개수 N(1 ≤ n ≤ 500,000,000)이 주어진다.

---

## 출력
첫째 줄에 최소 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 조약돌을 포함하는 직사각형의 둘레를 최소화하기 위해 조약돌의 개수 N을 가능한 한 가로와 세로가 비슷한 형태로 배치하는 것이다. 직사각형의 가로와 세로의 길이는 각각 N의 제곱근에 가까운 값으로 설정하면, 둘레가 최소화된다.

### 동작 과정
1. **입력 처리**: 조약돌의 개수 N을 입력받는다.
2. **가로 길이 결정**: i를 1부터 시작하여 N의 제곱근에 가까운 값을 찾는다. 이때 `i * i < n` 조건을 만족하는 최대의 i를 찾는다.
3. **세로 길이 결정**: 조약돌의 개수 N을 i로 나누어 세로 길이 j를 계산한다. 이때 `j = ceil(n / i)`로 구한다. 이 값은 가로 길이 i에 대해 조약돌을 포함할 수 있는 최소한의 세로 길이를 보장한다.
4. **특별 케이스 처리**: 조약돌의 개수가 1부터 4일 경우, 직사각형의 둘레는 항상 4이다.
5. **둘레 계산**: 둘레는 `2 * (i + j - 2)`로 계산하여 출력한다.

**예시 입력**: N = 10
- i = 4 (가로 길이)
- j = ceil(10 / 4) = 3 (세로 길이)
- 둘레 = 2 * (4 + 3 - 2) = 12

**예시 입력**: N = 3
- i = 2 (가로 길이)
- j = ceil(3 / 2) = 2 (세로 길이)
- 둘레 = 2 * (2 + 2 - 2) = 4

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(√N)이다. 이는 i를 찾는 과정에서 최대 √N까지 반복하기 때문이다.

## 코드
```python
import sys, math
input = sys.stdin.readline

n = int(input())
i = 1

while i * i < n:
    i += 1

j = math.ceil(n / i)

if 1 <= n <= 4:
    print(4)
else:
    print(2 * (i + j - 2))
```
