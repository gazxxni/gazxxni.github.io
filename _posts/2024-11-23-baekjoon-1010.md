## 문제 링크
[https://www.acmicpc.net/problem/1010](https://www.acmicpc.net/problem/1010)

## 풀이

### 풀이 핵심 로직
이 문제는 조합(combination) 문제로, 주어진 n과 m에 대해 m개의 물건 중 n개를 선택하는 경우의 수를 구하는 것이다. 이를 위해 조합 공식인 C(m, n) = m! / (n! * (m-n)!)를 사용하여 계산한다. 파이썬의 기본 기능인 정수 연산을 활용하여 factorial 함수를 통해 팩토리얼 값을 계산한다.

### 동작 과정
1. 첫 번째 줄에서 테스트 케이스의 수 t를 입력받는다.
2. 각 테스트 케이스에 대해 n과 m을 입력받는다.
   - 예시 입력: 2 (테스트 케이스 수)
   - 예시 입력: 2 4 (첫 번째 테스트 케이스)
   - 예시 입력: 3 6 (두 번째 테스트 케이스)
3. factorial 함수를 통해 m, n, (m-n)의 팩토리얼 값을 계산한다.
   - 첫 번째 테스트 케이스: m=4, n=2
     - 4! = 24
     - 2! = 2
     - (4-2)! = 2! = 2
4. 조합 공식을 적용하여 결과를 계산한다.
   - C(4, 2) = 4! / (2! * (4-2)!) = 24 / (2 * 2) = 6
5. 계산된 결과를 출력한다.
   - 첫 번째 테스트 케이스 결과: 6
   - 두 번째 테스트 케이스도 같은 방식으로 계산하여 결과를 출력한다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(t * m)이다. 여기서 t는 테스트 케이스의 수, m은 가장 큰 입력의 값이다. 이는 factorial 함수를 호출할 때마다 m까지의 곱셈을 수행하는데, 최악의 경우 m에 비례하는 시간이 소요되기 때문이다.

## 코드
```python
import sys

t=int(input())

def factorial(n):
    num = 1
    for i in range(1, n+1):
        num *= i
    return num

for _ in range(t):
    n,m=map(int,input().split())
    ans=factorial(m)//(factorial(n)*factorial(m-n))
    print(ans)
```
