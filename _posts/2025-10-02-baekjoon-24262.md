## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    i = ⌊n / 2⌋;
    return A[i]; # 코드1
}

---

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
문제에서 주어진 MenOfPassion 알고리즘은 입력 배열 A의 중간 인덱스에 있는 요소를 반환하는 간단한 알고리즘입니다. 코드1에서 수행되는 작업은 배열 A의 특정 인덱스에 접근하는 것으로, 이 경우에는 항상 1회의 접근만 발생하므로 수행 횟수는 1입니다. 최고차항의 차수는 상수항이므로 0입니다.

### 동작 과정
1. 입력으로 주어진 n의 값을 받습니다.
2. 코드1에서 `A[i]`에 접근하는 작업이 수행됩니다. 이 때 i는 `⌊n / 2⌋`로 계산되며, 배열 A의 중간 요소에 접근합니다.
3. 따라서 코드1의 수행 횟수는 1회로 결정됩니다.
4. 다항식으로 나타낼 경우, 수행 횟수는 상수항이므로 최고차항의 차수는 0입니다.

예를 들어, n이 5일 경우:
- `i = ⌊5 / 2⌋ = 2`로 계산되어 `A[2]`에 접근하게 됩니다. 이 과정에서의 수행 횟수는 1회입니다.
- 최고차항의 차수는 0입니다.

### 시간 복잡도
O(1) - 입력 크기 n에 관계없이 항상 일정한 수행 시간을 가집니다.
