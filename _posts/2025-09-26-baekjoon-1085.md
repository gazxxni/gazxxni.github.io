## 문제 링크
[https://www.acmicpc.net/problem/1085](https://www.acmicpc.net/problem/1085)

---

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 직사각형의 경계와 점 (x, y) 사이의 최단 거리를 구하는 것입니다. 직사각형의 좌측 하단 꼭지점이 (0, 0), 우측 상단 꼭지점이 (w, h)일 때, 점 (x, y)에서 각 경계까지의 거리를 계산하고 그 중 최솟값을 반환합니다.

### 동작 과정
1. **입력값**: x, y, w, h를 입력받는다. 예를 들어, `x=3, y=4, w=10, h=10`이라고 가정하자.
2. **거리 계산**:
   - 좌측 경계까지의 거리: `c = abs(x - 0) = abs(3 - 0) = 3`
   - 우측 경계까지의 거리: `a = abs(x - w) = abs(3 - 10) = 7`
   - 하단 경계까지의 거리: `d = abs(y - 0) = abs(4 - 0) = 4`
   - 상단 경계까지의 거리: `b = abs(y - h) = abs(4 - 10) = 6`
3. **최솟값 계산**: `min(a, b, c, d)`를 통해 각 거리 중 가장 작은 값을 찾는다. 이 경우 `min(7, 6, 3, 4) = 3`이므로 최종적으로 `3`이 출력된다.

### 시간 복잡도
이 알고리즘은 상수 시간에 수행되므로 시간 복잡도는 O(1)입니다.

---

## 코드
```python
x, y, w, h = map(int, input().split())

a = abs(x - w)
b = abs(y - h)
c = abs(x)
d = abs(y)

print(min(a, b, c, d))
```
