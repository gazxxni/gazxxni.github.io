## 문제
문자열 S가 주어졌을 때, S의 서로 다른 부분 문자열의 개수를 구하는 프로그램을 작성하시오.

부분 문자열은 S에서 연속된 일부분을 말하며, 길이가 1보다 크거나 같아야 한다.

예를 들어, ababc의 부분 문자열은 a, b, a, b, c, ab, ba, ab, bc, aba, bab, abc, abab, babc, ababc가 있고, 서로 다른것의 개수는 12개이다.

---

## 입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있고, 길이는 1,000 이하이다.

---

## 출력
첫째 줄에 S의 서로 다른 부분 문자열의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 문자열 S에서 서로 다른 부분 문자열을 구하기 위해, 이중 반복문을 사용하여 모든 길이의 부분 문자열을 생성하고 이를 집합(set)에 저장합니다. 집합은 중복된 값을 허용하지 않기 때문에, 최종적으로 집합의 크기를 구하면 서로 다른 부분 문자열의 개수를 알 수 있습니다.

### 동작 과정
1. 입력 문자열 S가 "ababc"라고 가정합니다.
2. 길이 1의 부분 문자열을 생성합니다: `a`, `b`, `a`, `b`, `c` → 집합에 저장 → `{a, b, c}`
3. 길이 2의 부분 문자열을 생성합니다: `ab`, `ba`, `ab`, `bc` → 집합에 저장 → `{a, b, c, ab, ba, bc}`
4. 길이 3의 부분 문자열을 생성합니다: `aba`, `bab`, `abc` → 집합에 저장 → `{a, b, c, ab, ba, bc, aba, bab, abc}`
5. 길이 4의 부분 문자열을 생성합니다: `abab`, `babc` → 집합에 저장 → `{a, b, c, ab, ba, bc, aba, bab, abc, abab, babc}`
6. 길이 5의 부분 문자열을 생성합니다: `ababc` → 집합에 저장 → `{a, b, c, ab, ba, bc, aba, bab, abc, abab, babc, ababc}`
7. 최종적으로 집합의 크기를 출력하면, 서로 다른 부분 문자열의 개수는 12가 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n^2)입니다. 여기서 n은 문자열 S의 길이입니다. 이중 반복문을 통해 모든 부분 문자열을 생성하므로, 최대 n(n+1)/2개의 부분 문자열을 생성하는 데 O(n^2) 시간이 소요됩니다.

## 코드
```python
import math
from collections import defaultdict
s = input().rstrip()
a = set()
for i in range(1, len(s) + 1):
    for j in range(0, len(s)):
        a.add(s[j:j + i])
 
print(len(a))
```
