## 문제
알고리즘 입문방 오픈 채팅방에서는 새로운 분들이 입장을 할 때마다 곰곰티콘을 사용해 인사를 한다. 이를 본 문자열 킬러 임스는 채팅방의 기록을 수집해 그 중 곰곰티콘이 사용된 횟수를 구해 보기로 했다.

ENTER는 새로운 사람이 채팅방에 입장했음을 나타낸다. 그 외는 채팅을 입력한 유저의 닉네임을 나타낸다. 닉네임은 숫자 또는 영문 대소문자로 구성되어 있다.

새로운 사람이 입장한 이후 처음 채팅을 입력하는 사람은 반드시 곰곰티콘으로 인사를 한다. 그 외의 기록은 곰곰티콘을 쓰지 않은 평범한 채팅 기록이다.

채팅 기록 중 곰곰티콘이 사용된 횟수를 구해보자!

---

## 입력
첫 번째 줄에는 채팅방의 기록 수를 나타내는 정수 N$N$ 이 주어진다. (1≤N≤100000$1 \le N \le 100\,000$)

두 번째 줄부터 N$N$ 개의 줄에 걸쳐 새로운 사람의 입장을 나타내는 ENTER, 혹은 채팅을 입력한 유저의 닉네임이 문자열로 주어진다. (문자열길이1≤문자열 길이≤20$1 \le \texttt{문자열 길이} \le 20$)

첫 번째 주어지는 문자열은 무조건 ENTER이다.

---

## 출력
채팅 기록 중 곰곰티콘이 사용된 횟수를 출력하시오.

## 풀이

### 풀이 핵심 로직
우선, 채팅방에 새로 입장한 사용자가 인사하는 경우(즉, "ENTER"가 입력될 때)마다 현재까지 입장한 사용자들의 수만큼 곰곰티콘이 사용되었다고 카운트합니다. 사용자가 채팅을 입력하면 그 사용자의 닉네임을 집합에 추가하여 중복을 방지합니다. "ENTER"가 입력될 때마다 집합을 초기화하여 새로운 사용자 그룹을 관리합니다.

### 동작 과정
1. **입력 예시**: 
   ```
   7
   ENTER
   User1
   User2
   ENTER
   User3
   ENTER
   User4
   ```
   
2. **단계별 설명**:
   - 첫 번째 입력 `"ENTER"`: 아무도 없으므로 진행.
   - `"User1"` 입력: `arr` 집합에 `User1` 추가 → `arr = {User1}`.
   - `"User2"` 입력: `arr` 집합에 `User2` 추가 → `arr = {User1, User2}`.
   - 두 번째 입력 `"ENTER"`: 현재 `arr`의 길이 2를 `cnt`에 추가 (곰곰티콘 2회 사용) → `cnt = 2`. `arr` 초기화.
   - `"User3"` 입력: `arr` 집합에 `User3` 추가 → `arr = {User3}`.
   - 세 번째 입력 `"ENTER"`: 현재 `arr`의 길이 1을 `cnt`에 추가 (곰곰티콘 1회 사용) → `cnt = 3`. `arr` 초기화.
   - `"User4"` 입력: `arr` 집합에 `User4` 추가 → `arr = {User4}`.
   - 마지막으로 `arr`의 길이 1을 `cnt`에 추가 (곰곰티콘 1회 사용) → `cnt = 4`.

3. **출력**: 최종적으로 `cnt`는 4가 되어 출력됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 입력을 한 번씩만 처리하며, 집합에 대한 추가 및 초기화는 상수 시간에 이루어지기 때문에 전체적으로 선형 시간 복잡도를 가집니다.

## 코드
```python
n = int(input())

cnt = 0
arr = set()
for _ in range(n):
    a = input().rstrip()
    if a == 'ENTER':
        cnt += len(arr)
        arr = set()
        continue
    
    arr.add(a)

cnt += len(arr)
print(cnt)
```
