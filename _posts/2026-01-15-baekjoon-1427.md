## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 자연수 N의 각 자리수를 내림차순으로 정렬하기 위해, 먼저 입력된 수를 문자열로 받아 각 자리수를 리스트로 변환합니다. 이후 리스트를 내림차순으로 정렬한 후, 정렬된 리스트의 요소를 하나의 문자열로 합쳐 최종 결과를 출력합니다.

### 동작 과정
1. **입력 예시**: N = 2143
2. **문자열 변환**: 입력된 문자열 '2143'를 리스트로 변환하여 [2, 1, 4, 3]이 됩니다.
3. **정렬**: 리스트를 내림차순으로 정렬하여 [4, 3, 2, 1]이 됩니다.
4. **출력 형식**: 리스트의 요소를 하나의 문자열로 합쳐 '4321'을 출력합니다.

### 시간 복잡도
입력된 수 N의 자리수에 따라 정렬하는 과정의 시간 복잡도는 O(d log d)입니다. 여기서 d는 N의 자리수(최대 10자리)입니다. 따라서 현재 문제에서는 거의 상수 시간 복잡도로 간주할 수 있습니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
