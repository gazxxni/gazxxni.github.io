[https://www.acmicpc.net/problem/1427](https://www.acmicpc.net/problem/1427)

## 문제
배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.

---

## 입력
첫째 줄에 정렬하려고 하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.

---

## 출력
첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 자연수 N의 각 자리수를 분리하여 리스트로 만든 후, 이 리스트를 내림차순으로 정렬하고 다시 문자열 형태로 합쳐서 출력하는 방식으로 문제를 해결한다. 파이썬의 정렬 기능을 활용하여 간편하게 내림차순 정렬을 수행할 수 있다.

### 동작 과정
1. 입력 예시: `2143`
2. `input()` 함수를 사용하여 문자열로 입력을 받고, 이를 `strip()`으로 공백을 제거한다.
3. `map(int, n)`을 통해 각 자리수를 정수로 변환하여 리스트 `a`를 만든다. 이 경우 `a = [2, 1, 4, 3]`가 된다.
4. `a.sort(reverse=True)`를 호출하여 리스트를 내림차순으로 정렬한다. 정렬 후 `a = [4, 3, 2, 1]`이 된다.
5. `print(*a, sep='')`를 사용하여 리스트의 요소를 공백 없이 연결하여 출력한다. 최종 출력 결과는 `4321`이 된다.

### 시간 복잡도
주어진 문제의 시간 복잡도는 O(d log d)이다. 여기서 d는 N의 자리수의 개수(최대 10자리)이며, 정렬 알고리즘에 따라 로그 시간 복잡도가 추가된다. 하지만 상수에 가까운 작은 숫자 d에 대해 선형 시간 복잡도로 간주할 수 있다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n = input().strip()
a = list(map(int, n))
a.sort(reverse=True)
print(*a, sep='')
```
