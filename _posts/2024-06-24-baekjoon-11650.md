## 풀이

### 풀이 핵심 로직
주어진 문제는 2차원 평면에 있는 점들을 x좌표를 기준으로 오름차순으로 정렬하는 것입니다. 만약 x좌표가 같은 점들이 여러 개 있을 경우, 그 점들은 y좌표를 기준으로 오름차순으로 정렬됩니다. 이를 위해 각 점을 튜플로 저장한 후, 파이썬의 내장 정렬 기능을 이용해 간단히 정렬할 수 있습니다.

### 동작 과정
1. 먼저, 입력받을 점의 개수 `n`을 입력받습니다.
   - 예시 입력: `5`
   
2. 이후, `n`번 반복하여 각 점의 x, y 좌표를 입력받고, 이를 튜플 `(x, y)` 형태로 리스트 `a`에 추가합니다.
   - 예시 입력: 
     ```
     3 4
     1 2
     1 -1
     2 2
     3 3
     ```
   - 리스트 `a`의 내용: `[(3, 4), (1, 2), (1, -1), (2, 2), (3, 3)]`

3. 리스트 `a`를 정렬합니다. 기본적으로 파이썬의 정렬은 첫 번째 요소(x좌표)를 기준으로 정렬하며, 동일한 경우 두 번째 요소(y좌표)를 기준으로 정렬합니다.
   - 정렬 후 리스트 `a`: `[(1, -1), (1, 2), (2, 2), (3, 3), (3, 4)]`

4. 최종적으로 정렬된 리스트의 각 점을 출력합니다.
   - 출력 결과:
     ```
     1 -1
     1 2
     2 2
     3 3
     3 4
     ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. 이는 n개의 점을 정렬하는 데 필요한 시간 복잡도입니다. 각 점을 입력받는 과정은 O(n)이며, 정렬 과정이 O(n log n)으로 지배적이기 때문에 전체 시간 복잡도는 O(n log n)으로 표현할 수 있습니다.

## 코드
```python
import sys
input=sys.stdin.readline

n=int(input())
a=[]

for _ in range(n):
    x,y=map(int,input().split())
    a.append((x,y))
a.sort()

for i in a:
    print(i[0],i[1])  
```
