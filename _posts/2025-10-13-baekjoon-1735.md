## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 분수를 더하는 문제로, 분수를 동일한 분모로 변환한 뒤 분자를 합친 후, 결과 분수를 기약분수 형태로 출력하는 것이 핵심입니다. 유클리드 알고리즘을 사용하여 최대공약수(GCD)를 구하고, 최소공배수(LCM)를 통해 분모를 통일한 후, 새로운 분자를 계산하여 최종 결과를 도출합니다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 `a`와 `b`를, 두 번째 줄에서 `c`와 `d`를 입력받습니다. 이를 통해 두 분수 \( \frac{a}{b} \)와 \( \frac{c}{d} \)를 정의합니다.
   - 예시 입력: `1 2` (즉, \( \frac{1}{2} \))와 `1 3` (즉, \( \frac{1}{3} \))
   
2. **최소공배수 계산**: 두 분모 `b`와 `d`의 최소공배수 `n`을 계산합니다.
   - \( n = \text{lcm}(2, 3) = 6 \)

3. **분자 변환**: 각 분수를 통일된 분모 `n`으로 변환합니다. 
   - \( aa = n // b = 6 // 2 = 3 \)
   - \( bb = n // d = 6 // 3 = 2 \)
   - 새로운 분자는 \( cc = aa * a + bb * c = 3 * 1 + 2 * 1 = 5 \)

4. **기약분수로 변환**: 새로운 분자 `cc`와 분모 `n`의 최대공약수 `dd`를 구합니다. 이를 통해 기약분수 형태로 나누어줍니다.
   - \( dd = \text{gcd}(5, 6) = 1 \)

5. **결과 출력**: 최종 결과는 \( \frac{cc // dd}{n // dd} = \frac{5 // 1}{6 // 1} = \frac{5}{6} \)이므로, `5 6`을 출력합니다.

### 시간 복잡도
- GCD와 LCM 계산의 시간 복잡도는 \( O(\log(\min(x, y))) \)로, 여기서 \( x \)와 \( y \)는 분모 `b`와 `d`입니다. 따라서 전체 알고리즘의 시간 복잡도는 \( O(\log(\min(b, d))) \)입니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
