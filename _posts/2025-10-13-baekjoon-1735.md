## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 두 분수의 합을 구하기 위해 공통 분모를 사용하여 두 분수를 통합합니다. 그런 다음, 분자의 합을 구한 후 최대 공약수(GCD)로 분자와 분모를 나누어 기약분수 형태로 변환합니다.

### 동작 과정
1. 입력으로 두 개의 분수를 받습니다. 예를 들어, 첫 번째 분수는 1/2, 두 번째 분수는 1/3입니다.
2. 두 분수의 분모인 2와 3의 최소 공배수를 계산합니다. 이 경우, LCM(2, 3)는 6입니다.
3. 각 분수를 공통 분모로 변환합니다:
   - 첫 번째 분수 1/2를 6으로 변환하면 (1 * 3)/(2 * 3) = 3/6
   - 두 번째 분수 1/3을 6으로 변환하면 (1 * 2)/(3 * 2) = 2/6
4. 이제 두 분수의 분자를 더합니다: 3 + 2 = 5.
5. 결과 분수는 5/6입니다. 이를 기약분수 형태로 만들기 위해 GCD(5, 6)를 구합니다. GCD는 1이므로, 5와 6은 이미 기약분수입니다.
6. 최종적으로 5와 6을 출력합니다.

### 시간 복잡도
O(log(min(b, d))) - GCD를 계산하는 데 유클리드 호제법을 사용하므로, 이 방법의 시간 복잡도는 로그 시간에 비례합니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
