## 문제
정수를 저장하는 큐를 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 여섯 가지이다.


	push X: 정수 X를 큐에 넣는 연산이다.
	pop: 큐에서 가장 앞에 있는 정수를 빼고, 그 수를 출력한다. 만약 큐에 들어있는 정수가 없는 경우에는 -1을 출력한다.
	size: 큐에 들어있는 정수의 개수를 출력한다.
	empty: 큐가 비어있으면 1, 아니면 0을 출력한다.
	front: 큐의 가장 앞에 있는 정수를 출력한다. 만약 큐에 들어있는 정수가 없는 경우에는 -1을 출력한다.
	back: 큐의 가장 뒤에 있는 정수를 출력한다. 만약 큐에 들어있는 정수가 없는 경우에는 -1을 출력한다.

---

## 입력
첫째 줄에 주어지는 명령의 수 N (1 ≤ N ≤ 2,000,000)이 주어진다. 둘째 줄부터 N개의 줄에는 명령이 하나씩 주어진다. 주어지는 정수는 1보다 크거나 같고, 100,000보다 작거나 같다. 문제에 나와있지 않은 명령이 주어지는 경우는 없다.

---

## 출력
출력해야하는 명령이 주어질 때마다, 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 명령어에 따라 정수를 저장하는 큐를 구현하는 것입니다. Python의 `collections` 모듈에 있는 `deque`를 사용하여 큐의 성능을 최적화하고, 각각의 명령어를 처리하여 결과를 출력합니다. `deque`는 양쪽 끝에서 O(1) 시간 복잡도로 삽입과 삭제가 가능하므로 큐의 요구 사항을 효율적으로 만족합니다.

### 동작 과정
1. **입력 처리**: 첫째 줄에서 명령의 수 N을 입력받고, 이후 N개의 줄에서 각각의 명령어를 입력받습니다.
2. **명령어 처리**:
   - `push X`: 입력받은 X를 큐에 추가합니다.
   - `pop`: 큐의 가장 앞에 있는 요소를 제거하고 출력합니다. 큐가 비어있다면 -1을 출력합니다.
   - `size`: 큐에 있는 요소의 개수를 출력합니다.
   - `empty`: 큐가 비어있으면 1을, 아니면 0을 출력합니다.
   - `front`: 큐의 가장 앞에 있는 요소를 출력합니다. 비어있으면 -1을 출력합니다.
   - `back`: 큐의 가장 뒤에 있는 요소를 출력합니다. 비어있으면 -1을 출력합니다.

예시 입력:
```
7
push 1
push 2
pop
size
empty
front
back
```
예시 출력:
```
1
1
0
2
2
```

- 첫 번째 명령어 `push 1`으로 큐에 1이 추가됩니다.
- 두 번째 명령어 `push 2`로 큐에 2가 추가됩니다.
- `pop` 명령어는 큐에서 1을 제거하고 출력합니다.
- `size`는 현재 큐에 1개의 요소가 남아 있으므로 1을 출력합니다.
- `empty`는 큐가 비어있지 않으므로 0을 출력합니다.
- `front`는 큐의 앞에 있는 2를 출력합니다.
- `back`도 큐의 뒤에 있는 2를 출력합니다.

### 시간 복잡도
모든 명령어는 O(1) 시간 복잡도로 처리할 수 있으므로, 전체적인 시간 복잡도는 O(N)입니다. N은 주어진 명령어의 수입니다.

## 코드
```python
import sys
from collections import deque
input = sys.stdin.readline
n = int(input())
q = deque()

for _ in range(n):
    command = input().split()
    
    a = command[0]
    
    if a == 'push':
        b = int(command[1])
        q.append(b)
        
    elif a == 'pop':
        if q:
            print(q.popleft())
        else:
            print(-1)
            
    elif a == 'size':
        print(len(q))
        
    elif a == 'empty':
        if not q:
            print(1)
        else:
            print(0)
            
    elif a == 'front':
        if q:
            print(q[0])
        else:
            print(-1)
            
    elif a == 'back':
        if q:
            print(q[-1])
        else:
            print(-1)
```
