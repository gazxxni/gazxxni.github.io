[https://www.acmicpc.net/problem/17387](https://www.acmicpc.net/problem/17387)

## 문제
2차원 좌표 평면 위의 두 선분 L1, L2가 주어졌을 때, 두 선분이 교차하는지 아닌지 구해보자. 한 선분의 끝 점이 다른 선분이나 끝 점 위에 있는 것도 교차하는 것이다.

L1의 양 끝 점은 (x1, y1), (x2, y2), L2의 양 끝 점은 (x3, y3), (x4, y4)이다.

---

## 입력
첫째 줄에 L1의 양 끝 점 x1, y1, x2, y2가, 둘째 줄에 L2의 양 끝 점 x3, y3, x4, y4가 주어진다.

---

## 출력
L1과 L2가 교차하면 1, 아니면 0을 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

## 풀이

### 풀이 핵심 로직
선분 교차 문제를 해결하기 위해서 두 선분의 방향성을 비교하는 ccw(시계 반대 방향) 함수를 사용합니다. 이를 통해 선분의 끝 점들이 서로의 선분에 대해 어떤 위치에 있는지를 판단할 수 있습니다. 또한, 선분이 일직선상에 있을 때는 두 선분의 교차 여부를 확인하기 위해 끝 점들이 서로의 선분 구간 안에 위치하는지를 검사합니다.

### 동작 과정
1. 두 선분 L1 (x1, y1) -> (x2, y2)와 L2 (x3, y3) -> (x4, y4)의 끝 점을 입력받습니다.
2. ccw 함수를 사용하여 각 선분 끝 점에 대한 방향성을 계산합니다:
   - o1 = ccw(x1, y1, x2, y2, x3, y3)
   - o2 = ccw(x1, y1, x2, y2, x4, y4)
   - o3 = ccw(x3, y3, x4, y4, x1, y1)
   - o4 = ccw(x3, y3, x4, y4, x2, y2)
3. 모든 ccw 값이 0인 경우(일직선 상에 있는 경우)에는 선분이 서로 겹치는지를 확인합니다.
   - 두 선분의 교차 여부를 확인하기 위해 각 선분의 x와 y 좌표의 범위를 비교합니다.
4. ccw 값 중 두 개가 0 이하이고 나머지 두 개가 0 이상인 경우 (즉, 선분이 서로 교차하는 경우) 1을 출력하고, 그렇지 않으면 0을 출력합니다.

**예시 입력:**
```
0 0 2 2
1 1 3 3
```
- L1의 끝 점은 (0, 0)과 (2, 2), L2의 끝 점은 (1, 1)과 (3, 3)입니다.
- 모든 ccw 값이 0이므로 두 선분은 일직선 위에 있습니다.
- (0, 0)과 (2, 2) 모두 (1, 1)과 (3, 3) 구간 안에 위치하므로, 출력은 1입니다.

### 시간 복잡도
O(1) - 입력되는 좌표의 개수가 정해져 있어 상수 시간 내에 결과를 도출할 수 있습니다.

## 코드
```python
import sys
input = sys.stdin.readline

def ccw(xa, ya, xb, yb, xc, yc):
    cross = (xb - xa) * (yc - ya) - (yb - ya) * (xc - xa)
    return cross

x1, y1, x2, y2 = map(int, input().split())
x3, y3, x4, y4 = map(int, input().split())

o1 = ccw(x1, y1, x2, y2, x3, y3)
o2 = ccw(x1, y1, x2, y2, x4, y4)
o3 = ccw(x3, y3, x4, y4, x1, y1)
o4 = ccw(x3, y3, x4, y4, x2, y2)

if o1 == 0 and o2 == 0 and o3 == 0 and o4 == 0:
    if (x1, y1) > (x2, y2):
        x1, x2 = x2, x1
        y1, y2 = y2, y1
    if (x3, y3) > (x4, y4):
        x3, x4 = x4, x3
        y3, y4 = y4, y3

    
    if max(min(x1,x2), min(x3,x4)) <= min(max(x1,x2), max(x3,x4)):
        if max(min(y1,y2), min(y3,y4)) <= min(max(y1,y2), max(y3,y4)):
            print(1)
            exit()
        else:
            print(0)
            exit()
    else:
        print(0)
        exit()

if o1 * o2 <= 0 and o3 * o4 <= 0:
    print(1)
    exit()
else:
    print(0)
    exit()

```
