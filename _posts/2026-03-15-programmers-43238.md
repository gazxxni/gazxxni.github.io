## 문제
n명이 입국심사를 위해 줄을 서서 기다리고 있습니다. 각 입국심사대에 있는 심사관마다 심사하는데 걸리는 시간은 다릅니다. 

처음에 모든 심사대는 비어있습니다. 한 심사대에서는 동시에 한 명만 심사를 할 수 있습니다. 가장 앞에 서 있는 사람은 비어 있는 심사대로 가서 심사를 받을 수 있습니다. 하지만 더 빨리 끝나는 심사대가 있으면 기다렸다가 그곳으로 가서 심사를 받을 수도 있습니다.

모든 사람이 심사를 받는데 걸리는 시간을 최소로 하고 싶습니다.

입국심사를 기다리는 사람 수 n, 각 심사관이 한 명을 심사하는데 걸리는 시간이 담긴 배열 times가 매개변수로 주어질 때, 모든 사람이 심사를 받는데 걸리는 시간의 최솟값을 return 하도록 solution 함수를 작성해주세요.

제한사항


입국심사를 기다리는 사람은 1명 이상 1,000,000,000명 이하입니다.
각 심사관이 한 명을 심사하는데 걸리는 시간은 1분 이상 1,000,000,000분 이하입니다.
심사관은 1명 이상 100,000명 이하입니다.


입출력 예

        
n
times
return


        
6
[7, 10]
28


      
입출력 예 설명

가장 첫 두 사람은 바로 심사를 받으러 갑니다. 

7분이 되었을 때, 첫 번째 심사대가 비고 3번째 사람이 심사를 받습니다. 

10분이 되었을 때, 두 번째 심사대가 비고 4번째 사람이 심사를 받습니다.

14분이 되었을 때, 첫 번째 심사대가 비고 5번째 사람이 심사를 받습니다.

20분이 되었을 때, 두 번째 심사대가 비지만 6번째 사람이 그곳에서 심사를 받지 않고 1분을 더 기다린 후에 첫 번째 심사대에서 심사를 받으면 28분에 모든 사람의 심사가 끝납니다.

## 풀이
### 풀이 핵심 로직
이 문제의 핵심 아이디어는 이분 탐색을 사용하여 모든 사람이 심사를 받는 최소 시간을 찾는 것입니다. 심사관의 수와 각 심사관의 심사 시간을 고려하여 주어진 시간 내에 몇 명이 심사를 받을 수 있는지를 계산합니다. 이 정보를 바탕으로 가능한 최소 시간을 좁혀가는 방식으로 해결합니다.

### 동작 과정
1. **초기값 설정**: `st`는 1로 설정하고, `ed`는 `n * max(times)`로 설정합니다. 여기서 `n`은 대기 인원 수, `max(times)`는 최장 심사 시간입니다.
   
2. **이분 탐색 수행**:
   - `mid`를 `(st + ed) // 2`로 계산합니다. 이 `mid`는 현재 고려 중인 시간입니다.
   - 각 심사관에 대해 `mid` 시간 내에 몇 명을 심사할 수 있는지 계산합니다. 각 심사관 `t`에 대해 `mid // t`를 더해가며 총 심사 가능한 인원 수 `p`를 계산합니다.
   - 만약 `p`가 `n`보다 크거나 같다면, 현재 `mid` 시간은 가능하므로 `ans`를 `mid`로 갱신하고 `ed`를 `mid - 1`로 줄입니다. 그렇지 않으면 `st`를 `mid + 1`로 늘립니다.

3. **결과 반환**: 이분 탐색이 끝나면 `ans`에 저장된 값이 모든 사람이 심사를 받는 최소 시간이 됩니다.

예를 들어, `n = 6`이고 `times = [7, 10]`인 경우:
- 처음 `st = 1`, `ed = 60` (6 * 10)
- 이분 탐색을 통해 `mid`를 계산하고 심사 가능한 인원 수를 확인하면서 `st`와 `ed`의 범위를 조정합니다.
- 결국 모든 사람이 심사를 받는데 걸리는 최소 시간인 `28`을 찾게 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 `O(m log k)`입니다. 여기서 `m`은 심사관의 수(`len(times)`), `k`는 최대 심사 시간이 `n * max(times)`로 결정되므로, 실제로는 매우 큰 수입니다. 이분 탐색이 로그 시간 복잡도를 가지므로 전체적으로 효율적인 방식입니다.

## 코드
```python
def solution(n, times):
    ans = 0
    st, ed = 1, n * max(times)
    
    while st <= ed:
        mid = (st + ed) // 2
        
        p = 0
        for t in times:
            p += mid // t
            
            if p >= n:
                break
                
        if p >= n:
            ans = mid
            ed = mid - 1
        else:
            st = mid + 1
    
    
    return ans
```
