## 문제
네트워크란 컴퓨터 상호 간에 정보를 교환할 수 있도록 연결된 형태를 의미합니다. 예를 들어, 컴퓨터 A와 컴퓨터 B가 직접적으로 연결되어있고, 컴퓨터 B와 컴퓨터 C가 직접적으로 연결되어 있을 때 컴퓨터 A와 컴퓨터 C도 간접적으로 연결되어 정보를 교환할 수 있습니다. 따라서 컴퓨터 A, B, C는 모두 같은 네트워크 상에 있다고 할 수 있습니다.

컴퓨터의 개수 n, 연결에 대한 정보가 담긴 2차원 배열 computers가 매개변수로 주어질 때, 네트워크의 개수를 return 하도록 solution 함수를 작성하시오.

제한사항


컴퓨터의 개수 n은 1 이상 200 이하인 자연수입니다.
각 컴퓨터는 0부터 n-1인 정수로 표현합니다.
i번 컴퓨터와 j번 컴퓨터가 연결되어 있으면 computers[i][j]를 1로 표현합니다.
computer[i][i]는 항상 1입니다.

n
computers
return

## 풀이
### 풀이 핵심 로직
이 문제는 깊이 우선 탐색(DFS) 또는 너비 우선 탐색(BFS)을 사용하여 연결된 네트워크의 개수를 찾는 문제입니다. 각 컴퓨터가 연결된 상태를 그래프로 표현하고, 방문하지 않은 컴퓨터를 시작점으로 하여 탐색을 진행합니다. 탐색 시 모든 연결된 컴퓨터를 방문하면 하나의 네트워크를 발견한 것이므로, 방문할 때마다 네트워크의 개수를 증가시킵니다.

### 동작 과정
예를 들어, 입력이 다음과 같다고 가정하겠습니다.

```
n = 3
computers = [[1, 1, 0], 
             [1, 1, 0], 
             [0, 0, 1]]
```

1. `visited` 리스트를 초기화합니다: `[False, False, False]`.
2. `cnt` 변수를 0으로 초기화합니다. 이 변수는 네트워크의 개수를 셉니다.
3. 첫 번째 컴퓨터(인덱스 0)부터 시작하여, `visited[0]`가 False이므로 BFS를 호출합니다.
   - BFS는 컴퓨터 0에서 시작하여, 연결된 컴퓨터 1을 방문합니다.
   - BFS가 종료되면 `visited`는 `[True, True, False]`로 업데이트됩니다.
   - `cnt`는 1 증가하여 1이 됩니다.
4. 다음 인덱스인 1은 이미 방문했으므로 건너뜁니다.
5. 인덱스 2를 확인하면 `visited[2]`가 False이므로 다시 BFS를 호출합니다.
   - 컴퓨터 2는 자기 자신만 연결되어 있으므로 탐색 후 `visited`는 `[True, True, True]`가 됩니다.
   - `cnt`는 1 증가하여 2가 됩니다.
6. 모든 컴퓨터를 탐색했으므로 최종적으로 `cnt` 값을 반환합니다. 결과는 2입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^2)입니다. 여기서 N은 컴퓨터의 개수입니다. 모든 컴퓨터를 한 번씩 방문하고, 각 컴퓨터에 대해 연결된 컴퓨터를 확인하므로 O(N^2)의 복잡도를 가집니다.

## 코드
```python
from collections import deque

def solution(n, computers):
    visited = [False] * n
    
    def bfs(start):
        q = deque([start])
        visited[start] = True
        
        while q:
            cur = q.popleft()
            
            for next in range(n):
                if not visited[next] and cur != next and computers[cur][next] == 1:
                    q.append(next)
                    visited[next] = True
                    
    cnt = 0
    for i in range(n):
        if not visited[i]:
            bfs(i)
            cnt += 1
        
    
    return cnt
```
