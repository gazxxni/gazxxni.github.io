## 문제
영선이는 길이가 a, b, c인 세 막대를 가지고 있고, 각 막대의 길이를 마음대로 줄일 수 있다.

영선이는 세 막대를 이용해서 아래 조건을 만족하는 삼각형을 만들려고 한다.


	각 막대의 길이는 양의 정수이다
	세 막대를 이용해서 넓이가 양수인 삼각형을 만들 수 있어야 한다.
	삼각형의 둘레를 최대로 해야 한다.


a, b, c가 주어졌을 때, 만들 수 있는 가장 큰 둘레를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 100)가 주어진다.

---

## 출력
첫째 줄에 만들 수 있는 가장 큰 삼각형의 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 세 막대의 길이로 삼각형을 만들기 위해서는 삼각형의 성질인 두 변의 길이의 합이 나머지 한 변의 길이보다 커야 한다. 따라서, 가장 긴 막대를 기준으로 나머지 두 막대의 길이를 조정하여 최대의 둘레를 구하는 것이 중요하다. 만약 두 막대의 합이 가장 긴 막대의 길이보다 작거나 같다면, 둘레를 최대화하기 위해 두 막대의 길이를 조정하여 가능한 최대의 둘레를 계산한다.

### 동작 과정
1. 입력으로 세 막대의 길이 \(a\), \(b\), \(c\)를 받는다.
2. 세 막대의 길이를 정렬하여 \(a \leq b \leq c\)로 만든다.
3. 삼각형을 만들 수 있는지를 확인한다:
   - 만약 \(a + b \leq c\)라면, 두 막대의 합이 가장 긴 막대의 길이보다 작거나 같으므로, 막대의 길이를 줄여서 \(a + b\)의 최대값을 만들고, 이때의 둘레는 \((a + b) \times 2 - 1\)이 된다.
   - 만약 \(a\), \(b\), \(c\)가 모두 같다면(즉, 정삼각형의 경우), 둘레는 \(a \times 3\)이 된다.
   - 그렇지 않으면 세 막대 그대로 사용할 수 있으므로, 둘레는 \(a + b + c\)가 된다.

예시:
- 입력: \(3, 4, 5\)
  - 정렬 후: \(3, 4, 5\)
  - \(3 + 4 > 5\)이므로, 둘레는 \(3 + 4 + 5 = 12\)
  
- 입력: \(1, 1, 3\)
  - 정렬 후: \(1, 1, 3\)
  - \(1 + 1 \leq 3\)이므로, 둘레는 \((1 + 1) \times 2 - 1 = 3\)

### 시간 복잡도
O(1) - 입력의 크기가 상수이므로, 고정된 시간 내에 계산이 완료된다.

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
