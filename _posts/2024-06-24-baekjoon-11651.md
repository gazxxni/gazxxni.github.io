## 풀이

### 풀이 핵심 로직
문제는 주어진 좌표를 y좌표를 기준으로 오름차순으로 정렬하고, y좌표가 같을 경우 x좌표를 기준으로 정렬하는 것입니다. 이를 위해 Python의 내장 sort 함수를 사용하며, 정렬 기준을 튜플로 설정하여 간단히 구현합니다.

### 동작 과정
1. 입력된 좌표의 수 `n`을 읽습니다.
2. `n`개의 좌표를 입력받아 리스트 `a`에 튜플 형태로 저장합니다.
   - 예시 입력: 
     ```
     5
     0 3
     1 2
     1 -1
     2 2
     3 3
     ```
   - 리스트 `a`는 다음과 같이 구성됩니다: `[(0, 3), (1, 2), (1, -1), (2, 2), (3, 3)]`
3. 리스트 `a`를 정렬합니다. 정렬 기준은 첫 번째로 y좌표 (x[1]), 두 번째로 x좌표 (x[0])입니다.
   - 정렬 후 리스트 `a`는 다음과 같이 됩니다: `[(1, -1), (1, 2), (2, 2), (0, 3), (3, 3)]`
4. 정렬된 결과를 출력합니다:
   - 출력 결과:
     ```
     1 -1
     1 2
     2 2
     0 3
     3 3
     ```

### 시간 복잡도
정렬에 사용된 `sort()` 함수의 시간 복잡도는 O(n log n)입니다. 여기서 n은 입력된 좌표의 수입니다. 입력과 출력에 대한 시간 복잡도는 O(n)으로, 전체 시간 복잡도는 O(n log n)입니다.

## 코드
```python
import sys
input=sys.stdin.readline

n=int(input())
a=[]

for _ in range(n):
    x,y=map(int, input().split())
    a.append((x,y))
    
a.sort(key=lambda x:(x[1],x[0]))

for i in a:
    print(i[0],i[1])

```
