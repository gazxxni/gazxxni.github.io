## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 풀이

### 풀이 핵심 로직
팩토리얼(N!)은 N과 그보다 작은 모든 양의 정수의 곱으로 정의된다. 이를 재귀적으로 정의하면, N! = N * (N-1)! 이며, 0!은 1로 정의된다. 이 문제에서는 재귀 함수를 사용하여 N의 팩토리얼 값을 계산한다.

### 동작 과정
1. 사용자로부터 정수 N을 입력받는다. 예를 들어, N이 5일 경우 입력은 `5`이다.
2. `fac`라는 재귀 함수를 정의한다. 이 함수는 입력값 N이 1일 때 1을 반환하고, 그렇지 않을 경우 N과 `fac(N-1)`의 곱을 반환한다.
3. 만약 N이 0일 경우, 팩토리얼 값은 1이므로 바로 1을 출력한다.
4. N이 1보다 클 경우, `fac(n)` 함수를 호출하여 N의 팩토리얼 값을 계산하고 출력한다.

예시:
- 입력: `5`
- 동작: 
  - `fac(5)` 호출 → 5 * `fac(4)`
  - `fac(4)` 호출 → 4 * `fac(3)`
  - `fac(3)` 호출 → 3 * `fac(2)`
  - `fac(2)` 호출 → 2 * `fac(1)`
  - `fac(1)` 호출 → 1 반환
  - 따라서, `fac(2)`는 2, `fac(3)`은 6, `fac(4)`는 24, `fac(5)`는 120이 되어 최종적으로 120이 출력된다.

### 시간 복잡도
시간 복잡도는 O(N)이다. N의 크기만큼 재귀 호출이 발생하므로, 각 호출에서 상수 시간만큼의 작업을 수행한다.

---

## 코드
```python
n = int(input())

def fac(n):
    if n == 1:
        return 1
    
    return n * fac(n - 1)

if n == 0:
    print(1)
else:
    print(fac(n))
```
