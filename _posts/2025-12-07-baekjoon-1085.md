## 문제
한수는 지금 (x, y)에 있다. 직사각형은 각 변이 좌표축에 평행하고, 왼쪽 아래 꼭짓점은 (0, 0), 오른쪽 위 꼭짓점은 (w, h)에 있다. 직사각형의 경계선까지 가는 거리의 최솟값을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 x, y, w, h가 주어진다.

---

## 출력
첫째 줄에 문제의 정답을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 직사각형의 경계선까지 가는 최소 거리를 계산하기 위해, 현재 위치 (x, y)에서 직사각형의 각 네 변까지의 거리를 구합니다. 이 네 변은 각각 x축의 0, w와 y축의 0, h로 정의되며, 각각의 거리 중 최솟값을 선택하여 결과를 도출합니다.

### 동작 과정
1. 입력으로 (x, y, w, h) 값을 받습니다. 예를 들어, 입력이 `6 2 10 3`일 경우:
   - x = 6
   - y = 2
   - w = 10
   - h = 3
2. 각 변까지의 거리를 계산합니다:
   - 왼쪽 변까지의 거리: `abs(x - 0) = abs(6 - 0) = 6`
   - 오른쪽 변까지의 거리: `abs(x - w) = abs(6 - 10) = 4`
   - 아래 변까지의 거리: `abs(y - 0) = abs(2 - 0) = 2`
   - 위 변까지의 거리: `abs(y - h) = abs(2 - 3) = 1`
3. 계산된 거리들: [6, 4, 2, 1]
4. 이 중 최솟값인 `1`을 출력합니다.

### 시간 복잡도
O(1) - 입력값의 개수가 고정되어 있어 상수 시간에 결과를 도출할 수 있습니다.

---

## 코드
```python
x, y, w, h = map(int, input().split())

a = abs(x - w)
b = abs(y - h)
c = abs(x)
d = abs(y)

print(min(a, b, c, d))
```
