## 문제
영어 소문자와 대문자로 이루어진 단어를 입력받은 뒤, 대문자는 소문자로, 소문자는 대문자로 바꾸어 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 영어 소문자와 대문자로만 이루어진 단어가 주어진다. 단어의 길이는 최대 100이다.

---

## 출력
첫째 줄에 입력으로 주어진 단어에서 대문자는 소문자로, 소문자는 대문자로 바꾼 단어를 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 문자열의 각 문자를 순회하면서, 대문자인 경우 소문자로 변환하고, 소문자인 경우 대문자로 변환하여 새로운 문자열을 구성합니다. 이를 위해 Python의 `isupper()`와 `islower()` 메서드를 사용하여 문자의 대소문자를 판별하고, `lower()`와 `upper()` 메서드를 통해 변환을 수행합니다.

### 동작 과정
1. 입력으로 "HelloWorld"라는 문자열이 주어졌다고 가정합니다.
2. 첫 번째 문자 'H'는 대문자이므로 `lower()` 메서드를 통해 'h'로 변환합니다.
3. 두 번째 문자 'e'는 소문자이므로 `upper()` 메서드를 통해 'E'로 변환합니다.
4. 같은 방식으로 나머지 문자들도 순서대로 변환합니다:
   - 'l' → 'L'
   - 'l' → 'L'
   - 'o' → 'O'
   - 'W' → 'w'
   - 'o' → 'O'
   - 'r' → 'R'
   - 'l' → 'L'
   - 'd' → 'D'
5. 최종적으로 변환된 문자열 'hELLOwORLD'가 출력됩니다.

### 시간 복잡도
O(n), 여기서 n은 입력 문자열의 길이입니다. 각 문자를 한 번씩만 검사하고 변환하기 때문에 선형 시간 복잡도를 가집니다.

---

## 코드
```python
for i in input():
    if i.isupper():
        print(i.lower(), end='')
    else:
        print(i.upper(), end='')
```
