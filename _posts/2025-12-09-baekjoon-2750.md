## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 110760 KB, 시간: 100 ms

---

## 풀이

### 풀이 핵심 로직
주어진 N개의 정수를 오름차순으로 정렬하기 위해 Python의 내장 함수 `sort()`를 사용합니다. 이 함수는 Timsort 알고리즘을 기반으로 하며, 효율적으로 데이터를 정렬합니다. 입력받은 수들은 중복되지 않으므로, 단순히 정렬 후 출력하면 됩니다.

### 동작 과정
1. 첫 번째 줄에서 수의 개수 N을 입력받습니다.
2. 다음 N개의 줄에서 각각의 정수를 입력받아 리스트 `arr`에 저장합니다.
3. `arr.sort()`를 호출하여 리스트를 오름차순으로 정렬합니다.
4. 정렬된 리스트의 각 요소를 한 줄에 하나씩 출력합니다.

예시 입력:
```
5
5
3
1
4
2
```
- N은 5이고, 입력된 수들은 [5, 3, 1, 4, 2]입니다.
- `arr.sort()`를 호출하면 리스트는 [1, 2, 3, 4, 5]로 정렬됩니다.
- 이후 출력 과정에서 각 요소가 순서대로 출력됩니다:
```
1
2
3
4
5
```

### 시간 복잡도
정렬 알고리즘의 평균 및 최악의 시간 복잡도는 O(N log N)입니다. 따라서 이 문제의 시간 복잡도는 O(N log N)입니다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
