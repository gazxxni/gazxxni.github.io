## 문제
세준이는 N개의 영어 단어를 이용해 길이가 M인 새로운 단어를 만들려고 한다. 새로운 단어는 N개의 단어를 순서대로 이어 붙이고, 각 단어의 사이에 _을 넣어서 만든다. 이렇게 만든 새로운 단어의 길이가 M이 아닌 경우 _를 추가해서 길이가 M이 되게 만들어야 한다.


	_는 단어와 단어 사이에만 추가할 수 있다. 따라서, 새로운 단어는 _으로 시작하거나, _로 끝날 수 없다.
	단어와 단어 사이에 있는 _의 개수는 모두 같아야 한다.
	
		모두 같게 만드는 것이 불가능한 경우 단어와 단어 사이에 있는 _의 개수의 최댓값과 최솟값의 차이는 1이 되어야 한다.
	
	


새로운 단어 중 사전 순으로 가장 앞서는 단어를 구해보자.

---

## 입력
첫째 줄에 N과 M이 주어진다. 둘째 줄부터 N개의 줄에 단어가 한 줄에 하나씩 주어진다.

---

## 출력
첫째 줄에 사전 순으로 가장 앞서는 단어를 출력한다.

알파벳 대문자, 소문자, 밑 줄의 순서는 'A'

## 풀이

### 풀이 핵심 로직
이 문제의 핵심은 주어진 N개의 단어를 이어붙이고, 단어 사이에 넣을 밑줄(_)의 개수를 조정하여 최종 길이가 M이 되도록 하는 것입니다. 단어 사이의 밑줄 개수는 모두 같아야 하며, 불가능한 경우에는 최대-최소 차이가 1이 되어야 합니다. 최종적으로 사전 순으로 가장 앞서는 문자열을 생성하는 것이 목표입니다.

### 동작 과정
1. **입력 받기**: N과 M을 입력받고, N개의 단어를 리스트에 저장합니다.
2. **길이 계산**: 모든 단어의 길이를 합산한 후, M에서 이 총 길이를 빼서 넣어야 할 밑줄(_)의 총 개수를 구합니다.
3. **밑줄 분배**: `greedy` 함수를 통해 남은 밑줄을 N-1개의 구간에 분배합니다. 이때, 각 구간에 넣는 밑줄의 개수는 최대한 균일하게 나누되, 불균형이 발생할 경우 최대-최소의 차이가 1이 되도록 합니다.
   - 먼저, 기본적으로 각 구간에 `a // b` 만큼의 밑줄을 넣고, 남은 밑줄을 순차적으로 각 구간에 추가합니다.
4. **최종 문자열 생성**: 첫 번째 단어를 시작으로, 각 단어와 그 사이에 계산된 밑줄을 추가하여 최종 문자열을 완성합니다.
5. **출력**: 최종적으로 만들어진 문자열을 출력합니다.

예시 입력으로 `3 20`과 단어 `a`, `b`, `c`가 주어진 경우:
- 총 단어 길이는 3이므로, 남은 길이는 20 - 3 = 17입니다.
- 밑줄은 17을 2(N-1)로 나누면 8, 나머지 1이 생깁니다. 따라서 두 구간에 8개의 밑줄을 넣고, 마지막 구간에 9개를 넣습니다.
- 최종 문자열은 `a________b__________c`로 만들어집니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 이유는 단어의 길이를 합산하고, 각 단어 사이에 밑줄을 추가하는 과정이 각각 N에 비례하기 때문입니다.

## 코드
```python
n, m = map(int, input().split())
arr = [input().rstrip() for _ in range(n)]

def greedy(a, b):
    aa = a // b
    bb = a % b

    result = [aa] * b
    for i in range(b):
        if ord(arr[i + 1][0]) >= 97 and bb > 0:
            result[i] += 1
            bb -= 1
    
    for i in range(b - 1, 0, -1):
        if result[i] == aa and bb > 0:
            result[i] += 1
            bb -= 1

    result2 = ["_" * x for x in result]
    return result2

total = 0
for i in range(n):
    total += len(arr[i])

result = greedy(m - total, n- 1)

ans = [arr.pop(0)]
for i in range(n - 1):
    ans.append(result[i])
    ans.append(arr[i])

print(*ans, sep = '')
```
