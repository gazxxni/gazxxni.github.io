## 문제
사전에 알파벳 모음 'A', 'E', 'I', 'O', 'U'만을 사용하여 만들 수 있는, 길이 5 이하의 모든 단어가 수록되어 있습니다. 사전에서 첫 번째 단어는 "A"이고, 그다음은 "AA"이며, 마지막 단어는 "UUUUU"입니다.

단어 하나 word가 매개변수로 주어질 때, 이 단어가 사전에서 몇 번째 단어인지 return 하도록 solution 함수를 완성해주세요.

제한사항


word의 길이는 1 이상 5 이하입니다.
word는 알파벳 대문자 'A', 'E', 'I', 'O', 'U'로만 이루어져 있습니다.




입출력 예

        
word
result

6


"AAAE"
10


"I"

1189


      
입출력 예 설명

입출력 예 #1

사전에서 첫 번째 단어는 "A"이고, 그다음은 "AA", "AAA", "AAAA", "AAAAA", "AAAAE", ... 와 같습니다. "AAAAE"는 사전에서 6번째 단어입니다.

입출력 예 #2

"AAAE"는  "A", "AA", "AAA", "AAAA", "AAAAA", "AAAAE", "AAAAI", "AAAAO", "AAAAU"의 다음인 10번째 단어입니다.

입출력 예 #3

"I"는 1563번째 단어입니다.

입출력 예 #4

"EIO"는 1189번째 단어입니다.

## 풀이
### 풀이 핵심 로직
주어진 문제는 모음 'A', 'E', 'I', 'O', 'U'로 생성할 수 있는 모든 단어를 구하고, 특정 단어가 이 목록에서 몇 번째에 위치하는지를 찾는 것입니다. 이를 위해 각 자리수에 대해 가능한 조합을 생성한 후, 해당 단어를 찾아 인덱스를 반환하는 방식으로 접근합니다.

### 동작 과정
1. **모음 정의**: 문제에서 사용할 모음 목록을 정의합니다: `['A', 'E', 'I', 'O', 'U']`.
2. **조합 생성**: 길이가 1부터 5까지의 모든 조합을 생성합니다. `itertools.product`를 사용하여 각 길이에 대해 모든 가능한 조합을 만듭니다.
3. **정렬**: 생성된 조합을 사전식으로 정렬합니다.
4. **위치 찾기**: 원하는 단어의 인덱스를 찾아 1을 더해 결과로 반환합니다.

예시 입력 "AAAAE"에 대해:
- 길이 1 조합: A, E, I, O, U
- 길이 2 조합: AA, AE, AI, AO, AU, EA, EE, EI, EO, EU, IA, IE, II, IO, IU, OA, OE, OI, OO, OU, UA, UE, UI, UO, UU
- ...
- 생성된 조합을 정렬한 후, "AAAAE"는 6번째 위치에 있습니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(5^n) (n은 최대 길이 5)입니다. 이는 각 자리수에서 5개의 선택이 가능하기 때문이며, 최악의 경우 모든 조합을 생성해야 하므로 이와 같은 복잡도를 가집니다. 다만, 입력 길이가 제한되어 있기 때문에 실제로는 매우 효율적으로 작동합니다.

### 코드
```python
from itertools import product

def solution(word):
    alpha = ['A', 'E', 'I', 'O', 'U']
    arr = []
    for i in range(1, 6):
        for p in product(alpha, repeat=i):
            arr.append("".join(p))
        
    arr.sort()
    ans = arr.index(word) + 1
    return ans
```

이 코드는 주어진 조건을 충족하고, 단어가 생성되는 순서에 따라 정확한 위치를 반환합니다.
