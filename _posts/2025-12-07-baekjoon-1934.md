## 문제
두 자연수 A와 B에 대해서, A의 배수이면서 B의 배수인 자연수를 A와 B의 공배수라고 한다. 이런 공배수 중에서 가장 작은 수를 최소공배수라고 한다. 예를 들어, 6과 15의 공배수는 30, 60, 90등이 있으며, 최소 공배수는 30이다.

두 자연수 A와 B가 주어졌을 때, A와 B의 최소공배수를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T(1 ≤ T ≤ 1,000)가 주어진다. 둘째 줄부터 T개의 줄에 걸쳐서 A와 B가 주어진다. (1 ≤ A, B ≤ 45,000)

---

## 출력
첫째 줄부터 T개의 줄에 A와 B의 최소공배수를 입력받은 순서대로 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
최소공배수(LCM)는 두 수 A와 B의 곱을 두 수의 최대공약수(GCD)로 나눈 값으로 정의됩니다. 즉, LCM(A, B) = (A * B) / GCD(A, B)입니다. 이 방식을 이용하면, 효율적으로 최소공배수를 구할 수 있습니다.

### 동작 과정
1. 테스트 케이스 수 T를 입력받습니다.
2. 각 테스트 케이스에 대해, 두 자연수 A와 B를 입력받습니다.
3. GCD를 구하는 함수를 정의하고, 이를 사용하여 LCM을 계산합니다.
4. 계산된 LCM 값을 출력합니다.

예시 입력:
```
3
6 15
8 12
5 10
```

단계별 설명:
- 첫 번째 테스트 케이스에서 A=6, B=15를 입력받습니다.
  - GCD(6, 15)를 계산하여 3을 얻습니다.
  - LCM(6, 15) = (6 * 15) / 3 = 30이므로 30을 출력합니다.
  
- 두 번째 테스트 케이스에서 A=8, B=12를 입력받습니다.
  - GCD(8, 12)를 계산하여 4을 얻습니다.
  - LCM(8, 12) = (8 * 12) / 4 = 24이므로 24를 출력합니다.
  
- 세 번째 테스트 케이스에서 A=5, B=10을 입력받습니다.
  - GCD(5, 10)를 계산하여 5을 얻습니다.
  - LCM(5, 10) = (5 * 10) / 5 = 10이므로 10을 출력합니다.

출력 결과:
```
30
24
10
```

### 시간 복잡도
최대공약수(GCD)를 유클리드 호제법으로 계산할 경우, 시간 복잡도는 O(log(min(A, B)))입니다. 주어진 문제에서 최대 1000개의 테스트 케이스를 처리하므로 전체 시간 복잡도는 O(T * log(min(A, B)))가 됩니다.

## 코드
```python
t = int(input())

def gcd(a, b):
    while b > 0:
        a, b = b, a % b
    return a

def lcm(a, b):
    return a * b / gcd(a, b)

for _ in range(t):
    a, b = map(int, input().split())
    
    print(int(lcm(a, b)))
```
