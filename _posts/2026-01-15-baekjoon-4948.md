## 문제
베르트랑 공준은 임의의 자연수 n에 대하여, n보다 크고, 2n보다 작거나 같은 소수는 적어도 하나 존재한다는 내용을 담고 있다.

이 명제는 조제프 베르트랑이 1845년에 추측했고, 파프누티 체비쇼프가 1850년에 증명했다.

예를 들어, 10보다 크고, 20보다 작거나 같은 소수는 4개가 있다. (11, 13, 17, 19) 또, 14보다 크고, 28보다 작거나 같은 소수는 3개가 있다. (17,19, 23)

자연수 n이 주어졌을 때, n보다 크고, 2n보다 작거나 같은 소수의 개수를 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 케이스는 n을 포함하는 한 줄로 이루어져 있다.

입력의 마지막에는 0이 주어진다.

---

## 출력
각 테스트 케이스에 대해서, n보다 크고, 2n보다 작거나 같은 소수의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
베르트랑 공준에 따라, 주어진 자연수 n에 대해 n보다 크고 2n 이하의 소수를 구하기 위해, 먼저 에라토스테네스의 체를 사용하여 0부터 300,000까지의 소수를 미리 계산합니다. 이후 각 테스트 케이스에서 n을 입력받고, n부터 2n까지의 범위에서 소수를 카운트하여 결과를 출력합니다.

### 동작 과정
1. **입력**: 여러 개의 테스트 케이스가 주어지며, 각 테스트 케이스는 자연수 n이다. 마지막 입력은 0으로, 이는 종료 신호이다.
2. **소수 계산**: `prime_numbers` 함수에서 0부터 300,000까지의 소수를 미리 계산하여 배열 `arr`에 저장합니다. 배열의 인덱스가 소수인 경우 `True`로 표시됩니다.
3. **소수 카운트**:
   - 입력받은 n에 대해 n + 1부터 2n까지의 범위에서 소수를 찾습니다.
   - 해당 범위의 인덱스가 `True`인 경우 카운트를 증가시킵니다.
4. **출력**: 각 테스트 케이스마다 카운트된 소수의 개수를 출력합니다.

**예시 입력**:
```
10
13
20
0
```
**단계별 설명**:
- 첫 번째 입력 `10`에 대해, 소수는 11, 13, 17, 19로 총 4개.
- 두 번째 입력 `13`에 대해, 소수는 17, 19로 총 2개.
- 세 번째 입력 `20`에 대해, 소수는 23, 29로 총 2개.
- 마지막으로 `0`이 입력되면 프로그램 종료.

### 시간 복잡도
에라토스테네스의 체를 사용하는 소수 판별의 시간 복잡도는 O(n log(log n))입니다. 각 테스트 케이스에서 소수를 카운트하는 과정은 O(n)이고, 최대 n의 값이 300,000인 경우에 대해 상수 시간의 작업을 하므로 전체적으로 O(n log(log n)) + O(m) (m은 테스트 케이스 수)로 볼 수 있습니다.

## 코드
```python
def prime_numbers(n):
    arr = [True] * (n + 1) 
    arr[0] = arr[1] = False

    for i in range(2, int(n ** 0.5) + 1):
        if arr[i]:
            for j in range(i * i, n + 1, i):
                arr[j] = False
    
    return arr

a = prime_numbers(300000)

while True:
    n = int(input())
    
    if n == 0:
        exit()
        
    cnt = 0
    for i in range(n + 1, 2 * n + 1):
        if a[i] == True:
            cnt += 1
    
    print(cnt)
```
