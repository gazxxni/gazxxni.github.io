## 문제
이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다.
별(*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.



제한 조건


n과 m은 각각 1000 이하인 자연수입니다.




예시

입력
5 3

출력
*****
*****
*****


> 출처: 프로그래머스 코딩 테스트 연습, https://school.programmers.co.kr/learn/challenges

## 풀이
### 풀이 핵심 로직
주어진 문제는 직사각형의 가로 길이(n)와 세로 길이(m)에 맞춰 별(*)을 출력하는 것입니다. 이 문제는 이중 반복문을 사용하여 세로 방향으로 m번 반복하면서, 가로 방향으로 n개의 별을 출력하는 방식으로 해결할 수 있습니다.

### 동작 과정
1. 입력으로 주어진 두 정수 n(가로 길이)와 m(세로 길이)을 읽습니다. 예를 들어, 입력이 `5 3`이라면 n=5, m=3입니다.
2. 외부 루프를 m번 반복합니다. 이는 세로 방향으로 몇 줄을 출력할지를 결정합니다.
3. 내부 루프를 n번 반복하여 별(*)을 출력합니다. 이 루프는 한 줄에 몇 개의 별을 출력할지를 결정합니다.
4. 각 줄을 출력한 후에는 다음 줄로 넘어갑니다.
5. 최종적으로, 주어진 크기의 직사각형 별이 출력됩니다.

예시 입력 `5 3`의 경우:
- 첫 번째 줄: `*****`
- 두 번째 줄: `*****`
- 세 번째 줄: `*****`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(m * n)입니다. m은 세로 길이, n은 가로 길이로, 모든 셀에 대해 별을 출력해야 하므로 두 값을 곱한 시간 복잡도를 가집니다.

### 전체 코드
```python
a, b = map(int, input().strip().split(' '))

for i in range(b):
    ans = ''
    for j in range(a):
        ans += '*'
    print(ans)
``` 

이 코드는 주어진 n과 m에 따라 직사각형 형태로 별을 출력합니다. 각 줄마다 n개의 별을 생성하고, m개의 줄을 반복하여 출력합니다.

## 코드
```python
a, b = map(int, input().strip().split(' '))


for i in range(b):
    ans = ''
    for j in range(a):
        ans += '*'
    print(ans)
```
