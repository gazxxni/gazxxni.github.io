## 문제 링크
[https://www.acmicpc.net/problem/4134](https://www.acmicpc.net/problem/4134)

## 풀이

### 풀이 핵심 로직
주어진 수 `a`보다 크거나 같은 첫 번째 소수를 찾기 위한 알고리즘입니다. 소수 판별을 위한 `prime` 함수를 정의하여 입력된 수가 소수인지 확인하고, 소수가 아닐 경우 `a`를 1씩 증가시키면서 반복합니다. `a`가 소수일 때 그 값을 출력합니다.

### 동작 과정
1. 입력으로 테스트 케이스의 수 `t`를 받습니다.
2. 각 테스트 케이스마다 수 `a`를 입력받습니다.
3. `a`가 2 이하일 경우, 소수 중 가장 작은 수인 2를 출력합니다.
4. 그렇지 않은 경우, 다음과 같은 과정을 반복합니다:
   - `prime` 함수를 호출하여 `a`가 소수인지 확인합니다.
   - 소수일 경우, 현재의 `a` 값을 출력하고 반복을 종료합니다.
   - 소수가 아닐 경우, `a`를 1 증가시킵니다.
5. 이 과정을 `t`번 반복합니다.

예시 입력:
```
3
3
4
10
```
- 첫 번째 테스트 케이스: `a = 3` → `3`은 소수이므로 출력: `3`.
- 두 번째 테스트 케이스: `a = 4` → `4`는 소수가 아니므로 `5`를 확인하고 출력: `5`.
- 세 번째 테스트 케이스: `a = 10` → `10`은 소수가 아니므로 `11`을 확인하고 출력: `11`.

### 시간 복잡도
소수 판별 함수 `prime`는 `O(√n)`의 시간 복잡도를 가집니다. 테스트 케이스마다 `a`를 증가시키면서 소수를 찾으므로 최악의 경우 각 테스트 케이스에 대해 `O(m√m)`의 시간 복잡도를 가집니다. 여기서 `m`은 입력된 수 `a`입니다. 전체적인 복잡도는 `O(t * m√m)`입니다.

## 코드
```python
import sys
input = sys.stdin.readline

def prime(n):
    for i in range(2, int(n**(1/2)) + 1):
        if n % i == 0:
            return False
    
    return True

t = int(input())

for _ in range(t):
    a = int(input())
    
    if a <= 2:
        print(2)
    else:
        while True:
            if prime(a):
                print(a)
                break
            a += 1
```
