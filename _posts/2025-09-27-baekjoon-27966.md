## 문제
N$N$개의 정점으로 이루어진 트리의 모든 정점 쌍에 대하여, 두 정점 사이의 거리의 합을 최소화하시오. 정점에는 1$1$부터 N$N$까지 번호가 매겨져 있다.

즉, 정점 i$i$와 정점 j$j$ 사이의 거리를 dist(i,j)$\textrm {dist}(i,\, j)$라 할 때  ∑1≤ij≤Ndist(i,j)$\sum_{1 \leq i 가 최소인 트리를 구성해야 한다.

---

## 입력
첫째 줄에 정점의 개수 N$N$이 주어진다. (2≤N≤100000)$(2\leq N \leq 100\,000)$

---

## 출력
첫째 줄에 모든 정점 쌍에 대하여 두 정점 사이 거리의 합의 최솟값을 출력한다.

둘째 줄부터 N−1$N-1$개의 줄에 걸쳐, 트리를 이루는 간선의 정보를 출력한다. 각 줄에 간선이 연결하는 두 정점의 번호를 공백을 사이에 두고 출력한다.

조건을 만족하는 트리가 여럿이라면 아무거나 하나 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 트리의 구조를 통해 모든 정점 쌍 간의 거리 합을 최소화하는 것입니다. 모든 정점을 연결하기 위해 하나의 중심 정점(예: 1번 정점)과 나머지 정점들을 연결하여, 이 중심 정점으로부터 다른 정점까지의 거리를 1로 설정하고, 나머지 정점들 간의 거리는 2로 설정하여 거리를 최소화합니다. 이렇게 하면 정점 쌍 간의 거리 합을 효과적으로 줄일 수 있습니다.

### 동작 과정
1. **입력**: 정점의 개수 N을 입력받습니다.
2. **거리 합 계산**: 거리 합을 계산하기 위해 `(N-1)^2`을 출력합니다. 이는 중심 정점(1번 정점)과 N-1개의 다른 정점 모두 간의 거리가 1이므로, 각각의 정점 간 거리의 합은 `(N-1) * N / 2`로 계산된 거리 2의 경우로 더해집니다.
3. **간선 출력**: 정점 1과 나머지 정점들(2, 3, ..., N) 간의 간선을 출력합니다. 각 정점은 정점 1에 연결되어 있으며, 이는 N-1개의 간선을 형성합니다.

예시 입력으로 N = 5일 경우:
- 거리 합: (5-1)^2 = 16
- 출력할 간선: 
  ```
  1 2
  1 3
  1 4
  1 5
  ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 입력을 처리하고 N-1개의 간선을 출력하는 데 각각 O(N) 시간이 소요되기 때문입니다.

## 코드
```python
'''
1. 가운데 정점과 모든 정점과의 거리 1
2. 나머지 정점들끼리의 거리 2

n개 일때
1. n-1
2. (n-2 + n-3 + n-4 + ... + 1) * 2
    = (n-2)(n-1)
    
1, 2번은 더하면 (n-1) ** 2
'''

n = int(input())

print((n-1) ** 2)
for i in range(2, n+1):
    print(1, i)
```
