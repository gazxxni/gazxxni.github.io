## 문제
N$N$개의 정점으로 이루어진 트리의 모든 정점 쌍에 대하여, 두 정점 사이의 거리의 합을 최소화하시오. 정점에는 1$1$부터 N$N$까지 번호가 매겨져 있다.

즉, 정점 i$i$와 정점 j$j$ 사이의 거리를 dist(i,j)$\textrm {dist}(i,\, j)$라 할 때  ∑1≤ij≤Ndist(i,j)$\sum_{1 \leq i 가 최소인 트리를 구성해야 한다.

---

## 입력
첫째 줄에 정점의 개수 N$N$이 주어진다. (2≤N≤100000)$(2\leq N \leq 100\,000)$

---

## 출력
첫째 줄에 모든 정점 쌍에 대하여 두 정점 사이 거리의 합의 최솟값을 출력한다.

둘째 줄부터 N−1$N-1$개의 줄에 걸쳐, 트리를 이루는 간선의 정보를 출력한다. 각 줄에 간선이 연결하는 두 정점의 번호를 공백을 사이에 두고 출력한다.

조건을 만족하는 트리가 여럿이라면 아무거나 하나 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 트리를 구성할 때 한 정점을 중심으로 모든 다른 정점과의 거리를 최소화하는 것입니다. 하나의 정점을 선택하여 이 정점과 나머지 모든 정점을 직접 연결하면, 모든 정점 쌍 간의 거리 합을 최소화할 수 있습니다. 즉, 중심 정점에서 다른 정점까지의 거리는 1로 설정되고, 나머지 정점들 간의 거리는 2가 됩니다.

### 동작 과정
1. 입력으로 정점의 개수 N을 받습니다.
2. 모든 정점 쌍의 거리 합을 계산하기 위해, 다음과 같은 수식을 사용합니다:
   - 중심 정점과 N-1개의 정점 간의 거리: (N-1) * 1 = N-1
   - 나머지 정점들 간의 거리 합: (N-2 + N-3 + ... + 1) * 2 = (N-2)(N-1)
3. 따라서 최종 거리 합은 (N-1) + (N-2)(N-1) = (N-1)^2로 계산됩니다.
4. 마지막으로, 정점 1을 중심으로 하여 2부터 N까지의 정점과의 간선을 출력합니다.

예를 들어, N=5인 경우:
- 중심 정점 1과 나머지 정점 2, 3, 4, 5를 직접 연결합니다.
- 거리 합은 (5-1)^2 = 16이 됩니다.
- 출력은 다음과 같습니다:
```
16
1 2
1 3
1 4
1 5
```

### 시간 복잡도
이 알고리즘은 O(N)입니다. 입력으로 정점 수 N을 받아서, N-1개의 간선을 출력하는 과정이 포함되어 있습니다.

## 코드
```python
'''
1. 가운데 정점과 모든 정점과의 거리 1
2. 나머지 정점들끼리의 거리 2

n개 일때
1. n-1
2. (n-2 + n-3 + n-4 + ... + 1) * 2
    = (n-2)(n-1)
    
1, 2번은 더하면 (n-1) ** 2
'''

n = int(input())

print((n-1) ** 2)
for i in range(2, n+1):
    print(1, i)
```
