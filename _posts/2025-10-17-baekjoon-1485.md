## 문제
네 점이 주어졌을 때, 네 점을 이용해 정사각형을 만들 수 있는지 없는지를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 네 줄로 이루어져 있으며, 점의 좌표가 한 줄에 하나씩 주어진다. 점의 좌표는 -100,000보다 크거나 같고, 100,000보다 작거나 같은 정수이다. 같은 점이 두 번 이상 주어지지 않는다.

---

## 출력
각 테스트 케이스마다 입력으로 주어진 네 점을 이용해서 정사각형을 만들 수 있으면 1을, 없으면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 네 점이 정사각형을 이루기 위해서는 두 가지 조건을 만족해야 합니다. 첫째, 네 점 간의 거리 중에서 정사각형의 변의 길이에 해당하는 거리와 대각선의 길이에 해당하는 거리가 각각 4개와 2개 존재해야 합니다. 이를 통해 정사각형의 변과 대각선의 길이를 구분할 수 있습니다.

### 동작 과정
1. 입력으로 주어진 점 4개를 읽어와서 리스트에 저장합니다.
2. 두 점 간의 거리를 계산하여 사각형의 변과 대각선을 식별하기 위해 거리의 제곱을 딕셔너리에 저장합니다. (거리를 제곱하여 사용하는 이유는 제곱근을 계산할 필요 없이 비교할 수 있기 때문입니다.)
3. 각 거리에 대해 몇 번 등장했는지를 카운트합니다.
4. 거리가 4번 등장하는 값(변의 길이)와 2번 등장하는 값(대각선의 길이)이 모두 존재하는지 확인합니다.
5. 위 조건을 만족하면 정사각형을 만들 수 있으므로 1을 출력하고, 그렇지 않으면 0을 출력합니다.

예시 입력:
```
2
0 0
1 0
1 1
0 1
0 0
2 0
2 2
0 2
```

단계별 설명:
- 첫 번째 테스트 케이스에서 점 (0,0), (1,0), (1,1), (0,1)의 거리 계산 결과는 다음과 같습니다:
  - (0,0)과 (1,0): 거리^2 = 1
  - (0,0)과 (1,1): 거리^2 = 2
  - (0,0)과 (0,1): 거리^2 = 1
  - (1,0)과 (1,1): 거리^2 = 1
  - (1,0)과 (0,1): 거리^2 = 2
  - (1,1)과 (0,1): 거리^2 = 1
  - 1이 4번, 2가 2번 등장하여 정사각형을 형성할 수 있습니다. 따라서 1을 출력합니다.
  
- 두 번째 테스트 케이스에서 점 (0,0), (2,0), (2,2), (0,2)의 거리 계산 결과는:
  - 0이 4번, 8이 2번 등장하여 정사각형을 형성할 수 없습니다. 따라서 0을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(T)이며, 각 테스트 케이스마다 O(1) 시간 복잡도로 4개의 점에서 가능한 쌍을 계산하기 때문에 총 O(6) = O(1)로 상수 시간에 해당합니다.

## 코드
```python
from collections import defaultdict
t = int(input())

for _ in range(t):
    arr = []
    
    for i in range(4):
        x, y = map(int, input().split())
        arr.append((x, y))
    
    dic = defaultdict(int)
    for i in range(4):
        for j in range(4):
            if i != j:
                a = (arr[i][0] - arr[j][0]) ** 2 + (arr[i][1] - arr[j][1]) ** 2
                dic[a] += 1
    
    aa = False
    bb = False  
    for k, v in dic.items():
        if v == 8:
            aa = True
        
        elif v == 4:
            bb = True
            
    if aa and bb:
        print(1)
        
    else:
        print(0)
```
