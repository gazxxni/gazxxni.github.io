## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
주어진 정수 N을 소인수분해하기 위해, 먼저 2로 N을 나누어 2의 배수를 모두 출력합니다. 이후, 홀수인 소수(3, 5, 7, ...)를 차례로 나누어가며 N이 더 이상 나누어지지 않을 때까지 반복합니다. 최종적으로 N이 1보다 큰 경우, 그 값 또한 소인수로 출력합니다.

### 동작 과정
1. **입력**: N = 28
2. **2로 나누기**:
   - 28 % 2 == 0 → 출력: 2, N = 28 // 2 = 14
   - 14 % 2 == 0 → 출력: 2, N = 14 // 2 = 7
   - 7 % 2 != 0 → 종료
3. **홀수로 나누기**:
   - i = 3: 7 % 3 != 0 → i 증가
   - i = 5: 7 % 5 != 0 → i 증가
   - i = 7: 7 % 7 == 0 → 출력: 7, N = 7 // 7 = 1
4. **최종 출력**: 2, 2, 7

위의 과정으로 28의 소인수는 2, 2, 7로 출력됩니다.

### 시간 복잡도
O(√N) - 최악의 경우, N의 제곱근까지 반복해야 하므로 시간 복잡도는 O(√N)입니다.

---

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
