## 문제
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

---

## 출력
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

## 풀이

### 풀이 핵심 로직
소인수분해는 주어진 정수 N을 소수의 곱으로 표현하는 과정입니다. 먼저, 2로 나누어 떨어지는 만큼 2를 출력하고, 그 다음에는 홀수 소수들로 나누어 떨어지는지 검사하여 소인수를 차례로 출력합니다. 마지막으로, N이 1보다 큰 경우에는 남은 N 자체가 소수이므로 출력합니다.

### 동작 과정
예를 들어, 입력 N이 60이라고 가정해 보겠습니다.

1. **초기 N**: 60
2. **2로 나누기**:
   - 60 % 2 == 0이므로 2를 출력하고, N을 60 // 2 = 30으로 갱신합니다.
   - 30 % 2 == 0이므로 2를 다시 출력하고, N을 30 // 2 = 15로 갱신합니다.
   - 15 % 2 != 0이므로 2로 나누는 과정을 종료합니다.
3. **홀수 소수 검사**:
   - i=3일 때, 15 % 3 == 0이므로 3을 출력하고, N을 15 // 3 = 5로 갱신합니다.
   - 5 % 3 != 0이므로 3으로 나누는 과정을 종료합니다.
   - i=5일 때, 5 % 5 == 0이므로 5를 출력하고, N을 5 // 5 = 1로 갱신합니다.
4. **N 검사**: N이 1보다 크지 않으므로 과정 종료.

출력 결과는 다음과 같습니다:
```
2
2
3
5
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(√N)입니다. 이는 N을 소인수로 나누는 과정에서 최대 √N까지 소수를 검사하기 때문입니다.

---

## 코드
```python
import sys
import math
input = sys.stdin.readline

n = int(input())

while n % 2 == 0:
    print(2)
    n //= 2

for i in range(3, int(math.sqrt(n)) + 1, 2):
    while n % i == 0:
        print(i)
        n //= i

if n > 1:
    print(n)
```
