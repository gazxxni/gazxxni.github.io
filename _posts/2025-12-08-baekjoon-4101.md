## 문제
두 양의 정수가 주어졌을 때, 첫 번째 수가 두 번째 수보다 큰지 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 두 정수가 주어진다. 두 수는 백만보다 작거나 같은 양의 정수이다. 입력의 마지막 줄에는 0이 두 개 주어진다.

---

## 출력
각 테스트 케이스마다, 첫 번째 수가 두 번째 수보다 크면 Yes를, 아니면 No를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
두 양의 정수 a와 b를 입력받아 a가 b보다 큰지를 비교하여 결과를 출력하는 프로그램입니다. 입력이 "0 0"일 경우 종료하며, 각 테스트 케이스에 대해 "Yes" 또는 "No"를 출력합니다.

### 동작 과정
1. 입력을 무한 루프를 통해 계속 받아옵니다.
2. 각 입력에서 두 개의 정수 a와 b를 공백으로 구분하여 읽어옵니다.
3. a와 b가 모두 0인 경우 루프를 종료합니다.
4. a가 b보다 크면 "Yes"를 출력하고, 그렇지 않으면 "No"를 출력합니다.

**예시 입력 및 출력**
- 입력:  
```
5 3
2 4
0 0
```
- 출력:
```
Yes
No
```
- 설명:
  - 첫 번째 줄에서 5가 3보다 크므로 "Yes".
  - 두 번째 줄에서 2가 4보다 작으므로 "No".
  - 마지막 줄 "0 0"은 프로그램 종료 신호입니다.

### 시간 복잡도
O(N), 여기서 N은 입력된 테스트 케이스의 수입니다. 각 테스트 케이스에 대해 상수 시간 O(1)만큼의 작업이 수행되므로 전체적으로 N개의 테스트 케이스에 대해 O(N) 시간이 소요됩니다.

---

## 코드
```python
while True:
    a, b = map(int, input().split())
    if a == 0 and b == 0:
        break
    if a > b:
        print("Yes")
    else:
        print("No")
```
