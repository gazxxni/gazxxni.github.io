## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 110760 KB, 시간: 100 ms

---

## 풀이

### 풀이 핵심 로직
주어진 N개의 정수를 오름차순으로 정렬하기 위해 Python의 내장 정렬 함수인 `sort()`를 사용한다. 이 함수는 Timsort 알고리즘을 기반으로 하며, 최악의 경우에도 O(N log N)의 시간 복잡도를 보장한다. 입력된 수는 중복이 없으므로, 정렬 후 결과를 그대로 출력하면 된다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 수의 개수 N을 입력받고, 이어서 N개의 정수를 각각 입력받아 리스트 `arr`에 저장한다.
   - 예시 입력:
     ```
     5
     5
     3
     8
     1
     4
     ```
   - 이 경우, `arr` 리스트는 [5, 3, 8, 1, 4]가 된다.

2. **정렬**: `arr.sort()`를 호출하여 리스트를 오름차순으로 정렬한다.
   - 정렬 후 `arr` 리스트는 [1, 3, 4, 5, 8]이 된다.

3. **출력**: 정렬된 리스트의 각 요소를 한 줄씩 출력한다.
   - 출력 결과:
     ```
     1
     3
     4
     5
     8
     ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N log N)이다. 여기서 N은 입력된 수의 개수이다. 정렬을 위한 시간 복잡도가 O(N log N)이며, 리스트를 출력하는 데 O(N)의 시간이 소요되지만, 정렬이 지배적인 요소가 되므로 전체 시간 복잡도는 O(N log N)으로 간주된다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
