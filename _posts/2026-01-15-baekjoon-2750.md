## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 110760 KB, 시간: 100 ms

---

## 풀이

### 풀이 핵심 로직
주어진 N개의 정수를 오름차순으로 정렬하기 위해, 리스트에 입력된 수를 저장한 후 Python의 내장 정렬 함수를 사용하여 정렬합니다. 정렬된 리스트의 각 요소를 출력하여 문제의 요구사항을 충족합니다.

### 동작 과정
1. 입력으로 정수 N을 받습니다. (예: N = 5)
2. N동안 반복하여, 각 줄에 주어진 정수를 입력받아 리스트에 저장합니다. (예: 입력값: 3, 1, 4, 2, 5)
   - 최종 리스트: `[3, 1, 4, 2, 5]`
3. 리스트를 정렬합니다. (정렬 후: `[1, 2, 3, 4, 5]`)
4. 정렬된 리스트의 각 숫자를 한 줄에 하나씩 출력합니다.

입력 예시:
```
5
3
1
4
2
5
```
출력 결과:
```
1
2
3
4
5
```

### 시간 복잡도
정렬 알고리즘의 시간 복잡도는 O(N log N)입니다. 여기서 N은 입력되는 수의 개수입니다. 입력 크기 N이 최대 1,000이므로, 이 알고리즘은 충분히 효율적입니다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
