## 풀이

### 풀이 핵심 로직
이 문제는 집합(set)을 이용하여 1부터 20까지의 숫자를 관리하는 문제입니다. 주어진 명령어에 따라 집합에 숫자를 추가, 삭제, 확인, 토글(추가 또는 삭제)하는 작업을 수행합니다. "all" 명령어는 집합을 1부터 20까지의 모든 숫자로 초기화하고, "empty" 명령어는 집합을 비우는 기능을 제공합니다.

### 동작 과정
예시 입력으로 다음과 같은 명령어를 고려해봅시다:

```
5
add 3
check 3
remove 3
check 3
all
check 1
```

1. **add 3**: 집합에 3을 추가합니다. 현재 집합 `s = {3}`.
2. **check 3**: 3이 집합 `s`에 존재하므로 1을 출력합니다.
3. **remove 3**: 집합에서 3을 제거합니다. 현재 집합 `s = {}`.
4. **check 3**: 3이 집합에 없으므로 0을 출력합니다.
5. **all**: 집합을 1부터 20까지의 모든 숫자로 초기화합니다. 현재 집합 `s = {1, 2, ..., 20}`.
6. **check 1**: 1이 집합 `s`에 존재하므로 1을 출력합니다.

### 시간 복잡도
모든 명령어는 집합(set)의 기본 연산을 사용하므로 평균적으로 O(1) 시간 복잡도를 가집니다. 따라서 전체 명령어의 수가 m일 때, 전체 시간 복잡도는 O(m)입니다.

## 코드
```python
import sys

s=set()
input=sys.stdin.readline

m=int(input())

for _ in range(m):
    a=input().strip().split()

    if len(a)==1:
        if a[0]=="all":
            s=set([i for i in range(1,21)])
        else:
            s=set()

    else:
        func,x=a[0],a[1]
        x=int(x)

        if func=='add':
            s.add(x)
        elif func=='remove':
            s.discard(x)
        elif func=='check':
            print(1 if x in s else 0)
        elif func=='toggle':
            if x in s:
                s.discard(x)
            else:
                s.add(x)

                
```
