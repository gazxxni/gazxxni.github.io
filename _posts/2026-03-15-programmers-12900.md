## 문제
가로 길이가 2이고 세로의 길이가 1인 직사각형모양의 타일이 있습니다. 이 직사각형 타일을 이용하여 세로의 길이가 2이고 가로의 길이가 n인 바닥을 가득 채우려고 합니다. 타일을 채울 때는 다음과 같이 2가지 방법이 있습니다.


타일을 가로로 배치 하는 경우
타일을 세로로 배치 하는 경우


예를들어서 n이 7인 직사각형은 다음과 같이 채울 수 있습니다.



직사각형의 가로의 길이 n이 매개변수로 주어질 때, 이 직사각형을 채우는 방법의 수를 return 하는 solution 함수를 완성해주세요.

경우의 수가 많아 질 수 있으므로, 경우의 수를 1,000,000,007으로 나눈 나머지를 return해주세요.

n
result

## 풀이
### 풀이 핵심 로직
이 문제는 `n` 길이의 바닥을 2x1 타일로 채우는 방법의 수를 계산하는 문제로, 피보나치 수열과 유사한 성격을 가집니다. `n` 길이를 채우기 위해서는 마지막 타일이 가로로 놓이거나 세로로 놓일 수 있는데, 이 두 가지 경우를 통해 재귀적으로 문제를 해결할 수 있습니다. 따라서 DP(동적 프로그래밍)를 사용하여 이전 두 길이의 경우의 수를 합산하여 현재 길이의 경우의 수를 구할 수 있습니다.

### 동작 과정
1. 입력으로 `n = 4`가 주어졌다고 가정합니다.
2. DP 배열을 초기화합니다. `dp[0] = 1` (길이가 0인 경우의 수)과 `dp[1] = 1` (길이가 1인 경우의 수)로 설정합니다.
3. `dp` 배열을 사용하여 2부터 `n`까지 반복하면서 각 길이에 대한 경우의 수를 계산합니다.
   - `dp[2] = dp[1] + dp[0] = 1 + 1 = 2`
   - `dp[3] = dp[2] + dp[1] = 2 + 1 = 3`
   - `dp[4] = dp[3] + dp[2] = 3 + 2 = 5`
4. 최종적으로 `dp[4]`의 값을 반환합니다. 결과는 5입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 이는 길이 `n`에 대해 한 번의 반복문을 수행하기 때문입니다. DP 배열을 사용하므로 공간 복잡도 또한 O(n)입니다.

## 코드
```python
def solution(n):
    MOD = 1000000007
    dp = [0]*(n+1)
    dp[0] = 1
    dp[1] = 1
    for i in range(2, n+1):
        dp[i] = (dp[i-1] + dp[i-2]) % MOD
    return dp[n]
```
