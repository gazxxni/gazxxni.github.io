## 문제
창영이는 삼각형의 종류를 잘 구분하지 못한다. 따라서 프로그램을 이용해 이를 외우려고 한다.

삼각형의 세 각을 입력받은 다음, 


	세 각의 크기가 모두 60이면, Equilateral
	세 각의 합이 180이고, 두 각이 같은 경우에는 Isosceles
	세 각의 합이 180이고, 같은 각이 없는 경우에는 Scalene
	세 각의 합이 180이 아닌 경우에는 Error


를 출력하는 프로그램을 작성하시오.

---

## 입력
총 3개의 줄에 걸쳐 삼각형의 각의 크기가 주어진다. 모든 정수는 0보다 크고, 180보다 작다.

---

## 출력
문제의 설명에 따라 Equilateral, Isosceles, Scalene, Error 중 하나를 출력한다.

## 풀이

### 풀이 핵심 로직
삼각형의 세 각을 입력받아, 각의 크기에 따라 삼각형의 종류를 판별하는 문제이다. 세 각의 합이 180도인지 확인한 후, 각의 값을 비교하여 Equilateral, Isosceles, Scalene 중 하나로 분류하며, 조건을 만족하지 않으면 Error를 출력한다.

### 동작 과정
1. 세 각 \(a\), \(b\), \(c\)를 각각 입력받는다.
2. 세 각이 모두 60도인지 확인한다. 만약 모두 60도라면 "Equilateral"을 출력한다.
3. 세 각의 합이 180도인지 확인한다.
   - 만약 180도이고 두 각이 같은 경우, "Isosceles"을 출력한다.
   - 그렇지 않고 모든 각이 서로 다른 경우, "Scalene"을 출력한다.
4. 세 각의 합이 180도가 아닌 경우, "Error"를 출력한다.

예시 입력:
```
60
60
60
```
단계별 설명:
- 입력받은 각 \(a\), \(b\), \(c\)는 각각 60, 60, 60이다.
- 첫 번째 조건을 만족하므로 "Equilateral"을 출력한다.

예시 입력:
```
70
70
40
```
단계별 설명:
- 입력받은 각 \(a\), \(b\), \(c\)는 각각 70, 70, 40이다.
- 세 각의 합은 180이며, 두 각(70)이 같으므로 "Isosceles"을 출력한다.

예시 입력:
```
90
30
60
```
단계별 설명:
- 입력받은 각 \(a\), \(b\), \(c\)는 각각 90, 30, 60이다.
- 세 각의 합은 180이며, 모든 각이 다르므로 "Scalene"을 출력한다.

예시 입력:
```
100
80
10
```
단계별 설명:
- 입력받은 각 \(a\), \(b\), \(c\)는 각각 100, 80, 10이다.
- 세 각의 합은 190으로 180도가 아니므로 "Error"를 출력한다.

### 시간 복잡도
O(1) - 입력받는 값의 수가 고정되어 있어, 시간 복잡도는 상수 시간이다.

## 코드
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
c = int(input())


if a == b == c == 60:
    print('Equilateral')

elif a + b + c == 180:
    if a == b and b != c:
        print('Isosceles')
    elif a == c and b != c:
        print('Isosceles')
    elif b == c and a != c:
        print('Isosceles')
    else:
        print('Scalene')

else:
    print('Error')
```
