## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 M개를 고른 수열
	같은 수를 여러 번 골라도 된다.

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
이 문제는 백트래킹을 사용하여 1부터 N까지의 자연수 중에서 M개의 수를 선택하는 모든 조합을 구하는 문제입니다. 같은 수를 여러 번 선택할 수 있기 때문에, 재귀적으로 수를 선택하고 조합이 완성되면 결과를 출력합니다.

### 동작 과정
1. 입력으로 `N`과 `M`이 주어진다. 예를 들어, 입력이 `3 2`일 경우, N은 3, M은 2이다.
2. 빈 리스트 `arr`를 생성하여 수열을 저장한다.
3. 재귀 함수 `aa()`를 호출한다.
   - 함수가 호출될 때, `arr`의 길이가 M과 같으면 현재 조합을 출력하고 함수를 종료한다.
   - 그렇지 않으면 1부터 N까지의 숫자에 대해 반복하며, 각 숫자를 `arr`에 추가하고 다음 재귀 호출을 진행한다.
   - 재귀 호출이 끝난 후에는 마지막으로 추가한 숫자를 `arr`에서 제거하여 다음 조합을 위한 준비를 한다.
4. 이 과정을 반복하여 모든 가능한 수열을 출력한다.

예시 입력을 가지고 살펴보면:
- 입력: `3 2`
- 출력: 
  ```
  1 1
  1 2
  1 3
  2 1
  2 2
  2 3
  3 1
  3 2
  3 3
  ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N^M)입니다. M의 길이만큼 재귀적으로 호출이 이루어지며, 각 호출에서 N개의 선택이 가능하기 때문입니다.

## 코드
```python
n, m = map(int, input().split())

arr = []
def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        arr.append(i)
        aa()
        arr.pop()
        
aa()
```
