## 풀이

### 풀이 핵심 로직
이 문제는 주어진 정수 \( n \)에 대해 \( n^2 \)을 계산하고, 알고리즘의 시간 복잡도를 출력하는 간단한 문제입니다. \( n^2 \)을 출력하는 이유는 주어진 문제의 알고리즘이 \( O(n^2) \)의 시간 복잡도를 갖는다는 것을 나타내기 위함입니다.

### 동작 과정
1. 사용자로부터 정수를 입력받습니다. 예를 들어, 입력으로 5가 주어졌다고 가정합니다.
2. 입력받은 정수 \( n \)에 대해 \( n^2 \)를 계산합니다. \( 5^2 = 25 \)가 됩니다.
3. 계산된 \( n^2 \)의 결과인 25를 출력합니다.
4. 알고리즘의 시간 복잡도인 2를 출력합니다. 이는 알고리즘의 명세에 따라 고정된 출력입니다.

예시 입력:
```
5
```
출력:
```
25
2
```

### 시간 복잡도
이 문제의 시간 복잡도는 \( O(1) \)입니다. 입력 크기에 관계없이 상수 시간에 결과를 출력하기 때문입니다. 하지만 문제에서 요구하는 알고리즘의 시간 복잡도는 \( O(n^2) \)로 표현되어 있습니다.

---

## 코드
```python
n = int(input())
print(n ** 2)
print(2)
```
