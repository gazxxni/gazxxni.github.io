## 문제
오늘도 서준이는 알고리즘의 수행시간 수업 조교를 하고 있다. 아빠가 수업한 내용을 학생들이 잘 이해했는지 문제를 통해서 확인해보자.

입력의 크기 n이 주어지면 MenOfPassion 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

MenOfPassion 알고리즘은 다음과 같다.

MenOfPassion(A[], n) {
    sum

## 입력
첫째 줄에 입력의 크기 n(1 ≤ n ≤ 500,000)이 주어진다.

---

## 출력
첫째 줄에 코드1 의 수행 횟수를 출력한다.

둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 수행 시간은 입력 크기 \( n \)에 따라 \( n^2 \)으로 증가합니다. 이는 알고리즘이 두 개의 중첩 루프를 통해 각 요소를 반복적으로 처리하기 때문입니다. 따라서, 코드의 수행 횟수를 계산할 때 \( n^2 \)을 출력하고, 이 다항식의 최고차항은 2가 됩니다.

### 동작 과정
1. 입력값 \( n \)을 받습니다. 예를 들어, \( n = 3 \)이라고 가정합시다.
2. 수행 횟수를 계산합니다: \( n^2 = 3^2 = 9 \). 따라서 첫 번째 출력은 9입니다.
3. 최고차항의 차수를 확인하여 2를 출력합니다. 이는 \( n^2 \)에서 \( n \)의 차수가 2이기 때문입니다.

따라서, 입력이 3일 때 출력은 다음과 같습니다:
```
9
2
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(n^2) \)입니다. 이는 입력 크기 \( n \)에 대해 수행 시간이 제곱으로 증가함을 나타냅니다.
