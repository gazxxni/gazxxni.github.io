## 출력

## 풀이

### 풀이 핵심 로직
MenOfPassion 알고리즘의 수행 시간은 입력 크기 n에 따라 n^2번의 연산이 발생합니다. 이 문제에서는 이러한 수행 횟수를 출력하고, 이를 다항식으로 나타낼 때 최고차항의 차수를 구하는 것이 요구됩니다. 따라서 수행 횟수는 n^2이고, 최고차항의 차수는 2입니다.

### 동작 과정
1. 입력: n = 1인 경우
   - 수행 횟수: 1^2 = 1
   - 최고차항의 차수: 2
   - 출력: 
     ```
     1
     2
     ```
   
2. 입력: n = 2인 경우
   - 수행 횟수: 2^2 = 4
   - 최고차항의 차수: 2
   - 출력: 
     ```
     4
     2
     ```

3. 입력: n = 3인 경우
   - 수행 횟수: 3^2 = 9
   - 최고차항의 차수: 2
   - 출력: 
     ```
     9
     2
     ```

4. 입력: n = 500,000인 경우
   - 수행 횟수: 500,000^2 = 250,000,000,000
   - 최고차항의 차수: 2
   - 출력: 
     ```
     250000000000
     2
     ```

### 시간 복잡도
시간 복잡도는 O(1)입니다. 입력 n에 대한 연산은 단순히 n^2을 계산하고, 상수 값을 출력하는 것이기 때문에 입력 크기에 관계없이 항상 일정한 시간 내에 수행됩니다.
