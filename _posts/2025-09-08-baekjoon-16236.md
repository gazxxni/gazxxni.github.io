## 문제 링크
[https://www.acmicpc.net/problem/16236](https://www.acmicpc.net/problem/16236)

## 풀이

### 풀이 핵심 로직
문제는 상어가 주어진 바다에서 작은 물고기를 먹어가며 성장하는 과정을 시뮬레이션하는 것입니다. 상어는 BFS(너비 우선 탐색)를 사용하여 현재 위치에서 먹을 수 있는 물고기를 탐색하고, 가장 가까운 물고기를 먹으며 이동합니다. 먹을 수 있는 물고기가 없으면 시뮬레이션을 종료합니다.

### 동작 과정
1. **입력 및 초기화**: 
   - 바다의 크기 `n`과 각 칸의 내용을 입력받아 `board` 리스트에 저장합니다. 
   - 상어의 초기 위치(값이 9인 위치)를 찾고, 해당 위치를 0으로 변경합니다.

2. **BFS 함수 호출**:
   - `bfs` 함수를 호출하여 현재 상어의 위치와 크기를 전달합니다. 이 함수는 상어가 이동할 수 있는 위치를 탐색하고, 먹을 수 있는 물고기의 후보를 찾습니다.

3. **BFS 내부 로직**:
   - 큐를 사용하여 현재 위치에서 상어가 이동할 수 있는 모든 방향을 탐색합니다. 
   - 이동 가능한 위치는 상어의 크기보다 작거나 같은 칸이며, 이미 방문하지 않은 칸입니다.
   - 먹을 수 있는 물고기(크기가 0보다 크고 상어 크기보다 작은 칸)를 찾으면 후보 리스트에 추가하고, 거리 정보를 함께 저장합니다.

4. **가장 가까운 물고기 선택**:
   - 모든 후보 물고기를 거리, x, y 순으로 정렬하여 가장 가까운 물고기를 선택합니다. 
   - 선택된 물고리를 먹기 위해 상어의 위치를 업데이트하고, 시간을 증가시킵니다.

5. **상어 크기 변화**:
   - 물고기를 먹은 후, 먹은 개수가 상어의 크기와 같아지면 상어의 크기를 증가시킵니다.

6. **루프 종료**:
   - 더 이상 먹을 물고기가 없으면 루프를 종료하고, 총 시간을 출력합니다.

### 시간 복잡도
BFS를 사용하여 각 칸을 최대 한 번씩 방문하므로, 최악의 경우 시간 복잡도는 O(n^2)입니다. 여기서 n은 바다의 크기입니다. BFS는 모든 인접한 칸을 탐색하기 때문에 이 복잡도가 발생합니다.

## 코드
```python
import sys
from collections import deque
input = sys.stdin.readline

n = int(input())
board = [list(map(int, input().split())) for _ in range(n)]

for i in range(n):
    for j in range(n):
        if board[i][j] == 9:
            shark_x, shark_y = i, j
            board[i][j] = 0 

directions = [(-1, 0), (0, -1), (0, 1), (1, 0)]

size = 2
eat = 0
time = 0

def bfs(x, y, size):
    visited = [[-1] * n for _ in range(n)]
    q = deque()
    q.append((x, y))
    visited[x][y] = 0

    fish = []

    while q:
        x, y = q.popleft()

        for dx, dy in directions:
            nx, ny = x + dx, y + dy

            if 0 <= nx < n and 0 <= ny < n:
                if visited[nx][ny] == -1:
                    # 상어가 이동할 수 있는 경우
                    if board[nx][ny] <= size:
                        visited[nx][ny] = visited[x][y] + 1
                        q.append((nx, ny))
                        # 먹을 수 있는 물고기면 후보 등록
                        if 0 < board[nx][ny] < size:
                            fish.append((visited[nx][ny], nx, ny))

    # 가장 가까운 물고기 선택
    if not fish:
        return None
    fish.sort()  # 거리, x, y 순으로 정렬
    return fish[0]  # (거리, x, y)

while True:
    result = bfs(shark_x, shark_y, size)

    if result is None:  # 더 이상 먹을 물고기 없음
        break

    dist, nx, ny = result

    # 상어 이동
    shark_x, shark_y = nx, ny
    time += dist

    # 물고기 먹기
    board[nx][ny] = 0
    eat += 1

    # 크기 증가
    if eat == size:
        size += 1
        eat = 0

print(time)

```
