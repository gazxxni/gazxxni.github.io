## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이를 통해 삼각형의 종류를 판별하기 위해 먼저, 세 변의 길이를 정렬한 후 가장 긴 변(c)와 나머지 두 변(a, b)의 합이 c보다 큰지 확인한다. 삼각형 조건을 만족하지 않으면 "Invalid"를 출력하고, 그렇지 않으면 변의 길이에 따라 Equilateral, Isosceles, Scalene 중 하나를 출력한다.

### 동작 과정
1. **입력 받기**: 사용자가 세 개의 변의 길이를 입력합니다. 예를 들어, `6 3 2`를 입력한다고 가정합니다.
2. **종료 조건 확인**: 입력된 값이 `0 0 0`일 경우 프로그램을 종료합니다.
3. **정렬**: 입력된 변의 길이를 정렬하여 `a <= b <= c`가 되도록 합니다. 위의 예시에서 정렬하면 `2, 3, 6`이 됩니다.
4. **삼각형 조건 판별**:
   - 가장 긴 변(c)이 나머지 두 변(a, b)의 합보다 크거나 같으면 "Invalid"를 출력합니다. (여기서는 `6 >= 2 + 3`이므로 Invalid)
5. **삼각형 유형 판별**:
   - 만약 `a == b == c`이면 "Equilateral"을 출력합니다.
   - 만약 두 변만 같다면 "Isosceles"을 출력합니다.
   - 세 변이 모두 다르면 "Scalene"을 출력합니다.

위의 과정을 반복하여 다음 입력을 처리합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력으로 주어지는 세 변의 길이를 정렬하고 비교하는 과정은 상수 시간 내에 수행됩니다. 전체 입력을 처리하는 데 필요한 시간은 입력의 개수에 비례하므로 O(n)으로 볼 수 있습니다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
