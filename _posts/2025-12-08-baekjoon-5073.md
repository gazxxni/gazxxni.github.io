## 문제
삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.


	Equilateral :  세 변의 길이가 모두 같은 경우
	Isosceles : 두 변의 길이만 같은 경우
	Scalene : 세 변의 길이가 모두 다른 경우


단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 출력한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.

세 변의 길이가 주어질 때 위 정의에 따른 결과를 출력하시오.

---

## 입력
각 줄에는 1,000을 넘지 않는 양의 정수 3개가 입력된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.

---

## 출력
각 입력에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 출력하시오.

## 풀이

### 풀이 핵심 로직
주어진 세 변의 길이로 삼각형을 구성할 수 있는지를 확인한 후, 변의 길이에 따라 삼각형의 유형을 구분하는 방식으로 문제를 해결합니다. 먼저 변의 길이를 정렬한 후 가장 긴 변과 나머지 두 변의 합을 비교하여 유효성을 검사하고, 이후 변의 길이의 동등성을 비교하여 삼각형의 유형을 결정합니다.

### 동작 과정
1. 사용자로부터 세 개의 변의 길이를 입력받습니다.
2. 입력받은 값이 모두 0이면 종료합니다.
3. 세 변을 정렬하여 가장 긴 변을 `c`, 나머지 두 변을 `a`와 `b`로 설정합니다.
4. 삼각형의 조건을 확인합니다:
   - `c >= a + b`일 경우 "Invalid"를 출력합니다.
5. 변의 길이에 따라 다음과 같이 분류합니다:
   - 세 변의 길이가 모두 같으면 "Equilateral"을 출력합니다.
   - 두 변의 길이가 같으면 "Isosceles"을 출력합니다.
   - 세 변의 길이가 모두 다르면 "Scalene"을 출력합니다.

예시 입력:
```
6 3 2
7 7 7
5 5 2
3 4 5
0 0 0
```
출력:
```
Invalid
Equilateral
Isosceles
Scalene
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 입력의 크기와 관계없이 항상 세 개의 변에 대해서만 계산을 수행하기 때문에 상수 시간 내에 결과를 도출할 수 있습니다.

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
