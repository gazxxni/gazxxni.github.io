## 문제 링크
[https://www.acmicpc.net/problem/30802](https://www.acmicpc.net/problem/30802)

## 풀이

### 풀이 핵심 로직
이 문제는 주어진 수들의 리스트에서 각 수를 특정 값으로 나누어 필요한 최소의 몫을 계산하고, 리스트의 총합을 다른 특정 값으로 나누어 몫과 나머지를 구하는 문제입니다. 첫 번째 부분에서는 각 수를 `t`로 나누어 몫을 구하고, 두 번째 부분에서는 리스트의 총합을 `p`로 나누어 몫과 나머지를 계산합니다.

### 동작 과정
1. 입력을 받은 후, 첫 번째 줄에서 수의 개수 `n`을 읽어옵니다.
2. 두 번째 줄에서는 `n`개의 수를 읽어와 리스트 `ex`에 저장합니다.
3. 세 번째 줄에서는 두 개의 정수 `t`와 `p`를 읽어옵니다.
4. `ans` 변수를 초기화하고, `ex` 리스트의 각 요소에 대해 반복문을 실행합니다:
   - 각 요소 `ex[i]`를 `t`로 나누어서 몫을 계산합니다. 만약 나머지가 0이면 몫을 그대로 더하고, 그렇지 않으면 몫에 1을 더하여 `ans`에 추가합니다.
5. 리스트 `ex`의 총합을 구하고 이를 `p`로 나누어 몫과 나머지를 계산하여 `ans2` 리스트에 저장합니다.
6. 최종적으로 `ans`와 `ans2`의 내용을 출력합니다.

예시 입력:
```
5
10 20 30 40 50
5 12
```
단계별 설명:
- `n = 5`, `ex = [10, 20, 30, 40, 50]`, `t = 5`, `p = 12`
- `ex[0] = 10` -> `10 % 5 == 0` -> `ans += 2` (비율 2)
- `ex[1] = 20` -> `20 % 5 == 0` -> `ans += 4` (비율 4)
- `ex[2] = 30` -> `30 % 5 == 0` -> `ans += 6` (비율 6)
- `ex[3] = 40` -> `40 % 5 == 0` -> `ans += 8` (비율 8)
- `ex[4] = 50` -> `50 % 5 == 0` -> `ans += 10` (비율 10)
- 최종 `ans = 30`
- 총합 `sum(ex) = 150`, `150 // 12 = 12`, `150 % 12 = 6`
- 최종 출력: `30` `12 6`

### 시간 복잡도
이 알고리즘은 `O(n)`의 시간 복잡도를 가집니다. 리스트를 한 번 순회하며 계산을 진행하므로, `n`이 리스트의 길이일 때, 입력 크기에 비례하여 실행 시간이 증가합니다.

## 코드
```python
import sys

n = int(input())
ex = list(map(int,input().split()))
t,p = map(int,input().split())

ans = 0
for i in range(len(ex)):
    if ex[i]%t == 0:
        ans += int(ex[i]/t)
    else:
        ans += (int(ex[i]/t)+1)

print(ans)

ans2 = [int(sum(ex)/p),sum(ex)%p]
print(*ans2)
```
