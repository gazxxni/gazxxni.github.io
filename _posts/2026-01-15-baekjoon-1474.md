## 문제
세준이는 N개의 영어 단어를 이용해 길이가 M인 새로운 단어를 만들려고 한다. 새로운 단어는 N개의 단어를 순서대로 이어 붙이고, 각 단어의 사이에 _을 넣어서 만든다. 이렇게 만든 새로운 단어의 길이가 M이 아닌 경우 _를 추가해서 길이가 M이 되게 만들어야 한다.


	_는 단어와 단어 사이에만 추가할 수 있다. 따라서, 새로운 단어는 _으로 시작하거나, _로 끝날 수 없다.
	단어와 단어 사이에 있는 _의 개수는 모두 같아야 한다.
	
		모두 같게 만드는 것이 불가능한 경우 단어와 단어 사이에 있는 _의 개수의 최댓값과 최솟값의 차이는 1이 되어야 한다.
	
	


새로운 단어 중 사전 순으로 가장 앞서는 단어를 구해보자.

---

## 입력
첫째 줄에 N과 M이 주어진다. 둘째 줄부터 N개의 줄에 단어가 한 줄에 하나씩 주어진다.

---

## 출력
첫째 줄에 사전 순으로 가장 앞서는 단어를 출력한다.

알파벳 대문자, 소문자, 밑 줄의 순서는 'A'

## 풀이

### 풀이 핵심 로직
문제를 해결하기 위해 주어진 N개의 단어를 연결할 때, 단어 사이에 추가해야 하는 `_`의 개수를 균등하게 나누는 방식으로 접근한다. 추가로, 단어 사이의 `_` 개수가 같은 경우의 사전 순으로 가장 앞서는 조합을 만들기 위해, 필요한 `_`의 개수를 계산하고 이를 통해 최적의 조합을 생성한다.

### 동작 과정
1. **입력 처리**: 첫 줄에서 N과 M을 읽고, 그 다음 N개의 단어를 리스트에 저장한다.
2. **총 길이 계산**: 모든 단어의 길이를 합산하여 `total` 변수에 저장한다.
3. **필요한 `_` 개수 계산**: `m - total`을 통해 추가해야 하는 `_`의 총 개수를 계산하고, 이를 N-1로 나누어 각 단어 사이에 추가할 `_`의 기본 개수와 나머지를 구한다.
4. **균등하게 `_` 분배**:
   - 기본 개수는 `aa`, 나머지는 `bb`로 저장한다.
   - 각 단어 사이에 기본 개수 만큼 `_`을 추가하고, 나머지를 사전 순으로 가장 앞서는 방식으로 배분한다.
5. **결과 조합**: 첫 번째 단어를 결과 리스트에 추가한 후, 각 단어와 `_`을 번갈아가며 결과 리스트에 추가한다.
6. **출력**: 최종 조합된 문자열을 출력한다.

**예시 입력**:
```
3 15
hello
world
python
```
- 총 길이: `5 + 5 + 6 = 16`
- 필요한 `_`의 개수: `15 - 16 = -1` (즉, `_`를 추가하지 않아도 된다)
- 결과: `hello_world_python` (사전 순으로 가장 앞서는 조합)

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)이다. 여기서 N은 입력으로 주어진 단어의 개수이다. 각 단어의 길이를 합산하고, 결과 문자열을 구성하는 과정에서 단어 개수에 비례하는 연산이 이루어지기 때문이다.

## 코드
```python
n, m = map(int, input().split())
arr = [input().rstrip() for _ in range(n)]

def greedy(a, b):
    aa = a // b
    bb = a % b

    result = [aa] * b
    for i in range(b):
        if ord(arr[i + 1][0]) >= 97 and bb > 0:
            result[i] += 1
            bb -= 1
    
    for i in range(b - 1, 0, -1):
        if result[i] == aa and bb > 0:
            result[i] += 1
            bb -= 1

    result2 = ["_" * x for x in result]
    return result2

total = 0
for i in range(n):
    total += len(arr[i])

result = greedy(m - total, n- 1)

ans = [arr.pop(0)]
for i in range(n - 1):
    ans.append(result[i])
    ans.append(arr[i])

print(*ans, sep = '')
```
