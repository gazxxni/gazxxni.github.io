## 문제
부산사이버대학교 학생 성택이는 엄마의 의뢰를 받아 주어진 문자열이 현관문 비밀번호에 사용 가능한지 알아내야 한다. 성택이는 공부해야 하므로 우리가 도와주자!

사용할 수 있는 비밀번호의 규칙은 다음과 같다.


	비밀번호는 6자리 이상 9자리 이하여야 한다.


예를 들어, 123124는 올바른 비밀번호이고, 1202727161은 잘못된 비밀번호이다. 문자열이 주어졌을 때 현관문 비밀번호로 사용할 수 있는지 판단하자.

---

## 입력
첫째 줄에 문자열의 총개수 N$N$이 주어진다.

둘째 줄부터 N$N$개의 줄에 걸쳐 숫자, 영어 대소문자로만 구성된 문자열이 주어진다.

---

## 출력
줄마다 사용할 수 있는 비밀번호면 yes, 그렇지 않으면 no를 출력한다.

## 풀이

### 풀이 핵심 로직
문자열의 길이를 확인하여 비밀번호가 6자리 이상 9자리 이하인지 판단합니다. 주어진 문자열의 길이가 이 범위에 속하면 "yes"를 출력하고, 그렇지 않으면 "no"를 출력합니다.

### 동작 과정
1. 첫 번째 줄에서 문자열의 총개수 \( N \)을 입력받습니다.
2. 다음 \( N \)개의 줄에 걸쳐 비밀번호 후보 문자열을 각각 입력받습니다.
3. 각 문자열의 길이를 측정합니다.
4. 길이가 6 이상 9 이하인 경우 "yes"를 출력하고, 그렇지 않으면 "no"를 출력합니다.

예시 입력:
```
3
password
12345
abcdefghij
```

- 첫 번째 문자열 "password"의 길이는 8로, 6 이상 9 이하의 조건을 만족하므로 "yes" 출력.
- 두 번째 문자열 "12345"의 길이는 5로, 조건에 맞지 않으므로 "no" 출력.
- 세 번째 문자열 "abcdefghij"의 길이는 10으로, 조건에 맞지 않으므로 "no" 출력.

출력 결과:
```
yes
no
no
```

### 시간 복잡도
입력 문자열의 개수를 \( N \)이라고 할 때, 각 문자열의 길이를 측정하는 작업의 시간 복잡도는 \( O(1) \)입니다. 따라서 전체 알고리즘의 시간 복잡도는 \( O(N) \)입니다.

---

## 코드
```python
for _ in range(int(input())):
    password = len(input())
    print("yes") if 6 <= password and password <= 9 else print("no")

```
