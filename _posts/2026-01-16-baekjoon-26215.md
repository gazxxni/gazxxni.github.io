## 문제
지난 밤 겨울 숲에는 눈이 많이 내렸다. 당신은 숲의 주민들을 위해 눈이 오지 않는 동안 모든 집 앞의 눈을 치우고자 한다.

당신은 1분에 한 번씩 두 집을 선택해서 두 집 앞의 눈을 각각 1만큼 치우거나, 한 집을 선택해서 그 집 앞의 눈을 1만큼 치울 수 있다.

모든 집 앞의 눈을 전부 치울 때까지 걸리는 최소 시간은 얼마일까?

---

## 입력
첫 줄에 집의 수를 의미하는 정수 N$N$ (1≤N≤100$1 \leq N \leq 100$)이 주어진다.

다음 줄에는 각각의 집 앞에 쌓여 있는 눈의 양을 나타내는 정수 ai$a_{i}$ (1≤ai≤2000$1 \leq a_{i} \leq 2000$)이 주어진다.

---

## 출력
모든 집 앞의 눈을 치우는 데 최소 몇 분이 걸리는지를 출력한다. 24시간(1440분)이 넘게 걸릴 경우 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 집 앞에 쌓인 눈을 효율적으로 치우기 위해 항상 가장 많은 눈이 쌓인 두 집을 선택하는 것입니다. 이를 통해 최대한 많은 눈을 줄이는 방향으로 작업을 진행하며, 시뮬레이션을 통해 눈을 치우는 데 걸리는 시간을 계산합니다.

### 동작 과정
1. **입력 예시**: 
   ```
   4
   5 8 3 10
   ```
   - 집의 수 \( N = 4 \)
   - 집 앞의 눈의 양 \( arr = [5, 8, 3, 10] \)

2. **1차 반복**:
   - 현재 배열: `[5, 8, 3, 10]`
   - 정렬 후: `[10, 8, 5, 3]`
   - 가장 큰 두 집의 눈을 치우기: 
     - 집 앞의 눈: `10 - 8 = 2` (10번 집)
     - 두 번째 집의 눈: `8 - 8 = 0` (8번 집)
   - 배열 업데이트: `[2, 5, 3]`
   - 시간 카운트: `cnt = 8`

3. **2차 반복**:
   - 현재 배열: `[2, 5, 3]`
   - 정렬 후: `[5, 3, 2]`
   - 가장 큰 두 집의 눈을 치우기:
     - 집 앞의 눈: `5 - 3 = 2`
     - 두 번째 집의 눈: `3 - 3 = 0`
   - 배열 업데이트: `[2, 2]`
   - 시간 카운트: `cnt = 11`

4. **3차 반복**:
   - 현재 배열: `[2, 2]`
   - 정렬 후: `[2, 2]`
   - 가장 큰 두 집의 눈을 치우기:
     - 집 앞의 눈: `2 - 2 = 0`
     - 두 번째 집의 눈: `2 - 2 = 0`
   - 배열 업데이트: `[0]`
   - 시간 카운트: `cnt = 13`

5. **최종 확인**:
   - 모든 집 앞의 눈이 치워진 경우: `arr = [0]`
   - 최종 시간 카운트: `cnt = 13`
   - 1440분 이내이므로 출력: `13`

### 시간 복잡도
이 알고리즘은 매 반복마다 배열을 정렬하고 최대 두 집의 눈을 줄이는 과정을 수행합니다. 정렬에 걸리는 시간은 \( O(N \log N) \)이며, 이러한 과정을 \( N \)번 수행할 수도 있습니다. 따라서 전체 시간 복잡도는 \( O(N^2 \log N) \)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

cnt = 0

while True:
    if len(arr) == 1:
        break
    
    arr.sort(reverse=True)
    arr[0] -= arr[1]
    cnt += arr[1]
    arr.pop(1)
    
cnt += arr[0]
if cnt > 1440:
    print(-1)
else:
    print(cnt)
```
