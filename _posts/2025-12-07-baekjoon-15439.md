## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 상의와 하의의 색상이 서로 다를 때의 조합을 구하는 문제입니다. N개의 상의와 N개의 하의가 있을 때, 각 상의에 대해 선택할 수 있는 하의는 N-1개입니다. 따라서 총 가능한 조합의 수는 N 곱하기 (N-1)입니다.

### 동작 과정
1. 입력으로 N을 받습니다. 예를 들어, N = 3일 때:
   - 상의 색상: 1, 2, 3
   - 하의 색상: 1, 2, 3
2. 각 상의 색상에 대해 선택할 수 있는 하의 색상은 그 상의 색상과 다른 색상이므로, 각 상의는 N-1개의 하의와 조합될 수 있습니다.
   - 상의 1 (색상 1) 선택 시: 하의 색상 2, 3 → 2가지
   - 상의 2 (색상 2) 선택 시: 하의 색상 1, 3 → 2가지
   - 상의 3 (색상 3) 선택 시: 하의 색상 1, 2 → 2가지
3. 따라서 총 조합은 3 (상의 갯수) * 2 (각 상의에 대해 선택할 수 있는 하의 갯수) = 6입니다.
4. 최종적으로 결과를 출력합니다.

### 시간 복잡도
O(1) - 입력을 받아서 단순한 산술 연산을 수행하므로 상수 시간 복잡도를 가집니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
