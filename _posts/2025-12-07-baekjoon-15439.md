## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제에서 주어진 조건에 따르면 N개의 상의와 N개의 하의가 있으며, 각각의 옷은 서로 다른 색상을 가지고 있습니다. 상의와 하의의 색상이 서로 다르도록 조합하기 위해서는 상의의 색상 선택 후 N-1개의 하의 색상 중에서 선택해야 합니다. 따라서 총 조합 수는 `N * (N - 1)`이 됩니다.

### 동작 과정
1. 입력으로 N이 주어진다고 가정합니다. (예: N = 3)
2. 상의의 색상 선택: 3개의 상의 중 1개를 선택합니다. (예: 상의 색상 1)
3. 선택한 상의와 다른 하의 색상을 선택할 수 있는 경우의 수는 N-1개입니다. (예: 하의 색상 2, 3 중 선택)
4. 따라서, 총 조합 수는 3 * (3 - 1) = 6이 됩니다.
5. 출력 결과는 6입니다.

예시 입력:
```
3
```
출력:
```
6
```

### 시간 복잡도
이 문제의 시간 복잡도는 O(1)입니다. 입력 N에 따라 단순한 곱셈 연산만 수행되므로 상수 시간에 해결할 수 있습니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
