## 문제
두 정수 X, Y의 임의의 자리에서 공통으로 나타나는 정수 k(0 ≤ k ≤ 9)들을 이용하여 만들 수 있는 가장 큰 정수를 두 수의 짝꿍이라 합니다(단, 공통으로 나타나는 정수 중 서로 짝지을 수 있는 숫자만 사용합니다). X, Y의 짝꿍이 존재하지 않으면, 짝꿍은 -1입니다. X, Y의 짝꿍이 0으로만 구성되어 있다면, 짝꿍은 0입니다.

예를 들어, X = 3403이고 Y = 13203이라면, X와 Y의 짝꿍은 X와 Y에서 공통으로 나타나는 3, 0, 3으로 만들 수 있는 가장 큰 정수인 330입니다. 다른 예시로 X = 5525이고 Y = 1255이면 X와 Y의 짝꿍은 X와 Y에서 공통으로 나타나는 2, 5, 5로 만들 수 있는 가장 큰 정수인 552입니다(X에는 5가 3개, Y에는 5가 2개 나타나므로 남는 5 한 개는 짝 지을 수 없습니다.)
두 정수 X, Y가 주어졌을 때, X, Y의 짝꿍을 return하는 solution 함수를 완성해주세요.

제한사항


3 ≤ X, Y의 길이(자릿수) ≤ 3,000,000입니다.
X, Y는 0으로 시작하지 않습니다.
X, Y의 짝꿍은 상당히 큰 정수일 수 있으므로, 문자열로 반환합니다.

## 풀이
### 풀이 핵심 로직
주어진 두 숫자 X와 Y에서 공통된 숫자를 찾아서, 그 숫자들을 최대한 많이 조합하여 가장 큰 수를 만들어야 합니다. 각 숫자의 빈도를 계산한 후, 공통된 숫자 중에서 사용할 수 있는 만큼 조합하여 내림차순으로 정렬하여 결과를 생성합니다.

### 동작 과정
1. **입력 처리**: 문자열로 주어진 X와 Y를 각 자리수로 나누어 리스트로 변환합니다.
2. **빈도 계산**: 각 숫자의 빈도를 세기 위해 `collections.defaultdict`를 사용하여 0부터 9까지의 숫자에 대해 각각의 빈도를 계산합니다.
3. **공통 숫자 찾기**: 두 숫자에서 공통적으로 나타나는 숫자를 확인하고, 이 숫자들을 둘 중 최소 빈도만큼 조합하여 결과 리스트에 추가합니다.
4. **결과 조합**: 결과 리스트를 내림차순으로 정렬한 후, 리스트의 합이 0이라면 "0"을 반환하고, 그렇지 않다면 리스트를 문자열로 합쳐서 반환합니다.
5. **조건 처리**: 공통 숫자가 없을 경우 "-1"을 반환합니다.

### 예시 입력으로 단계별 설명
- 입력: `X = "5525"`, `Y = "1255"`
- 각 숫자의 빈도:
  - X의 빈도: {5: 3, 2: 1}
  - Y의 빈도: {1: 1, 2: 1, 5: 2}
- 공통 숫자와 최대 사용 가능 빈도:
  - 5: min(3, 2) = 2개 사용 가능
  - 2: min(1, 1) = 1개 사용 가능
- 사용 가능한 숫자 리스트: [5, 5, 2]
- 내림차순 정렬 후 결과: "552"

### 시간 복잡도
- O(n + m), 여기서 n은 X의 길이, m은 Y의 길이입니다. 각 숫자의 빈도를 계산하는 데 O(n + m) 시간이 소요되고, 최종 결과를 생성하는 데 추가적인 O(k log k) (k는 공통 숫자의 총 개수) 시간이 소요됩니다. 전체적으로는 O(n + m)으로 볼 수 있습니다.

아래는 수정된 코드입니다:

```python
from collections import defaultdict

def solution(X, Y):
    # 각 숫자의 빈도를 세기 위한 딕셔너리
    dic_x = defaultdict(int)
    dic_y = defaultdict(int)
    
    # X의 각 숫자 빈도 계산
    for char in X:
        dic_x[int(char)] += 1
        
    # Y의 각 숫자 빈도 계산
    for char in Y:
        dic_y[int(char)] += 1
    
    arr = []
    
    # 공통된 숫자 찾기
    for digit in range(10):
        if digit in dic_x and digit in dic_y:
            cnt = min(dic_x[digit], dic_y[digit])
            arr.extend([digit] * cnt)
    
    # 결과를 내림차순 정렬
    arr.sort(reverse=True)
    
    # 결과 처리
    if not arr:  # 공통 숫자가 없음
        return '-1'
    elif all(num == 0 for num in arr):  # 모든 숫자가 0
        return '0'
    else:
        return ''.join(map(str, arr))  # 리스트를 문자열로 변환하여 반환
```

## 코드
```python
from collections import defaultdict
def solution(X, Y):
    answer = ''
    x = [int(i) for i in X]
    y = [int(i) for i in Y]
    dic_x = defaultdict(int)
    dic_y = defaultdict(int)
    for i in range(10):
        if i in x:
            dic_x[i] += x.count(i)
        else:
            dic_x[i] = 0
            
        if i in y:
            dic_y[i] += y.count(i)
        else:
            dic_y[i] = 0
            
    arr = []
    for x_i, x_v in dic_x.items():
        for y_i, y_v in dic_y.items():
            if x_i == y_i:
                cnt = max(x_v, y_v) - abs(x_v - y_v)
                for _ in range(cnt):
                    arr.append(x_i)
                break
                
    arr.sort(reverse=True)
    if arr:
        if sum(arr) == 0:
            return '0'
        else:
            return ''.join(map(str, arr))

    else:
        return '-1'
    
    
    return answer
```
