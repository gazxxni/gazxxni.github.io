## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 정수 \( a \)와 \( b \)를 입력받아 그 곱을 계산하는 간단한 문제입니다. Python의 기본 데이터 타입인 `int`는 큰 정수를 지원하므로, 두 수의 곱을 계산하는 데에 별다른 제한이 없습니다. 따라서 두 수를 입력받고 곱한 값을 출력하는 것이 핵심 아이디어입니다.

### 동작 과정
1. **입력 받기**: `input()` 함수를 사용하여 한 줄의 입력을 받고, `map(int, ...)`를 통해 공백으로 구분된 두 개의 정수를 \( a \)와 \( b \)로 변환합니다.
   - 예시 입력: `123456789012345678 987654321098765432`
   - 이 경우, \( a = 123456789012345678 \), \( b = 987654321098765432 \)가 됩니다.

2. **곱셈 수행**: 두 수를 곱한 결과를 계산합니다.
   - \( a \times b = 12193263113702179522374638011112635269 \)

3. **결과 출력**: `print()` 함수를 사용하여 계산된 결과를 출력합니다.
   - 출력: `12193263113702179522374638011112635269`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(1) \)입니다. 입력받는 두 정수의 크기와 관계없이, 곱셈 연산은 상수 시간 내에 수행됩니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a * b)
```
