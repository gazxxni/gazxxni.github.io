## 문제
세준이는 N개의 영어 단어를 이용해 길이가 M인 새로운 단어를 만들려고 한다. 새로운 단어는 N개의 단어를 순서대로 이어 붙이고, 각 단어의 사이에 _을 넣어서 만든다. 이렇게 만든 새로운 단어의 길이가 M이 아닌 경우 _를 추가해서 길이가 M이 되게 만들어야 한다.


	_는 단어와 단어 사이에만 추가할 수 있다. 따라서, 새로운 단어는 _으로 시작하거나, _로 끝날 수 없다.
	단어와 단어 사이에 있는 _의 개수는 모두 같아야 한다.
	
		모두 같게 만드는 것이 불가능한 경우 단어와 단어 사이에 있는 _의 개수의 최댓값과 최솟값의 차이는 1이 되어야 한다.
	
	


새로운 단어 중 사전 순으로 가장 앞서는 단어를 구해보자.

---

## 입력
첫째 줄에 N과 M이 주어진다. 둘째 줄부터 N개의 줄에 단어가 한 줄에 하나씩 주어진다.

---

## 출력
첫째 줄에 사전 순으로 가장 앞서는 단어를 출력한다.

알파벳 대문자, 소문자, 밑 줄의 순서는 'A'

## 풀이

### 풀이 핵심 로직
문제를 해결하기 위해, 주어진 단어를 이어 붙인 후 필요한 밑줄(_)의 개수를 계산하고, 이를 고르게 또는 최대 1 차이로 분배하여 사전 순으로 가장 앞서는 문자열을 만들도록 한다. 각 단어 사이에 들어갈 밑줄의 수를 균일하게 배분하고, 여유가 있는 경우에는 먼저 사전 순이 앞서는 방향으로 추가한다.

### 동작 과정
1. **입력 처리**: N과 M을 입력받고, N개의 단어를 리스트에 저장한다.
2. **길이 계산**: 모든 단어의 길이를 합산하여 `total`을 구한다.
3. **필요한 밑줄 계산**: 새로운 단어의 길이 M에서 `total`을 빼서 필요한 밑줄의 총 개수를 계산한다. 이때, 단어 사이에 들어갈 밑줄의 개수는 N-1개가 된다.
4. **밑줄 분배**:
   - 고르게 나누어 떨어지는 경우와 나누어 떨어지지 않는 경우를 처리한다. 
   - 각각의 경우에 대해 밑줄을 균일하게 분배하고, 여유가 있는 경우에는 사전 순을 고려하여 앞쪽 단어에 더 추가한다.
5. **단어 결합**: 첫 단어와 사전 순으로 정렬된 밑줄을 사이에 두고 나머지 단어들을 결합하여 최종 문자열을 만든다.
6. **출력**: 최종 결과를 출력한다.

**예시 입력**:
```
3 18
ab
c
de
```
1. 단어의 길이 합: `len('ab') + len('c') + len('de') = 2 + 1 + 2 = 5` 
2. 필요한 밑줄의 수: `18 - 5 = 13`
3. 단어 사이의 밑줄 수: `13 // 2` (사이의 단어 수는 2개, 즉 N-1)
4. 균일하게 6개를 각 사이에 배치하고 남은 1개를 사전 순을 고려하여 앞쪽에 추가
5. 최종 문자열: `ab______c______de` (사전 순으로 가장 앞서는 조합)

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)이다. 단어의 길이를 계산하고, 밑줄을 배분하는 과정에서 한 번씩 리스트를 순회하기 때문이다.

## 코드
```python
n, m = map(int, input().split())
arr = [input().rstrip() for _ in range(n)]

def greedy(a, b):
    aa = a // b
    bb = a % b

    result = [aa] * b
    for i in range(b):
        if ord(arr[i + 1][0]) >= 97 and bb > 0:
            result[i] += 1
            bb -= 1
    
    for i in range(b - 1, 0, -1):
        if result[i] == aa and bb > 0:
            result[i] += 1
            bb -= 1

    result2 = ["_" * x for x in result]
    return result2

total = 0
for i in range(n):
    total += len(arr[i])

result = greedy(m - total, n- 1)

ans = [arr.pop(0)]
for i in range(n - 1):
    ans.append(result[i])
    ans.append(arr[i])

print(*ans, sep = '')
```
