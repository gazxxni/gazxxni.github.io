## 문제
배열 arr가 주어집니다. 배열 arr의 각 원소는 숫자 0부터 9까지로 이루어져 있습니다. 이때, 배열 arr에서 연속적으로 나타나는 숫자는 하나만 남기고 전부 제거하려고 합니다. 단, 제거된 후 남은 수들을 반환할 때는 배열 arr의 원소들의 순서를 유지해야 합니다. 예를 들면,


arr = [1, 1, 3, 3, 0, 1, 1] 이면 [1, 3, 0, 1] 을 return 합니다.
arr = [4, 4, 4, 3, 3] 이면 [4, 3] 을 return 합니다.


배열 arr에서 연속적으로 나타나는 숫자는 제거하고 남은 수들을 return 하는 solution 함수를 완성해 주세요.

제한사항


배열 arr의 크기 : 1,000,000 이하의 자연수
배열 arr의 원소의 크기 : 0보다 크거나 같고 9보다 작거나 같은 정수




입출력 예

        
arr
answer


        
[1,1,3,3,0,1,1]
[1,3,0,1]


[4,4,4,3,3]
[4,3]

## 풀이
### 풀이 핵심 로직
주어진 배열에서 연속적으로 나타나는 숫자를 제거하기 위해, 새로운 리스트를 생성하고 첫 번째 원소를 추가한 후, 두 번째 원소부터 순회하면서 마지막 원소와 비교합니다. 만약 현재 원소가 마지막 원소와 다르면 새로운 리스트에 추가합니다. 이렇게 하면 연속된 숫자는 하나만 남게 됩니다.

### 동작 과정
1. 입력 배열 `arr`가 `[1, 1, 3, 3, 0, 1, 1]`라고 가정합니다.
2. `answer` 리스트를 초기화하여 첫 번째 원소인 `1`을 추가합니다. 현재 `answer = [1]`.
3. 인덱스 1부터 시작하여 각 원소를 확인합니다.
   - 인덱스 1: `1` (마지막 원소 `1`과 같음) → 추가하지 않음.
   - 인덱스 2: `3` (마지막 원소 `1`과 다름) → 추가하여 `answer = [1, 3]`.
   - 인덱스 3: `3` (마지막 원소 `3`과 같음) → 추가하지 않음.
   - 인덱스 4: `0` (마지막 원소 `3`과 다름) → 추가하여 `answer = [1, 3, 0]`.
   - 인덱스 5: `1` (마지막 원소 `0`과 다름) → 추가하여 `answer = [1, 3, 0, 1]`.
   - 인덱스 6: `1` (마지막 원소 `1`과 같음) → 추가하지 않음.
4. 최종적으로 `answer` 리스트를 반환합니다: `[1, 3, 0, 1]`.

### 시간 복잡도
O(n) - 배열의 각 원소를 한 번씩만 확인하므로, n은 배열의 길이입니다.

## 코드
```python
def solution(arr):
    answer = [arr[0]]
    n = len(arr)
    for i in range(1, n):
        if answer[-1] != arr[i]:
            answer.append(arr[i])
    return answer
```
