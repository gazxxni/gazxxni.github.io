## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 출입 기록을 통해 현재 회사에 있는 사람들을 추적하는 것입니다. `enter` 명령이 들어오면 해당 직원의 출근을 기록하고, `leave` 명령이 들어오면 해당 직원의 퇴근을 기록합니다. 최종적으로 회사에 있는 직원들을 사전 순의 역순으로 정렬하여 출력합니다.

### 동작 과정
1. **입력 받기**: 첫 줄에 출입 기록의 수 `n`이 주어지고, 이후 `n`개의 줄에 직원의 이름과 출입 기록(enter 또는 leave)이 주어집니다.
2. **기록 저장**:
   - `enter`가 들어오면 해당 직원의 기록을 `dic`에 추가합니다. 이때, `dic`에서 직원 이름을 키로 사용하고, 값은 출근한 횟수를 카운트합니다.
   - `leave`가 들어오면 해당 직원의 기록을 삭제합니다. 이 경우, `dic`에서 해당 직원의 값을 0으로 설정하여 현재 회사에 없음을 표시합니다.
3. **현재 직원 추출**: 최종적으로 `dic`에서 값이 1인 직원(즉, 현재 회사에 있는 직원)을 리스트에 추가합니다.
4. **정렬 및 출력**: 리스트를 사전 순의 역순으로 정렬한 후, 각 이름을 출력합니다.

예시 입력:
```
3
alice enter
bob enter
alice leave
```
- 첫 번째 입력에서 `alice`가 들어오고 `dic`에 `{'alice': 1}`이 추가됩니다.
- 두 번째 입력에서 `bob`이 들어오고 `dic`에 `{'bob': 1, 'alice': 1}`이 추가됩니다.
- 세 번째 입력에서 `alice`가 나가므로 `dic`에서 `alice`의 값을 0으로 설정합니다.
- 마지막으로 회사에 있는 직원인 `bob`을 리스트에 추가하고 역순 정렬하여 출력합니다.

출력:
```
bob
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. n개의 출입 기록을 처리하고, 최종적으로 현재 회사에 있는 직원들의 이름을 정렬하는 과정에서 log n의 시간이 소요되기 때문입니다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
