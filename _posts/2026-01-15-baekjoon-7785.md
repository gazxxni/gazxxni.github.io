## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 출입 기록을 바탕으로 현재 회사에 있는 사람들을 추적하는 것입니다. 'enter' 기록이 있으면 해당 사람을 회사에 있다고 간주하고, 'leave' 기록이 있으면 회사를 떠났다고 간주합니다. 이를 위해 `defaultdict`를 사용하여 각 직원의 출입 상태를 관리하고, 최종적으로 회사에 남아 있는 직원의 이름을 사전 순의 역순으로 출력합니다.

### 동작 과정
1. **입력 받기**: 첫 번째 줄에서 총 로그의 수 `n`을 입력받습니다. 그 다음 `n`개의 줄에 각각 직원의 이름과 출입 상태('enter' 또는 'leave')를 입력받습니다.
2. **상태 관리**: 
   - 각 로그를 순회하며, 만약 상태가 'enter'라면 해당 직원의 카운트를 1 증가시키고, 'leave'라면 1 감소시킵니다. 이때 `defaultdict`를 사용하여 직원 이름을 키로, 출입 상태를 값으로 관리합니다.
3. **현재 직원 추출**: 로그가 모두 처리된 후, 직원의 출입 상태가 1인 경우를 찾아 리스트에 추가합니다. 이는 현재 회사에 있는 직원을 의미합니다.
4. **정렬 및 출력**: 현재 회사에 있는 직원의 이름을 사전 순의 역순으로 정렬한 후, 한 줄씩 출력합니다.

#### 예시 입력
```
7
John enter
Doe enter
John leave
Jane enter
Doe leave
Jane leave
Doe enter
```
#### 단계별 설명
- "John enter": John이 회사에 들어옴 -> 상태: {John: 1}
- "Doe enter": Doe가 회사에 들어옴 -> 상태: {John: 1, Doe: 1}
- "John leave": John이 회사에서 나감 -> 상태: {John: 0, Doe: 1}
- "Jane enter": Jane이 회사에 들어옴 -> 상태: {John: 0, Doe: 1, Jane: 1}
- "Doe leave": Doe가 회사에서 나감 -> 상태: {John: 0, Doe: 0, Jane: 1}
- "Jane leave": Jane가 회사에서 나감 -> 상태: {John: 0, Doe: 0, Jane: 0}
- "Doe enter": Doe가 회사에 들어옴 -> 상태: {John: 0, Doe: 1, Jane: 0}

현재 회사에 있는 사람: Doe

### 시간 복잡도
입력 로그를 한 번 순회하며 각 직원의 상태를 업데이트하고, 최종적으로 현재 회사에 있는 직원의 리스트를 정렬하는 과정이 포함됩니다. 이때, 로그 수는 `n`이고, 정렬 과정에는 O(m log m) (여기서 m은 현재 회사에 있는 직원 수, 최대 n까지 가능) 가 소요됩니다. 따라서 전체 시간 복잡도는 O(n + m log m)입니다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
