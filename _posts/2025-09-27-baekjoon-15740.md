## 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 A와 B (-1010000 ≤ A, B ≤ 1010000)가 주어진다.

---

## 출력
첫째 줄에 A+B를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 96 ms

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 매우 큰 정수를 입력받아 그 합을 출력하는 문제이다. 파이썬의 `int` 타입은 임의의 정밀도를 지원하므로, 범위가 넓은 정수도 문제 없이 처리할 수 있다. 따라서 입력받은 두 정수를 더한 값을 출력하면 된다.

### 동작 과정
1. 첫째 줄에 두 정수 A와 B가 공백으로 구분되어 입력된다. 예를 들어, 입력이 `123456789012345678901234567890 987654321098765432109876543210`이라면 A는 `123456789012345678901234567890`, B는 `987654321098765432109876543210`이다.
2. `map(int, input().split())` 함수를 사용하여 입력된 문자열을 정수로 변환하여 A와 B에 각각 저장한다.
3. A와 B를 더한 결과를 `print()` 함수를 사용하여 출력한다. 이 경우, 결과는 `1111111110111111111011111111100`이 된다.

### 시간 복잡도
O(N) - 여기서 N은 입력되는 숫자의 자릿수이다. 정수를 더하는 과정에서 각 자릿수에 대해 연산이 필요하므로, 자릿수에 비례하는 시간 복잡도를 가진다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a + b)
```
