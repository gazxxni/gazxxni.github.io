## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 수들을 오름차순으로 정렬하기 위해 Python의 내장 `sort()` 함수를 이용한다. 이 함수는 기본적으로 Timsort 알고리즘을 사용하여 효율적으로 정렬을 수행하며, 입력된 수는 중복되지 않기 때문에 단순히 정렬하면 된다.

### 동작 과정
1. 첫 번째 줄에서 정수 N을 입력받아 정렬할 수의 개수를 저장한다.
2. 이후 N개의 줄에 걸쳐 각 수를 입력받아 리스트 `arr`에 저장한다.
3. `arr.sort()`를 호출하여 리스트를 오름차순으로 정렬한다.
4. 정렬된 리스트의 각 원소를 한 줄에 하나씩 출력한다.

#### 예시 입력
```
5
5
3
2
4
1
```
#### 단계별 설명
- N = 5가 입력된다.
- 다음 5개의 수가 입력되어 리스트 `arr`는 `[5, 3, 2, 4, 1]`이 된다.
- `arr.sort()`를 호출하면 리스트는 `[1, 2, 3, 4, 5]`로 정렬된다.
- 이 후, 각 원소를 한 줄에 하나씩 출력하면 다음과 같은 결과가 나온다:
```
1
2
3
4
5
```

### 시간 복잡도
정렬 알고리즘의 시간 복잡도는 O(N log N)이다. 여기서 N은 정렬할 숫자의 개수로, 최대 1,000개의 수를 처리할 수 있다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
