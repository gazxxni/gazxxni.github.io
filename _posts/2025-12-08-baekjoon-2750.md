## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
주어진 N개의 정수를 오름차순으로 정렬하기 위해, Python의 내장 정렬 기능인 `sort()`를 사용한다. 입력받은 수들은 리스트에 저장하고, 이 리스트를 정렬한 후, 정렬된 결과를 출력한다. 중복되는 수는 없으므로 단순히 정렬하면 된다.

### 동작 과정
1. 첫 번째 줄에서 정수 N을 입력받는다. (예: N = 5)
2. 다음 N개의 줄에서 각 정수를 입력받아 리스트 `arr`에 저장한다. (예: 입력 수 = [3, 1, -2, 5, 0])
3. `arr.sort()`를 호출하여 리스트를 오름차순으로 정렬한다. (정렬 결과 = [-2, 0, 1, 3, 5])
4. 정렬된 리스트를 반복문을 통해 한 줄씩 출력한다.
   - 출력: 
     ```
     -2
     0
     1
     3
     5
     ```

### 시간 복잡도
정렬 알고리즘의 시간 복잡도는 O(N log N)이다. 여기서 N은 입력으로 주어진 수의 개수이다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
