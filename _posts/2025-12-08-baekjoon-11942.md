## 문제
고려대학교는 사랑입니다.

---

## 입력
Empty

---

## 출력
첫째 줄에 “고려대학교”를 출력하세요.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
이 문제는 단순히 문자열을 출력하는 문제입니다. 입력이 없고, 요구되는 출력만 있기 때문에, 문제의 핵심은 해당 문자열을 정확하게 출력하는 것입니다.

### 동작 과정
1. 프로그램이 실행되면 입력을 기다리지만, 이 문제에서는 입력이 필요하지 않습니다.
2. `print` 함수를 사용하여 "고려대학교"라는 문자열을 출력합니다.
3. 이때, 문자열은 정확한 형식으로 출력되어야 합니다.

예시 입력:
```
(입력이 없음)
```

예시 출력:
```
고려대학교
```

### 시간 복잡도
O(1) - 출력하는 데 걸리는 시간은 문자열의 길이에 관계없이 항상 일정합니다.

## 코드
```python
print("고려대학교")
```
