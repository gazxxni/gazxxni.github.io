## 풀이

### 풀이 핵심 로직
이 문제는 주어진 세 변의 길이가 형성할 수 있는 삼각형의 종류를 판별하는 것입니다. 세 변의 길이를 입력받아, 삼각형의 성질에 따라 'Invalid', 'Equilateral', 'Isosceles', 'Scalene' 중 하나를 출력합니다. 삼각형이 성립하기 위해서는 가장 긴 변이 다른 두 변의 합보다 작아야 하며, 이를 바탕으로 삼각형의 종류를 판별합니다.

### 동작 과정
1. 사용자로부터 세 개의 정수를 입력받습니다. 이 값들은 삼각형의 세 변의 길이를 나타냅니다.
2. 입력된 값이 모두 0인 경우, 프로그램을 종료합니다.
3. 입력받은 세 변의 길이를 정렬하여 `a`, `b`, `c`로 할당합니다. 여기서 `c`는 가장 긴 변이 됩니다.
4. `c`가 `a + b`보다 크거나 같으면 'Invalid'를 출력합니다. 이는 삼각형의 성립 조건을 위반하는 경우입니다.
5. 세 변이 모두 같으면 'Equilateral'을 출력합니다.
6. 두 변이 같고 한 변이 다르면 'Isosceles'를 출력합니다.
7. 세 변이 모두 다르면 'Scalene'을 출력합니다.

예시 입력:
```
3 4 5
```
- 입력받은 값은 3, 4, 5입니다.
- 정렬 후 `a = 3`, `b = 4`, `c = 5`가 됩니다.
- `c(5)`는 `a(3) + b(4) = 7`보다 작으므로 삼각형이 성립합니다.
- 세 변이 모두 다르므로 'Scalene'이 출력됩니다.

예시 입력:
```
0 0 0
```
- 입력받은 값이 모두 0이므로 프로그램이 종료됩니다.

### 시간 복잡도
O(1) 
- 입력값의 개수는 항상 3개로 고정되어 있어, 입력받고 처리하는 데 소요되는 시간은 상수 시간입니다.

---

## 코드
```python
while True:
    a, b, c = map(int, input().split())
    
    if a == 0 and b == 0 and c == 0:
        exit()
        
    a, b, c = sorted((a, b, c))
    if c >= a + b:
        print('Invalid')       
    
    elif a == b and b == c:
        print('Equilateral')
        
    elif a == b or b == c or a == c:
        print('Isosceles')
        
    else:
        print('Scalene')
```
