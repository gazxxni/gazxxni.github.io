## 문제
육각형 블록들로 이루어진 RPG 세계가 있다. 그 세계에 나라를 세우려고 하는 군주 캐릭터 송유진은 일반 블록을 울타리 블록으로 바꿀 수 있는 아이템을 N개 가지고 있다. 유진이가 이 N개의 아이템을 이용해서 점령할 수 있는 최대의 영토의 넓이를 구해보자. 울타리 안에 둘러싸인 블록들은 당연히 넓이에 포함 시키고, 울타리를 세운 블록도 넓이에 포함을 시킨다. 울타리는 항상 이어져 있어야 하며, 맵의 넓이는 무한하다.

---

## 입력
첫 줄에는 송유진이 가지고 있는 아이템의 수 N(1≤N≤1,000,000)이 주어진다.

---

## 출력
N개의 아이템을 이용하여 점령할 수 있는 최대의 블록 수를 출력한다.

---

## 풀이

### 풀이 핵심 로직
문제의 핵심은 울타리 블록이 6개로 구성된 육각형 패턴을 만들어 최대한 많은 블록을 포함하는 것이다. 주어진 N개의 아이템을 사용하여 가능한 최대 면적을 계산하기 위해, 울타리를 만들기 위해 필요한 블록 수를 감소시키며 그에 따라 포함되는 블록의 수를 누적해 나간다.

### 동작 과정
입력으로 N이 주어지면, 먼저 N이 6으로 나누어 떨어지는지 검사하여 초기 카운트를 설정한다. 이후, N이 0이 될 때까지 6씩 감소시키며 현재의 N 값을 카운트에 추가한다.

예시 입력:
- N = 12
1. 초기 cnt = 0
2. 12는 6으로 나누어 떨어지므로, cnt += 1 -> cnt = 1
3. n = 12, cnt += n -> cnt = 1 + 12 = 13, n -= 6 -> n = 6
4. n = 6, cnt += n -> cnt = 13 + 6 = 19, n -= 6 -> n = 0
5. n = 0이므로 종료.

결과적으로 cnt = 19가 출력된다.

### 시간 복잡도
O(N) - N의 값에 따라 루프가 최대 N/6번 반복되므로, 전체 반복 횟수는 N에 비례한다.

---

## 코드
```python
n = int(input())

cnt = 0

if n % 6 == 0:
    cnt += 1
    
while n > 0:
    cnt += n
    n -= 6

print(cnt)
```
