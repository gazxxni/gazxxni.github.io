## 풀이

### 풀이 핵심 로직
이 코드는 입력된 문자열을 10자씩 나누어 출력하는 문제를 해결합니다. 주어진 문자열의 길이를 기준으로 10자씩 잘라서 출력하고, 마지막에 남은 문자는 별도로 출력합니다. 이를 위해 문자열을 리스트로 변환하고, 두 개의 배열을 사용하여 데이터를 저장합니다.

### 동작 과정
1. 입력 문자열을 리스트로 변환하고, 공백을 제거합니다.
   - 예시 입력: `abcdefghijklmnopqrstu`
   - 변환 후: `n = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k', 'l', 'm', 'n', 'o', 'p', 'q', 'r', 's', 't', 'u']`
   
2. 문자열의 길이를 10으로 나누어 몇 개의 완전한 줄이 만들어질 수 있는지 계산합니다.
   - `a = len(n) // 10` → `a = 2`
   - `b = len(n) % 10` → `b = 2`

3. 10자씩 자른 부분을 저장할 2D 배열 `arr`를 초기화합니다.
   - `arr = [[0] * 10 for _ in range(a)]` → `arr = [['0'] * 10, ['0'] * 10]`

4. 이중 루프를 사용하여 각 줄에 10자를 할당합니다.
   - 첫 번째 줄: `['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j']`
   - 두 번째 줄: `['k', 'l', 'm', 'n', 'o', 'p', 'q', 'r', 's', 't']`

5. 남은 문자는 별도의 리스트 `brr`에 저장합니다.
   - `brr = ['u']`

6. 각 줄을 출력합니다.
   - 출력: 
     ```
     abcdefghij
     klmnopqrst
     ```

7. 남은 문자를 출력합니다.
   - 출력: `u`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 여기서 n은 입력 문자열의 길이입니다. 전체 문자열을 한번 순회하여 10자씩 나누고 나머지를 처리하기 때문입니다.

---

## 코드
```python
n = list(input().rstrip())

a = len(n) // 10
b = len(n) % 10

arr = [[0] * 10 for _ in range(a)]
brr = []

for i in range(a):
    for j in range(10):
        arr[i][j] = n[10 * i + j]
        
for i in range(len(n) - b, len(n)):
    brr.append(n[i])

for row in arr:
    print(*row, sep='')

print(*brr, sep='')
```
