## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 각 사람의 대기 시간을 최소화하기 위해, 주어진 시간들을 오름차순으로 정렬한 후, 누적 대기 시간을 계산하는 것입니다. 이렇게 하면, 대기 시간이 짧은 사람부터 처리함으로써 전체 대기 시간을 최소화할 수 있습니다.

### 동작 과정
1. 입력받은 사람 수 `n`과 각 사람의 대기 시간 리스트 `a`를 받습니다.
2. 리스트 `a`를 오름차순으로 정렬합니다.
3. 대기 시간의 누적 합을 계산하기 위해 `cnt` 변수를 사용합니다. 
   - 첫 번째 사람의 대기 시간이 `a[0]`이므로 `cnt`에 `a[0]`을 더하고, 현재까지의 대기 시간 `ans`에 `cnt`를 더합니다.
   - 두 번째 사람의 대기 시간 `a[1]`을 `cnt`에 더하고, 현재까지의 대기 시간 `ans`에 `cnt`를 다시 더합니다.
   - 이 과정을 반복하여 마지막 사람까지 진행합니다.
4. 최종적으로 계산된 `ans`를 출력합니다.

#### 예시 입력
- 입력: 
  ```
  5
  3 1 4 3 2
  ```
- 단계별 설명:
  1. `n`은 5, `a`는 `[3, 1, 4, 3, 2]`입니다.
  2. 정렬 후 `a`는 `[1, 2, 3, 3, 4]`로 됩니다.
  3. 대기 시간 계산:
     - `cnt = 0`, `ans = 0`
     - 1번째 사람: `cnt += 1` → `cnt = 1`, `ans += 1` → `ans = 1`
     - 2번째 사람: `cnt += 2` → `cnt = 3`, `ans += 3` → `ans = 4`
     - 3번째 사람: `cnt += 3` → `cnt = 6`, `ans += 6` → `ans = 10`
     - 4번째 사람: `cnt += 3` → `cnt = 9`, `ans += 9` → `ans = 19`
     - 5번째 사람: `cnt += 4` → `cnt = 13`, `ans += 13` → `ans = 32`
  4. 최종 대기 시간 합인 `ans = 32`를 출력합니다.

### 시간 복잡도
정렬하는 과정에 O(n log n)의 시간이 필요하고, 대기 시간 계산은 O(n)입니다. 따라서 전체 시간 복잡도는 O(n log n)입니다.

## 코드
```python
import sys

n=int(input())
a=list(map(int,input().split()))

a.sort()

cnt=0
ans=0
for i in a:
    cnt+=i
    ans+=cnt

print(ans)


```
