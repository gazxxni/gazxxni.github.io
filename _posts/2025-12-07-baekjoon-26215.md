## 문제
지난 밤 겨울 숲에는 눈이 많이 내렸다. 당신은 숲의 주민들을 위해 눈이 오지 않는 동안 모든 집 앞의 눈을 치우고자 한다.

당신은 1분에 한 번씩 두 집을 선택해서 두 집 앞의 눈을 각각 1만큼 치우거나, 한 집을 선택해서 그 집 앞의 눈을 1만큼 치울 수 있다.

모든 집 앞의 눈을 전부 치울 때까지 걸리는 최소 시간은 얼마일까?

---

## 입력
첫 줄에 집의 수를 의미하는 정수 N$N$ (1≤N≤100$1 \leq N \leq 100$)이 주어진다.

다음 줄에는 각각의 집 앞에 쌓여 있는 눈의 양을 나타내는 정수 ai$a_{i}$ (1≤ai≤2000$1 \leq a_{i} \leq 2000$)이 주어진다.

---

## 출력
모든 집 앞의 눈을 치우는 데 최소 몇 분이 걸리는지를 출력한다. 24시간(1440분)이 넘게 걸릴 경우 -1을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 두 집의 눈을 동시에 치우는 방식으로 최소 시간을 계산하는 것입니다. 가장 눈이 많이 쌓인 두 집을 선택하여 동시에 눈을 치우는 방식으로 진행하며, 이를 반복하여 모든 집의 눈을 치웁니다. 이를 통해 시간을 최적화할 수 있습니다.

### 동작 과정
1. **입력 예시**: 집의 수 N = 5, 각 집 앞의 눈의 양 = [4, 7, 2, 8, 5]
2. **초기 상태**: 집 앞의 눈 = [4, 7, 2, 8, 5]
3. **첫 번째 반복**:
   - 집 앞의 눈을 내림차순으로 정렬: [8, 7, 5, 4, 2]
   - 가장 많은 눈이 쌓인 두 집(8, 7)을 선택하여 눈을 치움: [7, 6, 5, 4, 2]
   - 시간 카운트 증가: cnt = 7
4. **두 번째 반복**:
   - 정렬: [7, 6, 5, 4, 2]
   - 두 집(7, 6)을 선택하여 눈을 치움: [6, 5, 5, 4, 2]
   - cnt = 13
5. **세 번째 반복**:
   - 정렬: [6, 5, 5, 4, 2]
   - 두 집(6, 5)을 선택하여 눈을 치움: [5, 4, 5, 4, 2]
   - cnt = 18
6. **이 과정을 반복하여 모든 집의 눈이 0이 될 때까지 진행**.
7. **최종 카운트가 1440을 초과하면 -1을 출력**.

### 시간 복잡도
이 알고리즘의 주요 반복문은 집의 수 N에 대해 O(N log N)으로 정렬을 수행하며, 집의 수가 최대 100이므로 최악의 경우 O(N^2 log N) 정도의 시간이 걸립니다. 따라서, 전체 시간 복잡도는 O(N^2 log N)입니다.

## 코드
```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))

cnt = 0

while True:
    if len(arr) == 1:
        break
    
    arr.sort(reverse=True)
    arr[0] -= arr[1]
    cnt += arr[1]
    arr.pop(1)
    
cnt += arr[0]
if cnt > 1440:
    print(-1)
else:
    print(cnt)
```
