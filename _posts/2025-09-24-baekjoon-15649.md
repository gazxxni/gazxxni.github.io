## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.


	1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열

---

## 입력
첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)

---

## 출력
한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

## 풀이

### 풀이 핵심 로직
이 문제는 백트래킹을 활용하여 자연수 1부터 N까지 중복 없이 M개를 고르는 모든 수열을 생성하는 것입니다. 현재 수열의 길이가 M에 도달했을 때 수열을 출력하고, 그렇지 않다면 가능한 숫자를 추가하여 재귀적으로 탐색합니다.

### 동작 과정
1. **입력**: N과 M을 입력받습니다. 예를 들어, N=4, M=2라면 1부터 4까지의 수 중 2개를 선택해야 합니다.
   
2. **초기화**: 빈 리스트 `arr`를 생성하여 선택된 숫자를 저장합니다.

3. **재귀 함수 `aa()` 호출**:
   - `aa()` 함수는 현재 리스트 `arr`의 길이가 M에 도달했는지를 확인합니다.
   - 만약 M에 도달하면 리스트의 모든 요소를 공백으로 구분하여 출력합니다.
   - 그렇지 않으면 1부터 N까지의 숫자에 대해 반복문을 실행하여:
     - 해당 숫자가 `arr`에 포함되어 있지 않으면 추가하고, 함수 `aa()`를 재귀적으로 호출합니다.
     - 재귀 호출이 끝난 후에는 마지막에 추가한 숫자를 제거하여 다음 숫자를 선택할 수 있게 합니다.

4. **출력**: 모든 가능한 조합이 탐색이 끝나면, 각 조합이 사전 순으로 출력됩니다.

**예시 입력**: 
```
4 2
```
- 출력: 
```
1 2
1 3
1 4
2 1
2 3
2 4
3 1
3 2
3 4
4 1
4 2
4 3
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(M! * N)입니다. M!은 M개의 숫자를 선택하는 순열의 수이며, 각 선택에 대해 N개의 숫자 중에서 후보를 선택하는 과정이 있기 때문입니다. N은 최대 8이므로, 이 범위에서는 충분히 효율적으로 실행됩니다.

## 코드
```python
n, m = map(int, input().split())
arr = []

def aa():
    if len(arr) == m:
        print(*arr)
        return
    
    for i in range(1, n + 1):
        if i not in arr:
            arr.append(i)
            aa()
            arr.pop()
            
aa()
```
