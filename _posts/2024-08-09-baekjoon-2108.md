## 풀이

### 풀이 핵심 로직
이 코드는 주어진 수의 통계적 특성(산술 평균, 중앙값, 최빈값, 범위)을 계산하는 문제를 해결합니다. 입력된 수들을 정렬하고, 각 수의 빈도를 계산하여 최빈값을 찾으며, 간단한 수학적 계산을 통해 평균과 범위를 구합니다.

### 동작 과정
1. **입력 받기**: 주어진 수의 개수 `n`을 입력받고, `n`개의 수를 리스트 `stack`에 저장합니다. 이와 동시에 합계 `my_sum`과 각 수의 빈도를 저장하는 딕셔너리 `cnt`를 업데이트합니다.
   
   예를 들어, 입력이 다음과 같다고 가정해 보겠습니다.
   ```
   5
   1
   2
   2
   3
   4
   ```
   초기화 후 `stack`은 `[1, 2, 2, 3, 4]`, `my_sum`은 `12`, `cnt`는 `{1: 1, 2: 2, 3: 1, 4: 1}`이 됩니다.

2. **산술 평균 계산**: `my_sum`을 `n`으로 나누고 반올림하여 출력합니다. 위 예시에서는 `12 / 5`를 반올림하여 `3`이 출력됩니다.

3. **중앙값 계산**: `stack`을 정렬한 후, 중앙값을 찾습니다. 리스트의 길이가 홀수이므로 인덱스 `n // 2`의 값을 출력합니다. 위 예시에서는 `2`가 출력됩니다.

4. **최빈값 계산**: `cnt` 딕셔너리에서 가장 높은 빈도를 찾고, 해당 빈도를 가진 모든 수를 리스트 `number`에 추가합니다. 
   - 위 예시에서는 `2`가 가장 많이 등장하므로 `number`는 `[2]`가 됩니다. 최빈값이 하나일 경우 직접 출력하고, 여러 개일 경우 두 번째로 작은 값을 출력합니다.

5. **범위 계산**: `stack`의 마지막 값에서 첫 번째 값을 빼서 범위를 계산하고 출력합니다. 위 예시에서는 `4 - 1 = 3`이 출력됩니다.

### 시간 복잡도
이 코드는 입력받은 수의 개수 `n`에 대해 다음과 같은 작업을 수행합니다:
- 입력 수를 읽고 리스트에 저장하는 과정: O(n)
- 리스트를 정렬하는 과정: O(n log n)
- 최빈값을 찾기 위한 과정: O(n)

따라서 전체 시간 복잡도는 O(n log n)입니다.

## 코드
```python
import sys

input = sys.stdin.readline

n=int(input())
stack=[]

my_sum=0
cnt=dict()

for _ in range(n):
    a=int(input())
    stack.append(a)

    my_sum+=a

    if a not in cnt:
        cnt[a]=1
    else:
        cnt[a]+=1

stack.sort()

print(round(my_sum/n))
print(stack[n//2])

freq=max(cnt.values())
number=[]

for key,value in cnt.items():
    if value == freq:
        number.append(key)

if len(number) ==1:
    print(number[0])
else:
    print(sorted(number)[1])

print(stack[-1]-stack[0])


```
