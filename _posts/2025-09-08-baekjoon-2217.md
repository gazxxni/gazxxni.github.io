## 풀이

### 풀이 핵심 로직
이 문제는 각 로프의 인장 강도를 최대화하기 위해, 각 로프의 최대 중량을 계산하고, 그 중 최대값을 찾는 문제입니다. 로프의 인장 강도는 로프의 수와 해당 로프가 견딜 수 있는 최대 중량의 곱으로 계산되며, 주어진 로프의 수에 따라 최대 중량을 조정하여 최적의 값을 찾아야 합니다.

### 동작 과정
1. 입력으로 로프의 개수 `n`을 받고, 각 로프가 견딜 수 있는 최대 중량을 리스트 `arr`에 저장한다.
   예시: 
   ```
   n = 3
   arr = [10, 20, 30]
   ```

2. 리스트 `arr`를 오름차순으로 정렬한다.
   ```
   arr = [10, 20, 30]  # 이미 정렬됨
   ```

3. 변수 `a`를 -1로 초기화하고, `n`을 로프의 개수로 두고 반복문을 통해 각 로프의 인장 강도를 계산한다.
   - 첫 번째 로프(30kg)의 경우: `30 * 3 = 90` (3개의 로프)
   - 두 번째 로프(20kg)의 경우: `20 * 2 = 40` (2개의 로프)
   - 세 번째 로프(10kg)의 경우: `10 * 1 = 10` (1개의 로프)

4. 각 계산된 인장 강도 중 최대값을 갱신하며 진행한다.
   ```
   a = max(-1, 30 * 3) = 90
   a = max(90, 20 * 2) = 90
   a = max(90, 10 * 1) = 90
   ```

5. 최종적으로 `a`를 출력한다.
   ```
   print(a)  # 결과: 90
   ```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. 입력받은 로프의 중량을 정렬하는 데 O(n log n)의 시간이 소요되고, 그 후의 반복문은 O(n) 시간이 걸리므로 전체적으로 O(n log n)입니다.

---

## 코드
```python
n = int(input())

arr = [int(input()) for _ in range(n)]
arr.sort()
a = -1

for i in range(n):
    a = max(a, arr[i] * n)
    n -= 1
    
    
print(a)
```
