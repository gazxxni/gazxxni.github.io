## 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

---

## 출력
첫째 줄에 N!을 출력한다.

---

## 성능 요약

## 풀이

### 풀이 핵심 로직
이 문제는 재귀를 이용하여 팩토리얼을 계산하는 방식으로 해결할 수 있습니다. N!은 N * (N-1)!으로 정의되며, 기저 사례로 0!과 1!은 1로 정의됩니다. 이를 활용하여 함수가 호출될 때마다 N의 값을 하나씩 줄여가며 곱셈을 수행합니다.

### 동작 과정
입력으로 N이 5라고 가정해 보겠습니다.
1. `factorial(5)` 호출: 5가 1보다 크므로 `factorial(4) * 5`로 재귀 호출.
2. `factorial(4)` 호출: 4가 1보다 크므로 `factorial(3) * 4`로 재귀 호출.
3. `factorial(3)` 호출: 3이 1보다 크므로 `factorial(2) * 3`으로 재귀 호출.
4. `factorial(2)` 호출: 2가 1보다 크므로 `factorial(1) * 2`로 재귀 호출.
5. `factorial(1)` 호출: 1이 1과 같으므로 1을 반환 (기저 사례).
6. 이제 반환이 시작됨: `factorial(2)`는 `1 * 2`로 2 반환, `factorial(3)`은 `2 * 3`으로 6 반환, `factorial(4)`는 `6 * 4`로 24 반환, 마지막으로 `factorial(5)`는 `24 * 5`로 120 반환.
7. 최종적으로 120이 출력됩니다.

### 시간 복잡도
재귀 호출의 깊이는 N에 비례하므로 시간 복잡도는 O(N)입니다.

---

## 코드
```python
def factorial(n):
    if n <= 1:
        ans = 1
    else:
        ans = factorial(n-1) * n
        
    return ans

print(factorial(int(input())))
```
