## 문제 링크
[https://www.acmicpc.net/problem/7891](https://www.acmicpc.net/problem/7891)

---

## 문제
Given two integers, calculate and output their sum.

---

## 입력
The input contains several test cases. The first line contains and integer t (t ≤ 100) denoting the number of test cases. Then t tests follow, each of them consisiting of two space separated integers x and y (−109 ≤ x, y ≤ 109).

---

## 출력
For each test case output output the sum of the corresponding integers.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

## 풀이

### 풀이 핵심 로직
주어진 문제는 여러 테스트 케이스에 대해 두 정수를 입력받아 그 합을 출력하는 간단한 사칙연산 문제입니다. 각 테스트 케이스마다 두 정수를 읽고, 그들의 합을 계산하여 출력하면 됩니다.

### 동작 과정
1. 첫 번째 줄에서 테스트 케이스의 수 `t`를 입력받습니다.
2. 그 다음 `t` 만큼 반복하여 각 테스트 케이스에 대해 두 개의 정수 `x`와 `y`를 입력받습니다.
3. 입력받은 두 정수를 더한 값을 출력합니다.

예시 입력:
```
3
1 2
-1 1
1000000000 -1000000000
```

단계별 설명:
- 첫 번째 줄에서 `t`가 3이므로 3개의 테스트 케이스를 처리합니다.
- 첫 번째 테스트 케이스에서 `1`과 `2`를 입력받아, `1 + 2 = 3`을 출력합니다.
- 두 번째 테스트 케이스에서 `-1`과 `1`을 입력받아, `-1 + 1 = 0`을 출력합니다.
- 세 번째 테스트 케이스에서 `1000000000`과 `-1000000000`을 입력받아, `1000000000 + (-1000000000) = 0`을 출력합니다.

결과 출력:
```
3
0
0
```

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(t)입니다. 여기서 `t`는 테스트 케이스의 수로, 각 테스트 케이스마다 상수 시간 O(1) 내에 결과를 계산하고 출력하기 때문입니다.

---

## 코드
```python
t = int(input())

for _ in range(t):
    a, b = map(int, input().split())
    
    print(a + b)
```
