## 문제 링크
[https://www.acmicpc.net/problem/7891](https://www.acmicpc.net/problem/7891)

---

## 문제
Given two integers, calculate and output their sum.

---

## 입력
The input contains several test cases. The first line contains and integer t (t ≤ 100) denoting the number of test cases. Then t tests follow, each of them consisiting of two space separated integers x and y (−109 ≤ x, y ≤ 109).

---

## 출력
For each test case output output the sum of the corresponding integers.

---

## 성능 요약
메모리: 108384 KB, 시간: 92 ms

---

## 풀이

### 풀이 핵심 로직
주어진 문제는 여러 테스트 케이스에 대해 두 개의 정수를 입력받아 그 합을 출력하는 간단한 사칙연산 문제입니다. 각 테스트 케이스의 합을 계산하기 위해 반복문을 사용하여 입력된 두 정수를 더한 후 결과를 출력합니다.

### 동작 과정
1. 첫 번째 줄에서 테스트 케이스의 수 `t`를 입력받습니다.
2. `t`번 반복하여 각 테스트 케이스에 대해 다음을 수행합니다:
   - 두 개의 정수 `x`와 `y`를 공백으로 구분하여 입력받습니다.
   - `x`와 `y`를 더하여 그 결과를 출력합니다.

예를 들어, 입력이 다음과 같다고 가정합니다:
```
3
1 2
-3 5
0 0
```
- 첫 번째 테스트 케이스에서 `1`과 `2`를 더하여 `3`을 출력합니다.
- 두 번째 테스트 케이스에서 `-3`과 `5`를 더하여 `2`을 출력합니다.
- 세 번째 테스트 케이스에서 `0`과 `0`을 더하여 `0`을 출력합니다.

결과적으로 출력은 다음과 같습니다:
```
3
2
0
```

### 시간 복잡도
이 문제의 시간 복잡도는 O(t)입니다. 여기서 `t`는 테스트 케이스의 수로, 각 테스트 케이스마다 상수 시간 O(1) 내에 연산을 수행하므로 전체적으로는 `t`에 비례하는 시간 복잡도를 가집니다.

---

## 코드
```python
t = int(input())

for _ in range(t):
    a, b = map(int, input().split())
    
    print(a + b)
```
