## 문제
총 N개의 문자열로 이루어진 집합 S가 주어진다.

입력으로 주어지는 M개의 문자열 중에서 집합 S에 포함되어 있는 것이 총 몇 개인지 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 문자열의 개수 N과 M (1 ≤ N ≤ 10,000, 1 ≤ M ≤ 10,000)이 주어진다. 

다음 N개의 줄에는 집합 S에 포함되어 있는 문자열들이 주어진다.

다음 M개의 줄에는 검사해야 하는 문자열들이 주어진다.

입력으로 주어지는 문자열은 알파벳 소문자로만 이루어져 있으며, 길이는 500을 넘지 않는다. 집합 S에 같은 문자열이 여러 번 주어지는 경우는 없다.

---

## 출력
첫째 줄에 M개의 문자열 중에 총 몇 개가 집합 S에 포함되어 있는지 출력한다.

## 풀이

### 풀이 핵심 로직
문자열 집합 S에 포함된 문자열의 개수를 세기 위해, 집합 자료구조를 활용하여 문자열의 포함 여부를 효율적으로 확인합니다. 입력받은 M개의 문자열 각각에 대해 집합 S에 존재하는지를 검사하고, 존재할 경우 카운트를 증가시킵니다.

### 동작 과정
1. 입력으로 N과 M을 읽어옵니다. 예를 들어, N=5, M=4라고 가정합니다.
2. 집합 S에 N개의 문자열을 추가합니다.
   - 입력 예시:
     ```
     apple
     banana
     cherry
     date
     elderberry
     ```
   - 그러면 집합 S는 `{'apple', 'banana', 'cherry', 'date', 'elderberry'}`가 됩니다.
3. M개의 문자열을 차례대로 입력받고, 각 문자열이 집합 S에 포함되어 있는지 확인합니다.
   - 입력 예시:
     ```
     apple
     fig
     cherry
     grape
     ```
   - 'apple'과 'cherry'는 집합 S에 포함되어 있지만, 'fig'와 'grape'는 포함되어 있지 않습니다.
4. 포함된 문자열의 개수를 카운트하여 최종 결과를 출력합니다. 위 예시에서는 2가 출력됩니다.

### 시간 복잡도
입력받는 문자열의 개수 N과 M이 최대 10,000이므로, 문자열을 집합에 추가하는 과정은 O(N)이고, 각 문자열의 포함 여부를 검사하는 과정은 O(1)의 평균 시간 복잡도를 가집니다. 따라서 전체 시간 복잡도는 O(N + M)입니다.

## 코드
```python
n, m = map(int, input().split())

arr = set()
for _ in range(n):
    arr.add(input().rstrip())

cnt = 0
for _ in range(m):
    s = input().rstrip()
    if s in arr:
        cnt += 1

print(cnt)
```
