## 문제
영선이는 길이가 a, b, c인 세 막대를 가지고 있고, 각 막대의 길이를 마음대로 줄일 수 있다.

영선이는 세 막대를 이용해서 아래 조건을 만족하는 삼각형을 만들려고 한다.


	각 막대의 길이는 양의 정수이다
	세 막대를 이용해서 넓이가 양수인 삼각형을 만들 수 있어야 한다.
	삼각형의 둘레를 최대로 해야 한다.


a, b, c가 주어졌을 때, 만들 수 있는 가장 큰 둘레를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 100)가 주어진다.

---

## 출력
첫째 줄에 만들 수 있는 가장 큰 삼각형의 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 세 막대의 길이로 삼각형을 만들기 위해서는 삼각형의 성질을 만족해야 합니다. 즉, 두 막대의 길이의 합이 나머지 막대의 길이보다 커야 합니다. 이 조건을 이용하여 가능한 최대 둘레를 계산합니다. 만약 삼각형을 만들 수 없다면, 가장 큰 둘레를 만들기 위해 두 가장 짧은 막대의 합에서 1을 빼서 둘레를 계산합니다.

### 동작 과정
1. 입력으로 길이 a, b, c가 주어집니다. 예를 들어 `a = 5`, `b = 3`, `c = 2`라고 가정합니다.
2. 세 막대의 길이를 정렬하여 `a <= b <= c`가 되도록 합니다. 이 경우 `a = 2`, `b = 3`, `c = 5`가 됩니다.
3. 삼각형의 성질을 검사합니다. `a + b`가 `c`보다 크면(즉, `2 + 3 > 5`가 아니라면) 정상적인 삼각형을 만들 수 있습니다.
4. 만약 `a + b <= c`라면, 삼각형을 만들 수 없으므로 가능한 최대 둘레는 `(a + b) * 2 - 1`이 됩니다. 예를 들어 `a + b = 5`이면, `5 * 2 - 1 = 9`가 됩니다.
5. 반면, 삼각형을 만들 수 있다면 `a + b + c`를 출력합니다. 예를 들어 `2 + 3 + 5 = 10`이 됩니다.

최종적으로 출력은 `10`입니다.

### 시간 복잡도
전체 알고리즘은 입력된 길이를 정렬하고 조건을 체크하는 상수 시간 연산으로 이루어져 있으므로, 시간 복잡도는 O(1)입니다.

---

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
