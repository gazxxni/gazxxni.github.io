## 문제
그룹 단어란 단어에 존재하는 모든 문자에 대해서, 각 문자가 연속해서 나타나는 경우만을 말한다. 예를 들면, ccazzzzbb는 c, a, z, b가 모두 연속해서 나타나고, kin도 k, i, n이 연속해서 나타나기 때문에 그룹 단어이지만, aabbbccb는 b가 떨어져서 나타나기 때문에 그룹 단어가 아니다.

단어 N개를 입력으로 받아 그룹 단어의 개수를 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 단어의 개수 N이 들어온다. N은 100보다 작거나 같은 자연수이다. 둘째 줄부터 N개의 줄에 단어가 들어온다. 단어는 알파벳 소문자로만 되어있고 중복되지 않으며, 길이는 최대 100이다.

---

## 출력
첫째 줄에 그룹 단어의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
그룹 단어는 각 문자가 연속해서 나타나는 경우만을 허용하므로, 문자열을 순회하면서 문자의 연속성을 확인해야 한다. 현재 문자가 이전 문자와 같지 않으면서 이미 등장한 문자라면 그룹 단어가 아님을 알 수 있다. 이를 통해 문자열을 검사하고, 그룹 단어의 개수를 세면 된다.

### 동작 과정
1. 입력으로 주어진 단어의 개수 N을 읽어온다.
2. N개의 단어를 각각 검사한다.
3. 각 단어에 대해:
   - 첫 번째 문자를 별도로 처리하여 등장한 문자를 저장할 집합 `alpha`를 초기화한다.
   - 두 번째 문자부터 마지막 문자까지 순회하면서:
     - 현재 문자가 `alpha`에 이미 존재하는 경우:
       - 이전 문자와 다르다면 그룹 단어가 아님을 표시한다.
     - 현재 문자가 `alpha`에 존재하지 않는 경우, `alpha`에 추가한다.
4. 모든 문자를 검사한 후 그룹 단어로 판별되면 카운트를 증가시킨다.
5. 최종적으로 그룹 단어의 개수를 출력한다.

예를 들어, 입력이 다음과 같을 경우:
```
3
happy
new
yummy
```
- "happy"는 'h', 'a', 'p', 'y'가 연속하므로 그룹 단어이다.
- "new"는 'n', 'e', 'w'가 연속하므로 그룹 단어이다.
- "yummy"는 'y', 'u', 'm'이 연속하나 'm'이 떨어져 나타나므로 그룹 단어가 아니다.
- 결과적으로 그룹 단어는 2개이므로 출력은 2가 된다.

### 시간 복잡도
O(N * M), 여기서 N은 단어의 개수, M은 각 단어의 최대 길이이다. 각 단어에 대해 문자들을 한 번씩 검사하므로 이와 같은 시간 복잡도를 가진다.

## 코드
```python
n = int(input())

cnt = 0
for _ in range(n):
    string = input().rstrip()
    alpha = set(string[0])
    flag = True
    for i in range(1, len(string)):
        if string[i] in alpha:
            if string[i - 1] != string[i]:
                flag = False
        else:
            alpha.add(string[i])
    
    if flag:
        cnt += 1

print(cnt)
```
