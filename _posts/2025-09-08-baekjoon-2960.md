[https://www.acmicpc.net/problem/2960](https://www.acmicpc.net/problem/2960)

## 풀이

### 풀이 핵심 로직
이 문제는 에라토스테네스의 체를 이용하여 소수의 배수를 제거하는 방식으로 k번째 지우는 숫자를 찾는 문제입니다. 기본적으로 1부터 n까지의 숫자를 포함하는 배열을 만들고, 소수를 기준으로 그 배수를 지우면서 순서대로 카운트를 하여 k번째로 지운 수를 출력합니다.

### 동작 과정
예시 입력으로 `n = 7`, `k = 4`를 고려해보겠습니다.

1. 초기화: `arr` 배열은 `[0, 1, 2, 3, 4, 5, 6, 7]`로 설정합니다. (0은 사용하지 않음)
2. 첫 번째 반복 (i=2):
   - 2는 0이 아니므로, 2의 배수인 2, 4, 6을 순서대로 지웁니다.
   - `cnt`가 1 (2 지움), 2 (4 지움), 3 (6 지움)으로 증가합니다.
3. 두 번째 반복 (i=3):
   - 3은 0이 아니므로, 3의 배수인 3, 6를 지웁니다.
   - `cnt`가 4 (3 지움)으로 증가합니다. 이때, k와 같으므로 `ans`에 3을 저장하고 반복을 중단합니다.
4. 결과 출력: `ans` 값인 3을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log log n)입니다. 이는 에라토스테네스의 체 알고리즘의 복잡도와 유사하며, 각 숫자를 지우는 과정이 대부분 정수의 배수로 이루어져 있기 때문입니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())

arr = [i for i in range(n+1)] 

cnt = 0
ans = 0

for i in range(2, n + 1):
    
    if arr[i] == 0: 
        continue
    
    for j in range(i, n+1, i):
        
        if arr[j] != 0:
            arr[j] = 0
            cnt += 1
            
            if cnt == k:
                ans = j
                break
        
print(ans)
```
