## 문제
알고리즘 입문방 오픈 채팅방에서는 새로운 분들이 입장을 할 때마다 곰곰티콘을 사용해 인사를 한다. 이를 본 문자열 킬러 임스는 채팅방의 기록을 수집해 그 중 곰곰티콘이 사용된 횟수를 구해 보기로 했다.

ENTER는 새로운 사람이 채팅방에 입장했음을 나타낸다. 그 외는 채팅을 입력한 유저의 닉네임을 나타낸다. 닉네임은 숫자 또는 영문 대소문자로 구성되어 있다.

새로운 사람이 입장한 이후 처음 채팅을 입력하는 사람은 반드시 곰곰티콘으로 인사를 한다. 그 외의 기록은 곰곰티콘을 쓰지 않은 평범한 채팅 기록이다.

채팅 기록 중 곰곰티콘이 사용된 횟수를 구해보자!

---

## 입력
첫 번째 줄에는 채팅방의 기록 수를 나타내는 정수 N$N$ 이 주어진다. (1≤N≤100000$1 \le N \le 100\,000$)

두 번째 줄부터 N$N$ 개의 줄에 걸쳐 새로운 사람의 입장을 나타내는 ENTER, 혹은 채팅을 입력한 유저의 닉네임이 문자열로 주어진다. (문자열길이1≤문자열 길이≤20$1 \le \texttt{문자열 길이} \le 20$)

첫 번째 주어지는 문자열은 무조건 ENTER이다.

---

## 출력
채팅 기록 중 곰곰티콘이 사용된 횟수를 출력하시오.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 사용자가 채팅방에 입장할 때마다 곰곰티콘으로 인사를 하는데, 각 입장 후 최초 채팅 시 해당 닉네임의 수만큼 곰곰티콘을 사용하게 된다는 점입니다. 따라서 `ENTER`가 입력될 때마다 현재까지 입장한 사용자들(닉네임)을 세고, 새로운 사용자가 입장하면 그 수를 카운트하여 초기화하는 방식으로 곰곰티콘의 사용 횟수를 계산합니다.

### 동작 과정
1. 입력값 `N`을 읽고, 초기 곰곰티콘 사용 횟수 `cnt`를 0으로 설정합니다.
2. 사용자 닉네임을 저장할 집합 `arr`를 초기화합니다.
3. 다음 N번 반복:
   - 입력값이 `ENTER`일 경우:
     - 현재까지 저장된 닉네임 수만큼 `cnt`에 더하고, `arr`를 초기화합니다.
   - 그 외의 경우 (닉네임이 입력된 경우):
     - 해당 닉네임을 `arr`에 추가합니다.
4. 반복이 끝나면 마지막으로 `arr`에 남아있는 닉네임 수를 `cnt`에 더합니다.
5. 최종적으로 `cnt`를 출력합니다.

예시 입력:
```
6
ENTER
Alice
Bob
ENTER
Charlie
ENTER
David
```
과정:
- 첫 번째 `ENTER`: `cnt` = 0, `arr` = {}
- `Alice`: `arr` = {Alice}
- `Bob`: `arr` = {Alice, Bob}
- 두 번째 `ENTER`: `cnt` = 2 (Alice, Bob), `arr` = {}
- `Charlie`: `arr` = {Charlie}
- 세 번째 `ENTER`: `cnt` = 2, `arr` = {}
- `David`: `arr` = {David}
- 최종적으로 `cnt`에 마지막 `arr`의 크기 1을 더하여 `cnt` = 3.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 입력을 한 번씩 처리하며, 집합에 원소를 추가하거나 크기를 구하는 연산이 평균적으로 O(1) 시간이 소요되므로, 총 N개의 입력에 대해 선형 시간 복잡도를 가집니다.

## 코드
```python
n = int(input())

cnt = 0
arr = set()
for _ in range(n):
    a = input().rstrip()
    if a == 'ENTER':
        cnt += len(arr)
        arr = set()
        continue
    
    arr.add(a)

cnt += len(arr)
print(cnt)
```
