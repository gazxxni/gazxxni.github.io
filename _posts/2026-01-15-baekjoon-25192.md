## 문제
알고리즘 입문방 오픈 채팅방에서는 새로운 분들이 입장을 할 때마다 곰곰티콘을 사용해 인사를 한다. 이를 본 문자열 킬러 임스는 채팅방의 기록을 수집해 그 중 곰곰티콘이 사용된 횟수를 구해 보기로 했다.

ENTER는 새로운 사람이 채팅방에 입장했음을 나타낸다. 그 외는 채팅을 입력한 유저의 닉네임을 나타낸다. 닉네임은 숫자 또는 영문 대소문자로 구성되어 있다.

새로운 사람이 입장한 이후 처음 채팅을 입력하는 사람은 반드시 곰곰티콘으로 인사를 한다. 그 외의 기록은 곰곰티콘을 쓰지 않은 평범한 채팅 기록이다.

채팅 기록 중 곰곰티콘이 사용된 횟수를 구해보자!

---

## 입력
첫 번째 줄에는 채팅방의 기록 수를 나타내는 정수 N$N$ 이 주어진다. (1≤N≤100000$1 \le N \le 100\,000$)

두 번째 줄부터 N$N$ 개의 줄에 걸쳐 새로운 사람의 입장을 나타내는 ENTER, 혹은 채팅을 입력한 유저의 닉네임이 문자열로 주어진다. (문자열길이1≤문자열 길이≤20$1 \le \texttt{문자열 길이} \le 20$)

첫 번째 주어지는 문자열은 무조건 ENTER이다.

---

## 출력
채팅 기록 중 곰곰티콘이 사용된 횟수를 출력하시오.

## 풀이

### 풀이 핵심 로직
이 문제의 핵심은 새로운 사용자가 채팅방에 들어올 때마다 그 사용자가 곰곰티콘으로 인사하는 횟수를 카운트하는 것입니다. 사용자가 채팅방에 들어올 때마다 현재까지 있는 사용자들의 닉네임을 저장하는 집합(set)을 사용하여 중복을 제거하고, 'ENTER' 명령이 들어올 때마다 현재 집합의 크기만큼 카운트를 증가시킵니다.

### 동작 과정
주어진 입력을 통해 단계별로 설명하겠습니다.

1. **입력 받기**: 
   - 첫 번째 줄에 들어오는 정수 N은 채팅방 기록의 수입니다.
   - 그 이후 N개의 줄에는 'ENTER' 또는 사용자의 닉네임이 주어집니다.

2. **초기화**:
   - 카운트 변수 `cnt`를 0으로 초기화합니다.
   - 사용자 닉네임을 저장할 비어 있는 집합 `arr`을 초기화합니다.

3. **입력 처리**:
   - 반복문을 통해 N번의 입력을 처리합니다.
   - 만약 입력이 'ENTER'인 경우:
     - 현재 집합 `arr`의 크기를 카운트 `cnt`에 더합니다. 이는 새로 들어온 사용자가 이전 사용자 수만큼 곰곰티콘으로 인사했음을 의미합니다.
     - 집합 `arr`을 비웁니다.
   - 그렇지 않은 경우(닉네임 입력):
     - 해당 닉네임을 집합 `arr`에 추가합니다. 집합은 중복을 허용하지 않으므로, 같은 사용자가 여러 번 인사하는 경우를 방지합니다.

4. **마지막 카운트**:
   - 반복문이 끝난 후, 마지막으로 입력된 사용자들의 수(현재 집합의 크기)를 카운트에 더합니다. 이는 마지막 'ENTER' 이후에 들어온 사용자의 인사 횟수를 포함합니다.

5. **결과 출력**:
   - 최종적으로 카운트 `cnt`를 출력합니다.

예시 입력:
```
5
ENTER
user1
user2
ENTER
user3
ENTER
```

- 첫 번째 'ENTER' 후: `cnt = 0`, `arr = {}` (아무도 없음)
- 'user1' 입력: `arr = {'user1'}`
- 'user2' 입력: `arr = {'user1', 'user2'}`
- 두 번째 'ENTER' 후: `cnt += 2 (user1, user2)`, `cnt = 2`, `arr = {}`
- 'user3' 입력: `arr = {'user3'}`
- 마지막 'ENTER' 후: `cnt += 1 (user3)`, `cnt = 3`

결과적으로 출력은 3이 됩니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(N)입니다. 각 입력을 한 번씩만 처리하며, 집합에 원소를 추가하고 크기를 측정하는 연산은 평균적으로 O(1)입니다. 따라서 전체 입력 수 N에 비례하는 시간 복잡도를 가집니다.

## 코드
```python
n = int(input())

cnt = 0
arr = set()
for _ in range(n):
    a = input().rstrip()
    if a == 'ENTER':
        cnt += len(arr)
        arr = set()
        continue
    
    arr.add(a)

cnt += len(arr)
print(cnt)
```
