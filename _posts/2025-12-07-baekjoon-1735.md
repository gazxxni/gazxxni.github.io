## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
두 분수 A/B와 C/D의 합을 구하기 위해, 두 분수의 분모의 최소공배수를 사용하여 동치의 분수로 변환한 다음, 분자를 합산하고 기약분수 형태로 변환합니다. 이를 위해 유클리드 호제법을 사용하여 최대공약수를 구하고, 이를 통해 분수를 약분합니다.

### 동작 과정
1. 입력: 두 분수 A/B와 C/D가 주어진다. 예를 들어, A=1, B=2, C=1, D=3.
2. 최소공배수 계산: 분모 B와 D의 최소공배수를 구한다. 예를 들어, lcm(2, 3) = 6.
3. 분수 변환:
   - A/B는 1/2 → (1 * 3) / (2 * 3) = 3/6
   - C/D는 1/3 → (1 * 2) / (3 * 2) = 2/6
4. 분자 합산: 3 + 2 = 5.
5. 기약분수로 변환: 분자 5와 분모 6의 최대공약수를 구한다. gcd(5, 6) = 1.
6. 약분: (5 / 1) / (6 / 1) = 5/6.
7. 결과 출력: 5 6.

### 시간 복잡도
O(log(min(B, D))) - 유클리드 호제법에 의한 최대공약수 계산의 시간 복잡도입니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
