## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
두 분수의 합을 구할 때, 두 분수의 분모를 통일하여 계산한 후, 합한 분수를 기약분수 형태로 변환하는 방법을 사용합니다. 이를 위해 유클리드 호제법을 사용하여 최대공약수(GCD)를 구하고, 분수를 약분합니다.

### 동작 과정
1. 입력으로 두 분수 A/B와 C/D가 주어진다고 가정합니다. 예를 들어, A=1, B=2, C=3, D=4의 경우를 생각해봅시다.
2. 두 분수의 분모인 B와 D의 최소공배수(LCM)를 계산합니다. 
   - LCM(2, 4) = 4
3. 두 분수를 통일된 분모로 변환합니다.
   - A/B = 1/2 = (1 * 2) / (2 * 2) = 2/4
   - C/D = 3/4 = (3 * 1) / (4 * 1) = 3/4
4. 이제 두 분수를 더합니다.
   - (2/4) + (3/4) = (2 + 3) / 4 = 5/4
5. 합한 분수의 분자와 분모를 최대공약수로 나누어 기약분수로 변환합니다.
   - GCD(5, 4) = 1 이므로, 기약분수는 5/4가 됩니다.
6. 최종적으로 5와 4를 출력합니다.

### 시간 복잡도
O(log(min(B, D))) - 유클리드 호제법을 사용하여 GCD를 계산하는 과정의 시간 복잡도입니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
