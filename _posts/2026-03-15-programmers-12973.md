## 문제
짝지어 제거하기는, 알파벳 소문자로 이루어진 문자열을 가지고 시작합니다. 먼저 문자열에서 같은 알파벳이 2개 붙어 있는 짝을 찾습니다. 그다음, 그 둘을 제거한 뒤, 앞뒤로 문자열을 이어 붙입니다. 이 과정을 반복해서 문자열을 모두 제거한다면 짝지어 제거하기가 종료됩니다. 문자열 S가 주어졌을 때, 짝지어 제거하기를 성공적으로 수행할 수 있는지 반환하는 함수를 완성해 주세요. 성공적으로 수행할 수 있으면 1을, 아닐 경우 0을 리턴해주면 됩니다.

예를 들어, 문자열 S = baabaa 라면

b aa baa → bb aa → aa →

의 순서로 문자열을 모두 제거할 수 있으므로 1을 반환합니다.

## 풀이
### 풀이 핵심 로직
주어진 문자열에서 같은 알파벳이 2개 붙어 있는 짝을 제거하는 과정을 스택을 이용하여 구현합니다. 스택의 최상단에 있는 문자와 현재 문자가 같으면 짝이 되어 제거하고, 다르면 스택에 추가합니다. 문자열의 끝까지 진행한 후, 스택이 비어 있으면 모든 짝이 제거된 것이므로 1을 반환하고, 남아있으면 0을 반환합니다.

### 동작 과정
예시 입력으로 문자열 `s = "baabaa"`를 고려해보겠습니다.

1. 초기 상태: `stack = []`
2. 첫 번째 문자 'b'를 스택에 추가: `stack = ['b']`
3. 두 번째 문자 'a'를 스택에 추가: `stack = ['b', 'a']`
4. 세 번째 문자 'a'를 확인: 스택의 최상단 'a'와 같으므로 제거: `stack = ['b']`
5. 네 번째 문자 'b'를 확인: 스택의 최상단 'b'와 같으므로 제거: `stack = []`
6. 다섯 번째 문자 'a'를 스택에 추가: `stack = ['a']`
7. 여섯 번째 문자 'a'를 확인: 스택의 최상단 'a'와 같으므로 제거: `stack = []`

이 과정을 마친 후, 스택이 비어 있으므로 모든 문자가 짝지어 제거된 것이고, 결과적으로 1을 반환합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 여기서 n은 문자열의 길이입니다. 각 문자는 스택에 한 번만 추가되거나 제거되므로, 총 n번의 연산을 수행하게 됩니다.

## 코드
```python
def solution(s):
    answer = -1
    stack = [s[0]]
    
    for i in range(1, len(s)):
        if stack and stack[-1] == s[i]:
            stack.pop(-1)
        else:
            stack.append(s[i])
    
    if stack:
        return 0
    else:
        return 1

```
