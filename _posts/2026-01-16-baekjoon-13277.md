## 문제
두 정수 A와 B가 주어졌을 때, 두 수의 곱을 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 정수 A와 B가 주어진다. 두 정수는 0보다 크거나 같은 정수이며, 0을 제외한 정수는 0으로 시작하지 않으며, 수의 앞에 불필요한 0이 있는 경우도 없다. 또한, 수의 길이는 300,000자리를 넘지 않는다.

---

## 출력
두 수의 곱을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 개의 큰 정수를 곱하는 것을 요구합니다. Python에서는 기본적으로 `int` 타입이 임의 정밀도를 지원하기 때문에, 큰 수의 연산을 별도의 라이브러리 없이도 직접적으로 수행할 수 있습니다. 따라서 입력 받은 두 수를 정수형으로 변환한 후 곱셈을 수행하고 결과를 출력하면 됩니다.

### 동작 과정
1. **입력**: 예를 들어, 입력으로 `12345678901234567890 98765432109876543210`이 주어졌다고 가정해보겠습니다.
2. **변환**: `input()`을 통해 입력을 받고, `map(int, input().split())`을 사용하여 두 수를 정수형으로 변환합니다. 이 경우, A는 `12345678901234567890`, B는 `98765432109876543210`이 됩니다.
3. **곱셈 수행**: A와 B를 곱합니다. `a * b`의 결과는 `1219326311370217952237463801111263526900`입니다.
4. **출력**: 최종적으로 결과를 `print()`를 통해 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n * m)입니다. 여기서 n과 m은 두 수 A와 B의 자릿수입니다. 그러나 Python의 내부 구현에서는 큰 수의 곱셈을 매우 효율적으로 수행하므로 현실적인 경우에는 훨씬 빠르게 동작합니다.

---

## 코드
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())

print(a * b)
```
