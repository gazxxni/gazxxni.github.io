## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 주어진 수식에서 '-' 연산자를 기준으로 항을 나눈 후, 각 항을 '+' 연산자로 다시 나누어 계산하는 것입니다. 이렇게 하면 '-' 연산자는 우선 순위가 높기 때문에, 첫 번째 항에서 나머지 항들을 빼는 방식으로 최소값을 구할 수 있습니다.

### 동작 과정
1. **입력 처리**: 수식을 '-'를 기준으로 나누어 리스트 `exp`에 저장합니다.
   - 예시 입력: `55-50+40`
   - `exp`는 `['55', '50+40']`가 됩니다.

2. **각 항의 덧셈 계산**: 각 항에 대해 '+' 연산자를 기준으로 나누고, 각 요소를 정수로 변환하여 합산합니다.
   - 첫 번째 항: `55` → `num`에 `55` 추가
   - 두 번째 항: `50+40` → `50 + 40 = 90` → `num`에 `90` 추가
   - 결과적으로 `num`은 `[55, 90]`이 됩니다.

3. **최종 계산**: 첫 번째 항의 값에서 나머지 항의 합을 빼서 결과를 계산합니다.
   - 초기 값 `n = 55`
   - `n -= 90` → 최종 결과 `n = 55 - 90 = -35`

4. **결과 출력**: 최종 결과인 `-35`를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. n은 입력 수식의 길이에 비례하는데, 각 항을 한 번씩 순회하고 각 항 내의 '+' 연산을 처리하기 때문에 선형 시간에 해결할 수 있습니다.

---

## 코드
```python
import sys

exp=input().split('-')  # '-'를 기준으로 split해서 exp 리스트에 저장

num=[]   # '-'로 나눈 항들의 합을 저장할 리스트

for i in exp:
    sum=0
    tmp=i.split('+')  # 덧셈을 하기 위해서 '+'를 기준으로 split

    for j in tmp:   # split한 리스트의 각 요소들을 더해줌
        sum+=int(j)

    num.append(sum)  # 각 항의 연산 결과(덧셈)를 num에 저장


n=num[0]  # 식의 제일 처음은 숫자로 시작하기 때문에 
          # 0번째 값에서 나머지 값들을 빼준다

for i in range(1,len(num)):  # 1번째 값부터 n에서 빼준다
    n-=num[i]

print(n)


```
