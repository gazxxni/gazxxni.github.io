## 문제
당신은 N개의 조약돌을 가지고 있다. 이 조약돌을 좌표평면의 격자점 위에 아무렇게나 떨어뜨렸다. 격자점이란, x좌표와 y좌표 모두가 정수인 지점을 말한다. 이를테면 (1, 1)이나 (0, -9)는 격자점이며, (-2, 3.5)이나 (π, 7.14)는 격자점이 아니다.

모든 조약돌을 포함하는 가장 작은 직사각형을 생각할 수 있다. 예를 들어 세 개의 조약돌을 (2,4), (4, 8), (5,5)에 떨어뜨렸다면, 이 세 조약돌을 모두 포함하는 가장 작은 직사각형은 가로 3, 세로 4인 직사각형이다. 이 경우 직사각형의 둘레는 14가 된다. 직사각형의 가로와 세로 길이는 반드시 1 이상이어야 한다.

조약돌의 개수 N이 주어졌을 때, 조약돌을 좌표평면의 격자점에 적절히 떨어뜨려서 모든 조약돌을 포함하는 직사각형의 둘레를 최소로 하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 조약돌의 개수 N(1 ≤ n ≤ 500,000,000)이 주어진다.

---

## 출력
첫째 줄에 최소 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 주어진 N개의 조약돌을 포함하는 가장 작은 직사각형의 둘레를 최소화하는 것입니다. 직사각형의 가로와 세로 길이를 각각 i와 j로 설정하고, i * j ≥ N을 만족하도록 하여 둘레를 계산합니다. 이를 통해 N을 가능한 한 균형 잡힌 두 차원으로 나누어 직사각형의 둘레를 최소화합니다.

### 동작 과정
1. 입력으로 주어진 조약돌의 개수 N을 읽습니다.
2. i를 1부터 시작하여 i * i < N 조건이 만족되는 동안 i를 증가시킵니다. 이 과정에서 i는 직사각형의 가로 길이를 나타냅니다.
3. j를 N을 i로 나눈 값을 올림하여 계산합니다. j는 직사각형의 세로 길이를 나타냅니다.
4. 조약돌의 개수가 1에서 4 사이일 경우, 둘레는 항상 4로 고정됩니다.
5. 그 외의 경우, 최소 둘레는 `2 * (i + j - 2)`로 계산하여 출력합니다.

예를 들어, N=10일 경우:
- i는 4로 증가하고, j는 3이 됩니다. (4 * 3 = 12, 조건 만족)
- 둘레는 `2 * (4 + 3 - 2) = 2 * 5 = 10`으로 출력됩니다.

### 시간 복잡도
알고리즘의 시간 복잡도는 O(√N)입니다. 이는 i를 증가시키는 과정에서 i가 대략 √N까지 증가하기 때문입니다.

## 코드
```python
import sys, math
input = sys.stdin.readline

n = int(input())
i = 1

while i * i < n:
    i += 1

j = math.ceil(n / i)

if 1 <= n <= 4:
    print(4)
else:
    print(2 * (i + j - 2))
```
