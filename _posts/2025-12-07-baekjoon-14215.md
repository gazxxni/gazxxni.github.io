## 문제
영선이는 길이가 a, b, c인 세 막대를 가지고 있고, 각 막대의 길이를 마음대로 줄일 수 있다.

영선이는 세 막대를 이용해서 아래 조건을 만족하는 삼각형을 만들려고 한다.


	각 막대의 길이는 양의 정수이다
	세 막대를 이용해서 넓이가 양수인 삼각형을 만들 수 있어야 한다.
	삼각형의 둘레를 최대로 해야 한다.


a, b, c가 주어졌을 때, 만들 수 있는 가장 큰 둘레를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 100)가 주어진다.

---

## 출력
첫째 줄에 만들 수 있는 가장 큰 삼각형의 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
세 막대의 길이를 조정하여 삼각형을 만들기 위해서는 삼각형의 성질인 '두 변의 길이의 합이 나머지 한 변의 길이보다 커야 한다'는 조건을 만족해야 한다. 주어진 세 막대의 길이를 정렬한 후, 가장 긴 변을 기준으로 두 변의 합과 비교하여 조건을 만족하지 않으면 최대 둘레를 구하기 위해 가장 긴 변의 길이를 줄여야 한다.

### 동작 과정
1. 입력으로 주어진 세 막대의 길이 a, b, c를 읽고 정렬합니다. (가장 작은 것부터 가장 큰 것 순서)
   - 예: 입력이 `3 5 2`이면 정렬 후 `(2, 3, 5)`가 됩니다.
   
2. 정렬된 막대의 길이에서 가장 긴 변 c와 나머지 두 변 a, b의 합을 비교합니다. 
   - 만약 `a + b <= c`라면, 삼각형이 성립하지 않으므로, 둘레를 최대화하기 위해 `a + b`에 1을 더한 값을 곱하여 둘레를 결정합니다.
     - 예: `(2 + 3 <= 5)` → 둘레는 `(2 + 3) * 2 - 1 = 9`
   
3. 만약 a, b, c가 모두 같으면, 정삼각형을 만들 수 있으므로, 둘레는 `a * 3`입니다.
   - 예: 입력이 `4 4 4`이면 둘레는 `4 * 3 = 12`.
   
4. 두 변의 합이 가장 긴 변보다 큰 경우에는 세 막대 길이를 그대로 사용하여 삼각형을 만들 수 있으므로, 둘레는 `a + b + c`가 됩니다.
   - 예: `(2 + 3 > 5)` → 둘레는 `2 + 3 + 5 = 10`.

### 시간 복잡도
이 알고리즘은 입력된 세 개의 값을 정렬하는 과정이 포함되어 있으므로, 시간 복잡도는 O(1)입니다. (정렬할 항목이 항상 3개로 고정되어 있기 때문에 상수 시간으로 간주)

---

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
