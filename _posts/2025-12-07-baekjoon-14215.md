## 문제
영선이는 길이가 a, b, c인 세 막대를 가지고 있고, 각 막대의 길이를 마음대로 줄일 수 있다.

영선이는 세 막대를 이용해서 아래 조건을 만족하는 삼각형을 만들려고 한다.


	각 막대의 길이는 양의 정수이다
	세 막대를 이용해서 넓이가 양수인 삼각형을 만들 수 있어야 한다.
	삼각형의 둘레를 최대로 해야 한다.


a, b, c가 주어졌을 때, 만들 수 있는 가장 큰 둘레를 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 a, b, c (1 ≤ a, b, c ≤ 100)가 주어진다.

---

## 출력
첫째 줄에 만들 수 있는 가장 큰 삼각형의 둘레를 출력한다.

## 풀이

### 풀이 핵심 로직
문제의 핵심은 세 막대의 길이를 조정하여 유효한 삼각형을 만들고, 그 삼각형의 둘레를 최대화하는 것입니다. 삼각형의 성질에 따라 두 변의 길이의 합이 세 번째 변의 길이보다 커야 하며, 이를 활용하여 주어진 막대의 길이를 조정합니다.

### 동작 과정
1. 입력으로 주어진 세 막대의 길이 `a, b, c`를 읽습니다.
2. 세 막대를 정렬하여 `a <= b <= c`가 되도록 합니다.
3. 삼각형의 조건을 확인합니다:
   - 만약 `a + b <= c`라면, 삼각형을 만들 수 없으므로 가능한 최대 둘레는 `a + b`에서 1을 빼고 2를 곱한 값입니다. (즉, 두 막대의 길이를 최대한 늘리고 한 막대의 길이를 1 줄여서 둘레를 최대화)
   - 만약 `a`, `b`, `c`가 모두 동일하다면, 삼각형의 둘레는 `a * 3`입니다.
   - 그 외의 경우에는 세 막대의 길이를 그대로 사용하여 둘레를 계산합니다: `a + b + c`.

#### 예시 입력
- 입력: `2 3 5`
    1. 정렬된 길이는 `2, 3, 5`입니다.
    2. `2 + 3 = 5`, 따라서 `2 + 3 <= 5`가 성립하므로 둘레는 `(2 + 3) * 2 - 1 = 9`입니다.
    3. 출력: `9`

- 입력: `3 3 3`
    1. 정렬된 길이는 그대로 `3, 3, 3`입니다.
    2. 모든 막대가 동일하므로 둘레는 `3 * 3 = 9`입니다.
    3. 출력: `9`

### 시간 복잡도
이 알고리즘은 상수 시간 복잡도 O(1)입니다. 입력으로 주어지는 세 개의 숫자를 정렬하는 데 필요한 연산은 고정된 크기이기 때문에 입력 크기에 의존하지 않습니다.

## 코드
```python
a, b, c = map(int, input().split())

a, b, c = sorted((a, b, c))

if a + b <= c:
    print((a + b) * 2 - 1)
    
elif a == b and b == c:
    print(a * 3)
    
else:
    print(a + b + c)
```
