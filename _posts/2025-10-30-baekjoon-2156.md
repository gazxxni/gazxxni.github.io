## 풀이

### 풀이 핵심 로직
이 문제는 포도주를 마시는 방법을 최적화하여 최대의 포도주 양을 구하는 문제로, 동적 프로그래밍을 사용하여 해결합니다. 각 포도주를 마실 때 이전 포도주와의 관계를 고려하여, 연속해서 세 잔 이상 마시지 않도록 하면서 최적의 값을 구합니다.

### 동작 과정
예를 들어, 입력으로 포도주 양이 [6, 10, 13, 9, 8]인 경우를 살펴보겠습니다.

1. **초기화**: `dp` 배열을 선언하여 각 인덱스에 해당하는 포도주를 마셨을 때의 최대 양을 저장합니다.
   - `dp[0] = 6` (첫 번째 포도주)
   - `dp[1] = 16` (첫 번째와 두 번째 포도주)
   - `dp[2] = max(13 + 6, 13 + 10, 16) = 19` (첫 번째, 두 번째와 세 번째 포도주를 고려)

2. **이후 단계**: 포도주 잔이 3개 이상일 경우, `dp[i]`는 다음과 같이 계산됩니다:
   - `dp[i] = max(arr[i] + dp[i - 2], arr[i] + arr[i - 1] + dp[i - 3], dp[i - 1])`
   - 이 식은 현재 포도주를 마시는 경우와 마시지 않는 경우를 비교하여 최대 값을 저장합니다.

3. **최종 출력**: `dp[n-1]`에서 최대 포도주 양을 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n)입니다. 각각의 포도주에 대해 상수 시간 내에 최대 포도주 양을 계산하므로, 전체적으로는 n개의 포도주에 대해 선형적으로 진행됩니다.

## 코드
```python
n = int(input())
arr = []
for _ in range(n):
    arr.append(int(input()))

if n == 0:
    print(0)
elif n == 1:
    print(arr[0])
elif n == 2:
    print(arr[0] + arr[1])
else:
    dp = [0] * n
    
    dp[0] = arr[0]
    dp[1] = arr[0] + arr[1]
    dp[2] = max(arr[2] + arr[0], arr[2] + arr[1], dp[1])
    
    for i in range(3, n):
        dp[i] = max(arr[i] + dp[i - 2], arr[i] + arr[i - 1] + dp[i - 3], dp[i - 1])
        
    print(dp[n-1])
```
