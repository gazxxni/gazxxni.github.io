## 문제
정수를 저장하는 덱을 구현한 다음, 입력으로 주어지는 명령을 처리하는 프로그램을 작성하시오.

명령은 총 여덟 가지이다.


	1 X: 정수 X를 덱의 앞에 넣는다. (1 ≤ X ≤ 100,000)
	2 X: 정수 X를 덱의 뒤에 넣는다. (1 ≤ X ≤ 100,000)
	3: 덱에 정수가 있다면 맨 앞의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	4: 덱에 정수가 있다면 맨 뒤의 정수를 빼고 출력한다. 없다면 -1을 대신 출력한다.
	5: 덱에 들어있는 정수의 개수를 출력한다.
	6: 덱이 비어있으면 1, 아니면 0을 출력한다.
	7: 덱에 정수가 있다면 맨 앞의 정수를 출력한다. 없다면 -1을 대신 출력한다.
	8: 덱에 정수가 있다면 맨 뒤의 정수를 출력한다. 없다면 -1을 대신 출력한다.

---

## 입력
첫째 줄에 명령의 수 N이 주어진다. (1 ≤ N ≤ 1,000,000)

둘째 줄부터 N개 줄에 명령이 하나씩 주어진다.

출력을 요구하는 명령은 하나 이상 주어진다.

---

## 출력
출력을 요구하는 명령이 주어질 때마다 명령의 결과를 한 줄에 하나씩 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제에서는 덱(deque) 자료구조를 이용하여 다양한 명령을 처리합니다. 각 명령에 따라 덱의 앞이나 뒤에 정수를 추가하거나 삭제하고, 덱의 상태를 확인하는 등의 작업을 수행합니다. 파이썬의 `collections.deque`를 사용하면 이러한 작업이 효율적으로 처리됩니다.

### 동작 과정
1. 입력으로 주어진 명령의 수 `N`을 읽고, 빈 덱을 생성합니다.
2. 각 명령을 읽어와서, 명령의 타입에 따라 적절한 덱 연산을 수행합니다:
   - `1 X`: 정수 `X`를 덱의 앞에 추가합니다.
   - `2 X`: 정수 `X`를 덱의 뒤에 추가합니다.
   - `3`: 덱의 앞에서 정수를 제거하고 출력합니다. 덱이 비어있으면 `-1`을 출력합니다.
   - `4`: 덱의 뒤에서 정수를 제거하고 출력합니다. 덱이 비어있으면 `-1`을 출력합니다.
   - `5`: 덱의 크기를 출력합니다.
   - `6`: 덱이 비어있으면 `1`, 아니면 `0`을 출력합니다.
   - `7`: 덱의 앞에 있는 정수를 출력합니다. 덱이 비어있으면 `-1`을 출력합니다.
   - `8`: 덱의 뒤에 있는 정수를 출력합니다. 덱이 비어있으면 `-1`을 출력합니다.

예시 입력:
```
8
1 10
2 20
3
4
5
6
7
8
```
동작 과정:
- `1 10`: 덱에 10을 추가 → 덱 상태: [10]
- `2 20`: 덱에 20을 추가 → 덱 상태: [10, 20]
- `3`: 10을 제거하고 출력 → 출력: 10, 덱 상태: [20]
- `4`: 20을 제거하고 출력 → 출력: 20, 덱 상태: []
- `5`: 덱의 크기 출력 → 출력: 0
- `6`: 덱 비어있음 확인 → 출력: 1
- `7`: 덱의 앞 정수 출력 → 출력: -1
- `8`: 덱의 뒤 정수 출력 → 출력: -1

### 시간 복잡도
각 명령에 대해 덱의 연산은 평균적으로 O(1)의 시간 복잡도를 가집니다. 따라서 전체 명령 처리의 시간 복잡도는 O(N)입니다. 여기서 N은 입력으로 주어진 명령의 수입니다.

## 코드
```python
import sys
input = sys.stdin.readline
from collections import deque

n = int(input())
q = deque()

for _ in range(n):
    command = input().split()
    
    a = int(command[0])
    
    if a == 1:
        q.appendleft(command[1])
        
    elif a == 2:
        q.append(command[1])
        
    elif a == 3:
        if q:
            print(q.popleft())
        else:
            print(-1)
            
    elif a == 4:
        if q:
            print(q.pop())
        else:
            print(-1)
            
    elif a == 5:
        print(len(q))
        
    elif a == 6:
        if q:
            print(0)
        else:
            print(1)
            
    elif a == 7:
        if q:
            print(q[0])
        else:
            print(-1)
            
    elif a == 8:
        if q:
            print(q[-1])
        else:
            print(-1)
```
