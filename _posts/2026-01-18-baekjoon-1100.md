## 문제
체스판은 8×8크기이고, 검정 칸과 하얀 칸이 번갈아가면서 색칠되어 있다. 가장 왼쪽 위칸 (0,0)은 하얀색이다. 체스판의 상태가 주어졌을 때, 하얀 칸 위에 말이 몇 개 있는지 출력하는 프로그램을 작성하시오.

---

## 입력
첫째 줄부터 8개의 줄에 체스판의 상태가 주어진다. ‘.’은 빈 칸이고, ‘F’는 위에 말이 있는 칸이다.

---

## 출력
첫째 줄에 문제의 정답을 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

## 풀이

### 풀이 핵심 로직
체스판의 각 칸은 색깔이 번갈아가며 칠해져 있으며, 가장 왼쪽 위에 있는 칸(0,0)은 하얀색입니다. 따라서, 하얀색 칸의 좌표는 `(i, j)`에서 `i + j`가 짝수인 경우입니다. 주어진 체스판 상태에서 하얀색 칸에 말('F')이 있는지 확인하여 그 개수를 세는 방식으로 문제를 해결합니다.

### 동작 과정
1. 입력으로 8줄의 체스판 상태를 받습니다. 각 줄은 8개의 문자로 이루어져 있으며, '.'는 빈 칸, 'F'는 말이 있는 칸을 의미합니다.
2. 이중 반복문을 사용하여 각 칸의 좌표 `(i, j)`를 순회합니다.
3. `(i + j) % 2 == 0` 인 경우(즉, 하얀색 칸)에서 해당 칸의 값이 'F'이면 카운트를 증가시킵니다.
4. 최종적으로 카운트된 값이 하얀 칸 위에 있는 말의 개수가 됩니다.

예시 입력:
```
........
.F...... 
........
..F.....
........
....F...
........
........
```
- 첫 번째 줄: 모든 칸이 비어있음
- 두 번째 줄: (1,1)에 말이 있음 → 카운트 증가
- 세 번째 줄: 모든 칸이 비어있음
- 네 번째 줄: (3,2)에 말이 있음 → 카운트 증가
- 다섯 번째 줄: (5,4)에 말이 있음 → 카운트 증가
- 나머지 줄은 모두 비어 있음

결과적으로 하얀 칸 위에 있는 말의 총 개수는 3입니다.

### 시간 복잡도
O(1) - 체스판의 크기가 고정되어 있어 (8x8), 반복문의 최대 반복 횟수는 64회로 일정합니다.

## 코드
```python
graph = [input().rstrip() for _ in range(8)]

cnt = 0
for i in range(8):
    for j in range(8):
        if (i % 2 == 0 and j % 2 == 0) or (i % 2 == 1 and j % 2 == 1):
            if graph[i][j] == 'F':
                cnt += 1
                
print(cnt)
```
