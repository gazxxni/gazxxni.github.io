## 문제
한자리 숫자가 적힌 종이 조각이 흩어져있습니다. 흩어진 종이 조각을 붙여 소수를 몇 개 만들 수 있는지 알아내려 합니다.

각 종이 조각에 적힌 숫자가 적힌 문자열 numbers가 주어졌을 때, 종이 조각으로 만들 수 있는 소수가 몇 개인지 return 하도록 solution 함수를 완성해주세요.

제한사항


numbers는 길이 1 이상 7 이하인 문자열입니다.
numbers는 0~9까지 숫자만으로 이루어져 있습니다.
"013"은 0, 1, 3 숫자가 적힌 종이 조각이 흩어져있다는 의미입니다.


입출력 예

        
numbers
return


        
"17"
3


"011"
2


      
입출력 예 설명

예제 #1
[1, 7]으로는 소수 [7, 17, 71]를 만들 수 있습니다.

예제 #2
[0, 1, 1]으로는 소수 [11, 101]를 만들 수 있습니다.


11과 011은 같은 숫자로 취급합니다.

## 풀이
### 풀이 핵심 로직
이 문제는 주어진 숫자 조각으로 만들 수 있는 모든 조합을 통해 소수를 찾는 문제입니다. 먼저, 가능한 모든 조합의 수를 생성한 다음, 각 조합이 소수인지 확인합니다. 소수는 2 이상의 자연수 중에서 1과 자기 자신만으로 나누어 떨어지는 수입니다.

### 동작 과정
1. **입력받은 숫자 조각을 이용해 모든 조합 생성**: 주어진 문자열에서 가능한 모든 길이의 조합을 만든다. 예를 들어, "17"의 경우 1, 7, 17, 71을 생성할 수 있다.
  
2. **중복 제거**: 소수의 중복을 피하기 위해 생성된 숫자를 `set`에 저장한다.

3. **소수 판별**: 소수 판별을 위한 함수를 정의하고, 최대 숫자까지 소수를 미리 계산하여 리스트로 저장한다. 이를 통해 각 조합이 소수인지 빠르게 확인할 수 있다.

4. **결과 반환**: 소수인 숫자의 개수를 반환한다.

#### 예시 입력에 대한 단계별 설명
- 입력: "17"
  - 조합 생성: 1, 7, 17, 71
  - 소수 판별: 7, 17, 71이 소수이므로 총 3개의 소수가 존재한다.

- 입력: "011"
  - 조합 생성: 0, 1, 11, 10, 1, 0 (중복 제거 후: 0, 1, 11, 10)
  - 소수 판별: 11이 소수이므로 총 1개의 소수가 존재한다. (0은 소수가 아니고, 1도 소수가 아님)

### 시간 복잡도
- 조합 생성: `O(n!)` (n은 숫자 조각의 길이) 
- 소수 판별: 미리 계산한 소수 리스트를 사용하므로 `O(1)`로 확인 가능.
- 전체 시간 복잡도: 대략 `O(n! + m)` (m은 최대 숫자의 크기, 소수 계산 시), 하지만 n이 최대 7이므로 실제로는 효율적이다.

### 코드
```python
from itertools import permutations

def solution(numbers):    
    def prime(x):
        is_prime = [True] * (x + 1)
        is_prime[0] = is_prime[1] = False
        
        for i in range(2, int(x ** 0.5) + 1):
            if is_prime[i]:
                for j in range(i*i, x + 1, i):
                    is_prime[j] = False
        
        return [i for i in range(2, x + 1) if is_prime[i]]            
                    
    # 숫자 조각을 리스트로 변환
    new_num = list(numbers)
    
    # 최대 숫자 계산
    max_num = int(''.join(sorted(new_num, reverse=True)))
    
    # 소수 리스트 생성
    prime_num = prime(max_num)
    prime_set = set(prime_num)
    
    # 모든 조합 생성
    arr = set()
    for i in range(1, len(new_num) + 1):
        for p in permutations(new_num, i):
            arr.add(int(''.join(p)))
    
    # 소수 개수 세기
    cnt = sum(1 for i in arr if i in prime_set)
    
    return cnt
```

위 코드는 문제를 해결하는 데 효율적이며, 모든 조합을 고려하고 소수를 올바르게 판별하여 결과를 반환합니다.

## 코드
```python
from itertools import permutations

def solution(numbers):    
    def prime(x):
        is_prime = [True] * (x + 1)
        is_prime[0] = is_prime[1] = False
        
        for i in range(2, int(x ** 0.5) + 1):
            if is_prime[i]:
                for j in range(i*i, x + 1, i):
                    is_prime[j] = False
        
        return [i for i in range(2, x + 1) if is_prime[i]]            
                    
        
    new_num = list(map(str, numbers))
    new_num.sort(reverse=True)
    
    max_num = ''.join(new_num)
    
    prime_num = prime(int(max_num))
    
    per = [p for i in range(1, len(new_num) + 1) for p in permutations(new_num, i)]
    
    arr = set()
    for i in per:
        arr.add(int(''.join(i)))
    
    cnt = 0
    for i in list(arr):
        if i in prime_num:
            cnt += 1
            
    return cnt
```
