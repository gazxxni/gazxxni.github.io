## 문제
당신은 N개의 테스트케이스들에게 반드시 인사를 해야 이 문제를 풀 수 있다.

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

---

## 입력
N이 주어진다. (1 ≤ N ≤ 200)

---

## 출력
한 줄에 하나의 Hello World, Judge i! 를 출력한다.

---

## 성능 요약
메모리: 108384 KB, 시간: 84 ms

---

## 풀이

### 풀이 핵심 로직
주어진 숫자 N에 대해 1부터 N까지의 각 숫자를 사용하여 "Hello World, Judge i!" 형식의 문자열을 출력하는 문제이다. 반복문을 사용하여 각 줄의 번호에 맞는 인사를 출력하면 된다.

### 동작 과정
1. 사용자로부터 N을 입력받는다. (예: 3)
2. 반복문을 통해 1부터 N까지의 숫자를 순회하며, 각 숫자 i에 대해 "Hello World, Judge i!"를 출력한다.
   - i=1: "Hello World, Judge 1!"
   - i=2: "Hello World, Judge 2!"
   - i=3: "Hello World, Judge 3!"

입력 예시:
```
3
```
출력 결과:
```
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

### 시간 복잡도
O(N) - N의 범위에 따라 반복문이 N번 실행되므로, 시간 복잡도는 선형이다.

---

## 코드
```python
n = int(input())

for i in range(1, n + 1):
    print(f'Hello World, Judge {i}!')
```
