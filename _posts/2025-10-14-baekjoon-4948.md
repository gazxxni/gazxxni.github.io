## 문제
베르트랑 공준은 임의의 자연수 n에 대하여, n보다 크고, 2n보다 작거나 같은 소수는 적어도 하나 존재한다는 내용을 담고 있다.

이 명제는 조제프 베르트랑이 1845년에 추측했고, 파프누티 체비쇼프가 1850년에 증명했다.

예를 들어, 10보다 크고, 20보다 작거나 같은 소수는 4개가 있다. (11, 13, 17, 19) 또, 14보다 크고, 28보다 작거나 같은 소수는 3개가 있다. (17,19, 23)

자연수 n이 주어졌을 때, n보다 크고, 2n보다 작거나 같은 소수의 개수를 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 케이스는 n을 포함하는 한 줄로 이루어져 있다.

입력의 마지막에는 0이 주어진다.

---

## 출력
각 테스트 케이스에 대해서, n보다 크고, 2n보다 작거나 같은 소수의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
문제는 주어진 자연수 \( n \)에 대해 \( n \)보다 크고 \( 2n \)보다 작거나 같은 소수를 찾는 것입니다. 이를 위해 에라토스테네스의 체를 사용하여 최대 \( 2n \)까지의 소수를 미리 계산해 놓고, 각 테스트 케이스에 대해 소수의 개수를 세는 방식으로 문제를 해결합니다.

### 동작 과정
1. **입력**: 여러 개의 테스트 케이스가 주어지며, 각 케이스는 한 줄에 자연수 \( n \)이 입력됩니다. 입력의 마지막에는 0이 주어져 종료 조건을 나타냅니다.
2. **소수 계산**: 최대 \( n = 123456 \)를 고려하여 \( 2n \)까지의 소수를 미리 구합니다. 이 예제에서는 \( n \)이 최대 300,000까지 이르므로, \( 2n \)은 600,000이 됩니다. 에라토스테네스의 체를 통해 소수 목록을 생성합니다.
3. **소수 개수 세기**: 각 테스트 케이스에 대해 \( n+1 \)부터 \( 2n \)까지의 수 중 소수인 개수를 카운트합니다.
4. **출력**: 각 케이스 별로 소수의 개수를 출력합니다.

**예시 입력**:
```
10
13
16
0
```
**단계별 설명**:
- \( n = 10 \)일 때, \( n+1 \)에서 \( 2n \)까지(11부터 20) 소수를 셉니다. 소수는 11, 13, 17, 19 → 총 4개.
- \( n = 13 \)일 때, 14부터 26까지의 소수는 17, 19, 23 → 총 3개.
- \( n = 16 \)일 때, 17부터 32까지의 소수는 17, 19, 23, 29, 31 → 총 5개.
- 마지막 입력 0은 종료를 의미합니다.

출력:
```
4
3
5
```

### 시간 복잡도
에라토스테네스의 체를 사용하여 소수를 찾는 데 걸리는 시간 복잡도는 \( O(n \log \log n) \)입니다. 각 테스트 케이스에 대해 소수 개수를 계산하는 과정은 \( O(n) \)이므로 전체 시간 복잡도는 \( O(n \log \log n) \)입니다.

## 코드
```python
def prime_numbers(n):
    arr = [True] * (n + 1) 
    arr[0] = arr[1] = False

    for i in range(2, int(n ** 0.5) + 1):
        if arr[i]:
            for j in range(i * i, n + 1, i):
                arr[j] = False
    
    return arr

a = prime_numbers(300000)

while True:
    n = int(input())
    
    if n == 0:
        exit()
        
    cnt = 0
    for i in range(n + 1, 2 * n + 1):
        if a[i] == True:
            cnt += 1
    
    print(cnt)
```
