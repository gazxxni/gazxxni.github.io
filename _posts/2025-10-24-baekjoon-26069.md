## 풀이

### 풀이 핵심 로직
이 문제의 핵심 아이디어는 'ChongChong'이라는 이름이 포함된 쌍을 찾고, 이를 통해 서로 연결된 이름들을 추적하는 것입니다. 'ChongChong'과 직접 연결된 이름뿐만 아니라, 연결된 이름들이 또 다른 이름과 연결될 수 있음을 고려하여 최종적으로 연결된 이름의 개수를 세는 방식입니다.

### 동작 과정
1. 입력받은 정수 `n`은 쌍의 개수를 나타냅니다.
2. `arr`이라는 집합을 생성하여 연결된 이름들을 저장합니다.
3. `flag` 변수를 사용하여 'ChongChong'과 연결된 이름이 발견되었는지를 추적합니다.
4. `for` 루프를 통해 각 쌍을 입력받고, 이를 공백으로 나누어 변수 `b`와 `c`에 저장합니다.
5. 만약 `b` 또는 `c`가 'ChongChong'이라면, `flag`를 `True`로 설정하고 두 이름을 `arr`에 추가합니다.
6. 이후, 이미 `flag`가 `True`인 상태에서는 `b` 또는 `c`가 `arr`에 있는지 확인하고, 있다면 두 이름을 다시 `arr`에 추가하여 연결된 이름들을 갱신합니다.
7. 모든 입력을 처리한 후, `arr`의 길이를 출력하여 연결된 이름의 총 개수를 반환합니다.

예시 입력:
```
4
ChongChong F
F G
G H
H ChongChong
```
- 첫 번째 쌍에서 'ChongChong'이 발견되어 `arr`에 'ChongChong'과 'F'가 추가됩니다.
- 두 번째 쌍에서 'F'가 `arr`에 있으므로 'G'가 추가됩니다.
- 세 번째 쌍에서 'G'가 `arr`에 있으므로 'H'가 추가됩니다.
- 네 번째 쌍에서 'H'가 `arr`에 있으므로 'ChongChong'이 이미 있으므로 추가되지 않습니다.
- 최종적으로 `arr`에는 'ChongChong', 'F', 'G', 'H'가 포함되어 총 4개의 이름이 연결됩니다.

### 시간 복잡도
입력받는 쌍의 수 `n`에 비례하여 O(n)입니다. 각 쌍에 대해 상수 시간에 집합에 이름을 추가하고 확인하므로, 전체적으로 선형 시간이 소요됩니다.

---

## 코드
```python
n = int(input())

arr = set()
flag = False
for _ in range(n):
    a = input().rstrip().split()
    b = a[0]
    c = a[1]
    if b == 'ChongChong' or c == 'ChongChong':
        flag = True
        arr.add(b)
        arr.add(c)
    
    if flag:
        if b in arr or c in arr:
            arr.add(b)
            arr.add(c)
    
print(len(arr))
```
