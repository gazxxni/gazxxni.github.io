## 문제
Leo는 카펫을 사러 갔다가 아래 그림과 같이 중앙에는 노란색으로 칠해져 있고 테두리 1줄은 갈색으로 칠해져 있는 격자 모양 카펫을 봤습니다.



Leo는 집으로 돌아와서 아까 본 카펫의 노란색과 갈색으로 색칠된 격자의 개수는 기억했지만, 전체 카펫의 크기는 기억하지 못했습니다.

Leo가 본 카펫에서 갈색 격자의 수 brown, 노란색 격자의 수 yellow가 매개변수로 주어질 때 카펫의 가로, 세로 크기를 순서대로 배열에 담아 return 하도록 solution 함수를 작성해주세요.

## 풀이
### 풀이 핵심 로직
문제의 핵심은 주어진 갈색 격자 수(brown)와 노란색 격자 수(yellow)를 통해 카펫의 가로와 세로 길이를 계산하는 것입니다. 카펫의 총 면적은 갈색과 노란색 격자의 합으로 구할 수 있으며, 카펫의 구조상 갈색 격자는 카펫의 테두리를 구성하므로, 가로와 세로의 관계를 이용해 갈색 격자의 수가 맞는지 체크해야 합니다.

### 동작 과정
1. **총 면적 계산**: 먼저 카펫의 전체 면적을 `total = brown + yellow`로 계산합니다.
2. **약수 찾기**: 전체 면적의 약수를 찾아서 카펫의 가로와 세로 길이를 후보로 설정합니다. 가로 길이는 세로 길이보다 크거나 같아야 합니다.
3. **조건 검사**: 각 후보 쌍에 대해 `(가로 + 세로 - 2) * 2`가 갈색 격자 수와 같은지 확인합니다. 이 식은 카펫의 테두리에 해당하는 갈색 격자의 수를 계산합니다.
4. **결과 반환**: 조건을 만족하는 경우가 있으면 해당 가로와 세로 길이를 반환합니다.

예시로, `brown = 10`, `yellow = 2`일 경우:
- `total = 10 + 2 = 12`
- `12`의 약수는 `(3, 4)`, `(4, 3)`이므로 이 두 쌍을 체크합니다.
- `(4 + 3 - 2) * 2 = 10`으로 갈색 격자 수에 맞으므로 `[4, 3]`을 결과로 반환합니다.

### 시간 복잡도
약수를 찾는 과정에서 최대 `O(√n)`의 시간 복잡도를 가지므로, 전체 시간 복잡도는 `O(√(brown + yellow))`입니다.

## 코드
```python
import math

def solution(brown, yellow):
    ans = []
    total = brown + yellow
    
    def div(n):
        di = []
        
        for i in range(3, int(n ** 0.5) + 1):
            if n % i == 0:
                di.append((i, n // i))

        return di

    arr = div(total)
    for a, b in arr:
        if (a + b - 2) * 2 == brown:
            if a >= b:
                ans.append(a)
                ans.append(b)
            else:
                ans.append(b)
                ans.append(a)
                
    return ans
```
