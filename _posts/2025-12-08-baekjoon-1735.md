## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
두 분수를 더하기 위해서는 공통 분모를 설정한 후, 분자를 계산하여 기약분수 형태로 만들기 위해 최대공약수(GCD)를 이용해 약분합니다. GCD를 이용하여 최종 분자와 분모를 구한 후, 기약분수 형태로 출력합니다.

### 동작 과정
1. 입력을 통해 첫 번째 분수 \( A/B \)와 두 번째 분수 \( C/D \)의 분자와 분모를 읽어옵니다.
   - 예시 입력: 
     ```
     1 2
     1 3
     ```
   - 여기서 \( A = 1 \), \( B = 2 \), \( C = 1 \), \( D = 3 \)

2. 두 분수의 공통 분모를 구합니다. 공통 분모는 두 분모의 최소공배수(LCM)로 설정합니다.
   - \( LCM(B, D) = LCM(2, 3) = 6 \)

3. 각 분수를 공통 분모에 맞추어 분자를 변환합니다.
   - \( \text{first\_numerator} = A \times \frac{n}{B} = 1 \times \frac{6}{2} = 3 \)
   - \( \text{second\_numerator} = C \times \frac{n}{D} = 1 \times \frac{6}{3} = 2 \)

4. 두 분자의 합을 계산합니다.
   - \( \text{sum\_numerator} = 3 + 2 = 5 \)

5. 최종 분모는 공통 분모 \( n \)으로 설정하며, 기약분수를 위해 \( sum\_numerator \)와 \( n \)의 GCD를 구합니다.
   - \( GCD(5, 6) = 1 \)

6. 기약분수 형태로 출력합니다.
   - \( \frac{5/1}{6/1} = \frac{5}{6} \)
   - 출력: `5 6`

### 시간 복잡도
이 알고리즘의 시간 복잡도는 \( O(\log(\min(B, D))) \)입니다. 이는 GCD를 구하는 유클리드 호제법의 시간 복잡도입니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
