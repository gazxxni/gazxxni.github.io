## 문제
분수 A/B는 분자가 A, 분모가 B인 분수를 의미한다. A와 B는 모두 자연수라고 하자.
두 분수의 합 또한 분수로 표현할 수 있다. 두 분수가 주어졌을 때, 그 합을 기약분수의 형태로 구하는 프로그램을 작성하시오. 기약분수란 더 이상 약분되지 않는 분수를 의미한다.

---

## 입력
첫째 줄과 둘째 줄에, 각 분수의 분자와 분모를 뜻하는 두 개의 자연수가 순서대로 주어진다. 입력되는 네 자연수는 모두 30,000 이하이다.

---

## 출력
첫째 줄에 구하고자 하는 기약분수의 분자와 분모를 뜻하는 두 개의 자연수를 빈 칸을 사이에 두고 순서대로 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 두 분수의 합을 구하기 위해, 먼저 두 분수의 공통 분모를 찾고, 이를 통해 분수를 통합한 후 기약분수로 만들어 출력합니다. 공통 분모는 최소공배수(LCM)를 이용하여 구하며, 최종적으로는 유클리드 호제법을 사용하여 분자와 분모의 최대공약수(GCD)를 찾아 약분합니다.

### 동작 과정
예를 들어, 입력으로 다음과 같은 두 분수가 주어진다고 가정해 보겠습니다:
```
1 2
1 3
```
1. 첫 번째 분수는 1/2이고, 두 번째 분수는 1/3입니다.
2. 두 분수의 분모(2, 3)의 최소공배수를 계산합니다. LCM(2, 3) = 6입니다.
3. 각 분수를 공통 분모로 변환합니다:
   - 1/2 → (1 * 3) / (2 * 3) = 3/6
   - 1/3 → (1 * 2) / (3 * 2) = 2/6
4. 두 분수를 합칩니다:
   - 3/6 + 2/6 = (3 + 2) / 6 = 5/6
5. 기약분수로 만들기 위해 분자(5)와 분모(6)의 GCD를 계산합니다. GCD(5, 6) = 1입니다.
6. 약분 후 최종 결과는 5/6으로, 출력은 `5 6`입니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(log(min(b, d)))입니다. 이는 GCD를 계산하는 유클리드 호제법의 복잡도에 기반합니다. LCM 계산의 경우, GCD를 한번 더 구해야 하므로 전체적으로는 GCD 계산의 시간 복잡도가 지배적입니다.

## 코드
```python
a, b = map(int, input().split())
c, d = map(int, input().split())

def gcd(x, y):
    while y > 0:
        x, y = y, x % y
    return x

def lcm(x, y):
    return x * y / gcd(x, y)

n = lcm(b, d)
aa = n // b
bb = n // d

cc = aa * a + bb * c
dd = gcd(cc, n)
print(int(cc // dd), int(n // dd))
```
