## 문제
상근이는 세계적인 소프트웨어 회사 기글에서 일한다. 이 회사의 가장 큰 특징은 자유로운 출퇴근 시간이다. 따라서, 직원들은 반드시 9시부터 6시까지 회사에 있지 않아도 된다.

각 직원은 자기가 원할 때 출근할 수 있고, 아무때나 퇴근할 수 있다.

상근이는 모든 사람의 출입카드 시스템의 로그를 가지고 있다. 이 로그는 어떤 사람이 회사에 들어왔는지, 나갔는지가 기록되어져 있다. 로그가 주어졌을 때, 현재 회사에 있는 모든 사람을 구하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 로그에 기록된 출입 기록의 수 n이 주어진다. (2 ≤ n ≤ 106) 다음 n개의 줄에는 출입 기록이 순서대로 주어지며, 각 사람의 이름이 주어지고 "enter"나 "leave"가 주어진다. "enter"인 경우는 출근, "leave"인 경우는 퇴근이다.

회사에는 동명이인이 없으며, 대소문자가 다른 경우에는 다른 이름이다. 사람들의 이름은 알파벳 대소문자로 구성된 5글자 이하의 문자열이다.

---

## 출력
현재 회사에 있는 사람의 이름을 사전 순의 역순으로 한 줄에 한 명씩 출력한다.

## 풀이

### 풀이 핵심 로직
주어진 출입 기록을 기반으로 현재 회사에 있는 사람들을 추적하기 위해, 각 사람의 출입 상태를 저장하는 해시 맵(딕셔너리)을 사용합니다. 'enter' 로그가 기록될 때 해당 이름을 해시 맵에 추가하고, 'leave' 로그가 기록되면 해당 이름을 삭제합니다. 마지막으로 현재 회사에 있는 사람들의 이름을 사전 순의 역순으로 정렬하여 출력합니다.

### 동작 과정
1. 입력으로 출입 기록의 수 `n`을 읽습니다. 예를 들어 `n = 5`라고 가정합니다.
2. 다음으로 5개의 출입 기록이 입력됩니다:
   ```
   Alice enter
   Bob enter
   Alice leave
   Charlie enter
   Bob leave
   ```
3. 출입 기록에 따라 해시 맵을 업데이트합니다:
   - `Alice`가 들어오므로 맵에 `{'Alice': 1}` 추가
   - `Bob`이 들어오므로 맵에 `{'Alice': 1, 'Bob': 1}` 추가
   - `Alice`가 나가므로 맵에서 `Alice`를 삭제
   - `Charlie`가 들어오므로 맵에 `{'Bob': 1, 'Charlie': 1}` 추가
   - `Bob`이 나가므로 맵에서 `Bob`을 삭제
4. 현재 회사에 남아 있는 사람들은 `{'Charlie': 1}`로, 즉 `Charlie`만 남습니다.
5. 이 이름들을 사전 순의 역순으로 정렬하면 `['Charlie']`가 됩니다.
6. 최종적으로 `Charlie`를 출력합니다.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(n log n)입니다. n은 출입 기록의 수이며, 정렬 과정에서 O(n log n)의 시간이 소요되기 때문입니다. 해시 맵에 대한 삽입 및 삭제는 평균적으로 O(1) 시간이 걸리므로, 전체적인 과정에서 가장 시간이 많이 소요되는 부분은 정렬입니다.

## 코드
```python
from collections import defaultdict

n = int(input())
dic = defaultdict(int)

for _ in range(n):
    a, b = map(str, input().rstrip().split())
    
    if b == 'enter':
        dic[a] += 1
        
    else:
        dic[a] -= 1
        
ans = []    
for k, v in dic.items():
    if v == 1:
        ans.append(k)
        
ans.sort(reverse=True)
for i in ans:
    print(i)
```
