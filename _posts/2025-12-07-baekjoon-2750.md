## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 110760 KB, 시간: 100 ms

---

## 풀이

### 풀이 핵심 로직
주어진 N개의 정수를 오름차순으로 정렬하기 위해 리스트에 입력된 수들을 저장한 후, `sort()` 메서드를 이용하여 정렬합니다. 정렬된 결과는 리스트를 순회하며 차례대로 출력합니다.

### 동작 과정
1. 입력을 통해 정수의 개수 N을 읽습니다.
2. N개의 정수를 차례대로 입력받아 리스트 `arr`에 저장합니다.
   - 예시 입력:
     ```
     5
     3
     1
     4
     2
     5
     ```
   - 이때 `arr` 리스트는 `[3, 1, 4, 2, 5]`가 됩니다.
3. `arr.sort()` 메서드를 호출하여 리스트를 오름차순으로 정렬합니다.
   - 정렬 후 `arr` 리스트는 `[1, 2, 3, 4, 5]`가 됩니다.
4. 정렬된 리스트를 순회하며 각 요소를 한 줄씩 출력합니다.
   - 출력 결과:
     ```
     1
     2
     3
     4
     5
     ```

### 시간 복잡도
정렬 알고리즘의 시간 복잡도는 일반적으로 O(N log N)입니다. 여기서 N은 입력된 수의 개수입니다. 따라서 이 프로그램의 시간 복잡도는 O(N log N)입니다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
