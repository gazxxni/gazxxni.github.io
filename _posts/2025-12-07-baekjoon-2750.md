## 문제
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

---

## 입력
첫째 줄에 수의 개수 N(1 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 수가 주어진다. 이 수는 절댓값이 1,000보다 작거나 같은 정수이다. 수는 중복되지 않는다.

---

## 출력
첫째 줄부터 N개의 줄에 오름차순으로 정렬한 결과를 한 줄에 하나씩 출력한다.

---

## 성능 요약
메모리: 110760 KB, 시간: 100 ms

## 풀이

### 풀이 핵심 로직
주어진 N개의 정수를 오름차순으로 정렬하기 위해, Python의 내장 정렬 함수인 `sort()`를 사용한다. `sort()` 함수는 효율적인 정렬 알고리즘을 기반으로 하며, 이를 통해 손쉽게 정렬된 리스트를 얻을 수 있다. 정렬된 리스트는 이후에 각 요소를 출력하여 요구된 형식에 맞춘다.

### 동작 과정
1. 첫 번째 줄에서 정수 N을 입력받는다. 예를 들어, N이 5라고 가정하자.
2. 다음 줄부터 5개의 정수를 입력받는다. 예를 들어, 입력된 수가 3, 1, 4, 1, 5라고 하자.
3. 이 수들을 리스트에 저장한 후, `sort()` 메서드를 호출하여 리스트를 오름차순으로 정렬한다. 정렬된 리스트는 [1, 3, 4, 5]가 된다.
4. 정렬된 리스트의 각 요소를 한 줄에 하나씩 출력한다. 출력 결과는 다음과 같다:
   ```
   1
   3
   4
   5
   ```

### 시간 복잡도
정렬 알고리즘의 평균 및 최악의 시간 복잡도는 O(N log N)이다. 여기서 N은 입력된 수의 개수이다.

---

## 코드
```python
n = int(input())
arr = [int(input()) for _ in range(n)]

arr.sort()
for i in arr:
    print(i)
```
