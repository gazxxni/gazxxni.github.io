## 문제
2차원 좌표 평면 위의 두 선분 L1, L2가 주어졌을 때, 두 선분이 교차하는지 아닌지 구해보자. 한 선분의 끝 점이 다른 선분이나 끝 점 위에 있는 것도 교차하는 것이다.

L1의 양 끝 점은 (x1, y1), (x2, y2), L2의 양 끝 점은 (x3, y3), (x4, y4)이다.

---

## 입력
첫째 줄에 L1의 양 끝 점 x1, y1, x2, y2가, 둘째 줄에 L2의 양 끝 점 x3, y3, x4, y4가 주어진다.

---

## 출력
L1과 L2가 교차하면 1, 아니면 0을 출력한다.

## 풀이

### 풀이 핵심 로직
이 문제는 두 선분이 교차하는지를 판단하는 기하학적 문제로, 선분의 방향성을 이용하여 교차 여부를 확인합니다. 두 선분의 끝 점들을 이용하여 각 선분이 상대 선분의 끝 점에 대해 어떤 방향에 있는지를 계산하고, 이를 통해 교차 여부를 결정합니다.

### 동작 과정
1. **입력 받기**: 두 선분 L1과 L2의 끝 점 좌표를 입력받습니다. L1의 끝 점은 (x1, y1), (x2, y2)이고, L2의 끝 점은 (x3, y3), (x4, y4)입니다.
   
2. **CCW 계산**: `ccw` 함수를 통해 각 선분과 점의 방향성을 계산합니다. CCW(Counter Clock Wise) 값이 0이면 세 점이 일직선상에 있다는 의미이고, 양수면 반시계 방향, 음수면 시계 방향입니다. `o1`, `o2`, `o3`, `o4`는 각각 다음을 나타냅니다:
   - `o1`: L1의 시작점에서 L1의 끝점, L2의 시작점에 대한 CCW
   - `o2`: L1의 시작점에서 L1의 끝점, L2의 끝점에 대한 CCW
   - `o3`: L2의 시작점에서 L2의 끝점, L1의 시작점에 대한 CCW
   - `o4`: L2의 시작점에서 L2의 끝점, L1의 끝점에 대한 CCW

3. **선분이 겹치는 경우 처리**: 모든 CCW 값이 0인 경우, 두 선분이 일직선상에 있는 경우입니다. 이 경우, 두 선분의 끝 점들이 서로 겹치는지를 검사합니다. 이를 위해 각 선분의 x, y 좌표의 최소값과 최대값을 비교합니다.

4. **교차 여부 판단**: CCW 값들이 서로 다른 부호를 가지면 선분이 교차하는 것으로 판단합니다. 즉, `o1 * o2 <= 0` 및 `o3 * o4 <= 0`이 참이면 두 선분이 교차합니다.

5. **결과 출력**: 교차 여부에 따라 1 또는 0을 출력합니다.

#### 예시 입력:
```
0 0 2 2
0 2 2 0
```
- 이 경우 `o1`, `o2`, `o3`, `o4`의 계산 결과는 각각 1, -1, -1, 1이므로 교차합니다. 출력은 `1`.

### 시간 복잡도
이 알고리즘의 시간 복잡도는 O(1)입니다. 선분의 교차 여부를 판단하기 위해 필요한 연산은 정해진 수의 산술 연산이므로 입력 크기와 무관하게 일정합니다.

## 코드
```python
import sys
input = sys.stdin.readline

def ccw(xa, ya, xb, yb, xc, yc):
    cross = (xb - xa) * (yc - ya) - (yb - ya) * (xc - xa)
    return cross

x1, y1, x2, y2 = map(int, input().split())
x3, y3, x4, y4 = map(int, input().split())

o1 = ccw(x1, y1, x2, y2, x3, y3)
o2 = ccw(x1, y1, x2, y2, x4, y4)
o3 = ccw(x3, y3, x4, y4, x1, y1)
o4 = ccw(x3, y3, x4, y4, x2, y2)

if o1 == 0 and o2 == 0 and o3 == 0 and o4 == 0:
    if (x1, y1) > (x2, y2):
        x1, x2 = x2, x1
        y1, y2 = y2, y1
    if (x3, y3) > (x4, y4):
        x3, x4 = x4, x3
        y3, y4 = y4, y3

    
    if max(min(x1,x2), min(x3,x4)) <= min(max(x1,x2), max(x3,x4)):
        if max(min(y1,y2), min(y3,y4)) <= min(max(y1,y2), max(y3,y4)):
            print(1)
            exit()
        else:
            print(0)
            exit()
    else:
        print(0)
        exit()

if o1 * o2 <= 0 and o3 * o4 <= 0:
    print(1)
    exit()
else:
    print(0)
    exit()

```
