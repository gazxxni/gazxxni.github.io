## 풀이

### 풀이 핵심 로직
이 코드는 주어진 숫자가 완전 제곱수인지 확인하는 문제를 해결합니다. 완전 제곱수란 어떤 정수를 제곱했을 때 얻어지는 수로, 예를 들어 1, 4, 9, 16 등이 이에 해당합니다. 이 코드에서는 각 숫자에 대해 제곱근을 구하고, 제곱근을 정수로 변환하여 다시 제곱해 원래 숫자와 비교함으로써 완전 제곱수를 판별합니다.

### 동작 과정
1. 사용자로부터 입력을 받아 `n`개의 숫자를 처리합니다.
2. 각 숫자에 대해 `search` 함수를 호출합니다.
3. `search` 함수는 다음과 같이 동작합니다:
   - 입력 숫자의 제곱근을 구한 후, 이를 정수로 변환합니다.
   - 이 정수를 제곱하여 원래 숫자와 비교합니다.
   - 같으면 완전 제곱수이므로 `True`를 반환하고, 그렇지 않으면 `False`를 반환합니다.
4. 각 숫자에 대해 결과를 출력합니다. `1`은 완전 제곱수, `0`은 그렇지 않음을 의미합니다.

예를 들어, 입력으로 `4 7 9`가 주어진 경우:
- `4`의 제곱근은 `2`이며, `2*2`는 `4`로 같으므로 출력은 `1`.
- `7`의 제곱근은 약 `2.645`로 정수로 바꾸면 `2`가 되고, `2*2`는 `4`로 다르므로 출력은 `0`.
- `9`의 제곱근은 `3`이며, `3*3`는 `9`로 같으므로 출력은 `1`.

결과적으로 출력은 `1 0 1`이 됩니다.

### 시간 복잡도
이 풀이의 시간 복잡도는 O(n)입니다. 각 숫자에 대해 제곱근을 계산하고 비교하는 작업이 O(1) 시간에 이루어지므로, n개의 숫자를 처리하는 데 전체 O(n) 시간이 소요됩니다.

## 코드
```python
import sys
input = sys.stdin.readline

def search(x):
    r = int(x**0.5)
    
    if r * r == x:
        return True
    else:
        return False

n = int(input().strip())

for _ in range(n):
    arr = list(map(int, input().split()))
    
    for i in arr:
            
        if search(i):
            print(1)
        else:
            print(0)

```
