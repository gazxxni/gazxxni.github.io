## 문제
베르트랑 공준은 임의의 자연수 n에 대하여, n보다 크고, 2n보다 작거나 같은 소수는 적어도 하나 존재한다는 내용을 담고 있다.

이 명제는 조제프 베르트랑이 1845년에 추측했고, 파프누티 체비쇼프가 1850년에 증명했다.

예를 들어, 10보다 크고, 20보다 작거나 같은 소수는 4개가 있다. (11, 13, 17, 19) 또, 14보다 크고, 28보다 작거나 같은 소수는 3개가 있다. (17,19, 23)

자연수 n이 주어졌을 때, n보다 크고, 2n보다 작거나 같은 소수의 개수를 구하는 프로그램을 작성하시오.

---

## 입력
입력은 여러 개의 테스트 케이스로 이루어져 있다. 각 케이스는 n을 포함하는 한 줄로 이루어져 있다.

입력의 마지막에는 0이 주어진다.

---

## 출력
각 테스트 케이스에 대해서, n보다 크고, 2n보다 작거나 같은 소수의 개수를 출력한다.

## 풀이

### 풀이 핵심 로직
베르트랑 공준에 따라, 주어진 자연수 n에 대해 n보다 크고 2n보다 작거나 같은 소수를 찾기 위해, 먼저 에라토스테네스의 체를 사용하여 최대 2n까지의 소수를 미리 계산합니다. 이후 각 테스트 케이스에 대해 n과 2n 사이의 소수 개수를 카운트하여 출력합니다.

### 동작 과정
1. **입력 처리**: 여러 개의 테스트 케이스를 입력받으며, 0이 입력되면 종료합니다.
2. **소수 계산**: `prime_numbers` 함수를 호출하여 최대 300,000까지의 소수를 미리 계산합니다. 이 함수는 소수 여부를 나타내는 불리언 배열을 반환합니다.
3. **소수 개수 세기**: 각 n에 대해 n+1부터 2n까지의 범위에서 소수인 숫자를 카운트합니다.
4. **출력**: 각 테스트 케이스에 대해 소수의 개수를 출력합니다.

예를 들어, 입력이 `10`이라면:
- n = 10일 때, n+1 = 11, 2n = 20입니다.
- 11, 13, 17, 19가 소수이므로, 총 4개의 소수가 존재합니다.

### 시간 복잡도
- 에라토스테네스의 체를 사용하여 소수를 계산하는 과정의 시간 복잡도는 O(n log log n)입니다. 
- 각 테스트 케이스에 대해 n과 2n 사이의 소수를 카운트하는 과정은 최악의 경우 O(n)입니다.
- 전체적으로 볼 때, 각 테스트 케이스에 대한 소수 개수 세는 과정은 O(n)이며, 에라토스테네스의 체에 의해 미리 계산된 배열을 사용하는 것이므로 매우 효율적입니다.

## 코드
```python
def prime_numbers(n):
    arr = [True] * (n + 1) 
    arr[0] = arr[1] = False

    for i in range(2, int(n ** 0.5) + 1):
        if arr[i]:
            for j in range(i * i, n + 1, i):
                arr[j] = False
    
    return arr

a = prime_numbers(300000)

while True:
    n = int(input())
    
    if n == 0:
        exit()
        
    cnt = 0
    for i in range(n + 1, 2 * n + 1):
        if a[i] == True:
            cnt += 1
    
    print(cnt)
```
