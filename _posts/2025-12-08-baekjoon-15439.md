## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
베라의 상의와 하의는 각각 N벌씩 있으며, 각 의상의 색상은 서로 다릅니다. 상의와 하의가 서로 다른 색상인 조합의 수는 각 상의에 대해 선택할 수 있는 하의의 수를 곱하여 구할 수 있습니다. 즉, 상의가 i 색상일 때, 하의는 i 색상이 아닌 (N-1)개의 색상 중 하나를 선택해야 하므로 총 조합 수는 N * (N - 1)입니다.

### 동작 과정
1. 입력으로 N을 받습니다. (예: N = 3)
2. 상의와 하의의 색상 조합을 구합니다.
   - 상의 색상 1을 선택하면 하의는 색상 2 또는 3 중 선택 가능 → 2가지
   - 상의 색상 2를 선택하면 하의는 색상 1 또는 3 중 선택 가능 → 2가지
   - 상의 색상 3을 선택하면 하의는 색상 1 또는 2 중 선택 가능 → 2가지
3. 따라서, 총 조합 수는 3 (상의 색상) * 2 (하의 선택 가능 색상) = 6이 됩니다.
4. 결과를 출력합니다.

### 시간 복잡도
이 알고리즘은 상수 시간에 계산이 완료되므로 시간 복잡도는 O(1)입니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
