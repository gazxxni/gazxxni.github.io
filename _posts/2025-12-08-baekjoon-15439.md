## 문제
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?

---

## 입력
입력은 아래와 같이 주어진다.

N

---

## 출력
상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.

## 풀이

### 풀이 핵심 로직
베라는 N벌의 상의와 N벌의 하의를 가지고 있으며, 상의와 하의는 각각 서로 다른 색상을 가지고 있습니다. 상의와 하의가 서로 다른 색상인 조합을 만들기 위해서는 상의를 선택한 후, 그와 다른 색상의 하의를 선택해야 합니다. 따라서, 상의의 선택지 N개에 대해 각 상의에 대해 N-1개의 하의를 선택할 수 있으므로 조합의 수는 N * (N - 1)입니다.

### 동작 과정
1. 입력으로 N(상의와 하의의 개수)을 받습니다.
2. 상의의 색상 중 하나를 선택할 수 있는 경우의 수는 N입니다.
3. 선택한 상의와 다른 색상을 가진 하의를 선택할 수 있는 경우의 수는 N - 1입니다.
4. 따라서 총 조합의 수는 N * (N - 1)입니다.

예를 들어, N이 3이라면:
- 상의 색상: 1, 2, 3
- 하의 색상: 1, 2, 3
- 상의 1을 선택하면, 하의로 2 또는 3을 선택할 수 있어 2가지 조합이 가능합니다.
- 상의 2를 선택하면, 하의로 1 또는 3을 선택할 수 있어 또 2가지 조합이 가능합니다.
- 상의 3을 선택하면, 하의로 1 또는 2를 선택할 수 있어 또 2가지 조합이 가능하므로 총 6가지 조합(3 * 2)입니다.

### 시간 복잡도
O(1) - 입력 N에 상관없이 상수 시간 내에 결과를 계산할 수 있습니다.

---

## 코드
```python
n = int(input())

print(n * (n - 1))
```
